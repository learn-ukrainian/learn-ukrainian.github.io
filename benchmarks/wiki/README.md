# Wiki Dimensional-Review Benchmark Corpus

Holds the seeded-defect corpus used by `scripts/wiki/benchmark.py` to
freeze per-dim thresholds and agent assignments. Design: see
[`docs/design/dimensional-review.md`](../../docs/design/dimensional-review.md) §7b.

## Layout

Each case is a directory with one clean article and one or more
defective variants:

```
benchmarks/wiki/
├── README.md                        ← this file
├── .example/                        ← template; driver skips dotdirs
│   ├── clean.md                     ← hand-authored clean article
│   ├── defects-light.yaml           ← 2-3 defects
│   ├── defects-heavy.yaml           ← 8-10 defects
│   ├── light/
│   │   ├── defective.md             ← generated by benchmark_seed.py
│   │   └── ground_truth.yaml        ← generated by benchmark_seed.py
│   └── heavy/
│       ├── defective.md
│       └── ground_truth.yaml
├── aspect/                          ← real case
│   └── ...
└── vocative/                        ← real case
    └── ...
```

## Authoring a new case

1. Copy a real compiled wiki article to `benchmarks/wiki/{case}/clean.md`.
2. Write a defect spec (`defects-light.yaml`, `defects-heavy.yaml`) —
   see `.example/defects-light.yaml` for the schema.
3. Generate defective variants:
   ```bash
   .venv/bin/python scripts/wiki/benchmark_seed.py \
       --clean benchmarks/wiki/aspect/clean.md \
       --defects benchmarks/wiki/aspect/defects-light.yaml \
       --out-article benchmarks/wiki/aspect/light/defective.md \
       --out-truth benchmarks/wiki/aspect/light/ground_truth.yaml
   ```
4. Verify (no-op run that checks every `find:` string is present):
   ```bash
   .venv/bin/python scripts/wiki/benchmark_seed.py \
       --clean benchmarks/wiki/aspect/clean.md \
       --defects benchmarks/wiki/aspect/defects-light.yaml \
       --verify
   ```

## Running the benchmark

Smoke test (1 case × 1 variant × 1 agent × 1 dim × 1 rerun — proves wiring):

```bash
.venv/bin/python scripts/wiki/benchmark.py --corpus benchmarks/wiki --smoke
```

Full run (cost: ~540 calls per §7b — check budget first):

```bash
.venv/bin/python scripts/wiki/benchmark.py \
    --corpus benchmarks/wiki \
    --agents claude,gemini,codex \
    --reruns 3 \
    --out benchmarks/wiki/.results/$(date +%Y-%m-%d).json
```
