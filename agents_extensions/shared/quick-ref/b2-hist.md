# B2 History Quick Reference

This quick reference is retired.

B2 no longer uses the old relocated-history quick-ref as the seminar source of
truth. Use:

- `docs/prompts/orchestrators/hist/suite-orchestrator.md` for HIST seminar work.
- `docs/prompts/orchestrators/b2/production-build-orchestrator.md` for active B2
  core production work.
- `docs/prompts/orchestrators/shared/seminar-source-rules.md` and
  `docs/prompts/orchestrators/shared/reading-section-rules.md` when the work is
  seminar-source-heavy.

The old references to `docs/l2-uk-en/templates/ai/history-module-template.md`
and `docs/l2-uk-en/templates/history-synthesis-module-template.md` were stale.
For structural compliance on legacy B2 history-style modules, the mapping file
now points to existing `b2-history-*` templates.
