# BIO Quick Reference

This quick reference is retired.

BIO is no longer a C1 template-driven lane. Use the current seminar prompt
suite instead:

- `docs/prompts/orchestrators/bio/suite-orchestrator.md`
- `docs/prompts/orchestrators/shared/seminar-source-rules.md`
- `docs/prompts/orchestrators/shared/seminar-track-checklists.md`
- `docs/templates/bio-research-dossier-template.md`

Do not use `docs/l2-uk-en/templates/*biography*` or the old 9-phase
`/module bio` workflow for BIO production. Current BIO work uses source-tier
dossiers, V2 `inline`/`workbook` activities, generated MDX parity checks,
independent review, telemetry for module builds, and
`scripts/validate/bio_surface_quality.py` for leakage/fingerprint reporting.
