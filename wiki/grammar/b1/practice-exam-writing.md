## Як це пояснюють у школі

Українська шкільна програма розглядає навички письма та говоріння як комплексний процес, що об'єднує макроструктуру (побудова тексту) та мікрорівень (милозвучність, орфографія, лексична точність). У 10 класі учні концептуально вивчають текст як цілісний продукт мовленнєвої діяльності [S1]. Текст характеризується цілісністю, зв’язністю (лінійною та вертикальною), членованістю, лінійністю, інформаційністю та завершеністю [S1]. Текстоутворювальні засоби вимагають від автора чіткого членування матеріалу на зачин, основну частину та висновки [S1].

На більш ранніх етапах (5–6 класи) закладається міцний фундамент фонетичної та орфографічної грамотності. У 5 класі детально розглядаються закони української милозвучності: чергування голосних і приголосних, зміна `у/в`, `і/й`, а також використання варіантів прийменника `з/із/зі` [S4]. Саме ці фонетичні інструменти забезпечують плавність говоріння і відсутність акустично важких збігів приголосних, що є критично важливим критерієм для оцінювання усного та писемного мовлення на рівні B1. Водночас формуються стійкі правописні навички: уживання м’якого знака, апострофа, подвоєння та подовження приголосних, а також їх спрощення (наприклад, у слові *радісний*) [S4]. Згідно з вимогами до культури мовлення, учень має повсякчас дотримуватися літературних норм вимови, чистоти, логічності та доречності висловлення [S2].

У 6 класі фокус зміщується на лексичне багатство української мови. Учні засвоюють різницю між синонімами й антонімами, вчаться доречно використовувати фразеологізми, неологізми (на кшталт *кіберпартизан*, *дистанційка*) та питомі українські фемінітиви [S3]. Тут також розглядаються займенники як ключовий засіб лінійної зв’язності речень у тексті, що дозволяє усувати лексичну тавтологію [S5]. Врешті-решт, узагальнення з лексикології закріплює правильне використання слів та запобігає плутанині між паронімами (як-от *освічений* та *освітлений*) і вживанню кальок [S6].

## Повна парадигма

### 1. Текстоутворювальні категорії
Для успішного написання есе чи продукування монологу на екзамені B1, автор має чітко структурувати текст і дотримуватися композиційних елементів [S1]:

| Структурна частина | Опис та функція |
| --- | --- |
| **Зачин** | Початок думки або теми. Зазвичай концентрується в першому реченні, яке може виділятися в окремий абзац. |
| **Основна частина** | Розвиток і детальний виклад теми. Поділяється на мікротеми (абзаци). Між ними встановлюються причиново-наслідкові та логічні відношення. |
| **Кінцівка** | Загальний підсумок усього висловлення з погляду як змісту, так і синтаксичної структури. |

*Обмеження:* Структурна організація тексту жорстко залежить від типу мовлення, стилістичних завдань і жанру (художній чи науковий стилі вимагають різних мовних засобів для передачі тієї самої інформації) [S1].

### 2. Орфографічні ланцюжки: Спрощення, Апостроф, Подвоєння
Усне мовлення B1 має базуватися на бездоганній артикуляції, а письмове — на міцній орфографії. Правила подаються як ланцюжки механічних перетворень.

**Спрощення в групах приголосних:** [S4]
- `{іменникова основа} → {основа} → {+суфікс -н- або -лив-} → {спрощена форма}`
- `радість → радіст- → + -н- + -ий → радісний` (випадає `[т]`).
- `тиждень → тижд- → + -н- + -ев- + -ий → тижневий` (випадає `[д]`).
- `щастя → щаст- → + -лив- + -ий → щасливий` (випадає `[т]`).
*Обмеження:* Спрощення відбувається у вимові, але НЕ фіксується на письмі у словах-винятках: `кістлявий, хвастливий, шістнадцять, зап'ястний, контрастний` [S4].

**Творення питомих фемінітивів:** [S3]
- `{іменник ч.р.} → {основа} → {+суфікс -к-, -иц-, -ин-, -ес-} → {фемінітив}`
- `автор → автор- → + -к- + -а → авторка`.
- `майстер → майстер- → + -иня → майстриня`.
*Обмеження:* Від деяких іменників нормативно утворювати паралельні варіанти фемінітивів з однаковим значенням: `акторка` й `актриса` [S3].

**Подвоєння букв (на межі морфем):** [S4]
- `{основа на приголосний} → {основа} → {+суфікс або корінь на той самий приголосний} → {подвоєння}`
- `екран → екран- → + -н- + -ий → екранний`.
- `від- → від- + дати → віддати`.
*Обмеження:* У прикметниках, утворених за допомогою суфіксів `-ан- (-ян-), -ин-, -їн-`, подвоєння не відбувається: `глина → глин- + -ян- + -ий → глиняний`, `соловей → солов- + -їн- + -ий → солов'їний` [S4].

**Подовження приголосних між голосними:** [S4]
- `{основа іменника ж.р.} → {основа на м'який приголосний} → {+закінчення Орудного відмінка -ю} → {подовжена форма}`
- `тінь → тін- → + -ю → тінню`.
- `сіль → сіл- → + -ю → сіллю`.
*Обмеження:* Подовження НЕ відбувається, якщо основа закінчується на губний приголосний або `[р]`: `любов → любов- + -ю → любов'ю`, `матір'ю` [S4]. В іменниках середнього роду на `-я` подовження не відбувається в назвах малих за віком тварин: `гусеня`, `вовченя` [S4].

**Вживання апострофа:** [S4]
- `{основа на губний б, п, в, м, ф} → {основа} → {+суфікс/закінчення на я, ю, є, ї} → {апостроф}`
- `соловей → солов- → + -ї → солов'ї`.
- `в'язати → в- → + ' + язати → в'язати`.
*Обмеження:* Якщо перед губним звуком стоїть інший приголосний, що належить до кореня (окрім `[р]`), апостроф не пишеться: `свято → свят- + -о → свято`, `цвях` [S4].

### 3. Творення займенників (засіб лінійної зв'язності тексту) [S5]
Займенники запобігають лексичній тавтології та зв'язують речення.
- `{питальний займенник} → {префікс аби-, де-, ні-} → {написання разом}`: `хто → аби- + хто → абихто`.
- `{питальний займенник} → {префікс казна-, хтозна-, будь-} → {написання через дефіс}`: `що → будь- + що → будь-що`.
*Обмеження:* Якщо між префіксом і займенником з'являється прийменник, усі три слова пишуться окремо: `будь-хто`, але `будь до кого`; `хтозна-який`, але `хтозна з ким` [S5].

### 4. Парадигма лексичної норми [S6]
Сучасний екзаменаційний текст вимагає використання нормативних відповідників до поширених іншомовних або спотворених слів.

| Іншомовне слово / Калька | Питомий український відповідник |
| --- | --- |
| мурал | стінопис |
| баг | помилка |
| анонс | оголошення |
| стикер | наліпка |
| піцца | піца (не подвоюємо приголосні в загальних назвах іншомовного походження) |
| вірна відповідь | правильна відповідь |

### 5. Засоби милозвучності (Фонетичні чергування)
Українська мова принципово уникає важких скупчень приголосних. 

**Чергування У / В** [S4]
| Позиція | Вживаємо У | Вживаємо В |
| --- | --- | --- |
| Між голосними/приголосними | Між двома приголосними (`вечір у місті`) | Між двома голосними (`живе в Одесі`) |
| На початку речення | Перед приголосним (`У лісі тепло`) | Перед голосним (`В Одесі тепло`) |
| Перед специфічними буквами | Перед `в, ф, зв, св, дв, тв, хв, гв, льв` (`живе у Львові`) | Після голосного перед більшістю приголосних (`пішла в садок`) |
| Після паузи (коми, крапки) | Перед приголосним (`Знаю, у чому секрет.`) | Після приголосного перед голосним (`човен в очереті`) |

*Обмеження щодо З / ІЗ / ЗІ:* Форма `ЗІ` вживається жорстко перед сполученням приголосних, якщо першим є `[з], [с], [ш]` (`вийшли зі школи`) [S4]. Варіант `ЗО` можливий лише з числівниками `два`, `три` (`разів зо два`) [S4].

## Частотність і пріоритети

На рівні B1 пріоритетом номер один є макроструктура — текстова цілісність та логічна побудова. Студент повинен уміти чітко розмежовувати зачин, основну частину і кінцівку есе чи усного монологу [S1]. 
Другим за важливістю фактором є фонетична милозвучність (чергування `у/в`, `і/й`, `з/із/зі`). Ці явища мають найвищу частотність у реальному використанні, адже стосуються ледь не кожного українського речення. Швидке говоріння і письмо без автоматичного дотримання цих правил звучить неприродно і ламає ритміку [S4]. Складні орфограми (подовження та подвоєння приголосних) трапляються рідше, проте вони є ключовими маркерами глибокого засвоєння правописних норм і напряму впливають на підсумковий бал за письмову частину екзамену.

## Деколонізаційні застереження

Українська лінгводидактика протягом тривалого часу потерпала від колоніальної спадщини, коли правила української мови викладалися через призму російської граматики. Це призводило до формування хибних стереотипів, які слід рішуче відкидати при підготовці студентів.

Перш за все, слід наголосити на принциповій різниці в підході до милозвучності. В українській мові чергування `у/в` та `і/й` — це жорстка системна вимога, що забезпечує плавний вокалічний ритм мовлення [S4]. Російськомовна традиція звикла ігнорувати такі акустичні потреби, жорстко закріплюючи прийменники "в" і "у" виключно за їхнім лексичним значенням (напрямок проти приналежності). Пояснюючи цю тему англомовним учням, категорично заборонено проводити паралелі з російською граматикою. Українські прийменники є фонетичними дублетами, і їх вибір зумовлений лише сусідніми звуками.

Другим важливим аспектом є культура слововживання [S2]. Радянська школа часто таврувала питомі українські слова як "діалектизми" чи "архаїзми", нав'язуючи кальки з російської. При підготовці екзаменаційних текстів учні повинні активно використовувати органічні синоніми, неологізми та фемінітиви [S3], рішуче уникаючи росіянізмів. Наприклад, слід використовувати нормативне **прізвище**, а не розмовно-застаріле `фамілія` <!-- VERIFY: surname_name_term — СУМ-11 позначає «фамілія» як розм./заст./діал.; нормативне — прізвище Canonical form: «прізвище» -->, а при зустрічі вітатися **Добрий день** замість маргінального `Здрастуйте` <!-- VERIFY: greeting_hello_formal — Прямий русизм (рос. здравствуйте); VESUM класифікує як розмовне Canonical form: «Добрий день» -->. Звертаючись до батька, дитина каже **тато** (а не кальковане `папа`). Також замість російського `приймати участь` необхідно говорити питоме `брати участь` [S3]. Для позначення віку слід використовувати правильну українську конструкцію з давальним відмінком особи: **Мені 20 років**, а не калькувати європейське `Я маю 20 років` <!-- VERIFY: age_model — Калька рос. Мне N лет + англ. I have N years; українська — датив особи + кількість Canonical form: «Мені [N] років» -->. При прощанні слід вживати **До побачення**, рішуче уникаючи запозиченого `Пока` <!-- VERIFY: farewell_goodbye — Рос. запозичення; нейтральний варіант — «До побачення» Canonical form: «До побачення» -->.

У текстах екзаменаційного рівня B1 студенти часто обирають соціокультурні, історичні або політичні теми. При їх розкритті необхідно суворо дотримуватися національно свідомої термінології. Державний прапор нашої країни — це виключно **синьо-жовтий** стяг (порядок слів є непорушним). Малий Державний Герб України — це **Тризуб**, і будь-які алюзії на двоголових <!-- VERIFY: trident — Двоголовий orел is the Russian imperial symbol, not Ukrainian Canonical form: «Тризуб» --> орлів тут є ворожими імперськими наративами. Столиця держави — **Київ** (використання транслітераційного покруча `Kiev` <!-- VERIFY: capital — Kiev is the Russian-imperial romanization; Kyiv is the Ukrainian form (official English since 2018) Canonical form: «Київ» --> є абсолютно неприпустимим). Національна валюта — **гривня** (будь-які згадки про рубль <!-- VERIFY: currency — Рубль is the Russian/Soviet currency, not Ukrainian Canonical form: «гривня» --> не мають стосунку до України). Наша рідна мова — **українська мова**.

Згадуючи трагічні сторінки нашої історії, як-от Голодомор, необхідно вказувати точні дати **1932–1933** років, уникаючи абстрактних узагальнень на кшталт "у 1930-х", що штучно розмивають відповідальність колонізаторів. Якщо учень наводить приклади сучасних політичних діячів, варто пам'ятати, що Прем'єр-міністр України — **Юлія Свириденко**. Звертаючись до національного літературного надбання, слід пам'ятати, що безперечно український літературний канон становлять **Тарас Шевченко, Іван Франко, Леся Українка, Василь Стус, Ліна Костенко, Оксана Забужко, Юрій Андрухович, Сергій Жадан, Валер'ян Підмогильний**. Залучення авторів із російсько-імперською ідентичністю (наприклад, Анни Ахматової чи Михайла Булгакова) у ролі представників "української літератури" є абсолютно неприпустимим, адже їхній світогляд був ворожим до української державності.

## Природні приклади

Нижче наведені автентичні приклади з українських підручників, згруповані за моделлю застосування навичок.

**Модель 1: Текстова цілісність та займенникова зв'язність**
1. У місті за добу виявили шістнадцять нових випадків інфікування коронавірусом. [S4]
2. Я побачив стільки неймовірних мальовничих міст! Хочеш, розкажу? [S3]
3. У ньому багато приміщень з історичними декораціями. [S3]
4. Хто розуму не має, тому й коваль не вкує. [S5]

**Модель 2: Милозвучність (чергування у/в, і/й, з/із/зі)**
5. Вода у Світязі холодна, тому ми купалися влітку. [S4]
6. Садок уже одцвівся й густо вкрив землю білими пелюстками. [S4]
7. Бджоли сонячним дощем падають на медвяний лан. [S4]
8. Знаю, у чому секрет. [S4]

**Модель 3: Фонетичні варіанти та складна орфографія (апостроф, подовження)**
9. Дерево стояло ще голе, однак на вітті його вже починалося життя. [S4]
10. Маю я святе синівське право з матір'ю побуть на самоті. [S4]
11. Немає на світі нічого радіснішого й приємнішого, ніж людська праця. [S4]
12. Космічне агентство розробило суперракету для польоту астронавтів на Місяць. [S4]

**Модель 4: Лексичне багатство тексту (синоніми, сталі вислови)**
13. Запорожці дванадцятьма мовами вміли говорити, із води могли сухими виходити. [S3]
14. Без праці щасливий тільки лінивий. [S4]
15. Як мені пізніше розказав його помічник, Арнольд довго роздивлявся вишиванку. [S3]

## Рекомендації для вправ

Конструювання практичних завдань для модуля "Письмо і Говоріння (Екзамен)" має відбуватися за принципом поетапного нарощування складності: від мікрорівня форми до створення самостійного макротексту.

- **Фаза 1: Розпізнавання та механічне фонетичне перетворення.** Автор повинен створити серію вправ формату "заповніть пропуски", де учень має самостійно вибрати між варіантами `у/в`, `і/й`, `з/із/зі` залежно від акустичного оточення [S4]. Сюди ж належать завдання на вибір правильного лексичного відповідника або синоніма з ряду запропонованих для уникнення лексичної помилки (наприклад, вибір `правильна` замість `вірна відповідь` або `розгорнути` замість `відкрити підручник`) [S6].
- **Фаза 2: Редагування та текстова зв'язність.** Учням пропонується дефектний абзац, у якому порушена лінійна зв'язність через надмірну лексичну тавтологію [S1]. Завдання полягає у заміні повторюваних іменників відповідними займенниками (особовими, відносними, неозначеними) [S5] та паралельному виправленні навмисно закладених помилок у розстановці складних орфограм (подвоєння, м'який знак, апостроф) [S4].
- **Фаза 3: Кероване письмо.** Написання короткого структурованого есе (10–15 речень) за заданою мікротемою [S1]. Завдання повинно містити жорстку вимогу: використати у своєму тексті мінімум 3 фразеологізми, 2 питомі фемінітиви та правильно застосувати всі правила української милозвучності без винятків [S3]. Автор модуля має чітко передбачити прозорі критерії перевірки для таких інтегрованих завдань.

## Зв'язки з іншими темами

Ця граматична та стилістична конфігурація безпосередньо спирається на базові фонетичні навички рівня A1–A2, де студенти засвоювали основи вимови та відмінкові закінчення. Без глибокого розуміння фонетичної різниці між голосними та приголосними звуками учень не зможе правильно застосувати правила спрощення чи подовження приголосних [S4].
Водночас навички текстоутворення, здобуті тут, є фундаментальними для опанування складніших синтаксичних конструкцій рівня B2–C1. Зокрема, уміння правильно встановлювати причиново-наслідкові відношення між абзацами (забезпечення вертикальної зв'язності) [S1] дозволить у майбутньому легко імплементувати складнопідрядні речення експлікативного й допустового типу. Сформована орфографічна дисципліна також готує міцний фундамент для професійного академічного письма, де будь-які пунктуаційні чи лексичні помилки кардинально знижують загальний комунікативний бал. Упевнене володіння займенниковими формами як інструментом зв'язності [S5] відкриває шлях до розуміння складного синтаксису, властивого публіцистичному та суворо науковому стилям, що будуть вивчатися пізніше.
