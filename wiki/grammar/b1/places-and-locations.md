## Як це пояснюють у школі

Тема опису місця та просторового розташування є наскрізною в українській шкільній програмі, поступово ускладнюючись від базових прийменників до комплексних синтаксичних структур та офіційно-ділових форматів. Для автора-письменника розуміння цієї прогресії є ключовим для створення релевантного контенту для учнів рівня B1.

Уже в 2 класі учні знайомляться з базовими просторовими орієнтирами в художніх текстах, використовуючи прийменники Місцевого та Орудного відмінків для опису природи: птахи сидять «на гілці», відлітають «за морями» або «на півдні» [S2]. Це закладає фундамент розуміння статики. 

У 6 класі просторова термінологія активно інтегрується в міжпредметні зв'язки, зокрема на уроках історії. Учні стикаються з комплексними географічними описами, що вимагають розуміння Родового відмінка для локалізації («на південний схід від Балканського півострова», «вздовж західного узбережжя») [S10]. На уроках мови в цьому ж класі відпрацьовується словотвір і правопис, де просторові поняття використовуються як базові моделі («за лісом», «на річці») [S8].

У 7 класі фокус зміщується на практичне застосування просторової інформації в офіційно-діловому стилі. Учні вчаться правильно оформлювати поштову адресу. Шкільна програма чітко регламентує, що поштова адреса — це «розташування житла людини або установи... відносно країни, регіону (області), району, міста (села, селища), вулиці (провулку)» [S3]. Учням подають офіційні графічні скорочення: м. (місто), с. (село), вул. (вулиця), пров. (провулок), пл. (площа), кв. (квартира) [S3].

У 8 класі настає кульмінація теми у вигляді комунікативного завдання — «Твір-опис місцевості». Учні вчаться словесно зображувати подвір'я, площу, парк, вулицю чи місто, використовуючи просторові прислівники («навсібіч», «згори», «удалині», «навпроти», «неподалік») [S5] та багаті художні засоби: епітети, персоніфікації, метафори [S1]. Структура опису чітко регламентована: від загального місця розташування до деталей забудови, озеленення та загального враження [S1].

У старшій школі (9–10 класи) тема переходить у площину синтаксису та стилістики. У 9 класі просторові конструкції формально ідентифікуються як «обставина місця», що відповідає на питання «де?» («Сідає (де?) за горою — обставина місця») у складнопідрядних реченнях [S4]. У 10 класі учні вивчають лексичну сполучуваність та нормативність, зокрема виправляючи поширені помилки-кальки щодо опису місць (наприклад, заміну калькованого «живописна місцевість» на питоме «мальовнича місцевість») [S9].

## Повна парадигма

Опис розташування в українській мові реалізується через систему прийменників, які жорстко керують певними відмінками (Місцевим, Орудним або Родовим), та незалежних просторових прислівників.

### Таблиця 1: Прийменники Місцевого відмінка (Locative Case)
Використовуються для позначення знаходження на поверхні, всередині об'єкта або у певній стороні світу.

| Прийменник | Модель | Ланцюжок творення | Приклад із джерел |
| :--- | :--- | :--- | :--- |
| **на** | Поверхня (множина) | `{Н.в. мн.} → {основа} → {+ах / +ях} → {кінцева форма}` | `дахи → дах- → +ах → на дахах` [S5] |
| **на** | Сторона світу | `{Н.в. одн.} → {основа} → {+і / +ї} → {кінцева форма}` | `південь → півдн- → +і → на півдні` [S2] |
| **у / в** | Контейнер / ліс | `{Н.в. одн.} → {основа} → {+і} → {кінцева форма}` | `ліс → ліс- → +і → у лісі` [S1] |
| **у / в** | Чергування к/ц | `{Н.в. одн.} → {основа зі зміною к/ц} → {+і} → {кінцева форма}` | `річка → річк- → +і → у **річці**` [S1] |
| **у / в** | Чергування і/є | `{Н.в. одн.} → {основа зі зміною і/є} → {+і} → {кінцева форма}` | `Київ → Києв- → +і → у **Києві**` [S3] |

**Обмеження:** Прийменник `на` обов'язково використовується з назвами сторін світу (на півдні, на сході) [S2]. Використання `у/в` для сторін світу є неможливим.

### Таблиця 2: Прийменники Орудного відмінка (Instrumental Case)
Використовуються для позначення взаємного просторового розташування об'єктів по вертикалі або між собою.

| Прийменник | Модель | Ланцюжок творення | Приклад із джерел |
| :--- | :--- | :--- | :--- |
| **над** | Вертикаль (вище) | `{Н.в. одн.} → {основа} → {+ею / +ою} → {кінцева форма}` | `круча → круч- → +ею → над кручею` [S1] |
| **перед** | Фронтально | `{Займенник Н.в.} → {основа} → {+им} → {кінцева форма}` | `він → н- → +им → перед ним` [S1] |
| **за** | Позаду | `{Н.в. одн.} → {основа} → {+ою / +ею} → {кінцева форма}` | `гора → гор- → +ою → за горою` [S4] |
| **між** | Позиція посередині | `{Н.в. одн.} → {основа} → {+єю / +ою} → {кінцева форма}` | `Коломия → Коломий- → +єю → між Коломиєю` [S5] |

**Обмеження:** Прийменники `між`, `над`, `під`, `перед`, `за` вимагають Орудного відмінка ТІЛЬКИ тоді, коли позначають статичне місце і відповідають на питання «де?» [S4]. Якщо дієслово вказує на напрямок руху («куди?»), ці самі прийменники вимагають Знахідного відмінка (Accusative). 

### Таблиця 3: Прийменники Родового відмінка (Genitive Case)
Використовуються для вказівки на просторову близькість, межі або простір уздовж ліній.

| Прийменник | Модель | Ланцюжок творення | Приклад із джерел |
| :--- | :--- | :--- | :--- |
| **біля** | Близькість | `{Н.в. одн.} → {основа} → {+я / +а} → {кінцева форма}` | `підніжжя → підніжж- → +я → біля підніжжя` [S1] |
| **серед** | Оточення (множина) | `{Н.в. мн.} → {основа} → {+Ø} (нульове) → {кінцева форма}` | `дерева → дерев- → +Ø → серед дерев` [S1] |
| **уздовж** | Вздовж лінії | `{Н.в. одн.} → {основа} → {+і / +и} → {кінцева форма}` | `вулиця → вулиц- → +і → уздовж вулиці` [S5] |
| **від** | Відстань | `{Н.в. одн.} → {основа} → {+у / +а} → {кінцева форма}` | `центр → центр- → +у → від центру` [S2] |

**Обмеження:** Прийменник `уздовж` та `з-поміж` керують ВИКЛЮЧНО Родовим відмінком [S1, S5]. Використання з іншими відмінками неможливе.

### Таблиця 4: Утворення пасивних дієприкметників для опису локацій
Для опису місцевості часто використовуються дієприкметники.
| Статус | Ланцюжок творення | Приклад із джерел |
| :--- | :--- | :--- |
| **Дієприкметник** | `{інфінітив доконаного виду} → {основа} → {+ен / +н} + {закінчення} → {кінцева форма}` | `вимостити → вимост- (ст/щ) → +ен- + -а → вимощена` [S5] |

**Обмеження:** Пасивні дієприкметники минулого часу утворюються ТІЛЬКИ від перехідних дієслів (transitive verbs), які можуть мати прямий додаток. Від неперехідних дієслів такі форми не утворюються [S1, S5].

## Частотність і пріоритети

На рівні B1 найвищий пріоритет має бездоганне володіння Місцевим («в/на») та Родовим («біля», «уздовж», «серед») відмінками для статики. Саме ці конструкції забезпечують 80% потреб під час опису локації або навігації. Особливу увагу слід приділити вивченню просторових прислівників («навсібіч», «навпроти», «удалині», «згори» [S5]), оскільки вони дозволяють описувати місцевість без складного відмінкового керування залежними іменниками, роблячи мовлення іноземця значно природнішим.
Офіційно-ділова лексика (оформлення адреси з абревіатурами м., вул., кв. [S3]) має абсолютний пріоритет для практичного виживання в україномовному середовищі.
Складні географічні конструкції типу «на південний схід від» [S10] є менш частотними в побуті і можуть бути відкладені до рівня B2 (або засвоюватися пасивно).

## Деколонізаційні застереження

Автору-письменнику критично важливо уникати російсько-імперських лексичних і синтаксичних кальок під час створення модулів.
1. **Пароніми «адрес» та «адреса»:** У російській мові слово «адрес» позначає місце проживання. В українській — виключно «адреса» [S4]. Якщо учень каже «мій адрес», це маркер не лише англійської (address), але й російської інтерференції. Автор має створити вправи на чітке розрізнення цих паронімів.
2. **Калькування епітетів простору:** Вкрай поширеною помилкою є використання прикметника «живописна» (калька з російського «живописная») для опису ландшафту. Нормативним українським відповідником, який слід закладати в лексичний мінімум B1, є «мальовнича» місцевість [S9].
3. **Безприйменниковий Орудний простору:** Російська мова активно використовує конструкцію «по + Давальний/Місцевий відмінок» для позначення руху в просторі (идти по улице). В українській мові для опису місцевості та простору частіше використовується прийменник «уздовж» з Родовим відмінком («уздовж вулиці» [S5]) або автентичний безприйменниковий Орудний відмінок (іти вулицею, гуляти лісом).
4. **Топоніміка:** Назва столиці України — Київ. Неприпустимо використовувати імперські транслітерації (Kiev <!-- VERIFY: capital — Kiev is the Russian-imperial romanization; Kyiv is the Ukrainian form (official English since 2018) Canonical form: «Київ» -->). Усі приклади адрес і місцевостей мають спиратися на українську культурну географію.

## Рекомендації для вправ

Для якісного засвоєння теми автор має дотримуватися такої методичної послідовності:

1. **Фаза 1: Рецептивне розпізнавання (Receptive recognition)**
   Використай автентичний текст формату «твір-опис місцевості» (наприклад, опис площі або селянської садиби [S6]). Завдання для учня: знайти всі обставини місця (слова, що відповідають на питання «де?») та згрупувати їх: прийменникові конструкції в одну колонку, просторові прислівники («навсібіч», «удалині», «неподалік» [S5]) — в іншу. Це формує мовну здогадку.
2. **Фаза 2: Керована практика (Controlled practice)**
   Вправи на множинний вибір та заповнення пропусків. Учень має обрати правильний прийменник простору залежно від контексту та поставити іменник у правильний відмінок. Наприклад: завдання змушує відрізняти пароніми «адреса/адрес» [S4] та виправляти типову помилку у фразі «живописна місцевість» на «мальовнича місцевість» [S9]. Окремий блок вправ має бути присвячений сторонам світу («на півдні», а не «у півдні») [S2].
3. **Фаза 3: Продукція (Production)**
   - *Мікро-письмо:* Учень має записати власну (або вигадану) поштову адресу, чітко дотримуючись українського стандарту та графічних скорочень (м., вул., кв.) [S3].
   - *Комунікативне завдання:* Написати короткий гід-опис свого району для туриста, обов'язково використавши 3 прийменники (біля, серед, між) та 2 просторові прислівники (навпроти, неподалік). 

## Зв'язки з іншими темами

- **Передумови:** Тема спирається на знання форм Місцевого, Родового та Орудного відмінків, які учні вивчали на рівні A1–A2. Також потрібне базове розуміння прикметників для створення епітетів при описі (напр., «білостінні хати» [S1]).
- **Що уможливлює:** Володіння статичними просторовими прийменниками та концептом «обставини місця» (де?) [S4] є критичним містком для переходу до складної теми рівня B2 — «Дієслова руху» (Verbs of Motion), де відбувається зміщення фокусу зі статики («де?») на напрямок («куди?»), що повністю змінює відмінкове керування для прийменників в, на, між, під, над, перед, за.
