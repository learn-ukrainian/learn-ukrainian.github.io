## Як це пояснюють у школі (How Schools Teach This)
Активні дієприкметники — це форма дієслова, що виражає ознаку предмета за дією, яку цей предмет виконує сам [S39]. Ця тема зазвичай вводиться в 7 класі [S40][S41].

Шкільна програма відразу розмежовує два типи активних дієприкметників за часом: теперішній і минулий.

1.  **Активні дієприкметники минулого часу** (на `-лий`) подаються як продуктивна і нормативна частина мови. Правила їх утворення від основи інфінітива доконаного виду пояснюються чітко: `посивіти -> посивілий` [S42].

2.  **Активні дієприкметники теперішнього часу** (на `-учий, -ючий, -ачий, -ячий`) одразу характеризуються як такі, що мають **обмежене вживання** [S43][S44]. Підручники наголошують, що в сучасній мові вони "не властиві" [S45] або вживаються "рідко" [S46].

Основний акцент у викладанні робиться на **заміні** невластивих активних дієприкметників теперішнього часу на більш природні конструкції [S42]:
*   Підрядні речення: `танцюючі дівчата -> дівчата, які танцюють`.
*   Іменники: `керуючий -> керівник`.
*   Прикметники: `наступаюче свято -> прийдешнє свято`, `миючий засіб -> мийний засіб`.

Особлива увага приділяється повному запереченню форм на `-вший, -ший` як грубого порушення мовної норми [S43][S45].

## Повна парадигма (Full Paradigm)

Активні дієприкметники виражають ознаку предмета, який сам виконує дію (`пожовкле листя` — листя саме пожовкло) [S46]. Вони бувають теперішнього та минулого часу.

### Активні дієприкметники теперішнього часу

Утворюються від основи дієслова теперішнього часу (форми 3-ої особи множини без закінчення) недоконаного виду [S47].

| Дієвідміна | Основа (3 ос. мн.) | Суфікс | Приклад дієприкметника |
| :--- | :--- | :--- | :--- |
| **I дієвідміна** | `зна-ють` | **-уч- / -юч-** | `зна**юч**ий` |
| | `нес-уть` | | `нес**уч**ий` |
| | `працю-ють` | | `працю**юч**ий` |
| **II дієвідміна** | `леж-ать` | **-ач- / -яч-** | `леж**ач**ий` |
| | `сто-ять` | | `сто**яч**ий` |
| | `тремт-ять` | | `тремт**яч**ий` |

[S42][S39]

### Активні дієприкметники минулого часу

Утворюються від основи інфінітива неперехідних дієслів **доконаного виду** за допомогою суфікса **-л-** [S39][S47].

| Інфінітив (доконаний вид) | Основа інфінітива | Суфікс | Приклад дієприкметника | Примітка |
| :--- | :--- | :--- | :--- | :--- |
| `посивіти` | `посиві-` | **-л-** | `посиві**л**ий` | |
| `зів'янути` | `зів'я-` | **-л-** | `зів'я**л**ий` | Суфікс `-ну-` випадає |
| `розквітнути` | `розквіт-` | **-л-** | `розквіт**л**ий` | Суфікс `-ну-` випадає |
| `змерзнути` | `змерз-` | **-л-** | `змерз**л**ий` | Суфікс `-ну-` випадає |
| `засохнути` | `засох-` | **-л-** | `засох**л**ий` | Суфікс `-ну-` випадає |

[S42][S39][S47]

## Частотність і пріоритети

1.  **Високий пріоритет: Активні дієприкметники минулого часу (`-лий`)**. Ці форми є абсолютно нормативними, стилістично нейтральними і широко вживаними. Вони є ключовим інструментом для опису стану, що настав у результаті попередньої дії (`опале листя`, `почорніла земля`, `промоклий дах`). Учень на рівні B1 повинен вміти їх вільно утворювати і використовувати.

2.  **Низький пріоритет (для творення), високий (для розпізнавання): Активні дієприкметники теперішнього часу (`-учий`/`-ячий`)**. Сучасна літературна мова активно уникає цих форм, вважаючи їх книжними, застарілими або невдалими кальками [S44]. Їх продуктивне використання не є метою для учня. Натомість, учень повинен:
    *   **Розпізнавати** їх у текстах, особливо в науковому та художньому стилях (`блукаючий нерв`, `синіючі далі`) [S48].
    *   **Знати усталені форми**, що перейшли у прикметники і позначають постійну ознаку: `лежачий камінь`, `співучий народ`, `палюче сонце`, `живуча істота` [S49].
    *   **Активно володіти механізмами заміни** цих форм на природні відповідники (підрядні речення, прикметники, іменники). Це є головним пріоритетом на рівні B1/B2.

## Типові помилки L2

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я побачив дівчину, читаючу книгу.` | `Я побачив дівчину, яка читала книгу.` | Це пряма калька з російської мови. В українській мові для опису тимчасової дії перевага надається підрядним реченням з **який, що**. [S46] |
| `Люди, бажаючі поїхати, повинні купити квиток.` | `Люди, які бажають поїхати, повинні купити квиток.` АБО `Охочі поїхати повинні купити квиток.` | Слово `бажаючий` є типовим канцеляризмом. Краще використовувати підрядне речення або прикметник/іменник `охочий`. [S44] |
| `Це був зробивший домашнє завдання учень.` | `Це був учень, що зробив домашнє завдання.` | Суфікси **-вш-, -ш-** для творення активних дієприкметників в українській мові **не використовуються**. Це грубе порушення норми. [S43][S45] |
| `Де тут миючий засіб?` | `Де тут мийний засіб?` | Для позначення призначення предмета часто вживаються прикметники з суфіксом **-н-** або **-альн-**. `Миючий` означало б "той, що миє просто зараз". [S45][S50] |
| `Керуючий проєкту прийшов на зустріч.` | `Керівник проєкту прийшов на зустріч.` | Якщо для позначення діяча існує спеціальний іменник, перевага завжди надається йому, а не дієприкметнику. [S46] |
| `Слідуюча зупинка — "Університет".` | `Наступна зупинка — "Університет".` | `Слідуючий` — калька. В українській мові для позначення порядку в черзі використовується слово `наступний`. [S49] |

## Деколонізаційні застереження

**Це ключовий аспект теми.** На відміну від російської мови, де активні дієприкметники теперішнього (`-ущ-`/`-ющ-`, `-ащ-`/`-ящ-`) та минулого (`-вш-`/`-ш-`) часу є абсолютно нормативними і широко вживаними, сучасна українська літературна мова має значно суворіші обмеження.

1.  **Заборона суфіксів `-вш-` та `-ш-`:** Форми на кшталт `посинівший`, `сказавший`, `прийшовший` є **категорично неправильними** в українській мові. Це груба помилка і яскравий маркер російського впливу. Підручники прямо називають їх "грубим порушенням мовної норми" [S45] та зазначають, що "в українській мові для творення дієприкметників не використовуємо" ці суфікси [S42].

2.  **Обмеженість суфіксів `-уч-`/`-юч-`, `-ач-`/`-яч-`:** На відміну від російської, де `читающий`, `лежащий`, `говорящий` є нейтральними, відповідні українські форми `читаючий`, `лежачий`, `говорячий` є стилістично маркованими (книжними, поетичними) або обмеженими у вжитку. Форми типу `ідучий`, `бігучий`, `кричащий`, `говорячий` в українській мові взагалі не утворюються або не вживаються [S48].

3.  **Історична перспектива:** Обмеження на активні дієприкметники — це результат природного розвитку української мови, а не її "неповноцінність". Давньоруська мова мала систему дієприкметників, але з часом короткі активні дієприкметники теперішнього часу перетворилися на дієприслівники (`несучи`, `носѧчи` -> `несучи`, `носячи`) [S51]. Деякі форми (`лежачий`) були присутні в староукраїнській діловій мові, можливо, як калька з польської (`leżący`), але вони були адаптовані українським суфіксом `-ачий` [S52]. Це свідчить про власну історію розвитку цих форм, відмінну від російської.

**Інструкція для письменника:** Не можна пояснювати українські дієприкметники через порівняння з російськими. Українська граматика є самодостатньою. Слід наголошувати на властивих українській мові способах вираження думки, таких як підрядні речення, які є більш природними та поширеними.

## Природні приклади

### Група 1: Активні дієприкметники минулого часу (-л-)
Ці форми є стилістично нейтральними та поширеними.
1. `Спорожнілий колос вище за всіх стоїть.` (Народний вислів [S53])
2. `Попадало додолу покручене, зчорніле листя.` [S42]
3. `Солодким хлібом пахнуть стерні, ... і чуйно соняшник поверне пожовклу голову на схід.` (М. Стельмах [S47])
4. `Стиха лущиться зерно з перестиглого колоса.` (М. Коцюбинський [S54])

### Група 2: Активні дієприкметники теперішнього часу, що стали прикметниками (постійна ознака)
Ці форми втратили дієслівне значення часу і позначають сталу властивість.
5. `Під лежачий камінь вода не тече.` (Народна творчість [S46])
6. `Край шляху серед палаючих маків зостався свіжий горбик землі.` (Олесь Гончар [S54])
7. `Українська нація — це співоча нація.` (Порівняння з `співучий народ` [S49])

### Група 3: Заміна дієприкметника на підрядне речення
Це найпоширеніший і найприродніший спосіб уникнути невдалих дієприкметників.
8. ❌ `Танцюючі дівчата` → ✅ `Дівчата, які танцюють.` (Зразок з підручника [S42])
9. ❌ `Щури тікають з потопаючого корабля.` → ✅ `Щури тікають з корабля, що потопає.` [S45]
10. ❌ `...дівчина, сказавша...` → ✅ `Дівчина, яка сказала...` [S50]

### Група 4: Заміна дієприкметника на іменник або інший прикметник
Для уникнення канцеляризмів та суржику.
11. ❌ `завідуючий кафедрою` → ✅ `завідувач кафедри` [S45]
12. ❌ `знеболюючий засіб` → ✅ `знеболювальний засіб` [S45]
13. ❌ `наступаюче свято` → ✅ `прийдешнє свято` [S44]

## Рекомендації для вправ

- **Phase 1: Ідентифікація та аналіз.** Надати учням речення з різними дієприкметниками (`посивілий`, `написаний`, `лежачий`, `зроблений`). Завдання: визначити стан (активний/пасивний) і час (теперішній/минулий). Це закріпить базову теорію.

- **Phase 2: Утворення нормативних форм.** Дати список дієслів доконаного виду (`засохнути`, `пожовтіти`, `опасти`, `зникнути`). Завдання: утворити активні дієприкметники минулого часу (`засохлий`, `пожовклий`, `опалий`, `зниклий`). (На основі вправи 2 [S42]).

- **Phase 3: Редагування та заміна (ключова вправа).** Надати учням речення з ненормативними або стилістично невдалими активними дієприкметниками теперішнього часу. Завдання: відредагувати речення, використовуючи природні відповідники.
    - `Стоячі місця` -> `Місця для сидіння` [S53]
    - `Хвилюючі слова` -> `Зворушливі слова` [S53]
    - `Діючий закон` -> `Чинний закон` [S45]
    - `Плачуча дитина` -> `Дитина, яка плаче` [S49]

- **Phase 4: Контекстне застосування.** Запропонувати написати короткий текст (5-7 речень) на задану тему, використовуючи утворені на Phase 2 дієприкметники (`пожовкле листя`, `зів'ялі трави` тощо) та уникаючи ненормативних форм. Це перевірить засвоєння матеріалу на практиці.

## Зв'язки з іншими темами

- **Передумови:**
    - **Дієвідміни дієслів:** Необхідно для визначення правильного суфікса (`-уч-`/`-юч-` для I дієвідміни, `-ач-`/`-яч-` для II). [S39]
    - **Вид дієслова:** Активні дієприкметники теперішнього часу утворюються від дієслів недоконаного виду, а минулого — від доконаного. [S47]
    - **Основа дієслова:** Потрібно вміти визначати основу інфінітива та основу теперішнього часу.
- **Наступні теми:**
    - **Пасивні дієприкметники:** Тема є логічним продовженням, завершуючи загальну картину дієприкметників.
    - **Дієприкметниковий зворот:** Правила пунктуації та використання зворотів базуються на вмінні утворювати і розпізнавати дієприкметники.
    - **Дієприслівник:** Розуміння історії переходу дієприкметників у дієприслівники (`-чи`/`-ши`) допомагає глибше зрозуміти систему дієслівних форм. [S51]
