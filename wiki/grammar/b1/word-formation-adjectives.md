## Як це пояснюють у школі

В українській шкільній програмі ґрунтовне вивчення будови та способів творення прикметників (adjective formation) відбувається у 6-му класі. Автори підручників (наприклад, Заболотний) запроваджують цю тему після опанування будови слова та іменника, акцентуючи увагу на тому, що прикметники найчастіше утворюються від іменників (фізика → фізичний), дієслів (колоти → колючий) та інших прикметників, а найпродуктивнішим способом є суфіксальний [S1]. 

У старших класах (10 клас, Караман) тема розширюється й поглиблюється з акцентом на розряди за значенням (якісні, відносні, присвійні) та стилістичні функції прикметників [S10]. Шкільна програма детально аналізує правила вибору голосних у суфіксах (-ов-, -ев-, -єв-, -ичн-, -ічн-, -їчн-) [S5], подвоєння приголосних (-н- та -нн-) [S2] і закономірності зміни приголосних під час додавання суфіксів -ськ- [S7] та -ин- [S6]. Підручники також чітко розмежовують смислові відтінки прикметників із наголошеними суфіксами, що виражають міру ознаки (незліче́нний), і дієприкметників (незлі́чений) [S2, S3].

## Повна парадигма

Творення прикметників регулюється низкою суворих морфонологічних правил. Нижче наведено ключові парадигми суфіксального творення у вигляді механічних ланцюжків перетворень.

### 1. Творення відносних прикметників (суфікси -ов-, -ев-, -єв-)

Вибір суфікса залежить виключно від кінцевого приголосного твірної основи іменника та місця наголосу [S5, S6, S7].
**Обмеження:** ці суфікси додаються переважно до іменникових основ для позначення матеріалу, належності до групи тощо [S5].

| Правило вибору суфікса | Ланцюжок перетворення | Приклад |
|---|---|---|
| Твердий нешиплячий приголосний → `-ов-` | `{іменник} → {основа на твердий} → {+ов-} → {прикметник}` | `казка → казк- → +ов- → казковий` [S7] |
| Будь-який приголосний + наголос на закінченні → `-ов-` | `{іменник} → {основа} → {+ов-} → {наголос на -и́й}` | `дощ → дощ- → +ов- → дощови́й` [S6, S7] |
| М'який або шиплячий + наголос на основі → `-ев-` | `{іменник} → {основа на м'як/шип} → {+ев-} → {наголос на основі}` | `липень → липн- → +ев- → липневий` [S7] |
| Основа на `й` або подовжений м'який → `-єв-` | `{іменник} → {основа на й/подовж} → {+єв-} → {прикметник}` | `життя → житт- → +єв- → життєвий` [S5, S7] |

### 2. Творення відносних прикметників (суфікс -ськ- і його варіанти)

Суфікс `-ськ-` використовується переважно для творення прикметників від географічних назв і назв осіб [S4, S7].
**Обмеження:** при додаванні суфікса `-ськ(ий)` до основ, що закінчуються на г, к, х та їхні дзвінкі чи шиплячі відповідники, відбуваються обов'язкові історичні чергування [S7]. Додавання неможливе без цієї трансформації.

| Кінцевий приголосний основи | Зміна (чергування) | Ланцюжок перетворення | Приклад |
|---|---|---|---|
| г, ж, з | `+ ськ` = **-зьк-** | `{іменник} → {основа} → {г,ж,з + ськ} → {зьк}` | `Прага → Праг- → +ськ- → празький` [S6, S7] |
| к, ч, ц | `+ ськ` = **-цьк-** | `{іменник} → {основа} → {к,ч,ц + ськ} → {цьк}` | `Бахмач → Бахмач- → +ськ- → бахмацький` [S4, S7] |
| х, ш, с | `+ ськ` = **-ськ-** | `{іменник} → {основа} → {х,ш,с + ськ} → {ськ}` | `чех → чех- → +ськ- → чеський` [S7] |
| Інші приголосні | без змін | `{іменник} → {основа} → {+ськ-}` | `брат → брат- → +ськ- → братський` [S7] |

### 3. Творення присвійних прикметників

Присвійні прикметники вказують на індивідуальну належність конкретній особі чи тварині [S5, S10].
**Обмеження:** творяться ЛИШЕ від іменників — назв істот (людей і тварин) [S5, S10]. Присвійні прикметники, утворені від власних назв, завжди пишуться з великої літери [S5].

| Твірна основа | Суфікс | Ланцюжок перетворення | Приклад |
|---|---|---|---|
| Іменники І відміни (на -а, -я) | `-ин-` / `-їн-` | `{іменник} → {основа} → {+ин-/-їн-} → {прикметник}` | `Микола → Микол- → +ин- → Миколин` [S5] |
| Назви тварин (після голосного) | `-їн-` | `{тварина} → {основа} → {+їн-} → {прикметник}` | `соловей → солов'й- → +їн- → солов'їний` [S5] |
| Іменники ІІ відм. (чол. рід) | `-ів` (-їв), `-ов-`, `-ев-` | `{іменник} → {основа} → {+ів/ова} → {прикметник}` | `Василь → Васил- → +ів → Василів` (жін: `Василева`) [S5] |

**Важливо:** При додаванні суфікса `-ин-` можуть відбуватися чергування в корені [S5, S6]:
* `г → ж`: `Олег → Олеж- → +ин- → Олежин` [S6]
* `к → ч`: `Оленка → Оленч- → +ин- → Оленчин` [S5]
* `х → ш`: `Солоха → Солош- → +ин- → Солошин` [S6]

### 4. Творення ступенів порівняння

**Обмеження:** ступені порівняння творяться ТІЛЬКИ від якісних прикметників [S10]. Відносні та присвійні прикметники морфологічних ступенів порівняння не мають.

| Ступінь | Форма | Ланцюжок перетворення | Приклад |
|---|---|---|---|
| **Вищий** | Проста | `{початкова форма} → {основа} → {+іш-/-ш-} → {вищий}` | `довгий → довг- → +ш- → довший` [S10] |
| **Вищий** | Складена | `більш/менш + {початкова форма}` | `більш + вагомий → більш вагомий` [S10] |
| **Найвищий** | Проста | `най- + {форма вищого ступеня}` | `най- + спритніший → найспритніший` [S10] |
| **Найвищий** | Складена | `найбільш/найменш + {початкова форма}` | `найбільш + складний → найбільш складний` [S10] |

### 5. Написання -н- та -нн- у прикметниках

**Обмеження:** подвоєння `-нн-` у прикметниках зумовлене або збігом приголосних на межі морфем, або лексичним значенням перебільшеної міри ознаки [S2, S3].

| Правило | Ланцюжок перетворення | Приклад |
|---|---|---|
| Збіг приголосних (основа на -н + суфікс -н-) | `{іменник на -н} → {основа} → {+н-} → {-нн-}` | `туман → туман- → +н- → туманний` [S2] |
| Наголошені суфікси міри `-енн-`, `-анн-` | `{твірна основа} → {+е́нн-/-а́нн-} → {перебільшення}` | `товстий → товст- → +е́нн- → товстенний` [S2, S3] |
| Звичайні суфікси `-ин-`, `-ан-`, `-ян-` (одна `н`) | `{іменник} → {основа} → {+ян-} → {-н-}` | `глина → глин- → +ян- → глиняний` [S2] |

### 6. Правопис складних прикметників
* **Разом:** утворені від залежних одне від одного слів (`білогривий` ← біла грива), терміни (`водоочисний`), та слова `глухонімий`, `хитромудрий`, `зловорожий` [S2].
* **Через дефіс:** утворені від рівноправних слів (`синьо-жовтий`), відтінки кольорів (`темно-зелений`), повторення слів (`білий-білий`), з першою частиною `військово-` (`військово-морський`) [S2]. **Винятки:** кольори `жовтогарячий`, `червоногарячий` завжди пишуться разом [S2].

## Частотність і пріоритети

Для учнів рівня B1 першочерговим є засвоєння:
1. **Відносних прикметників на `-ов-`/`-ев-`** (матеріали, абстрактні поняття): це високочастотна модель у побутовому та діловому мовленні (`життєвий`, `яблуневий`, `паперовий`) [S5, S7].
2. **Ступенів порівняння якісних прикметників:** це базова комунікативна потреба. Складена форма (`більш/менш + початкова форма`) семантично легша для іноземців на ранніх етапах, однак проста форма (`-іш-`/`-ш-`) є значно питомішою та частіше використовується носіями усної мови [S10].
3. **Присвійних прикметників (`мамин`, `Олежин`)**: в українській мові ця модель превалює над описовими конструкціями з родовим відмінком (`мамина сумка` звучить природніше, ніж `сумка мами`). Це критично важливо для формування автентичного мовлення [S5].
4. **Географічних прикметників на `-ськ-`:** необхідні для опису походження, мов, міст і культур (`празький`, `бахмацький`) [S4, S7].

## Типові помилки L2 (англомовні учні)

Англомовні учні часто спираються на синтаксичні конструкції рідної мови, ігноруючи українські словотвірні моделі або застосовуючи правила некоректно.

| ❌ Помилково | ✅ Правильно | Чому |
|---|---|---|
| `Ця задача більш складніша.` | `Ця задача більш складна.` або `Ця задача складніша.` | Плеоназм (подвійне утворення ступеня порівняння). Не можна поєднувати слово `більш` із простою формою вищого ступеня. Складена форма вимагає виключно початкової форми прикметника [S10]. |
| `Це зошит Катерини.` (як єдина можлива модель) | `Це Катеринин зошит.` | Англійська структура `noun + of + noun` або `noun's` часто калькується через родовий відмінок приналежності. В українській мові значно природніше утворювати суфіксальні присвійні прикметники від імен [S5]. |
| `Він п'є яблуковий сік.` | `Він п'є яблуневий сік.` (або `яблучний`) | Ігнорування чергування суфіксів `-ов-`/`-ев-`. Оскільки основа `яблун-` закінчується на м'який приголосний, а наголос падає на основу, граматика вимагає використання суфікса `-ев-` [S5, S7]. |
| `Ми навчалися у Прагському університеті.` | `Ми навчалися у Празькому університеті.` | Ігнорування обов'язкового чергування приголосних (`г` + `ськ` = `зьк`) під час творення відносних прикметників від географічних назв [S6, S7]. |
| `Це дуже нескінчений процес.` | `Це дуже нескінченний процес.` | Плутання дієприкметників (ознака за дією, одна `-н-`) із прикметниками з наголошеним суфіксом `-енн-`, що позначає перебільшену міру або неможливість дії і завжди вимагає подвоєння `-нн-` [S2, S3]. |

## Деколонізаційні застереження

Російська та українська граматичні традиції мають суттєві відмінності у творенні та вживанні прикметників, на які обов'язково слід зважати:
1. **Калькування словосполучень:** В українській мові слід уникати російських лексичних кальок. Наприклад, правильно казати `слушна думка` або `правильна думка`, а не російський покруч `вірна думка` (вірним може бути лише друг, пес чи дружина) [S6]. Так само правильно `надійний спосіб`, а не `вірний спосіб` [S6].
2. **Семантика суфіксів подвоєння:** Українська мова тонко розрізняє смислові відтінки за допомогою наголосу та подвоєння, що часто відсутнє в російській. Зверніть увагу на різницю: `незліче́нний` (який складається з великої кількості — незліченні зорі) та `незлі́чений` (той, якого фізично ще не порахували). Подібно функціонують `несказанний` (якого не можна виразити словами) та `несказаний` (просто не висловлений голосними звуками) [S2, S3]. Це питомо український механізм розрізнення ознаки та стану.
3. **Правило «дев'ятки»:** Під час творення прикметників іншомовного походження в українській мові діє правило «дев'ятки»: суфікс `-ичн-` пишеться після приголосних `д, т, з, с, ц, ч, ш, ж, р` (`демократичний`, `історичний`, `ортопедичний` [S5, S7]). Учні, які мали досвід вивчення російської, часто порушують це правило через вплив російського «-ический» і пишуть «історічний».
4. **Ступені порівняння:** Українська мова категорично уникає російського префікса «самый» для творення найвищого ступеня. Форми на кшталт `самий кращий` чи `самий розумний` є грубим суржиком. Правильно використовувати префікс `най-`: `найкращий`, `найрозумніший` [S10]. У поетичному та художньому стилях українська мова також має автентичну модель «вищий ступінь + від усіх / над усе» (`дорожчий над усе`) [S10].

## Природні приклади

**Ступені порівняння та присвійні прикметники**
* `Рої бджолинні золоті над річкою про літо дзвонять.` [S2] (Присвійний прикметник від назви тварини, подвоєння збережене).
* `І я згадав своє село. Кого я там коли покинув.` [S9] (Займенник у ролі присвійного прикметника).
* `І сплеск весла, і місяць уповні, і ніч пливе на золотім човні в зір частий луг ромашкового неба, і шепчеш ти, що кращого не треба на світі щастя і тобі, й мені.` [S10] (Вищий ступінь прикметника як абсолютна оцінка стану).

**Відносні прикметники (суфікси -ов-, -ев-, -єв-, -ськ-)**
* `У великих містах легко знайти першу-ліпшу роботу.` [S9] (Складний прикметник, написання через дефіс).
* `— Будь ласка, грушовий сік, маленьку скляночку. — Прошу, ваш грушевий, маленька.` [S6] (Діалог, що чудово демонструє варіативність вибору суфікса залежно від місця наголосу: грушови́й — гру́шевий).
* `Комп’ютер — наш незамінний помічник у роботі з текстом.` [S7] (Прикметник на -нн- від іменника іншомовного походження).

**Прикметники інтенсивної міри ознаки (суфікси -енн-, -анн-)**
* `Ще сніг ковтала повінь широченна, і рала ждав іще тужавий лан.` [S2] (Суфікс інтенсивності -енн- для позначення надмірності).
* `Історія – це святая святих народу, недоторканна для злодійських рук.` [S2] (Наголошений суфікс -анн-).
* `Огні незліченні, мов стрічки огненні, до моря спускаються з міста.` [S2] (Подвоєння -нн- у прикметниках старослов'янського походження).

**Складні прикметники та написання частки "не"**
* `І соняшник злотоголовий лице за сонцем поверта.` [S2] (Складний прикметник, утворений від залежних слів, написання разом).
* `Вогка березнева ніч неохоче підіймає і збиває набакир червону гостроверху шапку вогню.` [S2] (Складні прикметники в описовому контексті).
* `Не одяг красить людину, а добрі справи.` [S7] (Написання частки "не" окремо через наявність явного протиставлення).

## Рекомендації для вправ

Для автора-письменника пропонується наступна чітка методична послідовність розробки активностей у модулях B1:

* **Фаза 1: Рецептивна ідентифікація (Розпізнавання)**
  * *Формат:* Робота з текстом (наприклад, стаття про тайм-менеджмент або комп'ютерні технології [S7, S10]). Учні повинні знайти прикметники та визначити їхню твірну основу й розряд за значенням.
  * *Фокус:* Навчити розпізнавати суфікси матеріалу/належності `-ов-`/`-ев-` [S5] та відрізняти їхні форми від випадкових збігів.

* **Фаза 2: Керована трансформація (Механічне відпрацювання)**
  * *Дія 1 (Чергування приголосних):* Дано перелік географічних назв (`Прага`, `Бахмач`, `Калуш`). Учень має утворити відносні прикметники (`празький`, `бахмацький`, `калуський`) за допомогою суфікса `-ськ-` [S4, S6, S7]. Це закріплює фонетичні зміни г/ж/з → зьк.
  * *Дія 2 (Присвійність):* Трансформація синтаксичних конструкцій. Наприклад, замінити `планшет Остапа` на `Остапів планшет`, `пенал Анастасії` на `Анастасіїн пенал` [S5].

* **Фаза 3: Вибір ступенів порівняння та логіка семантики**
  * *Формат:* Дано базові якісні прикметники. Учень повинен утворити просту та складену форми вищого й найвищого ступенів (наприклад, `вагомий → більш вагомий / найвагоміший`) [S10].
  * *Пастки:* Завдання формату "Знайди та виправ помилку", де свідомо закладені плеоназми типу `більш довший` або суржикові форми `самий кращий` [S2, S10].

* **Фаза 4: Орфографічна практика (Подвоєння та складні слова)**
  * *Формат:* Заповнення пропусків `-н-` або `-нн-` у словах типу `дерев'яний`, `широченний`, `нескінченний`, `пташиний` з обов'язковим обґрунтуванням (звичайна ознака чи перебільшена) [S2, S3].
  * *Формат:* Вибір між написанням разом і через дефіс для складних прикметників: `північно-східний` (дефіс) проти `двоповерховий` (разом) [S2].

## Зв'язки з іншими темами

* **Передумови (А1-А2):** Ця граматична тема спирається на базові знання про рід, число, відмінок прикметників та здатність узгоджувати прикметник з іменником у тексті. Без цих основ словотвір не має сенсу.
* **Синтаксис (В1-В2):** Розуміння розрядів (якісні, відносні, присвійні) є критичним для стилістики та правильної побудови речень. Наприклад, лише якісні прикметники можуть виступати іменною частиною складеного присудка в короткій формі (`певен`, `ясен`, `повен`) [S10].
* **Фонетика та словотвір:** Формування прикметників міцно прив'язане до теми історичного чергування приголосних і голосних звуків у коренях українських слів (тема "Фонетика"). Явище переходу частин мови (прономіналізація, субстантивація) [S9] готує учнів до розуміння складніших синтаксичних конструкцій на рівнях B2 і C1.
* **Дієприкметник (В2):** Тема подвоєння `-нн-` та специфіка написання `не` з прикметниками [S2, S3, S7] закладає міцний фундамент для майбутнього розрізнення прикметників та дієприкметників (наприклад, `нездола́нний` (прикм.) і `нездо́ланий` (дієприкм.)).
