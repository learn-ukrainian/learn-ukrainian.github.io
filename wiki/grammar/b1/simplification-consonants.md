## Як це пояснюють у школі

В українській шкільній програмі тема спрощення приголосних уводиться як фундаментальний прояв милозвучності (euphony) української мови. Підручники пояснюють це через фізіологічну та артикуляційну необхідність: якщо під час творення або зміни слів виникає збіг кількох приголосних (особливо проривних), вимовляти їх стає незручно [S1]. Щоб уникнути важкого нагромадження звуків, мова «спрощує» кластер, викидаючи середній приголосний звук [S5].

У 5 класі (підручник О. Заболотного) тема подається через емпіричний досвід: учням пропонують свідомо спробувати вимовити слово *проїзний*, зберігши звук [д] (*проїздний*), а потім вимовити його правильно, щоб фізично відчути полегшення артикуляції [S1]. 

У старших класах (10 клас, О. Авраменко, О. Глазова) матеріал систематизується на вищому теоретичному рівні. Автори чітко розмежовують спрощення, яке **закріплене на письмі** (відповідає орфографічній нормі), і те, що **відбувається лише в усному мовленні** (орфоепічна норма), але не фіксується літерами [S2, S3]. Підкреслюється, що найчастіше випадають проривні (stop consonants) [д], [т], а рідше — [к], [р], [л] [S5].

## Повна парадигма

Процес спрощення є морфонологічним і найчастіше активується під час додавання суфіксів (особливо прикметникового `-н-` та дієслівного `-ну-`) до основи, що закінчується на кластер приголосних. 

Нижче наведено повні парадигми трансформацій для кожної групи звуків. Правила творення подано як ланцюжок: `{базове слово/інфінітив} → {основа} → {+суфікс/закінчення} → {кінцева форма}`.

### 1. Випадіння звука [т]
Це наймасовіша група. Звук [т] випадає між щілинними [с], [з] та сонорними [н], [л].

| Базове слово | Ланцюжок перетворення | Кінцева форма | Група |
|---|---|---|---|
| честь | `честь → чест- → + -н- (ий) → *честний` | **чесний** | `-стн-` → `-сн-` [S1] |
| користь | `користь → корист- → + -н- (ий) → *користний` | **корисний** | `-стн-` → `-сн-` [S2] |
| радість | `радість → радіст- → + -н- (ий) → *радістний` | **радісний** | `-стн-` → `-сн-` [S3] |
| щастя | `щастя → щаст- → + -лив- (ий) → *щастливий` | **щасливий** | `-стл-` → `-сл-` [S2] |
| лестощі | `лестощі → лест- → + -лив- (ий) → *лестливий` | **улесливий** | `-стл-` → `-сл-` [S1] |

### 2. Випадіння звука [д]
Відбувається у прикметниках, утворених від іменників із коренем на `-їзд` або `-жд`.

| Базове слово | Ланцюжок перетворення | Кінцева форма | Група |
|---|---|---|---|
| проїзд | `проїзд → проїзд- → + -н- (ий) → *проїздний` | **проїзний** | `-здн-` → `-зн-` [S1] |
| виїзд | `виїзд → виїзд- → + -н- (ий) → *виїздний` | **виїзний** | `-здн-` → `-зн-` [S5] |
| тиждень | `тиждень → тижд- → + -н- (евий) → *тиждневий` | **тижневий** | `-ждн-` → `-жн-` [S2] |
| тиждень | `тиждень → тижд- → + -н- (я) → *тиждня` | **тижня** | Форма Р.в. однини [S5] |

Характерне переважно для дієслів раптової, одноразової дії (semelfactive verbs) із суфіксом `-ну-`.

| Інфінітив / База | Ланцюжок перетворення | Кінцева форма | Група |
|---|---|---|---|

| писк | `писк → писк- → + -ну- (ти) → *пискнути` | **писнути** | `-скн-` → `-сн-` [S2] |
| тиск | `тиск → тиск- → + -ну- (ти) → *тискнути` | **тиснути** | `-скн-` → `-сн-` [S2] |
| брязкіт | `брязкіт → брязк- → + -ну- (ти) → *брязкнути` | **брязнути** | `-зкн-` → `-зн-` [S2] |

### 4. Унікальні кореневі спрощення ([л], [д], [р])
У деяких питомих словах спрощення відбулося історично глибоко в корені.

| Базове слово | Ланцюжок перетворення | Кінцева форма | Група |
|---|---|---|---|
| масло | `масло → масл- → + -н- (ий) → *маслний` | **масний** | `-слн-` → `-сн-` [S1] |
| сердець | `сердець → серд- → + -ц- (е) → *сердце` | **серце** | `-рдц-` → `-рц-` [S2] |
| чернець | `чернець → чернец- → + -і (множина) → *чернці` | **ченці** | `-рнц-` → `-нц-` [S2] |

### ⚠️ Жорсткі обмеження та винятки

Граматика встановлює суворі обмеження, за яких спрощення **відбувається лише в усному мовленні (вимові), але категорично НЕ позначається на письмі**:

1. **Іншомовна лексика (Loanwords):** При творенні прикметників від запозичених слів із кінцевими кластерами `-ст`, `-нт` літера `т` зберігається на письмі для збереження візуальної етимології. 
   - `контраст → контраст- → + -н- (ий) → контрастний` (вимовляється [контрасний], але пишеться з `т`) [S1, S3].
   - `студент → студент- → + -ськ- (ий) → студентський` [S1, S5].
   - `баласт → баласт- → + -н- (ий) → баластний` [S2].
2. **Слова-винятки питомої лексики:** Невелика закрита група слів, у яких літера `т` традиційно зберігається на письмі попри те, що у вимові звук повністю редукується: `кістлявий, пестливий, хвастливий, хвастнути, хворостняк, зап’ястний, шістнадцять` [S1, S2, S3].
3. **Давальний та місцевий відмінки іменників:** В іменниках жіночого роду на `-стка`, `-нтка` у формах Д.в. та М.в. однини перед закінченням `-ці` літера `т` зберігається, хоча у вимові звук випадає. 
   - `невістка → невіст- → + -ці → невістці` (вимовляється [неив'іс'ц'і]) [S2, S4].
   - `артистка → артист- → + -ці → артистці` [S2].
4. **Обмеження щодо звука [к]:** У низці лексем звук [к] не спрощується ні на письмі, ні у вимові: `випускний`, `тоскно`, `скніти`, `скнара` [S2].

## Частотність і пріоритети

Для рівня B1 найвищий пріоритет мають:
1. **Якісні прикметники емоцій та станів** (`-стн-` → `-сн-`): *радісний, чесний, щасливий, корисний*. Вони утворюють лексичне ядро комунікації.
2. **Лексика часу та подорожей** (`-здн-` → `-зн-`, `-ждн-` → `-жн-`): *тижневий, проїзний, виїзний*.
3. **Числівники-винятки:** *шістнадцять, шістдесят, шістсот*. Хоча семантично вони засвоюються на рівні A1/A2, орфографічний контроль цього винятку критичний саме для B1, оскільки тут учні починають писати більше зв'язних текстів [S5].

Слова на кшталт *хворостняк* чи *зап'ястний* є екстремально низькочастотними. Їх варто згадувати лише як довідковий матеріал (reference material), не витрачаючи час на активне тренування у вправах.

## Типові помилки L2 (англомовні учні)

Англомовні студенти часто припускаються помилок у цій темі через морфологічну інтерференцію (morphological interference) — англійська орфографія тяжіє до збереження букв кореня (напр., *fast* → *fastness*).

| ❌ Помилково | ✅ Правильно | Чому |
|---|---|---|
| Він дуже **честний**. | Він дуже **чесний**. | **Сліпе збереження кореня.** Учень знає іменник «честь» і механічно додає суфікс `-ний`, очікуючи, що, як і в англійській, корінь залишиться візуально незмінним. В українській мові фонетичний принцип домінує над морфологічним [S1]. |
| Ми купили **проїздний**. | Ми купили **проїзний**. | **Глухота до сонорних збігів.** Кластер [здн] є важким для слов'янської артикуляції, але англомовний апарат може намагатися його вимовити, тому учень не відчуває інтуїтивної потреби викинути «д» [S1]. |
| Це **контрасний** душ. | Це **контрастний** душ. | **Гіперкорекція.** Учень вивчив правило спрощення і хибно застосовує його до слів іншомовного походження (loanwords). Українська орфографія зберігає кластери в іншомовних словах на письмі [S1, S2]. |
| Дай зошит **студенці**. | Дай зошит **студентці**. | **Втрата морфеми у непрямих відмінках.** Учень орієнтується на слух (де звук [т] дійсно випадає: [студе5нс'ц'і]), але забуває, що на письмі в іменниках жіночого роду на `-нтка` літера категорично зберігається [S3, S4]. |
| Вимова: [ш'іс**т**надц'ат'] | Вимова: [ш'існац':ат'] | **Читання за літерами (spelling pronunciation).** Оскільки виняток `шістнадцять` зберігає «т» на письмі, англомовний учень артикулює його з повним вибухом, ламаючи українську милозвучність [S5]. |

## Деколонізаційні застереження

При викладанні цієї теми англомовним студентам критично важливо уникати російськоцентричного фреймінгу. 

У російській мові орфографія є глибоко морфологічною: вона вимагає зберігати на письмі всі літери твірної основи, навіть якщо у вимові вони повністю зникають (наприклад, рос. *честный*, *радостный*, *сердце*, *солнце*). Через це іноземці, які стикалися з російською, часто вважають українські форми `чесний` чи `радісний` якоюсь «девіацією», де нібито «загубилася» літера.

**Методична вимога:** Авторам курсу категорично заборонено пояснювати українське спрощення як «виняток» чи «відхилення». Навпаки, форми на кшталт `чесний` і `серце` є еталонними й органічними для української системи. Вони ілюструють базовий закон милозвучності (евфонії) — усунення артикуляційних перешкод і тяжіння до відкритих складів [S1, S8]. Українська орфографія тут є чесною фонетичною транскрипцією мовленнєвої реальності.

Окремо слід звернути увагу на базову лексему `сонце`. Російський відповідник — *солнце*, де `л` не читається, але зберігається на письмі. Вживання форми *солнце* в українському тексті є не просто помилкою, а прямим русизмом і калькуванням, яке маркується як грубе порушення норми [S1 явно вказує на `солнце` як на помилку, що підлягає виправленню]. Ніколи не використовуйте російські парадигми для пояснення української фонетики.

## Природні приклади

Усі наведені речення є автентичними зразками літературної української мови.

**Група 1: Якісні прикметники (спрощення -стн- → -сн-, -стл- → -сл-)**
1. «Правдою, совістю світиться нам людина щира і **чесна**.» [S3]
2. «У вивченні наук приклади **корисніші** від правил.» [S3]
3. «Заздрісник сам причина свого **нещастя** і сам собі ворог.» [S3]
4. «І метелик біліє, мов згадка про нездійснену **радісну** мить.» [S3]

**Група 2: Відносні прикметники та іменникові форми (спрощення -рдц- → -рц-, -ждн- → -жн-)**
5. «Мова розуму буде почута, якщо проходитиме через **серце**.» [S3]
6. «На **серці** якось особливо святково.» [S1]
7. «**Сонце** хилилося до заходу.» [S1]
8. «Могутній антициклон обіцяв триматися два-три **тижні**.» [S1]

**Група 3: Збереження приголосних на письмі (винятки та іншомовна лексика)**
9. «Багатого з **хвастливим** не розпізнаєш.» [S1]
10. «І очі **пестливо** гладять землі кожну впадину й схил.» [S3]
11. «Щоб піднятись вище, треба було відкрити балони зі стисненим повітрям, аби витиснути з **баластних** цистерн воду.» [S1]
12. «Якось, побачивши **урочистий** стіл, а на ньому на **почесному** місці святковий хліб, підступна гадюка позаздрила...» [S2]

## Рекомендації для вправ

Автор-письменник має будувати діяльнісний блок (activities) за такою методичною прогресією:

1. **Фаза аудіювання та розпізнавання (Input & Awareness):** Студентам пропонується прослухати аудіо та зіставити іменник-джерело з прикметником (напр., *радість* → *радісний*). Завдання: позначити літеру, яка «зникла» в усному мовленні. Це формує первинне розуміння фонетичної плавності [S1].
2. **Фаза механічної трансформації (Word Formation):** Вправи на словотвір. Дано базове слово і суфікс (напр., `тиждень` + `-евий`), студент має надрукувати кінцеву форму `тижневий`, свідомо видаляючи зайву літеру `д` [S2]. Важливо давати змішаний пул слів, щоб учень постійно ухвалював рішення (спрощувати чи ні).
3. **Фаза сортування винятків (Sorting/Categorization):** Завдання drag-and-drop. Студент розподіляє слова на дві колонки: «спрощення відбувається на письмі» (напр., *пісний, масний*) та «спрощення відбувається лише у вимові» (напр., *шістнадцять, гігантський, контрастний*) [S1]. Це критично важливо для закріплення орфографічної норми іншомовних слів [S2].
4. **Фаза комунікативного контексту (Cloze Tests):** Заповнення пропусків у зв'язному мікротексті побутової тематики (напр., купівля *проїзного* квитка, обговорення *тижневого* розкладу, емоційні стани — *щасливий, радісний*), де студент має вписати правильну форму самостійно [S1, S3].

## Зв'язки з іншими темами

- **Передумови (Prerequisites):** Тема спирається на базові знання фонології рівня A1/A2 (розуміння різниці між приголосними та голосними) [S9], а також на навички суфіксального словотворення (утворення прикметників за допомогою суфіксів `-н-`, `-лив-`, `-ськ-`) [S10]. Учні вже повинні візуально впізнавати такі слова, як *тиждень* і *серце*.
- **Подальший розвиток (Enables):** Правильне засвоєння цієї теми розблоковує здатність учня безпомилково утворювати складні абстрактні прикметники на рівнях B2–C1. Вона також готує фундамент для складнішої теми чергування приголосних (напр., *рука → руці*, *їздити → їжджу*) [S6, S7] та глибокого розуміння української евфонії (чергування у/в, і/й) [S8]. Крім того, це безпосередньо пов'язано з орфографією числівників (десятків і сотень), що є ключовим для фінансової та побутової комунікації [S5].
