## Як це пояснюють у школі (How Schools Teach This)
Українські шкільні підручники вводять правила узгодження числівників з іменниками поступово, починаючи з 6-го класу. Основний підхід полягає в поділі кількісних числівників на три ключові групи (Заболотний, 6 клас, Source 31: `6-klas-ukrmova-zabolotnyi-2020_s0171`):
1.  **Цілі числа** (`десять`, `сорок шість`)
2.  **Дробові** (`одна восьма`, `півтора`)
3.  **Збірні** (`двоє`, `шестеро`)

На початковому етапі (6 клас) основна увага приділяється правилам узгодження в називному відмінку. Заболотний (Source 7: `11-klas-ukrmova-zabolotnyi-2019_s0054`) та Авраменко (Source 29: `11-klas-ukrajinska-mova-avramenko-2019_s0047`) чітко розмежовують три основні моделі:
*   **Числівник `один`:** Повне узгодження з іменником у роді, числі та відмінку (`одна пісня`, `одного посібника`).
*   **Числівники `два`, `три`, `чотири`:** Вимагають іменника у формі **називного відмінка множини** (`два фахівці`, `три сценарії`). Це ключове правило, яке активно тренується.
*   **Числівники `п'ять` і більше:** Вимагають іменника у формі **родового відмінка множини** (`п'ять озер`, `сорок будинків`).

Збірні числівники (`двоє`, `троє` тощо) вводяться як стилістично маркована група, поширена в розмовному та художньому мовленні (Source 24: `6-klas-ukrmova-zabolotnyi-2020_s0172`). Підкреслюється їх обмежена сполучуваність: вони не вживаються з іменниками жіночого роду (окрім `обидві`) та назвами неістот чоловічого роду (Source 6: `6-klas-ukrmova-zabolotnyi-2020_s0173`, Source 25: `6-klas-ukrmova-litvinova-2023_s0233`).

У старших класах (10-11) фокус зміщується на складніші випадки: відмінювання складених і дробових числівників, узгодження в непрямих відмінках та узгодження з прикметниками (Source 4: `11-klas-ukrajinska-mova-glazova-2019_s0036`). Наприклад, пояснюється, що при складених числівниках форму іменника визначає останнє слово (`двадцять два зошити` vs `двадцять п'ять зошитів`) (Source 7: `11-klas-ukrmova-zabolotnyi-2019_s0054`).

## Повна парадигма (Full Paradigm)

### Узгодження в називному відмінку (Nominative Case Agreement)

Це фундаментальне правило для кількісних виразів.

| Числівник | Правило узгодження | Приклад | Джерело |
| :--- | :--- | :--- | :--- |
| **`один`, `одна`, `одне`** | Іменник стоїть в **однині** того ж роду й відмінка. | `один стілець`, `одна книжка`, `одне вікно` | Source 22: `6-klas-ukrmova-betsa-2023_s0162` |
| **`два`, `три`, `чотири`**<br>**`обидва`, `обидві`** | Іменник стоїть у **називному відмінку множини**. | `два студенти`, `три метри`, `чотири подруги` | Source 29: `11-klas-ukrajinska-mova-avramenko-2019_s0047` |
| | *Виняток 1:* Іменники з суфіксом `-ин-`, що зникає у множині (киянин → кияни), ставляться у **родовий відмінок однини**. | `два киянина`, `три селянина`, `чотири громадянина` | Source 29: `11-klas-ukrajinska-mova-avramenko-2019_s0047` |
| | *Виняток 2:* Іменники IV відміни (назви малят тварин) ставляться у **родовий відмінок однини**. <!-- VERIFY --> | `два імені`, `три поросяти` | Source 29: `11-klas-ukrajinska-mova-avramenko-2019_s0047` |
| **`п'ять` — `двадцять`**<br>та всі, що закінчуються на 5, 6, 7, 8, 9, 0 | Іменник стоїть у **родовому відмінку множини**. | `п'ять студентів`, `десять метрів`, `сто книжок` | Source 22: `6-klas-ukrmova-betsa-2023_s0162` |
| **Складені числівники**<br>(21, 22, 34, 101, etc.) | Іменник узгоджується з **останнім словом** числівника. | `двадцять один студент`<br>`двадцять два студенти`<br>`двадцять п'ять студентів` | Source 7: `11-klas-ukrmova-zabolotnyi-2019_s0054` |

### Збірні числівники (`двоє`, `троє`...)

Збірні числівники позначають кількість як єдине ціле і мають обмежену сполучуваність (Source 25: `6-klas-ukrmova-litvinova-2023_s0233`). Після збірних числівників іменник завжди ставиться у **родовому відмінку множини** (Source 1: `11-klas-ukrajinska-mova-glazova-2019_s0039`).

| Збірні числівники поєднуються з: | Приклади | Джерело |
| :--- | :--- | :--- |
| 1. Іменниками **чоловічого роду** (назви осіб, тварин) | `двоє хлопців`, `троє коней`, `п'ятеро горобців` | Source 1: `11-klas-ukrajinska-mova-glazova-2019_s0039` |
| 2. Іменниками **середнього роду** | `двоє сердець`, `троє вікон` | Source 1: `11-klas-ukrajinska-mova-glazova-2019_s0039` |
| 3. Іменниками **IV відміни** (назви малят) | `двоє ягнят`, `троє поросят`, `п’ятеро немовлят` | Source 1: `11-klas-ukrajinska-mova-glazova-2019_s0039` |
| 4. Іменниками, що мають **лише форму множини** | `двоє воріт`, `троє дверей`, `четверо ножиць` | Source 1: `11-klas-ukrajinska-mova-glazova-2019_s0039` |
| **НЕ поєднуються з:** | | |
| 1. Іменниками **жіночого роду** (крім `обидві`) | ❌ `троє дівчат` → ✅ `три дівчини` | Source 26: `11-klas-ukrajinska-mova-avramenko-2019_s0050` |
| 2. Іменниками **чоловічого роду** (назви неістот) | ❌ `двоє столів` → ✅ `два столи` | Source 1: `11-klas-ukrajinska-mova-glazova-2019_s0039` |

У непрямих відмінках збірні числівники втрачають свою унікальну форму (`-оє`, `-еро`) і відмінюються як відповідні кількісні числівники, узгоджуючись з іменником у відмінку (Source 9: `6-klas-ukrmova-litvinova-2023_s0243`, Source 10: `6-klas-ukrmova-zabolotnyi-2020_s0180`).
*   Н. `двоє друзів` → Р. `двох друзів` → Д. `двом друзям` (Source 1: `11-klas-ukrajinska-mova-glazova-2019_s0039`)

### Дробові та мішані числівники

| Тип | Правило узгодження | Приклад | Джерело |
| :--- | :--- | :--- | :--- |
| **Прості дроби** (`дві треті`) | Іменник стоїть у **родовому відмінку однини**. | `дві третіх кілограма`, `три п'яті площі` | Source 22: `6-klas-ukrmova-betsa-2023_s0162` |
| **`півтора` / `півтори`** | `півтора` (ч.р., с.р.), `півтори` (ж.р.) + іменник у **родовому відмінку однини**. Не відмінюються. | `півтора метра`, `півтора яблука`, `півтори тонни` | Source 22: `6-klas-ukrmova-betsa-2023_s0162`, Source 15: `11-klas-ukrmova-zabolotnyi-2019_s0048` |
| **`півтораста` (150)** | `півтораста` + іменник у **родовому відмінку множини**. Не відмінюється. | `півтораста учнів` | Source 22: `6-klas-ukrmova-betsa-2023_s0162` |
| **Мішані числа** (`з половиною`, `з чвертю`) | Іменник узгоджується з **цілою частиною** числівника. | `два з половиною кілометри` (як з `два`)<br>`п'ять із чвертю літрів` (як з `п'ять`) | Source 7: `11-klas-ukrmova-zabolotnyi-2019_s0054`, Source 22: `6-klas-ukrmova-betsa-2023_s0162` |

## Частотність і пріоритети

Для рівня B1/B2 пріоритети наступні:

1.  **Найвищий пріоритет:** Бездоганне володіння узгодженням у **називному-знахідному відмінку**. Це основа для побудови речень, де кількісний вираз виступає підметом або прямим додатком. Розрізнення `два студенти` (Н.в. мн.) vs `п'ять студентів` (Р.в. мн.) є абсолютно критичним.
2.  **Середній пріоритет:** Правильне вживання збірних числівників (`двоє`, `троє`) та їх обмежень. Вони дуже поширені в усному мовленні (Source 24: `6-klas-ukrmova-zabolotnyi-2020_s0172`), і їх неправильне використання (напр., з іменниками жіночого роду) одразу видає іноземця.
3.  **Середній пріоритет:** Узгодження з `півтора`/`півтори` та мішаними числами (`два з половиною`). Це часто вживані конструкції в повсякденному житті.
4.  **Нижчий пріоритет:** Відмінювання складних числівників у непрямих відмінках (`ста вісімдесятьма шістьма веслами`). Хоча це є ознакою високого рівня володіння мовою, на рівні B1/B2 важливіше правильно побудувати конструкцію в основних відмінках, ніж безпомилково відміняти `768` в орудному відмінку. Часто носії мови самі уникають таких конструкцій.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я бачу *два студента*.` | `Я бачу **двох студентів**.`<br>або (у Н.в.) `Ось **два студенти**.` | Це прямий перенос (трансфер) російської граматичної норми, де після `два`, `три`, `четыре` вживається родовий відмінок однини (`два студента`). В українській мові в називному відмінку вживається **називний відмінок множини** (Source 29: `11-klas-ukrajinska-mova-avramenko-2019_s0047`). У знахідному відмінку для істот форма збігається з родовим (`двох студентів`). |
| `У нас в групі *троє дівчат*.` | `У нас в групі **три дівчини**.` | Збірні числівники (крім `обидві`) не вживаються з іменниками жіночого роду. Замість них використовуються відповідні кількісні числівники (Source 26: `11-klas-ukrajinska-mova-avramenko-2019_s0050`). |
| `Мені потрібно *півтора години*.` | `Мені потрібно **півтори години**.` | Помилка в родовому узгодженні. Слово `година` — жіночого роду, тому вимагає числівника `півтори`, а не `півтора` (Source 22: `6-klas-ukrmova-betsa-2023_s0162`). |
| `У цьому звіті *двадцять одна сторінок*.` | `У цьому звіті **двадцять одна сторінка**.` | У складених числівниках іменник узгоджується з останнім словом. `одна` вимагає називного відмінка однини. Помилка виникає через аналогію до `двадцять п'ять сторінок` (Source 7: `11-klas-ukrmova-zabolotnyi-2019_s0054`). |
| `Ми говорили з *п'ятьма студентів*.` | `Ми говорили з **п'ятьма студентами**.` | У непрямих відмінках числівники `два, три, чотири` та `п'ять і більше` вимагають, щоб іменник стояв у тому ж відмінку, що й числівник (тобто, узгодження у відмінку). Помилка полягає в тому, що іменник залишається в родовому відмінку множини, як у називному (Source 3: `10-klas-ukrmova-karaman-2018_s0309`, приклад `з 567 постраждалими`). |

## Деколонізаційні застереження (Decolonization Notes)

Українська система числівників, хоч і має спільне праслов'янське коріння з російською, розвинула низку унікальних рис. Ігнорування цих відмінностей веде до суржику та русифікації мовлення.

1.  **`Два`/`три`/`чотири` + Називний Відмінок Множини:** Це **найважливіша відмінність** від російської мови. Українська конструкція `два брати` (де `брати` — Н.в. мн.) зберегла давньоруську форму двоїни, яка пізніше злилася з множиною (Source 27: `ext-istoria_movy-37`). Російська ж використовує родовий відмінок однини: `два брата` (де `брата` — Р.в. одн.). Навчання через російські аналоги тут є абсолютно неприпустимим.
2.  **Спрощення відмінювання `40, 90, 100`:** В українській мові числівники `сорок`, `дев'яносто`, `сто` у всіх непрямих відмінках мають однакове закінчення **-а** (`сорока`, `дев'яноста`, `ста`). У російській мові ці числівники також мають спрощене відмінювання, але форми збігаються (`сорока`, `девяноста`, `ста`), що може призводити до помилкового припущення про ідентичність правил. Важливо наголосити, що це паралельний, але незалежний розвиток у кожній мові. (Source 16: `ext-other_blogs-67`).
3.  **Збірні числівники:** Українська мова має значно активнішу та багатшу систему збірних числівників (`двоє, трійко, четверо, п'ятеро... двадцятеро, тридцятеро`) (Source 24: `6-klas-ukrmova-zabolotnyi-2020_s0172`). Правила їх вживання (напр., заборона на використання з жіночим родом) є унікальними і не збігаються з російськими.
4.  **Форми `півтори` та `півтораста`:** Форма `півтори` для жіночого роду та числівник `півтораста` (150) є характерними рисами української мови, які не мають прямих аналогів у сучасній російській літературній мові (Source 22: `6-klas-ukrmova-betsa-2023_s0162`).

## Природні приклади (Natural Examples)

**Група 1: Узгодження з `два, три, чотири` (Н.в. мн.)**
*   `Якщо двоє друзів сваряться, не будь третім сперечальником.` (Source 4: `11-klas-ukrajinska-mova-glazova-2019_s0036`)
*   `У судді два вуха мають бути однакові.` (Джерело: `11-klas-ukrajinska-mova-glazova-2019_s0039`, адаптовано)
*   `Минуло два дні, чотири дні, біжить знов Лисичка польовою стежкою.` (Source 11: `6-klas-ukrmova-litvinova-2023_s0248`)

**Група 2: Узгодження з `п'ять` і більше (Р.в. мн.)**
*   `Наговорив десять міхів горіхів!` (Source 4: `11-klas-ukrajinska-mova-glazova-2019_s0036`)
*   `Славетну «Мрію» піднімають у небо шість пілотів.` (Джерело: `6-klas-ukrmova-avramenko-2023_s0170`, адаптовано)
*   `Польські війська обороняли Львів 10 днів, але шансів втримати місто у них не було.` (Source 19: `ext-realna_istoria-7`)

**Група 3: Збірні числівники**
*   `У його батьків народилось аж чотирнадцятеро дітей, але дванадцятеро з них померли ще дітьми, залишилось в сім'ї тільки двоє дітей.` (Source 2: `ext-ulp_youtube-142`)
*   `Сергій приніс троє відер води.` (Source 6: `6-klas-ukrmova-zabolotnyi-2020_s0173`)
*   `За столом сиділо не багато й не мало, одинадцятеро малих голованчиків.` (Source 9: `6-klas-ukrmova-litvinova-2023_s0243`)

**Група 4: Дробові та мішані числівники**
*   `Найбільші звірі Арктики – моржі. Вони важать дві з половиною тонни.` (Source 10: `6-klas-ukrmova-zabolotnyi-2020_s0180`)
*   `Для приготування соку потрібно два з половиною апельсини.` (Source 29: `11-klas-ukrajinska-mova-avramenko-2019_s0047`)
*   `Розмір доплат збільшено на 12,7 (дванадцять цілих і сім десятих) відсотка.` (Source 15: `11-klas-ukrmova-zabolotnyi-2019_s0048`)

## Рекомендації для вправ (Activity Concepts)

**Phase 1: Розпізнавання та базове узгодження (Н.в.)**
*   **Вправа "Вибір":** Дати речення з дужками: `На столі лежить (два/двоє) (олівець/олівці)`. Студент має обрати правильну форму числівника та іменника.
*   **Вправа "Заповнення пропусків":** Дати число цифрою і іменник у початковій формі. `Я купив ___. (5, книга) → Я купив п'ять книжок`.

**Phase 2: Збірні числівники та обмеження**
*   **Вправа "Можна / Не можна":** Дати пари "збірний числівник + іменник". Студент має визначити, чи є сполучення правильним. `Троє сестер` (не можна), `П'ятеро хлоп'ят` (можна), `Четверо столів` (не можна).
*   **Вправа "Трансформація":** Перетворити речення з кількісними числівниками на речення зі збірними, де це можливо. `Три хлопці грають у футбол → Троє хлопців грають у футбол.` `Три дівчини читають → (трансформація неможлива)`.

**Phase 3: Непрямі відмінки та складні конструкції**
*   **Вправа "Конструктор речень":** Дати елементи в початковій формі: `Я / говорити / з / (4) / друг`. Студент має побудувати повне речення: `Я говорив з чотирма друзями`.
*   **Вправа "Переказ":** Дати короткий текст з числами, написаними цифрами. Студент має переказати його, використовуючи числівники словами в правильних відмінках. Це тренує використання в природному контексті.

## Зв'язки з іншими темами

*   **Відмінювання іменників:** Знання форм родового та називного відмінків множини є абсолютною передумовою для цієї теми. Без цього неможливо правильно застосувати навіть базові правила (`п'ять книжок` vs `дві книжки`).
*   **Рід іменників:** Правильне використання `один/одна/одне` та `півтора/півтори` повністю залежить від знання роду іменника.
*   **Відмінки (Cases):** Узгодження в непрямих відмінках (`з двома друзями`) вимагає розуміння системи відмінків загалом.
*   **Прикметники:** Узгодження прикметників у кількісних виразах є наступним кроком. Правило `два великих вікна` (прикметник у Р.в. мн.) vs `два нові будинки` (прикметник у Н.в. мн.) є складною, але важливою темою (Source 4: `11-klas-ukrajinska-mova-glazova-2019_s0036`).
