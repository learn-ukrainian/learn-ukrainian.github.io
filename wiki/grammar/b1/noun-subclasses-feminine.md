### Правила творення орудного відмінка однини

Центральною особливістю третьої відміни є механізм творення орудного відмінка. У формі орудного відмінка однини перед закінченням `-ю` відбувається асимілятивне подовження приголосного звука (на письмі це передається подвоєнням відповідних літер). Цей процес відбувається виключно тоді, коли основа іменника закінчується на один м’який або шиплячий приголосний звук після голосного [S1, S3].

Ланцюжок творення з подовженням: `{Н. в.} → {основа} → {подовження кінцевого приголосного} → {+ -ю} → {Ор. в.}`
- `тінь → тін- → цінн- → + -ю → тінню` [S3]
- `ніч → ніч- → нічч- → + -ю → ніччю` [S3]
- `мить → мит- → митт- → + -ю → миттю` [S3]
- `річ → річ- → річч- → + -ю → річчю` [S1]

**Явні фонетичні та графічні обмеження (коли подовження НЕ відбувається)** [S1, S3]:

1. **Збіг двох приголосних.** Якщо основа іменника закінчується збігом двох або більше приголосних звуків (наприклад, `-ст-`), фонетичне подовження неможливе, і подвоєння літер на письмі не відбувається.
   Ланцюжок творення: `{Н. в.} → {основа без змін} → {+ -ю} → {Ор. в.}`
   - `радість → радіст- → + -ю → радістю` [S3]
   - `честь → чест- → + -ю → честю` [S1]
   - `ніжність → ніжніст- → + -ю → ніжністю` [S3]

2. **Губні приголосні та сонорний `[р]`.** Якщо основа закінчується на губні приголосні (`б`, `п`, `в`, `м`, `ф`) або на сонорний вібралант `р`, ці звуки в українській мові не подовжуються перед напівголосним `[й]` (який прихований у літері `ю`). У такому разі перед закінченням `-ю` обов'язково з'являється апостроф, що позначає роздільну вимову.
   Ланцюжок творення: `{Н. в.} → {основа} → {+ '-ю} → {Ор. в.}`
   - `любов → любов- → + '-ю → любов’ю` [S1, S3]
   - `мати → матір- → + '-ю → матір’ю` [S1, S3]
   - `кров → кров- → + '-ю → кров’ю` [S1, S3]

### Паралельні закінчення родового відмінка однини

Іменники третьої відміни, що закінчуються на `-ть` після приголосного (наприклад, `радість`, `смерть`, `честь`), а також певний історичний перелік слів (`кров`, `любов`, `осінь`, `сіль`, `Русь`), можуть мати у формі родового відмінка однини паралельні закінчення `-і` або `-и` [S3].
Ланцюжок творення: `{Н. в.} → {основа} → {+ -і / -и} → {Р. в.}`
- `радість → радост- → + -і / -и → радості / радости` [S3]

### Чергування голосних у корені

Під час відмінювання іменників третьої відміни діє фундаментальний фонетичний закон української мови: чергування голосних `[і]` з `[о]` або `[е]` [S6]. Голосний `[і]` фіксується у закритому складі (у називному, знахідному та орудному відмінках однини). Коли під час словозміни склад стає відкритим (через появу голосного закінчення), `[і]` закономірно переходить в `[о]` чи `[е]`.
Ланцюжок творення: `{Н. в.} → {основа з чергуванням і→о/е} → {+ закінчення} → {Непрямий відмінок}`
- `сіль → сол- → + -і → солі` (Д. в., М. в.) [S6]
- `піч → печ- → + -і → печі` (Р. в.) [S6]
- `осінь → осен- → + -і → осені` (Р. в.) [S3]

### Відмінювання слова «мати»

Іменник `мати` є унікальним реліктом давньої слов'янської відміни на приголосний. Тому при його відмінюванні в усіх непрямих відмінках однини та множини (окрім знахідного та орудного однини) в основі з'являється історичний суфікс `-ер-` [S3, S8].
Ланцюжок творення (Р., Д., М. в.): `мати → мат- → {+ -ер-} → {+ -і} → матері` [S8].
Ланцюжок творення (Зн. в.): `мати → мат- → {+ -ір} → матір` [S8].
Ланцюжок творення (Ор. в.): `мати → мат- → {+ -ір-} → {+ '-ю} → матір'ю` [S3].

## Частотність і пріоритети

Для іноземних учнів на рівні B1 кількість нової лексики стрімко зростає, зокрема за рахунок абстрактних понять. Абстрактні іменники із суфіксом `-ість` (`незалежність`, `свідомість`, `відповідальність`, `совість`, `щирість`) утворюють найбільшу і найпродуктивнішу лексичну групу в межах третьої відміни [S2, S4]. Саме їхнє правильне розпізнавання та використання у мовленні має стати пріоритетом номер один.

Методичні пріоритети для рівня B1:
1. **Збіг форм у непрямих відмінках**: Факт, що родовий, давальний і місцевий відмінки однини мають абсолютно однакове закінчення `-і` (або паралельне `-и`), є колосальною перевагою для тих, хто вивчає українську мову. Учень має засвоїти просте правило: "Якщо це абстрактний іменник жіночого роду на приголосний (зокрема на `-ість`) — у більшості непрямих відмінків однини сміливо додавай `-і`".
2. **Частотність орудного відмінка**: Форми орудного відмінка третьої відміни (конструкції типу `з радістю`, `з гордістю`, `з любов'ю`) є вкрай високочастотними. Вони використовуються для вираження емоційного стану суб'єкта, способу дії чи супроводу. Ці форми варто відпрацьовувати як готові лексичні фрази (чанки) ще до глибокого розбору всіх нюансів орфографічних правил [S3].
3. **Слово `мати`**: Оскільки це базове поняття сімейної сфери, його парадигму необхідно засвоїти на рівні абсолютного автоматизму. Суфікс `-ер-` не має аналогів у сучасній регулярній словозміні, тому слово `мати` потребує ізольованого механічного тренування.

## Деколонізаційні застереження

**Орудний відмінок слова `мати`**: В українській літературній мові нормативна форма орудного відмінка слова `мати` — `матір'ю` (з суфіксом `-ір-` і апострофом) [S1, S3]. У російській мові форма інша — «матерью» (через `е` та м'який знак). Під час навчання англомовних студентів вчителі та укладачі модулів іноді стикаються з тим, що учні, які мають бекграунд вивчення російської мови або користуються неякісними електронними словниками-перекладачами, утворюють суржикові покручі типу «матер'ю». Важливо наголошувати, що українська форма зберігає історичний суфікс `-ір-` у закритому складі перед апострофом і відповідає питомим законам української фонетики та морфології. Ніколи не слід пояснювати це учням як "форму, схожу на російську, але з апострофом" — це самостійна, історично інша словотвірна модель.

**Паралельні закінчення родового відмінка (`-и`)**: Згідно з українською літературною нормою, іменники на `-ть` після приголосного, а також низка слів (`кров`, `любов`, `осінь`, `сіль`, `Русь`) можуть мати в родовому відмінку однини паралельні закінчення `-і` або `-и` (наприклад, `радости`, `любови`, `незалежности`) [S3]. Ця питомо українська морфологічна риса була цілеспрямовано репресована та штучно вилучена з граматик у радянський період з метою примусового зближення української граматики з російською (де у відповідних словах пишеться «-и», що фонетично реалізується як `[і]`). Відновлення паралельного використання закінчення `-и` в сучасній мові підкреслює фонетичну та морфологічну самостійність української мови. Вчителі мають заохочувати учнів розпізнавати цю форму як ознаку автентичної української літературної традиції, а не сприймати її як орфографічну помилку.

**Кличний відмінок як повноцінна категорія**: В українській мові іменники третьої відміни мають повноцінну форму кличного відмінка із закінченням `-е`: `радосте`, `ноче`, `тіне` [S3]. Це глибока синтаксична і прагматична відмінність від російської мови, яка майже повністю втратила кличний відмінок для цієї категорії слів і послуговується формами називного відмінка. Під час побудови навчальних діалогів чи текстів, де персонажі звертаються до абстрактних понять або явищ природи (наприклад, у поетичному чи стилістично піднесеному мовленні), використання правильної форми кличного відмінка є обов'язковою вимогою деколонізованої української лінгводидактики.

## Природні приклади

Наведені нижче приклади демонструють використання іменників жіночого роду з нульовим закінченням у природних синтаксичних та комунікативних контекстах. Усі приклади згруповано за основними граматичними моделями.

**Модель 1: Називний та Знахідний відмінки (суб'єкти дії та прямі об'єкти просторового напрямку)**
Ці приклади ілюструють, що для неістот форми називного та знахідного відмінків повністю збігаються.
1. «А журавлі летять у синю даль...» [S3]
2. «...І знають вони, де є моя і де твоя печаль…» [S3]
3. «Кімнатні птахи мовчать перед заметіллю.» [S3]
4. «Буду тінню, Ти світанком будь...» [S3]

**Модель 2: Орудний відмінок без прийменника (знаряддя або спосіб дії)**
Ця конструкція використовується для вираження засобу, за допомогою якого виконується дія, або якісної характеристики процесу. Зверніть увагу на орфографічні явища подовження.
5. «Бики вже люттю наливають очі.» [S1]
6. «Разом із розповіддю в душу вливалося щось хороше, тепле.» [S1]
7. «Сонце вже зсередини просвічувало насичені памороззю сади.» [S1]

**Модель 3: Орудний відмінок із прийменником (супровід, емоційний стан, просторове відношення)**
Дуже поширена конструкція для вираження емоційного ставлення до дії або фізичного розташування об'єкта.
8. «Як людина працює з любов’ю — один результат.» [S3]
9. «Він би й стежив, але, стоячи за піччю, нічого не бачив.» [S3]
10. «Хлопець підхопив дівчину на руки, закрутився раптом вихором з ношею своєю на руках...» [S1]

**Модель 4: Давальний і Місцевий відмінки (адресат, мета, місце)**
Демонстрація форми на `-і`, яка спільно використовується для давального та місцевого відмінків.
11. «Він теж поспішав до молоді, щасливий, веселий, як ніколи.» [S3]
12. «Про звичаї й побут козаків-запорожців пишуть очевидці, що були на Січі...» [S3]

## Рекомендації для вправ

Автор навчального модуля повинен дотримуватися чіткої трифазної методичної послідовності впровадження цієї граматичної теми. Це дозволить уникнути когнітивного перевантаження та розбити складне правило на керовані кроки.

**Фаза 1: Сортування та розпізнавання роду.** 
Оскільки англомовні учні постійно спираються на хибний патерн (будь-яке слово на приголосний дорівнює чоловічому роду), найперша вправа має бути візуальною та категоризаційною.
- **Формат тренування:** Drag and Drop або сортування списку слів у два кошики.
- **Завдання:** Розподілити надані слова з нульовим закінченням (наприклад, `кінь`, `шлях`, `дуб` проти `ніч`, `сіль`, `осінь`, `радість`) у дві колонки: чоловічий рід та жіночий рід.
- **Підказка для учня:** Щоб гарантовано визначити рід незнайомого слова, автор має навчити учня підставляти прикметник у контексті: `довгий шлях` (тому це ч.р.), але `темна ніч` (тому це ж.р.). Учні також мають вивчити, що всі слова на `-ість` — завжди жіночого роду.

**Фаза 2: Механіка орудного відмінка (трансформація форм).** 
На цьому етапі відбувається глибоке опрацювання найскладнішого аспекту — правил подовження та вживання апострофа.
- **Формат тренування:** Вправа на трансформацію у контексті (Fill-in-the-blanks).
- **Завдання:** Учень отримує інфінітивну форму (називний відмінок) у дужках і має утворити правильну морфологічну форму орудного відмінка у запропонованому реченні (найчастіше поруч із прийменником `з` або `перед`).
- **Послідовність подачі:** Спочатку учням пропонуються прості випадки, де відбувається подовження (`(тінь) → Він непомітно став тінню`). Наступний крок — слова з двома приголосними наприкінці основи, де подовження заборонене (`(радість) → Він кричав з величезною радістю`). І лише наостанок вводяться лексеми з апострофом (`(любов) → Зроблено з любов'ю`). Автор модуля має явно виводити пояснювальний фідбек у разі виникнення орфографічних помилок (наприклад, якщо учень написав типову помилку "радісттю", система має нагадати про збіг двох приголосних).

**Фаза 3: Контекстне чергування голосних у комунікативних діалогах.** 
Практичне відпрацювання фонетичного чергування `і` → `о`/`е` у реальних, живих комунікативних ситуаціях.
- **Формат тренування:** Побудова коротких діалогів-реплік.
- **Завдання:** Один віртуальний спікер ставить запитання із формою називного або знахідного відмінка (де у корені стоїть закритий звук `і`), а учень має правильно відповісти, використовуючи місцевий або родовий відмінок (де з'являється відкритий `о` або `е`).
- **Приклад інтеракції:** 
  — Ти вже додав у страву *сіль*?
  — Ні, вибач, вдома зовсім немає *солі*. [S6]
  — Де лежить спечений хліб?
  — Він досі лежить у гарячій *печі*. [S6]

## Зв'язки з іншими темами

- **Передумови (що має бути засвоєно раніше):** Для успішного, немеханічного засвоєння цієї складної теми учні вже повинні абсолютно чітко розрізняти тверді, м'які та шиплячі приголосні звуки. Ця базова фонетична тема природно вводиться значно раніше під час вивчення парадигм першої та другої відмін [S2, S4]. Також критично важливою є передумова знайомства із загальним та всеосяжним фонетичним законом української мови — послідовним чергуванням `[і]` з `[о]` або `[е]` у закритих та відкритих складах [S6].
- **Що ця тема уможливлює у майбутньому:** Глибоке розуміння парадигми третьої відміни відкриває для іноземного учня двері для масового та системного засвоєння абстрактної лексики (зокрема слів із суфіксами `-ість`, `-ність`). Без впевненого володіння цією лексикою неможливо генерувати і розуміти складні академічні, публіцистичні чи професійні тексти на просунутих рівнях B2–C1. Крім того, правильне розуміння жіночого роду цих специфічних іменників є необхідним синтаксичним фундаментом для подальшого вивчення збірних числівників (оскільки збірні числівники, за правилами української граматики, ніколи не поєднуються з іменниками жіночого роду).
