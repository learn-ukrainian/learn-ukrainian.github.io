## Як це пояснюють у школі

В українській шкільній програмі розмовний стиль мовлення (на відміну від художнього, наукового та офіційно-ділового) розглядається поступово, починаючи із загальних відомостей про типи текстів у 5–8 класах, де акцентується на його «невимушеності, жвавості бесіди та вияві авторського ставлення» [S4689]. Поглиблене вивчення граматичних та лексичних особливостей розмовного стилю припадає на 10–11 класи в межах курсів стилістики та риторики.

Зокрема, підручники для 10 класу зазначають, що розмовне мовлення є найдавнішим стилем, яким користуються здебільшого в побуті [S712]. Шкільна програма звертає увагу на те, що типовою формою розмовного мовлення є діалог, а звідси випливає використання неповних речень, натяків, емоційно забарвлених слів із суфіксами суб'єктивної оцінки, а також просторічних і сленгових слів (наприклад, *роззява*, *гепнути*, *кроси*) [S712]. Окремо виділяються граматичні риси: використання власне неповних речень (еліптичних, обірваних), слів-речень та широке вживання звертань [S713].

У 10–11 класах пояснюється стилістична роль засобів словотвору (суфіксів та префіксів), які надають словам емоційно-експресивного забарвлення [S1454], а також специфіка вживання розмовних фразеологізмів, що позбавлені офіційності та формують колорит простоти [S1454, S1233]. Водночас українська педагогіка активно застерігає учнів від використання суржику (суміші двох мов) у побутовому мовленні, наголошуючи на необхідності дотримуватися літературних норм навіть у невимушеному спілкуванні [S711, S712].

## Повна парадигма: Морфологічні та синтаксичні маркери розмовного стилю

Розмовний стиль не є окремою вузькою граматичною категорією (як-от відмінок чи час), проте він має чіткий набір граматичних маркерів, що відрізняють його від нейтрального чи формального тексту. Для навчання на рівні B1/B2 ці маркери мають бути формалізовані.

### 1. Емоційно-оцінний словотвір (Суфіксація)
Використання суфіксів для надання додаткових відтінків (здрібнілості, пестливості, ласкавості, зневаги, згрубілості) є однією з найяскравіших ознак українського розмовного мовлення [S1454]. 

**Зменшувально-пестливі форми (Димінутиви)**
Такі суфікси утворюють слова з позитивним емоційним, експресивним забарвленням [S1454].

| Тип основи / Частина мови | Суфікси | Правило творення (Ланцюжок) | Приклад у розмовній мові |
| :--- | :--- | :--- | :--- |
| Іменники (чоловічий рід) | `-ик`, `-ок`, `-очок` | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `кіт → кот- → +ик → котик` |
| | | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `голос → голос- → +очок → голосочок` |
| Іменники (жіночий рід) | `-очк(а)`, `-еньк(а)` | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `хатина → хатин- → +очк(а) → хатиночка` [S1454] |
| | | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `рука → руч- (чергування) → +к(а) → ручка` |
| Прикметники | `-еньк(ий)`, `-есеньк(ий)`, `-ісіньк(ий)` | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `рідний → рідн- → +еньк(ий) → рідненький` [S1454] |
| | | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `рівний → рівн- → +есеньк(ий) → рівнесенький` [S1454] |
| | | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `тонкий → тон- (відсікання -к) → +ісіньк(ий) → тонісінький` [S1454] |

**Збільшувально-згрубілі форми (Аугментативи)**
Використовуються для надання зневажливого, іронічного або посиленого (збільшеного) значення [S1454].

| Тип основи / Частина мови | Суфікси | Правило творення (Ланцюжок) | Приклад у розмовній мові |
| :--- | :--- | :--- | :--- |
| Іменники | `-ищ(е)`, `-ущ(е)`, `-иськ(о)` | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `вовк → вовч- (чергування) → +ищ(е) → вовчище` [S1454] |
| | | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `хлопець → хлопч- → +иськ(о) → хлопчисько` [S1454] |
| | | `{називний} → {основа} → {+суфікс} → {кінцева форма}` | `вітер → вітр- → +иськ(о) → вітрисько` [S1454] |

*Обмеження:* Димінутиви та аугментативи використовуються переважно в розмовному та художньому стилях. Якщо слова зі зменшувально-пестливими суфіксами потрапляють до наукового стилю як терміни (наприклад, анатомічний термін *шлуночок*), вони повністю позбавляються емоційно-експресивних відтінків [S1454]. Окрім того, активні та пасивні дієприкметники ніколи не приймають зменшувально-пестливих суфіксів.

### 2. Стягнені (короткі) форми дієслів
В усному неформальному мовленні та фольклорі часто використовуються стягнені форми дієслів теперішнього часу [S1454]. Це явище передбачає усічення кінцевого голосного або цілого складу, що надає мовленню ритмічності та розмовної легкості.

| Особа / Число | Повна (літературна) форма | Правило усічення (Ланцюжок) | Стягнена (розмовна) форма |
| :--- | :--- | :--- | :--- |
| 3-тя особа однини | співає, знає, гуляє | `{повна форма} → {усічення -є} → {кінцева форма}` | `співає → співа` [S1454] |
| 1-ша особа множини | ходимо, робимо, знаємо | `{повна форма} → {усічення -о} → {кінцева форма}` | `ходимо → ходим` [S1454] |

*Обмеження:* Стягнені форми 3-ї особи однини утворюються ТІЛЬКИ від дієслів I дієвідміни, які в повній формі закінчуються на `-ає` або `-яє` (`знає` → `зна`, `співає` → `співа`). Для дієслів II дієвідміни (`ходить`, `робить`) таке стягнення в 3-й особі однини неможливе. Водночас усічення кінцевого `-о` в 1-й особі множини можливе для обох дієвідмін (`знаємо` → `знаєм`, `ходимо` → `ходим`).

### 3. Синтаксична редукція (Неповні та еліптичні речення)
Синтаксис розмовного стилю базується на законі економії мовленнєвих зусиль. Найхарактернішими є структури еліптичного типу [S713].

*   **Еліптичні речення:** Конструкції, у яких фізично відсутній присудок (дієслово), але його лексичне значення легко й однозначно відновлюється з контексту або мовленнєвої ситуації [S713].
    *   *Механізм:* `{Повне двоскладне речення} → {Вилучення дієслова-присудка} → {Залишок із залежними обставинами чи додатками}`
    *   *Приклад:* `Я йду на роботу, а він іде до школи.` → `Я на роботу, а він до школи.`
*   **Слова-речення (Етикетні одиниці):** Стверджувальні, заперечні або спонукальні слова, що замінюють цілу граматичну репліку [S713]. До них належать розмовні вигуки та частки: `Авжеж`, `Еге`, `Овва`, `Ой`.

### 4. Розмовні фразеологізми
До розмовного стилю належать фразеологізми, які мають «зневажливо-знижене експресивне забарвлення» та «формують колорит простоти» [S1454]. Вони допомагають «влучно, точно, образно, дотепно висловити думки, почуття» [S1233].

| Книжний / Нейтральний еквівалент | Розмовний фразеологізм [S1454] |
| :--- | :--- |
| Забагато говорити | `розпустити язика` |
| Дуже міцно спати | `спати без задніх ніг` |
| Безпідставно, зненацька | `ні сіло ні впало` |
| Зазнаватися, хизуватися | `задирати ніс` |
| Посваритися | `розбити глека` |

## Частотність і пріоритети

Для англомовних учнів на рівні B1 найвищий пріоритет засвоєння мають:
1.  **Неповні речення та слова-речення:** Учні повинні навчитися відмовлятися від звички відповідати на запитання повними синтаксичними структурами (як їх вчили на рівні А1). Вміння спілкуватися лаконічно, оминаючи відомий об'єкт розмови [S712] та використовуючи репліки-реакції — це основа природного звучання.
2.  **Розмовні фразеологізми:** Частотність побутових фразеологізмів (на зразок `бити байдики`, `морочити голову` [S1454]) в українській мові надзвичайно висока. Їх потрібно вводити як неподільні лексичні блоки.
3.  **Димінутиви (зменшувально-пестливі форми):** Учні мають усвідомити, що форми на кшталт `сонечко`, `хвилинка`, `кавочка` є не просто «дитячою мовою» (baby talk), а звичайним потужним інструментом пом'якшення тону розмови та прояву емпатії у повсякденній українській комунікації.
4.  **Стягнені дієслова:** Відіграють роль виключно пасивного (рецептивного) пріоритету. Учень має легко впізнавати форми `зна`, `співа`, `ходим` на слух чи в літературних текстах [S1454], проте стимулювати їхнє активне використання на письмі іноземцями не варто, щоб уникнути плутанини зі структурами наказового способу чи іншими часами.

## Типові помилки L2 (англомовні учні)

Англомовні учні, опановуючи розмовний український стиль, найчастіше переносять патерни англійського жорсткого синтаксису або використовують занадто формальну (книжну) лексику, оскільки зазвичай це єдиний регістр, яким вони володіють.

| ❌ Помилково | ✅ Правильно | Чому (механізм помилки) |
| :--- | :--- | :--- |
| **А:** Де ти був? <br> **Б:** Я був у школі. | **А:** Де ти був? <br> **Б:** У школі. | Учні намагаються будувати симетричні, граматично повні відповіді, механічно копіюючи структуру поставленого питання. В українському розмовному стилі переважають еліптичні речення [S713], де повторення підмета і присудка є стилістично надлишковим. |
| Привіт, Антон! Як ти? | Привіт, **Антоне**! Як справи? | Ігнорування кличного відмінка у звертаннях. В англійській мові форма імені при звертанні не змінюється (zero morphology), тому учні часто забувають про обов'язковість використання кличного відмінка, який є базовою ознакою розмовного стилю [S713]. |
| Ця проблема є дуже велика. | Це **здоровенна** / **величезна** проблема. | Учні вдаються до аналітичного способу вираження міри ознаки (англійська структура "very" + прикметник), ігноруючи багатший інструментарій українських емоційно-оцінних суфіксів (аугментативів `-енн-`, `-елезн-`) [S1454]. |
| Він ніколи не закриває свій рот. | Він часто **розпускає язика**.<br>Він **балакучий**. | Буквальне синтаксичне калькування (переклад слово в слово) англійських ідіоматичних зворотів ("he never closes his mouth") замість використання питомих українських фразеологізмів розмовно-побутового стилю [S1454]. |
| Ми ходимо до кінотеатру кожні вихідні. | Ми **ходим** у кіно щовихідних. | Використання виключно повних, книжних дієслівних форм у неформальному мовленні. Стягнені форми першої особи множини (на зразок `ходим`, `робим`) є природними маркерами невимушеності [S1454]. |
| Да, получається, що так. | **Авжеж**, **виходить**, що так. | Вживання лексичного суржику (суміші двох мов) [S712] під впливом вуличного мовлення, запозиченого з інтернету чи від російськомовних носіїв, замість питомих українських стверджувальних часток і дієслів. |

## Деколонізаційні застереження

Працюючи з розмовним стилем, критично важливо відокремити органічну українську розмовну мову від **суржику** (суміші української та російської мов, що виникла внаслідок системної русифікації) [S711, S712].

1. **Небезпека суржику для іноземців:** Англомовні учні часто чують на вулицях або у відеоматеріалах слова на кшталт `да`, `карочє`, `получається`, `понятно`, `блін`. Вони можуть помилково сприйняти їх як «сучасний міський сленг». Проте шкільні підручники прямо застерігають: «Мовлення людини... часто зазнає шкідливих впливів... сумішшю двох мов, так званим суржиком. Тому культура мовлення багато в чому залежатиме від опанування особливостями [питомого] розмовного стилю» [S711]. На рівні B1 автор зобов'язаний свідомо навчати питомих українських розмовних маркерів-замінників: `так`, `авжеж`, `еге`, `виходить`, `зрозуміло`, `ой`, `овва`, `трясця`.
2. **Димінутиви як соціальна норма, а не інфантилізм:** В англійській мові зменшувально-пестливі форми міцно асоціюються зі спілкуванням із маленькими дітьми. У російській мові вони нерідко використовуються у сфері бюрократичного обслуговування з відтінком фальшивої ввічливості («справочка», «документики»). В українській мові димінутиви є органічною частиною доброзичливого повсякденного спілкування дорослих. Вони несуть соціальну функцію зменшення дистанції та вияву щирості (наприклад, традиційні звертання `дівчинонько`, описові форми `голосочок`, `дощичок` [S1454, S713]). Не слід подавати їх учням як «дитячу» мову, вони є індикаторами теплих стосунків.
3. **Займенник «Ви» у розмовній мові:** Звертання на «Ви» в українській культурі не вимикається повністю навіть у розмовно-побутовому стилі. Традиційним залишається звертання на «Ви» до батьків або старших родичів у багатьох регіонах України [S713]. Це принципово відрізняється від російської соціолінгвістичної парадигми, де викання в сім'ї вважається застарілим архаїзмом.

## Природні приклади

Наведені нижче речення з українських підручників ілюструють ключові механізми розмовного стилю.

**Група 1: Еліптичні (неповні) конструкції в діалогах**
Діалог — це типова форма розмовного мовлення, що спирається на неповні речення та контекстуальні натяки, уникаючи багатослівності [S712].
*   `— Алло! Куди я зателефонував? — Добрий день. З Вами говорить помічник директора... — Мені треба ваше начальство.` [S1188]
*   `— А чого ви за кожним разом гехкаєте? — Чого? — косує на мене чоловік очима і вусами. — Ніби не знаєш.` [S713]
*   `— А де твоя мама, Олесю? — Вона щойно вийшла до крамниці. Телефон ще теплий.` [S1188]

**Група 2: Емоційно-оцінна лексика (зменшувально-пестливі суфікси)**
Широке використання суфіксів оцінки для встановлення теплого емоційного контакту або передавання ласкавого ставлення [S1454].
*   `— Здорові, діточки. А чого ж це ви, голуб’ята, усі такі зелені?` [S1454]
*   `— Ну, а де ж ваша мама? — питає бабуся, приглядаючись до чорних тенеточок на дітях.` [S1454]
*   `— Гарні, любі діточки… А гостинчика немає, ластів’ята, — не купила.` [S1454]

**Група 3: Розмовні, просторічні та сленгові слова**
Вживання емоційно забарвлених слів, просторіччя та молодіжного сленгу як індикатора неформального середовища [S712].
*   `— Нічого собі! — крикнув Лобода. — Народ! Подивіться, які у Потапа кроси!` [S712]
*   `— Кльові! — то підійшов Ромко Загорський. Присів, помацав задник кросівок і прицмокнув від захвату.` [S712]
*   `Без «гех» дерева не осилиш. Не віриш — спробуй. Ставай на моє місце.` [S713]

**Група 4: Фразеологізми побутового стилю**
Використання сталих словосполучень для влучної та образної характеристики дій чи життєвих ситуацій [S1233].
*   `На старого Сірка трохи напала балакучка, як часто після доброї вдачі, він був у доброму гуморі.` [S1454]
*   `Тоді мама правду каже, що я в тебе вдався.` [S713]
*   `Суднобудування поки що сидить на мілині.` [S1454]

## Рекомендації для вправ

Для автора-письменника, який створює модулі з розмовної граматики, рекомендується наступна методична послідовність відпрацювання матеріалу:

*   **Фаза 1: Розпізнавання та аналіз регістра (Recognition).**
    *   *Тип вправи:* Порівняння текстів. Учням пропонуються два мікротексти з однаковим смислом (перший — офіційно-діловий або науковий, другий — розмовно-побутовий [S711]).
    *   *Завдання:* Виділити слова з емоційно-оцінними суфіксами, знайти розмовні фразеологізми та виокремити неповні речення. Мета фази — навчити іноземця бачити граматичні індикатори, які маркують стиль тексту.
*   **Фаза 2: Трансформація стилю (Stylistic Transformation).**
    *   *Тип вправи:* Переписування речень.
    *   *Завдання:* Учень отримує граматично повне, офіційне (або «підручникове» з рівня А1) речення і має перетворити його на природне розмовне, використовуючи еліпсис, стягнені форми дієслів (наприклад, перетворити `ходимо` на `ходим` [S1454]) або димінутиви (`вода` → `водичка`). *Зразок завдання:* Трансформуйте «Я йду до магазину, щоб купити хліб» → «Я в магазин по хліб».
*   **Фаза 3: Рольова гра з параметрами (Role-play Dialogues).**
    *   *Тип вправи:* Кероване продукування в межах заданого сценарію.
    *   *Завдання:* Скласти коротку розмову (наприклад, телефонну бесіду між старими друзями [S1188]) з обов'язковим виконанням заданих параметрів: вжити два звертання у кличному відмінку, використати дві еліптичні репліки-відповіді та інтегрувати один розмовний фразеологізм (наприклад, `бити байдики` [S1454]).

*Методичний пріоритет:* Починати відпрацювання розмовного стилю слід саме із синтаксису (неповні речення у бліц-діалогах), оскільки це найшвидше дає учням відчуття «природності» та швидкості власної мови. Після закріплення синтаксичних навичок можна переходити до лексико-морфологічного рівня (фразеологізми, стягнення, суфікси оцінки).

## Зв'язки з іншими темами

*   **Передумови:** Учні вже повинні міцно володіти загальною парадигмою дієвідмінювання теперішнього часу (щоб структурно розуміти механізм усічення кінцевих голосних у стягнених формах). Також потрібне розуміння синтаксису простого двоскладного речення (щоб знати, який саме головний член вилучається під час еліпсу). Форми кличного відмінка мають бути засвоєні заздалегідь.
*   **Що ця тема уможливлює:** Розуміння і практичне використання розмовного стилю відкриває двері до легшого споживання автентичного україномовного медіаконтенту (подкасти, влоги, сучасне ігрове кіно). Крім того, це формує базу для подальшого вивчення стилістики художньої літератури на високих рівнях (B2–C1), де розмовний стиль використовується авторами як ключовий засіб індивідуальної характеристики персонажів.
