## Як це пояснюють у школі

У системі шкільної освіти України тема найвищого ступеня порівняння прикметників (Superlative Degree of Adjectives) вводиться систематично й поступово. Базове знайомство з цією граматичною категорією відбувається у 6 класі. Заболотний та Літвінова у своїх підручниках для 6 класу подають цю тему безпосередньо після вивчення вищого ступеня або в єдиному блоці з ним [S1, S2, S4]. Основний акцент на цьому етапі робиться на семантиці: найвищий ступінь вказує на те, що в якомусь предметі певної якості найбільше або найменше порівняно з усіма іншими предметами [S1]. 

У старшій школі (10–11 класи) підручники Карамана та Глазової значно поглиблюють розгляд теми [S3, S7]. Тут фокус зміщується з механічного творення форм на стилістичну доцільність їх використання та культуру мовлення. Учні вивчають, що проста форма найвищого ступеня є універсальною для всіх стилів, тоді як складена форма тяжіє до наукового та офіційно-ділового [S7]. Особлива увага приділяється типовим помилкам: подвійному маркуванню ступенів та калькам із російської мови (використання займенника «самий»).

Методика подачі матеріалу в підручниках строго спирається на розуміння того, що найвищий ступінь є похідним від вищого. Тому всі правила творення базуються не на зміні початкової форми прикметника, а на модифікації вже утвореної форми вищого ступеня [S1, S3, S4].

## Повна парадигма

Українська мова має дві форми найвищого ступеня порівняння: просту (синтетичну) та складену (аналітичну). Творення простої форми є багатоетапним морфологічним процесом. Найважливіше правило, яке слід засвоїти: префікс `най-` **ніколи** не додається до початкової форми прикметника; він додається виключно до форми вищого ступеня [S1, S3, S4].

### Проста (синтетична) форма

Творення простої форми відбувається за чіткими ланцюжками морфонологічних перетворень.

**Модель 1: Регулярна афіксація (без зміни основи)**
До основи початкової форми додається суфікс `-іш-`, утворюючи вищий ступінь, після чого приєднується префікс `най-`.
*Ланцюжок:* `{початкова форма} → {основа} → {+ суфікс -іш-} → {форма вищого ступеня} → {префікс най- + форма вищого ступеня} → {кінцева форма}`
*   `новий → нов- → + -іш- → новіший → най- + новіший → найновіший` [S2]
*   `важливий → важливіш- → + -іш- → важливіший → най- + важливіший → найважливіший` [S4]

**Модель 2: Афіксація з випаданням суфіксів `-к-`, `-ок-`, `-ек-`**
Якщо основа прикметника закінчується на ці суфікси, вони відсікаються перед додаванням суфікса `-ш-`.
*Ланцюжок:* `{початкова форма} → {основа} → {відсікання -к-/-ок-/-ек-} → {+ суфікс -ш-} → {форма вищого ступеня} → {префікс най- + форма вищого ступеня} → {кінцева форма}`
*   `тонкий → тонк- → тон- → + -ш- → тонший → най- + тонший → найтонший` [S4]
*   `глибокий → глибок- → глиб- → + -ш- → глибший → най- + глибший → найглибший` [S4]
*   `далекий → далек- → дал- → + -ш- → дальший → най- + дальший → найдальший` [S4]

**Модель 3: Чергування приголосних `[г]`, `[ж]`, `[з]` + `[ш]` → `[жч]`**
При додаванні суфікса `-ш-` кінцеві приголосні основи асимілюються.
*Ланцюжок:* `{початкова форма} → {основа} → {відсікання -к- якщо є} → {+ суфікс -ш-} → {асиміляція в [жч]} → {форма вищого ступеня} → {префікс най- + форма вищого ступеня} → {кінцева форма}`
*   `дорогий → дорог- → + -ш- → дорожчий → най- + дорожчий → найдорожчий` [S1, S4]
*   **`близький → близьк- → близ- → + -ш- → ближчий → най- + ближчий → найближчий`** [S1, S4]
*   **`важкий → важк- → важ- → + -ш- → важчий → най- + важчий → найважчий`** [S4]

**Модель 4: Чергування приголосних `[с]`, `[ст]` + `[ш]` → `[шч]` (на письмі `щ`)**
*Ланцюжок:* `{початкова форма} → {основа} → {відсікання -ок- якщо є} → {+ суфікс -ш-} → {асиміляція в [шч]/щ} → {форма вищого ступеня} → {префікс най- + форма вищого ступеня} → {кінцева форма}`
*   **`високий → висок- → вис- → + -ш- → вишчий (вищий) → най- + вищий → найвищий`** [S1, S4]
*   **`товстий → товст- → + -ш- → товшчий (товщий) → най- + товщий → найтовщий`** [S4]

**Модель 5: Суплетивні форми (зміна кореня)**
Окремі високочастотні прикметники утворюють ступені порівняння від іншого кореня.
*Ланцюжок:* `{початкова форма} → {суплетивна форма вищого ступеня} → {префікс най- + форма вищого ступеня} → {кінцева форма}`
*   **`великий → більший → найбільший`** [S1, S4]
*   **`малий → менший → найменший`** [S1, S4]
*   **`гарний → кращий → найкращий`** [S1, S4]
*   **`поганий → гірший → найгірший`** [S1, S4]
*   **`добрий → ліпший (або добріший) → найліпший (або найдобріший)`** [S4]

### Складена (аналітична) форма та підсилення

Складена форма утворюється шляхом додавання незмінюваних слів `найбільш` або `найменш` до прикметника в **початковій формі**.
*Ланцюжок:* `{допоміжне слово найбільш / найменш} + {початкова форма прикметника} → {кінцева форма}`
*   `широкий → найменш широкий` [S1]
*   `вдалий → найбільш вдалий` [S1]

Крім того, українська мова має підсилені форми найвищого ступеня (елятивно-суперлятивні), які утворюються за допомогою часток `як-` або `що-`.
*Ланцюжок:* `{частка як- / що-} + {проста форма найвищого ступеня} → {кінцева форма}`
*   `найкращий → якнайкращий` [S4]
*   `найсильніший → щонайсильніший` [S4]

### Абсолютні обмеження на творення ступенів порівняння

Пропущене обмеження під час створення навчальних вправ є критичною методичною помилкою. Далеко не всі якісні прикметники здатні утворювати найвищий ступінь. Українські підручники [S1, S2, S3, S4, S7] чітко фіксують такі семантичні та структурні заборони:

1. **Ознаки абсолютної якості** (що не підлягають кількісному виміру).
   *   *Обмеження:* Якщо якість не може проявлятися більшою чи меншою мірою, ступенювання неможливе.
   *   *Перелік:* порожній, глухий, німий, лисий, босий, сліпий, хворий, готовий, кривий, живий, косий, мертвий, голий. Форми `*найніміший`, `*найпорожніший` категорично заборонені.
2. **Масті тварин**.
   *   *Обмеження:* Прикметники, що традиційно позначають масть коней та інших тварин.
   *   *Перелік:* вороний, гнідий, буланий, чалий, соловий, мурий, карий. Форми `*найвороніший`, `*найгнідіший` не існують.
3. **Ознаки з уже вбудованою мірою (суфікси)**.
   *   *Обмеження:* Якщо слово вже містить суфікс зменшеності, пестливості або згрубілості, воно не ступенюється.
   *   *Заборонені суфікси:* -уват-/-юват-, -езн-, -енн-, -есеньк-/-ісіньк-/-юсіньк-, -еньк-, -ав-/-яв-, -ущ-/-ющ-, -анн-/-янн-. Форми `*найчорнесенькіший`, `*найстарезніший`, `*найзначущіший` є помилковими.
4. **Ознаки з уже вбудованою мірою (префікси)**.
   *   *Обмеження:* Слова з префіксами абсолютної міри (елятив) не можуть утворювати суперлятив.
   *   *Заборонені префікси:* пре-, над-, пра-, за-, архі-, ультра-, гіпер-, транс-. Форми `*найпредобріший`, `*найзависокіший` не існують.
5. **Складні прикметники**.
   *   *Обмеження:* Слова, утворені складанням основ (часто для передачі складних кольорів).
   *   *Приклади:* білосніжний, жовтогарячий, темно-синій, чорнобокий. Форма `*найтемносиніший` неможлива.
6. **Кольори, утворені за подібністю до предмета** (відносно-якісні).
   *   *Обмеження:* Відносні прикметники, що перейшли в розряд якісних, але зберігають зв'язок із матеріалом/предметом.
   *   *Перелік:* вишневий, кремовий, малиновий, шоколадний, бордовий, салатовий, каштановий, кавовий, бузковий. Форми `*найвишневіший`, `*найшоколадніший` не утворюються.
7. **Безвідносний вияв ознаки (відтінки)**.
   *   *Обмеження:* Слова, що позначають неповний колір (білястий, голубуватий) [S7].
8. **Термінологічні сталі словосполучення**.
   *   *Обмеження:* Прикметники у формі вищого ступеня, що стали термінами (старша медсестра, нижчий клас ссавців), утрачають значення вищої міри і не утворюють найвищого ступеня у своєму термінологічному значенні [S7].

## Частотність і пріоритети

У живому усному мовленні, фольклорі та художній літературі українська мова віддає беззаперечну перевагу **простій (синтетичній) формі** найвищого ступеня (напр., `найкращий`, `найрозумніший`) [S7]. Складені форми (`найбільш гарний`, `найбільш розумний`) є громіздкими і сприймаються як стилістично марковані — їхнє природне середовище обмежується науковим та офіційно-діловим стилями, де вони забезпечують термінологічну точність без зміни морфемної структури слова [S7]. Авторам навчальних модулів слід зосередити 80% уваги на відпрацюванні простих форм, оскільки саме вони становлять основу органічного українського мовлення.

## Типові помилки L2 (англомовні учні)

Англомовні учні часто спираються на хибні аналогії з рідною мовою або на російські кальки, що призводить до грубих граматичних порушень.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| *The tallest of all*<br>Самий високий | **Найвищий**<br>або<br>**Вищий за всіх** | Це пряме калькування російської моделі формотворення (`самый` + прикметник) [S1, S3]. В українській літературній мові займенник `самий` має ідентифікуюче значення (той самий чоловік), а не ступеневе. Для вираження значення «of all» використовується префікс `най-` або синтаксична конструкція `вищий за всіх`. |
| *The most best*<br>Найбільш найкращий<br>Найбільш кращий | **Найкращий**<br>або<br>**Найбільш гарний** | Подвійне маркування (double marking). В українській мові категорично заборонено поєднувати аналітичні форманти `найбільш/найменш` із простою формою вищого чи найвищого ступеня [S3]. Слово `найбільш` може стояти **тільки** перед прикметником у початковій (словниковій) формі. |
| *The shortest*<br>Найкороткий | **Найкоротший** | Пропуск обов’язкового проміжного етапу творення. Англомовні учні часто сприймають префікс `най-` як самостійний еквівалент англійського `the most` і додають його безпосередньо до початкової форми прикметника. В українській мові `най-` завжди додається **виключно** до форми вищого ступеня (коротший), а не до початкової [S1, S3, S4]. |
| *The most heavy*<br>Найважкший<br>Найважчий | **Найважчий** | Ігнорування морфонологічних змін. В англійській мові додавання суфіксів не руйнує корінь слова (heavy -> heaviest). Учень L2 не усвідомлює необхідності випадання суфікса `-к-` і подальшого чергування приголосних `ж + ш → жч` при творенні проміжної форми вищого ступеня [S1, S4]. |
| *The most blind*<br>Найбільш сліпий | **Абсолютно сліпий**<br>або<br>**Зовсім сліпий** | Ігнорування семантичних обмежень якісних прикметників. Англійська мова іноді дозволяє метафоричне ступенювання абсолютних ознак (the blindest), проте українська граматична норма суворо забороняє утворювати ступені порівняння від прикметників, що позначають абсолютну ознаку (сліпий, глухий, мертвий) [S1, S4, S7]. |
| *More warmer*<br>Більш тепліший | **Тепліший**<br>або<br>**Більш теплий** | Ще один прояв подвійного маркування. Англомовні учні часто додають `більш` до вже зміненого прикметника. Це грубе порушення літературної норми; `більш/менш` поєднується лише з початковою формою [S1, S3, S5]. |

## Деколонізаційні застереження

Цей розділ є критично важливим для укладачів курсу, оскільки багато русизмів міцно вкоренилися в мовній свідомості. Письменник-автор модуля повинен бути особливо пильним, щоб не допустити російських кальок у навчальні матеріали.

1. **Сувора заборона займенника «самий» для утворення ступенів**. Російська мова використовує аналітичну конструкцію «самый + прикметник» (самый красивый, самый умный). Це найпоширеніша і найбільш шкідлива калька в недбалому українському мовленні. В українській літературній мові словосполучення типу «самий кращий», «самий розумний» є грубими, неприпустимими помилками [S1, S2, S3, S7]. Займенник «самий» в українській вживається лише у значенні ідентичності («той самий день») або для вказівки на просторову/часову межу («до самого вечора»).
2. **Неприпустимість граматичного міксування («більш тепліший»)**. У російському просторіччі іноді трапляється змішування аналітичних і синтетичних форм. В українській літературній нормі поєднання частки «більш/менш» або «найбільш/найменш» із простою формою вищого ступеня («більш швидший», «найбільш цікавіший») суворо заборонено [S2, S3, S5]. Це завжди свідчить про низьку мовну культуру.
3. **Пріоритет питомих синтетичних форм**. Російська мова частіше звертається до описових (складених) конструкцій. Українська мова, як зазначають провідні мовознавці, завжди віддавала і віддає перевагу простим, синтетичним формам (`найвищий`, `найрозумніший`) [S7].

## Природні приклади

Нижче наведено корпусні підручникові приклади функціонування найвищого ступеня прикметників в українській мові. Жоден із них не є перекладом з англійської.

### Група 1: Проста (синтетична) форма в побутовому та навчальному дискурсі
1. У лісі під час вітряної погоди страждають **найвищі** сосни. [S1]
2. **Найкращий** результат за контрольну роботу мала моя подруга. [S1]
3. Психологи радять робити **найпершою** або ту справу, що вимагає найменшої кількості часу, або **найважливішу**. [S4]
4. Дівчатка одягнули свої **насвятковіші** вишиванки. [S2]

### Група 2: Складена (аналітична) форма та контекст порівняння
5. І наостанок слід залишити **найменш важливу** справу. [S4]
6. Довкола запанувала така тиша, якої ще ніколи не було в нашому класі, що вважався **найбільш галасливим** і недисциплінованим у всій школі. [S4]
7. Унікальне зібрання з 250 зразків народного іконопису XVIII–ХХ століть сам автор колекції вважає чи не **найбільшим** у світі. [S3]
8. Я ж казала: **найменш помітне** те, що стирчить у вас перед самісіньким носом! [S4]

### Група 3: Підсилені форми та публіцистичний стиль
9. Українка Христина Стуй **найпершою** подолала стометрівку. [S3]
10. Усі люди прагнуть дістати **найсвіжіші** відомості протягом **якнайкоротшого** часу. [S7]
11. 14 квітня 2016 року Житомир увійшов до «Книги рекордів України» як **найблагодійніше** місто. [S3]
12. Гості музейного комплексу залишали свої **найзаповітніші** мрії та побажання на спеціальному сувої завдовжки 32,5 метра. [S3]

### Група 4: Художній та поетичний стиль (суб'єктивна найвища міра)
13. **Найогидніші** очі порожні, **найгрізніше** мовчить гроза, **найнікчемніші** дурні вельможні, **найпідліша** брехлива сльоза. [S7]
14. **Найпрекрасніша** мати щаслива, **найсолодші** кохані вуста, **найчистіша** душа незрадлива, **найскладніша** людина проста. [S7]

## Рекомендації для вправ

Для ефективного засвоєння теми автором-письменником рекомендовано впровадити таку методичну послідовність (прогресію):

1. **Фаза 1: Відпрацювання регулярних форм (афіксація без чергувань).** Навчання слід починати з прикметників, які не змінюють основи (новий, теплий, цікавий). Учні тренують алгоритм `{початкова форма} → {+ -іш-} → {+ най-}` [S1, S2]. Оптимальний формат вправ: трансформація (fill-in-the-blanks), де учень бачить початкову форму і має вивести найвищий ступінь.
2. **Фаза 2: Суплетивізм та фонетичні чергування.** Після міцного засвоєння базового алгоритму вводяться високочастотні суплетивні форми (великий — найбільший, гарний — найкращий) та слова з випаданням суфіксів і чергуванням (високий — найвищий, дорогий — найдорожчий) [S1, S4]. Формат вправ: matching (знайди пару початковій формі) та error correction (виправлення навмисних помилок у корені, напр., `найблизькший` → `найближчий`).
3. **Фаза 3: Аналітичні конструкції та пастки калькування.** Учні вивчають конструкції з `найбільш/найменш`, засвоюючи жорстке правило їхнього поєднання **виключно** з початковою формою. На цьому етапі критично важливо ввести distractor exercises (вправи з відволікаючими варіантами), де учневі пропонується обрати правильний варіант серед форм із займенником «самий» або подвійним маркуванням («самий кращий», «найбільш цікавіший»). Учень повинен розпізнати і відхилити ці суржикові форми [S2, S3].

## Зв'язки з іншими темами

*   **Вищий ступінь порівняння прикметників:** Тема найвищого ступеня є абсолютно залежною від засвоєння вищого ступеня. Без навички творення вищого ступеня (з усіма його чергуваннями) неможливо правильно утворити просту форму найвищого [S1, S4].
*   **Ступені порівняння прислівників:** Аналогічні морфонологічні процеси та заборони діють при творенні найвищого ступеня прислівників (`голосно → голосніше → найголосніше`) [S5]. Форманти та афікси є спільними.
*   **Префікс `пре-`:** Важливо розрізняти відносний найвищий ступінь (суперлятив із префіксом `най-`) та абсолютну високу міру ознаки (елятив із префіксом `пре-`). Слово `превисокий` означає «дуже високий», а не «найвищий з усіх» [S6]. Учні часто плутають ці поняття через схожість семантики.
