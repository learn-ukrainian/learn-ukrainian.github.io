## Як це пояснюють у школі

У системі української шкільної освіти граматичні конструкти, необхідні для опису щоденного життя, рутини та повсякденної взаємодії, не подаються одним масивним блоком. Натомість підручники використовують концентричний підхід, поступово поглиблюючи знання учнів з морфології та синтаксису впродовж кількох років. Для рівня B1 (де фокус робиться на розповідях про себе, свої звички, навчання та плани) матеріал синтезується з тем 7-го, 9-го та 11-го класів.

У 7 класі українські школярі детально вивчають морфологію дієслова. Саме тут закладається фундамент для розуміння часових форм. Підручники чітко розмежовують поняття часу і виду, пояснюючи, що теперішній час нерозривно пов'язаний із недоконаним видом [S5]. Вводиться концепція неозначеної форми (інфінітива) як незмінної та початкової [S3]. Особлива увага приділяється конструюванню майбутнього часу, де школярі вчаться розрізняти просту, складну та складену форми залежно від виду дієслова [S4]. 

У 9 класі фокус зміщується на лексичні норми та культуру мовлення. Шкільна програма активно бореться з типовими лексичними помилками та суржиком у побутових темах. Наприклад, учням детально пояснюють різницю між паронімами у сфері одягу та побуту, суворо розмежовуючи дієслова, як-от використання правильного слова для процесу вдягання [S2].

У старшій школі (11 клас) базові навички шліфуються через вивчення синтаксичної та стилістичної норми. Підручники акцентують увагу на правильному синтаксичному керуванні (який відмінок вимагає те чи інше дієслово) у повсякденній комунікації, наприклад, як правильно говорити про хвороби, навчання чи міжособистісні стосунки [S6]. Також на цьому етапі закріплюються жорсткі стилістичні норми щодо позначення часу, дат та регулярності дій, виправляючи поширені розмовні відхилення [S1].

Отже, для автора-письменника, який укладає модулі для рівня B1, важливо розуміти: англомовний учень має засвоїти не просто набір слів, а складну матрицю взаємодій між видом дієслова, часом та суворим прийменниковим керуванням, яка в українській мові значно жорсткіша, ніж в англійській.

## Повна парадигма

Нижче наведено вичерпні парадигми ключових граматичних категорій, необхідних для опису щоденного життя на рівні B1.

### 1. Неозначена форма дієслова (Інфінітив)

Інфінітив є базовою формою для опису звичок і рутинних планів. 
**Обмеження:** Неозначена форма є абсолютно незмінною; вона позначає узагальнену дію чи стан без вказівки на особу, число, рід чи час [S3].

**Правила творення:**
- Стандартна форма: `{основа} → {+суфікс -ти} → {кінцева форма}`. Приклад: `збира- → +ти → збирати` [S3].
- Зворотна форма: `{основа} → {+суфікс -ти} → {+постфікс -ся/-сь} → {кінцева форма}`. Приклад: `збира- → +ти → +ся → збиратися` [S3].

**Обмеження щодо суфікса -ть:** Використання усіченого суфікса `-ть-` (наприклад, `збирать`) має винятково розмовний характер і не є нормативним для академічного чи формального стилю [S3].

### 2. Парадигма часів дієслова (Опис рутини та планів)

Дієслова у формі дійсного способу виражають дію, що відбувалася, відбувається або відбуватиметься, та мають форми трьох часів [S5].

#### Теперішній час (Present Tense)
**Обмеження:** Дієслова теперішнього часу можуть бути ЛИШЕ недоконаного виду, оскільки вони позначають дію, що відбувається постійно чи в момент мовлення [S5]. Вони змінюються виключно за особами та числами [S5].

| Особа / Число | Однина (Singular) | Множина (Plural) |
|---|---|---|
| 1 особа (I / We) | `робити → робл- → +ю → роблю` [S5] | `робити → роби- → +мо → робимо` <!-- VERIFY --> |
| 2 особа (You) | `робити → роби- → +ш → робиш` [S5] | `робити → роби- → +те → робите` <!-- VERIFY --> |
| 3 особа (He, She, It / They) | `робити → роби- → +ть → робить` [S5] | `робити → робл- → +ять → роблять` <!-- VERIFY --> |

#### Минулий час (Past Tense)
**Обмеження:** Утворюється як від доконаного, так і від недоконаного виду. У минулому часі дієслова змінюються за родами (в однині) та числами, але НІКОЛИ не змінюються за особами [S5].

| Рід / Число | Правило творення (Ланцюжок) | Приклад форми |
|---|---|---|
| Чоловічий (однина) | `{основа інфінітива} → {+суфікс -в} → {кінцева форма}` | `роби- → +в → робив` [S5] |
| Жіночий (однина) | `{основа інфінітива} → {+суфікс -л-} → {+закінчення -а} → {кінцева форма}` | `роби- → +л- → +а → робила` [S5] |
| Середній (однина) | `{основа інфінітива} → {+суфікс -л-} → {+закінчення -о} → {кінцева форма}` | `роби- → +л- → +о → робило` <!-- VERIFY --> |
| Множина (всі роди) | `{основа інфінітива} → {+суфікс -л-} → {+закінчення -и} → {кінцева форма}` | `роби- → +л- → +и → робили` [S5] |

#### Майбутній час (Future Tense)
Для опису планів на майбутнє використовуються три різні форми.
**Обмеження:** Проста форма утворюється ТІЛЬКИ від дієслів доконаного виду. Складна та складена форми утворюються ТІЛЬКИ від дієслів недоконаного виду [S4, S5].

**А. Проста форма (Доконаний вид):**
Ланцюжок: `{інфінітив доконаного виду} → {основа майбутнього часу} → {+особове закінчення} → {кінцева форма}`.
- Приклад (1 ос. одн.): `написати → напиш- → +у → напишу` [S4].
- Приклад (3 ос. мн.): `зробити → зробл- → +ять → зроблять` [S5].

**Б. Складна форма (Недоконаний вид - Синтетична):**
Ланцюжок: `{повний інфінітив недоконаного виду} → {+формотворчий суфікс -м-} → {+особове закінчення} → {кінцева форма}` [S4].
- Приклад (1 ос. одн.): `будувати → будувати- → +м- → +у → будуватиму` [S4].
- Приклад (2 ос. одн.): `казати → казати- → +м- → +еш → казатимеш` [S4].
- Приклад (1 ос. мн.): `сумувати → сумувати- → +м- → +емо → сумуватимемо` [S4].

**В. Складена форма (Недоконаний вид - Аналітична):**
Ланцюжок: `{дієслово 'бути' в особовій формі майбутнього часу} + {незмінний інфінітив недоконаного виду}` [S4].
- Приклад: `буду + шепотіти → буду шепотіти` [S4].
- Приклад: `будемо + робити → будемо робити` [S4].

### 3. Синтаксичне керування у повсякденній рутині (Syntactic Government)

Жорсткі правила поєднання слів для рівня B1, які не збігаються з англійською семантикою [S6].

| Дієслово / Прикметник | Необхідна конструкція | Ланцюжок керування | Приклад |
|---|---|---|---|
| Навчатися (to study) | Родовий відмінок | `навчатися + {іменник у родовому відмінку}` | `навчатися робототехніки` [S6] |
| Опанувати (to master) | Знахідний відмінок | `опанувати + {іменник у знахідному відмінку}` | `опанувати премудрість` [S6] |
| Подякувати (to thank) | Давальний відмінок | `подякувати + {іменник у давальному відмінку}` | `подякувати однокласникові` [S6] |
| Хворіти (to be sick) | `на` + Знахідний відмінок | `хворіти + на + {іменник у знахідному відмінку}` | `хворіти на грип` [S6] |
| Сміятися (to laugh) | `з` + Родовий відмінок | `сміятися + з + {іменник у родовому відмінку}` | `сміятися з мене` [S6] |
| Вищий (taller than) | `за` + Знахідний відмінок | `{прикметник вищого ступеня} + за + {іменник у знах. в.}` | `вищий за батька` [S6] |

### 4. Вираження часу, дат та регулярності

**Обмеження щодо вказівки часу до повної години:** Заборонено використовувати прийменник "без". Необхідно вживати конструкцію з прийменником "за":
Ланцюжок: `за + {кількість хвилин у знахідному відмінку} + {порядковий числівник години в називному відмінку}`.
- Приклад: `за + десять + шоста → за десять шоста` [S1].

**Обмеження щодо регулярності (дні тижня):**
Ланцюжок 1: `по + {назва дня тижня у місцевому відмінку множини}`. Приклад: `по четвергах` [S1].
Ланцюжок 2: `що + {назва дня тижня}`. Приклад: `щочетверга` [S1].

**Обмеження щодо дат:**
Порядковий числівник дати завжди має форму середнього роду, а назва місяця невідмінювано стоїть у родовому відмінку:
Ланцюжок: `{порядковий числівник у середньому роді} + {назва місяця в родовому відмінку}`. Приклад: `третє лютого` [S1].

**Обмеження щодо дробових величин (півтора):**
Ланцюжок: `півтора + {іменник чоловічого або середнього роду в родовому відмінку однини}`. Приклад: `півтора місяця` [S1].

## Частотність і пріоритети

Для опису рутини (daily routines) найчастотнішим є теперішній час недоконаного виду, оскільки він позначає дію, що відбувається постійно [S5]. Саме його форми слід відпрацьовувати першими.

Під час вивчення майбутнього часу англомовним учням легше дається складена аналітична форма (`буду робити`), оскільки вона структурно нагадує англійське "will + verb" [S4]. Однак, в українському живому та художньому мовленні складна синтетична форма (`робитиму`) є надзвичайно частотною і питомою, надаючи мові легкості й динаміки [S4]. Тому автори повинні пріоритезувати складну форму у вправах на аудіювання та читання, щоб учні звикали до її звучання, поступово виводячи її в активний вжиток.

У сфері лексики конструкції типу `хворіти на...` та `навчатися чогось...` [S6] є критично важливими для B1, оскільки вони порушують очікування англомовного студента щодо прямої перехідності.

## Деколонізаційні застереження

При укладанні матеріалів критично важливо уникати лексичних та синтаксичних форм, що є наслідком зросійщення або прямими кальками з російської мови. Російська граматика часто нав'язується через суржик у повсякденному мовленні.

1. **Вказівка часу:** Російська модель «без десяти шесть» глибоко вкорінилася в побутове мовлення, проте вона є абсолютно ненормативною в українській мові. Єдиний правильний літературний варіант — «за десять шоста» (або «за десять хвилин шоста») [S1].
2. **Регулярність (дні тижня):** Конструкція з давальним відмінком у множині («по четвергам», рос. *по четвергам*) є русизмом. Українська мова послуговується місцевим відмінком множини («по четвергах») або префіксальним способом творення («щочетверга») [S1].
3. **Синтаксичне керування хворобами:** Використання безприйменникового орудного відмінка («хворіти грипом», рос. *болеть гриппом*) є грубим порушенням. Українська парадигма вимагає прийменникової конструкції «хворіти на грип» [S6].
4. **Синтаксичне керування емоціями:** Калька «сміятися наді мною» (рос. *смеяться надо мной*) замінюється питомо українським керуванням «сміятися з мене» [S6].
5. **Мова спілкування:** Уникайте виразу «спілкуватися на українській мові» (рос. *общаться на украинском языке*). Правильна українська модель — використання орудного відмінка інструмента дії: «спілкуватися українською мовою» [S6].
6. **Одяг:** Російське розрізнення «надевать/одевать» часто створює хибну аналогію для українських паронімів. Слід пам'ятати, що українською мовою сонцезахисні окуляри, штани та футболки ми саме «одягаємо» [S2].

## Природні приклади

Приклади згруповано за комунікативними моделями для зручного використання в розмовних вправах та діалогах.

**Група 3: Одяг і побутові дії**
- Коли сонце світить прямо в очі, я завжди одягаю сонцезахисні окуляри. [S2]

- Вже півтора місяця майстри ремонтують сходи біля нового будинку. [S1, S6]

**Група 4: Плани на майбутнє**
- Я неодмінно принесу із саду пізні квіти, і дощ до ранку буде шепотіти. [S4]
- Місто само перетворюватиме себе, воно щодня шукатиме іншу досконалість. [S4]
- Завтра я будуватиму новий маршрут для нашої подорожі. [S4]

**Група 5: Навчання та міжособистісні стосунки**
- Наш син прагне навчатися робототехніки вже з наступного семестру. [S6]
- Я обов'язково маю подякувати однокласникові за вчасну допомогу з проєктом. [S6]
- Ми домовилися спілкуватися виключно українською мовою за будь-яких обставин. [S6]

## Рекомендації для вправ

Для ефективного засвоєння граматики щоденного життя на рівні B1 авторам слід дотримуватися чіткої трифазної прогресії.

**Фаза 1: Рецептивне розпізнавання (Сприйняття)**
Учні працюють із короткими текстами про розпорядок дня, де рясно використано дієслова. Завдання полягає у сортуванні: учні повинні виділити дієслова, що означають дію в момент мовлення (теперішній час, лише недоконаний вид) [S5], і ті, що описують плани після моменту мовлення (майбутній час) [S5]. Також корисно давати завдання на ідентифікацію простої, складної (`робитиму`) та складеної (`буду робити`) форм майбутнього часу в автентичних текстах [S4].

**Фаза 2: Контрольована репродукція (Тренування форми)**
Тут ідеально працюють вправи на заповнення пропусків, що фокусуються на синтаксичному керуванні. Учням пропонується дієслово та базове слово, з якого потрібно утворити правильний відмінок із прийменником або без. Наприклад: `хворіти + (грип)` перетворюється на `хворіти на грип` [S6]; `навчатися + (робототехніка)` перетворюється на `навчатися робототехніки` [S6]. Окремим блоком слід відпрацювати конструювання часу: перетворення цифрових показників (5:50) на текстові нормативні форми («за десять шоста») [S1].

**Фаза 3: Вільна продукція (Комунікація)**
Учні отримують завдання описати свій робочий розклад або ідеальні вихідні. Вони повинні використати вивчені форми регулярності («щочетверга», «по четвергах») [S1], правильно застосувати дієслова одягання («одягати штани») [S2] та розповісти про плани, свідомо чергуючи синтетичну («я будуватиму») та аналітичну («я буду шепотіти») форми майбутнього часу [S4].

## Зв'язки з іншими темами

Матеріал цієї статті тісно переплітається з темами «Відмінювання іменників», оскільки правильне синтаксичне керування неможливе без автоматизованого знання родового, давального та знахідного відмінків. Також ця тема є базою для подальшого вивчення «Дієприкметників та дієприслівників» на вищих рівнях (B2-C1), де види дієслів відіграватимуть ключову роль у словотворенні. Розуміння обмежень теперішнього часу (лише недоконаний вид) [S5] запобігає глибоким структурним помилкам у побудові складних речень умови та часу.
