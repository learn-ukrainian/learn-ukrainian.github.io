## Як це пояснюють у школі (How Schools Teach This)

В українській шкільній програмі складнопідрядні речення з підрядними з'ясувальними (complex sentences with object clauses) вводяться як один із трьох основних типів підрядних речень, поряд з означальними (attributive) та обставинними (adverbial) [S38][S39]. Зазвичай ця тема розглядається у 9-му класі [S40][S41][S42].

Основний принцип, що пояснюється учням, полягає в тому, що з'ясувальна частина "з'ясовує", "роз'яснює" або "конкретизує" зміст певного слова в головній частині речення [S40][S43]. Ця підрядна частина функціонує подібно до додатка (object) у простому реченні і відповідає на питання непрямих відмінків: *кого? чого? кому? чому? кого? що? ким? чим? на кому? на чому?* [S44][S41][S42].

Педагогічний підхід зосереджується на "опорних" словах у головній частині, які вимагають такого з'ясування. Підручники наголошують, що цими словами найчастіше є дієслова, що позначають різні вияви **мовлення, мислення та відчуття** [S45][S42].

Наприклад, у підручнику Заболотного [S41][S46] наводиться такий перелік опорних слів:
- **Мовлення:** *говорити, кричати, заявляти, запитати, розповідати*
- **Мислення:** *думати, міркувати, припускати, переконатися, розуміти*
- **Сприйняття:** *бачити, помічати, чути*
- **Волевиявлення/Бажання:** *просити, забороняти, бажати, хотіти*
- **Емоції/Стан:** *радіти, вірити, дивуватися, чекати, боятися*

Окрім дієслів, з'ясувальну частину можуть вимагати й інші частини мови [S45][S42]:
- **Прикметники:** *радий, щасливий, певен, гордий*. Наприклад: *Я щасливий, (чому?) що мене оточують талановиті люди* [S41].
- **Прислівники та безособові форми (предикативи):** *видно, відомо, потрібно, важливо, приємно*. Наприклад: *Дуже важливо, (що?) щоб учням було дороге все створене старшими поколіннями* [S41].

Таким чином, учнів навчають ідентифікувати такі речення, ставлячи відмінкове питання від опорного слова в головній частині до цілої підрядної частини. Це дозволяє відрізнити їх від означальних (питання *який?*) та обставинних (питання *де? коли? чому? як?* тощо) речень [S38].

## Повна парадигма (Full Paradigm)

З'ясувальні речення не мають парадигми відмінювання, як іменники чи дієслова. Натомість їхня структура визначається комбінацією **опорного слова** в головній частині та **засобу зв'язку** (сполучника або сполучного слова), що вводить підрядну частину.

### Опорні слова в головній частині

Ці слова сигналізують про необхідність "з'ясування" або доповнення інформації.

| Категорія | Приклади слів [S45][S41][S42] |
| :--- | :--- |
| **Дієслова мовлення** | *говорити, казати, розповідати, повідомляти, питати, заявляти, кричати, нагадувати* |
| **Дієслова мислення** | *думати, вважати, розуміти, знати, гадати, міркувати, припускати, переконатися* |
| **Дієслова сприйняття** | *бачити, чути, помічати, відчувати, дивитися* |
| **Дієслова волевиявлення** | *хотіти, бажати, просити, вимагати, благати, радити, наказувати* |
| **Дієслова емоційного стану** | *радіти, сумувати, дивуватися, боятися, вірити, сподіватися, чекати* |
| **Прикметники** | *радий, щасливий, певен, впевнений, згоден, гордий* |
| **Предикативи** | *треба, потрібно, важливо, необхідно, видно, чути, відомо, зрозуміло, очевидно, дивно, шкода, соромно* |
| **Іменники** | *думка, питання, звістка, надія, відчуття* (часто з вказівним словом *про те*) |

### Засоби зв'язку (Connectors)

Підрядна з'ясувальна частина приєднується до головної за допомогою сполучників підрядності або сполучних слів [S47][S41]. Важливо розрізняти їх:
- **Сполучники** (`що, щоб, як, чи, ніби`) лише з'єднують частини і не є членами речення [S48][S49].
- **Сполучні слова** (`хто, який, де, коли` та ін.) не тільки з'єднують, але й є повнозначними словами та виступають членами підрядної частини (підметом, додатком, обставиною) [S48][S49].

| Тип засобу | Засіб зв'язку | Призначення |
| :--- | :--- | :--- |
| **Сполучники** | `що` | Передача факту, констатація (найчастіший). |
| | `щоб` (синонім `аби`) | Передача бажання, наказу, прохання, мети. |
| | `як` | Передача процесу, способу дії (часто з дієсловами сприйняття). |
| | `чи` | Передача непрямого питання (так/ні). |
| | `ніби`, `мов`, `наче` | Передача припущення, сумніву, уявної ситуації. |
| **Сполучні слова** | `хто`, `що` | Питання про особу або предмет. |
| (відносні займенники) | `який`, `чий`, `котрий` | Питання про ознаку або належність. |
| **Сполучні слова** | `де`, `куди`, `звідки` | Питання про місце. |
| (прислівники) | `коли` | Питання про час. |
| | `як`, `чому`, `навіщо`, `скільки` | Питання про спосіб, причину, мету, кількість. |

*Примітка:* Слово `що` може бути і сполучником, і сполучним словом (займенником). Якщо `що` можна замінити на `який`, то це сполучне слово [S50]. Однак у з'ясувальних реченнях `що` найчастіше є сполучником.

## Частотність і пріоритети

Для рівня B1 пріоритетним є засвоєння найуживаніших конструкцій, що є основою для передачі непрямої мови.

1.  **Найвища частотність (`що`)**: Конструкції з дієсловами мовлення та мислення + сполучник `що`. Це основа для переказу інформації.
    - *Він сказав, що прийде завтра.*
    - *Я думаю, що це гарна ідея.*

2.  **Висока частотність (`щоб`)**: Конструкції з дієсловами волевиявлення + сполучник `щоб`. Необхідні для вираження бажань, прохань, порад.
    - *Мама хоче, щоб я допоміг.*
    - *Лікар порадив, щоб я більше відпочивав.*

3.  **Середня частотність (`чи`, `як` та питальні слова)**: Конструкції для передачі непрямих запитань та сприйняття.
    - *Запитай, чи працює магазин.*
    - *Я не знаю, коли почнеться фільм.*
    - *Я бачив, як він це робив.*

4.  **Нижча частотність для B1 (`ніби`, `мов`, `наче`)**: Конструкції, що виражають сумнів чи припущення. Їх можна вводити пізніше, після впевненого засвоєння основних.
    - *Мені здалося, ніби хтось покликав.*

## Типові помилки L2

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| *Я хочу тебе **піти** зі мною.* | *Я хочу, **щоб ти пішов** (пішла) зі мною.* | Англійська модель "want + object + infinitive" не працює в українській. Для вираження бажання щодо дії іншої особи використовується складнопідрядне речення зі сполучником **щоб** [S45][S42][S51]. |
| *Він попросив мене **допомогти** йому.* | *Він попросив, **щоб я допоміг** (допомогла) йому.* | Аналогічно до попередньої помилки, прохання, спрямоване на іншу особу, оформлюється через з'ясувальну частину з **щоб**. Інфінітив можливий лише тоді, коли дія стосується самого суб'єкта (*Він попросив допомогти* = *Він попросив, щоб йому допомогли*). |
| *Я запитав, **якщо** він знає відповідь.* | *Я запитав, **чи** він знає відповідь.* | Англійське "if" перекладається як `якщо` в умовних реченнях і як `чи` в непрямих питаннях. Українська мова чітко розрізняє ці два сполучники. `Якщо` вводить умову, `чи` – непряме питання "так/ні" [S44][S51]. |
| *Я не знаю, **що** це таке є.* | *Я не знаю, **що** це таке.* (або *...чим це є*) | В українській мові дієслово "бути" (є) в теперішньому часі у функції зв'язки зазвичай опускається. Його наявність є калькою з англійської ("I don't know what this is"). |
| *Він сказав, **то** він втомився.* | *Він сказав, **що** він втомився.* | Це поширена помилка, що виникає під впливом російської мови, де використовується сполучник `что`. Український відповідник – `що` (вимовляється як [шчо]). |

## Деколонізаційні застереження

1.  **`Що` vs. `Что`**: Найважливіше розрізнення стосується сполучника `що`. В українській мові він вимовляється чітко як **[шчо]**. Російський аналог `что` ([што]) або поширений суржиковий варіант `шо` ([шо]) є грубими помилками в літературній українській мові. Навчаючи, слід наголошувати на правильній вимові та написанні саме українського сполучника.

2.  **Сполучник `аби`**: В українській мові сполучник `аби` є повноцінним синонімом до `щоб` у значенні мети або бажання (*Я прийшов, аби поговорити*). Це характерна риса української мови, яка не завжди має прямі відповідники в російській. Його варто вводити як стилістично забарвлений варіант [S52][S53][S54].

3.  **Неунікальність структури**: Хоча сама структура з'ясувальних речень є спільною для багатьох слов'янських мов, не варто подавати українські правила як "схожі на російські" або "відмінні від російських". Українську синтаксичну систему слід пояснювати як самодостатню, з власним набором сполучників, опорних слів та стилістичних нюансів.

4.  **Приклади**: Використовуйте приклади з української літератури, фольклору та сучасного вжитку, уникаючи кальок або перекладів з російської. Наприклад, речення `Давним-давно ... я почув дивовижну легенду про те, як пісня врятувала козака від смерті` [S48] є автентичним і культурно значущим.

## Природні приклади

### Група 1: Передача інформації (сполучник `що`)
- *І всі зрозуміли, що яскрава зірка над обрієм – ранкова зірка.* (О. Довженко) [S55]
- *Французький математик Лаплас говорив, що винайдення логарифмів продовжило життя обчислювачів.* (З посібника) [S41]
- *Я дуже щасливий, що мене оточують талановиті люди.* (З газети) [S41]

### Група 2: Передача бажання, прохання, мети (сполучник `щоб`/`аби`)
- *Я бажаю, щоб перше горе було останнім у твоїм житті.* (Леся Українка) [S45][S42]
- *Дбайте, щоб в житті їх зберегти.* (Д. Луценко) [S45][S42]
- *Я воскрес, щоб із вами жити під шаленством весняних злив.* (В. Симоненко) [S43]

### Група 3: Передача сприйняття та способу (сполучник `як`)
- *З дороги було видно, як вився у низині кривульками глибокий та каламутний Прут.* [S44]
- *Серед великої тиші чулося тільки, як шелестіли листочки.* (М. Коцюбинський) [S42]
- *У метушні людській не помічаю, як проминають і роки, і дні.* (М. Луків) [S40]

### Група 4: Передача непрямого питання (сполучник `чи` та сполучні слова)
- *Ніхто не знає, що його в житті чекає.* (Нар. творчість) [S39]
- *Маруся питає, чи дружу я з Тимком Степурою.* (За Донченком) [S51]
- *Хлопчик запитав, куди зникає вночі сонце.* [S51]

## Рекомендації для вправ

-   **Phase 1: Ідентифікація та аналіз.**
    -   *Вправа:* Дати учням речення і попросити підкреслити головну і підрядну частини, обвести опорне слово і сполучник/сполучне слово.
    -   *Приклад:* `Я не знаю, / (де він живе).` Опорне слово: *не знаю*. Сполучне слово: *де*.

-   **Phase 2: Трансформація.**
    -   *Вправа "Перетворення прямої мови на непряму":* Це ключова вправа для цієї теми.
        -   *Пряма:* `Мама сказала: "Купи хліба".` -> *Непряма:* `Мама сказала, щоб я купив хліба.`
        -   *Пряма:* `Він запитав: "Ти прийдеш?"` -> *Непряма:* `Він запитав, чи я прийду.`
        -   *Пряма:* `Подруга написала: "Я вчора була в кіно".` -> *Непряма:* `Подруга написала, що вона вчора була в кіно.`
    -   *Вправа "З'єднання речень":* З двох простих речень утворити одне складнопідрядне з'ясувальне.
        -   *Дано:* `Щось трапилося.` `Я не розумію.` -> *Результат:* `Я не розумію, що трапилося.`

-   **Phase 3: Контрольована та вільна продукція.**
    -   *Вправа "Вибір сполучника":* Дати речення з пропущеним засобом зв'язку і варіантами (`що`/`щоб`/`чи`/`де`). `Я хочу, ___ ти був щасливий.`
    -   *Вправа "Завершення речення":* Дати початок головної частини, учні мають додати з'ясувальну.
        -   `Я сподіваюся, що ...`
        -   `Мене запитали, чи ...`
        -   `Було б добре, щоб ...`
    -   *Рольова гра:* Один учень дає іншому прохання, наказ або ставить питання. Другий учень розповідає третьому, що сказав перший, використовуючи непряму мову.

## Зв'язки з іншими темами

-   **Передумови:**
    -   **Синтаксис простого речення:** Учень має розуміти поняття підмета, присудка і особливо **додатка**, оскільки з'ясувальна частина є його синтаксичним аналогом [S38].
    -   **Відмінки:** Питання, на які відповідає з'ясувальна частина, є відмінковими питаннями [S41].

-   **Подальші зв'язки:**
    -   **Пряма та непряма мова:** З'ясувальні речення є граматичною основою для утворення непрямої мови [S51].
    -   **Інші типи складнопідрядних речень:** Ця тема вивчається в порівнянні з **означальними** та **обставинними** підрядними частинами. Важливо навчити учнів розрізняти їх за питаннями та опорними словами [S44][S38].
    -   **Сполучники та сполучні слова:** Конструкції з'ясувальних речень надають чудовий матеріал для практичного розрізнення сполучників (не є членами речення) та сполучних слів (є членами речення) [S48][S49].
