## Як це пояснюють у школі (How Schools Teach This)
Пасивні дієприкметники є важливою частиною вивчення морфології в українській школі, зазвичай тема детально розглядається у 7 класі [S29][S30][S31].

Навчальний підхід зосереджується на кількох ключових аспектах:
1.  **Визначення**: Пасивні дієприкметники визначаються як форма дієслова, що виражає ознаку предмета, який зазнає на собі дію іншого предмета [S32][S33][S34]. Наприклад, у фразі "написаний учнем твір", твір не виконує дію, а зазнає її від учня.
2.  **Час**: Наголошується, що пасивні дієприкметники в українській мові мають лише форму минулого часу [S35][S36].
3.  **Творення**: Основна увага приділяється трьом словотворчим суфіксам: **-н-**, **-ен- (-єн-)**, та **-т-**, які додаються до основи інфінітива [S37]. Підручники детально розглядають, який суфікс використовується залежно від основи дієслова, і приділяють увагу чергуванню приголосних, що виникає при додаванні суфікса **-ен-** (наприклад, `образити -> ображений`) [S35][S36].
4.  **Правопис**: Розглядаються правила правопису, зокрема написання однієї літери `н` у суфіксах (`сказаний`, `зроблений`), на відміну від прикметників дієприкметникового походження, де може бути `нн`, та правопис букви `е` (а не `и`) в суфіксі `-ен-` (`скошений`, не `*скошиний`) [S38]. Також вивчається правопис частки `не` з дієприкметниками [S39].
5.  **Відмінювання**: Пояснюється, що дієприкметники відмінюються за родами, числами та відмінками, як прикметники твердої групи, і узгоджуються з іменником, який вони описують [S40].
6.  **Синтаксична роль**: У реченні дієприкметник найчастіше виступає означенням (`Який?`) або іменною частиною складеного присудка [S41][S42].

Вправи зазвичай включають творення дієприкметників від дієслів, перебудову речень з активного стану в пасивний (`учні написали твір` -> `твір, написаний учнями`), та постановку дієприкметників у правильну граматичну форму [S38].

## Повна парадигма (Full Paradigm)
Пасивні дієприкметники минулого часу утворюються від основи інфінітива перехідних дієслів доконаного і недоконаного виду.

### Творення пасивних дієприкметників

| Суфікс | Умова | Приклади творення | Результат |
| :--- | :--- | :--- | :--- |
| **-н-** | Від дієслів, основа яких закінчується на **-а-**. | `прочитати` ➝ `прочита-` + `-н-` | `прочитаний` |
| | | `посіяти` ➝ `посія-` + `-н-` | `посіяний` |
| | | `написати` ➝ `написа-` + `-н-` | `написаний` |
| **-ен- (-єн-)** | Від дієслів, основа яких закінчується на **приголосний** або на **-и-**, **-і- (-ї-)**. | `принести` ➝ `принес-` + `-ен-` | `принесений` |
| | | `побачити` ➝ `побач-` + `-ен-` | `побачений` |
| | | `вивчити` ➝ `вивч-` + `-ен-` | `вивчений` |
| | | `напоїти` ➝ `напоj-` + `-єн-` | `напоєний` |
| **-т-** | Від дієслів з основою на **-ну-**, **-у-** та односкладових основ. | `зігнути` ➝ `зігну-` + `-т-` | `зігнутий` |
| | | `розбити` ➝ `розби-` + `-т-` | `розбитий` |
| | | `взути` ➝ `взу-` + `-т-` | `взутий` |

[S35][S43][S36]

**Чергування приголосних при творенні з суфіксом `-ен-`:**

| Зміна | Приклад | Дієприкметник |
| :--- | :--- | :--- |
| `д` -> `дж` | `розбудити` | `розбу**дж**ений` |
| `з` -> `ж` | `вразити` | `вра**ж**ений` |
| `с` -> `ш` | `скосити` | `ско**ш**ений` |
| `т` -> `ч` | `сплатити` | `спла**ч**ений` |
| `зд` -> `ждж` | `об'їздити` | `об'ї**ждж**ений` |
| `ст` -> `щ` | `вимостити` | `вимо**щ**ений` |
| `б`, `п`, `в`, `м`, `ф` -> `бл`, `пл`, `вл`, `мл`, `фл` | `замовити` | `замо**вл**ений` |

[S37][S36]

**Варіантні форми:** Деякі дієслова дозволяють утворювати паралельні форми.
*   `замкнути` → `замкн**ен**ий` і `замкн**ут**ий`
*   `одягнути` → `одягн**ен**ий` і `одягн**ут**ий`
*   `колоти` → `кол**ен**ий` і `кол**от**ий`

[S37][S35]

### Відмінювання пасивних дієприкметників (на прикладі `зроблений`)
Дієприкметники відмінюються як прикметники твердої групи [S40].

| Відмінок | Чоловічий рід | Середній рід | Жіночий рід | Множина |
| :--- | :--- | :--- | :--- | :--- |
| **Н.** | зроблен**ий** | зроблен**е** | зроблен**а** | зроблен**і** |
| **Р.** | зроблен**ого** | зроблен**ого** | зроблен**ої** | зроблен**их** |
| **Д.** | зроблен**ому** | зроблен**ому** | зроблен**ій** | зроблен**им** |
| **Зн.** | як Н. або Р. | зроблен**е** | зроблен**у** | як Н. або Р. |
| **Ор.** | зроблен**им** | зроблен**им** | зроблен**ою** | зроблен**ими** |
| **М.** | (на) зроблен**ому** / -**ім** | (на) зроблен**ому** / -**ім** | (на) зроблен**ій** | (на) зроблен**их** |

## Частотність і пріоритети
В сучасній українській мові пасивні дієприкметники є значно більш уживаними, стилістично природними та продуктивними, ніж активні дієприкметники, особливо ті, що мають суфікси `-уч-`/`-юч-`, `-ач-`/`-яч-` [S42][S44].

**Пріоритети для вивчення (B1/B2):**
1.  **Найвища частотність:** Форми на `-ний` та `-ений` є найпоширенішими. Їх слід засвоїти в першу чергу. Вони постійно зустрічаються як в усному мовленні, так і в текстах.
    *   `зроблений`, `написаний`, `сказаний`, `побачений`, `куплений`, `запрошений`.
2.  **Середня частотність:** Форми на `-тий` також поширені, особливо з дієсловами, що позначають дію з очевидним, завершеним результатом.
    *   `розбитий`, `закритий`, `забутий`, `взутий`.
3.  **Конструкції з `-но`, `-то`:** Важливо зрозуміти, що безособові форми на `-но`, `-то` (`зроблено`, `написано`, `закрито`) є дуже поширеним і стилістично кращим способом виразити результат дії, коли виконавець невідомий або неважливий. Часто ці форми замінюють пасивні конструкції з дієприкметниками [S45].
    *   Замість `Будинок був побудований` часто кажуть `Будинок було побудовано`.

Пасивні дієприкметники, на відміну від активних, є невід'ємною і стилістично нейтральною частиною мови, тому їх впевнене володіння є обов'язковим для рівнів В1 і вище.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я був **запрошуючий** на вечірку. | Я був **запрошений** на вечірку. | Англійське "-ing" (present participle) може бути і активним, і пасивним за значенням. В українській мові для пасивного стану використовується пасивний дієприкметник. Активний дієприкметник `запрошуючий` є стилістично невдалим і означає "той, хто запрошує". |
| Лист був **написиний** мною. | Лист був **написаний** мною. | Неправильний вибір суфікса. Основа дієслова `написати` закінчується на `-а-`, тому використовується суфікс `-н-`. Також, помилка правопису `и` замість `а`. |
| Я прочитав **образиту** статтю. | Я прочитав **образливу** статтю. / Стаття мене **образила**. | Помилка утворення. Від `образити` утворюється `ображений` (`я ображений` - I am offended). Для ознаки "який ображає" (offensive), використовується прикметник `образливий`. Не всі англійські "-ed" прикметники є пасивними дієприкметниками. |
| Книжка була **прочитана мною**. | **Я прочитав** книжку. / Книжку **прочитано**. | Хоча `Книжка була прочитана мною` граматично можлива, в українській мові активні конструкції (`Я прочитав...`) є набагато природнішими та більш уживаними, коли виконавець відомий і важливий. Якщо виконавець неважливий, перевага надається формі на `-но` (`Книжку прочитано`) [S45]. |
| `Турнір виграно шахістом.` | `Шахіст виграв турнір.` | Конструкції на `-но`, `-то` є безособовими і не вживаються з іменником в орудному відмінку, що позначає виконавця дії [S45]. |
| Ця задача **нерозв'язана мною**. | Ця задача **не розв'язана** мною. | Якщо дієприкметник має залежне слово (тут `мною`), частка `не` пишеться окремо. Разом (`нерозв'язана задача`) пишеться, якщо залежних слів немає і дієприкметник виступає означенням [S39]. |

## Деколонізаційні застереження (Decolonization Notes)
**MANDATORY — never omit.** Правильне вживання дієприкметників є одним із чітких маркерів, що відрізняє грамотну українську мову від суржику, який часто виникає під впливом російської.

1.  **Категорична заборона активних дієприкметників минулого часу на `-вший`, `-ший`**: В російській мові форми типу `сделавший`, `написавший` є нормою. В українській мові їх аналоги (`*зробивший`, `*написавший`) є **грубим порушенням мовної норми** і сприймаються як яскравий приклад русифікації мови [S44]. Замість них слід вживати описові конструкції з `що`:
    *   рос. `человек, прочитавший книгу` -> укр. `людина, **що прочитала** книгу`.
2.  **Обмеженість активних дієприкметників теперішнього часу**: На відміну від російської, де форми на `-ущий`/`-ющий`, `-ащий`/`-ящий` (напр. `идущий`, `говорящий`) є дуже поширеними, в українській мові відповідні форми на `-уч(ий)`/`-юч(ий)`, `-ач(ий)`/`-яч(ий)` мають обмежене вживання і часто сприймаються як архаїчні, поетичні або переходять у прикметники [S42][S44]. Українська мова віддає перевагу конструкціям з `що`, іменникам або іншим синонімам:
    *   рос. `наступающий праздник` -> укр. `прийдешнє свято` (не `*наступаюче свято`).
    *   рос. `заведующий отделом` -> укр. `завідувач відділу` (не `*завідуючий відділом`).
3.  **Природність пасивних дієприкметників**: Пасивні дієприкметники на `-ний`, `-ений`, `-тий` є абсолютно природною, питомою частиною української граматики. Їх не слід сприймати як "складнішу" альтернативу активним конструкціям, а як повноцінний і часто стилістично кращий інструмент, особливо в офіційному та науковому стилях.

Ніколи не можна пояснювати українські граматичні явища через порівняння з російською як з "нормою". Українська граматична система є самодостатньою.

## Природні приклади (Natural Examples)
**Minimum 12 natural Ukrainian sentences** using this grammar point.

**Група 1: Форми на -н- (від дієслів на -ати)**
1.  Розстелений кожушок, на нім посіяний горошок. (Народна творчість [S46])
2.  Як радію за ту я людину, що загублену радість знайшла. (В. Сосюра [S35])
3.  В душі моїй печаль стоїть неждана, в проміннях пам’яті обличчя дорогі. (М. Вінграновський [S42])
4.  Скручена, зв’язана, на кілок посаджена, по подвір’ю гуляє. (Народна творчість [S46])

**Група 2: Форми на -ен- (від дієслів на приголосний, -ити)**
5.  До кипариса магнолія пишная чолом заквітчаним ніжно схилилася. (Леся Українка [S35])
6.  Спеціально створені мобільні застосунки дають змогу знаходити екскурсійні тури. [S38]
7.  Рівно й спокійно дише натомлена з праці земля. (М. Рильський [S35])
8.  Сумно в гаю заспокоєнім, вічною ніччю придавленім. (Леся Українка [S40])

**Група 3: Форми на -т-**
9.  О слово рідне, орле скутий, чужинцям кинуте на сміх! (Олександр Олесь [S47])
10. Дітьми безпам'ятно забутий... (Олександр Олесь [S47])
11. І от ростуть за нашої доби в добу колишню саджені дуби. (М. Рильський [S40])
12. Лиш згорточок старого полотна і вишите моє життя на ньому. (Дмитро Павличко [S48])

## Рекомендації для вправ (Activity Concepts)

- **Phase 1: Ідентифікація та аналіз.**
    - *Вправа 1:* Дати текст (напр., уривок з поезії Олеся [S47] або опис замку [S38]) і попросити знайти всі пасивні дієприкметники.
    - *Вправа 2:* Дати пари слів (`варений`/`варячий`, `смажений`/`смажачий`) і попросити пояснити різницю, визначивши, де дієприкметник, а де прикметник або ненормативна форма.
- **Phase 2: Творення форм.**
    - *Вправа 1 (Творення за суфіксами):* Дати список інфінітивів (`побудувати`, `розбити`, `звільнити`, `забути`, `скосити`) і попросити утворити пасивні дієприкметники, пояснюючи вибір суфікса та чергування приголосних. (На основі вправ з `7-klas-ukrmova-zabolotnyi-2024_s0102`, `7-klas-ukrmova-litvinova-2024_s0096`).
    - *Вправа 2 (Вибір форми):* Заповнити пропуски: `Двері були (зачинити) ______`, `Я отримав лист, (написати) ______ другом`.
- **Phase 3: Інтеграція в речення.**
    - *Вправа 1 (Трансформація):* Перетворити активну конструкцію на пасивну. Зразок: `Мати вишила сорочку. -> Сорочка, вишита матір'ю, ...`. (На основі вправи з `7-klas-ukrmova-zabolotnyi-2024_s0102`).
    - *Вправа 2 (Трансформація в `-но`/`-то`):* Перетворити речення з дієприкметником на речення з безособовою формою. Зразок: `Куплений скейтборд лежить у кутку. -> Скейтборд куплено.` (На основі вправи з `7-klas-ukrmova-avramenko-2024_s0128`).
    - *Вправа 3 (Узгодження):* Поставити дієприкметник у дужках у правильну форму (рід, число, відмінок). Зразок: `Я не бачив (змальований) картини.` -> `змальованої`. (На основі вправи з `7-klas-ukrmova-avramenko-2024_s0116`).

## Зв'язки з іншими темами
- **Дієслово (Prerequisite):** Необхідно розуміти поняття основи дієслова, виду (доконаний/недоконаний) та перехідності. Пасивні дієприкметники утворюються переважно від перехідних дієслів.
- **Прикметник (Prerequisite):** Пасивні дієприкметники морфологічно та синтаксично поводяться як прикметники (відмінювання, узгодження з іменником, роль означення).
- **Дієприкметниковий зворот (Follow-up):** Знання дієприкметників є основою для вивчення дієприкметникових зворотів та правил пунктуації при них.
- **Безособові форми на -но, -то (Follow-up):** Ці форми утворюються безпосередньо від пасивних дієприкметників (`зроблений` -> `зроблено`) і є важливою синтаксичною альтернативою [S45].
- **Активні дієприкметники (Parallel Topic):** Вивчення пасивних дієприкметників відбувається паралельно з активними, що дозволяє зрозуміти їхні відмінності у значенні, творенні та частоті вживання.
