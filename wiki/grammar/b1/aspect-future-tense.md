## Як це пояснюють у школі

В українській шкільній програмі концепція майбутнього часу вводиться поступово, починаючи з початкових класів, але глибоке розуміння видо-часових кореляцій формується в середній школі. У 4 класі діти вперше знайомляться з трьома формами майбутнього часу (простою, складною та складеною) і вчаться змінювати їх за особами й числами [S2]. 

У 6 та 7 класах підручники (наприклад, Заболотного) вже чітко розмежовують форми майбутнього часу залежно від виду дієслова: доконаного чи недоконаного [S1, S3]. Підручники підкреслюють, що дієслова майбутнього часу означають дію, яка відбуватиметься або відбудеться після моменту мовлення [S1, S3]. Важливим методичним акцентом є те, що школярам пояснюють алгоритми визначення дієвідмін та пов'язані з ними орфографічні правила (написання *е/є* або *и/ї* в особових закінченнях) [S2, S9].

Автору-письменнику варто звернути увагу на те, що шкільна граматика не розглядає складну (синтетичну) та складену (аналітичну) форми як семантично різні — вони подаються як абсолютні синоніми для дієслів недоконаного виду [S1]. Водночас підручники чітко артикулюють **жорстке обмеження**: дієслова доконаного виду мають лише одну, просту форму майбутнього часу [S1, S3].

## Повна парадигма

Майбутній час в українській мові реалізується у трьох формах, вибір яких строго детермінований граматичним видом дієслова.

### Обмеження видо-часової системи
Перед побудовою парадигм важливо зафіксувати ключові структурні обмеження, встановлені граматичною нормою [S1, S3, S4]:
1. **Проста форма** майбутнього часу утворюється **ВИКЛЮЧНО** від дієслів доконаного виду.
2. **Складна форма** утворюється **ВИКЛЮЧНО** від дієслів недоконаного виду.
3. **Складена форма** утворюється **ВИКЛЮЧНО** від дієслів недоконаного виду.
4. **Дієприслівники** не мають форм майбутнього часу взагалі — парадигма обмежується лише теперішнім (для недоконаного виду) та минулим часом [S4].

### 1. Проста форма (Доконаний вид)

Утворюється шляхом додавання особових закінчень теперішнього часу до основи дієслова доконаного виду [S2, S3]. Набір закінчень визначається належністю дієслова до І або ІІ дієвідміни [S9].

**Ланцюжок перетворення:**
`{інфінітив доконаного виду} → {основа} → {+ особове закінчення І/ІІ дієвідміни} → {кінцева форма}`
*Приклад (І дієвідміна):* `написати → напиш- → +у → напишу` [S1]
*Приклад (ІІ дієвідміна):* `купити → куп- → +лю → куплю` [S3]

| Особа / Число | І дієвідміна (напр., *написати*, *зробити*) | ІІ дієвідміна (напр., *закінчити*) |
| :--- | :--- | :--- |
| 1-ша однини (я) | напиш**у**, зроб**лю** | закінч**у** |
| 2-га однини (ти) | напиш**еш**, зроб**иш** | закінч**иш** |
| 3-тя однини (він/вона/воно) | напиш**е**, зроб**ить** | закінч**ить** |
| 1-ша множини (ми) | напиш**емо**, зроб**имо** | закінч**имо** |
| 2-га множини (ви) | напиш**ете**, зроб**ите** | закінч**ите** |
| 3-тя множини (вони) | напиш**уть**, зроб**лять** | закінч**ать** |

**Фонетичні зміни (Чергування приголосних):**
Під час утворення форм 1-ї особи однини (а іноді й усієї парадигми) кінцеві приголосні основи часто зазнають історичних чергувань [S3]:
- `[г] → [ж]`: допомогти → допомо**ж**у
- `[к] → [ч]`: спекти → спе**ч**у
- `[х] → [ш]`: заколихати → заколи**ш**у
- `[з] → [ж]`: розрізати → розрі**ж**у
- `[с] → [ш]`: заголосити → заголо**ш**у
- `[д] → [дж]`: сходити → схо**дж**у
- `[т] → [ч]`: заплатити → запла**ч**у
- `[ст] → [шч]`: помастити → пома**щ**у
- `[зд] → [ждж]`: поїздити → пої**ждж**у
- **Губні** (`б, п, в, м, ф`) отримують епентетичне `[л]`: зробити → зроб**лю**, зловити → злов**лю** [S3].

### 2. Складна форма (Недоконаний вид)

Це унікальна синтетична форма української мови, що історично утворилася від злиття інфінітива та дієслова «яти» (мати) [S1, S2].

**Ланцюжок перетворення:**
`{основа інфінітива недоконаного виду} → {+ суфікс -ти-} → {+ формотворчий суфікс -м-} → {+ особове закінчення} → {кінцева форма}`
*Приклад:* `чита- → +ти → +м → +у → читатиму` [S1]

| Особа / Число | Парадигма (напр., *писати*, *ходити*) |
| :--- | :--- |
| 1-ша однини (я) | писати**му**, ходити**му** |
| 2-га однини (ти) | писати**меш**, ходити**меш** |
| 3-тя однини (він/вона/воно) | писати**ме**, ходити**ме** |
| 1-ша множини (ми) | писати**мемо**, ходити**мемо** |
| 2-га множини (ви) | писати**мете**, ходити**мете** |
| 3-тя множини (вони) | писати**муть**, ходити**муть** |

*Примітка щодо зворотних дієслів:* Постфікс `-ся` / `-сь` завжди приєднується в самому кінці, після особового закінчення: `сміятимусь`, `навчатимешся` [S2].

### 3. Складена форма (Недоконаний вид)

Аналітична форма, що складається з допоміжного дієслова «бути» в особовій формі та інфінітива основного дієслова [S1, S3].

**Ланцюжок перетворення:**
`{особова форма 'бути' у майбутньому часі} + {інфінітив недоконаного виду} → {кінцева форма}`
*Приклад:* `буду + слухати → буду слухати` [S1]

| Особа / Число | Парадигма (напр., *робити*) |
| :--- | :--- |
| 1-ша однини (я) | **буду** робити |
| 2-га однини (ти) | **будеш** робити |
| 3-тя однини (він/вона/воно) | **буде** робити |
| 1-ша множини (ми) | **будемо** робити |
| 2-га множини (ви) | **будете** робити |
| 3-тя множини (вони) | **будуть** робити |

### 4. Нерегулярні дієслова (Архаїчний тип)

Дієслова *дати*, *їсти* та похідні з коренем *-вісти* (*розповісти*, *відповісти*) не належать до стандартних дієвідмін і мають специфічну, архаїчну систему закінчень у простій формі майбутнього часу (для доконаного виду) або в теперішньому (для недоконаного) [S9].

| Особа / Число | дати (док. вид) | відповісти (док. вид) |
| :--- | :--- | :--- |
| 1-ша однини (я) | **дам** | **відповім** |
| 2-га однини (ти) | **даси** | **відповіси** |
| 3-тя однини (він/вона/воно) | **дасть** | **відповість** |
| 1-ша множини (ми) | **дамо** | **відповімо** |
| 2-га множини (ви) | **дасте** | **відповісте** |
| 3-тя множини (вони) | **дадуть** | **відповідять** |

## Частотність і пріоритети

В реальному спілкуванні (і на рівні B1 зокрема) учень стикатиметься переважно з **простою формою майбутнього часу доконаного виду**. Це пов'язано з тим, що розмови про плани найчастіше стосуються намірів досягти результату (*я зроблю*, *ми поїдемо*, *вона напише*).

Щодо конкуренції форм недоконаного виду: складна форма (*читатиму*) та складена (*буду читати*) є абсолютними граматичними синонімами [S1]. Проте в сучасній українській мові синтетична форма (*-тиму*) відіграє вагому стилістичну роль: вона робить мовлення більш динамічним, компактним і виразно питомо українським. Складена форма часто переважає в розмовному мовленні через легкість утворення, але для академічного стандарту та письма складна форма є маркером високої мовної компетентності.

Пріоритет для введення (B1):
1. Механіка простої форми (фокус на чергуваннях, бо тут L2-учні роблять найбільше помилок).
2. Синтетична складна форма (*працюватиму*).
3. Аналітична складена форма (*буду працювати*).

## Типові помилки L2 (англомовні учні)

Англійська мова використовує єдину аналітичну конструкцію (will + verb) для передачі дій у майбутньому. Через відсутність граматичної категорії виду англомовні учні системно переносять цю модель на українську.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я **буду прочитати** цю статтю. | Я **прочитаю** цю статтю. | Складена форма з допоміжним дієсловом «бути» утворюється **лише** з інфінітивами недоконаного виду [S1, S3]. Доконаний вид завжди використовує просту (синтетичну) форму з особовими закінченнями. |
| Він **написатиме** листа завтра. | Він **напише** листа завтра. | Суфікс *-м-* (ознака складної форми) застосовується **винятково** до дієслів недоконаного виду [S1, S3]. Учень механічно приєднав маркер майбутнього часу до інфінітива доконаного виду. |
| Я **купу** квитки на потяг. | Я **куплю** квитки на потяг. | Учні ігнорують закон чергування приголосних [S3]. В українській мові після губних приголосних (`б, п, в, м, ф`) у першій особі однини простої форми майбутнього часу обов'язково з'являється епентетичне `[л]`. |
| Вони **будуть читають** текст. | Вони **будуть читати** текст. (або *читатимуть*) | Подвійне маркування особи. Англомовний учень відмінює і допоміжне дієслово, і основне, не розуміючи, що лексичне значення в складеній формі має виражатися інфінітивом [S1, S3]. |
| Я **сміятисяму** з цього жарту. | Я **сміятимуся** з цього жарту. | Порушення порядку морфем. Постфікс зворотності `-ся` / `-сь` в українській мові завжди замикає слово і ставиться строго після особового закінчення, а не перед формотворчим суфіксом *-м-* [S2]. |
| Ми **дамомо** вам відповідь. | Ми **дамо** вам відповідь. | Хибна аналогія. Учні намагаються застосувати стандартне закінчення 1-ї особи множини *-емо* / *-имо* до нерегулярного архаїчного дієслова «дати», ігноруючи його унікальну парадигму (дам, даси, дасть, дамо) [S9]. |

## Деколонізаційні застереження

Методика викладання української мови десятиліттями потерпала від намагань наблизити її до російської граматики. Для автора-письменника критично важливо усвідомлювати ці відмінності, щоб формувати в англомовних учнів автентичну українську мовну картину світу.

1. **Синтетична форма майбутнього часу.** Російська мова має лише дві форми майбутнього часу: просту («сделаю») та складену («буду делать»). Українська мова володіє унікальною складною формою для недоконаного виду («робитиму») [S1, S3]. В умовах зросійщення цю форму часто таврували як «діалектну» чи «поетичну», зводячи вжиток до калькованого «буду робити». Наш курс має активно інтегрувати форму на *-тиму*, презентуючи її як норму літературної та розмовної мови, рівноцінну аналітичній конструкції.
2. **Збереження літери «л» у парадигмі.** Під впливом російського «сыпешь», російськомовні українці часто кажуть «сипеш». Нормативна українська граматика вимагає збереження звука `[л]` в усіх особових формах: *сиплю, сиплемо, сиплеш, сиплете, сипле, сиплють* [S9]. На це треба звертати увагу при введенні відповідної лексики.
3. **Відсутність дієприкметників майбутнього часу.** Як і в російській, в українській мові не існує дієприкметників майбутнього часу. Проте, російський синтаксис стимулює вживання активних дієприкметників загалом. В українській мові слід уникати конструкцій на зразок «працюючий двигун» чи «існуючий порядок», замінюючи їх на описові конструкції (відрядні речення) або прикметники.

## Природні приклади

Наступні приклади демонструють природний, неперекладений синтаксис майбутнього часу в різних граматичних конфігураціях.

**Група 1: Проста форма (дія, спрямована на результат, доконаний вид)**
* «Наступного року режисер **зніме** байопік.» [S1]
* «Влітку **прочитаю** запропоновані вчителем книги.» [S1]
* «Сміливий там **знайде**, де боягуз **загубить**.» [S2]
* «З добрим **поживеш** — добре й **переймеш**.» [S2]

**Група 2: Складна форма (процес або повторюваність, недоконаний вид)**
* «Я **плаватиму** довго.» [S1]
* «Ми **займатимемось** волонтерством на канікулах.» [S1]
* «**Їхатиму** до тебе через два дні.» [S1]
* «І наші нащадки з майбутнього **читатимуть** про сучасні події, як і ми читаємо про Київську Русь...» [S3]

**Група 3: Складена форма (процес, аналітична структура)**
* «Я завтра **буду надсилати**.» [S1]
* «Ми **будемо відпочивати** в Трускавці.» [S5]
* «Петро **буде вивчати** історію.» [S1]
* «Український гелікоптер "Скаут" можна **буде використовувати** і як таксі.» [S6]

**Група 4: Нерегулярні дієслова та чергування в питаннях**
* «Коли ти **приготуєш** нам омлет?» [S1]
* «Тоді ти **поговори** з батьками й **попередь** мене. Ми **зможемо** вибрати інший день.» [S6]
* «Тепер, якщо хтось **вимкне** рубильник на щитку... машина **повідомить** йому про це...» [S3] (адаптовано контекст для ілюстрації)

## Рекомендації для вправ

Для ефективного засвоєння теми на рівні B1 рекомендується розгорнути тренування за трифазною моделлю.

**Фаза 1: Рецептивна дискримінація (Впізнавання виду)**
Мета: навчити учнів ідентифікувати, чому вжито саме таку форму. 
* *Формат:* Учневі дається мікротекст або набір речень. Завдання — підкреслити всі дієслова в майбутньому часі та вказати їхній вид. 
* *Когнітивний фокус:* Зв'язати наявність слова «бути» або суфікса «-м-» з процесом (недоконаний вид), а просту форму з префіксом — із завершеним результатом.

**Фаза 2: Керована генерація (Трансформація парадигм)**
Мета: автоматизувати механіку творення.
* *Формат:* Трансформаційні вправи. Дано речення в теперішньому часі («Олена працює дизайнеркою»), потрібно перевести його в майбутній («Олена працюватиме / буде працювати дизайнеркою») [S1].
* *Окремий фокус:* Вправи на утворення 1-ї особи однини для дієслів із чергуванням приголосних (напр., *писати* → *напишу*, *ходити* → *сходжу*, *купити* → *куплю*). Студент має самостійно вивести форму, спираючись на інфінітив.

**Фаза 3: Комунікативна практика (Прогнозування та планування)**
Мета: вільне використання обох видів у розповіді про майбутнє.
* *Формат:* Рольові ігри або відкриті запитання. Наприклад: «Розкажіть про свої плани на тиждень, використовуючи дієслова майбутнього часу» [S3].
* *Обмеження для інструкції:* Завдання має вимагати використання як процесів («Я буду летіти три години»), так і результатів («Я прилечу о п'ятій»).

## Зв'язки з іншими темами

**Що має бути засвоєно раніше:**
- Категорія виду дієслова (доконаний / недоконаний). Без цього розуміння пояснити вибір між *прочитаю* та *читатиму* неможливо.
- Теперішній час дієслів (І та ІІ дієвідміни), оскільки проста форма майбутнього часу повністю копіює цю систему морфологічних закінчень і чергувань.
- Відмінювання дієслова «бути» в майбутньому часі (*буду, будеш*).

**Що уможливлює ця тема:**
- Опанування умовного способу (де гіпотетичне майбутнє виражається специфічними конструкціями).
- Побудова складних синтаксичних конструкцій із підрядними часу (напр., «Коли я *приїду*, ми *підемо* в кіно»).
- Розуміння дієслівних форм у художніх та публіцистичних текстах, де часто чергуються теперішній історичний та майбутній часи.
