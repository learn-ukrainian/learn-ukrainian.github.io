## Як це пояснюють у школі

В українській шкільній програмі ґрунтовне вивчення прислівника (adverb) припадає на 7 клас [S1]. Учні вже знайомі з прикметником, тому творення прислівників та їхні ступені порівняння (degrees of comparison) подаються як похідна, паралельна тема. Автори підручників, зокрема О. Заболотний, наголошують на тому, що способи творення та форми прислівникових ступенів порівняння тотожні прикметниковим [S1]. 

У старших класах (10–11 класи) фокус зміщується на стилістику та культуру мовлення. О. Глазова та О. Авраменко акцентують на викоріненні типових суржикових конструкцій, розрізненні простої і складеної форм у різних стилях (проста форма тяжіє до художнього і розмовного, складена — до наукового й ділового) [S5], а також на запобіганні морфологічним помилкам на кшталт змішування форм (утворення типу *більш тепліше*) [S1, S5].

Методика викладання будується на тісному зв'язку морфології та орфографії: оскільки прислівники утворюються від інших частин мови шляхом зрощення, їхній правопис (разом, окремо, з дефісом) прямо залежить від способу словотворення [S6].

## Творення прислівників (Морфологічна парадигма)

Прислівники утворюються традиційними для української мови способами: суфіксальним, префіксальним, префіксально-суфіксальним, складанням основ та переходом з однієї частини мови в іншу [S2, S4].

### Ланцюжки творення

| Спосіб творення (Formation type) | Ланцюжок перетворення | Приклад |
| :--- | :--- | :--- |
| **Суфіксальний** (від прикметників) | `{основа прикметника} → {+о / +е} → {прислівник}` | `радісн- → +о → радісно` [S2] <br> `гаряч- → +е → гаряче` [S5] |
| **Префіксальний** (від інших прислівників) | `{префікс} + {вихідний прислівник} → {новий прислівник}` | `до- + тепер → дотепер` [S2] <br> `після- + завтра → післязавтра` [S2] |
| **Префіксально-суфіксальний** | `{префікс} + {основа} + {суфікс} → {прислівник}` | `в- + перш- + -е → вперше` [S2] <br> `по- + українськ- + -ому → по-українському` [S6] |
| **Складання основ** (Compounding) | `{основа 1} + [сполучний голосний] + {основа 2} → {прислівник}` | `бос(і) + ніж(ки) → босоніж` [S2] <br> `в + один + час → водночас` [S2] |
| **Перехід з іншої частини мови** | `{іменник в непрямому відмінку} → {завмирання форми} → {прислівник}` | `{іменник Оруд. в.: весною} → весною` [S2]<br> `{іменник Оруд. в.: миттю} → миттю` [S2] |

### Обмеження правопису, зумовлені творенням

Творення прислівників супроводжується жорсткими орфографічними правилами [S6]. 

**Пишемо з дефісом:**
*   Утворені від прикметників/займенників за схемою: `{по-} + {основа} + {-ому / -єму / -и / -е}`: *по-українському, по-твоєму, по-друге* [S6].
*   Утворені з частками `{казна- / хтозна- / бозна- / будь-} + {прислівник}` або `{прислівник} + {-небудь / -то / -от / -таки}`: *казна-де, звідки-небудь* [S6].
*   Складання синонімів/антонімів/повторів: *більш-менш, ледве-ледве, тишком-нишком* [S6].

**Пишемо разом:**
*   Злиття прийменника з прикметником, займенником, числівником: `{у} + {день} → удень`; `{на} + {двоє} → надвоє` [S6].
*   З частками `{аби- / ані- / де- / чи- / що- / як-} + {прислівник}`: *абиде, щотижня, якнайшвидше* [S6].

**Пишемо окремо:**
*   Злиття прийменника з повноцінним іменником у непрямому відмінку, який зберіг своє лексичне значення: *рік у рік, сам на сам, день у день, кінець кінцем* [S6].
*   `{по} + {збірний числівник}`: *по двоє, по троє* [S6].

## Ступені порівняння прислівників (Параметри і парадигма)

### Жорсткі обмеження творення ступенів порівняння
1.  Ступені порівняння можуть утворювати **виключно** прислівники, що мають кінцевий суфікс `-о` або `-е` і походять від **якісних прикметників** (qualitative adjectives) [S1, S5]. 
2.  **Заборонено** утворювати ступені порівняння від прислівників, що походять від відносних чи присвійних прикметників (наприклад, неможливо утворити ступені від *вчора, по-батьківськи*) [S8].
3.  **Заборонено** утворювати ступені порівняння від прислівників з абсолютною ознакою (наприклад, *сліпо, мертво, порожньо*) [S8].
4.  **Заборонено** утворювати ступені від прислівників, що мають префікси *за-, над-, пре-* (*зависоко, пречудово*) або суфікси *-еньк-, -есеньк-* (*тихесенько*) [S8].

### Повна парадигма форм

| Ступінь (Degree) | Форма (Form) | Правило творення (Transformation chain) | Приклади |
| :--- | :--- | :--- | :--- |
| **Вищий** (Comparative) | **Проста** | `{корінь прислівника} + {-іше / -ше} → {вищий ступінь}` | `важлив- + -іше → важливіше` [S1] <br> `цікав- + -іше → цікавіше` [S1] |
| | **Складена** | `більш/менш + {вихідний прислівник} → {вищий ступінь}` | `більш важливо` [S1]<br> `менш цікаво` [S1] |
| **Найвищий** (Superlative) | **Проста** | `{най-} + {проста форма вищого ступеня} → {найвищий ступінь}` | `най- + важливіше → найважливіше` [S1]<br> `най- + глибше → найглибше` [S5] |
| | **Складена** | `найбільш/найменш + {вихідний прислівник} → {найвищий ступінь}` | `найбільш важливо` [S1]<br> `найменш глибоко` [S5] |

### Морфонологічні зміни при творенні простої форми вищого ступеня

Творення за допомогою суфікса `-ше` часто спричиняє чергування приголосних та усічення основи [S1, S5]. Ці явища є суворо регулярними і вимагають автоматизації в учнів L2:

**1. Випадання суфіксів `-к-`, `-ок-`, `-ик-`:**
Суфікси твірної основи відпадають перед додаванням `-ше` [S5].
*   `{швидко} → {швид-} + {-ше} → швидше` [S5]
*   `{рідко} → {рід-} + {-ше} → рідше` [S5]
*   `{глибоко} → {глиб-} + {-ше} → глибше` [S5]

**2. Чергування приголосних [г, ж, з] + [ш] → [жч]:**
*   `{дорого} → {дорог-} + {-ш-} → дорожче` [S1, S5]
*   `{дуже} → {дуж-} + {-ш-} → дужче` [S1]
*   `{важко} → {важ-} + {-ш-} → важче` [S5]
*   `{вузько} → {вузьк-} (випадає к) → {вузь-} + {-ш-} → вужче` [S1, S5]

**3. Чергування приголосного [с] + [ш] → [шч] (на письмі — щ):**
*   `{високо} → {висок-} (випадає ок) → {вис-} + {-ш-} → вище` [S1, S5]

### Суплетивні форми (Irregular / Suppletive Forms)
Деякі найуживаніші прислівники утворюють ступені порівняння від іншого кореня [S1, S5]. Їх необхідно вивчати лексично:
*   `добре → краще → найкраще` [S1, S5]
*   `погано → гірше → найгірше` [S1]
*   `багато → більше → найбільше` [S5]
*   `мало → менше → найменше` [S5]

### Еліптичне та підсилювальне вживання
Для підсилення найвищого ступеня (superlative enhancement) до простої форми додаються префікси **`як-`** або **`що-`**: 
*   `якнайкраще, щонайцікавіше, щонайсильніше` [S1, S5].
Для підсилення вищого ступеня (comparative enhancement) вживаються слова **`ще, трохи, значно, куди, набагато`**:
*   `ще сильніше, куди швидше, набагато краще` [S5].

## Частотність і пріоритети

Авторам модулів варто враховувати стилістичну диференціацію. У текстах художнього та розмовного стилів (що складають левову частку введення L2) українська мова надає абсолютну перевагу **простій формі** ступенів порівняння прислівників (`тепліше, краще, найвище`) [S5]. 

**Складена форма** (`більш тепло, найбільш високо`) є менш частотною в живій комунікації і зарезервована переважно для офіційно-ділового та наукового стилів [S5]. Починати навчання слід із простих форм, зокрема суплетивних (`краще, гірше, більше`), оскільки вони є надвисокочастотними у побутовому спілкуванні.

## Типові помилки L2 (англомовні учні)

Англомовні учні часто проектують структуру своїх ступенів порівняння (more/most + adverb) на українську мову, ігноруючи морфологічні обмеження.

| ❌ Помилково (L2 Error) | ✅ Правильно (Target Form) | Чому (Why it is wrong) |
| :--- | :--- | :--- |
| *Він зробив це найбільш краще.* | **Він зробив це найкраще.** (або найбільш добре) | Змішування простої і складеної форм. Не можна додавати `найбільш` до вищого ступеня порівняння. Це грубе порушення морфологічної норми [S1, S5]. |
| *Сьогодні більш тепліше.* | **Сьогодні тепліше.** (або більш тепло) | Плеоназм, подвійне вираження вищого ступеня порівняння. Неправильне поєднання складеної форми (більш) з простою формою (тепліше) [S1, S5]. |
| *Це самий швидкий спосіб.* | **Це найшвидший спосіб.** (або якнайшвидший) | Калькування російської моделі «самый + прикметник/прислівник». В українській мові найвищий ступінь утворюється виключно префіксом `най-` [S8]. |
| *Я почуваюся краще всього.* | **Я почуваюся найкраще.** | Структурний русизм, калька з «лучше всего». В українській мові використовується чиста форма найвищого ступеня [S5, S8]. |
| *Він побіг швидкіше.* | **Він побіг швидше.** | Учень ігнорує правило випадання суфікса `-к-` (швид-к-о) і механічно прикріплює типовий суфікс `-іше` замість `-ше` [S3, S5]. |

## Деколонізаційні застереження

Прислівники та їхні ступені порівняння — одна з тем, де російський синтаксичний та морфологічний вплив найбільш відчутний в мовленні білінгвів. Розробляючи матеріали, суворо уникайте таких патернів:

1. **Табу на слово «самий» у порівняннях.** Конструкції `самий цікавий`, `саме весело`, `самий швидкий` є поширеними суржикізмами. Єдиний літературний інструмент — префікс `най-`: `найцікавіший, найвеселіше, найшвидше` [S8].
2. **Усунення російських аналітичних кальок.** Форми `частіше всього` (від рос. *чаще всего*), `легше всього`, `швидше за все` (в ролі вставного слова), `вірогідніше всього` необхідно повністю вилучити. Українська норма вимагає синтетичних форм: **найчастіше, найлегше, найшвидше, найімовірніше** [S5]. 
3. **Безособові конструкції стану.** Англомовне *«I feel bad»* чи *«I am cold»* не варто буквально перекладати як *«Я почуваюся погано»* чи *«Я є холодним»*. Природна українська модель (на відміну від прямого SVO-порядку в L1) використовує безособовий прислівник + давальний відмінок особи: **«Мені погано»**, **«Мені холодно»**, **«Мені страшно»** [S1].
4. **Вибір суфіксів у прислівниках способу дії.** Уникайте російських лекал на *-ски* (*по-человечески, по-украински*). Хоча форма *по-українськи* існує, найбільш органічною та продуктивною питомою моделлю є `{по-} + {-ому}`: **по-українському, по-людському, по-козацькому** [S6].

## Природні приклади

Наступні приклади демонструють природне використання різних форм прислівників у контексті.

**Модель 1: Проста форма вищого ступеня (зміна стану або порівняння)**
*   `А сонце підбивається все вище і вище, пече все дужче та дужче.` [S10]
*   `Що важливіше — бути сильними фізично чи морально?` [S1]
*   `Учителі керували роботою, посилали туди, де було найважче.` [S10]

**Модель 2: Суплетивні та підсилені форми (емоції, оцінка)**
*   `Дякую, друже! Мені вже набагато краще.` [S2]
*   `Один розум добре, а два ще краще!` [S6]
*   `Мені бувало страшно під конем, а ще страшніше бувало на коневі.` [S5]

**Модель 3: Найвищий ступінь (максимальний вияв ознаки)**
*   `Найважливіше було правильно зорієнтуватися у величезному обсязі інформації.` [S1]
*   `Хто найменше знає, той базікає найбільше.` [S5]
*   `Талант — іскра Божа, нею людина зазвичай спалює себе, якнайяскравіше освітлюючи цією пожежею шлях іншим.` [S5]

**Модель 4: Творення та орфографія (повсякденні дії, спосіб)**
*   `Зазвичай людина намагається якнайшвидше позбутися емоційного болю.` [S1]
*   `Влетів Іван надвір замку й відразу ж зустрів чародія.` [S6]
*   `Дівчина могла вдягнути національний стрій... Вона це робить мимоволі.` [S2]

## Рекомендації для вправ

Для ефективного засвоєння теми B1 автор має структурувати активності за такою прогресією:

**Фаза 1: Розпізнавання та деконструкція (Recognition & Error Correction)**
*   *Формат:* Учням пропонуються речення з правильно та неправильно утвореними формами (напр., *самий кращий, більш швидкіше, краще всього*).
*   *Завдання:* Знайти морфологічну помилку і замінити її літературною нормою (напр., *найкращий, швидше, найкраще*). Це відпрацьовує деколонізаційний аспект і прибирає типові L2-баги [S5, S8].

**Фаза 2: Механічна трансформація (Morphological Drills)**
*   *Формат:* Таблиці перетворень з фокусом на чергування приголосних та випадання суфіксів.
*   *Завдання:* Дано інфінітивний прислівник (`високо`, `дорого`, `глибоко`, `швидко`). Учень має побудувати ланцюжок і утворити просту форму вищого та найвищого ступенів (`вище - найвище`). Окремий блок для суплетивних форм (`добре, погано, багато, мало`). [S1, S5]

**Фаза 3: Продукція в контексті (Contextual Production)**
*   *Формат:* Заповнення пропусків у міні-діалогах, що описують емоції чи побутові ситуації.
*   *Завдання:* Трансформувати прикметник у дужках на відповідний прислівник потрібного ступеня: «Мені сьогодні (поганий) -> *гірше*, ніж учора». Або використання підсилювальних часток: «Зроби це (швидко, макс. пріоритет) -> *якнайшвидше*». [S1, S5]

## Зв'язки з іншими темами

*   **Прикметники (А2-В1):** Ступені порівняння прислівників базуються на розумінні якісних прикметників та їхнього ступенювання. Якщо учень не засвоїв винятки та суплетивізм у прикметниках (*хороший - кращий*), він не зможе правильно утворити прислівник (*добре - краще*) [S1].
*   **Синтаксис простого речення (А1-А2):** Роль прислівника як обставини. Засвоєння прислівника як незмінної частини мови, що не має закінчення, а лише суфікс [S1].
*   **Правопис часток (В1):** Написання часток `не` та `ні` з прислівниками (разом чи окремо) [S6], а також написання прислівників з дефісом безпосередньо впливають на грамотність письма ускладнених текстів.
*   **Безособові речення (В2):** Використання прислівників стану (холодно, страшно, соромно) у ролі головного члена безособового речення (присудка).
