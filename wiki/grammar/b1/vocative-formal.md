## Як це пояснюють у школі

В українській шкільній програмі концепт кличного відмінка вводиться поступово, розширюючись від базового розпізнавання до складного офіційного етикету. Заболотний уводить цю тему ще в 6 класі, акцентуючи на тому, що кличний відмінок виконує функцію звертання до особи або уособленого предмета [S1]. Учням пояснюють фундаментальну рису: кличний відмінок властивий лише іменникам і не відповідає на жодне питання [S1]. 

У 8 класі (за підручником Авраменка) тема поглиблюється: школярі вивчають специфічні закінчення чоловічих і жіночих імен, розрізняють поширені та непоширені звертання, а також засвоюють правила словозміни для складних конструкцій, таких як звертання з двох загальних назв («пане лейтенанте») або загальної назви та імені [S2, S4]. 

На етапі 10 класу (Заболотний, Глазова) фокус зміщується на офіційно-діловий стиль, морфологічну норму та культуру мовлення. Докладно розглядаються фонетичні явища (чергування приголосних), форми іменників ІІІ відміни, а також детальний алгоритм творення та відмінювання імен по батькові [S3, S5]. Акцентується увага на тому, що в сучасному офіційному мовленні кличний відмінок є абсолютно обов’язковим і не може замінюватися називним [S1, S3].

## Повна парадигма

Кличний відмінок утворюється шляхом додавання специфічних закінчень до основи іменника. Для коректного навчання L2 аудиторії важливо подавати ці правила як чіткі ланцюжки морфологічних перетворень.

### 1. Іменники І відміни (переважно жіночий рід, а також чоловічий на -а/-я)

| Група | Ланцюжок перетворення | Приклад |
|---|---|---|
| **Тверда** | `{початкова форма} → {основа на твердий} → {+о} → {клична форма}` | `Микола → Микол- → +о → Миколо` [S2] <br> `жінка → жінк- → +о → жінко` [S3] |
| **М'яка / Мішана** | `{початкова форма} → {основа} → {+е / +є} → {клична форма}` | `Надія → Наді- → +є → Надіє` [S2] <br> `душа → душ- → +е → душе` [S3] |
| **Пестливі (м'яка)** | `{початкова форма} → {пестлива основа} → {+ю} → {клична форма}` | `доня → дон- → +ю → доню` [S5] <br> `матуся → матус- → +ю → матусю` [S5] |

**Явні обмеження:**
- Іменники м’якої групи *Настя, Катря, Мотря, Христя* є винятками і не приймають пестливого закінчення `-ю`. Вони утворюють кличний відмінок із закінченням `-е`: `Насте, Катре, Христе` [S2, S5].

### 2. Іменники ІІ відміни (чоловічий рід без закінчення або на -о)

Це найскладніша група для іншомовних учнів через чергування приголосних.

| Група / Особливість | Ланцюжок перетворення | Приклад |
|---|---|---|
| **Тверда (без чергування)** | `{початкова форма} → {основа} → {+е} → {клична форма}` | `брат → брат- → +е → брате` [S2] <br> `Петро → Петр- → +е → Петре` [S2] |
| **Тверда (з чергуванням [г]→[ж])** | `{початкова форма} → {основа} → {[г]→[ж]} → {+е} → {клична форма}` | `друг → дру- → ж- → +е → друже` [S3] |
| **Тверда (з чергуванням [к]→[ч])** | `{початкова форма} → {основа} → {[к]→[ч]} → {+е} → {клична форма}` | `козак → коза- → ч- → +е → козаче` [S3] |
| **Тверда (з чергуванням [ц']→[ч])** | `{початкова форма} → {основа} → {[ц']→[ч]} → {+е} → {клична форма}` | `хлопець → хлоп- → ч- → +е → хлопче` [S3] |
| **Тверда (з чергуванням [х]→[ш])** | `{початкова форма} → {основа} → {[х]→[ш]} → {+е} → {клична форма}` | `пастух → пасту- → ш- → +е → пастуше` [S3] |
| **М'яка (загальна)** | `{початкова форма} → {основа} → {+ю} → {клична форма}` | `Сергій → Сергі- → +ю → Сергію` [S2] <br> `лікар → лікар- → +ю → лікарю` [S4] |
| **Іншомовні на г, к, х** | `{початкова форма} → {основа} → {+у} → {клична форма}` | `Жак → Жак- → +у → Жаку` [S2, S3] <br> `Людвіг → Людвіг- → +у → Людвігу` [S2] |

**Явні обмеження:**
- В іншомовних власних назвах з основою на [г], [к], [х], а також в іменниках із суфіксами `-ик`, `-к` чергування приголосних **ніколи не відбувається**. Вони послідовно набувають закінчення `-у`: `Джеку, Генріху, хлопчику, братику` [S3, S4].
- Чоловічі імена на `-а` мають закінчення `-о` (як І відміна): `Миколо, Микито` [S2].
- Ім'я `Олег` має дві нормативні форми: `Олегу` та `Олеже` [S2].

### 3. Творення та відмінювання імен по батькові

Офіційне українське звертання вимагає бездоганного використання імен по батькові. 

**Творення:**
- **Чоловічі:** `{початкова форма імені батька} → {основа} → {+ович} → {ім'я по батькові}`. `Назар → Назар- → +ович → Назарович` [S6].
- **Жіночі:** `{початкова форма імені батька} → {основа} → {+івн(а) / +ївн(а)} → {ім'я по батькові}`. `Сергій → Сергі- → +ївна → Сергіївна` [S6].

**Винятки творення:** `Григорій → Григорович / Григорівна`; `Ілля → Ілліч / Іллівна`; `Яків → Якович / Яківна`; `Кузьма → Кузьмич (Кузьмович) / Кузьмівна` [S3, S6].

**Кличний відмінок імен по батькові:**
Чоловічі імена по батькові відмінюються як іменники ІІ відміни мішаної групи, а жіночі — як іменники І відміни твердої групи [S5, S6].
- **Чоловічі:** `{по батькові Н.в.} → {основа на -ич} → {+у} → {клична форма}`. `Петрович → Петрович- → +у → Петровичу` [S5].
- **Жіночі:** `{по батькові Н.в.} → {основа на -івн} → {+о} → {клична форма}`. `Петрівна → Петрівн- → +о → Петрівно` [S5].

### 4. Конструювання офіційних багатокомпонентних звертань

| Структура | Правило | Приклад |
|---|---|---|
| **Дві загальні назви** | Обидва слова у кличному відмінку | `пане президенте`, `пані голово` [S1] |
| **Загальна назва + Ім'я** | Обидва слова у кличному відмінку | `колего Віталію`, `брате Петре` [S2, S4] |
| **Ім'я + По батькові** | Обидва слова у кличному відмінку | `Ігорю Андрійовичу`, `Маріє Василівно` [S4] |
| **Загальна назва + Прізвище (ч)** | Загальна назва в кличному, прізвище в кличному АБО називному | `друже Максименку` / `друже Максименко` [S4] |
| **Загальна назва + Прізвище (ж)** | Загальна назва в кличному, прізвище залишається в називному | `пані Гаркуше` (якщо прізвище на -а) [S4]; `доценте Глущук` (прізвище незмінне) [S5] |

**Явні обмеження:**
- Прикметники та займенники у складі поширеного звертання **завжди** залишаються у формі називного відмінка: `шановний пане` (ніколи не `*шановному пане`), `наш дорогий колего` [S1, S5].
- У множині форми кличного відмінка абсолютно тотожні формам називного (сестри, матері, учні). Єдиним винятком є іменник `пан`, який у множині має форму `панове` (Н.в. — пани) [S1, S3, S5].
- Якщо жіноче прізвище закінчується на приголосний або на `-о`, воно взагалі не відмінюється: `пані Костенко` [S6].

## Частотність і пріоритети

Для студентів рівня B1 найвищий пріоритет мають:
1. Конструкції з `Пане / Пані / Панове` + Посада / Прізвище (найчастотніший формат ділового листування).
2. Звертання на `Ім'я + По батькові` (критично важливо для офіційних установ, лікарень, університетів).
3. Базові чоловічі та жіночі імена у кличному відмінку (для щоденного офісного спілкування).
Іменники ІІІ відміни (радосте, ноче) мають низьку частотність і трапляються переважно в поезії [S3, S5], тому їх можна вводити лише пасивно для рецептивних навичок.

## Типові помилки L2 (англомовні учні)

В англійській мові категорія кличного відмінка відсутня, що спричиняє масивне структурне перенесення.

| ❌ Помилково | ✅ Правильно | Чому |
|---|---|---|
| Шановному пане президенте! | **Шановний** пане президенте! | В англійській мові слова в іменній групі не змінюються за відмінками. Учні, дізнавшись про кличний відмінок, гіперкоректно намагаються відмінювати і прикметники, тоді як в українській нормі прикметник у звертанні зберігає форму називного відмінка [S1, S5]. |
| Пан президент, вітаємо! | **Пане президенте**, вітаємо! | Калькування англійської моделі «Mr. President», де іменники не мають морфологічних маркерів звертання. В українській обидві загальні назви вимагають кличного відмінка [S1, S4]. |
| Іван Петрович, зайдіть! | **Іване Петровичу**, зайдіть! | Англомовні учні не мають концепту «по батькові» як змінного іменника. Вони або не змінюють жодного слова, або змінюють лише ім'я. Правило вимагає кличного відмінка для обох компонентів [S5, S6]. |
| Джек, іди сюди! | **Джеку**, іди сюди! | Англомовні учні сприймають іншомовні імена як принципово невідмінювані. Проте українська морфологія асимілює їх: імена на г, к, х отримують закінчення -у [S2, S3]. |
| Олена Юрієвна! | Олено **Юріївно**! | Вплив російських фонетичних патернів (фонетична інтерференція). Учні не ідентифікують специфічний український суфікс `-ївн(а)` після голосних та ігнорують перехід у кличну форму на `-о` [S5, S6]. |
| Мій друг, допоможи! | Мій **друже**, допоможи! | Ігнорування чергування приголосних ([г] → [ж]). Учні намагаються утворити кличну форму без модифікації основи, що є грубим порушенням морфологічної норми [S3]. |

## Деколонізаційні застереження

Кличний відмінок — це потужний маркер української мовної ідентичності. У російській мові кличний відмінок як повноцінна граматична категорія зник, залишившись лише в релігійних архаїзмах («Боже», «Господи») або в розмовних усіченнях («Мам», «Вань»). Російська імперська та радянська мовні політики десятиліттями намагалися маргіналізувати український кличний відмінок, нав'язуючи використання називного відмінка у звертаннях як «більш офіційне і сучасне». 

Для авторів модулів критично важливо уникати наступних колоніальних пасток:
- **Заміна кличного на називний:** Конструкції на кшталт «Шановний Іван Петрович!» є типовим русизмом. Українська літературна норма вимагає виключно «Шановний Іване Петровичу!» [S5].
- **Калькування суфіксів по батькові:** Російські форми на *-овна / -євич* є суржиком. Українські суфікси — це строго *-івна / -ївна* та *-ович* [S6].
- **Розмовні усічення:** Російські усічені форми звертання (Тьотя Свєта! Маш! Петь!) є невластивими українській мові. Українська використовує повноцінні кличні форми з пестливими суфіксами: «Тітко Світлано! Марійко! Петрику!» [S1].
Ніколи не подавайте український кличний відмінок як «виняток» чи «стилістичну прикрасу» — це обов'язкова, базова і невіддільна частина системи іменника.

## Природні приклади

Наведені нижче приклади демонструють природне використання кличного відмінка в різних комунікативних ситуаціях:

**Група 1: Офіційний етикет (Посади, титули, множина)**
- «Шановні пані та панове!» [S1]
- «Пане президенте, пані голово.» [S1]
- «Слухаю вас, пане лейтенанте!» [S4]
- «Пропонуємо вам, Іване Сергійовичу, відкрити рахунок у банку.» [S4]

**Група 2: Академічне та робоче середовище (Ім'я + По батькові)**
- «Володимире Івановичу! Коли буде наступне тренування?» [S4]
- «Ольго Василівно! Ви знаєте, що я людина нового покоління.» [S3]
- «Радий тебе бачити, побратиме Степане!» [S4]

**Група 3: Побутові та родинні ситуації**
- «Так, Дмитрику! Це було весело!» [S1]
- «Заспівай мені, мамо, колискову, хочу ще почути твоє рідне слово.» [S4]
- «Моє шанування, тітонько Марусю!» [S4]

**Група 4: Риторичні звертання (Поезія, публіцистика)**
- «Скажи ж мені, Дніпре, які вітровії все кличуть тебе крізь степи і гаї?..» [S3]
- «Без мови рідної, юначе, й народу нашого нема.» [S4]
- «Молода і бентежна Україно моя, якщо ти незалежна, незалежний і я.» [S3]
- «О краю, рідний краю! Твої степи безкраї.» [S4]

## Рекомендації для вправ

Автор модуля має проєктувати вправи за такою методичною послідовністю:

**Фаза 1: Розпізнавання та аналіз форми.**
- *Вправа:* Читання коротких офіційних емейлів або діалогів у банку, де студенти повинні знайти і виділити всі форми кличного відмінка.
- *Завдання:* Зіставити знайдену форму кличного відмінка з її початковою формою (називним відмінком).

**Фаза 2: Механічна трансформація та чергування.**
- *Вправа:* Заповнення пропусків у реченнях, де в дужках подано називний відмінок (наприклад, `Шановний (Олександр Петрович)` → `Шановний Олександре Петровичу` [S4]).
- *Фокус:* Окремі мікроблоки на відпрацювання чергування приголосних ([г]→[ж], [к]→[ч]) та відмінювання імен по батькові.

**Фаза 3: Продукція у контексті ділового спілкування.**
- *Вправа:* Написання власного короткого листа (відповідь викладачу або звернення до менеджера банку). Учень має самостійно згенерувати правильне поєднання: `Прикметник (Н.в.) + Загальна назва (Кл.в.) + Прізвище/Ім'я (Кл.в.)`.
- *Рольова гра:* Симуляція зустрічі (наприклад, реєстрація на конференції), де учні звертаються одне до одного, використовуючи картки з іменами та по батькові.

## Зв'язки з іншими темами

- **Передумови:** Перед вивченням офіційного кличного відмінка учні повинні впевнено володіти концептом трьох відмін іменника, розрізняти м'яку, тверду та мішану групи, а також знати історичні чергування приголосних звуків у коренях слів (теми А2).
- **Що уможливлює:** Ця тема є абсолютно критичною для переходу до модуля «Ділове листування та корпоративний етикет» (рівень B2). Без автоматизованого кличного відмінка неможливо написати жодного офіційного листа чи звернутися до керівництва.
