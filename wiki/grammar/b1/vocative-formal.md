## Як це пояснюють у школі (How Schools Teach This)
Кличний відмінок як форма звертання вводиться в українській шкільній програмі досить рано, зазвичай у 5-6 класах [S35]. Початково увага зосереджена на базовій функції — звертанні до особи (`дорогий друже`, `пані Інно`) — і на тому, що іменники в кличному відмінку не відповідають на питання і не є членами речення.

У старших класах (8-11) тема поглиблюється, фокусуючись на правилах утворення складних звертань, які є ключовими для офіційного та ділового мовлення [S36][S37]. Підручники для 10-11 класів систематизують правила для різних комбінацій:
1.  **Ім'я та по батькові**: обидва слова ставляться у кличний відмінок (`Сергію Петровичу`). [S37].
2.  **Загальна назва та ім'я**: обидва слова у кличному відмінку (`пане Володимире`). [S37].
3.  **Загальна назва та прізвище**: загальна назва у кличному, прізвище — у називному (`пане Семенюк`). [S36].
4.  **Дві загальні назви**: обидва слова можуть бути у кличному відмінку (`пане лейтенанте`), хоча іноді друге слово може залишатись у називному (`пане лейтенант`). [S36].

Особлива увага приділяється етикету та доречності використання форм `пан`, `пані`, `добродій`, `добродійко` в офіційних ситуаціях, що підкреслюється як важлива частина культури спілкування [S38][S39].

## Повна парадигма (Full Paradigm)
В офіційному мовленні звертання майже завжди є складеними. Нижче наведено основні моделі їх творення.

### 1. Звертання на ім'я та по батькові
Це найпоширеніша форма ввічливого звертання до старших за віком або статусом людей. Обидва слова — і ім'я, і по батькові — ставляться у кличний відмінок [S36][S40][S41].

| Називний відмінок | Кличний відмінок | Правило |
| :--- | :--- | :--- |
| **Чоловіки** | | |
| Сергій Васильович | Сергі**ю** ВасильОВИЧ**у** | Чоловічі імена по батькові завжди мають закінчення `-у` [S42]. |
| Іван Вікторович | Іван**е** ВікторОВИЧ**у** | |
| Олег Ігорович | Олег**у** Ігор**ю** | Імена `Ігор` та `Олег` мають паралельні форми, але в офіційному стилі `Ігорю` та `Олегу` є кращими. [S43][S44]. |
| **Жінки** | | |
| Ганна Дмитрівна | Ганн**о** ДмитрІВН**о** | Жіночі імена по батькові завжди мають закінчення `-о` [S45]. |
| Марія Іллівна | Марі**є** ІллІВН**о** | |
| Надія Петрівна | Наді**є** ПетрІВН**о** | |

### 2. Звертання з титулами `пан`, `пані`, `панна`, `добродій`
Ці форми є стандартними для ділового та офіційного спілкування.

| Тип звертання | Називний відмінок | Кличний відмінок | Приклад | Джерело |
| :--- | :--- | :--- | :--- | :--- |
| **Загальна назва + Ім'я** | пан Володимир | **пане** Володимир**е** | *Пане Володимире, колего Ірино.* | `11-klas-ukrajinska-mova-voron-2019_s0194` |
| | пані Оксана | **пані** Оксан**о** | *Пані Оксано, запрошуємо Вас...* | `6-klas-ukrmova-litvinova-2023_s0148` |
| **Загальна назва + Прізвище** | пан Семенюк | **пане** Семенюк | *Пане Петренку, пане майстре...* | `9-klas-ukrajinska-mova-avramenko-2017_s0179` |
| | добродійка Білецька | **добродійко** Білецька | *Добродійко Скирда.* | `10-klas-ukrmova-zabolotnyi-2018_s0223` |
| **Дві загальні назви** | пан президент | **пане президенте** | *Пане лейтенанте, добродію бригадире.* | `10-klas-ukrmova-zabolotnyi-2018_s0223` |
| | пані голова | **пані голово** | *Шановна пані голово...* | `6-klas-ukrmova-litvinova-2023_s0148` |
| **Множина** | пани | **панове** | *Шановні пані та панове!* | `6-klas-ukrmova-litvinova-2023_s0147` |

### 3. Прикметники у складі звертання
Прикметники та займенники, що входять до складу поширеного звертання, завжди мають форму **називного відмінка** [S46][S40].

*   `Шановний` пане, а не `Шановний`
*   `Вельмишановна` пані, а не `Вельмишановну`
*   `Наш дорогий` колего, а не `Нашого дорогого`

## Частотність і пріоритети
В офіційному спілкуванні (B1-B2) пріоритетними є наступні форми:

1.  **Шановний/а + Ім'я + По батькові (у кличному відмінку)**: Найбільш універсальна і поширена форма для листів, офіційних промов та звернень до викладачів, керівників, старших осіб. Приклад: `Шановний Сергію Васильовичу` [S47].
2.  **Пане/Пані + Ім'я (у кличному відмінку)**: Трохи менш формальний, але абсолютно прийнятний варіант у діловому середовищі, особливо при усному спілкуванні. Приклад: `Пані Катерино` [S38].
3.  **Пане/Пані + Посада (у кличному відмінку)**: Дуже поширена форма звертання до людей, чиї імена невідомі або коли посада є важливою в контексті. Приклад: `Пане професоре`, `пані вчителько` [S38].
4.  **Панове**: Стандартне звертання до аудиторії, що складається з чоловіків або є змішаною. Приклад: `А що, панове отамани, тепер будемо робити?` [S48].

Форми на кшталт `добродію/добродійко` є цілком нормативними, але в сучасному усному мовленні вживаються дещо рідше, ніж `пан/пані` [S38].

## Типові помилки L2
Англомовні студенти часто припускаються помилок через відсутність кличного відмінка в англійській мові та інтерференцію з російською.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Шановний Сергій Васильович...` | `Шановний Сергі**ю** ВасильОВИЧ**у**...` | **Помилка**: Використання називного відмінка замість кличного. **Причина**: В англійській мові звертання не змінює форму імені. Студенти просто додають "Dear" до імені в його базовій формі. |
| `Пані Оксана...` (як звертання) | `Пані Оксан**о**...` | **Помилка**: Неправильне закінчення для жіночого імені. **Причина**: Плутанина між твердою (`-о`) та м'якою (`-є`, `-ю`) групами. |
| `Пане Петренку...` | `Пане Петренк**о**...` | **Помилка**: Застосування кличного відмінка до прізвища, яке йде після загальної назви. **Причина**: Надмірна генералізація правила "все у звертанні має бути в кличному відмінку". Прізвища після `пан/пані` залишаються в називному [S36]. |
| `Шановний Олеже Петровичу...` | `Шановний Олег**у** Петровичу...` | **Помилка**: Використання розмовної/дружньої форми (`Олеже`) в офіційному звертанні. **Причина**: Існують паралельні форми, але вони стилістично розрізнені. Форма на `-у` (`Олегу`) є більш нейтральною та формальною. [S49]. |
| `Запрошуємо вас на збори.` (до однієї шанованої особи в листі) | `Запрошуємо **В**ас на збори.` | **Помилка**: Написання займенника `вас` з маленької літери при звертанні до однієї особи в офіційному листуванні. **Причина**: В англійській `you` не має такої диференціації. В українській мові `Ви/Ваш` з великої літери є обов'язковим маркером ввічливості в письмовій комунікації до однієї конкретної особи [S50]. |
| `Пане полковник...` | `Пане полковник**у**...` | **Помилка**: Неправильне утворення кличного відмінка для двох загальних назв. **Причина**: Плутанина правил. Хоча форма `пане полковник` іноді допускається, нормативною є форма, де обидва слова в кличному відмінку. [S40][S43]. |

## Деколонізаційні застереження
**1. `Пан/Пані/Панове` vs. `Товариш`:**
В українській мові звертання `пан`, `пані`, `панна`, `панове` мають глибоке історичне коріння і є органічною частиною національної культури спілкування [S39]. Наприклад: `Добрий вечір тобі, пане господарю!` [S35].
У радянський період ці форми були насильно витіснені уніфікованим звертанням `товариш`. Підручники прямо вказують на "фальшивість слова товариш у звертанні до першої-ліпшої людини" і рекомендують "без вагань повертаймо те, що належить нам як спадок" [S38]. Навчаючи студентів, необхідно підкреслювати, що `пан/пані` — це не просто "західний" чи "польський" вплив, а питома українська норма, відроджена після десятиліть русифікації.

**2. Уникати зневажливих сурогатних форм:**
Текст О. Пономарева [S38] чітко вказує, що вакуум, створений викоріненням традиційних звертань, заповнився "цілком недоречними замінниками: `мужчино`, `женщино`, `дєвушко`". Ці форми є прямими кальками з російської, не відповідають нормам української мови і вважаються вкрай нешанобливими [S51]. Потрібно активно навчати студентів уникати їх і натомість використовувати `пане/пані/добродію`.

**3. Українська граматика є самодостатньою:**
При поясненні правил творення кличного відмінка (напр., закінчення `-у` в іменах по батькові) не слід проводити паралелі з російською мовою, де система звертань інша (ім'я + по батькові в називному відмінку). Правила українського кличного відмінка слід пояснювати через внутрішню логіку української морфології (відміни, групи, суфікси), а не як "відмінності від російської".

## Природні приклади

#### Звертання на ім'я та по батькові
1.  Шановний Сергію Васильовичу, Я хочу записатися до вас на прийом. [S47]
2.  Володимире Олексійовичу, Маріє Іллівно. (Приклади з правила [S46])
3.  Шановний Олегу Петровичу, просимо надіслати фінансовий звіт. [S49]

#### Звертання з `пан`, `пані`, `добродій`
4.  Добрий вечір тобі, пане господарю! Радуйся! [S35]
5.  Дякую Вам, добродію, за тепле слово! [S38]
6.  Шановні пані та панове! (Приклад звертання до аудиторії [S35])
7.  Пані Олена, Вітаю вас, рада бачити! [S52]

#### Звертання з посадою чи роллю
8.  Рекомендовано обидва слова вживати у кличному відмінку: пане президенте, пані голово. [S46]
9.  Друже Остапе, пане Іваненку. (Приклади з правила [S46])
10. Пане полковнику, брате Петре, колего Євгене... [S44]

#### Історичні та літературні приклади
11. Княже, хотять тебе взяти, схопити. [S53]
12. А що, панове отамани, тепер будемо робити? [S48]

## Рекомендації для вправ

*   **Phase 1: Ідентифікація та базове утворення.**
    *   **Вправа**: Знайти у тексті (напр., у зразку електронного листа) всі форми звертань і визначити, до кого вони звернені (до друга, до лікаря, до викладача).
    *   **Drill**: Утворити кличний відмінок від списку поширених чоловічих та жіночих імен (Іван → Іване, Оксана → Оксано, Марія → Маріє, Сергій → Сергію).
    *   **Drill**: Трансформувати прості речення, додавши до них звертання: `Він прийшов.` → `Іване, він прийшов.`

*   **Phase 2: Складні офіційні звертання.**
    *   **Drill**: Утворити правильну форму звертання з імені та по батькові. (`(Віктор Володимирович) ..., вітаємо Вас!`)
    *   **Drill**: Утворити правильну форму звертання з `пан/пані` та імені/посади. (`(пан, професор) ..., чи можу я поставити питання?`)
    *   **Choice**: Вибрати правильний варіант: `Шановний (Олеже/Олегу) Петровичу...`

*   **Phase 3: Практичне застосування.**
    *   **Вправа**: Написати початок офіційного електронного листа до гіпотетичного `директора Петра Коваленка`, використовуючи відповідне звертання.
    *   **Вправа**: Написати коротке привітання з професійним святом для `вчительки Ганни Степанівни`, використовуючи поширене звертання.
    *   **Role-play**: Зімітувати телефонну розмову, де студент записується на прийом до лікаря `Ігоря Андрійовича`, використовуючи правильні форми звертання.

## Зв'язки з іншими темами

*   **Граматика**:
    *   **Іменник (Declension)**: Кличний відмінок є невід'ємною частиною системи відмінювання іменників І, ІІ та ІІІ відмін. Його утворення залежить від роду, відміни та групи іменника (тверда, м'яка, мішана).
    *   **Прикметник**: У складі поширеного звертання прикметники залишаються у називному відмінку, що є важливою синтаксичною особливістю.
*   **Синтаксис**:
    *   **Звертання**: Іменники в кличному відмінку синтаксично виступають у ролі звертань, які не є членами речення і виділяються комами або знаком оклику.
*   **Прагматика та стилістика**:
    *   **Офіційно-діловий стиль**: Правильне використання офіційних звертань є ключовим маркером цього стилю (листи, заяви, офіційні розмови).
    *   **Етикет і ввічливість**: Вибір форми звертання (`ти` vs. `ви`, `пан` vs. ім'я) є важливим елементом мовленнєвого етикету.
