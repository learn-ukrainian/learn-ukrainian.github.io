## Як це пояснюють у школі

У шкільній програмі української мови морфонологічні явища (чергування голосних і приголосних звуків) запроваджуються поступово, починаючи з початкових класів. У 4 класі учні вперше стикаються з чергуванням `[і]` з `[о], [е]` в іменниках жіночого та чоловічого роду під час словозміни (сіль — солі, кіт — кота) [S1]. У 5 класі автори підручників формалізують це правило через поняття відкритого та закритого складу: «у відкритих складах — [о], [е], у закритих — [і]» (ніж — ножі, осінь — осені) [S4]. Тоді ж уводять чергування голосних у коренях дієслів: `[о] // [а]`, `[е] // [і]`, `[е] // [и]` перед наголошеними суфіксами `-а-`, `-я-`, `-ува-` (пекти — випікати) [S4].

Чергування приголосних в іменниках ґрунтовно вивчається у 6 класі. Підручники пояснюють, що під час відмінювання іменників І відміни (у давальному та місцевому відмінках) кінцеві приголосні основи `[г], [к], [х]` обов'язково чергуються із м'якими свистячими `[з'], [ц'], [с']` (нога — нозі, рука — руці, муха — мусі) [S6]. У 7 класі, під час вивчення дієслова, учні засвоюють чергування приголосних під час творення особових форм теперішнього часу: `[г] // [ж]`, `[к] // [ч]`, `[д] // [дж]`, `[т] // [ч]` та появу епентетичного `[л]` після губних `[б], [п], [в], [м], [ф]` (зробити — зроблю, ловити — ловлю) [S3].

Окрему увагу шкільні підручники приділяють впливу кінцевого приголосного основи на вибір відмінкових закінчень. Іменники поділяються на тверду, м'яку та мішану групи, що визначає закінчення орудного відмінка: `-ою` / `-ом` для твердої, `-ею` / `-ем` для м'якої та мішаної, `-єю` / `-єм` після `[й]` [S1, S2, S7]. Підручник для 10 класу систематизує всі ці випадки, закріплюючи їх як морфологічну норму [S8].

## Повна парадигма

### 1. Чергування голосних `[і]` // `[о], [е]` в іменниках
Закономірність: звук `[і]` стоїть у закритому складі (склад закінчується на приголосний). Якщо під час відмінювання склад відкривається (з'являється голосне закінчення), `[і]` переходить в `[о]` або `[е]` [S4].

* **Механізм творення (основа з `[о]`):** `{називний відмінок} → {основа} → {і // о} → {+закінчення}`. Приклад: `кіт → кіт- → кот- → +а → кота`.
* **Механізм творення (основа з `[е]`):** `{називний відмінок} → {основа} → {і // е} → {+закінчення}`. Приклад: `піч → піч- → печ- → +і → печі`.

| Називний відмінок (закритий склад) | Родовий/Давальний відмінок (відкритий склад) |
| --- | --- |
| стіл | ст**о**ла, ст**о**лу |
| кіт | к**о**та, к**о**тові |
| ніж | н**о**жа, н**о**жу |
| сіль | с**о**лі |
| радість | рад**о**сті |
| осінь | ос**е**ні |
| річ | р**е**чі |

**Обмеження:** чергування не відбувається у словах іншомовного походження (портфель — портфеля, метр — метра) та в деяких питомих словах як виняток (хобот — хобота, робот — робота) [S4].

### 2. Чергування приголосних `[г, к, х]` // `[з', ц', с']` в іменниках
Закономірність: перед закінченням `-і` у давальному та місцевому відмінках іменників І відміни задньоязикові приголосні завжди переходять у м'які свистячі [S6].

* **Механізм творення:** `{називний відмінок} → {основа} → {г, к, х // з', ц', с'} → {+і}`. Приклад: `рука → рук- → руц- → +і → руці`.

| Основа на | Називний відмінок | Давальний / Місцевий відмінок |
| --- | --- | --- |
| `[г] // [з']` | нога, подруга, Ольга, крига | но**з**і, подру**з**і, Оль**з**і, кри**з**і |
| `[к] // [ц']` | рука, дівчинка, донька, річка | ру**ц**і, дівчин**ц**і, донь**ц**і, річ**ц**і |
| `[х] // [с']` | муха, комаха, черепаха | му**с**і, кома**с**і, черепа**с**і |

### 3. Чергування приголосних у дієсловах (Теперішній / Майбутній простий час)
Закономірність: під час утворення форм 1-ї особи однини (я) та іноді інших осіб у дієсловах недоконаного і доконаного виду відбувається зміна кінцевого приголосного основи [S3].

* **Механізм творення:** `{інфінітив} → {основа без -ти} → {заміна приголосного} → {+у/-ю}`. Приклад: `плакати → плака- → плак- → плач- → +у → плачу`.

| Чергування | Інфінітив | 1-ша особа одн. (я) | Інші особи (ти, він, ми) |
| --- | --- | --- | --- |
| `[г] // [ж]` | могти, берегти | мо**ж**у, бере**ж**у | мо**ж**еш, бере**ж**еш |
| `[к] // [ч]` | плакати, пекти | пла**ч**у, пе**ч**у | пла**ч**еш, пе**ч**еш |
| `[х] // [ш]` | колихати, дихати | коли**ш**у, ди**ш**у | коли**ш**еш, ди**ш**еш |
| `[з] // [ж]` | різати, возити | рі**ж**у, во**ж**у | рі**ж**еш, во**з**иш |
| `[с] // [ш]` | писати, носити | пи**ш**у, но**ш**у | пи**ш**еш, но**с**иш |
| `[д] // [дж]` | ходити, водити | хо**дж**у, во**дж**у | хо**д**иш, во**д**иш |
| `[т] // [ч]` | платити, летіти | пла**ч**у, ле**ч**у | пла**т**иш, ле**т**иш |
| `[ст] // [щ]` | мастити, чистити | ма**щ**у, чи**щ**у | ма**ст**иш, чи**ст**иш |
| `[ск] // [щ]` | полоскати | поло**щ**у | поло**щ**еш |
| `[зд] // [ждж]`| їздити | ї**ждж**у | ї**зд**иш |

**Поява епентетичного `[л]`:** губні приголосні `[б]`, `[п]`, `[в]`, `[м]`, `[ф]` вимагають вставного `[л]` перед закінченням `-ю` [S3].
* **Механізм творення:** `{інфінітив} → {основа без -ти} → {+л} → {+ю}`. Приклад: `любити → люби- → люб- → любл- → +ю → люблю`.

| Губний | Інфінітив | 1-ша особа одн. (я) | 3-тя особа множ. (вони) |
| --- | --- | --- | --- |
| `[б] + [л]` | робити, любити | роб**л**ю, люб**л**ю | роб**л**ять, люб**л**ять |
| `[п] + [л]` | ліпити, спати | ліп**л**ю, сп**л**ю | ліп**л**ять, сп**л**ять |
| `[в] + [л]` | ловити, мовити | лов**л**ю, мов**л**ю | лов**л**ять, мов**л**ять |
| `[м] + [л]` | ломити, шуміти | лом**л**ю, шум**л**ю | лом**л**ять, шум**л**ять |
| `[ф] + [л]` | графити | граф**л**ю | граф**л**ять |

### 4. Залежність закінчень від групи приголосних (Орудний відмінок)
Закономірність: тверді приголосні основи вимагають закінчень із голосними `[о]`, `[а]`. М'які, мішані (шиплячі `ж, ч, ш, щ`) та основа на `[й]` вимагають голосних `[е]`, `[є]`, `[я]` [S1, S2, S7].

* **Механізм творення:** `{називний відмінок} → {основа} → {оцінка кінцевого звука} → {+ою/+ею/+єю}`. Приклад: `вежа → веж- (шиплячий, мішана група) → +ею → вежею`.

| Відміна | Тверда група | М'яка група | Мішана група (шиплячі) | Основа на `[й]` |
| --- | --- | --- | --- | --- |
| **І (на -а/-я)** | мама — мам**ою** | зоря — зор**ею** | межа — меж**ею** | мрія — мрі**єю** |
| | артистка — артистк**ою** | миша — миш**ею** (виняток) | душа — душ**ею** | надія — наді**єю** |
| **ІІ (чол. р.)** | сир — сир**ом** | олівець — олівц**ем** | чиж — чиж**ем** | обрій — обрі**єм** |
| | батько — батьк**ом** | учень — учн**ем** | плащ — плащ**ем** | Сергій — Сергі**єм** |

**Обмеження:** іменники на `-ар`, `-ир`, `-яр` розподіляються між твердою, м'якою та мішаною групами залежно від наголосу під час відмінювання (лікар — лікарем, столяр — столяром / столярем) [S1, S7]. Написання цих слів слід перевіряти за словником.

## Частотність і пріоритети
Для засвоєння на рівні B1 найвищий пріоритет мають:
1. **Чергування `г, к, х // з', ц', с'` в іменниках І відміни.** Місцевий відмінок (де? на кому? на чому?) є одним із найбільш уживаних у базовій комунікації (у книжці, у руці, на нозі). Англомовні учні часто роблять тут помилки.
2. **Чергування `[і] // [о], [е]`.** Без цього правила неможливо правильно утворити навіть родовий відмінок найбільш базових слів (кіт — кота, стіл — стола, рік — року, Львів — Львова).
3. **Епентетичне `[л]` та чергування приголосних у дієсловах 1-ї особи.** Форми `я люблю`, `я роблю`, `я сиджу`, `я бачу`, `я ходжу` критично необхідні для висловлювання власних дій та думок. Учні повинні засвоїти їх до автоматизму.

## Типові помилки L2 (англомовні учні)
Англомовні учні схильні ігнорувати внутрішні зміни в основі слова, оскільки в англійській мові словозміна здебільшого зводиться до простого додавання суфіксів (dog — dogs, work — worked) без суттєвої зміни кореня.

| ❌ Помилково | ✅ Правильно | Чому |
| --- | --- | --- |
| *Я ходю до школи.* | Я **ходжу** до школи. | Учень застосовує регулярне закінчення `-ю` до основи `ход-`, ігноруючи обов'язкове чергування `д // дж` у першій особі однини. |
| *Вона тримає телефон в рукі.* | Вона тримає телефон у **руці**. | Калькування логіки незмінної основи. Учень додає закінчення місцевого відмінка `-і` до початкової основи `рук-`, забуваючи про фонетичне чергування `к // ц'`. |
| *Це іграшка мого Кіта.* | Це іграшка мого **кота**. | Нерозуміння правила закритого/відкритого складу. Учень залишає голосний `[і]`, хоча при появі закінчення `-а` склад відкривається, і звук закономірно змінюється на `[о]`. |
| *Я любю читати книги.* | Я **люблю** читати книги. | Ігнорування епентетичного `[л]`. Учень відкидає інфінітивний суфікс і просто додає `-ю` до губного приголосного `[б]`, що не відповідає правилам української фонетики. |
| *Він малює олівцом.* | Він малює **олівцем**. | Нерозрізнення твердої та м'якої груп. Учень застосовує універсальне закінчення `-ом`, хоча основа `олівець` закінчується на м'який `[ц']`, що вимагає закінчення `-ем`. |
| *Ми гуляли по вулицям.* | Ми гуляли по **вулицях**. | Помилкове керування прийменника `по`. Учень вживає давальний відмінок множини замість місцевого (закінчення `-ах / -ях`), що є поширеною помилкою внаслідок структурної аналогії чи перенесення з інших мов. |

## Деколонізаційні застереження
Морфонологічні процеси української мови мають глибокі історичні корені, які суттєво відрізняють її від російської мови. Авторам курсів слід уникати пояснень, що спираються на російську граматику або подають українські форми як «винятки».

1. **Чергування `г, к, х // з', ц', с'`**: В українській мові це чергування є послідовним і обов'язковим (у книжці, на березі, у рухi → у русі). Російська мова його історично втратила (в книге, на берегу, в руке). Ніколи не трактуйте `к // ц'` як ускладнення — це питома фонетична риса української мови.
2. **Чергування `[і] // [о], [е]`**: Це суто українське явище (наслідок історичного занепаду зведених голосних). Воно формує унікальний звуковий ритм мови: *Київ — Києва, Чернігів — Чернігова, радість — радості*. У російській мові корінь переважно залишається сталим (Киев — Киева, радость — радости). Слід наголошувати, що звук `[і]` існує переважно в закритих складах.
3. **Епентетичне `[л]`**: Українська мова суворо зберігає вставне `[л]` у дієсловах (люблю, роблю), тоді як російська іноді допускає розмовні чи діалектні форми без нього. В українській мові відсутність `[л]` — це груба помилка.
4. **Прийменник `по`**: В українській мові прийменник `по` з місцевим відмінком множини (по містах, по селах) вживається на позначення розподілу дії або місця дії [S1, S8]. Конструкції на зразок `по містам`, `по селам` є суржиком (російською калькою).

## Природні приклади
Наведені нижче речення ілюструють природне використання морфонологічних правил:

**Чергування голосних `[і]` // `[о], [е]`**
1. *Під лежачий камінь вода не тече.* [S1]
2. *З неба зорі хапає, а під носом не бачить.* [S1]
3. *Слово вилетить горобцем, а вернеться волом.* [S1]

**Чергування приголосних в іменниках (Місцевий/Давальний)**
4. *Град йшов стрімко густими рядами... стукали по стеклу, по вікну, по рамі, по трубі, по доріжці біля хати.* [S1]
5. *У неділю по обіді ми усією родиною ходили по гриби.* [S1]
6. *Справжні королівські рушники на покутті!* [S2]

**Відмінкові закінчення орудного відмінка**
7. *Спортсменкою-паралімпійкою Оксаною Ботурчук захоплюється увесь світ.* [S1]
8. *Любуюся лисицею. Вишиваю ниткою.* [S1]
9. *Вона була зі щирою вдачею.* [S1]

**Чергування в дієсловах (Теперішній/Майбутній час)**
10. *Завтра ми їдемо в Київ.* [S3]
11. *Я завжди кажу, що професійних екскурсоводів багато і з ними конкурувати я не хочу.* [S3]
12. *І наші нащадки з майбутнього читатимуть про сучасні події, як і ми читаємо про Київську Русь...* [S3]

## Рекомендації для вправ
Для якісного засвоєння цієї теми авторам слід структурувати вправи за такою прогресією:

* **Фаза 1: Рецептивне розпізнавання (Сприйняття).** Учні отримують текст або набір речень, де потрібно знайти слова з чергуванням і відновити їхню початкову форму.
  * *Формат:* Текст із виділеними словами (наприклад, *у руці*, *ходжу*, *кота*), до яких треба обрати початковий інфінітив чи називний відмінок зі списку.
* **Фаза 2: Контрольована трансформація.** Вправи на безпосереднє застосування правила до одного слова у контексті.
  * *Формат:* Заповнення пропусків. Надається початкова форма в дужках, а учень повинен ввести правильну форму, здійснюючи необхідне чергування.
  * *Приклад:* Вона тримає телефон у (рука) _______.
  * *Приклад:* Я кожного дня (їздити) _______ на роботу.
* **Фаза 3: Продуктивне використання.** Написання відповідей на запитання, які природно провокують вживання чергувань.
  * *Формат:* «Опишіть, чим ви пишете і малюєте» (олівець → олівцем, фарба → фарбою). «Де стоїть ваза?» (полиця → на полиці).

**Методична порада:** Завжди відпрацьовуйте іменникові чергування (`г, к, х // з', ц', с'`) у тісній зв'язці з прийменниками місця (`у, в, на, при`). Дієслівні чергування варто відпрацьовувати у парах-контрастах: `він ходить — я ходжу`, щоб учень фізично відчував різницю між особами.

## Зв'язки з іншими темами
* **Передумови:** Тема вимагає чіткого розуміння поділу звуків на голосні/приголосні, тверді/м'які/шиплячі [S5], а також знання концепції відмінків (особливо Давального, Місцевого та Орудного). Необхідне вміння розрізняти основу слова та його закінчення [S1].
* **Що ця тема уможливлює:** Засвоєння цих правил є надійним фундаментом для повноцінного опанування місцевого та давального відмінків іменників, правильного творення форм наказового способу дієслів (сядь, ріж), а також розуміння процесів словотворення на вищих рівнях (Київ — київський, юнак — юнацький).
