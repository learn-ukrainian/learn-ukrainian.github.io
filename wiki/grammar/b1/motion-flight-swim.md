## Як це пояснюють у школі (How Schools Teach This)
Українські шкільні підручники вводять дієслова руху, як-от «летіти» та «пливти», поступово, спочатку фокусуючись на їхніх прямих значеннях, а згодом розширюючи до переносних.

Ключовим моментом є розрізнення між однонаправленим рухом (unidirectional), який триває в цей момент (**летіти, пливти**), та багатонаправленим (multidirectional) або здатністю до руху (**літати, плавати**). Ця відмінність є фундаментальною для українських дієслів руху.

Підручник Авраменка для 10 класу [S41] чітко визначає багатозначність дієслова **летіти**:
1.  **Пересування в повітрі за допомогою крил** (про птахів, комах).
2.  **Падіння** (летіти в улоговину).
3.  **Дуже швидке переміщення** (санки летять).
4.  **Швидкий плин часу** (дні летять).

Цей підхід, що рухається від конкретного до абстрактного, є стандартним. Крім того, у 10 класі пояснюється чергування голосних `[е] — [і]` на прикладі пари **летіти — літати** [S42], що закріплює розуміння різниці між цими двома дієсловами. Вже у 5 класі учнів вчать розрізняти спільнокореневі слова, де `летіти` і `літати` наводяться як приклад слів з різними, хоч і пов'язаними, значеннями, на відміну від `літній` [S43].

Дієслово **пливти** вводиться аналогічно: спочатку для опису руху по воді (човен), а потім для руху інших об'єктів (хмари) [S44][S45].

## Повна парадигма (Full Paradigm)
Ці дієслова належать до I дієвідміни.

### Летіти (Imperfective, Unidirectional)
| Особа | Однина | Множина |
| :--- | :--- | :--- |
| **1 (я, ми)** | леч**у** | лет**имо** |
| **2 (ти, ви)** | лет**иш** | лет**ите** |
| **3 (він/вона/воно, вони)** | лет**ить** | лет**ять** |
| **Минулий час** | лет**ів** / лет**іла** / лет**іло** | лет**іли** |
| **Наказовий спосіб** | — | лет**імо** |
| | лет**и** | лет**іть** |

### Полетіти (Perfective, Unidirectional)
*Утворюється додаванням префікса `по-`.*
| Особа | Однина | Множина |
| :--- | :--- | :--- |
| **Майбутній час** | полеч**у** | полет**имо** |
| | полет**иш** | полет**ите** |
| | полет**ить** | полет**ять** |
| **Минулий час** | полет**ів** / полет**іла** / полет**іло** | полет**іли** |

### Пливти / Плисти (Imperfective, Unidirectional)
*Дієслово має паралельні форми. «Пливти» є більш поширеним.*
| Особа | Однина | Множина |
| :--- | :--- | :--- |
| **1 (я, ми)** | плив**у** | плив**емо** |
| **2 (ти, ви)** | плив**еш** | плив**ете** |
| **3 (він/вона/воно, вони)** | плив**е** | плив**уть** |
| **Минулий час** | плив / пл**ив** | пл**и**ли |
| | пливл**а** / пл**ил**а | |
| | пливл**о** / пл**ил**о | |
| **Наказовий спосіб** | — | плив**імо** |
| | плив**и** | плив**іть** |

### Попливти / Поплисти (Perfective, Unidirectional)
*Утворюється додаванням префікса `по-`.*
| Особа | Однина | Множина |
| :--- | :--- | :--- |
| **Майбутній час** | поплив**у** | поплив**емо** |
| | поплив**еш** | поплив**ете** |
| | поплив**е** | поплив**уть** |
| **Минулий час** | попл**и**в / попл**ивл**а / попл**ивл**о | попл**и**ли |

## Частотність і пріоритети
Для рівня B1 пріоритетними є:
1.  **Imperfective Present:** `летить`, `летять`, `пливе`, `пливуть`. Це форми для опису поточної дії. *«Дивись, птах летить! Хмари пливуть по небу»*.
2.  **Perfective Past:** `полетів / полетіла`, `поплив / попливла`. Для позначення завершеного початку руху. *«Літак щойно полетів. Човен поплив від берега»*.
3.  **Imperfective Past:** `летів / летіла`, `плив / пливла`. Для опису тривалого руху в минулому. *«Ми летіли три години. Човен довго плив по річці»*.
4.  **Figurative Usage:** `час летить`, `думка летить`. Ці вирази є дуже поширеними та природними [S41][S46].
5.  **Basic Prefixes:** `вилетіти` (fly out), `прилетіти` (fly in), `відпливти` (sail away), `припливти` (sail in). Ці префікси є високочастотними та інтуїтивно зрозумілими [S47][S48].

Менш пріоритетними є складніші префікси (`пере-`, `з-`, `до-`) та наказовий спосіб, хоча їх розуміння є корисним.

## Типові помилки L2 (Common L2 Errors)
Англомовні студенти часто припускаються помилок через відсутність в англійській мові аналогічних граматичних категорій.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Птахи **літають** на південь. | Птахи **летять** на південь. | `Літати` означає здатність літати або рух у різних напрямках. `Летіти` — це рух в одному конкретному напрямку, як під час сезонної міграції. |
| Я **плавав** до того берега. | Я **плив** до того берега. | Аналогічно до `летіти/літати`. `Плавати` — це загальна здатність або рух без конкретного напрямку. `Пливти` — це рух до певної мети. |
| Час **літає**, коли весело. | Час **летить**, коли весело. | В українській мові для метафоричного позначення швидкого плину часу використовується саме однонаправлене дієслово `летіти` [S41][S49]. |
| Літак **летів** до Києва вчора (мається на увазі завершена дія). | Літак **прилетів** до Києва вчора. / Літак **літав** до Києва вчора. | `Летів` (недоконаний вид) описує процес. Для позначення завершеної дії (прибуття) потрібен доконаний вид (`прилетів`). Якщо йдеться про подорож туди й назад, використовується `літав`. |
| Ми **будемо пливти** на човні завтра (якщо це одноразова дія). | Ми **попливемо** на човні завтра. | Проста форма майбутнього часу доконаного виду (`попливемо`) є більш природною для опису одноразової запланованої дії в майбутньому, ніж складена форма недоконаного виду. |
| Хмари **плавають** по небу. | Хмари **пливуть** по небу. | Хоча хмари рухаються хаотично, в українській мові для опису їхнього руху традиційно використовується `пливти`, що підкреслює видимий, хоч і повільний, рух в одному загальному напрямку в певний момент спостереження [S45]. |

## Деколонізаційні застереження (Decolonization Notes)
1.  **Уникати порівнянь з російською мовою.** Українська система дієслів руху є самодостатньою. Не варто пояснювати її як "схожу на російську, але...". Російські аналоги (напр. `лететь/летать`, `плыть/плавать`) мають власні семантичні відтінки, і пряме перенесення може призвести до помилок. Українська мова — вихідна точка.
2.  **Різниця в парах `пливти/плавати` та `плыть/плавать`.** В українській мові `плавати` також може означати "плавати на кораблі, подорожувати морем", що менш характерно для російського `плавать` у цьому контексті. Навчальний матеріал має базуватися виключно на українських прикладах.
3.  **Форми минулого часу.** Українські форми чоловічого роду минулого часу дієслова `пливти` — `плив` (з нульовим закінченням), що відрізняється від російського аналога (`плыл`). Необхідно наголошувати на правильній українській формі.
4.  **Не використовувати російськомовні джерела чи приклади.** Усі приклади, вправи та контекст мають бути автентично українськими, щоб уникнути суржику та семантичних кальок. Наприклад, вживання `плавати` замість `пливти` у деяких контекстах може бути впливом російської мови.

## Природні приклади (Natural Examples)
**Група 1: `Летіти` — рух у повітрі (пряме значення)**
1.  У небі **летіли** гуси з далекого краю. [S41]
2.  Журавлі **летять** у теплий, але чужий край. [S50]
3.  Христина **вилетіла** з Нью-Йорка. [S47][S51]

**Група 2: `Летіти` — швидкий рух, падіння (переносне значення)**
4.  Рипить сніжок, співає, санки **летять** під білії намети ялин лапатих. [S41]
5.  Раптом він посунувся вниз і зрозумів, що **летить** в улоговину. [S41]
6.  Просто на них **летить** човен і ось-ось стукнеться з їхнім. [S52]

**Група 3: `Летіти` — плин часу, вітер, думки (абстрактне/метафоричне значення)**
7.  І в’янеш ти, а дні **летять**, несуть все добре за собою. [S41]
8.  Він [вітер] **летить** іздалеку, понад тихими селами. [S53]
9.  Пташка летить — думка **летить**. [S46]

**Група 4: `Пливти/Плисти` — рух по воді**
10. Човен плавко загойдався на волі, а далі тихо й рівно **посунув** [синонім до поплив] по воді. [S54]
11. Кущі шелюги росли у воді й тепер човен **плив** через зелену отару. [S44]
12. **Попливла** б я на схід сонця, тим шляхом, що проложило ясне сонце через воду. [S55]

**Група 5: `Пливти` — рух у повітрі чи просторі (переносне значення)**
13. Хмаринки **пливуть** у небесній блакиті. [S45]
14. Планета Квітів **пливла** під кораблем порожня й непривітна. [S56]
15. І **пливли** багрові хмари в вишині. [S44]

## Рекомендації для вправ (Activity Concepts)
**Phase 1: Основи (Core Meanings)**
*   **Вправа "Хто/Що летить? Хто/Що пливе?"**: Студенти доповнюють речення, вибираючи з банку слів (птах, літак, човен, риба, хмара, час, думка). Це закріплює базові асоціації.
*   **Заповнення пропусків**: Вправи на вибір правильної форми дієслова в теперішньому часі. *«Я (летіти) додому. Ми (пливти) на інший берег»*.

**Phase 2: Протиставлення (Contrast)**
*   **Вправа "Летіти чи Літати? Пливти чи Плавати?"**: Завдання на вибір правильного дієслова залежно від контексту (однонаправлений рух vs. здатність/багатонаправлений рух). *«Горобці (летять/літають) біля годівнички. Восени птахи (летять/літають) у вирій»*.
*   **Трансформація**: Перетворити речення з теперішнього часу в минулий доконаний. *«Човен пливе.» → «Човен поплив.»*

**Phase 3: Префікси та переносні значення (Prefixes & Figurative Use)**
*   **Вправа на добір префіксів**: Студенти додають префікси `при-`, `ви-`, `від-`, `до-` до дієслів `летіти` та `пливти`, щоб змінити значення речення.
*   **Творче завдання "Час летить"**: Студенти пишуть 3-4 речення про те, як швидко минає час, використовуючи дієслово `летіти` в переносному значенні.

## Зв'язки з іншими темами
*   **Дієслова руху (Verbs of Motion)**: `Летіти` і `пливти` є ключовими парами в цій великій темі, яка також включає `іти/ходити`, `їхати/їздити`.
*   **Вид дієслова (Aspect)**: Ці дієслова ілюструють творення доконаного виду за допомогою префіксів (`летіти` → `полетіти`, `прилетіти`).
*   **Відмінки (Cases)**: Вживання цих дієслів вимагає знання знахідного (`куди?` — `в/на` + Acc.) та родового (`звідки?` — `з/від` + Gen.) відмінків для позначення напрямку.
*   **Чергування голосних**: Пара `летіти — літати` є класичним прикладом чергування `[е] — [і]` в коренях дієслів [S42].
