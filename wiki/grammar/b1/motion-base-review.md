## Як це пояснюють у школі (How Schools Teach This)

Українська педагогічна традиція вводить дієслова руху через систему протиставлень, що допомагає учням зрозуміти їхню функціональність. Основні дихотомії:

1.  **Напрямок руху:** Це ключовий аспект. Дієслова поділяються на ті, що позначають **односпрямований** рух (в один бік, в конкретний момент) і **багатоспрямований** або **регулярний** рух (в обидва боки, повторювана дія, звичка) [S39][S40].
    *   Односпрямовані: `іти`, `їхати`, `бігти`.
    *   Багатоспрямовані: `ходити`, `їздити`, `бігати`.

2.  **Спосіб пересування:** Проводиться чітке розрізнення між рухом **пішки** та рухом **транспортом**.
    *   Пішки: `іти` / `ходити`, `бігти` / `бігати`.
    *   Транспортом: `їхати` / `їздити` [S39][S41]. Це стосується будь-якого транспорту: машина, автобус, поїзд, велосипед [S39][S42].

Шкільні підручники для 6-7 класів вводять правила дієвідмінювання та чергування приголосних, які є характерними для цих дієслів [S43][S44]. Вправи зазвичай фокусуються на виборі правильного дієслова залежно від контексту та на правильному утворенні особових форм [S45]. На наступному етапі вводяться префікси (`по-`, `при-`, `ви-` тощо), які додають дієсловам руху нових значень (відправлення, прибуття, вихід) і є основою для творення доконаного виду [S46][S47].

## Повна парадигма (Full Paradigm)

Дієслова руху в українській мові належать до І та ІІ дієвідмін і часто мають чергування приголосних в основі.

### 1. Рух пішки: іти / ходити

| Форма | Односпрямований (Unidirectional) | Багатоспрямований (Multidirectional) |
| :--- | :--- | :--- |
| **Інфінітив** | **іти́** (І дієвідміна) | **ходи́ти** (ІІ дієвідміна) |
| **Теперішній час** | | |
| я | іду́ | **ходжу́** (`д` → `дж`) |
| ти | іде́ш | хо́диш |
| він / вона / воно | іде́ | хо́дить |
| ми | ідемо́ | хо́димо |
| ви | ідете́ | хо́дите |
| вони | іду́ть | хо́дять |
| **Минулий час** | **йшов**, **йшла**, йшло, йшли | ходи́в, ходи́ла, ходи́ло, ходи́ли |
| **Майбутній час (НДВ)** | буду йти / йти́му | буду ходи́ти / ходи́тиму |
| **Майбутній час (ДВ)** | **піду́**, пі́деш, пі́де... (префікс `по-` або `при-` та ін.) | (утворюється від іншого дієслова, напр. `сходити`) |
| **Наказовий спосіб** | іди́, іді́мо, іді́ть | ходи́, ході́мо, ході́ть |

*Примітка:* Чергування `д` → `дж` у дієслові `ходити` відбувається тільки в 1-й особі однини [S44]. Минула форма `йшов` є суплетивною.

### 2. Рух транспортом: їхати / їздити

| Форма | Односпрямований (Unidirectional) | Багатоспрямований (Multidirectional) |
| :--- | :--- | :--- |
| **Інфінітив** | **ї́хати** (І дієвідміна) | **ї́здити** (ІІ дієвідміна) |
| **Теперішній час** | | |
| я | ї́ду | **ї́жджу** (`зд` → `ждж`) |
| ти | ї́деш | ї́здиш |
| він / вона / воно | ї́де | ї́здить |
| ми | ї́демо | ї́здимо |
| ви | ї́дете | ї́здять |
| вони | ї́дуть | ї́здять |
| **Минулий час** | ї́хав, ї́хала, ї́хало, ї́хали | ї́здив, ї́здила, ї́здило, ї́здили |
| **Майбутній час (НДВ)** | буду ї́хати / ї́хатиму | буду ї́здити / ї́здитиму |
| **Майбутній час (ДВ)** | **пої́ду**, пої́деш, пої́де... (префікс `по-`, `при-` та ін.) | (утворюється від іншого дієслова, напр. `з'їздити`) |
| **Наказовий спосіб** | їдь, ї́дьмо, ї́дьте | їздь, ї́здьмо, ї́здьте |

*Примітка:* Чергування `зд` → `ждж` у дієслові `їздити` відбувається тільки в 1-й особі однини [S48][S49].

### 3. Рух бігом: бігти / бігати

| Форма | Односпрямований (Unidirectional) | Багатоспрямований (Multidirectional) |
| :--- | :--- | :--- |
| **Інфінітив** | **бі́гти** (І дієвідміна, змішаний тип) | **бі́гати** (І дієвідміна) |
| **Теперішній час** | | |
| я | **біжу́** | бі́гаю |
| ти | **біжи́ш** | бі́гаєш |
| він / вона / воно | **біжи́ть** | бі́гає |
| ми | **біжимо́** | бі́гаємо |
| ви | **біжите́** | бі́гаєте |
| вони | **біжа́ть** | бі́гають |
| **Минулий час** | **біг**, **бі́гла**, бі́гло, бі́гли | бі́гав, бі́гала, бі́гало, бі́гали |
| **Майбутній час (НДВ)** | буду бі́гти / бі́гтиму | буду бі́гати / бі́гатиму |
| **Майбутній час (ДВ)** | **побіжу́**, побіжи́ш... (префікс `по-`, `при-` та ін.) | (утворюється від іншого дієслова, напр. `збігати`) |
| **Наказовий спосіб** | біжи́, біжі́мо, біжі́ть | бі́гай, бі́гаймо, бі́гайте |

*Примітка:* Дієслово `бігти` є складним. Хоча воно належить до І дієвідміни, в особових формах відбувається чергування `г` → `ж` [S43], а закінчення схожі на ІІ дієвідміну (`-иш`, `-ить`, `-ать`). У минулому часі чоловічого роду відсутній суфікс `-в` [S50].

## Частотність і пріоритети

Для рівня B1 пріоритети вивчення наступні:

1.  **Основна концепція:** Найважливіше — засвоїти функціональну різницю між односпрямованими (`іти`, `їхати`) та багатоспрямованими (`ходити`, `їздити`) дієсловами. Це фундаментальна концепція, без якої неможливе природне мовлення.
2.  **Форми теперішнього часу:** Усі шість дієслів є високочастотними в теперішньому часі. Особливу увагу слід приділити формам 1-ї особи однини (`ходжу`, `їжджу`, `біжу`) через їхні чергування.
3.  **Минулий час:** Форми минулого часу є обов'язковими для опису минулих подій. Форми `йшов`/`йшла` та `біг`/`бігла` потребують окремого запам'ятовування.
4.  **Простий майбутній час (доконаний):** Найбільш вживаними є форми з префіксами `по-` (початок/намір руху: `піду`, `поїду`) та `при-` (прибуття: `прийду`, `приїду`) [S47]. Вони є ключовими для вираження планів.
5.  **Складні форми майбутнього (недоконаний):** Форми `буду ходити` або `ходитиму` менш пріоритетні для активного вжитку на початковому етапі B1, але важливі для розуміння тривалих або повторюваних дій у майбутньому.

## Типові помилки L2 (Common L2 Errors)

Англомовні студенти часто роблять помилки через те, що в англійській мові одне дієслово "to go" охоплює багато значень, які в українській розрізняються.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я іду в спортзал щовівторка.` | `Я ходжу в спортзал щовівторка.` | Для регулярної, повторюваної дії використовується багатоспрямоване дієслово `ходити`, а не `іти`, що позначає конкретний рух в один бік зараз [S40]. |
| `Ми ідемо в Карпати на машині.` | `Ми їдемо в Карпати на машині.` | Якщо рух відбувається за допомогою будь-якого транспорту, обов'язково вживаються дієслова `їхати` або `їздити` [S39]. |
| `Вчора він бігв марафон.` | `Вчора він біг марафон.` | У дієсловах з основою інфінітива на приголосний (`біг-ти`, `нес-ти`) суфікс `-в` у формі минулого часу чоловічого роду випадає [S50]. |
| `Я ходю на роботу пішки.` | `Я ходжу на роботу пішки.` | В 1-й особі однини дієслів ІІ дієвідміни відбувається чергування приголосних: `д` → `дж` (`ходити` → `ходжу`), `зд` → `ждж` (`їздити` → `їжджу`) [S43][S48]. |
| `Завтра я буду йти в кіно.` | `Завтра я піду в кіно.` | Хоча граматично форма `буду йти` правильна, для вираження одноразової запланованої дії в майбутньому значно природніше і частіше вживається доконана форма `піду` [S47]. |

## Деколонізаційні застереження (Decolonization Notes)

При викладанні українських дієслів руху важливо уникати порівнянь з російською мовою як з нормою. Українська мова має власну, самодостатню систему.

1.  **Самостійність системи:** Система протиставлення односпрямованих/багатоспрямованих дієслів є спільною для багатьох слов'янських мов, але реалізація в кожній мові унікальна. Форми, наголоси та чергування в українських дієсловах (`їжджу`, `біжиш`) слід вивчати як норму української мови, а не як "відмінність від російської".
2.  **Наголос:** Український наголос є рухомим і часто не збігається з російським. Наприклад, у формах `несемо́`, `ведемо́`, `ідемо́` наголос падає на закінчення [S51]. Це потрібно вивчати як самостійне правило.
3.  **Лексика:** Слід послідовно використовувати питомо українську лексику. Наприклад, `квиток` на поїзд [S47], а не `білет`.
4.  **Фразеологія:** Використовуйте автентичні українські фразеологізми, що пов'язані з рухом, наприклад: `з усіх ніг` (дуже швидко бігти), `куди вода несе` (пасивно слідувати обставинам), `іти своєю дорогою` [S52].
5.  **Дієслово `бігти`:** Це дієслово має змішаний тип дієвідмінювання в українській мові. Це унікальна риса, а не "схожість" чи "відмінність" від російського аналога `бежать`, який також має свої особливості. Пояснювати його слід через внутрішню логіку української граматики.

## Природні приклади (Natural Examples)

Приклади згруповано за семантичною парою для демонстрації контрасту.

#### `іти` / `ходити` (пішки)

1.  **іти** (зараз, в один бік): `«Навесні в гайок піду, першу квіточку знайду»` [S53]. (про заплановану майбутню дію)
2.  **іти** (зараз, в один бік): `Я зараз іду в магазин, тобі щось купити?` <!-- VERIFY -->
3.  **ходити** (регулярно): `«Інколи я ходжу в кіно чи в ресторан»` [S40].
4.  **ходити** (регулярно): `Діти ходять до школи п'ять днів на тиждень.` <!-- VERIFY -->

#### `їхати` / `їздити` (транспортом)

5.  **їхати** (зараз, в один бік): `«Цього року ми з дружиною їдемо в Європу на Різдво»` [S39].
6.  **їхати** (послідовність поїздки): `«Спочатку ми їдемо в Угорщину в Будапешт, тоді в Словаччину»` [S39].
7.  **їздити** (регулярно, звичка): `«Оксано, Ви часто їздите у відпустку?»` [S39].
8.  **їздити** (регулярно, звичка): `«Зазвичай я їжджу на дачу влітку»` [S39].

#### `бігти` / `бігати` (бігом)

9.  **бігти** (зараз, в один бік): `«Хмарки біжать по небу вдалечінь»` [S54].
10. **бігти** (опис процесу): `«На першій доріжці біжить чемпіонка з легкої атлетики»` [S43].
11. **бігати** (регулярно, хобі): `«Якщо я вдома, то я бігаю в парку чи займаюся спортом»` [S40].
12. **бігати** (як діяльність): `«Я дуже люблю займатись спортом, наприклад... бігати»` [S42].

## Рекомендації для вправ (Activity Concepts)

Для ефективного засвоєння матеріалу рекомендується триетапний підхід:

*   **Фаза 1: Розпізнавання та вибір.**
    *   **Вправа "або/або":** Студенти обирають правильне дієслово з пари.
        *   _Приклад:_ `Минулого літа ми (їхали / їздили) в Карпати тричі.`
        *   _Приклад:_ `Дивись, до нас (іде / ходить) бабуся!`
    *   **Вправа "Пішки чи транспортом?":** Студенти заповнюють пропуски дієсловами `іти/ходити` або `їхати/їздити` залежно від контексту (`...на автобусі`, `...через парк`).

*   **Фаза 2: Відтворення форм.**
    *   **Дієвідмінювання в таблиці:** Заповнення таблиць дієвідмінювання для всіх шести дієслів у теперішньому часі.
    *   **Трансформація речень:** Переписати речення, змінивши особу або час.
        *   _Приклад:_ `Я ходжу на ринок. → Ми ... . Вони ... .`
        *   _Приклад:_ `Він біжить. → (Минулий час) Він ... . Вона ... .`

*   **Фаза 3: Вільне мовлення та письмо.**
    *   **Опис тижня:** Студенти описують свій типовий тиждень, використовуючи `ходити`, `їздити`, `бігати` для регулярних дій.
    *   **Планування подорожі:** Студенти складають план уявної подорожі, використовуючи доконані форми майбутнього часу (`поїдемо`, `підемо`, `прилетимо`).
        *   _Приклад:_ `«На канікулах ми поїдемо до Львова. Спочатку ми підемо на площу Ринок...»` (за зразком [S47]).

## Зв'язки з іншими темами

*   **Дієвідміни (Conjugations):** Дієслова руху є яскравими представниками І (`іти`, `їхати`, `бігати`) та ІІ (`ходити`, `їздити`) дієвідмін. `Бігти` — приклад змішаного типу, що є корисним для ілюстрації складності системи.
*   **Аспект дієслова (Verb Aspect):** Пари односпрямованих/багатоспрямованих дієслів є фундаментом для розуміння недоконаного виду. Додавання префіксів (`по-`, `при-`) до односпрямованих дієслів (`по+іти` → `піти`) є основним способом творення доконаного виду.
*   **Майбутній час (Future Tense):** Ці дієслова демонструють усі три форми майбутнього часу: просту (`піду`, `поїду`), складну (`йтиму`, `їхатиму`) і складену (`буду йти`, `буду їхати`).
*   **Відмінки (Cases):** Дієслова руху керують іменниками в різних відмінках для позначення напрямку (`в/у` + Знахідний), мети (`по` + Знахідний), засобу (`...автобусом` — Орудний).
