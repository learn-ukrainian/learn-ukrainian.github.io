## Як це пояснюють у школі

Українська шкільна програма підходить до теми пасивного стану поступово, розбиваючи її на кілька взаємопов'язаних етапів. Основна концепція закладається ще в 7 класі, де автори підручників (наприклад, Літвінова) фокусуються на вивченні активних і пасивних дієприкметників (participles), їхньому творенні та відмінюванні [S9]. Тоді ж учні вперше стикаються з поняттям безособових дієслів (impersonal verbs) та незмінюваних безособових форм на *-но, -то* [S10]. 

У 8 класі, під час вивчення синтаксису простого речення, автори підручників, такі як Авраменко, поглиблюють знання про безособові речення, які повідомляють про стан природи чи людини, можливість або неможливість дії, де підмета не може бути в принципі [S5], [S11].

У старшій школі (10–11 класи) тема розкривається на рівні стилістики та синтаксичної норми. Караман розглядає категорію перехідності (transitivity) та стану дієслова (active/passive voice), наголошуючи, що українська мова уникає пасивних конструкцій [S7]. Підручники для 11 класу приділяють значну увагу редагуванню русизмів і кальок, пояснюючи, що замість пасивних дієприкметників у ролі присудка слід уживати активні форми дієслова або форми на *-но, -то* [S6]. Школярів навчають, що центром уваги в українській мові є особистість, а не процес, тому речення на зразок *справа розглядається* є неприродними [S6].

## Повна парадигма

Система пасивного стану в українській мові реалізується через пасивні дієприкметники (passive participles) та похідні від них безособові форми на *-но, -то*.

### 1. Творення пасивних дієприкметників

Пасивні дієприкметники виражають ознаку предмета за дією, якої він зазнає з боку іншого виконавця [S9], [S11]. 

**Обмеження 1:** Пасивні дієприкметники утворюються виключно від **перехідних дієслів** (transitive verbs) доконаного та недоконаного виду [S9], [S11]. Від неперехідних дієслів утворити пасивну форму неможливо (наприклад, не існує форми від *спати* чи *іти*).
**Обмеження 2:** Від дієслів на *-ся* (зворотних) дієприкметники не утворюються [S9].

Творення відбувається суфіксальним способом за допомогою суфіксів *-н-*, *-ен-*, *-т-* [S9], [S15].

| Тип основи / Інфінітив | Ланцюжок перетворення | Приклад |
|---|---|---|
| Дієслова на *-ати*, *-яти*, *-увати* | `{інфінітив} → {основа} → {+н-} → {кінцева форма}` | `прочитати → прочита- → +н- → прочитаний` [S15] |
| | | `намалювати → намалюва- → +н- → намальований` [S15] |
| Односкладові основи на *-и*, *-і*, *-у*, *-я*, *-ер* | `{інфінітив} → {основа} → {+т-} → {кінцева форма}` | `мити → ми- → +т- → митий` [S15] |
| | | `взути → взу- → +т- → взутий` [S15] |
| | | `терти → тер- → +т- → тертий` [S15] |
| Дієслова на приголосний або *-и*, *-і* (з чергуванням) | `{інфінітив} → {основа} → {+ен-} → {кінцева форма}` | `пекти → пек- → +ен- → печений` [S9], [S15] |
| | | `зробити → зроб- → +л+ен- → зроблений` [S9] |
| | | `виїздити → виїзд- → +ж+ен- → виїжджений` [S15] |

*Примітка щодо паралельних форм:* Деякі дієслова можуть утворювати паралельні форми, наприклад: `кинути → кинутий` та `кинений`; `колоти → колотий` та `колений` [S15]. 

Відмінюються дієприкметники як прикметники твердої групи (за родами, числами, відмінками) [S9], [S11].

### 2. Безособові форми на -но, -то

Найбільш автентичним засобом вираження результату дії в українській мові є незмінювані форми на *-но, -то*. Вони функціонують як головний член (присудок) в односкладному безособовому реченні [S6], [S10]. 

**Обмеження 3:** У реченнях із формами на *-но, -то* **не буває і не може бути підмета** (subject) у називному відмінку [S10], [S13]. Об'єкт дії завжди стоїть у формі знахідного відмінка (accusative) без прийменника: *Справу (Зн.в.) зроблено* [S10].

| Похідний дієприкметник | Ланцюжок перетворення | Приклад безособової форми |
|---|---|---|
| Із суфіксом *-н-* | `{пасивний дієприкметник} → {відкидання закінчення -ий} → {+о} → {кінцева форма}` | `підписаний → підписан- → +о → підписано` [S6] |
| Із суфіксом *-ен-* | `{пасивний дієприкметник} → {відкидання закінчення -ий} → {+о} → {кінцева форма}` | `зроблений → зроблен- → +о → зроблено` [S10] |
| Із суфіксом *-т-* | `{пасивний дієприкметник} → {відкидання закінчення -ий} → {+о} → {кінцева форма}` | `розбитий → розбит- → +о → розбито` [S13] |

**Обмеження 4:** Заборонено вживати ці форми, якщо за логікою не може бути діяча (наприклад, *Дике поле встелено ромашками* — хибно, бо ніхто не застеляв поле спеціально; правильно — *встелене*) [S6].
**Обмеження 5:** Категорично заборонено вводити в такі речення діяча у формі орудного відмінка (instrumental case). Можна позначати лише знаряддя дії (*написано олівцем*) [S6].

### 3. Конструкції з дієсловами на -ся

**Обмеження 6:** Українська мова використовує пасивні конструкції з дієсловами на *-ся* здебільшого для зображення природних явищ або стану істоти (*Очі налилися кров’ю*) [S6]. 
**Обмеження 7:** Якщо дія виконується людиною, вживання дієслів на *-ся* в пасивному значенні з орудним відмінком діяча є грубим порушенням мовної норми [S6].

## Частотність і пріоритети

В українській мові концепція пасивного стану суттєво відрізняється від англійської (passive voice) чи російської (страдательный залог). Під час вивчення української мови (особливо на рівні B2) учні повинні засвоїти такі пріоритети:

1. **Активний стан — абсолютний пріоритет.** Якщо дія має відомого виконавця-людину, українська мова вимагає активної конструкції. Речення *Учні пишуть твори* є єдино правильним природним варіантом замість пасивного *Учнями пишуться твори* [S6].
2. **Форми на -но, -то для фокусу на результаті.** Коли діяч невідомий або не має значення, а важливий результат дії, слід використовувати конструкції на *-но, -то* (*Рішення ухвалено одноголосно*) [S6]. Це найчастотніша і найприродніша форма безособового пасиву в офіційному, науковому та публіцистичному стилях [S6].
3. **Пасивний дієприкметник як означення.** Пасивні дієприкметники найчастіше використовуються в ролі означення (attributes), а не присудка. Тобто: *Вимовлені слова зігріли своїм теплом* (означення), а не *Ці слова вимовлені тихо* (присудок). В останньому випадку краще сказати *Ці слова вимовлено тихо* [S6].

## Типові помилки L2 (англомовні учні)

Англомовні учні часто переносять структуру англійського пасивного стану (*The book was written by him*, *The house is being built*) в українську мову. Це призводить до грубих синтаксичних і стилістичних помилок.

| ❌ Помилково (L2 / Калька) | ✅ Правильно (Природно) | Чому |
|---|---|---|
| Ця проблема **досліджується вченими**. | Цю проблему **досліджують вчені**. | В українській мові не можна використовувати дієслова на *-ся* з діячем в орудному відмінку для позначення процесу. Акцент робиться на діячі в називному відмінку [S6], [S11]. |
| Іспит **складено студентом**. | Студент **склав** іспит. | Введення діяча (agent) у формі орудного відмінка до безособових конструкцій на *-но, -то* є грубим порушенням. Орудний відмінок тут можливий лише для знаряддя (наприклад, *написано ручкою*) [S6]. |
| Ухвала **прийнята** одноголосно. | Ухвалу **прийнято** (або **ухвалено**) одноголосно. | Коли потрібно наголосити на дії та її результаті, а не на постійній ознаці предмета, замість дієприкметника-присудка вживають незмінювані форми на *-но, -то* [S6]. |
| Двері були **відкриті мною**. | Я **відкрив** двері. | Пряме перенесення англійського *were opened by me*. Українська мова уникає пасивних конструкцій із зазначенням виконавця-особи через орудний відмінок [S7]. |
| Мій тато є **працюючим** пенсіонером. | Мій тато — пенсіонер, **який працює**. | Активні дієприкметники теперішнього часу на *-уч-/-юч-* не властиві сучасній українській мові; їх слід замінювати підрядними реченнями, іменниками або прикметниками [S9], [S12]. |
| Це дуже **хвилюючий** момент. | Це дуже **зворушливий** момент. | Форми на *-юч(ий)* слід замінювати природними прикметниками-синонімами (наприклад, *діючий закон* → *чинний закон*, *хвилюючий* → *зворушливий*) [S12]. |
| Хлопець, **прочитавший** книгу. | Хлопець, **який прочитав** книгу. | В українській мові взагалі не існує дієприкметників із суфіксом *-ш(ий)* або *-вш(ий)*. Це груба граматична помилка. Натомість використовується підрядне означальне речення [S9], [S12]. |

## Деколонізаційні застереження

Тема пасивного стану і дієприкметників є однією з найпроблемніших точок контакту між українською та російською мовами. Тривала русифікація призвела до появи в українському мовленні невластивих, штучних конструкцій. Автору модуля необхідно суворо уникати російських граматичних моделей:

1. **Абсолютна заборона на активні дієприкметники минулого часу.** Російські форми на *-вший*, *-ший* (*победивший*, *сказавший*) є органічними для російської, але їхніх прямих аналогів в українській літературній мові **не існує**. Будь-які спроби створити слова *перемігший*, *сказавший*, *навчавшийся* є суржиком [S9], [S12]. Їх замінюють підрядними реченнями (*той, що сказав*) або описовими конструкціями [S12].
2. **Уникнення активних дієприкметників теперішнього часу.** Російська мова масово використовує форми на *-ущий*, *-ящий* (*идущий*, *говорящий*, *моющий*). В українській мові такі форми (*ідучий*, *миючий*, *бажаючий*) вважаються калькою та стилістичною хибою. Їх необхідно замінювати на:
   - Іменники: *бажаючий* → *охочий*; *завідуючий* → *завідувач*; *оточуюче середовище* → *довкілля* [S9], [S12].
   - Віддієслівні прикметники: *миючий засіб* → *мийний засіб*; *знеболюючий* → *знеболювальний* [S9], [S12].
   - Підрядні речення: *учащийся* → *той, хто навчається* [S9].
3. **Пасивний стан процесу.** Російській мові притаманне використання зворотних дієслів у пасивному значенні з орудним відмінком діяча (*Дом строится рабочими*). Українська норма категорично відкидає цю модель як неприродну. Особливістю української мови є те, що в центрі уваги перебуває особистість, тому діяч ставиться в називний відмінок: *Робітники будують будинок*, *Депутати ухвалюють закони* (а не *Закони ухвалюються депутатами*) [S6].
4. **Неозначено-особові речення замість пасиву.** Російське *Вопрос обсуждается* українською звучить як *Питання обговорюють* або *Питання обговорюється* (без вказівки діяча), але найприродніше — активне неозначено-особове речення: *Питання обговорюють* [S6].

## Природні приклади

Нижче наведено приклади автентичного вживання пасивних і безособових конструкцій в українській мові. Усі вони демонструють правильну синтаксичну побудову без використання діяча в орудному відмінку.

**Безособові форми на -но, -то (результат дії):**

2. *Рішення ухвалено одноголосно.* [S6]
3. *Шаблю вибито з рук.* [S6]

6. *Уже тієї ночі драконів було зневогнено, а на зорі — розбито.* [S10]

**Використання орудного відмінка лише для знаряддя (інструмента):**

8. *Вершки збито міксером.* [S6]

## Рекомендації для вправ

Для якісного засвоєння пасивного стану англомовними учнями автор-письменник має дотримуватися такої методичної послідовності:

* **Фаза 1: Розпізнавання конструкцій та їхніх обмежень.**
  Вправи на ідентифікацію форми. Дати учням список речень і попросити позначити ті, що звучать неприродно українською мовою (наприклад, пошук орудного відмінка діяча в пасиві). 
  *Завдання:* Знайдіть помилку. *Книга прочитана учнем.* (Помилка: учнем; правильно: *Учень прочитав книгу*).

* **Фаза 2: Трансформація пасиву в актив.**
  Це ключова навичка для подолання англійської синтаксичної матриці (transfer). Учні повинні перетворювати англомовні або штучні пасивні конструкції на природні українські активні речення.
  *Завдання:* Перебудуйте речення. *Закони приймаються депутатами* → *Депутати приймають закони* [S6]. *This poem was written by Taras Shevchenko* → *Тарас Шевченко написав цей вірш*.

* **Фаза 3: Використання форм на -но, -то для фокусу на результаті.**
  Відпрацювання ситуацій, де діяч невідомий або не має значення (офіційні звіти, новини, рецепти). 
  *Завдання:* Утворіть безособові речення. *Хтось скосив траву* → *Траву скошено*. *Хтось розбив вікно* → *Вікно розбито*.

* **Фаза 4: Заміна «хворих» дієприкметників (кальок).**
  Вправи на лексичне збагачення, де учні вчаться уникати форм на *-уч-/-юч-* та замінювати їх прикметниками із суфіксами *-альн-*, *-н-*.
  *Завдання:* Оберіть правильний варіант. *Миючий засіб / Мийний засіб* [S12], *Знеболюючі пігулки / Знеболювальні пігулки* [S9].

## Зв'язки з іншими темами

**Передумови:**
- **Відмінки іменників:** Тема вимагає бездоганного володіння знахідним відмінком (accusative) без прийменника, оскільки він позначає прямий об'єкт як у перехідних дієсловах, так і в безособових формах на *-но, -то* (*зроблено справу*). Також учні мають розуміти функції орудного відмінка (instrumental), щоб розрізняти знаряддя дії (дозволено: *написано ручкою*) від суб'єкта дії (заборонено: *написано студентом*) [S6].
- **Вид дієслова:** Учні повинні розуміти різницю між доконаним (perfective) та недоконаним (imperfective) видом, оскільки більшість форм на *-но, -то*, що виражають результат, творяться від дієслів доконаного виду [S9].

**Наступні кроки:**
- **Синтаксис складного речення:** Оскільки українська мова уникає активних дієприкметників (participles like *the sleeping boy*, *the student reading a book*), учням знадобиться вміння вільно формувати складнопідрядні означальні речення зі сполучником *який* (relative clauses: *хлопець, який спить*, *студент, який читає книгу*) [S9], [S12].

## Пов'язані статті
- Безособові речення та стани природи
- Підрядні означальні речення (займенник «який»)
- Орудний відмінок: інструмент дії
- Віддієслівні прикметники та їхні суфікси
