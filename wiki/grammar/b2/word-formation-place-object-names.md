## Як це пояснюють у школі (How Schools Teach This)

Українські шкільні підручники не виділяють словотворення назв місць в окрему велику тему на рівні B2, а інтегрують його в загальніші розділи морфології та словотвору. Основний акцент робиться на розпізнаванні частин слова (кореня, суфікса, префікса) та способів творення нових слів (суфіксальний, префіксальний тощо), як показано в загальних оглядах словотвору [S34].

Ключовий підхід, який стосується назв місць, полягає в їх граматичній класифікації, що визначає їхню відмінювану поведінку. Наприклад, підручник Заболотного для 10 класу приділяє значну увагу правилам вибору закінчень родового відмінка (`-а/-я` чи `-у/-ю`) для іменників чоловічого роду II відміни. У цих правилах назви місць та об'єктів групуються за семантикою (значенням):

1.  **Назви будівель, приміщень та їхніх частин** отримують закінчення `-у (-ю)`: *вокзалу, коридору, інституту* [S35]. Це правило є фундаментальним для правильного вживання назв будівель. Водночас існують винятки, які теж вивчаються: *гаража, хліва, млина*.
2.  **Назви, що позначають місце, простір**, також отримують закінчення `-у (-ю)`: *яру, майдану, світу* [S35].
3.  **Назви населених пунктів** мають закінчення `-а (-я)`: *Києва, Парижа, Тернополя* [S35][S36].

Щодо самого процесу творення, то він демонструється через словотвірні ланцюжки. Наприклад, утворення іменника `продавець` від `продати`, а потім `продавчиня` від `продавець` [S37]. Хоча в наданих джерелах немає прямих прикладів ланцюжків для назв місць, цей метод є стандартним. Учень має зрозуміти, що нове слово (`похідне`) утворюється від вже існуючого (`твірного`) шляхом додавання словотворчих афіксів [S38].

Отже, шкільний підхід полягає не стільки у запам'ятовуванні списку суфіксів для назв місць, скільки у розвитку навички аналізувати структуру слова та розуміти граматичні наслідки, що випливають із семантичної категорії слова (тобто, "це слово означає місце, тому в родовому відмінку буде закінчення -у").

## Повна парадигма (Full Paradigm)

Нижче наведено таблиці основних продуктивних суфіксів, що використовуються для творення назв місць та об'єктів. Оскільки надані джерела не містять вичерпних правил для кожного суфікса, деякі моделі позначено як такі, що потребують верифікації.

### 1. Суфікси для місць діяльності, виробництва, торгівлі

Ці суфікси є надзвичайно продуктивними в сучасній мові для позначення магазинів, майстерень, закладів.

| Суфікс | Твірне слово (і його значення) | → | Похідне слово (рід, значення) | Приклад |
| :--- | :--- | :-: | :--- | :--- |
| **-арня** / **-ярня** | `книга` (book) | → | `книгарня` (ж.р., bookstore) | `<!-- VERIFY -->` |
| | `пекар` (baker) | → | `пекарня` (ж.р., bakery) | `<!-- VERIFY -->` |
| | `кава` (coffee) | → | `кав'ярня` (ж.р., café) | `<!-- VERIFY -->` |
| | `цукор` (sugar) | → | `цукерня` (ж.р., confectionery) | `<!-- VERIFY -->` |
| **-льня** | `їсти` (to eat) | → | `їдальня` (ж.р., dining room) | `<!-- VERIFY -->` |
| | `голити` (to shave) | → | `голильня` (ж.р., barbershop, заст.) | `<!-- VERIFY -->` |
| **-івня** / **-ївня** | `короп` (carp) | → | `коропівня` (ж.р., carp pond) | `<!-- VERIFY -->` |

### 2. Суфікси для просторів, територій, місць

Ці суфікси часто позначають великі, відкриті або спустошені простори.

| Суфікс | Твірне слово (і його значення) | → | Похідне слово (рід, значення) | Приклад |

| | `місто` (city) | → | `городище` (с.р., ancient settlement site) | `<!-- VERIFY -->` |
| **-исько** / **-їсько** | `пасовище` (pasture) | → | `пасовисько` (с.р., pasture, often vast) | `<!-- VERIFY -->` |

| **-овище** | `кладовище` (cemetery) `<!-- VERIFY -->` | | `кладовище` (с.р., cemetery) | `<!-- VERIFY -->` |

| Суфікс | Твірне слово (і його значення) | → | Похідне слово (рід, значення) | Приклад |
| :--- | :--- | :-: | :--- | :--- |

### 4. Суфікс для позначення процесу як місця

Суфікс `-ння` утворює іменники на позначення дії, але в певному контексті вони можуть означати місце, де ця дія відбувається.

| Суфікс | Твірне слово (і його значення) | → | Похідне слово (рід, значення) | Приклад |
| :--- | :--- | :-: | :--- | :--- |
| **-ування** | `паркувати` (to park) | → | `паркування` (с.р., parking, a place for parking) | [S40] |

## Частотність і пріоритети

Для рівня В2 пріоритетним є засвоєння найбільш продуктивних моделей, що зустрічаються у повсякденному житті та на вивісках.

1.  **Високий пріоритет (must know):**
    *   **-арня / -ярня**: Цей суфікс є надзвичайно продуктивним для назв закладів: *книгарня, перукарня, кав'ярня, зброярня, лікарня*. Розуміння цього суфікса дозволяє миттєво ідентифікувати призначення місця.
    *   **-ник**: Дуже поширений суфікс для назв предметів-вмістилищ: *смітник, чайник, молочник*.
    *   **Запозичені назви місць**: Учень В2 повинен вільно орієнтуватися в таких словах, як *готель, хостел, офіс, аеропорт, вокзал, університет, стадіон, супермаркет*. [S41][S35][S36].

2.  **Середній пріоритет (good to know):**
    *   **-ище / -исько**: Важливий для розуміння як розмовної мови, так і текстів про історію та географію. Значення може варіюватися від простору (*летовище* - аеродром) до місця руїни (*попелище*). Слово *селище* є дуже поширеним.
    *   **-льня**: Менш продуктивний, ніж `-арня`, але зустрічається в важливих словах: *їдальня, спальня, вбиральня*.
    *   **-иця**: Важливий для окремих слів: *криниця, теплиця, паляниця*.

3.  **Низький пріоритет (passive understanding):**
    *   **-овище**: Зустрічається в спеціалізованій лексиці (*торфовище*).
    *   **-івня**: Рідкісний суфікс, зазвичай у назвах, пов'язаних із сільським господарством. `<!-- VERIFY -->`

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я шукаю *місце для книг*. | Я шукаю **книгарню**. | Пряме калькування англійської конструкції "place for books" замість використання відповідного українського іменника, утвореного суфіксальним способом. Українська мова тяжіє до синтетизму. |
| На цьому місці було старе *будинкище*. | На цьому місці було старе **попелище** (якщо згорів) або **руйновище**. | Неправильне застосування суфікса `-ище`. Цей суфікс додається до основи, а не замінює її, і часто має специфічне значення (простір, руїна), а не просто "великий будинок". Для "великий будинок" краще сказати *величезний будинок* або *будівля*. |
| Моя улюблена *каварня* в центрі. | Моя улюблена **кав'ярня** в центрі. | Орфографічна помилка. При творенні від слова `кава` за допомогою суфікса `-ярня` виникає збіг губного приголосного [в] перед [я], що вимагає апострофа. |
| Ми зупинились у *готелі* Львову. | Ми зупинились у готелі **у Львові**. | Плутанина відмінків. Назви міст у конструкціях з `в/у` вимагають місцевого відмінка (`у Львові`), а не родового (`Львова`). Родовий відмінок тут би означав "готель, що належить Львову". |
| Цей новий *паркінг* дуже дорогий. | Це нове **паркування** дуже дороге. Або: Цей новий **паркінг** (ч.р.) дуже дорогий. | Неправильне визначення роду запозиченого слова. Хоча `паркінг` часто використовується в розмовній мові як іменник чоловічого роду, нормативним є слово `паркування` (середній рід), що означає як процес, так і місце [S40]. |
| У селі є стара *пекарка*. | У селі є стара **пекарня**. | Плутанина між назвою особи (`пекарка` - a female baker) та назвою місця (`пекарня` - a bakery). [S42]. |

## Деколонізаційні застереження (Decolonization Notes)

1.  **Продуктивність суфікса `-арня/-ярня` vs. російська модель `-ская`**: В українській мові для назв закладів і майстерень дуже поширений і продуктивний суфікс `-арня/-ярня` (*перукарня, книгарня, кав'ярня, друкарня*). У російській мові для цих самих понять часто використовується субстантивований прикметник жіночого роду із закінченням `-ская` (*парикмахерская, книжная [лавка], кофейня, типография*). Навчаючи, слід наголошувати на українській моделі як на самостійній і живій нормі, а не як на "відмінності від російської".

2.  **Слово `їдальня` vs. `столовая`**: Українське слово `їдальня` (від `їсти`) є питомим і стилістично нейтральним. Російське `столовая` (від `стол`) є калькою, яка, на жаль, поширена в суржику. Важливо активно вживати саме форму `їдальня`.

3.  **Назви населених пунктів**: Процес `декомунізації` та `дерусифікації` в Україні супроводжується поверненням історичних назв або наданням нових, українських назв населеним пунктам. Наприклад, *Кіровоград* став *Кропивницьким*, а *Дніпропетровськ* — *Дніпром* [S39]. Це важливий культурний контекст, який показує, що назви місць є частиною національної ідентичності.

4.  **Уникати паралелей з російськими суфіксами**: Не варто пояснювати українські суфікси через їхні російські аналоги (напр., "український `-ище` схожий на російський `-ище`"). Український словотвір має власну логіку, продуктивність і стилістичне забарвлення. Наприклад, український суфікс `-ище` може мати значення простору (*летовище*), тоді як в російській він переважно має збільшувально-згрубіле значення (*домище*).

5.  **Питомі слова для місць**: Слід заохочувати використання питомих українських слів для позначення місць там, де вони існують, замість бездумного запозичення: `летовище` (аеродром), `паркування` (паркінг), `книгарня` (книжковий магазин).

## Природні приклади (Natural Examples)

### Група 1: Назви закладів та приміщень у місті

-   На Софійському **майдані** — пам’ятник Богданові Хмельницькому. [S36]
-   Поруч розташований **університет**, теж названий іменем Шевченка. [S36]
-   Треба зауважувати про наявність місць для **велопаркування** біля шкіл, **університетів**, **магазинів**. [S40]
-   Моя улюблена **кав'ярня** знаходиться на першому поверсі старого будинку. `<!-- VERIFY -->`

### Група 2: Назви місць для подорожей та відпочинку

-   Христина буде бронювати кімнату в **садибі** в селі Шешори в Карпатах. [S41]
-   Інший популярний варіант — це **готель**: тризірковий, чотиризірковий, п'ятизірковий **готель**. [S41]
-   Ви можете зупинитися там в **кімнаті** інше житло яке ви можете зняти в Україні **санаторій**. [S41]

### Група 3: Назви населених пунктів

-   Після прийняття закону було перейменовано 32 **міста**, 955 **сіл** і **селищ**. [S39]
-   **Дніпропетровськ** став **Дніпром**, а **Кіровоград** — **Кропивницьким**. [S39]
-   Столиця Чехії — **місто Прага**. [S43]

### Група 4: Назви просторів та територій

-   На тому місці, де колись була хата, залишилось тільки чорне **попелище**. `<!-- VERIFY -->`
-   За селом розкинулось велике **пасовисько** для худоби. `<!-- VERIFY -->`
-   Археологи знайшли залишки давнього **городища** на березі річки. `<!-- VERIFY -->`

## Рекомендації для вправ (Activity Concepts)

### Phase 1: Ідентифікація та аналіз

-   **Вправа "Знайди місце"**: Дати учням текст (наприклад, опис міста чи подорожі) і попросити знайти всі іменники, що позначають місця. Учні мають визначити їхній рід, число, відмінок та, якщо можливо, твірне слово і суфікс.
-   **Вправа "Класифікація"**: Дати список назв місць (*книгарня, кладовище, вокзал, Київ, спальня, паркування, офіс*). Учні мають згрупувати їх: 1) Утворені суфіксальним способом; 2) Запозичені; 3) Власні назви.

### Phase 2: Кероване творення

-   **Вправа "Відкрий бізнес"**: Дати список слів, що позначають товари або професії (*квіти, ліки, цукерки, окуляри, зброя, пекар, взуття*). Учні мають утворити назви відповідних магазинів або майстерень, використовуючи суфікси `-арня`, `-ярня`, `-ня` або інші. (Наприклад: *квіти → квіткарня*).
-   **Вправа "Трансформація"**: Перетворити описові конструкції на один іменник.
    *   *Місце, де їдять → їдальня*
    *   *Кімната, де сплять → спальня*
    *   *Майстерня, де ремонтують годинники → годинникарня*
    *   *Місце, де багато сміття → звалище / смітник*

### Phase 3: Вільне вживання

-   **Вправа "Карта мого міста"**: Попросити учнів намалювати вигадану карту району та підписати на ній різні об'єкти, використовуючи утворені слова: *кав'ярня "Затишок", книгарня "Слово", дитячий майданчик, паркування*. Потім вони мають усно чи письмово описати маршрут від одного місця до іншого, правильно вживаючи місцевий та родовий відмінки.
-   **Вправа "Оголошення"**: Учні мають написати коротке оголошення про оренду приміщення або продаж будинку, описуючи його розташування та об'єкти поруч, використовуючи вивчену лексику.

## Зв'язки з іншими темами

-   **Попередні теми (Prerequisites):**
    *   [[grammar/a2/noun-gender|Рід іменників]]: Обов'язкова тема, оскільки рід похідного слова часто визначається суфіксом (напр., `-арня` → ж.р., `-ище` → с.р.).
    *   [[grammar/b1/noun-cases-locative|Місцевий відмінок]]: Назви місць найчастіше вживаються у місцевому відмінку з прийменниками `в/у` та `на` для позначення локації.
    *   [[grammar/b1/noun-cases-genitive|Родовий відмінок]]: Критично важливий для вживання назв місць після прийменників `до, з, біля` та для правильного творення форм іменників чоловічого роду (`-а/-я` vs `-у/-ю`). [S35].
    *   [[morphology/a2/word-structure|Будова слова]]: Базове розуміння кореня, суфікса, префікса є необхідним для аналізу словотвору.

-   **Наступні теми (What this enables):**
    *   [[grammar/b2/adjectives-relative|Відносні прикметники]]: Утворення прикметників від назв місць (*Київ → київський, університет → університетський*). [S44].
    *   [[grammar/b2/verbs-of-motion|Дієслова руху з префіксами]]: Складні конструкції з дієсловами руху (`приїхати до вокзалу`, `вийти з книгарні`, `пройти повз майдан`).
    *   [[vocabulary/c1/urban-studies|C1 Лексика: Урбаністика та архітектура]]: Глибше розуміння термінології, пов'язаної з міським простором, плануванням та історією.
