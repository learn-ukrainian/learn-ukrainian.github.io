## Як це пояснюють у школі (How Schools Teach This)
Словотвір прикметників є центральною темою в українській шкільній програмі, яку докладно вивчають у 6 класі. Підручники, такі як автори Заболотний, Літвінова, Голуб та Авраменко, представляють цю тему послідовно, починаючи з основних способів творення слів і поступово переходячи до складніших орфографічних правил.

1.  **Основи словотвору (Grade 6):** Тема починається з розрізнення **змінювання слів** (відмінювання) та **творення слів** (словотвір) [S42]. Учні знайомляться з основними способами творення: префіксальним, суфіксальним, префіксально-суфіксальним та складанням основ [S43][S44]. Суфіксальний спосіб визначається як найпоширеніший для прикметників [S44].

2.  **Суфіксальне творення (Grade 6):**
    *   **Відносні прикметники:** Учні вчаться утворювати відносні прикметники від іменників за допомогою суфіксів. Особлива увага приділяється суфіксам `-ськ-`, `-цьк-`, `-зьк-` та змінам приголосних, що відбуваються при їх додаванні (`г, з, ж + ськ = зьк`; `к, ц, ч + ськ = цьк`; `х, с, ш + ськ = ськ`) [S45][S46][S47]. Це правило відпрацьовується на прикладах географічних назв [S48].
    *   **Присвійні прикметники:** Пояснюється творення присвійних прикметників від іменників І та ІІ відмін за допомогою суфіксів `-ин`/`-їн` та `-ів`/`-їв`, `-ов` [S49][S45][S50].
    *   **Якісні прикметники:** Розглядається творення прикметників зі значенням здрібнілості, пестливості або збільшення (`-еньк-`, `-есеньк-`, `-езн-`, `-енн-`) [S51][S52].
    *   **Правила правопису суфіксів:** Вивчаються правила написання суфіксів `-ов-` та `-ев-`/`-єв-` залежно від кінцевого приголосного основи та наголосу [S46]. Для слів іншомовного походження вводиться правило «дев'ятки» для суфіксів `-ичн-`, `-ічн-`, `-їчн-` [S53].

3.  **Складні прикметники (Grade 6-7):** Учні вчаться правил написання складних прикметників **разом** (утворених від залежних слів, напр. `легкоатлетичний`) та **через дефіс** (утворених від незалежних слів, відтінків кольору, напр. `фізико-математичний`, `темно-сірий`) [S54][S55][S56]. Окремо розглядаються винятки (`жовтогарячий`, `військовополонений`).

4.  **Перехід між розрядами (Grade 6):** Підручники ілюструють, як прикметники можуть переходити з одного розряду в інший у переносному значенні, наприклад, відносний у якісний (`срібна ложка` → `срібний місяць`) або присвійний у якісний (`заячі вуха` → `заяча душа`) [S57][S58].

Загалом, шкільний підхід є дуже практичним, з великою кількістю вправ на утворення слів, трансформацію словосполучень (`журнал учителя` → `учителя журнал`) та відпрацювання орфографічних правил.

## Повна парадигма (Full Paradigm)

Словотвір прикметників в українській мові відбувається переважно чотирма основними способами: суфіксальним, префіксальним, префіксально-суфіксальним та складанням основ.

### 1. Суфіксальний спосіб

Це найпродуктивніший спосіб творення прикметників. Суфікс додається до твірної основи (іменника, іншого прикметника, дієслова).

| Тип прикметника | Суфікси | Твірне слово (Основа) | → | Утворений прикметник | Приклади |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Відносні** | `-н-` | `осінь`, `стіна` | → | `осінній`, `стінний` | `осінній день` [S49], `машинна олія` [S59] |
| | `-ов-` | `дощ`, `стиль` (наголос на закінченні) | → | `дощовий`, `стильовий` | `дощові краплі` [S58], `книжковий магазин` [S49] |
| | `-ев-` / `-єв-` | `липень`, `життя` (наголос на основі) | → | `липневий`, `життєвий` | `грушевий сік`, `яблуневий сад` |
| | `-ан-` / `-ян-` | `глина`, `срібло`, `солома` | → | `глиняний`, `срібний`, `солом’яний`| `срібні прикраси` [S49], `дерев'яний стілець` [S57] |
| **Присвійні** | `-ин-` / `-їн-` | `сестра` (І відм.), `Софія` | → | `сестрин`, `Софіїн` | `Маринчина стрічка` [S49], `доччин` [S45] |
| | `-ів` / `-їв` | `батько` (ІІ відм.), `Андрій` | → | `батьків`, `Андріїв` | `батьків спінінг` [S58], `Назарів план` [S50] |
| | `-ач-` / `-яч-` | `собака`, `кріль` | → | `собачий`, `кролячий` | `собача буда` [S50], `лисячий хвіст` [S58] |
| **Якісні** | `-еньк-` / `-есеньк-` | `малий`, `тонкий` | → | `маленький`, `тонесенький` | `маленький` [S45], `гарнесенький` [S60] |
| | `-езн-` / `-енн-` | `здоровий`, `високий` | → | `здоровенний`, `височенний`| `здоровенний` [S45], `широченний` [S59] |
| | `-уват-` / `-юват-` | `синій`, `білий` | → | `синюватий`, `білуватий` | `синюватий` [S51], `жовтувата квітка` [S61] |
| | `-ист-` / `-яст-` | `камінь`, `піна` | → | `кам'янистий`, `пінястий`| `барвистий` [S45] |
| | `-лив-` | `крик`, `хвороба` | → | `крикливий`, `хворобливий` | `крикливий` [S45], `щасливий` [S57] |
| | `-уч-` / `-юч-` | `злий` | → | `злющий` | `злющий` [S51] |
| **Від іншомовних слів** | `-ичн-` | `історія`, `демократ` (після `д, т, з, с, ц, ч, ш, ж, р`)| → | `історичний`, `демократичний`| `історичний музей` [S53] |
| | `-ічн-` | `географія`, `академія` | → | `географічний`, `академічний`| `академічний театр` [S53] |
| | `-їчн-` | `проза`, `герой` (після голосних) | → | `прозаїчний`, `героїчний` | `героїчний` [S53] |

#### Зміни приголосних при додаванні суфікса `-ськ-`
Це правило є ключовим для утворення прикметників від географічних назв, національностей та професій.

| Твірний приголосний | + `-ськ(ий)` | → | Результат | Приклади |
| :--- | :--- | :--- | :--- | :--- |
| `г`, `з`, `ж` | + `-ськ(ий)` | → | `-зьк(ий)` | `Прага` → `празький`, `Париж` → `паризький` [S45][S46] |
| `к`, `ч`, `ц` | + `-ськ(ий)` | → | `-цьк(ий)` | `козак` → `козацький`, `Кременчук` → `кременчуцький` [S45][S46] |
| `х`, `с`, `ш` | + `-ськ(ий)` | → | `-ськ(ий)` | `чех` → `чеський`, `Сиваш` → `сиваський` [S45][S46][S47] |
| *Інші приголосні* | + `-ськ(ий)` | → | *Без змін* | `Київ` → `київський`, `солдат` → `солдатський` [S48][S47] |
| **Винятки** | | | | `казахський`, `герцогський`, `тюркський` [S47] |

### 2. Префіксальний спосіб
Додавання префікса до прикметника для зміни його значення (зазвичай для підсилення якості).

| Префікс | + Прикметник | → | Новий прикметник | Значення |
| :--- | :--- | :--- | :--- | :--- |
| `пре-` | `добрий` | → | `предобрий` | Найвищий ступінь ознаки |
| `над-` | `звичайний` | → | `надзвичайний` | Понад міру |
| `без-` | `межі` (іменник) | → | `безмежний` | Відсутність чогось |
[S43]

### 3. Префіксально-суфіксальний спосіб
Одночасне додавання префікса і суфікса до твірної основи.

| Префікс | + Основа | + Суфікс | → | Новий прикметник |
| :--- | :--- | :--- | :--- | :--- |
| `при-` | `кордон` | `-н(ий)` | → | `прикордонний` |
| `без-` | `меж` | `-н(ий)` | → | `безмежний` |
| `на-` | `стіл` | `-н(ий)` | → | `настільний` |
[S43][S44]

### 4. Складання основ (Складні прикметники)
Поєднання двох основ для утворення нового прикметника.

| Спосіб | Правило написання | Приклади | Джерело |
| :--- | :--- | :--- | :--- |
| **Від залежних слів** (`словосполучення`) | **Разом** | `густолистий` (листя яке? густе) <br> `легкоатлетичний` <br> `вагоноремонтний` (ремонт чого? вагонів) | [S54] |
| **Від рівноправних слів** (`можна вставити "і"`) | **Через дефіс** | `німецько-польський` (німецький і польський) <br> `фізико-математичний` (фізика і математика) | [S54][S55] |
| **Відтінки кольору, поєднання кольорів, смаків** | **Через дефіс** | `темно-червоний`, `синьо-жовтий`, `кисло-солодкий` | [S54] |
| **Винятки для кольорів** (означають один колір) | **Разом** | `жовтогарячий`, `червоногарячий`, `золотогарячий` | [S55] |
| **Перша частина - числівник (словом)** | **Разом** | `п'ятизірковий`, `триповерховий` | [S54] |
| **Перша частина - числівник (цифрою)** | **Через дефіс** | `5-зірковий`, `3-поверховий` | [S56] |
| **Перша частина на `-ико`, `-іко`** | **Через дефіс** | `історико-культурний`, `механіко-математичний` | [S54] |
| **Повтор слів** | **Через дефіс** | `чистий-чистий`, `великий-превеликий` | [S54] |
| **Перша частина `військово-`, `воєнно-`** | **Через дефіс** | `військово-морський`, `воєнно-політичний` | [S54] |
| **Винятки** для `військово-` | **Разом** | `військовополонений`, `військовозобов'язаний` | [S55] |
| **Від складних іменників** | *Так само, як іменник* | `лісостеп` (разом) → `лісостеповий` (разом) | [S56] |

## Частотність і пріоритети
Для рівня B2 ключовим є не просто знання суфіксів, а впевнене володіння орфографічними та семантичними нюансами.

1.  **Пріоритет №1: Суфікс `-ськ-` та його варіації (`-цьк-`, `-зьк-`).** Це найважливіша група для вільного спілкування. Вона використовується для утворення прикметників від назв країн, міст, регіонів, національностей та соціальних груп (`український`, `київський`, `запорізький`, `козацький`). Бездоганне володіння правилами чергування приголосних є обов'язковим.

2.  **Пріоритет №2: Складні прикметники.** На рівні B2 учні повинні активно використовувати складні прикметники для надання мовленню більшої виразності та точності, особливо для опису кольорів, якостей та складних понять (`світло-рожевий`, `народнопісенний`, `південно-східний`). Важливо розрізняти написання разом і через дефіс.

3.  **Пріоритет №3: Присвійні прикметники (`-ин`, `-ів`).** Хоча механізм їх утворення простий, важливо вміти правильно їх використовувати, особливо в контексті сімейних стосунків та належності (`мамина сорочка`, `батьків капелюх`). Також важливо знати, коли присвійний прикметник переходить у відносний (`Шевченкові твори` vs `шевченківські дні`) [S50].

4.  **Пріоритет №4: Якісні суфікси для вираження емоцій (`-еньк-`, `-езн-`, `-уват-`).** Ці суфікси додають мовленню емоційного забарвлення та суб'єктивної оцінки, що є характерною рисою вільного володіння мовою (`смачненький`, `здоровенний`, `кислуватий`).

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я з Праги, я прагський.` | `Я з Праги, я празький.` | Англомовні учні часто пропускають чергування приголосних `г` → `з` перед суфіксом `-ськ-`. Це одне з найпоширеніших орфографічних ускладнень. [S45][S46] |
| `Це історичний і культурний центр.` | `Це історико-культурний центр.` | Тенденція перекладати англійську структуру "X and Y" дослівно, замість використання українського складного прикметника з дефісом, утвореного від рівноправних слів. [S54] |
| `Вінницький` → `Вінницькщина` | `Вінницький` → `Вінниччина` | При творенні іменників від прикметників на `-цький` відбувається зміна `-цьк-` → `-чч-`. Учні часто забувають про цю зміну, механічно додаючи суфікс `-ин(а)`. [S62][S47] |
| `Це грушовий сік.` (з наголосом на `грУ`) | `Це грушевий сік.` | Правило вибору суфікса `-ов-` чи `-ев-` залежить від наголосу та кінцевого приголосного основи. Якщо наголос падає на основу, що закінчується на м'який приголосний або шиплячий, використовується `-ев-`. [S46] |
| `соціально-політичний` | `соціально-політичний` (правильно), але `західноукраїнський` (разом) | Учні плутають правила написання складних прикметників. Якщо частини рівноправні (`соціальний і політичний`), пишемо через дефіс. Якщо одна частина залежить від іншої (`Західна Україна` → `західноукраїнський`), пишемо разом. [S54] |
| `Це Миколин зошит.` | `Це Миколин зошит.` (правильно), але `Це Костів зошит.` | Плутанина в утворенні присвійних прикметників. Іменники І відміни (`Микола`) утворюють прикметники із суфіксом `-ин`. Іменники ІІ відміни (`Кость`) — із суфіксом `-ів` (`-ов`). [S63] |

## Деколонізаційні застереження (Decolonization Notes)

1.  **Чергування приголосних vs. Російська мова:** Українська мова має унікальні та послідовні правила чергування приголосних перед суфіксом `-ськ-`. Наприклад, `Запоріжжя` → `запорізький` (`ж`→`зьк`), `Кременчук` → `кременчуцький` (`ч`→`цьк`). Російська мова має інші рефлекси: `Запорожье` → `запорожский`, `Кременчуг` → `кременчугский`. Представлення українських форм як самодостатньої, логічної системи, а не як "відмінностей від російської", є критично важливим.

2.  **Творення назв регіонів:** Суфікс `-ин(а)` для творення іменників-назв регіонів від прикметників на `-ський`, `-цький` є характерною рисою українського словотвору: `Чернігівський` → `Чернігівщина`, `Донецький` → `Донеччина`, `Словацький` → `Словаччина` [S64][S62]. Це продуктивна модель, що відрізняється від російських аналогів.

3.  **Присвійні суфікси:** Українська мова чітко розрізняє присвійні суфікси `-ин/-їн` (для І відміни: `Ольжин`, `Наталчин`) та `-ів/-їв` (для ІІ відміни: `Ігорів`, `батьків`). Російська мова використовує переважно суфікс `-ин` для обох випадків (`Ольгин`, `папин`). Неправильне вживання суфіксів є ознакою суржику.

4.  **Уникання кальок:** Слід уникати калькування російських складних прикметників, де в українській мові існують власні відповідники або віддається перевага словосполученням. Наприклад, замість кальки з рос. `трудоёмкий` слід вживати `трудомісткий` або описово `той, що потребує багато праці`.

5.  **Історичний контекст прізвищ:** Прізвища на `-енко` (типові для Наддніпрянщини) та `-ук/-чук` (типові для Західної України) є маркерами української ідентичності та історії, що відображають різні хвилі формування прізвищ у різних регіонах України [S65]. Це не просто суфікси, а частина культурного коду.

## Природні приклади (Natural Examples)

### Група 1: Утворення від географічних назв та національностей
- `Пластиковий посуд, красивий посуд, цікаві уроки, шкільні уроки, ... харківські вулиці, широкі вулиці.` [S49]
- `Прилуки → прилуцький; Золотоноша → золотоніський; чех → чеський.` [S59]
- `Жителі Європи називають Дунай «річкою дружби». Могутня річка прикрашає та напуває німецькі, австрійські, словацькі, угорські, болгарські, румунські й українські землі.` [S66]

### Група 2: Присвійні прикметники
- `Ми вже говорили про спосіб творення таких прикметників. Згадаймо суфікси й особливості правопису: Маринчина стрічка, Софіїн гребінець.` [S49]
- `Присвійні прикметники означають належність предмета людині або тварині, наприклад: батьків спінінг, лисячий хвіст.` [S58]
- `...і несучи вагу ту страшную буду пісню веселу співать.` (Приклад нестягненої форми `страшную` у поезії) [S67]

### Група 3: Складні прикметники
- `Мені подобаються темно-червоні мальви. А тобі? — А я обожнюю світло-рожеві…` [S54]
- `Німецько-польський (німецький і польський), вишнево-яблучний (вишневий і яблучний).` [S54]
- `Слова жовтогарячий, червоногарячий, золотогарячий пишемо разом, бо вони означають колір, а не поєднання кольорів.` [S55]
- `Він збирав український фольклор — це народні пісні, легенди, прислів'я і так далі.` (Приклад складного прикметника `народнопісенний` від словосполучення `народна пісня`) [S68]

### Група 4: Перехід прикметників в інший розряд
- `Відносні: залізні ворота, срібна ложка. Якісні: залізна воля, срібне волосся.` [S57]
- `Присвійні → якісні: ведмежа лапа — ведмежа послуга, собачий хвіст — собачий холод.` [S57]
- `Присвійні → відносні: орлиний дзьоб — орлиний ніс.` [S58]

### Група 5: Суфікси з емоційним забарвленням
- `Від поданих прикметників утворіть інші прикметники за допомогою одного із суфіксів -еньк-, -есеньк-, -ісіньк-, -юсіньк-.` [S52]
- `Прикметники зі значенням здрібнілості: малесенький, тонюсінький.` [S69]
- `Дмитро Павличко був таким представником інтелектуальної еліти в Україні і на щастя він прожив досить довге життя...` (Приклад прикметника `довгий` та його форми з підсиленням `довжелезний` (нема в тексті) або `дуже довгий`) [S70]

## Рекомендації для вправ (Activity Concepts)

- **Phase 1: Ідентифікація та аналіз.**
  - **Вправа "Знайди твірне слово":** Дати список прикметників (`морський`, `учнівський`, `прегарний`, `синьоокий`). Учні повинні визначити твірне слово (`море`, `учень`) та спосіб творення. (За зразком [S44]).
  - **Вправа "Розбір за будовою":** Дати прикметники (`різдвяний`, `центральний`, `міжбрівний`). Учні розбирають їх за будовою, виділяючи корінь, префікс, суфікс. (За зразком [S45]).

- **Phase 2: Контрольована практика.**
  - **Вправа "Трансформація: Іменник → Прикметник":** Дати список іменників, від яких потрібно утворити прикметники з певними суфіксами.
    - `(Суфікс -ськ-): Париж, Кременчук, Прага, козак.` → `паризький, кременчуцький, празький, козацький.` (За зразком [S45]).
    - `(Суфікс -ин-/-ів-): сестра, тато, Софія, Андрій.` → `сестрин, татів, Софіїн, Андріїв.` (За зразком [S50]).
  - **Вправа "Склади прикметник":** Дати пари слів для утворення складних прикметників. Учні повинні визначити, писати їх разом чи через дефіс.
    - `(синій, жовтий)` → `синьо-жовтий`
    - `(ремонт вагонів)` → `вагоноремонтний` (За зразком [S54]).

- **Phase 3: Вільне мовлення та письмо.**
  - **Вправа "Опиши предмет":** Дати зображення (напр., прапор, предмет одягу, пейзаж) і попросити описати його, використовуючи якомога більше складних прикметників для кольорів та якостей (`яскраво-червоний`, `сніжно-білий`, `кисло-солодкий`).
  - **Вправа "Моя вигадана країна":** Учні вигадують назву країни, її столиці та національності. Потім утворюють відповідні прикметники (`Утопія` → `утопійський`) і пишуть короткий текст про культуру цієї країни, використовуючи утворені слова.

## Зв'язки з іншими темами
- **Морфологія: Групи прикметників (B1):** Словотвір безпосередньо пов'язаний з розрядами прикметників (якісні, відносні, присвійні), оскільки різні суфікси зазвичай утворюють прикметники певного розряду. Розуміння переходу прикметників з розряду в розряд є ключовим.
- **Орфографія: Чергування приголосних (A2/B1):** Правила творення прикметників на `-ськ-`, `-цьк-`, `-зьк-` є практичним застосуванням правил чергування приголосних.
- **Синтаксис: Словосполучення (A1-B2):** Багато відносних прикметників є синонімічними до конструкцій "іменник + іменник у родовому відмінку" (`день осені` → `осінній день`). Уміння трансформувати такі конструкції робить мовлення більш гнучким.
- **Лексикологія: Складні слова (B2):** Творення складних прикметників є частиною ширшої теми про основоскладання в українській мові, яка також включає складні іменники.
