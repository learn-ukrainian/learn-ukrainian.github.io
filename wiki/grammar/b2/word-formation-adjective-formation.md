## Як це пояснюють у школі (How Schools Teach This)
Словотвір прикметників є центральною темою в українській шкільній програмі, яку докладно вивчають у 6 класі. Підручники, такі як автори Заболотний, Літвінова, Голуб та Авраменко, представляють цю тему послідовно, починаючи з основних способів творення слів і поступово переходячи до складніших орфографічних правил.

1.  **Основи словотвору (Grade 6):** Тема починається з розрізнення **змінювання слів** (відмінювання) та **творення слів** (словотвір) (Source 20). Учні знайомляться з основними способами творення: префіксальним, суфіксальним, префіксально-суфіксальним та складанням основ (Source 8, 43). Суфіксальний спосіб визначається як найпоширеніший для прикметників (Source 43).

2.  **Суфіксальне творення (Grade 6):**
    *   **Відносні прикметники:** Учні вчаться утворювати відносні прикметники від іменників за допомогою суфіксів. Особлива увага приділяється суфіксам `-ськ-`, `-цьк-`, `-зьк-` та змінам приголосних, що відбуваються при їх додаванні (`г, з, ж + ськ = зьк`; `к, ц, ч + ськ = цьк`; `х, с, ш + ськ = ськ`) (Source 7, 10, 42). Це правило відпрацьовується на прикладах географічних назв (Source 19).
    *   **Присвійні прикметники:** Пояснюється творення присвійних прикметників від іменників І та ІІ відмін за допомогою суфіксів `-ин`/`-їн` та `-ів`/`-їв`, `-ов` (Source 1, 7, 14).
    *   **Якісні прикметники:** Розглядається творення прикметників зі значенням здрібнілості, пестливості або збільшення (`-еньк-`, `-есеньк-`, `-езн-`, `-енн-`) (Source 9, 12).
    *   **Правила правопису суфіксів:** Вивчаються правила написання суфіксів `-ов-` та `-ев-`/`-єв-` залежно від кінцевого приголосного основи та наголосу (Source 10). Для слів іншомовного походження вводиться правило «дев'ятки» для суфіксів `-ичн-`, `-ічн-`, `-їчн-` (Source 41).

3.  **Складні прикметники (Grade 6-7):** Учні вчаться правил написання складних прикметників **разом** (утворених від залежних слів, напр. `легкоатлетичний`) та **через дефіс** (утворених від незалежних слів, відтінків кольору, напр. `фізико-математичний`, `темно-сірий`) (Source 21, 29, 37). Окремо розглядаються винятки (`жовтогарячий`, `військовополонений`).

4.  **Перехід між розрядами (Grade 6):** Підручники ілюструють, як прикметники можуть переходити з одного розряду в інший у переносному значенні, наприклад, відносний у якісний (`срібна ложка` → `срібний місяць`) або присвійний у якісний (`заячі вуха` → `заяча душа`) (Source 3, 4).

Загалом, шкільний підхід є дуже практичним, з великою кількістю вправ на утворення слів, трансформацію словосполучень (`журнал учителя` → `учителя журнал`) та відпрацювання орфографічних правил.

## Повна парадигма (Full Paradigm)

Словотвір прикметників в українській мові відбувається переважно чотирма основними способами: суфіксальним, префіксальним, префіксально-суфіксальним та складанням основ.

### 1. Суфіксальний спосіб

Це найпродуктивніший спосіб творення прикметників. Суфікс додається до твірної основи (іменника, іншого прикметника, дієслова).

| Тип прикметника | Суфікси | Твірне слово (Основа) | → | Утворений прикметник | Приклади |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Відносні** | `-н-` | `осінь`, `стіна` | → | `осінній`, `стінний` | `осінній день` (Source 1), `машинна олія` (Source 11) |
| | `-ов-` | `дощ`, `стиль` (наголос на закінченні) | → | `дощовий`, `стильовий` | `дощові краплі` (Source 4), `книжковий магазин` (Source 1) |
| | `-ев-` / `-єв-` | `липень`, `життя` (наголос на основі) | → | `липневий`, `життєвий` | `грушевий сік`, `яблуневий сад` |
| | `-ан-` / `-ян-` | `глина`, `срібло`, `солома` | → | `глиняний`, `срібний`, `солом’яний`| `срібні прикраси` (Source 1), `дерев'яний стілець` (Source 3) |
| **Присвійні** | `-ин-` / `-їн-` | `сестра` (І відм.), `Софія` | → | `сестрин`, `Софіїн` | `Маринчина стрічка` (Source 1), `доччин` (Source 7) |
| | `-ів` / `-їв` | `батько` (ІІ відм.), `Андрій` | → | `батьків`, `Андріїв` | `батьків спінінг` (Source 4), `Назарів план` (Source 14) |
| | `-ач-` / `-яч-` | `собака`, `кріль` | → | `собачий`, `кролячий` | `собача буда` (Source 14), `лисячий хвіст` (Source 4) |
| **Якісні** | `-еньк-` / `-есеньк-` | `малий`, `тонкий` | → | `маленький`, `тонесенький` | `маленький` (Source 7), `гарнесенький` (Source 15) |

| | `-уват-` / `-юват-` | `синій`, `білий` | → | `синюватий`, `білуватий` | `синюватий` (Source 9), `жовтувата квітка` (Source 44) |

| | `-лив-` | `крик`, `хвороба` | → | `крикливий`, `хворобливий` | `крикливий` (Source 7), `щасливий` (Source 3) |
| | `-уч-` / `-юч-` | `злий` | → | `злющий` | `злющий` (Source 9) |
| **Від іншомовних слів** | `-ичн-` | `історія`, `демократ` (після `д, т, з, с, ц, ч, ш, ж, р`)| → | `історичний`, `демократичний`| `історичний музей` (Source 41) |
| | `-ічн-` | `географія`, `академія` | → | `географічний`, `академічний`| `академічний театр` (Source 41) |
| | `-їчн-` | `проза`, `герой` (після голосних) | → | `прозаїчний`, `героїчний` | `героїчний` (Source 41) |

#### Зміни приголосних при додаванні суфікса `-ськ-`
Це правило є ключовим для утворення прикметників від географічних назв, національностей та професій.

| Твірний приголосний | + `-ськ(ий)` | → | Результат | Приклади |
| :--- | :--- | :--- | :--- | :--- |
| `г`, `з`, `ж` | + `-ськ(ий)` | → | `-зьк(ий)` | `Прага` → `празький`, `Париж` → `паризький` (Source 7, 10) |
| `к`, `ч`, `ц` | + `-ськ(ий)` | → | `-цьк(ий)` | `козак` → `козацький`, `Кременчук` → `кременчуцький` (Source 7, 10) |
| `х`, `с`, `ш` | + `-ськ(ий)` | → | `-ськ(ий)` | `чех` → `чеський`, `Сиваш` → `сиваський` (Source 7, 10, 42) |
| *Інші приголосні* | + `-ськ(ий)` | → | *Без змін* | `Київ` → `київський`, `солдат` → `солдатський` (Source 19, 42) |
| **Винятки** | | | | `казахський`, `герцогський`, `тюркський` (Source 42) |

### 2. Префіксальний спосіб
Додавання префікса до прикметника для зміни його значення (зазвичай для підсилення якості).

| Префікс | + Прикметник | → | Новий прикметник | Значення |
| :--- | :--- | :--- | :--- | :--- |
| `пре-` | `добрий` | → | `предобрий` | Найвищий ступінь ознаки |
| `над-` | `звичайний` | → | `надзвичайний` | Понад міру |
| `без-` | `межі` (іменник) | → | `безмежний` | Відсутність чогось |
(Source 8)

### 3. Префіксально-суфіксальний спосіб
Одночасне додавання префікса і суфікса до твірної основи.

| Префікс | + Основа | + Суфікс | → | Новий прикметник |
| :--- | :--- | :--- | :--- | :--- |
| `при-` | `кордон` | `-н(ий)` | → | `прикордонний` |
| `без-` | `меж` | `-н(ий)` | → | `безмежний` |
| `на-` | `стіл` | `-н(ий)` | → | `настільний` |
(Source 8, 43)

### 4. Складання основ (Складні прикметники)
Поєднання двох основ для утворення нового прикметника.

| Спосіб | Правило написання | Приклади | Джерело |
| :--- | :--- | :--- | :--- |
| **Від залежних слів** (`словосполучення`) | **Разом** | `густолистий` (листя яке? густе) <br> `легкоатлетичний` <br> `вагоноремонтний` (ремонт чого? вагонів) | (Source 21) |
| **Від рівноправних слів** (`можна вставити "і"`) | **Через дефіс** | `німецько-польський` (німецький і польський) <br> `фізико-математичний` (фізика і математика) | (Source 21, 29) |
| **Відтінки кольору, поєднання кольорів, смаків** | **Через дефіс** | `темно-червоний`, `синьо-жовтий`, `кисло-солодкий` | (Source 21) |
| **Винятки для кольорів** (означають один колір) | **Разом** | `жовтогарячий`, `червоногарячий`, `золотогарячий` | (Source 29) |
| **Перша частина - числівник (словом)** | **Разом** | `п'ятизірковий`, `триповерховий` | (Source 21) |
| **Перша частина - числівник (цифрою)** | **Через дефіс** | `5-зірковий`, `3-поверховий` | (Source 37) |
| **Перша частина на `-ико`, `-іко`** | **Через дефіс** | `історико-культурний`, `механіко-математичний` | (Source 21) |
| **Повтор слів** | **Через дефіс** | `чистий-чистий`, `великий-превеликий` | (Source 21) |
| **Перша частина `військово-`, `воєнно-`** | **Через дефіс** | `військово-морський`, `воєнно-політичний` | (Source 21) |
| **Винятки** для `військово-` | **Разом** | `військовополонений`, `військовозобов'язаний` | (Source 29) |
| **Від складних іменників** | *Так само, як іменник* | `лісостеп` (разом) → `лісостеповий` (разом) | (Source 37) |

## Частотність і пріоритети
Для рівня B2 ключовим є не просто знання суфіксів, а впевнене володіння орфографічними та семантичними нюансами.

1.  **Пріоритет №1: Суфікс `-ськ-` та його варіації (`-цьк-`, `-зьк-`).** Це найважливіша група для вільного спілкування. Вона використовується для утворення прикметників від назв країн, міст, регіонів, національностей та соціальних груп (`український`, `київський`, `запорізький`, `козацький`). Бездоганне володіння правилами чергування приголосних є обов'язковим.

2.  **Пріоритет №2: Складні прикметники.** На рівні B2 учні повинні активно використовувати складні прикметники для надання мовленню більшої виразності та точності, особливо для опису кольорів, якостей та складних понять (`світло-рожевий`, `народнопісенний`, `південно-східний`). Важливо розрізняти написання разом і через дефіс.

3.  **Пріоритет №3: Присвійні прикметники (`-ин`, `-ів`).** Хоча механізм їх утворення простий, важливо вміти правильно їх використовувати, особливо в контексті сімейних стосунків та належності (`мамина сорочка`, `батьків капелюх`). Також важливо знати, коли присвійний прикметник переходить у відносний (`Шевченкові твори` vs `шевченківські дні`) (Source 14).

4.  **Пріоритет №4: Якісні суфікси для вираження емоцій (`-еньк-`, `-езн-`, `-уват-`).** Ці суфікси додають мовленню емоційного забарвлення та суб'єктивної оцінки, що є характерною рисою вільного володіння мовою (`смачненький`, `здоровенний`, `кислуватий`).

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я з Праги, я прагський.` | `Я з Праги, я празький.` | Англомовні учні часто пропускають чергування приголосних `г` → `з` перед суфіксом `-ськ-`. Це одне з найпоширеніших орфографічних ускладнень. (Source 7, 10) |
| `Це історичний і культурний центр.` | `Це історико-культурний центр.` | Тенденція перекладати англійську структуру "X and Y" дослівно, замість використання українського складного прикметника з дефісом, утвореного від рівноправних слів. (Source 21) |
| `Вінницький` → `Вінницькщина` | `Вінницький` → `Вінниччина` | При творенні іменників від прикметників на `-цький` відбувається зміна `-цьк-` → `-чч-`. Учні часто забувають про цю зміну, механічно додаючи суфікс `-ин(а)`. (Source 24, 42) |
| `Це грушовий сік.` (з наголосом на `грУ`) | `Це грушевий сік.` | Правило вибору суфікса `-ов-` чи `-ев-` залежить від наголосу та кінцевого приголосного основи. Якщо наголос падає на основу, що закінчується на м'який приголосний або шиплячий, використовується `-ев-`. (Source 10) |
| `соціально-політичний` | `соціально-політичний` (правильно), але `західноукраїнський` (разом) | Учні плутають правила написання складних прикметників. Якщо частини рівноправні (`соціальний і політичний`), пишемо через дефіс. Якщо одна частина залежить від іншої (`Західна Україна` → `західноукраїнський`), пишемо разом. (Source 21) |
| `Це Миколин зошит.` | `Це Миколин зошит.` (правильно), але `Це Костів зошит.` | Плутанина в утворенні присвійних прикметників. Іменники І відміни (`Микола`) утворюють прикметники із суфіксом `-ин`. Іменники ІІ відміни (`Кость`) — із суфіксом `-ів` (`-ов`). (Source 35) |

## Деколонізаційні застереження (Decolonization Notes)

1.  **Чергування приголосних vs. Російська мова:** Українська мова має унікальні та послідовні правила чергування приголосних перед суфіксом `-ськ-`. Наприклад, `Запоріжжя` → `запорізький` (`ж`→`зьк`), `Кременчук` → `кременчуцький` (`ч`→`цьк`). Російська мова має інші рефлекси: `Запорожье` → `запорожский`, `Кременчуг` → `кременчугский`. Представлення українських форм як самодостатньої, логічної системи, а не як "відмінностей від російської", є критично важливим.

2.  **Творення назв регіонів:** Суфікс `-ин(а)` для творення іменників-назв регіонів від прикметників на `-ський`, `-цький` є характерною рисою українського словотвору: `Чернігівський` → `Чернігівщина`, `Донецький` → `Донеччина`, `Словацький` → `Словаччина` (Source 22, 24). Це продуктивна модель, що відрізняється від російських аналогів.

3.  **Присвійні суфікси:** Українська мова чітко розрізняє присвійні суфікси `-ин/-їн` (для І відміни: `Ольжин`, `Наталчин`) та `-ів/-їв` (для ІІ відміни: `Ігорів`, `батьків`). Російська мова використовує переважно суфікс `-ин` для обох випадків (`Ольгин`, `папин`). Неправильне вживання суфіксів є ознакою суржику.

4.  **Уникання кальок:** Слід уникати калькування російських складних прикметників, де в українській мові існують власні відповідники або віддається перевага словосполученням. Наприклад, замість кальки з рос. `трудоёмкий` слід вживати `трудомісткий` або описово `той, що потребує багато праці`.

5.  **Історичний контекст прізвищ:** Прізвища на `-енко` (типові для Наддніпрянщини) та `-ук/-чук` (типові для Західної України) є маркерами української ідентичності та історії, що відображають різні хвилі формування прізвищ у різних регіонах України (Source 32). Це не просто суфікси, а частина культурного коду.

## Природні приклади (Natural Examples)

### Група 1: Утворення від географічних назв та національностей
- `Пластиковий посуд, красивий посуд, цікаві уроки, шкільні уроки, ... харківські вулиці, широкі вулиці.` (Source 1)
- `Прилуки → прилуцький; Золотоноша → золотоніський; чех → чеський.` (Source 11)
- `Жителі Європи називають Дунай «річкою дружби». Могутня річка прикрашає та напуває німецькі, австрійські, словацькі, угорські, болгарські, румунські й українські землі.` (Source 36)

### Група 2: Присвійні прикметники
- `Ми вже говорили про спосіб творення таких прикметників. Згадаймо суфікси й особливості правопису: Маринчина стрічка, Софіїн гребінець.` (Source 1)
- `Присвійні прикметники означають належність предмета людині або тварині, наприклад: батьків спінінг, лисячий хвіст.` (Source 4)
- `...і несучи вагу ту страшную буду пісню веселу співать.` (Приклад нестягненої форми `страшную` у поезії) (Source 33)

### Група 3: Складні прикметники
- `Мені подобаються темно-червоні мальви. А тобі? — А я обожнюю світло-рожеві…` (Source 21)
- `Німецько-польський (німецький і польський), вишнево-яблучний (вишневий і яблучний).` (Source 21)
- `Слова жовтогарячий, червоногарячий, золотогарячий пишемо разом, бо вони означають колір, а не поєднання кольорів.` (Source 29)
- `Він збирав український фольклор — це народні пісні, легенди, прислів'я і так далі.` (Приклад складного прикметника `народнопісенний` від словосполучення `народна пісня`) (Source 17)

### Група 4: Перехід прикметників в інший розряд
- `Відносні: залізні ворота, срібна ложка. Якісні: залізна воля, срібне волосся.` (Source 3)
- `Присвійні → якісні: ведмежа лапа — ведмежа послуга, собачий хвіст — собачий холод.` (Source 3)
- `Присвійні → відносні: орлиний дзьоб — орлиний ніс.` (Source 4)

### Група 5: Суфікси з емоційним забарвленням
- `Від поданих прикметників утворіть інші прикметники за допомогою одного із суфіксів -еньк-, -есеньк-, -ісіньк-, -юсіньк-.` (Source 12)
- `Прикметники зі значенням здрібнілості: малесенький, тонюсінький.` (Source 39)
- `Дмитро Павличко був таким представником інтелектуальної еліти в Україні і на щастя він прожив досить довге життя...` (Приклад прикметника `довгий` та його форми з підсиленням `довжелезний` (нема в тексті) або `дуже довгий`) (Source 31)

## Рекомендації для вправ (Activity Concepts)

- **Phase 1: Ідентифікація та аналіз.**
  - **Вправа "Знайди твірне слово":** Дати список прикметників (`морський`, `учнівський`, `прегарний`, `синьоокий`). Учні повинні визначити твірне слово (`море`, `учень`) та спосіб творення. (За зразком Source 43).
  - **Вправа "Розбір за будовою":** Дати прикметники (`різдвяний`, `центральний`, `міжбрівний`). Учні розбирають їх за будовою, виділяючи корінь, префікс, суфікс. (За зразком Source 7).

- **Phase 2: Контрольована практика.**
  - **Вправа "Трансформація: Іменник → Прикметник":** Дати список іменників, від яких потрібно утворити прикметники з певними суфіксами.
    - `(Суфікс -ськ-): Париж, Кременчук, Прага, козак.` → `паризький, кременчуцький, празький, козацький.` (За зразком Source 7).
    - `(Суфікс -ин-/-ів-): сестра, тато, Софія, Андрій.` → `сестрин, татів, Софіїн, Андріїв.` (За зразком Source 14).
  - **Вправа "Склади прикметник":** Дати пари слів для утворення складних прикметників. Учні повинні визначити, писати їх разом чи через дефіс.
    - `(синій, жовтий)` → `синьо-жовтий`
    - `(ремонт вагонів)` → `вагоноремонтний` (За зразком Source 21).

- **Phase 3: Вільне мовлення та письмо.**
  - **Вправа "Опиши предмет":** Дати зображення (напр., прапор, предмет одягу, пейзаж) і попросити описати його, використовуючи якомога більше складних прикметників для кольорів та якостей (`яскраво-червоний`, `сніжно-білий`, `кисло-солодкий`).
  - **Вправа "Моя вигадана країна":** Учні вигадують назву країни, її столиці та національності. Потім утворюють відповідні прикметники (`Утопія` → `утопійський`) і пишуть короткий текст про культуру цієї країни, використовуючи утворені слова.

## Зв'язки з іншими темами
- **Морфологія: Групи прикметників (B1):** Словотвір безпосередньо пов'язаний з розрядами прикметників (якісні, відносні, присвійні), оскільки різні суфікси зазвичай утворюють прикметники певного розряду. Розуміння переходу прикметників з розряду в розряд є ключовим.
- **Орфографія: Чергування приголосних (A2/B1):** Правила творення прикметників на `-ськ-`, `-цьк-`, `-зьк-` є практичним застосуванням правил чергування приголосних.
- **Синтаксис: Словосполучення (A1-B2):** Багато відносних прикметників є синонімічними до конструкцій "іменник + іменник у родовому відмінку" (`день осені` → `осінній день`). Уміння трансформувати такі конструкції робить мовлення більш гнучким.
- **Лексикологія: Складні слова (B2):** Творення складних прикметників є частиною ширшої теми про основоскладання в українській мові, яка також включає складні іменники.
