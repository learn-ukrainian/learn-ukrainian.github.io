## Як це пояснюють у школі (How Schools Teach This)

В українських школах поняття абстрактних іменників вводять доволі рано, зазвичай у 4-6 класах. Основний підхід полягає в протиставленні абстрактних іменників конкретним.

- **Конкретні іменники** позначають предмети та явища, які можна сприймати органами чуття: побачити, почути, доторкнутися [S53]. Приклади: `стіл`, `дощ`, `сорочка`, `дівчинка` [S54][S55].
- **Абстрактні іменники** називають поняття, явища, властивості, які не можна сприйняти органами чуття [S53][S54].

Підручник для 4 класу [S56] дає детальну класифікацію значень, які передають абстрактні іменники:
*   **Стан:** `сон`, `тиша`
*   **Почуття:** `кохання`, `ненависть`, `радість`
*   **Процес:** `хід`, `біг`, `змагання`
*   **Якість:** `чорнота`, `ясність`, `сміливість`
*   **Риси характеру:** `доброта`, `злоба`, `вихованість`
*   **Вияви інтелекту:** `знання`, `розум`, `мудрість`
*   **Відносини:** `дружба`, `мир`
*   **Поняття етикету:** `вітання`, `повага`

У старших класах (7-10) акцент зміщується на **словотвір** — як саме ці абстрактні іменники утворюються від інших частин мови, зокрема за допомогою суфіксів [S57][S58]. Ці суфікси часто мають книжне або стилістичне забарвлення, що є важливим для офіційно-ділового та наукового стилів [S59].

## Повна парадигма (Full Paradigm)

Абстрактні іменники в українській мові переважно є похідними й утворюються від дієслів, прикметників та інших іменників. Нижче наведено основні словотвірні моделі.

### 1. Відприкметникові іменники (from Adjectives)

Це найпродуктивніший спосіб творення абстрактних іменників, що позначають якість або властивість.

| Суфікс | Основа (Прикметник) | → | Абстрактний іменник | Приклади |
| :--- | :--- | :--- | :--- | :--- |
| **-ість** | `мудрий` | → | `мудрість` | `швидкість`, `щирість`, `незалежність`, `відданість` [S53], `можливість`, `небезпека` -> `небезпечність` |
| **-от(а)** | `добрий` | → | `доброта` | `чорнота`, `простота` [S60], `сліпота`, `теплота` |
| **-ин(а)** | `високий` | → | `височина` | `глибина`, `ширина`, `далина` [S61] |
| **безафіксний** | `зелений` | → | `зелень` | `синь`, `вись` [S62], `тиша` (`тихий`) |

### 2. Віддієслівні іменники (from Verbs)

Ці іменники позначають опредметнену дію, процес або результат. Вони є ключовими для наукового та офіційно-ділового стилів.

| Суфікс | Основа (Дієслово) | → | Абстрактний іменник | Приклади та правила |
| :--- | :--- | :--- | :--- | :--- |
| **-ння** | `вживати` → `вжива-` | → | `вживання` | `**-анн(я) / -янн(я)**` для дієслів на `-ати / -яти`: `читати → читання`, `змагатися → змагання`, `сприяти → сприяння` [S63]. Наголос зазвичай на суфіксі. |
| | `звільнити` → `звільн-` | → | `звільнення` | `**-енн(я) / -єнн(я)**` для дієслів на `-ити`, `-іти` та з приголосним в основі: `здійснити → здійснення`, `повернути → повернення`, `вивчити → вивчення`. Наголос зазвичай на корені [S63]. |
| | `ходити` → `ход-` | → | `ходіння` | `**-інн(я)**` для дієслів, де в основі інфінітива є `и` або `і`, або під наголосом: `горіти → горіння`, `носити → носіння` [S64][S65]. |
| **-ття** | `жити` | → | `життя` | Утворює іменники від дієслів з основою на голосний: `забути → забуття`, `прийняти → прийняття`, `сприйняти → сприйняття`. |
| **-б(а)** | `боротися` | → | `боротьба` | `просити → просьба`, `служити → служба` |
| **-ок** | `розвивати` | → | `розвиток` | `почати → початок`, `жартувати → жарто`к |
| **безафіксний** | `виходити` | → | `вихід` | `пошукати → пошук` [S62], `міняти → обмін`, `кричати → крик`, `бігти → біг` [S56]. |

### 3. Відіменникові іменники (from Nouns)

Ці іменники позначають узагальнені, збірні або абстрактні поняття, ідеології.

| Суфікс | Основа (Іменник) | → | Абстрактний іменник | Приклади |
| :--- | :--- | :--- | :--- | :--- |
| **-ств(о)** / **-цтв(о)** | `брат`, `людина` | → | `братство`, `людство` | `товариство` [S60], `суспільство`, `керівництво`, `мистецтво` |
| **-ізм** / **-изм** | `патріот` | → | `патріотизм` | `капіталізм`, `реалізм` [S66], `націоналізм` [S58] |

## Частотність і пріоритети (Frequency & Priorities)

Для рівня B2 пріоритети у вивченні та використанні абстрактних іменників такі:

1.  **Найвища пріоритетність:**
    *   Іменники на **-ість** (від прикметників): надзвичайно продуктивний суфікс, що використовується в усіх стилях мови для позначення якостей та властивостей. Це основа для вираження абстрактних думок. Приклади: `можливість`, `відповідальність`, `незалежність`, `справедливість`.
    *   Іменники на **-ння** (віддієслівні): ключові для розуміння та продукування текстів наукового, публіцистичного та ділового стилів. На рівні В2 учні активно працюють з такими текстами. Приклади: `використання`, `рішення`, `навчання`, `знищення` [S67].

2.  **Середня пріоритетність:**
    *   Іменники на **-ств(о) / -цтв(о)**: важливі для обговорення соціальних, культурних та особистих понять. Приклади: `суспільство`, `мистецтво`, `дитинство`, `боягузтво` [S60].
    *   **Безафіксний спосіб:** Багато високочастотних слів утворено саме так. Їх потрібно засвоювати як лексичні одиниці. Приклади: `біль`, `гнів`, `сум`, `вплив`, `попит`.
    *   Іменники на **-от(а)**: поширені для опису якостей, часто емоційно забарвлених. Приклади: `доброта`, `красота`, `турбота` [S60].

3.  **Нижча пріоритетність (для розпізнавання):**
    *   Іменники на **-ізм / -изм**: важливі для розуміння текстів на суспільно-політичні та культурні теми, але менш продуктивні у власному мовленні учня. Приклади: `капіталізм`, `імперіалізм` [S68], `романтизм`.
    *   Рідкісні суфікси, як-от `-ин(а)` (`височина`) або `-б(а)` (`боротьба`), слід вивчати в контексті конкретних лексичних одиниць, а не як продуктивну модель.

## Типові помилки L2 (Common L2 Errors)

Англомовні учні часто роблять помилки, пов'язані з переносом структур рідної мови та неправильним утворенням форм.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| *Його хоробрість була очевидна.* | *Його **хоробрість** була очевидною.* (Орудний відмінок) | Помилка в керуванні. Іменна частина складеного присудка після дієслова `бути` в минулому/майбутньому часі часто ставиться в орудному відмінку для позначення тимчасової ознаки. |
| *Я маю **відношення** до цієї організації.* | *Я маю **стосунок** до цієї організації.* | Калька з російської мови (`иметь отношение`). `Відношення` в українській мові переважно означає математичне або логічне співвідношення [S69]. Для позначення зв'язку, причетності вживають `стосунок` або `зв'язок`. |
| ***Важливість** - це пам'ятати історію.* | ***Важливо** пам'ятати історію.* | Структурний перенос з англійської ("The importance is to remember..."). Українська мова віддає перевагу простішій конструкції з прислівником (`важливо`) або `Головне — ...`, а не громіздким конструкціям з абстрактним іменником у ролі підмета. |
| *Я ціную її **добрість**.* | *Я ціную її **доброту**.* | Неправильний вибір суфікса. Для якості, що походить від прикметника `добрий`, використовується суфікс `-от(а)`, а не `-ість`. |
| ***Розуміння** проблеми є першим кроком.* | ***Розуміння** проблеми — перший крок.* | Неправильне використання дієслова-зв'язки `є`. У реченнях-визначеннях, де і підмет, і присудок виражені іменниками в називному відмінку, зв'язка `є` зазвичай опускається, а на її місці ставиться тире. |

## Деколонізаційні застереження (Decolonization Notes)

**MANDATORY — never omit.** Важливо чітко розмежовувати українські та російські словотвірні моделі, уникаючи подачі українських форм як варіантів російських.

1.  **`Ставлення` vs. `Відношення`:** Це класичний приклад. Російське слово `отношение` має широке значення. В українській мові слід розрізняти:
    *   **Ставлення** — attitude (`ставлення до людей`, `ставлення до роботи`).
    *   **Відношення** — relation, ratio (вживається переважно в науковому, математичному, офіційному контекстах: `відношення сторін трикутника`, `у відношенні до...`).
    *   **Стосунок** — relevance, connection (`мати стосунок до справи`).
    *   **Взаємини, стосунки** — relationships (between people).
    Нав'язування російської моделі, де `отношение` покриває всі ці значення, є поширеною помилкою.

2.  **Продуктивність суфіксів:** Український суфікс **-ість** є значно більш фонетично послідовним і продуктивним, ніж російський **-ость**. Порівняйте: `свіж**ість**` — `свеж**есть**`, `необхідн**ість**` — `необходим**ость**`.

3.  **Безафіксний спосіб:** Українська мова активно використовує безафіксний спосіб творення абстрактних іменників від прикметників: `зелений` → `зелень`, `синій` → `синь`, `глибокий` → `глибінь`. Це є живою та питомою рисою мови, яку не слід заміняти громіздкими конструкціями.

4.  **Уникання "канцеляриту":** За радянських часів в українську мову (як і в російську) проник "канцелярит" — надмірне вживання віддієслівних іменників на `-ння` там, де можна було б використати простішу дієслівну конструкцію. Хоча ці іменники є важливою частиною мови, слід вчити студентів розрізняти доречне (в офіційному стилі) та недоречне їх використання, заохочуючи більш природні дієслівні форми в розмовному мовленні. Наприклад, замість *"відбувається процес читання книги"* краще сказати *"я/він/вона читає книгу"*.

## Природні приклади (Natural Examples)

**Minimum 12 natural Ukrainian sentences** using this grammar point.

### Іменники на -ість (якість, властивість)
1.  *Результат — відсутності української **державності**.* [S67]
2.  *Світ шокований **жорстокістю** російських злочинів в Україні.* [S68]
3.  *Саме дякуючи Вашій **небайдужості** й підтримці, стало можливим якісне оздоровлення дітей.* [S63]

### Іменники на -ння, -ття (дія, процес)
4.  *Основна Радянська Ідея полягала в тому, щоб зрівняти людей і створити один єдиний народ... щодо їжі намагались запровадити **універсальність**, **уніфікованість**.* [S70] - *тут також -ість*.
5.  *Цей вплив насправді був дуже великий і це дуже цікава тема, яку частково я досліджувала зі статей та інтерв'ю з дослідницею української **гастрономії** Оленою Брайченко, але також я просто згадувала, що мені розповідали мої батьки, які виросли в Радянському Союзі, моя бабуся, а також свій досвід з **дитинства**...* [S70]
6.  *...для того щоб зрозуміти **передумови** голодомору, який відбувся в 1932-33 роках...* [S67]

### Іменники на -ств(о), -цтв(о) (збірне поняття, стан)
7.  *Бездіяльність у момент, коли можна вчинити по справедливості, означає **боягузтво**.* [S71][S60]
8.  *Турботи **товариської** ні з чим зрівняти не можна.* [S72][S60] - *хоча це прикметник, він походить від `товариство`*
9.  *Поки існує цей режим і ця політична культура, Росія залишатиметься **загрозою** для України.* [S68]

### Безафіксні та інші іменники
10. *Їжа мала бути тим, що немає приносити **задоволення**, **насолоду**.* [S70]
11. *Є тільки одне благо — **знання** й тільки одне зло — **неуцтво**.* [S73][S60]
12. *При сварці — швидше помиритись. Не варто з друзями сваритись! Потрібно виявить **терпіння**.* [S74]
13. *Відчуття **щастя**. Хто тебе, **щастя**, людям придумав?* [S75][S76]

## Рекомендації для вправ (Activity Concepts)

**Phase 1: Ідентифікація та аналіз**
*   **Drill 1 (Find the Noun):** Дати учням текст (наприклад, уривок з [S68] або 11) і попросити знайти всі абстрактні іменники.
*   **Drill 2 (Word Family):** Для знайдених іменників попросити визначити твірне слово і частину мови (напр. `справедливість ← справедливий` (прикметник); `рішення ← вирішити` (дієслово)).

**Phase 2: Формування**
*   **Drill 3 (Suffix Choice):** Дати список прикметників і попросити утворити іменники, вибравши правильний суфікс (`-ість` чи `-от(а)`).
    *   `сильний → сила`, `щирий → ?`, `новий → ?`, `глибокий → ?`
*   **Drill 4 (Verbal Nouns):** Дати список дієслів і попросити утворити віддієслівні іменники.
    *   `змінювати → ?`, `обговорювати → ?`, `пояснити → ?`, `жити → ?`

**Phase 3: Застосування в контексті**
*   **Drill 5 (Sentence Transformation):** Перефразувати речення, використовуючи абстрактний іменник.
    *   *Він сміливо боровся.* → *Його **сміливість** у боротьбі всіх вразила.*
    *   *Важливо допомагати іншим.* → ***Допомога** іншим є дуже важливою.*
*   **Drill 6 (Fill in the blanks):** Заповнити пропуски в реченнях, поставивши утворений іменник у правильну відмінкову форму.
    *   *Ми пишаємося \_\_\_\_\_\_\_\_\_\_\_ (незалежний) нашої країни.* → *...**незалежністю**...*
    *   *Я не сумніваюся в його \_\_\_\_\_\_\_\_\_\_\_ (чесний).* → *...**чесності**...*

## Зв'язки з іншими темами

*   **Прикметники:** Абстрактні іменники на `-ість`, `-ота` безпосередньо утворюються від прикметників, що вимагає знання прикметникових основ.
*   **Дієслова:** Віддієслівні іменники на `-ння`, `-ття` тісно пов'язані з дієслівними основами та категорією виду.
*   **Відмінювання іменників:** Більшість абстрактних іменників на `-ість` належать до III відміни (жіночий рід, нульове закінчення), що має свою специфічну парадигму відмінювання. Іменники на `-ння`, `-ство` переважно належать до II відміни (середній рід).
*   **Синтаксис:** Розуміння ролі абстрактних іменників як підмета, додатка чи іменної частини присудка є ключовим для побудови складних речень.
