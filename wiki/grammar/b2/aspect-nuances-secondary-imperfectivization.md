## Як це пояснюють у школі (How Schools Teach This)

В українських школах тема виду дієслова (доконаний і недоконаний) вводиться в 6-7 класах. Основний підхід полягає у представленні **видових пар** — дієслів, що мають однакове лексичне значення, але відрізняються за ознакою завершеності/незавершеності дії [S35][S36][S37].

Навчальний процес виглядає так:
1.  **Розрізнення за питаннями**: Учні вчаться розрізняти види за питаннями: дієслова недоконаного виду відповідають на питання *що робити?* (читати, малювати), а доконаного — *що зробити?* (прочитати, намалювати) [S35].
2.  **Способи творення пар**: Підручники пояснюють, що видові пари утворюються кількома способами. Найчастіше згадується **префіксація** для утворення доконаного виду (ДВ) від недоконаного (НДВ): `робити (НДВ) → зробити (ДВ)` [S35][S36].
3.  **Імперфективація суфіксами**: Процес утворення недоконаного виду від доконаного (вторинна імперфективація) подається як один зі способів творення видових пар. Заболотний [S36] та інші автори [S37][S35] наводять приклади, де форма НДВ утворюється від ДВ за допомогою:
    *   **Суфіксів**: `переписати (ДВ) → перепис**ува**ти (НДВ)`, `вислухати (ДВ) → вислух**овува**ти (НДВ)`, `охолодити (ДВ) → охоло**дж**увати (НДВ)` [S36][S37].
    *   **Чергування звуків у корені**: `витерти → вит**и**рати`, `перевозити → перевезти` [S36][S37].
    *   **Зміни наголосу**: `розрі́зати → розріза́ти` [S36].

Термін "вторинна імперфективація" в шкільних підручниках не використовується. Замість нього вживається більш загальне формулювання "утворення видових пар за допомогою суфіксів та чергувань" [S36]. Основний акцент робиться на тому, що дієслова існують у парах, і учні повинні вміти утворювати одну форму від іншої, щоб правильно передати тривалість, повторюваність або завершеність дії.

## Повна парадигма (Full Paradigm)

Вторинна імперфективація — це процес утворення дієслова недоконаного виду (НДВ) від основи дієслова доконаного виду (ДВ), яке, як правило, вже є префіксованим. Цей процес дозволяє виразити тривалість, повторюваність або процес дії, яка концептуально є єдиним, завершеним актом.

Основними механізмами є суфіксація, що часто супроводжується чергуванням голосних та приголосних у корені.

### 1. Імперфективація за допомогою суфікса `-ува- / -юва-`

Це найпродуктивніший спосіб. Суфікс додається до основи інфінітива ДВ.

| Дієслово ДВ (що зробити?) | Основа ДВ | + Суфікс | Дієслово НДВ (що робити?) | Джерело |
| :--- | :--- | :--- | :--- | :--- |
| прочитати | `прочита-` | `-ва-` (історично) -> `-ува-` | прочит**ува**ти | (Аналогія) |
| намалювати | `намалюва-` | `-ти` -> `-ува-ти` | маль**юва**ти (*просте*) / розмаль**ову**вати | [S36] |
| побудувати | `побудува-` | `-ти` -> `-ува-ти` | побуд**ову**вати | (Аналогія) |
| вислухати | `вислуха-` | `-ти` -> `-ува-ти` | вислух**ову**вати | [S37] |
| переписати | `переписа-` | `-ти` -> `-ува-ти` | перепис**ува**ти | [S36] |
| наголосити | `наголоси-` | `с`→`ш` + `-ува-ти` | наголош**ува**ти | [S36] |
| охолодити | `охолоди-` | `д`→`дж` + `-ува-ти` | охоло**дж**увати | [S36] |

### 2. Імперфективація за допомогою суфікса `-а- / -я-` (з чергуванням у корені)

Цей спосіб характерний для багатьох базових дієслів і майже завжди супроводжується змінами голосних або приголосних у корені.

#### 2.1. Чергування голосних

Найпоширеніші чергування в корені перед суфіксом `-а-` [S38][S39]:

| Чергування | Умови | Дієслово ДВ | → | Дієслово НДВ | Джерело |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **`[е] → [и]`** | Перед наголошеним суфіксом `-а-` | вит**е**рти | → | вит**и**р**а**ти | [S37][S38] |
| | | пом**е**рти | → | пом**и**р**а**ти | (Аналогія) |
| **`[о] → [а]`** | Перед суфіксом `-а-` | допом**о**гти | → | допом**а**г**а**ти | [S38] |
| | | ск**о**чити | → | ск**а**к**а**ти | [S39] |
| **`ø → [и]`** (нуль звука → `и`) | З'являється `и` | розірвати | → | розр**и**в**а**ти | (Аналогія) |
| | | назвати | → | наз**и**в**а**ти | (Аналогія) |
| **`[е] → [і]`** | Перед суфіксом `-а-` | вип**е**кти | → | вип**і**к**а**ти | [S38] |
| | | зач**е**пити | → | зач**і**п**а**ти | [S36] |
| **`[і] → [о] / [е]` (зворотне)** | Історичне чергування | привезти (віз) | → | привоз**и**ти | <!-- VERIFY --> |
| | | перенести (ніс) | → | перенос**и**ти | (Аналогія) |

#### 2.2. Чергування приголосних

Чергування приголосних часто відбувається разом із суфіксацією [S40][S41].

| Чергування | Дієслово ДВ | → | Дієслово НДВ | Джерело |
| :--- | :--- | :--- | :--- | :--- |
| **`[д] → [дж]`** | розбу**д**ити | → | розбу**дж**увати | [S41] |
| **`[с] → [ш]`** | запро**с**ити | → | запро**ш**увати | [S42] |
| **`[з] → [ж]`** | вира**з**ити | → | вира**ж**ати | (Аналогія) |
| **`[т] → [ч]`** | зусрі**т**ити | → | зусрі**ч**ати | (Аналогія) |
| **`[б] → [бл]`** | зро**б**ити | → | зро**бл**яти (*рідко*), частіше `роб**и**ти` | [S36] |
| **`[в] → [вл]`** | устано**в**ити | → | устано**вл**ювати | (Аналогія) |

### 3. Суплетивні та інші пари

Деякі видові пари утворені від різних основ (суплетивізм).
*   `взяти` (ДВ) — `брати` (НДВ) [S37][S36]
*   `піймати` (ДВ) — `ловити` (НДВ) [S36]
*   `знайти` (ДВ) — `шукати` (НДВ) [S35]

## Частотність і пріоритети

1.  **Найвищий пріоритет (B1/B2):**
    *   **Суфікс `-ува- / -юва-`**: Це найпродуктивніший і найрегулярніший спосіб. Учні повинні вміти активно його використовувати для утворення НДВ від нових та іншомовних дієслів (напр., `організувати → організ**ову**вати`).
    *   **Суфікс `-а-` з чергуванням `[е]→[и]` та `[о]→[а]`**: Цей патерн охоплює велику кількість високочастотних дієслів руху, мовлення та базових дій (`збирати`, `починати`, `допомагати`). Його розпізнавання та використання є критично важливим.

2.  **Середній пріоритет (B2):**
    *   **Суфікс `-а-` з чергуванням `[е]→[і]`**: `зачепити → зачіпати`. Менш поширений, ніж попередні, але важливий для природного мовлення.
    *   **Чергування приголосних**: `[д]→[дж]`, `[с]→[ш]` (`запросити → запрошувати`). Ці зміни є системними в українській мові [S40] і важливі для правильної вимови та орфографії.

3.  **Низький пріоритет (пасивне розпізнавання на B2):**
    *   **Суфікс `-ва-`**: `дати → давати`. Ці пари є базовими і, як правило, засвоюються на ранніх етапах як окремі лексичні одиниці.
    *   **Зміна наголосу**: `розки́дати → розкида́ти`. Цей патерн є непродуктивним і стосується обмеженої кількості дієслів. Розпізнавання достатньо.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я вчора цілий день **прочитав** цю книжку. | Я вчора цілий день **прочитував** цю книжку. | Для позначення дії, що тривала протягом певного періоду часу (цілий день), потрібен недоконаний вид. Оскільки базове дієслово префіксоване (`про-читати`), використовується вторинний імперфектив. |
| Він завжди мене **зачепує**, коли жартує. (Неправильне утворення) | Він завжди мене **зачіпає**, коли жартує. | Учень намагається утворити НДВ, але забуває про системне чергування голосного `[е] → [і]` в корені перед суфіксом `-а-` [S36][S38]. |
| Я буду **вибирати** подарунок завтра. (Наголос) | Я буду вибир**а́**ти подарунок завтра. | При утворенні НДВ з чергуванням `[е] → [и]` наголос завжди падає на суфікс `-а-` (`виберу` → `вибир**а́**ти`) [S38]. Англомовні учні часто зберігають наголос на корені. |
| Я кілька разів **переписав** тест. | Я кілька разів **переписував** тест. | Для позначення повторюваної дії (`кілька разів`) використовується недоконаний вид. Англійське "I rewrote the test several times" не передає цієї різниці. Тут потрібен вторинний імперфектив `переписувати` [S36]. |
| Він **запросив** її на танець на кожній вечірці. | Він **запрошував** її на танець на кожній вечірці. | Повторюваність дії (`на кожній вечірці`) вимагає НДВ. Учень правильно обирає лексику (`запросити`), але використовує доконаний вид замість вторинного імперфектива з чергуванням приголосного (`запрошувати`). |

## Деколонізаційні застереження

Хоча система аспектних пар є спільною рисою слов'янських мов, українська вторинна імперфективація має унікальні фонетичні та морфологічні особливості, які відрізняють її від російської. Ігнорування цих відмінностей призводить до русизмів.

1.  **Чергування голосних `[о], [е] → [і]`**: Це фундаментальна риса української фонетики. Хоча вона не завжди проявляється безпосередньо в парах ДВ-НДВ, вона є частиною всієї дієслівної системи. Наприклад, ДВ `допом**о**гти` → минулий час `допом**і**г`. Російська мова не має цього чергування (`помочь` → `помог`). Навчальні матеріали повинні пояснювати утворення НДВ `допом**а**гати` як частину української системи, а не за аналогією до російського `помогать`.
2.  **Чергування приголосних**: Українська мова має унікальні чергування, як-от `[д] → [дж]` (`охоло**д**ити` → `охоло**дж**увати`), `[т] → [ч]`, тоді як у російській мові аналогічні процеси виглядають інакше (напр., укр. `во**дж**у` vs рос. `во**ж**у`). Використання російських моделей чергування є грубою помилкою.
3.  **Суфікс `-ува-` vs російський `-ыва-`**: Український продуктивний суфікс імперфективації — `-ува-` (`прочит**ува**ти`, `перепис**ува**ти`). Пряме копіювання російського `-ыва-` (`прочит**ыва**ть`) в українську є неприпустимим і свідчить про калькування.
4.  **Пояснення від української бази**: Неприпустимо пояснювати українські форми як "варіанти" або "винятки" з російських норм. Наприклад, не можна казати "замість російського `-ывать` українці кажуть `-увати`". Пояснення має йти від внутрішньої логіки української мови: дієслово ДВ `переписати` → утворюємо НДВ за допомогою продуктивного суфікса `-ува-` → `переписувати` [S36][S43].

## Природні приклади

#### Група 1: Процес або тривала дія (`-ува-`)
*   Мама довго **розпитувала** мене про мої справи в школі. (Дія тривала в часі)
*   Ми **обговорювали** цей проєкт цілий тиждень, перш ніж ухвалити рішення. (Процес)
*   Художник **розмальовував** стіну яскравими фарбами, і я спостерігав за ним. ([S44], аналогія до [S36])

#### Група 2: Повторювана, звична дія (`-ува-`, `-а-`)
*   Коли я жив у Львові, то щоранку **випивав** філіжанку кави в улюбленій кав'ярні.
*   Мій дідусь **розповідав** цю історію щоразу, коли ми збиралися всією родиною.
*   Він завжди **запізнювався** на зустрічі, і це всіх дратувало.

#### Група 3: Дія з "анульованим" результатом або загальний факт
*   Ти сьогодні **відчиняв** вікно? Бо в кімнаті протяг. [S45][S46][S47]
*   Я вже **вмикав** комп'ютер, але потім згадав, що забув удома телефон, і вимкнув його. (Результат дії (ввімкнений комп'ютер) був анульований)
*   — Ти брав мої ключі? — Так, **брав**, але вже поклав на місце. [S48][S49][S47]

#### Група 4: Чергування в корені
*   Кожного літа ми **допомагали** бабусі збирати врожай. (`допомогти → допомагати`) [S38]
*   Він повільно **вмирав** від спраги в пустелі. (`вмерти → вмирати`)
*   Щойно я **починав** працювати, як хтось дзвонив у двері. (`почати → починати`)

## Рекомендації для вправ

*   **Фаза 1 (Розпізнавання патернів):**
    *   **Вправа "Знайди пару":** З'єднати дієслова ДВ з відповідними дієсловами НДВ. `(переписати, витерти, запросити)` з `(запрошувати, переписувати, витирати)`.
    *   **Вправа "Класифікація":** Розподілити пари дієслів за способом творення НДВ (суфікс `-ува-`, суфікс `-а-` + чергування, зміна приголосного).

*   **Фаза 2 (Формування):**
    *   **Трансформаційний дрил:** Утворити дієслово НДВ від поданого дієслова ДВ.
        *   `прочитати → прочитувати`
        *   `запитати → запитувати`
        *   `витерти → витирати`
        *   `допомогти → допомагати`

*   **Фаза 3 (Контекстне застосування):**
    *   **Вправа "Вибір виду":** У реченнях розкрити дужки, обравши правильний вид дієслова (ДВ або НДВ).
        *   `Він (прочитав / прочитував) цю книжку за один вечір.` (Результат → ДВ)
        *   `Він (прочитав / прочитував) цю книжку цілий вечір.` (Процес → НДВ)
    *   **Вправа "Перефразування":** Перетворити речення з ДВ на речення з НДВ, додавши обставини часу, що вказують на тривалість або повторюваність.
        *   `Я відремонтував машину.` → `Я цілий день **ремонтував** машину.`
        *   `Вона написала йому листа.` → `Вона часто **написувала** йому листи.`

## Зв'язки з іншими темами

*   **Базовий вид дієслова (A2/B1):** Вторинна імперфективація є поглибленням цієї теми. Без розуміння базової опозиції `робити/зробити` ця тема неможлива.
*   **Дієслівні префікси (B1/B2):** Вторинна імперфективація майже завжди застосовується до вже префіксованих дієслів.
*   **Часи дієслів:** Дієслова НДВ (включно з вторинними) мають форми теперішнього, минулого та складеного майбутнього часу. Дієслова ДВ не мають теперішнього часу.
*   **Дієприкметники та дієприслівники (B2):** Основа вторинного імперфектива часто використовується для утворення дієприслівників теперішнього часу (`переписуючи`, `обговорюючи`) [S50].
*   **Фонетика: Чергування голосних і приголосних:** Це явище є невід'ємною частиною механізму імперфективації [S38][S40].
