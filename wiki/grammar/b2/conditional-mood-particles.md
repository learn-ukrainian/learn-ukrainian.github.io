## Як це пояснюють у школі (How Schools Teach This)

Українська шкільна програма вводить поняття способів дієслова, часток та вигуків у середніх класах, зазвичай у 6-7 класах.

**1. Умовний спосіб (Conditional Mood):**
Цю тему вводять у 6-му або 7-му класі [S47][S48]. Підхід дуже прагматичний. У підручнику Літвінової (7 клас) тема починається з прямого порівняння речень у дійсному та умовному способах: `Я вам допомогла` vs. `Я б вам допомогла` [S48]. Це одразу фокусує учнів на ключовому маркері — частці `б/би` — та зміні значення: від реальної дії до бажаної або можливої за певних умов.

Основні пункти, що викладаються:
- **Творення:** Форма минулого часу дієслова + частка `би` (після приголосного) або `б` (після голосного) [S49].
- **Значення:** Дія бажана або можлива лише за певних умов (`І риба співала б, якби голос мала`) [S47].
- **Гнучкість частки:** Підкреслюється, що частка `би/б` пишеться окремо і може стояти не безпосередньо біля дієслова (`Я б тобі пісню приніс`) [S49].

**2. Частки (Particles):**
Частки як службову частину мови розглядають у 7-му класі [S50]. Їх одразу поділяють на три основні розряди за функцією:
- **Формотворчі:** `би/б` (умовний спосіб), `хай/нехай` (наказовий спосіб), `най-/якнай-` (ступені порівняння) [S51].
- **Словотворчі:** `аби-, де-, -сь, будь-` і т.д., які утворюють нові слова (переважно займенники та прислівники) [S52].
- **Модальні (Фразові):** Найбільша група, яка надає слову чи реченню додаткових смислових чи емоційних відтінків. Підручники, як-от Заболотного, дають детальну таблицю підгруп модальних часток (стверджувальні, заперечні, питальні, підсилювальні тощо) з прикладами [S53].

**3. Вигуки (Interjections):**
Вигуки вивчають у 7-му класі як "особливу частину мови", що не належить ні до самостійних, ні до службових [S54][S55]. Основний фокус на:
- **Групи за значенням:** емоційні (`ой, ах`), волевиявлення (`геть, цить`), мовний етикет (`добридень, будь ласка`), звуконаслідувальні (`ку-ку, няв`) [S55].
- **Правопис:** написання через дефіс (`го-о-ол`, `їй-богу`) [S56].
- **Пунктуація:** виділення комою або знаком оклику, з особливою увагою на випадки, коли `о` та `ой` перед звертанням комою не виділяються (`Ой ти, дівчино...`) [S57][S58].

Загалом, шкільний підхід є структурним і функціональним: спочатку визначається частина мови, потім її розряди та функції, і наостанок — правопис та пунктуація.

## Повна парадигма (Full Paradigm)

### Умовний спосіб (Conditional Mood)

Умовний спосіб дієслова не має власних закінчень. Він утворюється аналітично: **форма минулого часу дієслова + формотворча частка `би` або `б`**. Дієслова в умовному способі змінюються за числами, а в однині — за родами [S49].

| Рід / Число | Дієслово (минулий час) | Частка | Результат (Умовний спосіб) |
| :--- | :--- | :--- | :--- |
| **Однина** | | | |
| Чоловічий | `читав`, `міг`, `приніс` | `би` | `читав би`, `міг би`, `приніс би` |
| Жіночий | `читала`, `могла`, `принесла` | `б` | `читала б`, `могла б`, `принесла б` |
| Середній | `читало`, `могло`, `принесло` | `б` | `читало б`, `могло б`, `принесло б` |
| **Множина** | `читали`, `могли`, `принесли` | `б` | `читали б`, `могли б`, `принесли б` |

**Правило милозвучності:**
- **`би`** вживається після слів, що закінчуються на **приголосний**: `я сказав би`, `він прийшов би`.
- **`б`** вживається після слів, що закінчуються на **голосний**: `вона сказала б`, `вони прийшли б`.

### Частки (Particles)

Частки не змінюються. Вони класифікуються за функцією та значенням [S53][S52].

| Розряд | Група | Приклади часток | Приклад у реченні |
| :--- | :--- | :--- | :--- |
| **Формотворчі** | - | `б (би)` | `Я хотів **би** піти.` |
| | | `хай`, `нехай` | `**Хай** завжди буде сонце!` |
| | | `най-`, `якнай-`, `що-` | `Він **най**сильніший.` |
| **Словотворчі** | - | `аби-`, `де-`, `-сь` | `**аби**хто`, `**де**коли`, `хто**сь**` |
| | | `будь-`, `-небудь` | `**будь**-який`, `що-**небудь**` |
| | | `казна-`, `хтозна-` | `**казна**-де`, `**хтозна**-який` |
| | | `ні-`, `не-` | `**ні**хто`, `**не**друг` |
| **Модальні** | **Підсилювальні** | `ж (же)`, `таки`, `навіть`, `і`, `й`, `та`, `аж` | `Я **таки** це зроблю.`, `**Навіть** він прийшов.`|
| | **Обмежувально-видільні**| `тільки`, `лише`, `лиш`, `хоч`, `саме` | `**Лише** ти мене розумієш.` |
| | **Вказівні** | `ось`, `от`, `це`, `то`, `он`, `оце` | `**Ось** моя хата.` |
| | **Стверджувальні** | `так`, `авжеж`, `аякже`, `еге`, `еге ж` | `**Так**, я згоден.` |
| | **Заперечні** | `не`, `ні`, `ані` | `Я **не** знаю.`, `**Ні** слова правди.` |
| | **Питальні** | `чи`, `хіба`, `невже` | `**Чи** ти прийдеш?`, `**Невже** це правда?` |
| | **Власне-модальні** | `мов`, `ніби`, `наче`, `мовби` | `Він **ніби** спить.` |
| | **Спонукальні** | `-но`, `-бо`, `бодай`, `годі`, `ну` | `Скажи-**но** правду!`, `**Ну**мо до роботи!`|
| | **Окличні** | `що за`, `що то за`, `як`, `який` | `**Що за** краса!` |

### Вигуки (Interjections)

Вигуки — незмінна частина мови. Їхня парадигма — це класифікація за значенням і походженням [S55].

| Група за значенням | Призначення | Приклади вигуків |
| :--- | :--- | :--- |
| **Емоційні** | Вираження почуттів (радість, біль, страх, здивування) | `о!`, `ой!`, `ай!`, `ах!`, `ех!`, `ого!`, `овва!`, `фу!`, `тьху!`, `Боже мій!` |
| **Волевиявлення** | Спонукання до дії, привертання уваги, клич тварин | `агов!`, `гей!`, `цить!`, `тс-с!`, `стоп!`, `геть!`, `нумо!`, `гайда!`, `киць-киць!`, `гиля!` |
| **Мовний етикет** | Привітання, прощання, подяка, вибачення, прохання | `добридень`, `привіт`, `до побачення`, `дякую`, `спасибі`, `будь ласка`, `вибачте`, `перепрошую` |
| **Звуконаслідувальні** | Відтворення звуків природи, тварин, предметів | `ку-ку`, `тьох-тьох`, `гав-гав`, `няв`, `му-у`, `дзень`, `бах`, `шу-шу` |

## Частотність і пріоритети (B2)

На рівні B2 учень має вийти за межі простого констатування фактів і навчитися висловлювати нюанси: гіпотези, побажання, сумніви, емоції.

**Високий пріоритет:**
1.  **Умовний спосіб у складних реченнях з `якби`**: Це основа для вираження гіпотетичних ситуацій, умов та нереальних бажань. Конструкція `Якби..., [то] ... б...` є абсолютно ключовою. [S59].
2.  **Умовний спосіб для ввічливих прохань**: Форми `чи не могли б ви...`, `я хотів би...`, `чи не допомогла б ти...` є важливим інструментом соціальної взаємодії і демонструють вищий рівень володіння мовою [S60].
3.  **Обмежувально-видільні та підсилювальні частки (`тільки`, `лише`, `саме`, `ж/же`, `таки`, `навіть`)**: Вони дозволяють точно фокусувати думку, виділяти головне, сперечатися, підсилювати твердження. Без них мова залишається "пласкою". [S54].
4.  **Емоційні вигуки та вигуки мовного етикету**: На B2 учень повинен реагувати на ситуацію природно. `Ого!`, `Овва!`, `Жах!`, `Ой, лишенько!` та бездоганне використання `будь ласка`, `перепрошую`, `дякую` є обов'язковими.

**Середній пріоритет:**
1.  **Питальні та власне-модальні частки (`хіба`, `невже`, `мов`, `ніби`)**: Важливі для вираження сумніву, здивування, порівняння. Вони часто використовуються в розмовній мові та літературі. [S61].
2.  **Словотворчі частки**: На B2 учень вже повинен впізнавати та використовувати займенники `хтось`, `що-небудь`, `будь-який`, `ніхто`, але активне творення нових форм не є головним пріоритетом. Важливіше — правильне їх написання (разом, окремо, через дефіс).
3.  **Вигуки волевиявлення (`цить!`, `агов!`)**: Корисні, але ситуативні.

**Низький пріоритет:**
1.  **Звуконаслідувальні слова**: Учень повинен їх розуміти в тексті, але активне використання не є критичним для загального спілкування.
2.  **Рідковживані частки (`-бо`, `-но`, `бодай`)**: Хоча `скажи-но` є досить поширеним, учень може обійтися без активного використання цих форм, замінюючи їх іншими конструкціями.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Якщо я мав би гроші, я купив би машину. | **Якби** я мав гроші, я купив би машину. | В умовних реченнях, що виражають нереальну умову, використовується сполучник `якби`, а не `якщо`. `Якщо` вживається для реальних умов (`Якщо буде дощ, ми залишимось вдома`). |
| Давай(те) підемо в кіно. | **Ходімо** в кіно. / **Пішли** в кіно (розмовне). | Конструкція `давай(те) + дієслово` є поширеним русизмом. В українській мові для спонукання до спільної дії використовуються дієслова 1-ї особи множини наказового способу (`-імо`, `-ьмо`) [S62]. |
| Він ж прийшов. | Він **же** прийшов. / Він **таки** прийшов. | Частка `ж` після приголосних перетворюється на `же` для милозвучності. Крім того, сама по собі частка `ж` слабша за `таки` для вираження наполягання або факту всупереч очікуванням. |
| О, мій друже, я радий тебе бачити! | **О мій друже**, я радий тебе бачити! | Якщо вигук `о` стоїть перед особовим займенником або присвійним займенником у складі звертання, він не відділяється комою і виступає як підсилювальна частка [S57][S58]. |
| Це не є правильно. | Це **неправильно**. | Хоча конструкція `не є + прикметник` технічно можлива, вона часто є канцеляризмом або впливом англійської мови (`it is not correct`). Природніше в українській мові утворювати одне слово з префіксом `не-`, якщо протиставлення немає. |
| Я нічого не хочу сказати. | Я не хочу **нічого** казати. | В українській мові, на відміну від англійської, використовується "подвійне заперечення". Заперечна частка `не` ставиться перед присудком, а заперечний займенник (`ніхто`, `ніщо`) або прислівник (`ніколи`, `ніде`) займає свою позицію в реченні. |

## Деколонізаційні застереження (Decolonization Notes)

**MANDATORY — never omit.**

1.  **`Давай(те)` vs `Ходімо/Робімо`**: Найпоширеніший і найважливіший пункт. Використання конструкції `давай(те) + інфінітив` для 1-ї особи множини наказового способу (`давайте співати`) є прямим синтаксичним кальмуванням з російської мови. Автентична українська форма — дієслова із закінченнями `-імо`, `-ьмо`: `співаймо`, `читаймо`, `ходімо` [S62]. Це не просто "кращий" варіант, а єдино правильний літературний.
2.  **Частка `таки`**: Українська мова активно використовує підсилювальну частку `таки` (`він таки прийшов`, `я таки це зроблю`), часто надаючи висловлюванню відтінок наполегливості або констатації факту всупереч сумнівам. Хоча в російській є аналог, в українській ця частка є більш уживаною та ідіоматичною.
3.  **Розрізнення `чи` та `ли`**: Російська питальна частка `ли` (читаешь ли ты?) не має прямого відповідника в сучасній українській літературній мові. Питання формуються за допомогою питальної частки `чи` на початку речення (`Чи ти читаєш?`) або просто інтонацією. Використання `ли` в українському тексті є грубою помилкою.
4.  **Етикетні вигуки**: Формули ввічливості, як-от `будь ласка`, не є калькою з російського `пожалуйста` (яке походить від "пожалуй, ста" - "пожалій, старий"). Українське `будь ласка` походить від "будь ласкав". Наголошуйте на самостійності та унікальності українських етикетних формул.
5.  **Негативні конструкції**: Слід активно використовувати питомо українські заперечні конструкції з часткою `ані` (`ані хмариночки`, `ані слова`), які додають виразності та емоційності, замість того, щоб завжди покладатися на конструкції з `ні`, які мають прямі аналоги в російській.

## Природні приклади (Natural Examples)

**Група 1: Умовний спосіб (гіпотетичні ситуації)**
*   `Якби у мене була віза, я б полетів з тобою хоч завтра хоч на край світу.` [S59]
*   `І риба співала б, якби голос мала.` [S47]
*   `Коли б собака умів розказувати, то розказав би багато інтересних історій.` [S48]
*   `Я б залюбки залишилася працювати в Україні, якби не треба було виплачувати кредит за навчання.` [S59]

**Група 2: Умовний спосіб (ввічливе прохання)**
*   `Чи не допомогла б ти мені приготувати салат?` [S60]
*   `Відпросився б ти в командира, прийшов би до мене хоч на часину.` [S63]

**Група 3: Модальні частки (підсилення, виділення)**
*   `Так! Я буду крізь сльози сміятись.` (Леся Українка [S61])
*   `Поезія — це завжди неповторність, якийсь безсмертний дотик до душі.` (Ліна Костенко [S64])
*   `Тільки одного не забувай, сину: як би тобі не було тяжко в житті, не зроби жодного недоброго вчинку.` (В. Королів-Старий [S54])
*   `Дома й стіни допомагають.` (Народна творчість [S53])

**Група 4: Вигуки (емоції та етикет)**
*   `Говорили-балакали, сіли та й заплакали.` (Приказка [S65])
*   `О, розмотай шляхи мені, богине!` (Л. Костенко [S55])
*   `Боже, як багато у світі краси!` (О. Довженко [S55])
*   `Ой ти, дівчино, з горіха зерня…` (І. Франко [S56])

## Рекомендації для вправ (Activity Concepts)

**Phase 1: Foundation (Conditional Mood)**
1.  **Трансформація:** Перетворити речення з дійсного способу в умовний.
    - *Вхід:* `Вона читає книгу.` -> *Вихід:* `Вона читала б книгу.`
    - *Вхід:* `Ми поїдемо в Карпати, якщо буде сніг.` -> *Вихід:* `Ми поїхали б у Карпати, якби був сніг.`
2.  **Побудова речень:** Скласти умовні речення за схемою `Якби [умова], то [наслідок]`.
    - *Завдання:* `я / бути мільйонером -> я / купити острів.` -> *Вихід:* `Якби я був мільйонером, я б купив острів.`

**Phase 2: Nuance (Particles)**
1.  **Вставка частки:** Дати речення і частку в дужках. Учень має вставити частку в правильне місце, щоб передати заданий відтінок.
    - *Завдання:* `Він прийшов на вечірку. (навіть)` -> *Вихід:* `Навіть він прийшов на вечірку.` (неочікувано) або `Він прийшов навіть на вечірку.` (був і на інших заходах).
2.  **Вибір частки:** Дати речення з пропуском і варіанти часток.
    - *Завдання:* `___ ти розумієш, про що я. (тільки / невже / же)` -> *Вихід:* `Тільки ти...` (обмеження), `Невже ти...` (здивування/сумнів).

**Phase 3: Integration & Fluency (Dialogue with Interjections)**
1.  **Діалоги за ситуацією:** Використовувати ситуації з підручників [S66].
    - *Ситуація:* "Телефонний дзвінок від шахраїв, які повідомляють про виграш." Учень має скласти діалог, використовуючи вигуки здивування (`Ого!`, `Невже?`), сумніву (`Хм...`), відмови (`Геть!`).
2.  **"Озвучка" емодзі:** Показати ряд емодзі (😮, 😥, 😂, 😠) і попросити "підписати" їх відповідними українськими вигуками (`Овва!`, `Ой, лишенько!`, `Ха-ха!`, `Трясця!`) [S67].

## Зв'язки з іншими темами

- **Минулий час дієслова (B1):** Є абсолютною передумовою, оскільки форма умовного способу будується на його основі. Без вільного володіння формами `робив/робила/робили` учень не зможе утворити `робив би/робила б/робили б`.
- **Наказовий спосіб (B1/B2):** Прямий зв'язок через формотворчі частки `хай`/`нехай`, які використовуються для утворення 3-ї особи наказового способу (`хай робить`). Також умовний спосіб може вживатися у значенні наказового для ввічливості (`Ти б не міг допомогти?`).
- **Займенники (A2/B1):** Тісно пов'язаний через словотворчі частки `аби-, де-, -сь, будь-, ні-` і т.д., які утворюють неозначені та заперечні займенники (`хто` -> `хтось`, `будь-хто`, `ніхто`).
- **Ступені порівняння прикметників (B1):** Зв'язок через формотворчі частки `най-`, `якнай-` для утворення найвищого ступеня (`сильний` -> `найсильніший`).
- **Синтаксис складного речення (B2/C1):** Умовний спосіб є ключовим елементом підрядних речень умови, що виражають гіпотетичні, нереальні ситуації (`...якби...`).
