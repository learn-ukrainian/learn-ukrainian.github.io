## Як це пояснюють у школі (How Schools Teach This)
У старших класах української школи (10-11 класи) вводиться блок "Практична риторика", що є основою для навчання дискусійного мовлення. Підручники розрізняють кілька ключових понять, які важливо не плутати.

1.  **Риторика** подається як "наука про способи переконання та впливу на аудиторію" (Source 34, 35) і "мистецтво реалізації мовної особистості" (Source 6, 7). Це загальна рамка.

2.  **Суперечка** є загальним терміном, що має кілька різновидів. Підручник Заболотного (Source 16, 17) виділяє чотири основні види:
    *   **Дискусія**: Публічне обговорення з метою знайти спільне рішення або встановити істину (Source 13, 21). Це не обов'язково суперечка, а "можливість кожній стороні висловити свою точку зору" (Source 13).
    *   **Диспут**: Публічна суперечка на наукову або суспільно важливу тему, яка старанно планується (Source 16, 17).
    *   **Полеміка**: Публічна суперечка з конфронтацією, метою якої є не згода, а "перемога над супротивником, відстоювання й захист своєї позиції" (Source 16, 17, 41).
    *   **Дебати**: Суперечки під час обговорення доповідей, виступів; у сучасному світі часто використовуються в передвиборчій агітації з метою "публічної перемоги над суперником" (Source 16, 17). Дебати чітко структуровані, їх мета — переконати третю сторону (суддів), а не опонента (Source 41, 43).

Ключовим елементом є **аргументація**: наведення доказів для обґрунтування тези. Підручники наголошують на використанні достовірних джерел, статистичних даних, прикладів з життя та думок авторитетних людей (Source 11, 22).

Етикет посідає центральне місце: не можна переривати опонента, вживати образливі слова, переходити на особистості (Source 13, 18, 25).

## Повна парадигма (Full Paradigm)
Оскільки дискусійне мовлення є функціональним, а не морфологічним, парадигма складається з усталених фраз для виконання комунікативних завдань.

**Таблиця 1: Висловлення власної позиції (Expressing Your Position)**

| Функція | Фраза | Приклад |
| :--- | :--- | :--- |
| **Ствердження** | `Я вважаю / думаю / переконаний, що...` | `Я вважаю, що технології покращують наше життя.` |
| | `На мою думку, ...` / `На моє переконання, ...` | `На мою думку, традиційні книги ніколи не зникнуть.` |
| | `Дозволю собі стверджувати, що...` (формально) | `Дозволю собі стверджувати, що ця проблема значно глибша.` |
| | `Мені здається, що...` (менш категорично) | `Мені здається, що ми дещо упускаємо.` |
| **Підкреслення** | `Я хочу наголосити / підкреслити, що...` | `Хочу наголосити, що це лише один з аспектів проблеми.` |

**Таблиця 2: Згода та незгода (Agreement and Disagreement)**

| Функція | Фраза | Приклад (на тезу "Треба заборонити соцмережі") |
| :--- | :--- | :--- |
| **Повна згода** | `Я (цілком і повністю) згоден / згодна з вами.` | `Я повністю згоден з вами, вони забирають забагато часу.` |
| | `Безперечно.` / `Безсумнівно.` / `Абсолютно вірно.` | `Безперечно, це було б корисно для ментального здоров'я.` |
| **Часткова згода** | `Я погоджуюся з вами, але... / однак...` | `Я погоджуюся, що є негативні аспекти, однак вони також є інструментом комунікації.` |
| | `З одного боку, ви маєте рацію, але з іншого...` | `З одного боку, ви маєте рацію щодо залежності, але з іншого, це джерело інформації.` |
| **Ввічлива незгода** | `Дозвольте з вами не погодитися.` | `Дозвольте з вами не погодитися, заборона — не вихід.` |
| | `Я розумію вашу точку зору, проте...` | `Я розумію вашу точку зору, проте це обмежить свободу слова.` |
| | `На жаль, я не можу з цим погодитися.` | `На жаль, я не можу з цим погодитися, адже це нереалістично.` (Source 36) |
| **Пряма незгода** | `Я категорично проти.` / `Це не так.` | `Це не так, проблема не в самих мережах, а в культурі їх використання.` |

**Таблиця 3: Аргументація та структурування (Argumentation and Structuring)**

| Функція | Фраза | Приклад |
| :--- | :--- | :--- |
| **Введення аргументу** | `По-перше, ... По-друге, ...` | `По-перше, це економічно невигідно. По-друге, це порушує права людини.` |
| | `Можна навести такий доказ / аргумент:` | `Як доказ, можна навести статистику минулого року.` (Source 21) |
| | `Це можна підтвердити тим, що...` | `Це можна підтвердити тим, що успішні люди читають щодня.` |
| **Посилання на джерело** | `Як зазначає (автор), ...` / `Згідно з (джерелом),...` | `Як зазначає Платон, "на лікаря і дивитися б не схотіли, а обрали б того, хто володіє словом".` (Source 6, 7) |
| **Узагальнення** | `Отже, ...` / `Таким чином, ...` | `Отже, ми бачимо, що запропоноване рішення має більше недоліків, ніж переваг.` |
| **Висновки** | `З цього випливає, що...` / `На завершення, ...` | `З цього випливає, що потрібно шукати інший підхід.` |

## Частотність і пріоритети
Для рівня В2 пріоритетним є впевнене володіння базовими та ввічливими формами висловлення думки, згоди та незгоди.

1.  **Must-have (обов'язково)**:
    *   `Я думаю / вважаю, що...`, `На мою думку, ...`
    *   `Я згоден / не згоден.`, `Ви маєте рацію.`
    *   Прості сполучники для аргументації: `тому що`, `бо`, `але`.
    *   Структурні слова: `По-перше`, `по-друге`, `наприклад`.

2.  **Should-have (дуже бажано)**:
    *   Більш нюансовані вирази: `Мені здається`, `Я переконаний`.
    *   Ввічливі форми незгоди: `Дозвольте не погодитися`, `Я розумію, але...`.
    *   Часткова згода: `Так, але...`, `З одного боку..., з іншого...`.
    *   Складніші сполучники: `проте`, `однак`, `незважаючи на те, що...`.

3.  **Nice-to-have (для просунутого В2 / С1)**:
    *   Формальна термінологія: `теза`, `аргумент`, `спростування`, `опонент`.
    *   Риторичні прийоми: риторичні запитання, посилання на авторитети, використання гумору чи іронії (Source 13, 15).
    *   Фрази для управління дискусією: `Давайте повернемося до теми`, `Дозвольте уточнити ваше запитання`.

## Типові помилки L2 (Common L2 Errors)
Англомовні студенти часто припускаються помилок через прямий переклад та відмінності в комунікативних стратегіях.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я погоджуюся до певної міри.` (калька з "I agree to an extent") | `Я частково погоджуюся.` / `Я погоджуюся лише частково.` | В українській мові `до певної міри` використовується для опису інтенсивності, а не згоди. |
| `Це робить сенс.` (калька з "It makes sense") | `Це має сенс.` / `Це логічно.` / `У цьому є сенс.` | Конструкція "робити сенс" є грубою калькою. |
| `Я не згоден **з** твоєю думкою.` (надлишковість) | `Я не згоден **із** твоєю думкою.` або `Я не погоджуюся **з** твоєю думкою.` | Хоча обидва дієслова вимагають орудного відмінка, `не згоден із` є більш природним, а `не погоджуюся з` — стилістично кращим варіантом. |
| `Приймати участь у дискусії.` | `Брати участь у дискусії.` | `Приймати участь` — поширена калька з російської (`принимать участие`). Правильно — `брати участь`. |
| Використання `відношення` замість `ставлення`. | `Моє **ставлення** до цього питання таке...` | `Відношення` — це зв'язок між об'єктами (математичний, логічний). `Ставлення` — це поводження або думка про когось/щось. `Стосунки` — зв'язки між людьми. (Source 21) |
| `Я думаю, **що** це погана ідея, **що** вона не спрацює.` (повторення `що`) | `Я думаю, **що** це погана ідея, **бо** вона не спрацює.` / `Я думаю, це погана ідея, яка не спрацює.` | В англійській мові "that" може повторюватися, в українській це звучить неприродно. Слід використовувати різні сполучники або перебудовувати речення. |

## Деколонізаційні застереження (Decolonization Notes)
**MANDATORY — never omit.** Українська риторична традиція має глибоке коріння і відрізняється від російської.

1.  **Історична тяглість**: Риторика була важливою дисципліною в Києво-Могилянській академії (XVII-XVIII ст.), а видатні українські мислителі, такі як Іларіон Київський ("Слово про закон і благодать", XI ст.), Феофан Прокопович та Григорій Сковорода, були блискучими ораторами задовго до формування сучасної російської літературної мови (Source 6, 7, 23). Це не "спільна спадщина", а частина самобутньої української інтелектуальної історії.
2.  **Радянська руйнація**: В СРСР вільна дискусія була неможливою. Будь-яка незгода з лінією партії таврувалася як "антирадянська пропаганда", а її носії — "дисиденти" — зазнавали репресій (Source 1, 10). Тому для сучасної України відродження культури дебатів є актом деколонізації та повернення до європейської традиції вільного обміну думками.
3.  **Лексичні відмінності**: Необхідно активно уникати російських кальок, які проникли в мову під час русифікації.
    *   ❌ `на мій погляд` (рос. `на мой взгляд`) → ✅ `на мою думку`
    *   ❌ `в кінці кінців` (рос. `в конце концов`) → ✅ `зрештою`, `врешті-решт`
    *   ❌ `тим не менш` (рос. `тем не менее`) → ✅ `проте`, `однак`, `а втім`
4.  **"Холодна війна" і мова**: Протистояння СРСР і Заходу було також війною наративів. Українські підпільники у своїй відповіді Черчиллю наголошували на необхідності "активно діяти для цілковитої ліквідації самого джерела цієї небезпеки" (Source 10). Це показує, що український дискурс завжди прагнув до суб'єктності та перемоги, на відміну від нав'язаної радянської риторики "мирного співіснування".

## Природні приклади (Natural Examples)
**Minimum 12 natural Ukrainian sentences** using this grammar point.

**Група 1: Висловлення позиції та аргументація**
1. `Я стверджую, що якби до будь-якого міста прибули оратор і лікар... то на лікаря і дивитися б не схотіли, а обрали б того, хто володіє словом.` (Source 6, 7, цитата Платона)
2. `Хочеш бути розумним, навчися розумно запитувати, уважно слухати, спокійно відповідати і припиняти розмову, коли нічого більше сказати.` (Source 19, 20, цитата Лафатера)
3. `Хто не знає свого минулого, той не вартий майбутнього...` (Source 25, цитата Рильського)
4. `Закони риторики виходять далеко за межі публіцистичних промов і мають широке застосування в різних мовленнєвих ситуаціях.` (Source 6, 7, перефраз)

**Група 2: Позначення незгоди та протиставлення**
5. `Дисидент - це інакодумець, людина яка думає не так як влада, це людина яка незгодна з політичною ситуацією в країні.` (Source 1)
6. `Якщо учасники дискусії... намагаються дійти якоїсь єдиної думки..., то мета полеміки зовсім інша – не досягнення згоди, а перемога над супротивником.` (Source 16, 17)
7. `Тут говорили, що…, але це не так, адже…` (Source 21, приклад спростування)
8. `У японців співрозмовники всіляко уникають слів «ні», «не можу», «не знаю».` (Source 12, приклад культурних відмінностей у вираженні незгоди)

**Група 3: Структурування дискусії та риторичні питання**
9. `В Україні вже давно точаться дискусії щодо порядку кольорів на прапорі.` (Source 9)
10. `Тепер повернімось до Василя Стуса.` (Source 1, приклад управління ходом розповіді)
11. `Може, є і серед вас Виступайченко Панас? Ви йому, братове, не давайте слова!` (Source 12, риторичне питання і заклик)
12. `Чи справедливо це? Думаю, батьки не праві.` (Source 41, риторичне питання для початку дискусії)

## Рекомендації для вправ (Activity Concepts)

-   **Phase 1 → Впізнавання та відтворення:**
    *   **Вправа 1 (Вибір):** Дати студентам діалог, де пропущені фрази згоди/незгоди. Вони мають вибрати правильний варіант із запропонованих.
    *   **Вправа 2 (Трансформація):** Перетворити пряму незгоду (`Ти не правий!`) на ввічливу (`Дозвольте з вами не погодитися, тому що...`).
    *   **Вправа 3 (Класифікація):** Розподілити список фраз на три колонки: "Згода", "Незгода", "Часткова згода".

-   **Phase 2 → Контрольована практика:**
    *   **Вправа 4 (Ланцюжок аргументів):** Вчитель дає тезу (напр., "Дистанційне навчання краще за очне"). Перший студент дає аргумент "за". Другий спростовує його і дає свій аргумент "проти". Третій спростовує другий і дає новий "за", і т.д.
    *   **Вправа 5 ("Атака запитаннями"):** Один студент захищає просту тезу (напр., "Кожен повинен мати домашнього улюбленця"). Інші ставлять йому уточнювальні та складні запитання, тренуючись "змусити опонента захищатися" (Source 13).

-   **Phase 3 → Вільна практика (Дебати):**
    *   **Вправа 6 (Структуровані дебати):** Провести навчальні дебати за правилами, описаними в (Source 43, 41). Клас ділиться на дві команди ("за" і "проти") і суддів. Теми можна взяти з підручників, наприклад:
        *   `Хіба це важко — бути до всіх і завжди добрим?` (Source 43)
        *   `Важливо для кожного розуміти минуле: своєї сім’ї, народу, країни.` (Source 43)
        *   `Необхідно відмовитися від експериментів над природою.` (Source 41)
    *   Команди готують тези, аргументи і можливі контраргументи.

## Зв'язки з іншими темами
*   **Попередні теми:**
    *   **Модальні дієслова (`могти`, `мусити`, `варто`):** Необхідні для вираження можливості, необхідності та рекомендацій.
    *   **Складнопідрядні речення (причини, умови, допустові):** Основа для побудови аргументів (`тому що`, `якщо`, `хоча`).
    *   **Кличний відмінок (Вокатив):** Для звертання до опонентів (`Шановний опоненте`, `Пані Олено`).
*   **Наступні теми:**
    *   **C1: Академічне письмо:** Навички дискусії є фундаментом для написання есе та наукових робіт.
    *   **C1: Публічний виступ:** Риторичні прийоми, засвоєні в дебатах, використовуються для підготовки промов.
    *   **Пасивний стан:** Часто використовується у формальному мовленні для безособовості та зміщення фокусу з діяча на дію (`Це питання було розглянуто...`) (Source 32).
