## Як це пояснюють у школі (How Schools Teach This)
В українській шкільній програмі односкладні речення (речення з одним головним членом) вводяться приблизно у 8-му класі [S44][S45]. Основний принцип, який підкреслюється, полягає в тому, що другий головний член у таких реченнях "не потрібний, оскільки зміст речення і так зрозумілий" [S46]. Через це односкладні речення вважаються **повними**, на відміну від неповних речень, де член речення пропущений, але його можна відновити з контексту [S47][S48].

Підручники ділять односкладні речення на дві великі групи залежно від форми головного члена [S49][S50]:
1.  **З головним членом у формі присудка**. Ця група є основною і поділяється на чотири типи:
    *   **Означено-особові** (Definite-Personal): Діяч конкретний (`я`, `ти`, `ми`, `ви`), але не названий.
    *   **Неозначено-особові** (Indefinite-Personal): Діяч — невизначена група осіб (`вони`).
    *   **Узагальнено-особові** (Generalized-Personal): Дія стосується будь-якої особи, всіх загалом (часто у прислів'ях).
    *   **Безособові** (Impersonal): Дія або стан відбуваються без діяча.
2.  **З головним членом у формі підмета**. Ця група має один тип:
    *   **Називні** (Nominative): Стверджується існування предмета чи явища.

Навчання зазвичай починається з протиставлення двоскладних та односкладних речень-синонімів, щоб показати стилістичну функцію останніх, які роблять мовлення більш динамічним і невимушеним [S51][S47]. Наприклад: `Я люблю мандрувати` (двоскладне) vs. `Люблю мандрувати` (односкладне означено-особове) [S46].

## Повна парадигма (Full Paradigm)

### 1. Речення з головним членом-присудком

#### Означено-особові (Definite-Personal)
Дія виконується конкретною, визначеною особою (`я`, `ти`, `ми`, `ви`), на яку вказує закінчення дієслова.

| Спосіб вираження | Форма дієслова | Приклади |
| :--- | :--- | :--- |
| Дійсний спосіб (Теперішній/Майбутній час) | 1 ос. одн. (`-у`/`-ю`, `-м`) | `Люблю музику.` [S52]. `Зроблю собі човна із мрій.` [S53]. |
| | 2 ос. одн. (`-еш`/`-єш`, `-ш`) | `Давно читаєш?` [S47]. `Помиєш чашки?` [S54]. |
| | 1 ос. мн. (`-емо`/`-ємо`, `-мо`) | `Уболіваємо за збірну України.` [S52]. `Скоро зберемо врожай.` [S53]. |
| | 2 ос. мн. (`-ете`/`-єте`, `-те`) | `Чекаєте вже декілька хвилин?` [S55]. |
| Наказовий спосіб | 2 ос. одн. (`-и`/`-й`) | `Не лінуйся робити ранкову зарядку.` [S52]. `Завари чаю.` [S54]. |
| | 1 ос. мн. (`-імо`/`-ймо`) | `Ходімо в зоопарк.` [S56]. |
| | 2 ос. мн. (`-іть`/`-йте`) | `Швидше збирайте врожай.` [S53]. |

#### Неозначено-особові (Indefinite-Personal)
Дія виконується невизначеними особами (уявний підмет — `вони`). Увага зосереджена на самій дії, а не на її виконавцях [S57].

| Спосіб вираження | Форма дієслова | Приклади |
| :--- | :--- | :--- |
| Дійсний спосіб (Теп./Майб. час) | 3 ос. мн. | `У нас пихатих не люблять.` [S57]. `На Рівненщині відзначатимуть річницю.` [S56]. |
| Дійсний спосіб (Мин. час) | множина | `Мені написали лист.` [S57]. `Той монастир недавно збудували.` [S56]. |
| Умовний спосіб | множина | `Швидше зібрали б урожай.` [S53]. `Пішли б до школи.` [S51]. |

#### Узагальнено-особові (Generalized-Personal)
Дія стосується будь-якої особи, кожного. Часто це прислів'я, приказки, афоризми, загальні судження. Граматично найчастіше виражені як означено-особові (2 ос. одн.) або неозначено-особові (3 ос. мн.), але відрізняються узагальненим значенням [S58].

| Форма дієслова (найчастіші) | Приклади |
| :--- | :--- |
| 2 ос. одн. (як до будь-кого) | `Бережи честь змолоду.` [S47]. `Шила в мішку не сховаєш.` [S50]. |
| 3 ос. мн. (як "всі люди") | `За одного битого двох небитих дають.` [S58]. `Лежачого не б’ють.` [S47]. |

#### Безособові (Impersonal)
Дія або стан мисляться як незалежні від особи-діяча. Підмет неможливий [S59][S60].

| Спосіб вираження | Приклади |
| :--- | :--- |
| Безособове дієслово | `Уже світає.` [S60]. `Мені не спиться.` [S60]. |
| Особове дієслово в значенні безособового | `Мете.` [S59]. `У лісі пахне грибами.` [S60]. `Тягне холодом із лісу.` [S59]. |
| Безособова форма на **-но**, **-то** | `Стежки вже розчищено.` [S59]. `На лузі траву скошено.` [S60]. `Було зафіксовано перший випадок.` [S61]. |
| Прислівник (слова категорії стану) | `Сьогодні весь день холодно.` [S59]. `На вулиці було темно.` [S60]. |
| Прислівник + інфінітив | `Приємно допомагати друзям.` [S60]. `Сьогодні гріх працювати.` [S59]. |
| Слова `треба`, `можна`, `слід`, `жаль` тощо + інфінітив | `Треба вранці замовити таксі.` [S62]. `Можна побачити все.` [S60]. `Правди не сховати.` [S59]. |
| Інфінітив (незалежний) | `Як бути?` [S59]. `Не зупинити часу.` [S62]. |
| Заперечні слова `нема(є)`, `не було`, `не буде` | `Нема на світі України, немає другого Дніпра.` [S62]. `Для сміливих немає перешкод.` [S59]. |

### 2. Речення з головним членом-підметом

#### Називні (Nominative)
Стверджують наявність предмета або явища без вказівки на дію. Головний член — іменник у називному відмінку.
| Приклади |
| :--- |
| `Тиша. Пізній вечір.` [S47] |
| `Чудесна київська осінь.` [S49] |
| `Вечірня кімната. Розчинені вікна. Весна.` [S52] |

## Частотність і пріоритети
Для рівня B2 ключовими для активного засвоєння є:

1.  **Безособові речення на `-но`, `-то`**: Це найважливіший тип для розуміння та продукування формального мовлення. Вони є високочастотним аналогом пасивних конструкцій в новинах, офіційних документах, наукових текстах. Наприклад, `було зафіксовано`, `визнано`, `розроблено` [S61][S63]. Їхнє опанування є стрибком у мовній компетенції від B1 до B2.

2.  **Означено-особові та неозначено-особові**: Це основа природного розмовного мовлення. У діалогах займенники `я`, `ти`, `ми`, `ви`, `вони` часто опускаються, і вміння будувати такі речення (`(ти) знаєш?`, `(ми) йдемо`, `(вони) кажуть`) є ознакою вільного володіння мовою [S51].

3.  **Безособові для вираження стану**: Конструкції з прислівниками (`мені холодно`, `важко зрозуміти`) та безособовими дієсловами (`хочеться`, `не спиться`) є надзвичайно поширеними для опису фізичних та емоційних станів людини [S62].

4.  **Називні речення**: Важливі для розвитку навичок письма, особливо в описах та художніх текстах для створення ефекту "кадру", статичної картини.

5.  **Узагальнено-особові**: На рівні В2 їх потрібно переважно розпізнавати в прислів'ях та афоризмах. Активне використання менш пріоритетне, ніж у попередніх типів.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Татом куплено скейтборд.` [S64] | `Скейтборд куплено.` | В українській мові дієслівні форми на `-но`, `-то` **не вживаються** з діячем в орудному відмінку. Це грубий калька з російської (`мною прочитано`). Діяч або взагалі не згадується, або речення перебудовується на двоскладне (`Тато купив скейтборд`). [S64] |
| `Я хочу спати.` (як єдиний варіант) | `Мені хочеться спати.` | Хоча `Я хочу спати` є абсолютно правильним двоскладним реченням, учні часто не використовують синонімічне і дуже поширене безособове речення, яке передає мимовільне бажання. Діяч у таких реченнях стоїть у давальному відмінку (`мені`, `тобі`, `йому`). [S62] |
| `Я не сиджу на місці.` (коли контекст вимагає безособовості) | `Мені не сидиться на місці.` | Подібно до попередньої помилки, для вираження внутрішнього стану, неможливості виконати дію через психологічні чи фізичні причини, часто вживається безособове дієслово на `-ся`. [S47] |
| `Стежки є розчищені.` | `Стежки розчищені.` (ознака) АБО `Стежки розчищено.` (результат дії) | Прямий переклад з англійської ("The paths are cleared"). Учні часто плутають дієприкметник (`розчищені`), який виражає ознаку предмета, і безособову форму на `-но`/`-то` (`розчищено`), яка наголошує на результаті дії. [S64] |
| `Ти купи плитку шоколаду.` | `Купи плитку чорного шоколаду.` | У розмовному мовленні, особливо в спонуканнях (проханнях, наказах), особовий займенник `ти` або `ви` в означено-особових реченнях є надлишковим і часто опускається для природності. [S54] |
| `Вони вже закінчили жати жито.` | `Вже закінчили жати жито.` | Подібно до попереднього, в неозначено-особових реченнях займенник `вони` опускається, коли діячі неважливі або невідомі. Збереження займенника робить їх конкретними, а не невизначеними. [S53] |

## Деколонізаційні застереження (Decolonization Notes)
1.  **Форми на `-но`, `-то` — це не русизм.** Це одна з найяскравіших рис українського синтаксису, особливо в науковому та офіційно-діловому стилях. Важливо підкреслити, що конструкція **без діяча в орудному відмінку** (`Книгу написано`, а не `*Книгу написано автором`) є питомо українською нормою. Нав'язування конструкції з орудним відмінком (`мною сделано`, `автором написано`) є прямим впливом російської граматичної традиції, якого слід уникати [S64].
2.  **Українська мова має перевагу до активних конструкцій.** На відміну від російської, де пасивні дієприкметники на `-емый`, `-имый` є дуже поширеними, в українській мові часто надається перевага активним двоскладним реченням або безособовим на `-но`, `-то`. Слід уникати надмірного вживання пасивних дієприкметників, особливо коли можна сказати простіше й природніше.
3.  **Не пояснювати через російську.** Не можна подавати українські конструкції як "варіант російської" або "відмінність від російської". Український синтаксис має власну внутрішню логіку. Наприклад, поширеність безособових речень для вираження стану (`мені холодно`, `мені хочеться`) слід пояснювати як самостійну рису української мови, а не в порівнянні з російською.
4.  **Уникати суржику в прикладах.** Наприклад, використання `на днях` (`на днях уряд оголосив`) є калькою з російської. Правильно буде `цими днями` або `нещодавно`. (Приклад з `ext-ulp_youtube-99` є транскрипцією живого мовлення і може містити такі елементи, але в навчальних матеріалах їх слід уникати).

## Природні приклади (Natural Examples)

**Група 1: Означено-особові (дія конкретної особи)**
*   `Люблю чернігівську дорогу весною, влітку, восени.` [S53]
*   `Як парость виноградної лози, плекайте мову.` [S65]
*   `Заходимо у відділення банку.` [S56]
*   `Будемо через 10 хвилин.` [S56]

**Група 2: Неозначено-особові (дія невизначених осіб)**
*   `У нашому місті щороку проводять кулінарний фестиваль.` [S57]
*   `Про це озеро розповідають легенду.` [S53]
*   `Нас зустрічають хлібом-сіллю.` [S49]

**Група 3: Узагальнено-особові (дія стосується всіх)**
*   `Друга пізнають у біді.` [S49]
*   `І чужому научайтесь, й свого не цурайтесь.` [S58]

**Група 4: Безособові (дія/стан без діяча)**
*   `Заморосило осінь у сльозу.` [S60]
*   `Пахло в повітрі торішнім листом і терпкими бруньками.` [S62]
*   `3 березня було зафіксовано перший випадок інфікування коронавірусом.` [S61]
*   `Йому ще ніколи в житті не бігалося так легко.` [S62]
*   `Нема на світі України, немає другого Дніпра.` [S62]

**Група 5: Називні (констатація факту існування)**
*   `Вечірня кімната. Розчинені вікна. Весна.` [S52]
*   `Тиша, полуденна спека. Тиша... Блакить... Сонце...` [S66]

## Рекомендації для вправ (Activity Concepts)

*   **Phase 1: Ідентифікація та трансформація**
    *   **Вправа 1 (Ідентифікація):** Надати учням набір речень (двоскладних та всіх типів односкладних). Завдання — визначити тип кожного речення. Можна використовувати таблицю, як у вправі 268 [S56].
    *   **Вправа 2 (Трансформація "Двоскладне ↔ Односкладне"):** Перетворити двоскладні речення на односкладні синоніми і навпаки.
        *   `Я люблю мандрувати` ↔ `Люблю мандрувати` [S46].
        *   `Сніг замів дорогу` ↔ `Дорогу замело снігом` [S62].
        *   `Трава вже скошена` ↔ `Траву вже скошено` [S62].

*   **Phase 2: Контрольована практика**
    *   **Вправа 3 (Fill-in-the-blanks для безособових):** Надати речення з пропущеним головним членом і список варіантів для заповнення (`пахне`, `зачинено`, `треба`, `немає` тощо), як у вправі 296 [S67].
    *   **Вправа 4 (Створення речень з дієсловом у правильній формі):** Дати інфінітив і вказати тип речення, яке потрібно утворити. Наприклад: `(відкривати), означено-особове` -> `Відкриваю духовні острови!` [S57].

*   **Phase 3: Вільне продукування**
    *   **Вправа 5 (Ситуативні завдання):** Поставити комунікативне завдання, яке природно спонукає до використання певного типу речень.
        *   **Означено-особові:** "Напишіть другу коротке повідомлення в месенджері про ваші плани на вечір." (Приклад: `Будемо через 10 хвилин`).
        *   **Неозначено-особові:** "Розкажіть про традиції святкування Різдва у вашому місті." (Приклад: `У нас співають колядки`).
        *   **Безособові:** "Опишіть погоду сьогодні вранці." (Приклад: `Світає. Надворі ще тепло.`). "Напишіть коротку новину про відкриття нової школи." (Приклад: `У місті відкрито нову школу.`).
        *   **Називні:** "Напишіть початок оповідання, створюючи атмосферу тихого зимового вечора в селі." (Приклад: `Ніч. Тиша. Засніжені дахи.`).

## Зв'язки з іншими темами
*   **Дієслово:** Ця тема безпосередньо залежить від знання дієслівних форм: особи, числа, часу, способу. Особливо важливе розуміння наказового способу для означено-особових речень.
*   **Пасивний стан:** Безособові конструкції на `-но`, `-то` є ключовим і часто вживаним синтаксичним синонімом до пасивних конструкцій. Розуміння цих форм є обов'язковим для теми пасиву в українській мові.
*   **Відмінки:** Для правильної побудови безособових речень необхідно знати давальний відмінок для позначення особи, що відчуває стан (`мені`, `йому`), та знахідний відмінок для об'єкта дії в реченнях на `-но`, `-то` (`книгу прочитано`).
*   **Стилістика:** Односкладні речення є маркером стилю. Означено-особові — розмовного та художнього. Безособові на `-но`, `-то` — офіційно-ділового та наукового. Називні — художнього. Це дозволяє перейти до аналізу текстів різних стилів.
