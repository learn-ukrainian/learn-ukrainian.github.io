## Як це пояснюють у школі (How Schools Teach This)

В українській шкільній програмі тема присудків є центральною в синтаксисі простого речення. Її вводять у 8 класі, як показують змісти підручників Заболотного, Авраменка та ОНАТІЙ (Source 1, 2, 3).

Підхід є структурним та послідовним:
1.  **Визначення присудка**: Спочатку дається загальне визначення присудка як головного члена речення, що означає дію, стан або ознаку підмета і відповідає на питання *що робить підмет?*, *хто він є?*, *який він є?* тощо (Source 2, 11, 23).
2.  **Основний поділ**: Відразу вводиться ключова класифікація присудків за будовою: **прості** (з одного слова) та **складені** (з двох і більше слів) (Source 2, 11). Це фундаментальна дихотомія, на якій будується вся подальша тема.
3.  **Простий дієслівний присудок**: Детально розглядається першим. Наголошується, що він найчастіше виражений особовим дієсловом (`Парк зустрічає...` - Source 2). Важливими уточненнями, які подаються відразу, є те, що складена форма майбутнього часу (`будемо читати`) та фразеологізми (`бити байдики`) також вважаються **простими** присудками (Source 2, 34). Це важливий момент для уникнення плутанини зі складеними присудками.
4.  **Складені присудки**: Після засвоєння простого присудка вводяться два типи складених:
    *   **Складений дієслівний присудок**: Його структура "допоміжне слово + інфінітив" пояснюється через семантичні групи допоміжних слів: а) початок/кінець/продовження дії (*почати, перестати, продовжувати*); б) модальність і волевиявлення (*могти, мусити, хотіти, бажати*) та в) короткі прикметники (*радий, згоден, ладен*) (Source 2, 11).
    *   **Складений іменний присудок**: Пояснюється за аналогічною структурною формулою: "дієслово-зв'язка + іменна частина". Підкреслюється, що зв'язка (*бути, стати, здаватися*) несе граматичне значення (час, особа), а іменна частина (іменник, прикметник, займенник тощо) — лексичне (Source 2, 16). Особлива увага приділяється пропуску зв'язки `є` в теперішньому часі, що є характерною рисою української мови (`Київ — столиця України` - Source 2).

Протягом вивчення теми учні виконують вправи на ідентифікацію типів присудків у реченнях (Source 5, 7), трансформацію простих присудків у складені (Source 8) та побудову речень з різними типами присудків.

## Повна парадигма (Full Paradigm)

Присудок (предикат) в українській мові класифікується за структурою на три основні типи.

### 1. Простий дієслівний присудок (Simple Verbal Predicate)

Складається з одного граматично самостійного слова, яке виражає і лексичне, і граматичне значення.

| Спосіб вираження | Приклад | Джерело |
| :--- | :--- | :--- |
| Особова форма дієслова (дійсний, умовний, наказовий спосіб) | *Парк **зустрічає** нас прохолодою.* | Source 2 |
| | *Тихі сосни до ранку **шумлять**...* | Source 11 |
| | ***Насолоджуймося**!!!* | Source 5 |
| Складена форма майбутнього часу | *Ми **будемо відпочивати** в Трускавці.* | Source 2 |
| | *Так, я **буду** крізь сльози **сміятись**.* | Source 34 |
| Фразеологізм | *На курорті ми зазвичай **б'ємо байдики**.* | Source 2 |
| | *Дмитро **ламав голову** над іншими питаннями.* | Source 34 |
| Інфінітив (в односкладних інфінітивних реченнях) | *Ніяк **не втну** ні «тпру», ні «ну».* | Source 21 |
| | *Тих днів далеких нам ніколи **не забути**.* | Source 34 |
| Вигук або звуконаслідувальне слово | *А зозуля у верховітті все **«ку-ку!»*** | Source 34 |

### 2. Складений дієслівний присудок (Compound Verbal Predicate)

Складається з двох компонентів: допоміжного слова (змінювана частина) та інфінітива (незмінювана частина, що несе основне лексичне значення).

**Структура:** `Допоміжне слово + Інфінітив (що робити? / що зробити?)`

| Тип допоміжного слова | Допоміжні слова | Приклад | Джерело |
| :--- | :--- | :--- | :--- |
| **Фазові дієслова** (початок, продовження, кінець дії) | *починати, продовжувати, закінчувати, стати, перестати, кинутися, взятися* | *Уранці місто **починає оживати**.* | Source 2 |
| | | *По над берегом кошлатим сосни **почали дрімати**…* | Source 20 |
| | | *Футболісти збірної України **продовжують** активно **грати**.* | Source 5 |
| **Модальні дієслова** (бажання, прагнення, можливість, необхідність) | *могти, мусити, хотіти, бажати, намагатися, уміти, мріяти, любити, пробувати* | *Пішоходи **можуть переходити** вулицю...* | Source 2 |
| | | *Добре того вчити, хто **хоче** все **знати**.* | Source 12 |
| | | *Я **хочу купити** зелений чай.* | Source 4 |
| **Короткі прикметники** та дієприкметники | *радий, згоден, ладен, здатний, повинен* | *Ми **раді поїхати** до Чернівців.* | Source 2 |
| | | *Рада б душа потрапити в рай...* | Source 12 |

### 3. Складений іменний присудок (Compound Nominal Predicate)

Складається з двох компонентів: дієслова-зв'язки (виражає граматичне значення) та іменної частини (виражає основне лексичне значення).

**Структура:** `Дієслово-зв'язка + Іменна частина`

| Компонент | Варіанти | Приклади | Джерело |
| :--- | :--- | :--- | :--- |
| **Дієслово-зв'язка** | *бути (є, був, буду), стати, ставати, робитися, здаватися, вважатися, називатися, доводитися, залишатися* | *Наш Довженко **був** найсучаснішою **людиною**.* (минулий час) | Source 8 |
| | | *Річка **стала** глибшою.* | Source 16 |
| | | *Вода **здавалася** холодною.* | Source 16 |
| | *(є) — пропущена в теперішньому часі* | *Київ — **столиця** України.* | Source 2 |
| **Іменна частина** | **Іменник** (у Н.в. або Ор.в.) | *Запорожець в очах народу **був героєм**.* | Source 8 |
| | | *Книжка – найкращий **порадник**.* | Source 23 |
| | **Прикметник** | *Вулиці Львова **романтичні**.* (зв'язка `є` пропущена) | Source 2 |
| | | *Ніч **була ясна**.* | Source 30 |
| | **Займенник** | *Це місто **моє**.* | Source 2 |
| | **Числівник** | *Львів у змаганнях **перший**.* | Source 2 |
| | **Дієприкметник** | *Харків сьогодні **запорошений**.* | Source 2 |
| | **Фразеологізм** | *Людина на Хрещатику **здається краплею в морі**.* | Source 2 |
| | Іменна частина + порівняльний сполучник (*як, мов, ніби*) | *Річка стала **ніби глибшою**.* | Source 16 |
| | | *Промені **як вії** сонячних очей.* | Source 9 |

## Частотність і пріоритети

Для рівня B2 ключовим є не просто розпізнавання типів присудків, а впевнене й доречне їх використання.

1.  **Найвищий пріоритет (must master):**
    *   **Складений іменний присудок з пропущеною зв'язкою `є`**: Це найпоширеніша структура для опису, характеристики та ідентифікації в теперішньому часі (`Він — лікар`, `Погода чудова`). Нерозуміння цієї структури блокує сприйняття базових речень.
    *   **Складений дієслівний присудок з модальними дієсловами**: Конструкції з `хотіти`, `могти`, `мусити`, `треба` є основою для вираження намірів, можливостей та обов'язків. Вони надзвичайно частотні в розмовному та письмовому мовленні (`Я хочу поїхати`, `Ми мусимо закінчити`).
    *   **Простий присудок у минулому та майбутньому часі**: Це базовий інструмент для розповіді про події.

2.  **Середній пріоритет (should master):**
    *   **Складений іменний присудок в минулому/майбутньому часі**: Важливий для описів у минулому (`Він був студентом`, `Погода була гарною`). Тут критично важливо засвоїти вибір відмінка іменної частини (називний vs. орудний).
    *   **Складений дієслівний присудок з фазовими дієсловами**: `почати/починати`, `закінчити/закінчувати`. Ці дієслова є частотними для опису процесів у динаміці (`Він почав працювати`, `Вона закінчила писати`).
    *   **Простий присудок, виражений фразеологізмом**: Хоча конкретні фразеологізми можуть бути менш частотними, розуміння принципу, що стійка сполука діє як єдиний член речення, є важливим для розуміння ідіоматичної мови.

3.  **Нижчий пріоритет (can wait / exposure is enough):**
    *   **Складений іменний присудок з рідкісними дієсловами-зв'язками**: такi як `лишатися`, `доводитися`, `називатися`. Учні на рівні В2 мають їх розпізнавати, але активне використання можна відкласти на С1.
    *   **Складений дієслівний присудок з допоміжними прикметниками**: `згоден`, `ладен`, `здатний`. Ці конструкції менш частотні, ніж з модальними дієсловами.

Ключовий стрибок від В1 до В2 полягає у переході від простого розпізнавання структур до гнучкого використання складених присудків, особливо іменного з варіативністю відмінків.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я **є** студент. | Я студент. | В українській мові дієслово-зв'язка `є` в теперішньому часі у складеному іменному присудку зазвичай опускається (Source 2, 16). Це прямий переклад з англійської "I **am** a student". |
| Вчора він **був веселий**. (Nominative) | Вчора він **був веселим**. (Instrumental) | Іменна частина, що позначає тимчасову, набуту ознаку в минулому або майбутньому часі, ставиться в орудному відмінку. Називний відмінок вказує на постійну, незмінну характеристику (Source 10, 22). |
| Вони **будуть читати** книгу. (вважаючи це складеним дієслівним присудком) | Вони **будуть читати** книгу. (це простий присудок) | Учні плутають складену форму майбутнього часу (`буду робити`), яка є **простим** присудком, зі **складеним дієслівним** присудком (`хочу робити`). І там, і там два слова, що створює плутанину (Source 2, 34). |
| Він любить читати. Я думаю, що "любить читати" - це складений дієслівний присудок. | Він **любить** (що?) **читати**. ("читати" - додаток) | Дуже поширена помилка. Якщо дії стосуються різних виконавців або якщо перше дієслово не має модального/фазового значення, а є повноцінним, інфінітив виступає додатком. Порівняйте: `Він **почав читати**` (одна дія, одна особа) vs. `Він **наказав** (комусь) **читати**` (різні виконавці) (Source 21). |
| Я не міг **допомогти**. (переклад "I could not help") | Я не міг **не допомогти**. | Українська мова часто використовує подвійне заперечення для вираження неможливості утриматися від дії.
