## Як це пояснюють у школі (How Schools Teach This)

Морфологія в українській шкільній програмі вивчається як розділ граматики, що досліджує частини мови та їхні форми [S32][S33]. Вона тісно пов'язана із синтаксисом, який вивчає, як ці форми поєднуються у словосполучення та речення, але чітко розмежовується з ним [S34][S35].

Вивчення починається в середніх класах (5-6 клас) із загального поняття про частини мови (повнозначні та службові) і поступового розгляду кожної з них [S32][S36]. Типовий порядок:
1.  **Іменник** (рід, число, відмінок, відміни).
2.  **Прикметник** (групи, ступені порівняння).
3.  **Числівник** (типи, відмінювання).
4.  **Займенник** (розряди за значенням).
5.  **Дієслово** (форми, вид, час, спосіб, дієвідміни) — зазвичай найбільш розлогий розділ [S33].
6.  **Прислівник** та службові частини мови.

У старших класах (9-11 клас) акцент зміщується з базового вивчення на поглиблений аналіз, узагальнення та **стилістичну роль** морфологічних форм [S37][S38]. На цьому етапі розглядають **морфологічні норми** та типові помилки [S39]. B2 рівень відповідає цьому етапу, де метою є не просто знання форм, а свідоме й доречне їх використання в різних стилях мовлення.

## Повна парадигма (Full Paradigm)

На рівні B2 передбачається володіння основами всіх частин мови. Контрольна точка фокусується на складних аспектах, які відрізняють впевненого користувача.

### 1. Дієслово (Verb)

Це центральна тема для B2. Ключові аспекти: вид, стан, та особливі форми.

**Пасивні дієприкметники (Passive Participles)**
Утворюються від перехідних дієслів доконаного виду.
| Суфікс | Твірний дієслівний стовбур | Приклад | Переклад |
| :--- | :--- | :--- | :--- |
| **-н-** | На голосний | прочитати → прочита**н**ий | read |
| | | побачити → побаче**н**ий | seen |
| **-ен-** | На приголосний | принести → принес**ен**ий | brought |
| | | запекти → запеч**ен**ий | baked |
| **-т-** | На -ну-, -р-, -оро- | забути → забу**т**ий | forgotten |
| | | закрити → закри**т**ий | closed |
| | | колоти → коло**т**ий | pricked |

**Дієприслівники (Gerunds / Adverbial Participles)**
| Час / Вид | Суфікс | Приклад | Переклад |
| :--- | :--- | :--- | :--- |
| **Теперішній** (недоконаний вид) | `-учи` / `-ючи` (після шиплячих) | чита**ючи** | while reading |
| | `-ачи` / `-ячи` | сто**ячи**, леж**ачи** | while standing, lying |
| **Минулий** (доконаний вид) | `-вши` (після голосного) | прочита**вши** | having read |
| | `-ши` (після приголосного) | приніс**ши** | having brought |

**Безособові форми на -но, -то**
Це незмінна форма, що позначає дію, виконану невідомим суб'єктом, і виконує роль присудка в безособовому реченні.
*   Лист **написано**. (The letter has been written.) [S40]
*   Усіх ворогів **розбито**. (All enemies have been defeated.)

### 2. Іменник (Noun)

На B2 рівні ключовим є впевнене володіння всіма відмінками, особливо кличним.

**Кличний відмінок (Vocative Case)**
| Рід / Закінчення | Відмінкове закінчення | Приклад (Н.в. → Кл.в.) |
| :--- | :--- | :--- |
| Чол. (тверда група) | **-о** | пане → пан**е**; друже → друж**е**; козаче → коза**че** |
| Чол. (м'яка, мішана група) | **-ю** | Андрію → Андрі**ю**; лікарю → лікар**ю** |
| Жін. (на -а, -я) | **-о** | мамо → мам**о**; сестро → сестр**о** |
| | **-е** | земле → земл**е**; душе → душ**е** |
| | **-є** | мріє → мрі**є** |
| | **-ю** | бабусю → бабус**ю** |

### 3. Прикметник (Adjective)

Крім базового узгодження, B2 вимагає розуміння стилістичних нюансів ступенів порівняння.

| Ступінь | Форма | Спосіб творення | Приклад | Стиль |
| :--- | :--- | :--- | :--- | :--- |
| **Вищий** | Проста | корінь + `-іш-` / `-ш-` | цікав**іш**ий, молод**ш**ий | Нейтральний, розмовний |
| | Складена | `більш` / `менш` + прикметник | **більш** цікавий | Книжний, науковий |
| **Найвищий** | Проста | `най-` + проста вищого ступеня | **най**цікавіший | Нейтральний, розмовний |
| | Складена | `найбільш` / `найменш` + прикм. | **найбільш** цікавий | Книжний, офіційний |

## Частотність і пріоритети

Для вільного спілкування на рівні B2 пріоритети такі:
1.  **Дієслово:** Абсолютний пріоритет.
    *   **Вид (Aspect):** Бездоганне розрізнення доконаного/недоконаного виду є основою часової системи. Це важливіше за будь-яку складну лексику.
    *   **Пасивні дієприкметники:** Надзвичайно поширені в писемній мові (описи, характеристики). `зроблений`, `написаний`, `збудований` — обов'язкові.
    *   **Безособові форми на -но, -то:** Дуже часті в офіційному та інформаційному стилях для позначення факту виконання дії (`вирішено`, `сказано`, `зроблено`).
2.  **Відмінки іменників:**
    *   **Родовий відмінок заперечення:** Використання родового відмінка з заперечними дієсловами (`немає часу`, `не розумію сенсу`) є маркером грамотної мови [S41].
    *   **Кличний відмінок:** Обов'язковий у звертаннях. Ігнорування є грубою помилкою.
3.  **Дієприслівники:** Поширені в книжних стилях для ускладнення речення, але в усному мовленні використовуються рідше. Розуміти їх важливо, активно використовувати — корисно для збагачення мовлення.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| 1. Дякую **вас** за допомогу. | Дякую **вам** за допомогу. | Дієслово `дякувати` вимагає давального відмінка (кому?), а не знахідного (кого?). (За аналогією з Source [S42]) |
| 2. Це **самий цікавий** фільм. | Це **найцікавіший** фільм. | В українській мові найвищий ступінь порівняння утворюється префіксом `най-`, а не словом `самий`, що є калькою з російської. |
| 3. Студент, **читаючий** книгу, сидів біля вікна. | Студент, **що читав** книгу, сидів біля вікна. (Або: Студент сидів біля вікна, **читаючи** книгу.) | Активні дієприкметники теперішнього часу (`-ачий`, `-учий`) не властиві сучасній літературній мові. Замість них вживають підрядні речення з `що` або дієприслівникові звороти. (За аналогією з Source [S42]) |
| 4. Ми говорили **за** політику. | Ми говорили **про** політику. | Неправильне вживання прийменника. `Говорити про щось` є нормою. `Говорити за щось` може вживатися в діалектах у значенні "свідчити на користь". |
| 5. **Більш розумніший** учень. | **Більш розумний** учень. АБО **Розумніший** учень. | Не можна змішувати просту і складену форми ступенів порівняння. Це морфологічна помилка, що називається плеоназмом. [S39] |
| 6. Я не маю **питання**. | Я не маю **питань**. | При запереченні з дієсловами `мати`, `знати` тощо, додаток ставиться в родовому відмінку множини або однини (залежно від контексту). [S41] |

## Деколонізаційні застереження (Decolonization Notes)

**ОБОВ'ЯЗКОВО:** Завжди наголошувати, що українська граматика є самостійною системою, а не "варіантом" чи "відхиленням" від російської.
1.  **Активні дієприкметники:** На відміну від російської, де форми на `-ущий`/`-ющий`, `-ащий`/`-ящий` є нормою (`читающий`, `говорящий`), в українській літературній мові вони є архаїчними або стилістично обмеженими (переважно термінологія). Нормативним є вживання конструкцій `той, що...`: не `бігуча дівчина`, а `дівчина, що біжить`.
2.  **Кличний відмінок:** В українській мові кличний відмінок (`пане`, `Ольго`, `земле`) є живою і обов'язковою граматичною категорією у звертанні. У російській він зберігся лише в кількох архаїчних формах (`Боже`, `отче`).
3.  **Чергування `у`/`в`, `і`/`й`:** Це не просто фонетичне явище, а важлива морфологічна норма милозвучності (евфонії), яка системно регулює вибір прийменників та сполучників для уникнення збігу голосних чи приголосних [S43]. Це відрізняє українську від російської, де такі чергування менш системні.
4.  **Слово `точка`:** Російське слово `точка` має два українські відповідники: **`крапка`** (розділовий знак, цятка) і **`точка`** (геометричне поняття, певний пункт) [S44][S45]. Вживання `точка` в значенні "крапка" є поширеним русизмом.

## Природні приклади (Natural Examples)

**Група 1: Дієприкметники (пасивні)**
*   Купана-цілована хвилями Дніпровими, люблена-голублена сивими дібровами, з колоска пахучого... наша мова. [S46]
*   Широке вікно — ніби величезна рама, у якій красується жива картина, створена природою. [S46]
*   Він був на боці УНР, тобто він воював проти більшовиків. (Source ext-ulp_youtube-142, *приклад показує відсутність активного дієприкметника "воюючий"*)

**Група 2: Дієприслівники**
*   Ідеш, і слухаєш, і чуєш рідну землю, що годує тебе не тільки хлібом і медом, а й думками... (Source [S44], *показує послідовність дій*)
*   Отак, бува, німі вітри ячать, гойдаючи небес ясну колиску. (Source ext-other_blogs-53)
*   Поспішаючи, нас таки наздогнав дощ. (Source [S42], *приклад помилки вживання, але сама форма природна*)

**Група 3: Безособові форми та речення**
*   У газетах детально описують події тижня. (Неозначено-особове, Source ext-other_blogs-55)
*   Смеркалося… Огнем кругом запалало. (Безособове, Source ext-other_blogs-55)
*   Лист написано. Було вирішено почати негайно. (Безособові на -но, -то, Source ext-other_blogs-55)

**Група 4: Відмінкові форми (Кличний, Родовий заперечення)**
*   Ревеш, вітре, та не плач, бо тобі не важко. [S47][S44]
*   Любіть травинку і тваринку, і сонце завтрашнього дня… (Кличний, Source ext-other_blogs-55)
*   Ніде немає літа. (Родовий заперечення, Source ext-other_blogs-55)

## Рекомендації для вправ (Activity Concepts)

**Phase 1: Ідентифікація та аналіз**
*   **Drill 1 (Find the form):** Дати текст (напр., з Source ext-ulp_youtube-142). Завдання: знайти всі дієприкметники, дієприслівники, безособові конструкції. Пояснити їх роль у реченні.
*   **Drill 2 (Error analysis):** Надати речення з типовими морфологічними помилками (з секції "Типові помилки L2"). Завдання: знайти помилку, пояснити її суть і виправити. [S48][S42].

**Phase 2: Контрольована практика**
*   **Drill 3 (Transformation):** Перетворити два прості речення на одне складне з дієприслівниковим зворотом. *Приклад: "Він ішов додому. Він слухав музику." → "Ідучи додому, він слухав музику."*
*   **Drill 4 (Participle formation):** Дати список дієслів доконаного виду (прочитати, збудувати, написати). Завдання: утворити пасивні дієприкметники та ввести їх у речення.
*   **Drill 5 (Case drill):** Дати діалог із пропущеними звертаннями. Завдання: вставити імена та слова ввічливості у кличному відмінку.

**Phase 3: Вільне використання**
*   **Drill 6 (Creative writing):** Написати коротке есе (5-6 речень) на задану тему, обов'язково використавши: 1) пасивний дієприкметник, 2) дієприслівник, 3) безособове речення на `-но`/`-то`. (За моделлю вправи з Source [S49]).

## Зв'язки з іншими темами

*   **Синтаксис:** Морфологія є фундаментом для синтаксису. Правильне утворення форм слів (морфологія) є умовою для правильної побудови словосполучень (узгодження, керування) та речень. Теми як **односкладні речення** (Source ext-other_blogs-55) та **дієприкметникові/дієприслівникові звороти** є прямим мостом між морфологією (форми дієслів) і синтаксисом (структура речення).
*   **Словотвір:** Якщо словотвір створює **нові слова** (ліс → лісник), то морфологія змінює **форми одного й того ж слова** (ліс, лісу, лісом) для зв'язку в реченні (Source ext-other_blogs-61). На рівні B2 важливо розрізняти ці два процеси.
*   **Стилістика:** Вибір морфологічної форми часто визначає стиль тексту. Наприклад, вибір між `більш цікавий` (книжний) та `цікавіший` (нейтральний/розмовний) є стилістичним рішенням (Source ext-other_blogs-63, 11-klas-ukrmova-zabolotnyi-2019_s0300).
