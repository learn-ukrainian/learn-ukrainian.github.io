## Як це пояснюють у школі

В українській шкільній програмі вивчення другорядних членів речення (secondary sentence members) проходить кілька етапів, поступово ускладнюючись від базового розпізнавання до глибинного синтаксичного аналізу. 

У 5 класі автори підручників, такі як Літвінова та Авраменко, вводять поняття поширеного речення. Учням пояснюють, що у структурі речення, окрім головних членів, існують другорядні, які «доповнюють інформацію, яку містить граматична основа» [S14]. За наявністю цих членів речення поділяють на поширені (extended) та непоширені (unextended) [S14], [S9].

У 8 класі (зокрема, у підручнику Заболотного) учні детально аналізують морфологічне вираження другорядних членів. На цьому етапі вводиться розрізнення на узгоджені й неузгоджені означення [S11], а також глибоко досліджуються вісім видів обставин за значенням [S8]. Школярів навчають розуміти типи синтаксичного підпорядкування: узгодження, керування та прилягання [S6].

На рівні 11 класу (підручник Ворона) програма фокусується на «складних випадках кваліфікації» (синкретизмі) [S7]. Тут розглядаються ситуації, де межа між додатком і неузгодженим означенням або обставиною стає неочевидною (наприклад, *поїздка до моря* — куди? чи яка?). Також глибоко вивчається інфінітивний додаток, прикладка та пунктуація при порівняльних зворотах [S7]. Для іноземних студентів рівня B2 саме цей етап є найрелевантнішим, оскільки він розкриває складну синтаксичну гнучкість української мови.

## Повна парадигма

Система другорядних членів речення будується на жорстких синтаксичних зв'язках. Залежності чітко структуровані: означення найчастіше залежить від підмета або додатка; додаток та обставина завжди залежать від присудка [S13], [S7].

### 1. Означення (Attribute)
Означення вказує на ознаку предмета й відповідає на питання *який? чий? котрий?* [S13]. 

| Тип | Спосіб вираження | Приклад | Зв'язок (Linkage) |
|---|---|---|---|
| **Узгоджене** | Прикметник | *осінній сад* [S11] | Узгодження (рід, число, відмінок) |
| | Дієприкметник | *опале листя* [S11] | Узгодження |
| | Займенник | *свій сад* [S11] | Узгодження |
| | Числівник | *перші кроки* [S11] | Узгодження |
| **Неузгоджене** | Іменник у непрямому відм. | *пахощі квітів* [S11] | Керування |
| | Прислівник | *дорога ліворуч* [S7] | Прилягання |
| | Інфінітив | *бажання допомогти* [S7] | Прилягання |
| | Нерозкладне сполучення | *будинок у стилі бароко* [S11] | Керування |

**Ланцюжки перетворення для означень:**
Для створення неузгодженого означення з інфінітива форма просто прилягає до абстрактного іменника:
`допомогти (дієслово, інфінітив) → бажання (абстрактний іменник) + допомогти → бажання допомогти (неузгоджене означення)` [S7].
*Обмеження:* Неузгоджені означення, виражені інфінітивом, можуть залежати лише від абстрактних іменників, що виражають волевиявлення, стан або можливість (*наказ відступати*, *право вибирати*) [S7].

Творення дієприкметників для узгоджених означень:
`опасти (дієслово, док. вид, неперехідне) → опа- (основа) → +л- + -ий → опалий (дієприкметник, означення)` [S11].
*Обмеження:* Активні дієприкметники минулого часу утворюються **тільки** від неперехідних дієслів доконаного виду `<!-- VERIFY -->`. Утворення від перехідних дієслів (наприклад, `*прочиталий книгу`) абсолютно неможливе.

### 2. Прикладка (Apposition)
Прикладка — це різновид означення, вираженого іменником, що дає предмету другу назву та ширше характеризує його [S7].

| Семантика прикладки | Приклад | Правило написання |
|---|---|---|
| Професія, рід занять | *лікар-стоматолог* [S7] | Через дефіс |
| Національність, місце | *юнак-білорус*, *дівчина-киянка* [S7] | Через дефіс |
| Вужча + ширша назва | *Дніпро-ріка* [S7] | Через дефіс |
| Ширша + вужча назва | *річка Десна* [S7] | Окремо |
| Наукові терміни | *заєць-русак*, *гриб-паразит* [S7] | Через дефіс |
| Власні назви установ/творів | *журнал «Дзвін»* [S7] | У лапках, називний відмінок |

### 3. Додаток (Object)
Додаток означає предмет, на який спрямована дія, і відповідає на питання непрямих відмінків [S7].

| Тип додатка | Умови вживання | Приклад |
|---|---|---|
| **Прямий (Direct)** | 1. Знахідний відмінок без прийменника | *берегти (що?) честь* [S7] |
| | 2. Родовий відмінок при запереченні | *не знаєш (чого?) броду* [S7] |
| | 3. Родовий відмінок (частина від цілого) | *не вип’єш (чого?) водиці* [S7] |
| **Непрямий (Indirect)** | Інші відмінки та конструкції з прийменниками | *покладатися на брата* [S7] |
| **Інфінітивний** | Дія інфінітива виконується іншою особою | *наказую (комусь) мовчати* [S7] |

### 4. Обставина (Adverbial Modifier)
Обставина характеризує узагальнену дію за просторовими, часовими або логічними параметрами [S7].

| Вид обставини | Питання | Приклад |
|---|---|---|
| Способу дії | *як? яким способом?* | *Оцю науку шанобливо я в серці ніс* [S7] |
| Міри і ступеня | *наскільки? якою мірою?* | *Тричі крига розмерзала* [S7] |
| Місця | *де? куди? звідки?* | *Вдома і стіни допомагають* [S7] |
| Часу | *коли? як довго?* | *Давно пора дитинства одцвіла* [S7] |
| Причини | *чому? з якої причини?* | *Я аж занімів від здивування* [S7] |
| Мети | *навіщо? з якою метою?* | *Такого навмисне не придумаєш* [S7] |
| Умови | *за якої умови?* | *Не взявшись за сокиру, хати не збудуєш* [S7] |
| Допусту | *незважаючи на що?* | *Незважаючи на дощ, парило* [S7] |

**Ланцюжок творення обставини (дієприслівник):**
`нанизати (дієслово, док. вид) → наниза- (основа) → +вш- + -и → нанизавши (дієприслівник, обставина)` [S7].
*Обмеження:* Дієприслівник як обставина завжди описує додаткову дію тієї самої особи чи предмета, що є підметом у реченні `<!-- VERIFY -->`. Неможливо використовувати дієприслівник, якщо дії виконують різні суб'єкти.

### 5. Порівняльний зворот (Comparative phrase)
Порівняльний зворот є членом простого речення (найчастіше — обставиною способу дії) і зазвичай відокремлюється комами [S7].
*Винятки з відокремлення:*
1. Є частиною присудка: *Мамина ласка як сонце* [S7].
2. Стоїть після часток *майже, зовсім, цілком*: *Діти міркують цілком як дорослі* [S7].
3. Є усталеним фразеологізмом: *спав як убитий*, *ллє як із відра* [S7].

## Частотність і пріоритети

Для іноземців, що вивчають українську мову на рівні B2, найвищий пріоритет мають:
1. **Прямий додаток у родовому відмінку при запереченні** (*не маю часу*). Ця форма має вживатися автоматично, оскільки її відсутність одразу видає неносія мови [S7].
2. **Узгоджені означення** [S11]. Вони складають левову частку описових текстів і вимагають автоматизму у відмінковому узгодженні.
3. **Обставини допусту та умови** (*незважаючи на*, *попри*). Це маркери ускладненого синтаксису рівня B2/C1. Вони критично важливі для побудови складної аргументації, участі в дискусіях та написання есе [S7].
4. **Інфінітивний додаток**. Дуже частотний у повсякденному спілкуванні для передачі прохань, порад чи наказів (*просити принести*, *наказати відступати*) [S7].

## Типові помилки L2 (англомовні учні)

Англомовні учні часто спираються на синтаксичну структуру рідної мови, де порядок слів жорсткий, а відмінкова система керування майже відсутня.

| ❌ Помилково | ✅ Правильно | Чому |
|---|---|---|
| *Я не бачив цю книгу.* | *Я не бачив цієї книги.* | Англійська мова (I didn't see this book) не змінює форму об'єкта при запереченні. В українській мові заперечення перехідного дієслова вимагає прямого додатка в родовому відмінку (Genitive case) [S7]. |
| *Вона попросила його, щоб він допоміг.* | *Вона попросила його допомогти.* | Учні уникають інфінітивного додатка, намагаючись передати прохання через складне речення або кальковане "for to help". Насправді дієслова волевиявлення (*просити*, *наказувати*) природно керують інфінітивним додатком, коли дія стосується іншої особи [S7]. |
| *Це є будинок з каменю.* | *Це кам'яний будинок.* або *Це будинок із каменю.* | Надмірне використання дієслова-зв'язки *є* в теперішньому часі (калька з "This is a house made of stone"). Українська мова в теперішньому часі копулу опускає, а неузгоджене означення (*з каменю*) є абсолютно самодостатнім [S7]. |
| *Він стояв міцно як дуб.* | *Він стояв міцно як дуб.* (без коми) | Англомовні учні засвоюють загальне правило виділення ком для "як" (as/like), але ігнорують, що порівняльні звороти, які набули статусу стійких фразеологізмів (*міцний як дуб*, *спав як убитий*), не відокремлюються комами [S7]. |
| *Це будинок на право.* | *Це будинок праворуч.* | Буквальний переклад англійського прийменникового словосполучення "on the right". В українській мові це неузгоджене означення, виражене прислівником (зв'язок — прилягання) [S7]. |
| *Я прочитав статтю у "Дзвін" журналі.* | *Я прочитав статтю у журналі "Дзвін".* | Англійський порядок слів часто ставить власну назву як означення перед загальною ("the Time magazine"). В українській мові прикладка-назва йде суворо після загального слова і не відмінюється, якщо загальне слово наявне в реченні [S7]. |

## Деколонізаційні застереження

Авторам модулів необхідно уникати синтаксичних та морфологічних кальок із російської мови, які тривалий час спотворювали український синтаксис та підміняли природні конструкції:

1. **Активні дієприкметники теперішнього часу.** Російська мова широко використовує форми на *-ущий*, *-ящий* (рос. *читающий мальчик*, *бегущая строка*) у ролі узгоджених означень. В українській літературній мові такі форми неприродні та ненормативні. Їх необхідно замінювати підрядними означальними реченнями (*хлопець, який читає*) або віддієслівними іменниками (*читач*, *рухомий рядок*) `<!-- VERIFY -->`.
2. **Конструкції з прийменником «по».** Калькування російського просторового *по* (рос. *гулять по парку*) є вкрай поширеною помилкою при творенні обставин. Природна українська обставина місця чи способу дії вимагає орудного відмінка без прийменника: *гуляти парком*, *іти вулицею* `<!-- VERIFY -->`.
3. **Російське «в качестве».** У ролі прикладки для позначення посади або статусу англомовне "as a teacher" іноді перекладають як *в якості вчителя*. Це грубий русизм. Єдино правильна форма — використання сполучника *як*: *працювати як учитель* `<!-- VERIFY -->`.
4. **Словосполучення «приймати участь».** Це найпоширеніша калька з російської (рос. *принимать участие*). Правильне керування в українській мові, де іменник виступає прямим додатком — *брати участь* `<!-- VERIFY -->`.
5. **Дієслово «знаходитися».** Використання цього дієслова для обставин місця (*Київ знаходиться на Дніпрі*) — це стилістична калька з російського *находиться*. Природна українська: *Київ розташований на Дніпрі* або *Київ лежить на Дніпрі* `<!-- VERIFY -->`.

## Природні приклади

Приклади згруповано за синтаксичною роллю. Усі речення є зразками природної української літературної мови.

**Група 1: Означення (узгоджені та неузгоджені)**
- *Настала осінь із вітрами холодними, з дощами колючими.* [S11] (Узгоджені означення в постпозиції для стилістичного акцентування).
- *Щире слово — теплий промінь, а зле слово — гострий ніж у серце.* [S7] (Класичні узгоджені означення перед іменником).
- *Дорога ліворуч була наглухо закрита дубовими завалами.* [S7] (Неузгоджене означення, виражене прислівником напрямку).
- *За високим парканом темніло гілля дерев.* [S7] (Неузгоджене означення, виражене іменником у родовому відмінку).

**Група 2: Додатки (прямі та інфінітивні)**
- *З поганої криниці не вип’єш доброї водиці.* [S7] (Прямий додаток у родовому відмінку, зумовлений запереченням *не* та частковістю дії).
- *Осінь матусі їсти несе: борщик у горщику, кашка у жменьці.* [S7] (Інфінітивний додаток *їсти*, що виражає об'єкт дії).
- *Козак сам собі повинен знайти долю.* [S7] (Прямий додаток *долю* у знахідному відмінку; непрямий *собі* в давальному).

**Група 3: Обставини різної семантики**
- *За морями тепліше, а вдома миліше.* [S7] (Обставини місця, виражені прийменниковим сполученням та прислівником).
- *Незважаючи на дощ, парило і робилося душно.* [S7] (Обставина допусту з характерним прийменником, виділена комою).
- *Не взявшись за сокиру, хати не збудуєш.* [S7] (Обставина умови, виражена дієприслівниковим зворотом).

**Група 4: Прикладка та порівняльні звороти**
- *В Путивлі-граді вранці-рано співає-плаче Ярославна.* [S7] (Прикладка, написана через дефіс).
- *Час летів, немов на крилах, і, мов сон, життя минало.* [S7] (Порівняльні звороти, відокремлені комами з обох боків).
- *Наче хмари, стольний Київ печеніги облягли.* [S7] (Порівняльний зворот на початку речення; *Київ* — прямий додаток).

*Додаткові приклади для ілюстрації українського контексту:*
- *Тарас Шевченко, Леся Українка та Ліна Костенко формують безперечно український літературний канон.* `<!-- VERIFY -->` (Використання канонічних імен у функції підметів, що керують прямим додатком *канон* з узгодженим означенням *український*).
- *Синьо-жовтий прапор майорить над містом.* `<!-- VERIFY -->` (Канонічне позначення кольорів українського прапора в ролі узгодженого означення).

## Рекомендації для вправ

Для авторів модулів рекомендується наступна методична прогресія під час розробки завдань:

**Фаза 1: Рецептивна ідентифікація (Розрізнення синкретизму)**
Учні часто плутають додатки та обставини місця, або додатки й неузгоджені означення. Запропонуйте вправи на постановку правильного питання:
- *Люди у шлюпці мали шестеро весел.* [S7] Учні мають усвідомити, що *у шлюпці* відповідає на питання *які?* (неузгоджене означення, залежить від підмета *люди*), а не *де?*.
- **Формат:** Множинний вибір із поясненням синтаксичної ролі та слова, від якого залежить другорядний член.

**Фаза 2: Трансформація (Перебудова конструкцій)**
Навичка заміни неузгоджених означень узгодженими критично важлива для збагачення словникового запасу та мовного чуття.
- **Завдання:** Перетворити словосполучення з прийменником на прикметникову форму [S11].
- **Приклад:** *сік із яблук* → *яблучний сік*; *вітер із півночі* → *північний вітер* [S11].
- **Формат:** Текстове поле для введення трансформованого словосполучення.

**Фаза 3: Продуктивна інтеграція (Збагачення речень)**
На рівні B2 учні повинні вміти самостійно поширювати базові граматичні основи, створюючи складні синтаксичні конструкції.
- **Завдання:** Дано просте непоширене речення. Учень повинен додати обставину допусту, порівняльний зворот або прикладку.
- **Приклад:** *Дівчина йшла.* → *Дівчина-студентка йшла вулицею, мов королева.*
- **Завдання на відпрацювання заперечення:** трансформація стверджувальних речень у заперечні з обов'язковою зміною відмінка додатка (*Я побачив літак* → *Я не побачив літака*).

## Зв'язки з іншими темами

Ця тема є ключовим синтаксичним вузлом, що з'єднує морфологію та складний синтаксис:
- **Передумови:** Вивчення другорядних членів речення спирається на глибоке розуміння відмінкової системи іменників та прикметників (A1–B1), оскільки відмінок є головним маркером синтаксичної ролі (наприклад, знахідний для прямого додатка). Також вимагається розуміння видів дієслова (доконаний/недоконаний) для правильного творення обставин, виражених дієприслівниками.
- **Майбутні кроки:** Засвоєння узгоджених і неузгоджених означень та обставин готує учнів до вивчення відокремлених членів речення (дієприкметникових та дієприслівникових зворотів) [S6]. Своєю чергою, це є прямою базою для розуміння складнопідрядних речень (підрядні означальні, з'ясувальні, обставинні) на рівнях C1–C2 [S6], оскільки підрядні частини структурно та семантично дублюють функції другорядних членів простого речення.
