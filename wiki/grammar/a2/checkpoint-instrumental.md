## Як це пояснюють у школі

Українські шкільні програми вводять поняття відмінків, зокрема орудного, поступово. Вже у 4 класі учні знайомляться з основними закінченнями орудного відмінка для іменників І та ІІ відмін, зосереджуючись на практичному застосуванні [S4]. Наприклад, підручник для 4 класу чітко розмежовує закінчення `-ою` для іменників жіночого роду з твердою основою (`красою`), `-ею` для основ на м'який приголосний або шиплячий (`зозулею`, `дачею`) та `-єю` для основ на `[й]` (`півонією`) [S4]. Аналогічний підхід застосовується до чоловічого роду: `-ом` для твердих основ (`сиром`) та `-ем`/`-єм` для м'яких і на `[й]` (`олівцем`, `гравієм`) [S4].

У 6 класі тема поглиблюється. Учні систематизують знання про відміни та групи іменників [S2]. Підручники детально розглядають парадигми відмінювання. Наприклад, підручник Заболотного для 6 класу надає таблиці відмінювання для всіх трьох груп І відміни (твердої, м'якої, мішаної) і чітко формулює правило: `-ою` для твердої групи, `-ею` для м'якої та мішаної, і `-єю` після голосного та апострофа [S2]. Також вводяться паралельні форми орудного відмінка множини, як-от `сльозами` і `слізьми` [S2]. Для ІІ відміни пояснюються закінчення `-ом`, `-ем` (`-єм`) та `-ям` для іменників середнього роду на `-я` (`життям`) [S5].

Особлива увага приділяється правопису. Тема «Букви о, е (є) в закінченнях орудного відмінка іменників І і ІІ відмін» є однією з ключових для повторення у старших класах, що свідчить про її важливість і поширеність помилок [S1]. Також розглядається подовження приголосних в орудному відмінку однини іменників ІІІ відміни, як-от `сіллю`, `тінню` [S6, S1].

Отже, шкільний підхід є послідовним: від простих практичних правил у молодших класах до повних парадигм і орфографічних нюансів у середній школі. Основний акцент робиться на розпізнаванні основи іменника (тверда, м'яка, мішана), що є ключем до вибору правильного закінчення.

## Повна парадигма

Орудний відмінок (Instrumental case) відповідає на питання *ким? чим?* і є одним із найбільш уживаних непрямих відмінків. Його форми залежать від відміни, роду та групи іменника.

### І відміна

До І відміни належать іменники жіночого, чоловічого та спільного роду із закінченням `-а`, `-я` в називному відмінку однини.

**Орудний відмінок однини**

| Група | Кінцевий приголосний основи | Правило | Приклади |
| :--- | :--- | :--- | :--- |
| **Тверда** | Твердий (нешиплячий) | Основа + `-ою` | `книга` → `книгою`, `робота` → `роботою`, `Микола` → `Миколою` [S2] |
| **М'яка** | М'який | Основа + `-ею` | `земля` → `землею`, `пісня` → `піснею`, `вишня` → `вишнею` [S2] |
| **М'яка** | `[й]` | Основа + `-єю` | `мрія` → `мрією`, `надія` → `надією`, `Софія` → `Софією` [S3, S4] |
| **Мішана** | Шиплячий (`ж`, `ч`, `ш`) | Основа + `-ею` | `площа` → `площею`, `круча` → `кручею`, `тиша` → `тишею` [S2] |

**Орудний відмінок множини**

Для всіх груп І відміни використовуються закінчення `-ами` (для твердої) та `-ями` (для м'якої та мішаної).

| Група | Правило | Приклади |
| :--- | :--- | :--- |
| **Тверда** | Основа + `-ами` | `книги` → `книгами`, `роботи` → `роботами` |
| **М'яка** | Основа + `-ями` | `землі` → `землями`, `пісні` → `піснями`, `мрії` → `мріями` |
| **Мішана** | Основа + `-ями` | `площі` → `площами`, `кручі` → `кручами` |

*Примітка:* Деякі іменники мають паралельну рідковживану форму на `-ми`: `сльозами` і `слізьми`, `свинями` і `свиньми` [S2].

### ІІ відміна

До ІІ відміни належать іменники чоловічого роду з нульовим закінченням або на `-о` та іменники середнього роду на `-о`, `-е`, `-я`.

**Орудний відмінок однини**

| Група | Тип іменника | Правило | Приклади |
| :--- | :--- | :--- | :--- |
| **Тверда** | Чоловічий і середній рід | Основа + `-ом` | `стіл` → `столом`, `батько` → `батьком`, `вікно` → `вікном` [S5] |
| **М'яка** | Чоловічий і середній рід | Основа + `-ем` | `учитель` → `учителем`, `кінь` → `конем`, `море` → `морем` [S5] |
| **М'яка** | Основа на `[й]` | Основа + `-єм` | `край` → `краєм`, `соловей` → `солов'єм` [S5] |
| **Мішана** | Чоловічий і середній рід | Основа + `-ем` | `читач` → `читачем`, `кущ` → `кущем`, `плече` → `плечем` [S5] |
| **М'яка** | Середній рід на `-я` | Основа + `-ям` | `життя` → `життям`, `знання` → `знанням`, `обличчя` → `обличчям` [S5] |

**Орудний відмінок множини**

Закінчення `-ами` (тверда група) та `-ями` (м'яка і мішана групи) є стандартними.

| Група | Правило | Приклади |
| :--- | :--- | :--- |
| **Тверда** | Основа + `-ами` | `столи` → `столами`, `батьки` → `батьками` |
| **М'яка** | Основа + `-ями` | `коні` → `конями`, `учителі` → `учителями`, `моря` → `морями` |
| **Мішана** | Основа + `-ями` | `читачі` → `читачами`, `кущі` → `кущами`, `плечі` → `плечима` |

*Примітка:* Деякі іменники мають паралельні форми на `-ми` (`конями` і `кіньми`, `крилами` і `крильми`) або виняткові форми (`очі` → `очима`, `плечі` → `плечима`) [S5].

### ІІІ відміна

До ІІІ відміни належать іменники жіночого роду з нульовим закінченням у називному відмінку та слово `мати`.

**Орудний відмінок однини**

Правило творення: **Основа + `-ю`**. Ключовим моментом є орфографічні зміни в основі.

| Кінцевий приголосний основи | Правило творення | Приклади |
| :--- | :--- | :--- |
| Один м'який або шиплячий | Подовження приголосного + `-ю` | `сіль` → `сіллю`, `ніч` → `ніччю`, `мить` → `миттю` [S6] |
| Губний (`б`, `п`, `в`, `м`, `ф`) або `р` | Апостроф + `-ю` | `любов` → `любов'ю`, `кров` → `кров'ю`, `верф` → `верф'ю`, `мати` → `матір'ю` [S6] |
| Збіг приголосних | Без подовження + `-ю` | `радість` → `радістю`, `мудрість` → `мудрістю`, `ненависть` → `ненавистю` [S6] |

**Орудний відмінок множини**

Іменники ІІІ відміни в орудному відмінку множини мають закінчення `-ами`, `-ями` (після `р`) або `-ми` (для `матері`).

| Правило | Приклади |
| :--- | :--- |
| Основа + `-ами` | `ночі` → `ночами`, `речі` → `речами` |
| Основа + `-ями` | `матері` → `матерями` [S6] |

## Частотність і пріоритети

Для учнів рівнів А2-В1 пріоритетним є засвоєння форм **орудного відмінка однини** всіх трьох відмін, оскільки вони є найбільш частотними і покривають основні комунікативні функції відмінка:

1.  **Інструмент дії:** `писати ручкою`, `різати ножем`. Це найперше і найважливіше значення.
2.  **Супровід (з ким? з чим?):** `розмовляти з другом`, `пити чай з лимоном`.
3.  **Предикатив (бути/стати ким/чим):** `Він став лікарем`, `Вона хоче бути вчителькою`.
4.  **Спосіб дії:** `йти пішки`, `говорити пошепки`.
5.  **Час:** `працювати вдень і вночі` (де `вночі` - орудний від `ніч`).
6.  **Шлях, місце:** `йти лісом`, `їхати полем`.

Форми **орудного відмінка множини** також важливі, але можуть вивчатися після впевненого засвоєння однини. Найчастіше вони вживаються у значеннях супроводу (`з друзями`) та інструменту (`малювати фарбами`).

Особливу увагу слід приділити орфографії:
- Вибір `о` чи `е` в закінченнях І та ІІ відмін (`-ом`/`-ем`, `-ою`/`-ею`).
- Подовження та апостроф в іменниках ІІІ відміни (`тінню`, `любов'ю`).

Паралельні форми на кшталт `кіньми` чи `слізьми` є менш частотними у сучасному мовленні і можуть бути представлені як стилістичний варіант на рівні В1+, а не як обов'язкова норма для А2.

## Типові помилки L2 (англомовні учні)

## Деколонізаційні застереження

При вивченні орудного відмінка важливо уникати хибних аналогій з російською мовою, оскільки існують суттєві відмінності у формах та правописі. Подавати українську граматику слід як самостійну систему, а не як варіант російської.

1.  **Іменники ІІІ відміни:** Це одна з найпомітніших розбіжностей.
    *   **Українська:** Подовження приголосного + `-ю` (`ніччю`, `сіллю`).
    *   **Російська:** М'який знак + `-ью` (`ночью`, `солью`).
    *   **Українська:** Закінчення `-істю` в словах типу `радість` (`радістю`).
    *   **Російська:** Закінчення `-остью` (`радостью`).

2.  **Закінчення іменників ІІ відміни (чоловічий рід):**
    *   **Українська:** `-ем` у мішаній групі (`слухачем`, `борщем`).
    *   **Російська:** `-ем`, але з іншою орфографією після шиплячих (`слушателем`, `борщом`).

3.  **Закінчення іменників І відміни (жіночий рід):**
    *   **Українська:** Чітке розрізнення `-ою` (тверда група) та `-ею` (м'яка/мішана): `водою`, але `землею`, `грушею`.
    *   **Російська:** Аналогічне розрізнення, але з іншою фонетикою та орфографією (`водой`, `землёй`, `грушей`). Не можна механічно переносити правила.

4.  **Паралельні форми множини:** В українській мові форми на `-ми` (`кіньми`, `гістьми`, `чобітьми`) є архаїчними або стилістично забарвленими варіантами до стандартних `-ами`/`-ями` (`конями`, `гостями`, `чоботами`). У російській мові форми на `-ьми` (`лошадьми`, `детьми`) є єдино нормативними для деяких слів. Важливо наголошувати, що українські `-ами`/`-ями` є універсальним і завжди правильним варіантом для учня.

5.  **Лексичний вплив:** Пояснюючи значення, слід використовувати автентичні українські приклади, а не кальки з російської. Наприклад, `займатися спортом` (а не `займатися з спортом`), `цікавитися музикою`.

Методична пастка — це припущення, що якщо слово звучить схоже, то воно відмінюється так само. Це не так. Кожне слово слід розглядати в межах української граматичної системи.

## Природні приклади

*   Вишиваю **ниткою**. [S4]
*   Кошу **косою**. [S4]
*   Слово вилетить **горобцем**, а вернеться **волом**. [S4]

*   Заспівай мені, бабусю, тиху колискову. (приклад, де супровід імпліцитний, але часто виражається орудним) [S2]

*   Синє небо над зеленим **гаєм** рідний край вітає з добрим **урожаєм**. [S5]

**Модель 3: Предикатив (бути, стати ким/чим)**
*   Будь мені **розрадою**, **піснею** моя зоряна. [S2]

**Модель 4: Шлях / Простір**
*   І вишні по садках стояли з **рушниками**. [S5]
*   На землю по зоряних **сходах** задуманий вечір зійшов. [S5]

**Модель 5: Спосіб дії / Стан**
*   Як людина працює з **любов'ю** — один результат. [S6]
*   (Дощем) **пахучим** день промок. (Орудний часу/стану) [S5]

## Рекомендації для вправ

**Фаза 1: Розпізнавання та ідентифікація (Receptive Skills)**
*   **Вправа "Знайди орудний":** Учень читає короткий текст і підкреслює всі іменники в орудному відмінку.
*   **Вправа "З'єднай пари":** У двох колонках подано іменники в називному та орудному відмінках. Учень має з'єднати відповідні форми.
*   **Вправа "Визнач функцію":** Учень читає речення і визначає, яку функцію виконує орудний відмінок (інструмент, супровід, предикатив тощо).

**Фаза 2: Контрольоване відтворення (Controlled Production)**
*   **Вправа "Заповни пропуски":** `Я пишу ______ (ручка).` -> `Я пишу ручкою.` [S4]
*   **Вправа "Трансформація":** Перетворити речення. `Це мій друг. Я йду гуляти.` -> `Я йду гуляти з другом.`
*   **Вправа на вибір закінчення:** `Я пишаюся своєю сестр(ою/ею).` Учень має вибрати правильне закінчення на основі групи.
*   **Вправа на правопис ІІІ відміни:** `Він говорив з (любов).` -> `Він говорив з любов'ю.` `Ніч` -> `Він працював цілу ніччю.`

**Фаза 3: Вільне відтворення (Free Production)**
*   **Вправа "Чим ти це робиш?":** Учитель показує зображення (молоток, виделка, олівець) і питає: `Що це?` (`Це молоток.`), а потім: `Що ти робиш молотком?` Учень відповідає: `Я забиваю цвях молотком.`
*   **Вправа "Мій друг":** Описати свого друга, використовуючи конструкції `Я люблю гуляти з (ім'я)...`, `Ми з ним цікавимося (хобі)...`, `Він хоче стати (професія)...`.
*   **Ситуативні діалоги:** Розіграти сценку "В магазині", купуючи `хліб з сиром`, або "Плани на вечір", домовляючись про зустріч `з друзями`.

## Зв'язки з іншими темами

**Передумови:**
*   **Відміни і групи іменників:** Без розуміння, до якої відміни та групи (твердої, м'якої, мішаної) належить іменник, неможливо правильно утворити форму орудного відмінка [S2].
*   **Роди іменників:** Рід впливає на вибір відміни, а отже, і на закінчення.
*   **Поняття основи і закінчення:** Усі правила творення орудного відмінка базуються на аналізі кінцевого приголосного основи [S4].

**Подальші теми, що уможливлюються:**
*   **Пасивні конструкції:** Орудний відмінок використовується для позначення виконавця дії в пасивних дієприкметникових зворотах (`книга, написана письменником`).
*   **Дієприслівникові звороти:** `Йдучи вулицею, він зустрів друга.` (`вулицею` - орудний відмінок).
*   **Складні синтаксичні конструкції:** Вивчення орудного відмінка є кроком до розуміння складніших речень, де він виражає обставинні відношення (часу, місця, способу дії).
*   **Прикметники:** Узгодження прикметників з іменниками в орудному відмінку (`з новим другом`, `гарною ручкою`) є наступним логічним кроком.
