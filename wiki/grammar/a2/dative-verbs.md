## Як це пояснюють у школі

В українській шкільній програмі давальний відмінок (`кому? чому?`) вводять як одну з семи фундаментальних відмінкових форм іменника [S3]. Ця тема зазвичай з'являється в 4–6 класах. Підручники пояснюють відмінки як систему, що дозволяє словам поєднуватися в реченні [S3].

Особлива увага приділяється дієсловам, що керують іменниками в певному відмінку. Хоча джерела прямо не фокусуються на тріаді `допомагати`, `дякувати`, `дзвонити`, вони надають загальні рамки та приклади, які ілюструють це правило. Наприклад, у вправах пропонують поставити іменники в правильну форму, щоб завершити речення, як-от: «Не забудьте зателефонувати (Василь Олександрович)» або «Треба допомогти (однокласник Андрій)» [S8]. Це свідчить про підхід, де граматичне правило закріплюється через практичне застосування з конкретними дієсловами.

Відмінювання особових займенників, включно з давальним відмінком (`мені`, `тобі`, `йому`, `їй`), розглядається окремо і детально, з наведенням повних парадигм у таблицях [S2, S6]. Це підкреслює важливість правильного вживання займенників у ролі непрямих додатків.

Отже, шкільний підхід такий:
1.  **Введення системи відмінків**: Учень спершу знайомиться з існуванням семи відмінків та їхніми основними питаннями [S3].
2.  **Вивчення відмінкових закінчень**: Учні вивчають, як іменники та займенники змінюють свої закінчення в кожному відмінку [S3, S2].
3.  **Керування дієслів**: Через вправи учні засвоюють, що певні дієслова вимагають після себе іменника чи займенника в конкретному, а не будь-якому, відмінку [S8].

Ця тема є базовою для розуміння структури українського речення, де зв'язки між словами передаються переважно через відмінкові форми, а не лише за допомогою прийменників, як в англійській мові.

## Повна парадигма

Ключовим для цієї теми є давальний відмінок (Д. в.), що відповідає на питання `кому?` `чому?`. Дієслова `допомагати`, `дякувати` та `дзвонити` керують іменниками та займенниками саме в цьому відмінку.

### 1. Особові займенники

Займенники в давальному відмінку є високочастотними, тому їхні форми необхідно засвоїти першочергово [S2, S6].

| Особа | Однина | Множина |
| :--- | :--- | :--- |
| 1-ша | мені | нам |
| 2-га | тобі | вам |
| 3-тя (ч.р./с.р.) | **йому** | **їм** |
| 3-тя (ж.р.) | **їй** | **їм** |

**Важлива особливість 3-ї особи:** на відміну від родового чи орудного відмінків, у давальному відмінку приставний `н` після прийменників **не з'являється**. Однак з цими дієсловами прийменники зазвичай і не вживаються.

### 2. Іменники

Закінчення іменників у давальному відмінку залежить від їхньої відміни, роду та групи (тверда, м'яка, мішана) [S3].

#### І відміна (переважно жіночий рід, деякі чоловічі)

| Група | Однина | Множина |
| :--- | :--- | :--- |
| **Тверда** (основа на твердий приголосний) | `сестра` → `сестрі` | `сестри` → `сестрам` |
| | `мама` → `мамі` | `мами` → `мамам` |
| **М'яка** (основа на м'який приголосний) | `земля` → `землі` | `землі` → `землям` |
| | `мрія` → `мрії` | `мрії` → `мріям` |
| **Мішана** (основа на шиплячий) | `душа` → `душі` | `душі` → `душам` |
| | `круча` → `кручі` | `кручі` → `кручам` |

**Правило творення (однина):**
- У твердій та м'якій групах закінчення **-і**.
- У мішаній групі закінчення **-і**.
- **Чергування приголосних основи**: `г` → `з`, `к` → `ц`, `х` → `с` перед закінченням **-і**:
  - `подру**г**а` → `подру**з**і`
  - `коле**к**а` → `коле**ц**і`
  - `свекру**х**а` → `свекру**с**і` [S3]

#### II відміна (чоловічий та середній рід)

**Чоловічий рід**

Тут спостерігаються паралельні форми, що є важливою особливістю української мови [S8].

| Основа | Закінчення 1 (пріоритетне для істот) | Закінчення 2 (пріоритетне для неістот) |
| :--- | :--- | :--- |
| Тверда група | `директор` → `директоров**і**` | `директор` → `директор**у**` |
| | `брат` → `братов**і**` | `брат` → `брат**у**` |
| М'яка група | `учитель` → `учител**еві**` | `учитель` → `учител**ю**` |
| | `водій` → `воді**єві**` | `водій` → `воді**ю**` |
| Мішана група | `слухач` → `слухач**еві**` | `слухач` → `слухач**у**` |
| | `товариш` → `товариш**еві**` | `товариш` → `товариш**у**` |

**Рекомендація щодо вживання:**
- Для назв істот (людей, тварин) краще використовувати закінчення **-ові, -еві, -єві** [S8]. Це допомагає уникнути неоднозначності.
- Для назв неістот частіше вживається закінчення **-у, -ю**.
- При збігу двох іменників чоловічого роду в давальному відмінку рекомендується використовувати різні закінчення для уникнення тавтології: `директорові заводу`, `панові лейтенанту`.

**Середній рід**

| Група | Однина | Множина |
| :--- | :--- | :--- |
| Тверда | `місто` → `міст**у**` | `міста` → `міст**ам**` |
| М'яка | `море` → `мор**ю**` | `моря` → `мор**ям**` |
| | `обличчя` → `обличч**ю**` | `обличчя` → `обличч**ям**` |

## Частотність і пріоритети

Для рівня А2 пріоритетом є засвоєння найбільш уживаних конструкцій.
1.  **Найвищий пріоритет**: конструкція з особовими займенниками. Фрази `дякую тобі`, `допоможи мені`, `я подзвоню їй` є основою повсякденного спілкування.
2.  **Середній пріоритет**: конструкція з іменниками І відміни, що позначають людей (`мама`, `сестра`, `подруга`, `вчителька`). Наприклад: `дзвонити мамі`, `допомагати сестрі`.
3.  **Нижчий пріоритет (але важливий)**: конструкція з іменниками ІІ відміни чоловічого роду (`брат`, `друг`, `директор`, `лікар`). Учень має вміти утворювати форми на `-ові`/`-еві` та `-у`/`-ю` і розуміти, що вони є взаємозамінними, хоча й мають стилістичні відтінки [S8].

Дієслово `дякувати` майже завжди вживається з особою в давальному відмінку. `Допомагати` і `дзвонити` — аналогічно. Вивчення цих трьох дієслів створює міцний фундамент для розуміння концепції дієслівного керування в українській мові.

## Типові помилки L2 (англомовні учні)

Англомовні учні часто роблять помилки, переносячи структури рідної мови на українську.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я дзвоню **його** щодня. | Я дзвоню **йому** щодня. | В англійській мові "to call someone" використовує прямий додаток (accusative). В українській дієслово `дзвонити` вимагає непрямого додатка в давальному відмінку (`кому?`). Учень помилково вживає знахідний відмінок. |
| Він допомагає **мене** з роботою. | Він допомагає **мені** з роботою. | Аналогічно до `дзвонити`. В англійській "to help someone" — прямий додаток. В українській `допомагати` вимагає давального відмінка для позначення особи, якій допомагають. |
| Дякую **ви** за допомогу. | Дякую **вам** за допомогу. | Учень використовує називний відмінок займенника (`ви`) замість давального (`вам`). Це поширена помилка на початкових етапах, коли система відмінків ще не засвоєна. |
| Я подзвоню **до мого друга**. | Я подзвоню **моєму другові** (або **другу**). | Калькування англійської конструкції "call **to** someone". Хоча в розмовній мові іноді можна почути `дзвонити до когось`, літературна норма — безприйменникова конструкція з давальним відмінком. |
| Треба дякувати **вчитель**. | Треба дякувати **вчителеві** (або **вчите-лю**). | Використання іменника в називному відмінку замість давального. Учень ігнорує відмінювання. |
| Вона допомогла **хлопцю і другу**. | Вона допомогла **хлопцеві й другові** (або **хлопцю і другові**). | Невикористання паралельних форм для уникнення монотонності. Хоча формально помилки немає, стилістично краще чергувати закінчення `-у`/`-ю` та `-ові`/`-еві`, коли два іменники ч.р. стоять поруч [S8]<!-- VERIFY -->. |

## Деколонізаційні застереження

Хоча російська мова також використовує давальний відмінок з аналогічними дієсловами, існує низка відмінностей, які можуть призвести до помилок, якщо учень спирається на знання російської. Важливо вивчати українську граматику як самостійну систему.

1.  **Форма займенника `їй`**: В українській мові форма давального відмінка для займенника `вона` — **їй** (`Я допомагаю їй`). В російській мові це **ей**. Це одна з ключових фонетичних та орфографічних відмінностей.
2.  **Паралельні форми `-ові, -еві / -у, -ю`**: В українській мові для іменників чоловічого роду ІІ відміни активно використовуються паралельні закінчення `-ові, -еві` (для істот) та `-у, -ю` [S8]. У сучасній російській мові закінчення `-ови, -еви` є застарілими або просторічними, а домінантним є `-у, -ю`. Активне вживання форм на `-ові, -еві` — виразна риса української морфології.
3.  **Кличний відмінок**: Хоча він не стосується безпосередньо цих дієслів, його наявність та активне використання в українській мові (`Дякую, друже!`) на противагу його рудиментарному стану в російській підкреслює системну відмінність. Ніколи не слід пояснювати українські граматичні явища як «варіант російських».
4.  **Дієслово `дзвонити`**: В українській мові це дієслово послідовно вимагає давального відмінка. У російському просторіччі можливі коливання та вплив конструкцій зі знахідним відмінком, що може збивати з пантелику.

## Природні приклади

Ось приклади, що ілюструють вживання цих конструкцій у природному мовленні, взяті з наданих джерел.

**Група 1: Дієслово `допомагати`**
- Приємно **допомагати друзям**. [S5]
- Треба **допомогти однокласнику Андрію**. (з конструкції «допомогти (однокласник Андрій)» [S8])
- Велике щастя — **допомагати своїм друзям**. [S6] <!-- Увага: у джерелі це частина складнішої конструкції, але модель вживання `допомагати + Д.в.` очевидна -->

**Група 2: Дієслово `дякувати`**
- **Подякуйте опонентові** за цікаве запитання. [S7]
- Ми **дякуємо вам** за розуміння. (Створено на основі типових оголошень, як у [S5])
- Я **дякую тобі** за те, що ти є. (Типова фраза, що ілюструє модель). <!-- VERIFY -->

**Група 3: Дієслово `дзвонити`**
- Не забудьте **зателефонувати Василю Олександровичу**. (з конструкції «зателефонувати (Василь Олександрович)» [S8])
- Я часто **дзвоню бабусі** в неділю. <!-- VERIFY -->
- Він обіцяв **подзвонити мені** завтра.

**Група 4: Займенники в давальному відмінку з іншими дієсловами та конструкціями**
- Нагороду — **мені**, а роботу — **тобі**. [S2]
- **Їй** кажеш «ячмінь», а вона **тобі** — «гречка». [S2]
- **Мені** не спиться. [S5]
- **Йому** ще ніколи в житті не бігалося так легко. [S5]
- Любові й щастя хочеться **людині**. [S5]

## Рекомендації для вправ

Автор-письменник має створювати вправи, що ведуть учня від простого розпізнавання до самостійного вживання.

-   **Фаза 1: Розпізнавання та базове відтворення**
    -   **Вправа «Знайди адресата»**: Учень читає речення (`Мама допомагає синові.`) і відповідає на питання (`Кому допомагає мама?`).
    -   **Вправа «Заповни пропуск займенником»**: Дано речення з пропуском і займенник у початковій формі. Учень має поставити його в давальний відмінок.
        *   `Я дякую __(ти)__ за подарунок.` → `Я дякую **тобі** за подарунок.`
-   **Фаза 2: Контрольована практика**
    -   **Вправа «Трансформація»**: Учень отримує елементи речення і має скласти з них граматично правильну фразу.
        *   `(я / дзвонити / брат)` → `Я дзвоню братові/брату.`
        *   `(вона / дякувати / вчителька)` → `Вона дякує вчительці.`
    -   **Вправа «Вибери правильне закінчення»**: Для іменників чоловічого роду учень обирає між `-у`/`-ю` та `-ові`/`-еві`.
        *   `Директор подякував (інженер__ / інженерові__).`
-   **Фаза 3: Вільне вживання**
    -   **Вправа «Склади діалог»**: Учні в парах розігрують ситуації, використовуючи ці дієслова. Наприклад: «Ти дзвониш другові, щоб подякувати йому за допомогу з домашнім завданням».
    -   **Вправа «Напиши відповідь»**: Написати коротку електронну відповідь, де треба подякувати комусь за щось або запропонувати допомогу.

## Зв'язки з іншими темами

-   **Передумови**:
    -   **A1/A2: Іменник**: учень повинен знати поняття роду та числа іменників.
    -   **A1/A2: Особові займенники**: учень має знати особові займенники в називному відмінку (`я`, `ти`, `він`...).
    -   **A2: Основи відмінкової системи**: учень повинен знати про існування відмінків, зокрема знати питання знахідного відмінка (`кого? що?`), щоб бачити різницю.
-   **Подальший розвиток**:
    -   **A2/B1: Інші дієслова, що керують давальним відмінком**: `давати`, `дарувати`, `показувати`, `пояснювати`, `казати`, `відповідати`.
    -   **B1: Безособові конструкції**: тема давального відмінка є ключовою для розуміння безособових речень, що передають стан: `мені холодно`, `йому треба`, `їй боляче`, `нам весело` [S5].
    -   **B1: Прийменники з давальним відмінком**: `завдяки кому/чому`, `всупереч кому/чому`.
