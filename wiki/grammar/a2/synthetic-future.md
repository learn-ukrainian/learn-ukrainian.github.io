## Як це пояснюють у школі (How Schools Teach This)
В українських школах майбутній час (Future Tense) вводиться як система трьох форм, що залежать від виду дієслова (Aspect). **Проста форма майбутнього часу** (Simple Future Tense) є центральною для цього брифу і нерозривно пов'язана з **доконаним видом** (Perfective Aspect) (Джерело: `10-klas-ukrmova-karaman-2018_s0325`).

Основний принцип, який викладається учням, полягає в тому, що ця форма утворюється від дієслів доконаного виду, але **відмінюється за особами й числами так само, як дієслова теперішнього часу** (Джерела: `7-klas-ukrmova-litvinova-2024_s0055`, `4-klas-ukrayinska-mova-zaharijchuk-2021-1_s0112`).

Процес пояснюється так:
1.  Беремо дієслово недоконаного виду (Imperfective): `писати`.
2.  Створюємо доконаний вид (Perfective), найчастіше за допомогою префікса: `написати` (Джерело: `7-klas-ukrmova-avramenko-2024_s0078`).
3.  Відмінюємо нове дієслово (`написати`) за тими ж правилами та з тими ж особовими закінченнями, що й дієслово `писати` в теперішньому часі (Джерело: `4-klas-ukrayinska-mova-zaharijchuk-2021-1_s0111`).

Цей підхід дозволяє учням не вивчати нову систему відмінювання, а застосовувати вже відомі їм патерни теперішнього часу до нової лексичної категорії (дієслів доконаного виду) для вираження завершеної дії в майбутньому. Підручники для 4-го класу вже вводять цю паралель: `(що роблю?) пишу -> (що зроблю?) напишу` (Джерело: `2-klas-ukrmova-kravcova-2019-1_s0038`, `4-klas-ukrayinska-mova-zaharijchuk-2021-1_s0111`).

## Повна парадигма (Full Paradigm)
Проста форма майбутнього часу утворюється від основи дієслова **доконаного виду** за допомогою тих самих особових закінчень, що і в теперішньому часі.

**Формула утворення:** `[ПРЕФІКС] + [ОСНОВА НЕДОКОНАНОГО ДІЄСЛОВА] + [ОСОБОВЕ ЗАКІНЧЕННЯ ТЕПЕРІШНЬОГО ЧАСУ]`

### 1-ша дієвідміна (First Conjugation)

| Особа | Займенник | Питання | `прочитати` | `поїхати` | Закінчення |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Однина** | | | | |
| 1 | я | що зроблю? | прочита**ю** | поїд**у** | `-у`, `-ю` |
| 2 | ти | що зробиш? | прочита**єш** | поїд**еш** | `-еш`, `-єш` |
| 3 | він/вона/воно | що зробить? | прочита**є** | поїд**е** | `-е`, `-є` |
| **Множина** | | | | |
| 1 | ми | що зробимо? | прочита**ємо** | поїд**емо** | `-емо`, `-ємо` |
| 2 | ви | що зробите? | прочита**єте** | поїд**ете** | `-ете`, `-єте` |
| 3 | вони | що зроблять? | прочита**ють** | поїд**уть** | `-уть`, `-ють` |

(Джерела: `6-klas-ukrmova-betsa-2023_s0208`, `4-klas-ukrayinska-mova-zaharijchuk-2021-1_s0112`)

### 2-га дієвідміна (Second Conjugation)
Тут також зберігаються ті ж самі чергування приголосних в 1-й особі однини, що і в теперішньому часі (`д -> дж`, `з -> ж`, `с -> ш`, `т -> ч`, `ст -> щ`). (Джерело: `7-klas-ukrmova-litvinova-2024_s0055`)

| Особа | Займенник | Питання | `побачити` | `зробити` | Закінчення |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Однина** | | | | |
| 1 | я | що зроблю? | поба**чу** (`ч` < `т`) | зробл**ю** (`л` < `б+л`) | `-у`, `-ю` |
| 2 | ти | що зробиш? | побач**иш** | зроб**иш** | `-иш`, `-їш` |
| 3 | він/вона/воно | що зробить? | побач**ить** | зроб**ить** | `-ить`, `-їть` |
| **Множина** | | | | |
| 1 | ми | що зробимо? | побач**имо** | зроб**имо** | `-имо`, `-їмо` |
| 2 | ви | що зробите? | побач**ите** | зроб**ите** | `-ите`, `-їте` |
| 3 | вони | що зроблять? | побач**ать** | зробл**ять** | `-ать`, `-ять` |

## Частотність і пріоритети
Проста форма майбутнього часу (`напишу`) є **високочастотною** і пріоритетною для рівня А2. Це основний спосіб вираження **одноразової, завершеної дії в майбутньому**. У розмовній мові та текстах вона вживається постійно для опису планів, намірів та прогнозованих результатів.

-   **Пріоритет №1:** Вивчення найуживаніших дієслів руху з префіксами: `поїду, приїду, піду, прийду`. Вони є основою для розповіді про будь-які плани (Джерело: `ext-ulp_youtube-206`).
-   **Пріоритет №2:** Засвоєння пари `зробити` / `робити`. Дієслово `зроблю` (`я зроблю бутерброди`) є ключовим для вираження наміру виконати дію (Джерело: `ext-ulp_youtube-186`).
-   **Пріоритет №3:** Опанування дієслів комунікації та сприйняття: `скажу, напишу, прочитаю, побачу, почую, зателефоную`.

Складніші форми (`писатиму`, `буду писати`) використовуються для опису процесу або повторюваної дії в майбутньому, тому їхнє активне використання може бути відкладене на користь простої форми для вираження конкретних планів на рівні А2.

## Типові помилки L2 (Common L2 Errors)
| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я буду написати емейл.` | `Я напишу емейл.` (якщо дія одноразова) АБО `Я буду писати емейл.` (якщо це процес) | Поєднання допоміжного дієслова `бути` з доконаним інфінітивом є грубою граматичною помилкою. `Бути` вживається тільки з інфінітивом недоконаного виду. (Джерело: `6-klas-ukrmova-betsa-2023_s0208`). |
| `Завтра я буду їхати в Карпати.` | `Завтра я поїду в Карпати.` | Для позначення одноразового переміщення в майбутньому (початку подорожі) використовується доконане дієслово з префіксом. Форма `буду їхати` означає сам процес перебування в дорозі. (Джерело: `ext-ulp_youtube-206`). |
| `Що ти робиш завтра?` | `Що ти зробиш завтра?` АБО `Що ти будеш робити завтра?` | Теперішній час не використовується для позначення майбутньої дії, на відміну від англійської (Present Continuous for future). Потрібно вживати одну з форм майбутнього часу. |
| `Якщо я прийду, ми дивимося фільм.` | `Якщо я прийду, ми подивимося фільм.` | В умовних реченнях майбутнього часу (після `якщо`, `коли`) обидві частини речення повинні стояти в майбутньому часі. Українська мова не використовує теперішній час у підрядній частині, як англійська (`If I come...`). |
| `Я відповім на питання, але спочатку я подумаю.` | `Я відповім на питання, але спочатку я подумаю.` | Дієслово `подумаю` (від `подумати`) є доконаним і означає "поміркувати недовго". Використання недоконаного `думатиму` означало б тривалий процес, що тут нелогічно. (Джерело: `ext-other_blogs-23`). |

## Деколонізаційні застереження (Decolonization Notes)
Хоча українська та російська мови належать до однієї мовної групи і мають схожі принципи творення видо-часових форм, існують суттєві відмінності, ігнорування яких веде до суржику та помилок.

1.  **Не тотожність префіксів**: Не можна автоматично переносити російський префікс на український корінь. Значення префіксів може не збігатися, або ж для створення доконаного виду в українській мові використовується інший префікс.
2.  **Дієслова руху**: Система префіксації дієслів руху в українській мові є автономною і має власну логіку. Спроби калькувати російські моделі (напр. `пойти за хлебом`) часто призводять до не природних для української мови конструкцій. Правильно: `піти по хліб`.
3.  **Суржик**: Поширеною помилкою є використання російських дієслів доконаного виду в українському реченні. Наприклад, вживання `позвоню` замість `зателефоную` або `подзвоню` (Джерело: `ext-ulp_youtube-250`).
4.  **Форма `*-ся`**: Українська мова часто зберігає повну форму `-ся` там, де російська використовує скорочену `-сь` після голосної. Хоча це не стосується безпосередньо майбутнього часу, це частина загального патерну уникнення калькування.

**Ключове правило**: Українську граматику слід вивчати як самостійну систему, а не як "варіант" чи "виняток" з російської. Кожне дієслово доконаного виду має вивчатися як окрема лексична одиниця (`писати`/`написати`), а не через аналогії з іншими мовами.

## Рекомендації для вправ (Activity Concepts)

-   **Phase 1: Формування та розпізнавання**
    -   **Вправа "Зроби пару"**: Дати список інфінітивів недоконаного виду (`робити`, `читати`, `дивитися`). Учні мають утворити доконаний вид за допомогою префіксів (`зробити`, `прочитати`, `подивитися`). (Джерело: `ext-other_blogs-23`).
    -   **Вправа "Трансформація"**: Дати речення в теперішньому часі. Учні мають переписати їх у простій формі майбутнього часу, додавши маркер часу (завтра, скоро). Зразок: `Ярослав слухає пісні.` -> `Завтра Ярослав послухає пісні.` (Джерело: `6-klas-ukrmova-betsa-2023_s0209`).

-   **Phase 2: Вибір правильної форми (Вид)**
    -   **Вправа "Або/Або"**: Дати речення з пропуском, де в дужках запропоновано дві форми майбутнього часу: просту (доконаний вид) і складену/складну (недоконаний вид). Учень має вибрати правильну залежно від контексту (одноразова дія vs процес). Зразок: `Завтра я (напишу / буду писати) один лист.`
    -   **Вправа "Плани на тиждень"**: Створити список справ на тиждень (напр., `понеділок: зателефонувати мамі`). Учні мають написати повні речення про свої плани: `У понеділок я зателефоную мамі.`

-   **Phase 3: Вільна практика**
    -   **Вправа "Дай відповідь"**: Запитання-відповіді. `Що ти зробиш на вихідних?` -> `Я поїду за місто і прочитаю книгу.` (Джерело: `6-klas-ukrmova-betsa-2023_s0209`).
    -   **Вправа "Що буде далі?"**: Показати картинку або початок історії. Учні мають написати 3-4 речення про те, що станеться далі, використовуючи просту форму майбутнього часу. (На основі вправи 64, Джерело: `7-klas-ukrmova-litvinova-2024_s0054`).

## Зв'язки з іншими темами

-   **Попередні теми:**
    -   **Теперішній час:** Обов'язкова передумова. Відмінювання в простій формі майбутнього часу повністю збігається з відмінюванням у теперішньому. (Джерело: `4-klas-ukrmova-zaharijchuk_s0151`).
    -   **Вид дієслова (Aspect):** Учень має розуміти різницю між доконаним (perfective) та недоконаним (imperfective) видом, щоб знати, *коли* вживати просту форму майбутнього часу. (Джерело: `7-klas-ukrmova-avramenko-2024_s0064`).

-   **Наступні теми:**
    -   **Складений і складений майбутній час (`писатиму`, `буду писати`):** Проста форма слугує точкою контрасту для введення форм майбутнього часу недоконаного виду.
    -   **Дієслова руху з префіксами:** Глибоке вивчення префіксів (`ви-`, `при-`, `за-`, `до-` та ін.) розширює можливості використання простої форми майбутнього часу.
    -   **Складні речення:** Опанування простої форми майбутнього часу є необхідним для побудови складних речень з умовою або часом у майбутньому (`Коли я приїду, ми поговоримо`).
