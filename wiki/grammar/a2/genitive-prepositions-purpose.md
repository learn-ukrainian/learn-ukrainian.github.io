## Як це пояснюють у школі

Українські шкільні підручники вводять поняття відмінків через систему **відмін** та **груп**. Цей підхід є фундаментальним для розуміння того, як іменники змінюють свої закінчення. Зазвичай, ця тема детально розглядається у 6-му класі [S1].

Підхід полягає в тому, щоб спочатку класифікувати всі іменники за стабільною ознакою, яка не змінюється при відмінюванні — їхньою належністю до однієї з чотирьох відмін. Ця класифікація базується на роді іменника та його закінченні в називному відмінку однини (початковій формі) [S1].

1.  **І відміна:** іменники жіночого, чоловічого та спільного роду на `-а(-я)` (напр., `книга`, `Микола`, `сирота`).
2.  **ІІ відміна:** іменники чоловічого роду з нульовим закінченням або на `-о` (напр., `Київ`, `батько`) та середнього роду на `-о, -е, -я` (напр., `село`, `море`, `знання`).
3.  **ІІІ відміна:** іменники жіночого роду з нульовим закінченням та слово `мати` (напр., `радість`, `ніч`).
4.  **IV відміна:** іменники середнього роду на `-а(-я)`, що при відмінюванні отримують суфікси `-ат-/-ят-, -ен-` (напр., `лоша`, `ім'я`).

Далі, іменники I та II відмін поділяють на **тверду, м'яку та мішану групи** залежно від кінцевого приголосного основи [S1]. Ця система груп допомагає передбачити, яку голосну (`о` чи `е`, `и` чи `і`) матиме закінчення у непрямих відмінках, зокрема в родовому.

Родовий відмінок (`кого?`, `чого?`) вводиться не ізольовано, а як частина повної парадигми відмінювання для кожної відміни та групи. Підручники демонструють таблиці, де учні можуть побачити всі відмінкові форми разом [S1]. Прийменники, що вимагають родового відмінка (`для`, `без`, `біля`, `до`, `від`, `з-за`, `серед`, `край` тощо), вводяться контекстуально через приклади та вправи [S7, S9, S10]. Наприклад, у вправі може бути завдання поставити іменник у правильну форму в конструкції `збір коштів для (створення)` [S1] або `їхати до (Харків)` [S7].

Такий підхід дозволяє учням не просто зазубрювати ізольовані правила, а бачити системні зв'язки всередині морфології української мови.

## Повна парадигма

Родовий відмінок (Genitive) відповідає на питання `кого?` (для істот) і `чого?` (для неістот). Його закінчення залежать від відміни, роду, числа та групи іменника.

### Родовий відмінок однини

| Відміна | Група | Закінчення | Приклади | Джерело |
| :--- | :--- | :--- | :--- | :--- |
| **І** | тверда | `-и` | книг**и**, машин**и**, Полтав**и** | [S1] |
| | м'яка | `-і`, `-ї` | пісн**і**, земл**і**, надi**ї**, Марi**ї** | [S1] |
| | мішана | `-і` | площ**і**, круч**і**, меж**і** | [S1] |
| **ІІ** | тверда (ч.р.) | `-а`, `-у` | брат**а**, стол**а**, Лондон**у**, дощ**у** | [S1, S3] |
| | м'яка (ч.р.) | `-я`, `-ю` | кра**ю**, вчител**я**, Ігор**я** | [S1] |
| | мішана (ч.р.) | `-а`, `-я` | товариш**а**, слухач**а** | <!-- VERIFY --> |
| | с.р. (-о, -е) | `-а`, `-я` | сел**а**, мор**я**, пол**я** | [S1] |
| **ІІІ** | (ж.р.) | `-і` | ноч**і**, сол**і**, радост**і**, матер**і** | [S1] |
| **IV** | (с.р.) | `-и`, `-ені` | телят**и**, лошад**и**, ім**ені** | [S1] |

**Увага:** Вибір закінчення `-а(-я)` чи `-у(-ю)` для іменників чоловічого роду ІІ відміни в родовому відмінку є однією з найскладніших тем української граматики, що детально вивчається на рівні В1. На рівні А2 учень має засвоїти найбільш вживані форми через практику.

### Родовий відмінок множини

У множині більшість іменників мають нульове закінчення або закінчення `-ів`, `-ей`.

| Відміна | Закінчення | Приклади | Джерело |
| :--- | :--- | :--- | :--- |
| **І** | нульове | книг, доріг, машин, круч | [S1] |
| | `-ей` | стат**ей**, сім**ей**, миш**ей** | [S1] |
| | `-ів` | судд**ів** | [S1] |
| **ІІ** | `-ів` | брат**ів**, стол**ів**, кра**їв** | <!-- VERIFY --> |
| | нульове | місць, сердець, морів | <!-- VERIFY --> |
| **ІІІ** | `-ей` | ноч**ей**, матер**ей** | [S1] |
| **IV** | нульове | імен, племен | <!-- VERIFY --> |

### Прийменники, що вимагають родового відмінка

Велика кількість прийменників в українській мові керує родовим відмінком.

| Прийменник | Значення | Приклад | Джерело |
| :--- | :--- | :--- | :--- |
| **для** | мета, призначення | `збір коштів **для створення** машини часу` | [S1] |
| **без** | відсутність | `кава **без цукру**` | <!-- VERIFY --> |
| **біля, коло** | близькість | `він сідав **біля міської брами**` | [S10] |
| **до** | напрямок, кінцева точка | `іти **до школи**` | [S7] |
| **від** | початкова точка, причина | `рана **від слова**` | [S7] |
| **з, із** | походження, зсередини | `вода **із криниці**` | [S10] |
| **з-за, із-за** | з протилежного боку | `тече вода **із-за гаю**` | [S7] |
| **серед** | всередині групи | `**серед** яких темніють острівці типчака` | [S9] |
| **у/в** (також `в/у`) | належність, місцезнаходження | `**у мене** є`, `**у веселій хаті**` | [S7] |
| **проти** | опозиція | `гуртує спільноту **супроти норми**` | [S7] |
| **край, поблизу** | близькість | `посадила його **край дороги**` | [S10] |

## Частотність і пріоритети

На рівні А2 учень має впевнено володіти найчастотнішими конструкціями з родовим відмінком. Пріоритети для засвоєння:

1.  **Позначення відсутності з `немає`**: Це одна з найбільш вживаних конструкцій у повсякденному мовленні.
    *   `У мене немає **часу**.`
    *   `Сьогодні немає **дощу**.`
    *   `У магазині немає **хліба**.`

2.  **Позначення належності (Possession)**: Хоча англійською це 'of', в українській мові прийменник зазвичай відсутній.
    *   `це дім **мого брата**` (my brother's house)
    *   `колір **машини**` (the car's color / the color of the car)
    *   `столиця **України**` (the capital of Ukraine)

3.  **Ключові прийменники**:
    *   **`для` (for)**: `квіти для **мами**`, `подарунок для **друга**`.
    *   **`з` (from)**: `я з **Києва**`, `сік з **яблук**`.
    *   **`до` (to)**: `йти до **магазину**`, `їхати до **Львова**` [S7].
    *   **`без` (without)**: `чай без **цукру**`, `життя без **проблем**`.
    *   **`біля` (near)**: `парк біля **дому**`, `сидіти біля **вікна**` [S10].

4.  **Кількість (після чисел 5 і більше)**: У поєднанні з числівниками родовий відмінок множини є вкрай необхідним.
    *   `п'ять **книг**`
    *   `багато **людей**`
    *   `кілька **статей**` [S1]

Конструкції з прийменниками `серед`, `проти`, `з-за` є менш частотними на початковому етапі і можуть бути засвоєні пасивно.

## Типові помилки L2 (англомовні учні)

Англомовні учні часто роблять помилки, переносячи структури рідної мови на українську.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Це подарунок `для` моя мама. | Це подарунок `для` **моєї мами**. | Прийменник `для` завжди вимагає родового відмінка. В англійській мові займенники та іменники не змінюються після прийменників (`for my mom`), що призводить до ігнорування відмінювання в українській. |
| У кімнаті немає `стілець`. | У кімнаті немає **стільця**. | Конструкція `немає` (there is no/are no) для позначення відсутності завжди керує родовим відмінком. Англомовні учні часто використовують називний відмінок за аналогією з `There is no chair`. |
| Я з `Україна`. | Я з **України**. | Прийменник `з` (from) для позначення походження вимагає родового відмінка. Помилка виникає через незмінність іменників в англійській (`from Ukraine`). |
| Це центр `від` міста. | Це центр **міста**. | Для вираження належності (possession), де англійська використовує `of` (`center **of** the city`), українська найчастіше використовує родовий відмінок без прийменника. Учні помилково шукають еквівалент для `of` і обирають `від`. |
| Я не можу жити `без` музика. | Я не можу жити `без` **музики**. | Прийменник `без` (without) завжди вимагає родового відмінка. Як і в інших випадках, помилка полягає в ігноруванні відмінювання після прийменника. |
| Я йду `в` парку. | Я йду `до` **парку**. (або `в` **парк**) | Плутанина між `до + Gen` (у напрямку до) і `в + Acc` (всередину). `до парку` означає рух у напрямку парку. `в парк` означає вхід у сам парк. Форма `в парку` (з місцевим відмінком) відповідає на питання `де?` (I am in the park), а не `куди?`. [S7] |

## Деколонізаційні застереження

Чітке розрізнення українських і російських граматичних конструкцій є критично важливим для формування правильних мовних звичок і деколонізованого мовлення.

1.  **`В Україні`, а не `на Україні`**: Це найвідоміший маркер. Прийменник `в/у` вживається з назвами суверенних держав (`в Польщі`, `в Німеччині`). Форма `на Україні` історично склалася в період, коли українські землі були частинами інших імперій, і відображала статус несамостійної, нечітко окресленої території [S7]. У сучасному офіційному та стандартному мовленні єдиною правильною формою є **`в Україні`**. Іван Огієнко писав: «Мусимо змінити стару граматичну форму й уживати тільки в Україні, в Україну, викинувши остаточно з ужитку граматичну ознаку нашого колишнього поневолення» [S7].

2.  **Причина дії: `через` vs. `із-за`**: В українській мові для позначення причини (особливо негативної) використовується прийменник `через`. Конструкція `із-за` у цьому значенні є калькою з російської (`из-за`).
    *   **Правильно (укр.)**: `Я запізнився **через** затори.`, `Змагання скасували **через** дощ.` [S7]
    *   **Неправильно (русизм)**: `Я запізнився із-за заторів.`
    *   В українській `із-за` вживається переважно для позначення руху з протилежного боку: `виглянути **із-за** рогу`, `сонце вийшло **із-за** хмари` [S7].

3.  **Мовлення про когось/щось: `про` vs. `за`**: Обговорення теми в українській мові вимагає прийменника `про`. Використання `за` є поширеним русизмом (калька з `рассказывать за`).
    *   **Правильно (укр.)**: `розповідати **про** подорож`, `турбуватися **про** дітей` [S7].
    *   **Неправильно (русизм)**: `розповідати за подорож`, `турбуватися за дітей`.

4.  **Умова: `за умови` vs. `при умові`**: Для вираження умови варто надавати перевагу конструкції з `за`.
    *   **Правильно (укр.)**: `**За потреби** він допоможе.` (`При необхідності` є калькою) [S7].
    *   **Правильно (укр.)**: `**За згодою** сторін...`
    *   **Неправильно (русизм)**: `При необхідності...`, `При згоді...`

## Природні приклади

Ці приклади з підручників ілюструють природне вживання родового відмінка в різних контекстах.

**Група 1: Призначення (з `для`)**
*   То було збір коштів **для створення** машини часу. [S1]
*   У портфелі в нас книжки, пенали, зошити й ручки **для письма**. <!-- VERIFY, adapted from S5 -->
*   Паперовий кораблик щодня перевозить **для учнів** знання. [S5]

**Група 2: Відсутність (з `немає`, `без`)**
*   За дурною головою рукам і ногам немає **спокою**. [S7]
*   Ідеш-пливеш безкраїм запашним морем, а над головою — синє небо і тремтливо-радісні дзвіночки **жайворів**. [S9] (Тут мається на увазі "дзвіночки, що належать жайворам", але також створює відчуття простору, наповненого ними)

**Група 3: Походження та напрямок (з `з`, `до`)**
*   Тепер вода **із цієї криниці** освіжає всіх спраглих. [S10]
*   А рід їх вийшов **з-за Дністра**. [S7]
*   Як двері не розчиняються, а **до порога** повертаються. [S7]

**Група 4: Близькість (з `біля`, `коло`, `край`)**
*   Вечорами він сідав **біля міської брами**. [S10]
*   Три явори посадила сестра **при дорозі**. [S7]
*   Якось, ідучи лісом, Калина викопала тонке стебло й посадила його **край дороги**. [S10]

**Група 5: Кількість**
*   Написав кілька **статей**. [S1]
*   Вивчили багато **пісень**. [S1]
*   Наче коштовні камені, спалахують у цьому різнотрав’ї іриси, червоні й жовті тюльпани та безліч інших **квітів**. [S9]

## Рекомендації для вправ

**Фаза 1: Розпізнавання та ідентифікація**
*   **Вправа "Знайди прийменник і відмінок":** Дати учням короткі речення. Вони мають підкреслити прийменник, що вимагає родового відмінка, та іменник у цьому відмінку.
    *   Приклад: `Машина стоїть біля **будинку**.`
*   **Вправа "Питання-відповідь":** Дати речення і попросити поставити питання до виділеного слова.
    *   Приклад: `Це сік з (чого?) **яблук**.`

**Фаза 2: Контрольована практика**
*   **Вправа "Заповни пропуски":** Учень має поставити іменник з дужок у правильну форму.
    *   Приклад: `Я п'ю каву без ____ (цукор).` → `цукру`
    *   `Це подарунок для ____ (сестра).` → `сестри`
*   **Вправа "Трансформація":** Перетворити стверджувальне речення на заперечне з `немає`.
    *   Приклад: `У мене є час.` → `У мене немає **часу**.`
*   **Вправа "Вибір прийменника":** Запропонувати речення з вибором з кількох прийменників.
    *   Приклад: `Я йду (в / на / до) магазину.` → `до`

**Фаза 3: Вільна практика (Продукція)**
*   **Вправа "Дай відповідь":** Учні відповідають на відкриті питання, використовуючи конструкції з родовим відмінком.
    *   Приклад: `Для кого ти купуєш квіти?`, `Без чого ти не можеш уявити свій сніданок?`
*   **Вправа "Опис":** Описати свою кімнату чи місце біля дому, використовуючи прийменники `біля`, `недалеко від`, `праворуч від`, `ліворуч від`.
*   **Вправа "Склади речення":** Дати набір слів (напр., `я / їхати / з / Львів / до / Київ`), з яких треба скласти граматично правильне речення.

## Зв'язки з іншими темами

*   **Передумови**: Для успішного засвоєння родового відмінка учень повинен знати:
    *   **Рід іменників** (чоловічий, жіночий, середній).
    *   **Поняття відмінка** як зміни форми слова.
    *   **Систему відмін та груп** (І, ІІ, ІІІ, IV відміни; тверда, м'яка, мішана групи), оскільки вона визначає правильні закінчення [S1].
    *   **Називний відмінок** як початкову форму слова.

*   **Наступні теми**: Володіння родовим відмінком є ключем до вивчення більш складних тем:
    *   **Узгодження прикметників**: `без нов**ого** стол**а**`, `для мо**єї** сестр**и**`.
    *   **Вживання числівників**: `два столи` (Н.в. мн.), але `п'ять стол**ів**` (Р.в. мн.).
    *   **Складніші прийменникові конструкції**: `залежно від`, `на відміну від`.
    *   **Синтаксис**: Родовий відмінок є частиною багатьох ідіоматичних виразів та керується багатьма дієсловами (напр. `бажати (чого?) успіху`, `досягати (чого?) мети`).
