## Як це пояснюють у школі (How Schools Teach This)

Українські шкільні підручники вводять поняття, пов'язані з Родовим відмінком, поступово, починаючи з наймолодших класів. Підхід базується на практичному застосуванні прийменників і поступовому ускладненні граматичних концепцій.

**Початковий етап (1-2 клас):**
Насамперед учнів знайомлять з прийменниками як "малими, але важливими словами", що поєднують інші слова в речення. Буквар для 1-го класу вже містить прийменник `біля` в переліку foundational prepositions (Джерело: 1-klas-bukvar-zaharijchuk-2025-2_s0108).

У 2-му класі вводиться конкретна робота з прийменниками, що вимагають зміни закінчення. Підручник Большакової (Джерело: 2-klas-ukrmova-bolshakova-2019-2_s0068) пропонує вправу "Що для кого призначено?", де діти мають утворити словосполучення з прийменником `для`, змінюючи закінчення іменника в дужках. Це перше практичне знайомство з конструкцією "прийменник + Родовий відмінок" для вираження призначення, хоча сам термін "Родовий відмінок" ще не використовується.

**Середній етап (5-6 клас):**
У 5-му класі формалізується поняття відмінків. Учні вивчають, що Родовий відмінок відповідає на питання `кого?`, `чого?` (Джерело: 5-klas-ukrmova-litvinova-2022_s0216). Підручники пропонують таблиці з назвами відмінків, питаннями та їхньою роллю в реченні.

У 6-му класі ця тема поглиблюється. Учні вже активно працюють з відмінюванням іменників за повною парадигмою, де чітко вказані форми Родового відмінка (Родовий — кого? чого? коня, річки) (Джерело: 6-klas-ukrmova-avramenko-2023_s0241). Вправи ускладнюються: потрібно не просто вставити слово, а й правильно визначити його відмінкову форму в контексті речення з прийменником. Наприклад, у вправі з підручника Бетса (Джерело: 6-klas-ukrmova-betsa-2023_s0203) учні мають поставити слово `заправка` у правильну форму після прийменника `біля`: `Автобус стояв (біля, заправка)`.

Також на цьому етапі розглядаються особливості вживання особових займенників у непрямих відмінках. Підручник Авраменка (Джерело: 6-klas-ukrmova-avramenko-2023_s0209) пояснює наголос у формах `до мене`, `біля кого`, `від чого`, що безпосередньо стосується нашої теми.

Отже, шкільний підхід такий: від конкретного до загального. Спочатку учні засвоюють практичні моделі вживання (`для мами`, `біля школи`) на інтуїтивному рівні, і лише потім вивчають граматичну теорію, що стоїть за цими моделями (Родовий відмінок, відмінювання).

## Повна парадигма (Full Paradigm)

Родовий відмінок (Genitive case) в українській мові відповідає на питання `кого?`, `чого?`. Прийменники `для`, `без`, `біля` завжди вимагають після себе іменника або займенника в Родовому відмінку.

### Іменники (Nouns)
Відмінювання залежить від роду, числа та відміни іменника.

**Чоловічий рід (Masculine)**
Ключова складність — вибір закінчення **-а/-я** або **-у/-ю** для неістот.
- **-а, -я**: для більшості конкретних, чітко окреслених предметів, назв істот, мір, населених пунктів.
- **-у, -ю**: для абстрактних понять, речовин, збірних назв, явищ природи, географічних назв (країни, річки). <!-- VERIFY -->

| Відмінок | Істота (Animate) | Неістота (Inanimate -а) | Неістота (Inanimate -у) |
| :--- | :--- | :--- | :--- |
| **Н.в.** | брат, студент | стіл, олівець | будинок, ліс |
| **Р.в.** | брат**а**, студент**а** | стол**а**, олівц**я** | будинк**у**, ліс**у** |
| _Приклад_ | для брат**а** | без стол**а** | біля будинк**у** |

**Жіночий рід (Feminine)**

| Відмінок | I відміна (тверда) | I відміна (м'яка) | III відміна |
| :--- | :--- | :--- | :--- |
| **Н.в.** | сестра, книга | земля, мрія | ніч, радість |
| **Р.в.** | сестр**и**, кни́г**и** | земл**і**, мрі**ї** | ноч**і**, радост**і** |
| _Приклад_ | для сестр**и** | без земл**і** | після ноч**і** |

**Середній рід (Neuter)**

| Відмінок | Закінчення -о | Закінчення -е | Закінчення -я |
| :--- | :--- | :--- | :--- |
| **Н.в.** | вікно, село | море, сонце | життя, завдання |
| **Р.в.** | вікн**а**, сел**а** | мор**я**, сонц**я** | житт**я**, завданн**я** |
| _Приклад_ | без вікн**а** | біля мор**я** | для житт**я** |

**Множина (Plural)**
Закінчення переважно **-ів** (для ч.р.), **-ей** або нульове (для ж.р. та с.р.).

| Відмінок | Чоловічий рід | Жіночий рід | Середній рід |
| :--- | :--- | :--- | :--- |
| **Н.в.** | студенти, брати | сестри, книги | вікна, моря |
| **Р.в.** | студент**ів**, брат**ів** | сестер, книж**ок** | вікон, мор**ів** |
| _Приклад_ | для студент**ів** | без сестер | біля вікон |

### Займенники (Pronouns)

**Особові займенники (Personal Pronouns)**
| Nominative | Genitive (для, без, біля...) |
| :--- | :--- |
| я | **мене** (для ме́не) |
| ти | **тебе** (для те́бе) |
| він, воно | **нього** (для ньо́го) |
| вона | **неї** (без не́ї) |
| ми | **нас** (для нас) |
| ви | **вас** (без вас) |
| вони | **них** (біля них) |

Зверніть увагу на приставний **н-** у формах 3-ої особи (`нього`, `неї`, `них`) після прийменників (Джерело: 6-klas-ukrmova-avramenko-2023_s0186). Також важливим є зсув наголосу при вживанні з прийменником, як показано в підручнику Авраменка (`до мéне`, `до тéбе`) (Джерело: 6-klas-ukrmova-avramenko-2023_s0209).

## Частотність і пріоритети

На рівні А2 учень повинен впевнено володіти найуживанішими конструкціями з Родовим відмінком. Пріоритети варто вибудовувати так:

1.  **Конструкція `у + Genitive + є` для вираження володіння:** Це абсолютний пріоритет, оскільки є базовим способом сказати "I have". Наприклад: `У мене є брат`. `У сестри є машина`.
2.  **Конструкція `немає + Genitive` для вираження відсутності:** Не менш важлива, ніж попередня. Є стандартним способом заперечення. Наприклад: `У мене немає часу`. `У магазині немає хліба`. (Джерело: 6-klas-ukrmova-betsa-2023_s0203, `не маю словника`).
3.  **Прийменник `для` + Genitive для вираження призначення/бенефіціара:** Надзвичайно поширена конструкція. Вводиться ще в 2-му класі (Джерело: 2-klas-ukrmova-bolshakova-2019-2_s0068). Наприклад: `Це подарунок для мами`. `Книга для дітей`.
4.  **Прийменники `з`/`від` + Genitive для вираження походження/початкової точки:** `Я з України`. `Лист від друга`.
5.  **Прийменники `біля`/`коло` + Genitive для вираження місця:** Основний спосіб сказати "near". Наприклад: `Я живу біля парку`. `Він сидів коло вікна`. Прийменник `біля` згадується вже у букварі (Джерело: 1-klas-bukvar-zaharijchuk-2025-2_s0108).
6.  **Прийменник `без` + Genitive для вираження відсутності:** `Кава без цукру`. `Я не можу жити без тебе`.

Менш пріоритетними для активного засвоєння на рівні А2 є численні інші прийменники, що вимагають Родового відмінка (напр., `проти`, `серед`, `крім`, `замість`), хоча пасивне розуміння їх є корисним. Правило закінчень `-а/-у` для чоловічого роду є складною темою, і на рівні А2 важливо засвоїти найуживаніші слова як лексичні одиниці (`біля будинку`, `до чаю`), а не намагатися вивчити всі нюанси правила.

## Типові помилки L2

Вивчення Родового відмінка носіями англійської мови супроводжується низкою типових помилок через відсутність розвиненої системи відмінків в англійській.

| ❌ Помилково (Incorrect) | ✅ Правильно (Correct) | Чому (Why) |
| :--- | :--- | :--- |
| `Це подарунок *для моя сестра.` | `Це подарунок **для моєї сестри**.` | Англомовні учні часто забувають змінювати закінчення іменника та прикметника після прийменника. В українській мові прийменник `для` вимагає Родового відмінка. |
| `Я п'ю каву *без цукор.` | `Я п'ю каву **без цукру**.` | Та сама помилка: ігнорування відмінювання після прийменника `без`. Форма `цукор` - Називний відмінок, а потрібен Родовий - `цукру`. |
| `Я не маю *машину.` | `Я не маю **машини**.` (або `У мене немає **машини**`) | Це інтерференція з англійської ("I don't have a car"), де об'єкт заперечного речення стоїть у формі, аналогічній Знахідному відмінку. В українській мові заперечення з `немає` або `не мати` вимагає Родового відмінка. (Джерело: 6-klas-ukrmova-betsa-2023_s0203 `не маю словника`). |
| `Я живу біля *старий парк.` | `Я живу біля **старого парку**.` | Помилка в узгодженні прикметника. Недостатньо змінити лише іменник, прикметник також повинен стояти в Родовому відмінку і узгоджуватися з іменником у роді та числі. |
| `Це книга *його.` | `Це **його** книга.` (якщо "his book") АБО `Ця книга — **для нього**.` (якщо "for him") | Плутанина між присвійним займенником `його` (his) та особовим у Родовому відмінку `нього` (him). Хоча форма `його` може бути і тією, і тією (Джерело: 6-klas-ukrmova-avramenko-2023_s0186), контекстне вживання різне. `книга його` граматично можливе, але `його книга` звучить природніше. |
| `Ми були біля *будинка.` | `Ми були біля **будинку**.` | Помилка у виборі закінчення `-а` чи `-у` в Родовому відмінку чоловічого роду. Це складна тема навіть для носіїв, але для багатьох поширених слів є усталена норма, яку треба запам'ятовувати. |
| `Він прийшов *без нічого.` | `Він прийшов **ні з чим**.` | Подвійне заперечення. Хоча `без нічого` є поширеною розмовною формою, стилістично і граматично правильніше вживати конструкцію `ні з чим` (Орудний відмінок) для вираження "with nothing". Це більш просунута помилка. |

## Деколонізаційні застереження

При викладанні української мови як іноземної надзвичайно важливо уникати хибних аналогій з російською та наголошувати на унікальних рисах української граматики.

1.  **Присвійний займенник `їхній`**: В українській мові для вираження присвійності ("their") використовується займенник **`їхній`** (`їхня`, `їхнє`, `їхні`). Наприклад: `Це їхня хата`. `Я знаю їхніх батьків`. Вживання форми `їх` у присвійній функції (`*їх хата`) є грубою помилкою і вважається росіянізмом (калькою з рос. `их дом`). Підручники чітко розмежовують особовий займенник `їх` (Р.в. від `вони`) і присвійний `їхній` (Джерело: 6-klas-ukrmova-avramenko-2023_s0186).

2.  **Стабільність Родового відмінка при запереченні**: В українській мові конструкція `не мати + Родовий відмінок` (`не маю часу`, `не знаю відповіді`) є стабільною та літературною нормою. У сучасній російській мові спостерігається сильна тенденція до заміни Родового відмінка Знахідним при запереченні (`я не читал эту книгу` замість `я не читал этой книги`). Навчаючи української, слід наголошувати на обов'язковості Родового відмінка для уникнення калькування.

3.  **Закінчення `-а`/`-у` в Родовому відмінку чоловічого роду**: Хоча обидві мови мають цю особливість, розподіл слів за групами часто не збігається. Не можна покладатися на знання російської для вибору правильного закінчення в українській. Слова `листопад` (місяць), `футбол`, `документ`, `метр` в українській мові мають закінчення **-а**, а в російській аналогічні слова — **-а** або **-у**. Це треба вивчати як унікальну систему української мови.

4.  **Фонетичні особливості**: Зміна наголосу при вживанні займенників з прийменниками (`на мéне`, `до тéбе`, `від ньóго`) є характерною рисою української фонетики і може відрізнятися від російських аналогів. Це слід подавати як самостійне правило української мови (Джерело: 6-klas-ukrmova-avramenko-2023_s0209).

5.  **Прийменникове керування**: Не всі прийменники керують однаковими відмінками в обох мовах. Хоча `для`, `без`, `у` (рос. `у`) збігаються, багато інших відрізняються. Наприклад, український прийменник `через` (+ Знахідний відмінок) у значенні причини відповідає російському `из-за` (+ Родовий). Важливо з самого початку привчати учнів не переносити автоматично моделі керування з однієї мови на іншу.

## Природні приклади

Приклади згруповано за прийменниками та конструкціями для демонстрації їхнього вживання в природному контексті.

**1. Прийменник `для` (for the purpose of, for the benefit of)**
- `Кістка для собаки, молоко для кота, насіння для хом'яка.` (Джерело: 2-klas-ukrmova-bolshakova-2019-2_s0068)
- `У великих і маленьких містах для дорослих і дітей організовують театральні фестивалі.` (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0155)
- `Вечорами він сідав біля міської брами й пропонував людям... кухлик свіжої джерельної води... вода для спраглих.` (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0011)

**2. Прийменник `без` (without)**
- `Без води й борщу не звариш.` (Джерело: 8-klas-ukrmova-avramenko-2025_s0034)
- `Сестра не п’є чай без лимона.` (Джерело: 6-klas-ukrmova-betsa-2023_s0203)
- `На щастя життя без несподіванок не обходиться.` (Джерело: 8-klas-ukrmova-zabolotnyi-2025_s0218)

## Рекомендації для вправ

Для ефективного засвоєння теми Родового відмінка з прийменниками `для`, `без`, `біля` рекомендується трифазний підхід.

**Phase 1: Розпізнавання та відтворення форм (Recognition & Form reproduction)**
-   **Вправа на підстановку:** Дати речення з пропуском і словом у Називному відмінку в дужках. Учень має поставити слово в правильну форму.
    *   *Приклад:* `Цей подарунок для (моя мама). -> ...для моєї мами.`
    *   *Джерело ідеї:* (Джерело: 2-klas-ukrmova-bolshakova-2019-2_s0068)
-   **Механічні drills:** Дати іменник (напр., `школа`) і попросити утворити три фрази: `для школи`, `без школи`, `біля школи`.
-   **Вправа на вибір:** `Я п'ю чай без (цукор / цукру)?`. Учень має вибрати правильний варіант.

**Phase 2: Контрольоване продукування (Controlled Production)**
-   **Побудова речень:** Дати набір слів, з яких треба скласти граматично правильне речення.
    *   *Приклад:* `кава / я / без / п'ю / молоко -> Я п'ю каву без молока.`
-   **Трансформація речень:** Перетворити стверджувальне речення на заперечне, використовуючи конструкцію `немає`.
    *   *Приклад:* `У нього є машина.` -> `У нього немає машини.`
-   **Відповіді на питання:** Поставити учням запитання, що спонукають до використання Родового відмінка.
    *   *Приклад:* `Для кого ти купуєш квіти?` -> `Я купую квіти для своєї дівчини.`

**Phase 3: Вільне спілкування (Communicative Practice)**
-   **Рольові ігри:**
    *   "У магазині": один учень — продавець, інший — покупець, який просить товари (`дайте, будь ласка, пляшку води без газу`).
    *   "Планування вечірки": учні обговорюють, що потрібно купити для вечірки (`сік для дітей`, `салат без майонезу`).
-   **Опис місця:** "Опиши свою вулицю. Що знаходиться біля твого будинку?".
-   **Особисті питання:** `Без чого ти не можеш уявити свій ранок?`, `Що ти робиш для свого здоров'я?`, `Хто для тебе є прикладом у житті?`. Це спонукає до особистісно-орієнтованого мовлення.

## Зв'язки з іншими темами

Розуміння Родового відмінка є ключовим і пов'язане з багатьма іншими граматичними темами.

-   **Передумови (Prerequisites):**
    -   **Рід іменників (Noun Genders):** Неможливо правильно поставити іменник у Родовий відмінок, не знаючи його роду, оскільки від цього залежать закінчення. (Джерело: 6-klas-ukrmova-betsa-2023_s0071).
    -   **Істота/Неістота (Animacy):** Впливає на форму Знахідного відмінка, але розуміння цієї категорії корисне і для Родового, особливо при засвоєнні правил для чоловічого роду.
    -   **Основи системи відмінків:** Учень повинен розуміти, що закінчення слів в українській мові змінюються залежно від їхньої ролі в реченні.

-   **Наступні теми (What this enables):**
    -   **Конструкція `у мене є`:** Вивчення Родового відмінка особових займенників (`мене`, `тебе`, `нього`) є обов'язковою умовою для правильного вираження володіння.
    -   **Родовий відмінок після числівників:** Після числівників 2, 3, 4 вживається Називний відмінок множини, але після 5 і більше — Родовий відмінок множини (`два студенти`, але `п'ять студентів`).
    -   **Вираження дати:** Для позначення дати використовується Родовий відмінок (`п'яте травня` - Н.в., але `я народився п'ятого травня` - Р.в.). (Джерело: 6-klas-ukrmova-avramenko-2023_s0172).
    -   **Ступені порівняння прикметників:** У конструкціях порівняння може використовуватися Родовий відмінок (`Він старший за мене / від мене`).
    -   **Складні прийменники:** Багато складних прийменників вимагають Родового відмінка (`з-поза`, `залежно від`, `на відміну від`).
