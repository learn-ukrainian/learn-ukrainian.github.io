## Як це пояснюють у школі

В українській шкільній програмі тема узгодження числівників з іменниками вводиться поступово, починаючи з молодших класів, і поглиблюється в середній школі (5-7 класи), з особливим фокусом на складних випадках у старших класах (10-11 класи) [S2, S6]. Основний підхід полягає в демонстрації трьох ключових моделей узгодження, що залежать від самого числівника.

1.  **Числівник `один` (одна, одне)**: Узгоджується з іменником у роді, числі та відмінку, як прикметник. Іменник стоїть у називному відмінку однини [S2].
2.  **Числівники `два`, `три`, `чотири`**: Вимагають від іменника форми називного відмінка множини (`два стільці`, `три сестри`, `чотири вікна`). Підручники наголошують на цьому, виправляючи типові помилки на кшталт `три професора` (неправильно) на `три професори` (правильно) [S2]. Існують окремі винятки, як-от `два друга` (родовий відмінок однини), що розглядаються окремо [S2].
3.  **Числівники від `п’ять` і більше (включно з дробовими та збірними)**: Вимагають від іменника форми родового відмінка множини (`п'ять столів`, `багато учнів`, `дві третіх програми`) [S2].

Формування правильної форми родового відмінка множини є ключовим умінням, яке залежить від відміни та групи іменника [S1, S3]. Шкільні підручники приділяють значну увагу таблицям відмінювання та практичним вправам на утворення цих форм [S1, S4, S5]. Тема збірних числівників (`двоє`, `троє` тощо) вводиться з чіткими обмеженнями: вони вживаються з назвами істот чоловічого роду, назвами малят тварин, множинними іменниками та деякими іменниками середнього роду, але ніколи — з іменниками жіночого роду [S2, S6].

Термінологія, що використовується, є стандартною: «числівник» (кількісний, збірний), «іменник», «відмінок» (називний, родовий), «число» (однина, множина), «відміна» та «група» іменників [S1, S3, S4].

## Повна парадигма

Центральним у цій темі є правило вибору форми іменника, що стоїть після кількісного числівника.

### Основні правила узгодження

| Числівник | Форма іменника | Приклад |
| :--- | :--- | :--- |
| **1** (`один`, `одна`, `одне`) | **Називний відмінок однини** (Н.в. одн.) | `один стілець`, `одна книжка`, `одне вікно` |
| **2, 3, 4** (`два`, `дві`, `три`, `чотири`) | **Називний відмінок множини** (Н.в. мн.) | `два стільці`, `дві книжки`, `три хлопці`, `чотири вікна` |
| **5...20, 30...** | **Родовий відмінок множини** (Р.в. мн.) | `п'ять стільців`, `шість книжок`, `двадцять хлопців` |
| **Складені числівники** | Форма іменника залежить від **останнього** слова | `двадцять один стілець` (як з `один`)<br>`тридцять два стільці` (як з `два`)<br>`сорок п'ять стільців` (як з `п'ять`) |

**Особливі випадки з `два`, `три`, `чотири`**:
- Іменники з суфіксом `-ин-`, що зникає у множині (селянин, громадянин), набувають форми родового відмінка однини: `два селянина`, `три киянина`, `чотири громадянина` (але: `п'ять селян`). Ця модель часто є винятком [S6].
- Слово `друг` також є винятком: `два друга`, `три друга`, `чотири друга` (Р.в. одн.) [S2].

### Утворення родового відмінка множини

Це найскладніша частина теми, оскільки форма залежить від відміни, групи, роду та наголосу іменника [S1, S6].

| Відміна / Група | Закінчення | Правило та приклади |
| :--- | :--- | :--- |
| **І відміна** (ж.р., ч.р., сп.р. на `-а, -я`) | **Нульове** | Більшість іменників. Часто відбувається вставлення **-о-, -е-**. <br>`книжка` → `книжок`<br>`пісня` → `пісень`<br>`задача` → `задач`<br>`вишня` → `вишень` [S1] |
| | `-ів` | Деякі іменники: `суддя` → `суддів`, `староста` → `старостів`. |
| | `-ей` | Невелика група: `стаття` → `статей`, `сім'я` → `сімей`, `миша` → `мишей` [S1]. |
| **ІІ відміна** (ч.р. нульове/-о; с.р. на -о,-е,-я) | **Нульове** | Іменники середнього роду: `місто` → `міст`, `море` → `морів` (але!), `поле` → `полів` (але!), `питання` → `питань`, `обличчя` → `облич`. |
| | `-ів, -їв` | Більшість іменників чоловічого роду та багато середнього.<br>ч.р.: `стіл` → `столів`, `батько` → `батьків`, `трамвай` → `трамваїв`<br>с.р.: `море` → `морів`, `поле` → `полів`, `життя` → `життів` [S6]. |
| | **Нульове** (з суфіксом `-ин`) | Іменники, що втрачають суфікс `-ин`: `українець` → `українців` (але `киянин` → `киян`, `селянин` → `селян`, `болгарин` → `болгар`) [S6]. |
| **ІІІ відміна** (ж.р. з нульовим закінченням + `мати`) | `-ей` | `ніч` → `ночей`<br>`тінь` → `тіней`<br>`мати` → `матерів` (виняток) |
| **Множинні іменники** | `-ей` або нульове | `двері` → `дверей`<br>`сани` → `саней`<br>`ножиці` → `ножиць`<br>`штани` → `штанів` (або `штаней`)<br>`гроші` → `грошей` [S6]. |

### Збірні числівники

Збірні числівники (`двоє, троє, четверо...`) мають специфічну сферу вжитку і поєднуються з іменниками у родовому відмінку множини [S2].

| З чим вживаються | Приклади |
| :--- | :--- |
| Іменники **ч.р.** (назви людей та істот) | `двоє друзів`, `четверо чоловіків`, `п'ятеро хокеїстів` [S2] |
| Іменники **с.р.** (назви малят тварин) | `троє телят`, `семеро козенят`, `двоє курчат` [S2] |
| Іменники **с.р.** (деякі неістоти) | `троє вікон`, `двоє відер` [S2] |
| **Множинні іменники** | `двоє ножиць`, `троє дверей`, `четверо штанів` [S2, S7] |

**Обмеження**: Збірні числівники **НЕ** вживаються з іменниками жіночого роду [S2].
- `дві сестри` (не `двоє сестер`)
- `три жінки` (не `троє жінок`)

## Частотність і пріоритети

Для рівнів А2-В1 найважливішими для засвоєння є:

1.  **Основна модель**: `один + Н.в. одн.`, `2-4 + Н.в. мн.`, `5+ + Р.в. мн.`. Це основа для будь-якого кількісного опису.
2.  **Утворення родового відмінка множини для найуживаніших слів**. Учням не потрібно знати всі винятки, але форми для слів як `день, рік, людина, гривня, книжка, стіл, вікно, друг` є обов'язковими.
3.  **Узгодження зі складеними числівниками** (21, 22, 25...). Правило «дивись на останнє число» є простим і дуже корисним.
4.  **Збірні числівники `двоє`, `троє`, `четверо`** у їхньому найчастотнішому вжитку: з друзями, дітьми, братами.

Менш пріоритетним є запам'ятовування паралельних форм родового відмінка (`грам` і `грамів`) або рідкісних винятків.

## Типові помилки L2 (англомовні учні)

Англомовні учні часто переносять структуру "number + plural noun" на всі українські числівники, що призводить до системних помилок.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `п'ять студенти` | `п'ять студентів` | Пряме перенесення англійської моделі "five students". Після числівників 5 і більше українська мова вимагає родового відмінка множини. |
| `два рока`<br>`три долара` | `два роки`<br>`три долари` | Хибна аналогія з російською мовою, де після `два, три, чотири` вживається родовий відмінок однини. Українська норма — називний відмінок множини [S2]. |
| `троє дівчат` | `три дівчини` | Неправильне використання збірного числівника з іменником жіночого роду. Збірні числівники не поєднуються з жіночим родом [S2, S7]. |
| `двадцять один студентів` | `двадцять один студент` | Іменник узгоджується з останнім компонентом числівника. Якщо це `один`, іменник має бути в однині. Помилка виникає через сприйняття "21" як множини. |
| `багато час` | `багато часу` | Слова `багато, мало, кілька, скільки` функціонують як числівники і вимагають родового відмінка (для незлічуваних — однини, для злічуваних — множини) [S2]. Помилка — калька з "much time". |
| `чотири олівця` | `чотири олівці` | Помилкове утворення форми іменника. Хоча винятки існують (`два друга`), загальне правило для `2,3,4` — називний відмінок множини [S2]. |

## Деколонізаційні застереження

Ключова відмінність української граматики від російської в цій темі — узгодження з числівниками `два`, `три`, `чотири`. Цей момент є маркером грамотності та полем для частих помилок.

-   **Українська модель**: `два, три, чотири` + **Називний відмінок множини**.
    -   `два столи`, `три хлопці`, `чотири вікна`.
-   **Російська модель**: `два, три, четыре` + **Родовий відмінок однини**.
    -   `два стола`, `три мальчика`, `четыре окна`.

Припущення, що українська тут функціонує "як російська", призводить до грубих помилок (су́ржику), як-от `два стула` замість `два стільці`. Важливо пояснювати учням, що українська має свою власну, відмінну логіку узгодження.

Інші моменти:
- Форми родового відмінка множини можуть відрізнятися: укр. `полів`, `морів` — рос. `полей`, `морей`.
- Збірні числівники мають схожі, але не тотожні правила вжитку.

Навчання має базуватися виключно на нормах української мови, подаючи її як самостійну систему, а не як "варіант" чи "діалект" іншої мови [S3, S7].

## Природні приклади

Приклади згруповано за правилом, яке вони ілюструють.

**Модель 1: Числівники `два`, `три`, `чотири` + Н.в. мн.**

1.  Якщо ластівки або стрижі літають високо, буде сонячно. Якщо ж вони літають біля землі, слід чекати дощу [S1]. (Ілюструє форму множини, хоч і без числівника).
2.  Два чоловіка й три жінки. [S2]
3.  На малому деревці з такою гілкою рогатою лузають зиму два горобці. (Адаптовано з [S8], де було `горобці`).
4.  І дає листок в клітинку із задачами Іринка. (Адаптовано з [S1], де було `з задачами` -> `із задача` -> `дві задачі`).

**Модель 2: Числівники `5+` та `багато/кілька` + Р.в. мн.**

5.  Чекай у найближчі шість-вісім годин дощу [S1].
6.  Кільканадцять сторінок [S2].
7.  Мільйон запитань [S2].
8.  Після вистави публіка влаштувала овацію п'ятьом солістам балету. (Адаптовано з [S7], де був один соліст).

**Модель 3: Узгодження зі складеними числівниками**

9.  Двадцять один учень отримав найвищий бал. (Сконструйовано за правилом з [S2]).
10. Двадцять три селянина працювали в полі [S2]. (Виняток з Р.в. одн.).
11. Сімдесят чотири учні поїхали на екскурсію (Помилково в [S2] як `учнів`, має бути `учні`).

**Модель 4: Збірні числівники**

12. Двоє друзів пішли в кіно [S2].
13. У бабусі було семеро козенят [S2].
14. Нам поставили троє вікон [S2].
15. Для ремонту знадобилося двоє ножиць [S7].

## Рекомендації для вправ

Процес навчання варто розділити на три фази.

**Фаза 1: Розпізнавання та базове утворення форм**
-   **Вправа "Вибери правильний варіант"**: `У мене є (два брати / два брата)`. `Я купив (п'ять яблук / п'ять яблука)`.
-   **Вправа "З'єднай"**: Поєднати числівники (1, 2, 5, 10) з відповідними формами іменників (`стіл, столи, столів`).
-   **Вправа на утворення родового відмінка множини для 10-15 найчастотніших слів**: `одна книжка` → `п'ять книжок`.

**Фаза 2: Контрольоване застосування в реченнях**
-   **Вправа "Заповни пропуски"**: `У тижні ___ (7, день)`. → `сім днів`.
-   **Вправа "Дай відповідь на питання"**: `Скільки у вас уроків сьогодні? (6)` → `У нас сьогодні шість уроків`.
-   **Трансформація речень**: Змінити однину на множину: `Я бачу одного хлопця.` → `Я бачу п'ятьох хлопців.`

**Фаза 3: Вільне мовлення та письмо**
-   **Опис зображення**: "Розкажіть, що ви бачите на малюнку. Скільки там людей, тварин, предметів?"
-   **Рольова гра "У магазині"**: Практика використання числівників з іменниками при покупках (`дайте мені два кілограми помідорів`).
-   **Письмове завдання**: Написати короткий текст про свою сім'ю, друзів чи клас, вживаючи числівники (`У моїй групі десять студентів: шість дівчат і четверо хлопців...`).

## Зв'язки з іншими темами

-   **Передумови**:
    -   **Іменник: Рід і Число**: Учень має розуміти категорії роду та числа.
    -   **Відмінки іменників**: Обов'язкове знання називного та родового відмінків. Без розуміння системи відмінків ця тема неможлива [S4].
    -   **Відміни та групи іменників**: Знання відміни і групи допомагає правильно утворювати форму родового відмінка множини [S1, S3].

-   **Наступні теми**:
    -   **Час**: Для відповіді на питання "о котрій годині?" (`о п'ятій годині`).
    -   **Вік**: Розповідь про вік (`Мені двадцять один рік`).
    -   **Дати**: Називання дати (`шосте лютого`) [S8].
    -   **Присвійні займенники у множині**: Узгодження в конструкціях типу `мої три друга`.
