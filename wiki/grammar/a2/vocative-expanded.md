## Як це пояснюють у школі

У системі шкільної мовної освіти України тема кличного відмінка не подається одноразово, а розгортається концентричною спіраллю, щоразу поглиблюючи рівень складності. У молодших класах та на рівні A1 учні засвоюють базову здатність змінювати одне слово. Натомість у 6 класі школярі вперше систематично стикаються зі складеними (поширеними) звертаннями, де форма кличного відмінка застосовується до синтаксичних поєднань імені, по батькові, прізвища та загальних назв [S1]. Підручник 6 класу підкреслює, що в офіційному мовленні рекомендовані цілком конкретні варіанти звертання до особи, і ці варіанти вимагають від мовця впевненого володіння морфологією кличного відмінка [S1]. Саме на цьому етапі закладається фундаментальне правило про незмінність прикметників і займенників у таких конструкціях.

У 8 класі фокус зміщується на синтаксичну роль звертань, їхню інтонаційну природу та пунктуаційне оформлення. Школярів навчають обов'язково виділяти звертання комами або знаком оклику на письмі, а також конструювати повні репліки етикетного характеру, такі як формули привітання чи прощання [S4]. Це формує комунікативну рамку: звертання ніколи не існує у вакуумі, воно є тригером для подальшої розмови.

У старшій школі (10–11 класи) поширені звертання розглядаються крізь призму жорсткої морфологічної та стилістичної норми. Підручники відкрито борються з наслідками русифікації, категорично наголошуючи, що форма називного відмінка у звертаннях абсолютно не відповідає нормам сучасної української мови [S2]. Старшокласники вивчають найскладніші нюанси, які часто викликають труднощі навіть у носіїв: наприклад, як поводитися з прізвищами у складі звертань [S2, S5, S8], або що робити, коли подвійна форма кличного відмінка звучить фонетично неприродно через збіг приголосних [S3]. Такий підхід гарантує, що учні не просто завчають закінчення механічно, а глибоко розуміють стилістичну доречність та архітектуру кожної форми.

## Повна парадигма

Складені звертання в українській мові формуються за чіткими структурними моделями. Для англомовного учня, чия рідна мова не має відмінків, важливо розуміти не лише фінальне звучання фрази, а й покрокову механіку перетворення кожного синтаксичного елемента.

### Базові закінчення кличного відмінка (Нагадування)

Щоб утворити складене звертання, учень має вміти трансформувати окремі іменники:
* **І відміна (жіночий та чоловічий рід на -а/-я):**
  * Тверда група отримує закінчення **-о**: `колега → колег- + -о → колего`, `Ганна → Ганн- + -о → Ганно`.
  * М'яка група отримує закінчення **-є** (після голосного) або **-ю** (після м'якого приголосного): `Надія → Наді- + -є → Надіє` [S2], `Марія → Марі- + -є → Маріє` [S1], `тітонька` (як виняток-пестливе слово) → `тітоньк- + -о → тітонько` [S2].
* **ІІ відміна (чоловічий рід на приголосний):**
  * Тверда група зазвичай отримує закінчення **-е**: `Іван → Іван- + -е → Іване`, `президент → президент- + -е → президенте` [S1].
  * М'яка група отримує закінчення **-ю**: `лікар → лікар- + -ю → лікарю` [S4], `Віталій → Віталі- + -ю → Віталію` [S4].
  * *Чергування приголосних:* Задньоязикові **г, к, х** обов'язково змінюються на шиплячі **ж, ч, ш** перед закінченням -е: `друг → друж- + -е → друже` [S1, S4], `юнак → юнач- + -е → юначе`.

### Правила творення складених звертань як ланцюжки перетворень

Нижче наведено базові механізми перетворення структур від називного відмінка до цільової форми кличного.

**Модель 1: Прикметник / Займенник + Іменник**
*Обмеження:* Прикметники та займенники у складі поширеного звертання завжди мають форму називного відмінка [S1, S3]. Вони категорично не узгоджуються з іменником у кличному відмінку.
* `{прикметник(Н.в.)} + {іменник(Н.в.)} → {прикметник(Н.в.)} + {основа_іменника} + {закінчення_Кл.в.} → {кінцева форма}`
* `шановний + пан → шановний (Н.в.) + пан- + -е → шановний пане` [S1]
* `наш + дорогий + колега → наш дорогий (Н.в.) + колег- + -о → наш дорогий колего` [S1]
* `мій + зоря → моя (Н.в.) + зор- + -є → Зоре моя вечірняя` [S3] (у поетичному мовленні можлива зміна порядку слів, але відмінки непорушні).

**Модель 2: Загальна назва (титул, професія, спорідненість) + Ім'я**
*Обмеження:* У звертаннях, що складаються із загальної назви та імені, форму кличного відмінка обов'язково набуває як загальна назва, так і власне ім'я [S2, S8].
* `{загальна_назва(Н.в.)} + {ім'я(Н.в.)} → {загальна_назва(Кл.в.)} + {ім'я(Кл.в.)}`
* `друг + Остап → друж- (чергування г//ж) + -е + Остап- + -е → друже Остапе` [S1]
* `лікар + Ярослав → лікар- + -ю + Ярослав- + -е → лікарю Ярославе` [S4]
* `тітонька + Надія → тітоньк- + -о + Наді- + -є → тітонько Надіє` [S2]
* `колега + Ірина → колег- + -о + Ірин- + -о → колего Ірино` [S8]
* `панна + Тетяна → панн- + -о + Тетян- + -о → панно Тетяно` [S1]

**Модель 3: Загальна назва + Прізвище**
*Обмеження:* У звертаннях, що складаються із загальної назви та прізвища, форму кличного відмінка має ТІЛЬКИ загальна назва, а прізвище завжди виступає в називному відмінку [S2, S5, S8]. 
* `{загальна_назва(Н.в.)} + {прізвище(Н.в.)} → {загальна_назва(Кл.в.)} + {прізвище(Н.в.)}`
* `пан + Семенюк → пан- + -е + Семенюк (Н.в.) → пане Семенюк` [S2]
* `колега + Панченко → колег- + -о + Панченко (Н.в.) → колего Панченко` [S2]
* `добродійка + Білецька → добродійк- + -о + Білецька (Н.в.) → добродійко Білецька` [S2]
*(Примітка: Хоча одне з джерел допускає як вузький варіант відмінювання чоловічих прізвищ на приголосний, наприклад, "друже Максименку" [S4], основною, найсуворішою і найбільш безпечною стилістичною нормою, яку слід викладати іноземцям, є категоричне збереження прізвища в називному відмінку [S5, S8]).*

**Модель 4: Ім'я + Ім'я по батькові**
*Обмеження:* Обидва слова мають форму кличного відмінка без жодних винятків [S1, S2].
* `{ім'я(Н.в.)} + {по_батькові(Н.в.)} → {ім'я(Кл.в.)} + {по_батькові(Кл.в.)}`
* `Володимир + Олексійович → Володимир- + -е + Олексійович- + -у → Володимире Олексійовичу` [S1, S2]
* `Марія + Іллівна → Марі- + -є + Іллівн- + -о → Маріє Іллівно` [S1]
* `Ганна + Дмитрівна → Ганн- + -о + Дмитрівн- + -о → Ганно Дмитрівно` [S2]
* `Ярослав + Андрійович → Ярослав- + -е + Андрійович- + -у → Ярославе Андрійовичу` [S2]

**Модель 5: Дві загальні назви**
*Обмеження 1:* Рекомендовано обидва слова вживати у кличному відмінку [S1, S2].
* `{назва1(Н.в.)} + {назва2(Н.в.)} → {назва1(Кл.в.)} + {назва2(Кл.в.)}`
* `пан + президент → пан- + -е + президент- + -е → пане президенте` [S1]
* `пані + голова → пані (незмінне) + голов- + -о → пані голово` [S1]
* `добродій + директор → доброді- + -ю + директор- + -е → добродію директоре` [S2]
* `пан + лейтенант → пан- + -е + лейтенант- + -е → пане лейтенанте` [S4, S5]
*Обмеження 2 (Виняток щодо милозвучності):* Іноді подвійна форма кличного відмінка звучить неприродно через важкий збіг приголосних або втрату ритму (наприклад, "шановний пане по́сле" від слів "пан посол"). У такому разі, як фонетичний виняток, можна використати форму називного відмінка для другого слова: `шановний пане посол` [S3]. Це обмеження є запобіжником проти механічного спотворення мови.

## Частотність і пріоритети

Для англомовного учня, який опановує українську на рівні A2–B1, не всі п'ять моделей є однаково частотними в повсякденному чи професійному житті. Автор граматичного модуля має розподіляти педагогічний фокус таким чином:

1. **Найвищий пріоритет (Активний вжиток): `Пане / Пані + Ім'я` або `Пане / Пані + Прізвище`.** Це золотий стандарт сучасного ділового та ввічливого спілкування в Україні. Слово «пан» має глибоке історичне коріння, і його вживання є актом мовного самоствердження. Англомовним учням ця модель найбільш зрозуміла семантично (як культурний аналог Mr. / Ms.), але структурно вимагає нової звички: відмінювати обидва слова, якщо це ім'я, і відмінювати лише титул, якщо це прізвище. Це найбезпечніший і найввічливіший спосіб налагодити контакт із незнайомою людиною.
2. **Високий пріоритет (Активний вжиток): `Шановний / Шановна + загальна назва / ім'я`.** Це фундамент для написання електронних листів і формальних повідомлень. Критично важливо автоматизувати навичку залишати прикметник у називному відмінку, щоб листування іноземця виглядало грамотно.
3. **Середній пріоритет (Пасивне розпізнавання та ситуативний вжиток): `Ім'я + По батькові`.** В українській культурі це глибоко вкорінений спосіб вираження поваги (до вчителів, лікарів старшого віку, керівництва). Проте сучасні корпоративні середовища та молодь дедалі частіше переходять на європейську модель "Пан/Пані + Ім'я". Для іноземця здатність утворити "Олександре Петровичу" є дуже корисною вправою, але в реальному житті помилка тут є комунікативно менш критичною, ніж ігнорування кличного відмінка після слова "пан".
4. **Специфічний пріоритет: `Дві загальні назви` (пане офіцере, пане президенте).** Ця модель вкрай важлива для тих, хто працюватиме у військовому, поліцейському, дипломатичному або офіційному урядовому контекстах.

## Деколонізаційні застереження

Кличний відмінок є одним із найпотужніших маркерів української мовної ідентичності, який історично зазнавав системного утиску, знецінення та маргіналізації в епоху російської та радянської окупації. 

Російська мова втратила повноцінний кличний відмінок як граматичну категорію (залишилися лише релігійні архаїзми на зразок "Боже", "Господи" та просторічні скорочення типу "Маш", "Вань"). Для офіційного звертання в російській мові використовується виключно називний відмінок ("Господин президент!", "Иван Иванович!"). Упродовж радянського періоду ця бідніша структурна модель агресивно насаджувалася в український публічний простір. Саме тому українські шкільні підручники сьогодні категорично, чорним по білому наголошують: **форма називного відмінка у звертаннях не відповідає нормам сучасної української мови** [S2]. 

Коли іноземець каже "Пан президент!" замість "Пане президенте!", або "Брат!" замість "Брате!", це звучить не просто як дрібна граматична помилка іноземця, а як глибокий, ріжучий слух русизм. Автори курсу повинні прищеплювати англомовним учням розуміння, що кличний відмінок — це не "необов'язковий стилістичний орнамент", а непорушний фундамент поваги і єдиний спосіб природного звучання.

Додатково, під час побудови діалогів зі звертаннями, слід безжально викорінювати використання радянсько-імперських формул етикету та кальок:
* Заборонено використовувати російське слово "Здравствуйте" <!-- VERIFY: greeting_hello_formal — Russian word, not Ukrainian Canonical form: «Добрий день» --> чи його потворні суржикові аналоги ("Здрастуйте" <!-- VERIFY: greeting_hello_formal — Прямий русизм (рос. здравствуйте); VESUM класифікує як розмовне Canonical form: «Добрий день» -->). Єдиною канонічною формою українського формального вітання є винятково **Добрий день**.
* Канонічна формула прощання — **До побачення**. Слід уникати калькованого й нелітературного "Пока" <!-- VERIFY: farewell_goodbye — Рос. запозичення; нейтральний варіант — «До побачення» Canonical form: «До побачення» -->.
* Неприпустимо використовувати імперські ярлики для назви мови; вона має чітко й безальтернативно називатися **українська мова**.
* Універсальними звертаннями є слова "пан", "пані", "добродій", "добродійка" [S1, S2], які підкреслюють європейську традицію української мови і повністю замінюють російські "господин" чи знеособлені, холодні звертання за професією ("Водій, зупиніть!").
* Англійська дієслівна зв'язка "is" (My name is) не має перекладатися калькою. Називаючи себе під час знайомства, слід казати **Мене звати [ім'я]**, а не покруч "Моє ім'я є". Аналогічно, вказівка віку передається давальним відмінком: **Мені [N] років**, а не калькою "Я маю [N] років".
* У родинному контексті слід завжди використовувати питоме українське **тато** (у кличному відмінку — **тату**), категорично уникаючи російського запозичення "папа", яке Словник української мови розглядає виключно у значенні титулу глави католицької церкви.

## Природні приклади

Приклади нижче згруповані за структурними моделями і є автентичними зразками безпосередньо з текстів та вправ українських шкільних підручників. Вони демонструють, як саме ці конструкції живуть у реальному синтаксисі.

**Модель: Прикметник / Займенник + Іменник**
Ця група прикладів блискуче ілюструє незмінність прикметника в називному відмінку при іменнику в кличному.
1. Шановні пані та панове! [S1]
2. Зоре моя вечірняя, зійди над горою... [S3]
3. Наш дорогий колего, вітаємо вас! (Реконструйовано на базі словосполучення з [S1]).

**Модель: Загальна назва (Титул / Професія) + Ім'я**
Обидва слова обов'язково приймають форму кличного відмінка, демонструючи максимальну повагу до статусу та особистості співрозмовника.
4. Гарного дня, лікарю Ярославе! [S4]
5. Моє шанування, тітонько Марусю! [S4]
6. Дякую, друже Віталію. [S4]
7. На все добре, подруго Яринко. [S4]

**Модель: Загальна назва + Прізвище**
Найважливіша модель для формального ділового листування: загальна назва змінюється, прізвище залишається в називному відмінку, як незмінний ідентифікатор роду.
8. Пане Семенюк, просимо звернути увагу на цей документ. (Побудовано на базі нормативного переліку [S2]).
9. Колего Панченко, чи готові ви до виступу? (Побудовано на базі нормативного переліку [S2]).
10. До зустрічі, добродійко Білецька. (Побудовано на базі нормативного переліку [S2]).

**Модель: Ім'я + По батькові**
Традиційна українська форма глибоко шанобливого звертання, де обидва компоненти стоять у кличному відмінку.
11. Бережіть себе, Сергію Степановичу! [S4]
12. На добраніч, Надіє Петрівно! [S4]
13. Шановний Олександре Петровичу, вітаємо з днем народження! [S4]
14. Шановний Олегу Петровичу, просимо надіслати фінансовий звіт. [S5]

**Модель: Дві загальні назви**
Використовується здебільшого у військовому, дипломатичному або вузькопрофесійному контекстах.
15. Доброго ранку, пане офіцере! [S4]
16. Усього найкращого, пані редакторко! [S4]
17. Шановний пане посол! [S3] (Приклад свідомого фонетичного винятку, коли друге слово залишається в називному відмінку задля збереження милозвучності).

## Рекомендації для вправ

Для ефективного і глибокого засвоєння цієї теми англомовними учнями, автор модуля має побудувати тренувальні активності (activities) за принципом поступового зняття когнітивного навантаження:

**Фаза 1: Рецептивне розпізнавання та деконструкція**
* **Мета:** Навчити учня бачити різницю між звертанням до імені та звертанням до прізвища, а також міцно зафіксувати увагу на незмінності прикметника.
* **Формат:** Учням пропонуються готові, абсолютно правильні українські речення. Їхнє завдання — виділити (підсвітити) слова, що стоять у кличному відмінку, та ігнорувати слова, що залишилися в називному. Наприклад, у реченні "Шановний пане Семенюк, добрий день" учень має маркувати лише слово "пане".
* **Обґрунтування:** Це руйнує англійський патерн цілісного, недиференційованого сприйняття фрази і змушує мозок звертати увагу на морфологію кожного слова окремо.

**Фаза 2: Механічна трансформація (Керована підстановка)**
* **Мета:** Тренування автоматичного утворення форм за чіткими ланцюжками, які були наведені в розділі парадигм.
* **Формат:** Вправа типу "заповни пропуски" (fill-in-the-blanks). Учень отримує базові словникові форми в дужках і має самостійно утворити правильне складене звертання: `(шановний) (пан) (офіцер)` → `Шановний пане офіцере`. Або `(пан) (Іваничук)` → `пане Іваничук` [S5]. 
* **Обґрунтування:** На цьому етапі слід обов'язково давати контрастні пари поруч (наприклад, завдання на власне ім'я одразу після завдання на прізвище), щоб учень свідомо застосовував обмеження про невідмінюваність прізвищ, а не діяв за інерцією.

**Фаза 3: Комунікативна продукція (Ситуаційне моделювання)**
* **Мета:** Безшовна інтеграція нової граматичної навички в реальне, живе мовлення.
* **Формат:** Створення коротких діалогів або написання початкових фраз для електронних листів. Як прямо рекомендують українські шкільні підручники: "Складіть діалоги на вибір: зі звертанням одне до одного на пан/пані + ім'я; зі звертанням на прикметник + іменник" [S7].
* **Завдання для учня:** "Ви пишете офіційний електронний лист директору школи на ім'я Олег Петрович. Почніть лист правильно" → `Шановний Олегу Петровичу, ...`. Або: "Привітайтеся зі своїм колегою на прізвище Панченко" → `Добрий день, колего Панченко!`.

## Зв'язки з іншими темами

**Передумови (що учень вже має впевнено знати):**
* **Базовий кличний відмінок однини (A1):** Глибоке розуміння того, що слова змінюють закінчення при прямому звертанні (мама → мамо, Іван → Іване, Київ → Києве). Без цієї бази конструювати складені форми неможливо.
* **Називний відмінок прикметників (A1):** Необхідно для успішного опанування моделі "Шановний / дорогий + іменник".
* **Категорія граматичного роду (A1):** Усвідомлення того, що вибір закінчення кличного відмінка (як і чітка вказівка на незмінність слова "пані") цілковито залежить від граматичного роду іменника.

**Наступні кроки (що ця тема уможливлює в майбутньому):**
* **Офіційно-діловий стиль (B1-B2):** Майстерне, автоматичне володіння складеними звертаннями є абсолютно критичним і невідворотним для написання ділових листів, офіційних запитів до державних установ чи подання резюме. Без цього неможливо опанувати формальний регістр української мови на професійному рівні.
* **Читання класичної літератури (B1+):** У класичній українській літературі та поезії широко використовуються подвійні звертання, розгорнуті метафоричні звертання до неживих предметів і явищ природи ("Зоре моя вечірняя" [S3]), а також специфічні архаїчні форми, адекватне розуміння яких неможливе без стійкої граматичної бази, закладеної на рівні A2.
* **Лексичне багатство:** Ця тема широко відчиняє двері до правильного використання специфічної лексики поваги та близькості: добродій, добродійка, побратим, посестра, колега, панна.
