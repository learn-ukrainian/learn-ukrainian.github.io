# Граматика A2: Місцевий відмінок у нових контекстах

## Як це пояснюють у школі

У вітчизняній лінгводидактиці та шкільній практиці місцевий відмінок розглядається як унікальний граматичний феномен, що має суворі синтаксичні та морфологічні обмеження. Базове правило, яке закладається на початкових етапах вивчення мови, формулюється безапеляційно: «У називному відмінку іменники вживаються без прийменників, у місцевому — тільки з прийменниками, а в усіх інших відмінках можуть уживатися з прийменниками або без них» [S1]. Ця концептуальна аксіома є фундаментальною для розуміння природи місцевого відмінка: він ніколи не виступає самостійним членом речення без прийменникового керування (наприклад, неможливо сказати `*жити місті`).

У підручниках для молодших класів місцевий відмінок традиційно вводиться через прості просторові відношення (відповідь на питання «Де?»). Учні засвоюють конструкції з прийменниками «в/у» та «на», описуючи стабільне розташування об'єктів у просторі: «на сонних деревах», «на вузьких тротуарах», «на залізничній станції» [S1]. Водночас школа наголошує на нерозривному зв'язку іменника з його атрибутами. Правило вимагає повного узгодження: «Прикметник завжди стоїть у тому числі, що й іменник, з яким він зв’язаний. У множині всі прикметники мають закінчення -і» [S4]. Це означає, що при відмінюванні іменника («озеро» → «в озері») прикметник також синхронно трансформується, набуваючи відповідних локативних маркерів («чисте озеро» → «у чистому озері», «чисті озера» → «у чистих озерах»).

У міру переходу до середньої школи (5–7 класи) семантичне поле місцевого відмінка суттєво розширюється, охоплюючи нові абстрактні та темпоральні контексти. По-перше, місцевий відмінок починає обслуговувати абстрактні іменники, які «називають поняття, явища, властивості, які не можуть сприйматися органами чуття» [S3]. З'являються конструкції на кшталт «у темпі», «у центрі», «у декреті», «у прізвищі», «у слові» [S7]. 

По-друге, місцевий відмінок стає ключовим інструментом для вираження точного часу. Для відповіді на питання «О котрій годині?» школярів навчають використовувати специфічні прийменники «о» та «об» у поєднанні з порядковими числівниками саме у формі місцевого відмінка: «о сьомій», «об одинадцятій» [S5], [S6]. Особлива увага приділяється питомим українським конструкціям із прийменником «по», який у місцевому відмінку маркує часову межу після певної події: «по обіді», «по закінченні» [S7]. Шкільні підручники чітко диференціюють ці форми від російськомовних кальок, підкреслюючи їхню автентичність.

## Повна парадигма

Місцевий відмінок демонструє складну систему закінчень, вибір яких залежить від роду, відміни та семантики слова. Формотворення слід розглядати як чіткий ланцюжок морфологічних перетворень.

### Обмеження форми
1. **Прийменникова монополія:** Місцевий відмінок вживається ТІЛЬКИ з прийменниками («в/у», «на», «о/об», «по», «при») [S1].
2. **Семантика часу:** Для визначення точного часу без хвилин використовуються винятково прийменники «о» або «об» [S5].
3. **Обмеження прийменника «по»:** Для позначення часу з прийменником «по» використовується ТІЛЬКИ закінчення -і [S7].
4. **Узгодження:** Прикметник у місцевому відмінку завжди переймає число та рід керованого іменника [S4].

### Іменники: Чоловічий рід
Безсуфіксні іменники чоловічого роду часто мають паралельні закінчення `-і` (`-ї`) та `-у` (`-ю`), розподіл яких залежить від місця наголосу або семантичної традиції [S7]. Водночас перед закінченням `-і` відбувається закономірне чергування задньоязикових приголосних (`г//з`, `к//ц`, `х//с`).

| Категорія | Ланцюжок творення | Кінцева форма (з прийменником) |
| :--- | :--- | :--- |
| Базовий (тверда група) | `{називний} → {основа} → {+і}` | `стіл → стол- → +і → на столі` <!-- VERIFY --> |
| З чергуванням `г//з` | `{називний} → {основа} → {г//з} → {+і}` | `торг → торг- → г//з → торз- → +і → на торзі` [S7] |
| З закінченням `-у/-ю` | `{називний} → {основа} → {+у/+ю}` | `гай → га- → +ю → у гаю` [S7] |
| **Паралельні форми** | `{називний} → {основа} → {+і/+у}` | `край → кра- → +ї / +ю → у краї / у краю` [S7] |

### Іменники: Жіночий рід
Жіночий рід на `-а/-я` (І відміна) характеризується стабільним закінченням `-і` (або `-ї` після голосного). Чергування приголосних основи тут є обов'язковим.

| Категорія | Ланцюжок творення | Кінцева форма (з прийменником) |
| :--- | :--- | :--- |
| Тверда група | `{називний} → {основа} → {+і}` | `вулиця → вулиц- → +і → на вулиці` [S1] |
| Після голосного | `{називний} → {основа} → {+ї}` | `станція → станці- → +ї → на станції` [S1] |
| Жіночий рід (ІІІ відміна) | `{називний} → {основа} → {+і}` | `адреса → адрес- → +і → за адресою` [S1] <!-- VERIFY: "за адресою" є орудним відмінком, але для локації вживається "в адресі" --> `осінь → осен- → +і → в осені` <!-- VERIFY --> |

*Примітка щодо ІІІ відміни:* Слова жіночого роду, що закінчуються на нульове закінчення, у місцевому відмінку набувають закінчення `-і` (`ніч` → `у ночі`). Хоча джерела прямо не наводять парадигму ІІІ відміни для місцевого, вона виводиться з загальних правил української мови <!-- VERIFY -->.

### Іменники: Середній рід
Іменники середнього роду без суфікса `-к-` переважно отримують закінчення `-і` [S7].

| Категорія | Ланцюжок творення | Кінцева форма (з прийменником) |
| :--- | :--- | :--- |
| Без суфікса `-к-` | `{називний} → {основа} → {+і}` | `місто → міст- → +і → у місті` [S7] |
| Базові слова | `{називний} → {основа} → {+і}` | `село → сел- → +і → на селі` [S7] |
| Орудний/Локативний | `{називний} → {основа} → {+і}` | `слово → слов- → +і → у слові` [S7] |

### Множина (усі роди)
Множина місцевого відмінка є максимально уніфікованою: всі іменники отримують закінчення `-ах` (після твердих приголосних) або `-ях` (після м'яких і голосних).

| Категорія | Ланцюжок творення | Кінцева форма (з прийменником) |
| :--- | :--- | :--- |
| Після твердих приголосних | `{називний} → {основа} → {+ах}` | `дерева → дерев- → +ах → на деревах` [S1] |
| Після м'яких/шиплячих | `{називний} → {основа} → {+ях}` | `гроші → грош- → +ях → у грошах` <!-- VERIFY --> |

## Частотність і пріоритети

Для англомовних студентів рівня А2 засвоєння парадигми місцевого відмінка повинно відбуватися за чітко визначеними пріоритетами, що спираються на комунікативну частотність:

1. **Пріоритет 1: Базова локація простору (Де?).** Конструкції з прийменниками `в/у` та `на` («на залізничній станції», «у місті», «на вулиці» [S1], [S7]). Це найчастотніший патерн, який дозволяє учневі орієнтуватися у фізичному середовищі.
2. **Пріоритет 2: Маркування точного часу (О котрій?).** Використання прийменників `о/об` + порядковий числівник («о сьомій», «об одинадцятій» [S5]). Ці фрази є критичними для домовленостей та щоденної рутини.
3. **Пріоритет 3: Абстрактні стани та простори.** Вживання місцевого відмінка для невідчутних об'єктів («у темпі», «у центрі», «у слові» [S7]). Відкриває двері до складніших дискусій та аналітичного мислення.
4. **Пріоритет 4: Розподільні та часові конструкції з «по».** Форми типу «по місті», «по закінченні» [S7] є маркерами високої культури мовлення і вводяться на етапі шліфування граматики.

## Типові помилки L2 (англомовні учні)

Англомовні студенти стикаються з низкою структурних труднощів під час опанування місцевого відмінка. Перенесення патернів з англійської мови призводить до системних помилок.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я мешкаю Київ.` | `Я мешкаю в Києві.` | Англомовні учні часто застосовують модель прямого додатка або ігнорують локативні прийменники з топонімами (на зразок англійського "I navigate the city"). Однак українська граматика категорична: у місцевому відмінку іменники вживаються **тільки з прийменниками** [S1]. |
| `Зустрінемося в сьомій.` | `Зустрінемося о сьомій.` | Це пряма калька з англійського прийменника "at" (at seven) або бездумне використання універсального локативного "в". Українська норма для визначення точного часу вимагає специфічних прийменників `о/об` перед місцевим відмінком [S5]. |

Учень часто залишає прикметник у початковій формі множини, змінюючи лише іменник. Правило ж вимагає: прикметник завжди стоїть у тому числі й відмінку, що й іменник [S4], [S1]. |
| `Ми гуляли по місту.` | `Ми гуляли по місті.` | Вплив російської мови (де вживається давальний відмінок напрямку) або гіперкорекція. Хоча українська норма допускає паралельні закінчення `-у/-ю` для слова «місто», більш питомою, класичною конструкцією просторового розподілу є «по» + місцевий відмінок на `-і` [S7]. |
| `Він зателефонує по обіду.` | `Він зателефонує по обіді.` | Учень помилково застосовує правило про паралельні закінчення чоловічого роду (`-у` та `-і`) до часових конструкцій. Граматична норма чітко обмежує цей вибір: для позначення часу з прийменником «по» використовується **тільки** закінчення `-і` [S7]. |

## Деколонізаційні застереження

Під час викладання місцевого відмінка надзвичайно важливо уникати російських синтаксичних та морфологічних кальок. Українська мова має власний, глибоко вкорінений інструментарій для вираження простору й часу. 

Найчастіша зона інтерференції — конструкції з прийменником «по». У російській мові цей прийменник жорстко керує давальним відмінком («гулять по городу», «по селу»). Українська мова, натомість, історично віддає перевагу місцевому відмінку: «по місті», «по селі», «по Дніпрі» [S7]. Хоча сучасні словники фіксують форми на `-у/-ю` як паралельні («по місту», «по Дніпру») під впливом тривалої русифікації, автори навчальних матеріалів повинні пріоритезувати питомі українські форми на `-і/-ї` як зразки автентичного звучання.

Другий критичний аспект стосується вираження часу. Російська модель використовує прийменник «в» (в семь часов). Українська мова має унікальні прийменники «о» та «об» для точного часу: «о сьомій», «об одинадцятій» [S5]. Використання прийменника «в/у» для вказівки на годину є грубим порушенням норми. 

Крім того, російське «после обеда» часто некритично перекладається як «після обіду». Хоча ця форма граматично правильна (родовий відмінок), українська мова володіє значно елегантнішою темпоральною конструкцією з місцевим відмінком: «по обіді», «по закінченні» [S7]. Інтеграція цих форм у лексикон студента робить його мовлення природним і деколонізованим.

## Рекомендації для вправ

Для ефективного закріплення парадигми місцевого відмінка автор-письменник має використовувати багатоетапний підхід, поступово підвищуючи когнітивне навантаження.

**Фаза 1: Рецептивна ідентифікація та механічне сортування.** 
Перші вправи повинні фокусуватися на розпізнаванні закінчень. Рекомендується створити завдання, де учні мають згрупувати подані словосполучення за типом закінчення (наприклад, слова з закінченням `-і` проти слів із паралельними `-у/-ю`). Це візуалізує для них патерни: «у гаї» / «у гаю», «на торзі» / «на торгу» [S7]. 

**Фаза 2: Керована трансформація простору та часу.** 
На цьому етапі слід вводити вправи на підстановку. Учень отримує називний відмінок і повинен утворити правильну локативну або часову конструкцію. Особливий акцент потрібно зробити на вправах типу «Заповни пропуск правильним прийменником», щоб відпрацювати розрізнення `о/об` для часу [S5] та `в/на` для місця [S1]. Обов'язково додавайте завдання на узгодження прикметника з іменником у множині («чисті озера» → «у чистих озерах») [S4].

**Фаза 3: Продуктивне використання у деколонізованих патернах.** 
Фінальний етап має вимагати від учня створення цілих речень за моделлю. Наприклад, трансформація англійських ситуацій (He called after lunch) в питомі українські (Він зателефонував по обіді [S7]). Важливо давати ситуативні завдання: «Опишіть свій розклад на день, використовуючи прийменники часу о/об та порядкові числівники».

## Зв'язки з іншими темами

Місцевий відмінок є критичним мостом до складніших граматичних структур. По-перше, він тісно пов'язаний із вивченням числівників. Уміння ставити порядкові числівники у місцевий відмінок є необхідною передумовою для вільного висловлення дат та часу [S5]. По-друге, ця тема підготовлює учнів до розуміння відмінностей між місцевим і знахідним відмінком при використанні тих самих прийменників (наприклад, «на станції» — статика, місцевий; «на станцію» — динаміка, знахідний). Зрештою, засвоєння форм абстрактних іменників [S3] прокладає шлях до академічного мовлення (рівні B2–C1), де концептуальні дискусії відбуваються переважно за допомогою абстрактної лексики.
