## Як це пояснюють у школі (How Schools Teach This)

Місцевий відмінок (Locative case) вводиться в українських школах на ранніх етапах вивчення граматики, зазвичай у початкових класах, як один із семи відмінків української мови [S41]. Початкове пояснення фокусується на його основній функції: позначення місця дії. Він відповідає на питання `на кому?`, `на чому?`, `в кому?`, `в чому?`, `де?` і завжди використовується з прийменником [S42].

Основні моменти, що висвітлюються у шкільній програмі:
1.  **Завжди з прийменником:** На відміну від інших непрямих відмінків, місцевий ніколи не вживається без прийменників `в/у`, `на`, `по`, `при`.
2.  **Чергування приголосних:** У 6 класі звертається увага на важливі чергування приголосних в іменниках І відміни однини: `г → з`, `к → ц`, `х → с`. Приклади: `нога → на нозі`, `рука → в руці`, `муха → на мусі` [S42].
3.  **Розширення контекстів:** На рівні A2 (що відповідає середній школі) учні переходять від простого позначення місця (`в школі`, `на столі`) до більш абстрактних і складних контекстів:
    *   **Час:** Позначення місяців (`у березні`), років (`у 2020 році`), століть (`у XX столітті`).
    *   **Регулярні події:** Використання прийменника `по` з множиною для позначення повторюваної дії, наприклад, `по суботах` (щосуботи) [S43].
    *   **Абстрактні поняття:** Використання з абстрактними іменниками: `в житті`, `в душі`, `у роботі`.

Підхід є поступовим: від конкретного (фізичне місцезнаходження) до абстрактного (час, стан, сфера діяльності).

## Повна парадигма (Full Paradigm)

Місцевий відмінок є одним із найбільш різноманітних за формами закінчень, особливо в однині.

### Іменники І відміни (Feminine, some Masculine)

| Група | Однина (-і, -ї) | Множина (-ах, -ях) | Приклади | Чергування |
| :--- | :--- | :--- | :--- | :--- |
| **Тверда** | `-і` | `-ах` | `робота → в роботі`, `сестра → на сестрі` | `г→з`: `книга → в книзі`, `нога → на нозі` |
| | | | | `к→ц`: `рука → в руці`, `донька → на доньці` [S44] |
| | | | | `х→с`: `муха → на мусі`, `свекруха → при свекрусі` [S42] |
| **М'яка** | `-і` | `-ях` | `земля → на землі`, `пісня → в пісні` | Немає |
| **Мішана** | `-і` | `-ах` | `душа → в душі`, `межа → на межі` | Немає |

### Іменники ІІ відміни (Masculine, Neuter)

Це найскладніша група з двома варіантами закінчень в однині: `-і/-ї` та `-у/-ю`.

**Однина (Singular):**

| Рід | Закінчення | Правило (загальне) | Приклади |
| :--- | :--- | :--- | :--- |
| **Чоловічий** | `-і`, `-ові`, `-еві` | Назви істот, деякі конкретні предмети. | `на коні`, `при батькові`, `вчителеві` |
| | **-у**, **-ю** | Більшість назв неістот: місця, простори, явища, абстрактні поняття. | `в будинку`, `в лісі`, `на столі`, `у Києві` [S45], `у Парижі` [S46], `в центрі` [S47], `на районі` [S43] |
| **Середній** | `-і` | Іменники на `-е`, `-я` (крім `-еня`, `-ятя`). | `в морі`, `в житті` [S48], `на полі`, `в серці` |
| | **-у** | Іменники на `-о`. | `в місті` [S47], `у небі` |

**Множина (Plural):**

| Рід | Закінчення | Приклади |
| :--- | :--- | :--- |
| **Чоловічий** | `-ах`, `-ях` | `на столах`, `у лісах`, `в університетах` [S48] |
| **Середній** | `-ах`, `-ях` | `в містах`, `по полях`, `у вікнах` [S49] |

### Іменники ІІІ відміни (Feminine ending in a consonant)

| Однина | Множина | Приклади |
| :--- | :--- | :--- |
| `-і` | `-ах` | `в любові`, `по ночі` (форма `вночі` стала прислівником), `на Січі` [S50], `у подорожі` |

### Іменники IV відміни (Neuter in -а, -я from young animals)

| Однина | Множина | Приклади |
| :--- | :--- | :--- |
| `-ені` | `-атах` | `на теляті`, `в імені` | `на телятах` |

## Частотність і пріоритети (Frequency & Priorities)

Для рівня A2 пріоритети вивчення та використання такі:

1.  **Локація (Де?):** Найбільш частотне використання.
    *   **Прийменник `в/у` (inside):** `в Україні` [S44], `в Києві` [S45], `в квартирі` [S47], `в університеті` [S48].
    *   **Прийменник `на` (on/at):** `на столі` [S51], `на районі` [S43], `на ярмарку` [S43], `на кордоні` [S44].
2.  **Час (Коли?):** Друге за важливістю для розширення виразності.
    *   **Місяці:** `у березні` [S45], `у січні` [S46].
    *   **Роки:** `в 2020 році` [S45], `в 1919 році` [S46].
    *   **Повторювані дні:** `по суботах` [S43].
3.  **Абстрактні поняття:** Вживання з іменниками, що позначають стан, сферу, контекст.
    *   `в житті` [S45], `в роботі` [S45], `в дитинстві` [S47], `в біді` [S52].
4.  **Прийменник `при`:** Менш частотний, але важливий для розуміння.
    *   `при місячному освітленні` [S49].
    *   `при цьому учневі` [S53].

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| *Я живу **на** Києві.* | Я живу **в** Києві. | Плутанина прийменників `в` (для міст, країн, приміщень) і `на` (для районів, островів, подій, відкритих просторів). Англійське "in" та "at" не завжди прямо перекладаються. |
| *Вона була **в рукі**.* | Вона була **в руці**. | Забуте обов'язкове чергування приголосних `к → ц` в І відміні. Це правило треба завчити [S42]. |
| *Побачимось **в суботу**.* (для регулярної дії) | Побачимось **у суботу** / Ми бачимось **по суботах**. | `В + Знахідний відмінок` (`в суботу`) позначає одну конкретну суботу. Для регулярних подій використовується `по + Місцевий відмінок множини` (`по суботах`) [S43]. |
| *Я був **в будинку**.* | Я був **у домі** / **у будинку**. | Неправильний вибір закінчення `-у`/`-і` в ІІ відміні. Хоча обидва варіанти можуть бути правильними, їх розподіл складний. `В лісі`, `в саду`, `в домі` (більш розмовне), але `на поверсі`. Це одна з найскладніших тем для іноземців. |
| *Я думаю **о** тобі.* | Я думаю **про** тебе. | В українській мові прийменник `о` з місцевим відмінком для позначення теми думки є застарілим або діалектним. Сучасна норма — `про + Знахідний відмінок`. |
| *Він **домі**.* | Він **вдома**. | Спроба використати місцевий відмінок без прийменника, за аналогією з англійською. В українській мові прислівник `вдома` є окремим словом, а місцевий відмінок завжди вимагає прийменника. |

## Деколонізаційні застереження (Decolonization Notes)

1.  **`В Україні` vs. `На Україні`:** Це найважливіше правило. Єдиною граматично і політично коректною формою є **`в Україні`**. Прийменник `в` вживається з назвами незалежних держав. Форма `на Україні`, хоч і зустрічається в історичних текстах та піснях, сьогодні асоціюється з російським колоніальним наративом, що розглядав Україну як регіон ("окраину") Російської імперії, а не суверенну державу. Усі сучасні джерела, включно з наданими, послідовно вживають `в Україні` [S45][S43][S44][S48][S54].
2.  **Незалежність граматичних правил:** Не можна пояснювати українські граматичні явища як "винятки з російських правил". Наприклад, розподіл закінчень `-у/-ю` та `-і` в місцевому відмінку чоловічого роду має власну логіку та історію в українській мові. Хоча певні закономірності можуть збігатися, українську граматику слід вивчати як самостійну систему.
3.  **Прийменник `по`:** Конструкція `по + Місцевий відмінок` для позначення завершення дії (`по закінченні`, `по прибутті`) є характерною рисою української офіційно-ділової мови і відрізняється від аналогічних конструкцій у російській (`по окончании`). Це важливий стилістичний маркер.
4.  **Сім відмінків:** Наголошуйте, що українська мова має 7 відмінків, включаючи повноцінний і широко вживаний Кличний відмінок [S42][S41]. Ігнорування Кличного відмінка є поширеною помилкою, часто через вплив російської, де він є рудиментарним.

## Природні приклади (Natural Examples)

### Група 1: Позначення місця (Location)

- `Іванка живе **в Києві** і працює **в приватній школі** англійської мови.` [S45] — *Класичний приклад для міста та установи.*
- `Тоді багато країн закрили свої кордони і оголосили карантин **в Україні**.` [S45] — *Використання з назвою країни.*
- `Наступною сходинкою було навчання **в колегіумах**.` [S55] — *Використання у множині для типу установ.*
- `Чи знайшла Христина **на барахолці** фотоапарат?` [S43] — *Використання з `на` для події/місця.*

### Група 2: Позначення часу (Time)

- `...**у березні 2020 року в Європі** та світі став швидко поширюватись коронавірус.` [S45] — *Використання для місяця та року.*
- `...**у нас на районі** ярмарки **по суботах**.` [S43] — *Повторювана дія у часі.*
- `Куїнджі народився 27 січня **1842-го** у передмісті Маріуполя.` [S49] — *Рік як частина дати.*
- `А **в неділю** ми з Лесею разом приготували борщ і голубці.` [S43] — *Конкретний день тижня (Знахідний відмінок, але часто плутають з Місцевим; важливий для порівняння).*

### Група 3: Абстрактні поняття та стани (Abstract concepts)

- `**При цьому в біді** лишався самотнім — ніхто навіть не поцікавився його життям.` [S52] — *Стан, обставини.*
- `**У його творчості** важливу роль відіграли мотиви сенсу життя...` [S56] — *Сфера діяльності.*
- `Моя сім'я завжди жила **в квартирі**, яка розташована **на третьому поверсі п'ятиповерхового будинку** **у центрі міста**.` [S47] — *Комбінація конкретних локацій.*
- `**У давальному й місцевому відмінках** однини перед -і відбувається чергування.` [S42] — *Металінгвістичне використання.*

## Рекомендації для вправ (Activity Concepts)

-   **Phase 1 (Розпізнавання та базове відтворення):**
    -   **Вправа "Де це?":** Показати картинки (кімната, вулиця, місто, стіл, книжка) і попросити скласти прості речення `Предмет + в/на + іменник в М.в.`.
    -   **Сортування:** Дати список іменників (`Україна, стіл, район, школа, понеділок, рік`) і попросити розподілити, з яким прийменником (`в` чи `на`) вони зазвичай вживаються.
    -   **Заповнення пропусків (Прийменники):** `Я живу ___ Україні. Книжка лежить ___ столі.`

-   **Phase 2 (Контрольована практика):**
    -   **Трансформація:** Дати іменники в називному відмінку і попросити поставити їх у місцевий. Особливу увагу приділити чергуванням. `рука → в руці`, `Київ → в Києві`, `нога → на нозі`.
    -   **Складання речень за розкладом:** Дати розклад на тиждень (`понеділок - робота, вівторок - спортзал, ... субота - ярмарок`) і попросити написати, що людина робить і коли (`По понеділках я на роботі...`).
    -   **"Коли це було?":** Дати список історичних подій з роками і попросити скласти повні речення: `Козацька революція почалася ... (1648 рік).`

-   **Phase 3 (Вільна практика):**
    -   **Запитання-відповіді:** `Де ти живеш?`, `Де ти працюєш/вчишся?`, `В якому місяці у тебе день народження?`, `Що ти зазвичай робиш по вихідних?`
    -   **Опис:** "Опиши свою кімнату. Що де знаходиться?"
    -   **Міні-розповідь:** "Розкажи про своє типове дитинство. Де ти грався? Коли в тебе були канікули?" (за аналогією з [S47]).
