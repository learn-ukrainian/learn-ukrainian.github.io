## Як це пояснюють у школі

Українська шкільна програма вводить поняття дієслова як самостійної частини мови, що означає дію або стан предмета і відповідає на питання *що робити?* або *що зробити?* [S10, S3]. Це основа, на якій будується подальше вивчення.

Ключовим для рівнів А2-В1 є засвоєння двох фундаментальних категорій: **виду** і **часу**.

1.  **Вид дієслова (Доконаний / Недоконаний)**: Ця категорія вводиться як центральна, що відрізняє українську граматику від англійської. Дієслова **недоконаного виду** відповідають на питання *що робити?* і позначають дію тривалу, повторювану або незавершену [S1]. На противагу їм, дієслова **доконаного виду** відповідають на питання *що зробити?* і виражають дію завершену, результативну, одноразову. Важливо, що дієслова доконаного виду не мають форми теперішнього часу [S1]. Підручники відразу знайомлять учнів із тим, що види утворюють співвідносні пари (`писати — написати`, `робити — зробити`) і надають основні способи творення доконаного виду: за допомогою префіксів, суфіксів, чергування звуків, зміни наголосу та зміни основи слова [S1]. Також розглядаються поняття одновидових та двовидових дієслів.

2.  **Часи дієслова (Теперішній, минулий, майбутній)**: Час вивчається в межах **дійсного способу** — єдиного способу, де ця категорія існує [S5].
    *   **Теперішній час** існує тільки для дієслів недоконаного виду. Його форми утворюються зміною дієслів за особами та числами. Це явище називається **дієвідмінюванням** [S7].
    *   **Минулий час** вказує на дію, що вже відбулася. Його форми утворюються від основи інфінітива і змінюються за родами (в однині) та числами, але не за особами (`читав`, `читала`, `читали`).
    *   **Майбутній час** має три форми, вибір яких залежить від виду дієслова:
        *   **Проста форма (доконаний вид)**: `напишу`, `зроблю`. Формально збігається з формами теперішнього часу, але стосується майбутньої дії.
        *   **Складена форма (недоконаний вид)**: `буду писати`, `будеш робити`. Утворюється поєднанням особових форм дієслова *бути* та інфінітива.
        *   **Аналітична форма (недоконаний вид)**: `писатиму`, `робитимеш`. Вона менш поширена у розмовному мовленні, але є повноцінною літературною нормою.

3.  **Дієвідмінювання (I та II)**: Це одна з найскладніших тем для початківців. Підручники пропонують два основні методи визначення дієвідміни, від якої залежать особові закінчення (`-е-`/`-є-` для I та `-и-`/`-ї-` для II) [S8, S4]:
    *   **За закінченням 3-ї особи множини**: I дієвідміна має закінчення `-уть, -ють` (`пиш**уть**`); II дієвідміна — `-ать, -ять` (`говор**ять**`) [S8].
    *   **За основою інфінітива**: Якщо перший спосіб викликає сумніви, аналізують суфікс перед *-ти* в інфінітиві. До II дієвідміни належать дієслова з суфіксами `-и-`, `-і-`, `-ї-` або `-а-` (після шиплячого), які випадають в 1-й особі однини (`сид**і**ти` — `сидж**у**`). Решта належить до I дієвідміни [S4, S8]. Окремо розглядаються дієслова-винятки (`хотіти`, `бігти`, `спати` тощо).

Крім особових форм, шкільна програма вводить і незмінні форми дієслова, зокрема **дієприслівник**, що означає додаткову дію (`читаючи`, `прочитавши`) і в реченні виступає обставиною [S9].

## Повна парадигма

### Творення видових пар

Видові пари є основою для розуміння часу та дії в українській мові. Доконаний вид (ДВ) найчастіше утворюється від недоконаного (НДВ) [S1].

| Спосіб творення | Недоконаний вид (НДВ) (*що робити?*) | Перетворення | Доконаний вид (ДВ) (*що зробити?*) |
| :--- | :--- | :--- | :--- |
| **Префіксація** | `читати` | `читати → + про-` | `прочитати` |
| | `вітати` | `вітати → + при-` | `привітати` |
| | `летіти` | `летіти → + при-` | `прилетіти` |
| **Суфіксація** | `стукати` | `стукати → -а- + -ну-` | `стукнути` |
| | `викидати` | `викидати → -а- + -ну-` | `викинути` |
| | `відчиняти` | `відчиняти → -я- + -и-` | `відчинити` |
| **Чергування звуків** | `видирати` | `видирати → /и/ ↔ /е/` | `видерти` |
| | `збирати` | `збирати → /и/ ↔ /Ø/` | `зібрати` |
| | `допомагати` | `допомагати → /а/ ↔ /о/` | `допомогти` |
| **Зміна наголосу** | `склика́ти` | `склика́ти → наголос на корінь` | `скли́кати` |
| | `розріза́ти` | `розріза́ти → наголос на корінь` | `розрі́зати` |
| **Суплетивізм** (зміна основи) | `брати` | `брати → інша основа` | `**взяти**` |
| | `ловити` | `ловити → інша основа` | `**піймати**` |
| | `шукати` | `шукати → інша основа` | `**знайти**` |
| | `говорити` | `говорити → інша основа` | `**сказати**` |

**Обмеження:**
*   Не всі дієслова утворюють видові пари; існують **одновидові** дієслова [S1]:
    *   Тільки НДВ: `гордувати`, `чекати`, `мріяти`, `гомоніти`, `стежити`.
    *   Тільки ДВ: `розпитатися`, `натерпітися`, `провчитися`, `наврочити`.
*   Існують **двовидові** дієслова, де форма однакова для обох видів: `атакувати`, `гарантувати`, `женити`, `ранити` [S1]. Вид визначається з контексту.

*   Теперішній час утворюється **лише від дієслів недоконаного виду** [S1].

*   Складена та аналітична форми майбутнього часу утворюються **лише від дієслів недоконаного виду**.

**Обмеження:**
*   Дієприслівники теперішнього часу утворюються **тільки від дієслів недоконаного виду** [S9].
*   Дієприслівники минулого часу утворюються переважно від дієслів **доконаного виду**, хоча утворення від НДВ також можливе, але рідкісне (`писавши`) [S9].
*   Дія, виражена дієприслівником, має стосуватися того ж суб'єкта (підмета), що й основна дія в реченні [S9].

## Частотність і пріоритети

Для рівнів А2-В1 пріоритети вивчення слід розподілити так:

1.  **Найвищий пріоритет (must-know):**
    *   **Минулий час (НДВ/ДВ):** Це найпростіша для утворення і найчастотніша форма для розповіді про минулі події. Узгодження за родом (`він сказав`, `вона сказала`) є ключовою навичкою.
    *   **Теперішній час (НДВ):** Основа для опису теперішніх дій, станів, звичок. Пріоритет — засвоєння особових закінчень І та ІІ дієвідмін для ~50-100 найуживаніших дієслів.
    *   **Проста форма майбутнього (ДВ):** Найпоширеніший спосіб говорити про конкретні, завершені дії в майбутньому (`я зроблю`, `ми підемо`).

2.  **Середній пріоритет (should-know):**
    *   **Складена форма майбутнього (НДВ):** `буду робити`. Ця форма є дуже поширеною для опису тривалих або повторюваних дій у майбутньому і легша для засвоєння, ніж аналітична.
    *   **Умовний спосіб:** `зробив би`, `хотіла б`. Критично важливий для вираження бажань, порад, гіпотетичних ситуацій, що є частиною комунікативних потреб рівня А2+.
    *   **Видові пари:** Активне розпізнавання та утворення найпоширеніших пар (префіксація, суплетивізм). Це фундаментальна концепція, без якої неможливе вільне володіння мовою.

3.  **Нижчий пріоритет (good-to-know):**
    *   **Аналітична форма майбутнього (НДВ):** `робитиму`. Учні мають її розпізнавати, оскільки вона трапляється в літературі, але активне вживання можна відкласти на рівень В1+.
    *   **Дієприслівники:** `роблячи`, `зробивши`. На рівні А2 учень має розпізнавати найпростіші з них. Активне використання для побудови складних речень є задачею рівня В1.
    *   **Двовидові та одновидові дієслова:** Розпізнавання таких дієслів є важливим, але активне запам'ятовування списків не є пріоритетом на початковому етапі.

Учні переносять цю модель і використовують недоконаний вид, який в українській мові означає процес, а не результат. Для одноразової завершеної дії потрібен доконаний вид [S1]. |

Учні забувають, що в українській мові форма минулого часу узгоджується з родом підмета в однині (`-в` для ч.р., `-ла` для ж.р.) [S5]. |

## Деколонізаційні застереження

При вивченні української граматики важливо уникати хибних аналогій з російською, оскільки це може призвести до системних помилок.

1.  **Дієслово-зв'язка `є`:** В українській мові, на відміну від російської, збереглося і активно вживається дієслово-зв'язка `є` в теперішньому часі (`Я є студент`, `Це є проблема`). У російській воно опускається (`Я студент`, `Это проблема`). Хоча в українській мові опущення `є` теж можливе, його наявність є абсолютною нормою. Не варто уникати цієї форми.

2.  **Майбутній час:** В українській мові існують дві рівноправні форми майбутнього часу недоконаного виду: складена (`буду читати`) і аналітична (`читатиму`). У російській мові існує лише одна форма (`буду читать`). Потрібно наголошувати, що форма `читатиму` — це не архаїзм, а повноцінна частина літературної мови.

3.  **Форми дієприслівників:** Українські дієприслівники недоконаного виду творяться за допомогою суфіксів `-учи`/`-ючи` (І дієвідміна) та `-ачи`/`-ячи` (ІІ дієвідміна): `несучи`, `читаючи`, `стоячи`. Російські відповідники творяться за допомогою суфіксів `-а`/`-я`: `неся`, `читая`, `стоя`. Пряме перенесення російських форм (`читая`) в українську мову є грубою помилкою (суржиком).

4.  **Чергування приголосних:** В особових формах дієслів в українській мові відбуваються глибокі історичні чергування, які можуть відрізнятися від російських. Наприклад, `водити` — `воджу` (чергування `д`→`дж`), тоді як у російській `водить` — `вожу` (`д`→`ж`). `Просити` — `прошу` (`с`→`ш`), як і в російській, але `їздити` — `їжджу` (`зд`→`ждж`) проти російського `ездить` — `езжу` (`зд`→`зж`). Ці патерни слід вивчати як самостійну систему [S7].

5.  **Лексичні відмінності:** Не слід припускати, що дієслова зі спільним коренем мають однакові граматичні властивості. Наприклад, українське дієслово `чекати` (НДВ) утворює пару з `почекати` (ДВ). Російське `ждать` може мати пару `подождать`. Не можна механічно переносити префікси та моделі творення.

## Природні приклади

Приклади згруповано за граматичним явищем, яке вони ілюструють. Усі приклади взято з навчальних матеріалів.

**Група 1: Теперішній час (різні особи та дієвідміни)**
*   В дитинстві *відкриваєш* материк, котрий назветься потім — Батьківщина. [S4]
*   Україна *починається* з тебе. [S4]
*   Як *буде* робити, то *буде* й родити. [S7]
*   Що *посієш*, те й *пожнеш*. [S8]

**Група 2: Минулий час (узгодження за родом та числом)**
*   Командир *велів* готуватись до походу (ч.р.). [S1]
*   Коли *закінчилася* «Пікова дама» (ж.р.), актриса піднесла Чайковському лавровий вінок. [S3]
*   Я *була* малою горда, — щоб не плакать, я *сміялась* (ж.р.). [S10]

**Група 3: Майбутній час (різні форми)**
*   *Буду* я *навчатись* мови золотої (складена форма). [S1]
*   Ніхто нам не *збудує* держави, коли ми самі її не *збудуємо* (проста форма). [S4]
*   Я для тебе на світі *живу* (може інтерпретуватись як теперішній з відтінком майбутнього). [S6]

**Група 4: Умовний спосіб**
*   І риба *співала б*, якби голос мала. [S5]
*   *Пішов би* в гори на прогулянку, якби не великий сніг. [S5]

**Група 5: Дієприслівники**
*   Квіти, *стуливши* барвисті пелюстки, заснули. [S9]
*   Важко кидати землю, *не залишивши* на ній доброго сліду. [S9]
*   Люблю читати *лежачи*. [S9]

## Рекомендації для вправ

Для ефективного засвоєння теми рекомендується трифазна прогресія вправ.

**Фаза 1: Розпізнавання та ідентифікація**
1.  **Класифікація:** Дати текст і попросити учнів знайти всі дієслова, визначивши їх вид, час, особу/рід та число. (За зразком вправи 514 [S1] або 4 [S3]).
2.  **Визначення дієвідміни:** Надати список інфінітивів і попросити визначити дієвідміну, обґрунтовуючи вибір (за 3 ос. мн. або за основою інфінітива). (За зразком вправ 75, 76 [S4]).
3.  **Вибір правильної форми:** Вправи типу "multiple choice" або "заповни пропуск", де потрібно вибрати правильне закінчення: `ти пиш(е/и)ш`, `вони бор(ю/я)ться`. (За зразком вправи 77 [S4] та тестових завдань [S8]).

**Фаза 2: Контрольована продукція**
1.  **Трансформація часу:** Змінити речення з теперішнього часу на минулий і майбутній. *Я читаю книгу → Я читав книгу → Я буду читати книгу / Я читатиму книгу*.
2.  **Творення видових пар:** Дати список дієслів НДВ і попросити утворити від них ДВ за допомогою префіксів чи інших засобів. `робити → зробити`, `писати → написати`. (За зразком вправи 516 [S1]).
3.  **Створення особових форм:** Поставити дієслова у правильній особовій формі. *Ми (хотіти) поїхати → Ми хочемо поїхати*. (За зразком вправи 471 [S5]).
4.  **Творення дієприслівників:** Від поданих дієслів утворити дієприслівники доконаного та недоконаного виду. (За зразком вправи 556 [S9]).

**Фаза 3: Вільна продукція**
1.  **Побудова речень:** Скласти власні речення, що ілюструють різницю між видами. *Складіть по два речення з кожним двовидовим дієсловом так, щоб воно мало в одному доконаний, а в іншому — недоконаний вид.* (За зразком вправи 518 [S1]).
2.  **Короткий твір:** Написати короткий текст (5-7 речень) на задану тему ("Мій вчорашній день", "Мої плани на вихідні"), використовуючи дієслова у відповідних часах і видах. (За зразком вправи 6 [S4]).
3.  **Виправлення помилок:** Дати речення з типовими помилками L2 (видовими, родовими, дієприслівниковими) і попросити знайти та виправити їх. (За зразком вправи 561 [S9]).

## Зв'язки з іншими темами

*   **Передумови:**
    *   **Фонетика та алфавіт:** Для правильного написання закінчень і розуміння чергувань.
    *   **Іменник (відмінки):** Розуміння ролі підмета (хто/що виконує дію) в називному відмінку є базовим для узгодження з дієсловом.

*   **Подальші зв'язки:**
    *   **Відмінки:** Дієслова керують відмінками залежних від них іменників (перехідні дієслова вимагають знахідного відмінка: `читати (що?) книгу`).
    *   **Дієприкметник:** Це ще одна форма дієслова, яка поєднує ознаки дієслова та прикметника (`прочитана книга`) і є логічним продовженням теми дієслівних форм [S3].
    *   **Синтаксис:** Дієслово є центром речення (присудком). Вивчення способів, часів і видів дозволяє будувати складніші речення, зокрема з дієприслівниковими зворотами, що є темою синтаксису [S10, S9].
    *   **Дієслова руху:** Ця складна тема повністю базується на розумінні видів, префіксів та відмінкової системи.
