## Як це пояснюють у школі

Українські шкільні підручники вводять давальний відмінок (Д. в.) як один із семи відмінків української мови, що відповідає на питання *«кому? чому?»* [S3164]. Основна його семантична роль — позначення адресата або отримувача дії, тобто особи чи предмета, на користь або назустріч яким спрямована дія [S1].

Ключовий методичний акцент у початковій та середній школі робиться на розрізненні давального та місцевого відмінків, оскільки їхні форми, особливо в іменниках І відміни, можуть збігатися. Підручник для 4 класу чітко розмежовує їх за двома критеріями: 1) **питання** (*«кому?»* проти *«на кому?»*) та 2) **наявність прийменника** (давальний зазвичай безприйменниковий, місцевий — завжди з прийменником) [S1]. Наприклад: *«Ґрунт дає дереву (Д. в.) поживні речовини. По дереву (М. в.) повзе мурашка»* [S1].

Другий важливий аспект, який вивчається у 6-му класі, — це **чергування приголосних** основи `[г], [к], [х]` на `[з'], [ц'], [с']` перед закінченням `-і` в давальному (та місцевому) відмінках однини іменників І відміни [S1, S3164]. Наприклад: *нога → нозі, рука → руці, муха → мусі* [S3164].

У старших класах (6-10 клас) поглиблюється вивчення відмінювання іменників ІІ відміни, зокрема чоловічого роду. Тут центральною темою є **паралельні закінчення** `-ові, -еві (-єві)` та `-у (-ю)` в давальному відмінку однини. Підручники зазначають, що більшість таких іменників мають обидві форми [S3388, S672, S3164]. Рекомендації щодо їх уживання спрямовані на уникнення одноманітності в тексті: *«Щоб уникнути одноманітних відмінкових закінчень, спочатку вживаємо закінчення -ові, -еві (-єві), а тоді – -у (-ю)»*. Приклад: *панові директору, Василеві Симоненку* [S3388].

Таким чином, шкільний підхід є послідовним: від базової функції (адресат) і розрізнення з омонімічними формами — до складніших морфологічних явищ, як-от чергування приголосних і варіативність закінчень.

## Повна парадигма

Давальний відмінок є частиною системи відмінювання іменників, і його закінчення залежать від відміни, роду та групи іменника.

### I відміна

До І відміни належать іменники жіночого, чоловічого та спільного роду із закінченням `-а(-я)` в називному відмінку [S3387]. В давальному відмінку однини вони мають закінчення `-і` (тверда та мішана групи) або `-ї` (м'яка група). Ключова особливість — чергування кінцевих приголосних основи `г→з'`, `к→ц'`, `х→с'` перед закінченням `-і` [S3164].

| Група | Називний (хто? що?) | Кінцевий приголосний основи | Перетворення | Давальний (кому? чому?) |
| :--- | :--- | :--- | :--- | :--- |
| **Тверда** | кни**г**а | г | `г` → `з'` | кни**зі** |
| | ру**к**а | к | `к` → `ц'` | ру**ці** |
| | му**х**а | х | `х` → `с'` | му**сі** |
| | сестр**а** | р (твердий) | - | сестр**і** |
| | Микол**а** | л (твердий) | - | Микол**і** |
| **М'яка** | пíсн**я** | н' (м'який) | - | пісн**і** |
| | земл**я** | л' (м'який) | - | земл**і** |
| | Марí**я** | й | - | Марí**ї** |
| **Мішана** | пло**щ**а | щ (шиплячий) | - | пло**щі** |
| | кру**ч**а | ч (шиплячий) | - | кру**чі** |

У множині всі іменники І відміни в давальному відмінку мають закінчення `-ам` (тверда, мішана) або `-ям` (м'яка): *книжкам, сестрам, пісням, Маріям, площам*.

### II відміна

Це найскладніша для вивчення відміна, що включає іменники чоловічого роду (нульове закінчення або `-о`) та середнього роду (`-о, -е, -я`) [S3387].

**Чоловічий рід**

Головною особливістю є варіативність закінчень `-ові` та `-у` (тверда група), `-еві` та `-у` (мішана група), `-еві/-єві` та `-ю` (м'яка група) для назв істот. Для назв неістот переважно використовується `-у(-ю)` [S3388, S672].

| Група | Називний (хто? що?) | Давальний (кому? чому?) |
| :--- | :--- | :--- |
| **Тверда** | батько | батьк**ові**, батьк**у** |
| | студент | студент**ові**, студент**у** |
| | директор | директор**ові**, директор**у** |
| | Київ | Києв**у** |
| **М'яка** | учитель | учител**еві**, учител**ю** |
| | Андрій | Андрі**єві**, Андрі**ю** |
| | край | кра**єві**, кра**ю** |
| **Мішана** | читач | читач**еві**, читач**у** |
| | товариш | товариш**еві**, товариш**у** |

**Середній рід**

Іменники середнього роду в давальному відмінку однини мають закінчення `-у`: *озеру, селу, місту*. Іменники на `-я` мають закінчення `-ю`: *життю, знанню* [S3388, S672].

У множині всі іменники ІІ відміни мають закінчення `-ам(-ям)`: *студентам, учителям, читачам, озерам, містам, знанням*.

### III відміна

Іменники жіночого роду з нульовим закінченням та іменник *мати* [S3387]. В давальному відмінку однини вони мають закінчення `-і`. Якщо основа закінчується на один приголосний (крім губного та `р`), відбувається подовження: *сіль → солі, ніч → ночі, радість → радості*.

| Називний (хто? що?) | Давальний (кому? чому?) |
| :--- | :--- |
| радість | радост**і** |
| любов | любов**і** |
| ніч | ноч**і** |
| сіль | сол**і** |
| мати | матері |

У множині закінчення `-ам`: *радощам, ночам, матерям*.

### IV відміна

Іменники середнього роду на `-а(-я)`, що при відмінюванні отримують суфікси `-ат-/-ят-, -ен-` [S3387].

| Називний (хто? що?) | Основа в непрямих відмінках | Давальний (кому? чому?) |
| :--- | :--- | :--- |
| тел**я** | тел**ят**- | тел**яті** |
| курч**а** | курч**ат**- | курч**аті** |
| ім'**я** | ім**ен**- | ім**ені** |

У множині закінчення `-ам`: *телятам, курчатам, іменам*.

## Частотність і пріоритети

На рівні А2/В1 пріоритетним є засвоєння основних функцій та форм давального відмінка.

1.  **Функція адресата дії:** Це найчастіше вживання. Учень має впевнено будувати речення з дієсловами, що вимагають давального відмінка: *давати комусь, допомагати комусь, дзвонити комусь, дякувати комусь, надсилати комусь, казати комусь*.
    *   *Приклад*: *Олеся подарувала **Павлові** книжку* [S1].
2.  **Форми І відміни з чергуванням:** Оскільки іменники І відміни (особливо жіночі імена та назви родичів: *мама, сестра, подруга*) є високочастотними, правильне утворення форм `мамі`, `сестрі`, `подрузі` є критично важливим для базової комунікації.
3.  **Форми ІІ відміни (чол. рід, істоти):** Здатність утворювати паралельні форми, як-от *другові/другу*, *братові/брату*, *учителеві/учителю*, є ознакою вільного володіння. На початковому етапі достатньо засвоїти одну з них (наприклад, більш універсальну `-у(-ю)`), але для уникнення тавтології та природності мовлення важливо знати обидві [S3388].
4.  **Іменники IV відміни:** Форми на кшталт *дитині* (від *дитя*) є дуже поширеними. Форми *імені*, *теляті* трапляються рідше, але важливі для засвоєння системного правила.
5.  **Безособові конструкції:** Конструкції для вираження стану, віку, потреби (*мені холодно, сестрі 20 років, студенту треба відпочити*) є надзвичайно частотними і мають бути доведені до автоматизму.

## Типові помилки L2 (англомовні учні)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я дзвоню *мій друг*. | Я дзвоню *моєму другові* (або *другу*). | Прямий переклад з англійської "I call my friend", де "my friend" є прямим додатком. В українській мові дієслово *дзвонити* вимагає давального відмінка. |
| Це подарунок *тобі*. | Це подарунок *для тебе*. | Плутанина між давальним відмінком (адресат дії) і конструкцією `для + родовий відмінок` (призначення, "for someone"). Хоча розмовно перший варіант можливий, стилістично правильним є другий. |
| Я допомагаю *моя подруга*. | Я допомагаю *моїй подрузі*. | Помилка у відмінюванні І відміни. Учень ігнорує чергування `г → з` та закінчення `-і`. Це поширена помилка через складність морфонологічних змін. [S3164] |
| Я їду в гості *брату*. | Я їду в гості *до брата*. | Неправильне вживання давального відмінка замість родового з прийменником `до` для позначення напрямку до особи. Давальний не використовується для позначення руху *до* когось. |
| По *вулиці* їдуть машини. | По *вулиці* (М.в.) їдуть машини. | Формально збіг. Але учень може думати, що це Д. в. через питання "по чому?". Це місцевий відмінок. Плутанина виникає через омонімію форм та питання, як показано в підручнику [S1]. Розрізнення: `дати назву вулиці` (Д.в.) vs `йти по вулиці` (М.в.). |

## Деколонізаційні застереження

При вивченні української граматики важливо уникати хибних аналогій з російською мовою, оскільки це може призвести до системних помилок (суржику) та неправильного розуміння структури української мови.

1.  **Закінчення `-ові, -еві` є нормою, а не винятком.** В українській мові для іменників чоловічого роду ІІ відміни (істот) закінчення `-ові, -еві` є повноцінним, продуктивним і часто стилістично кращим варіантом, ніж `-у, -ю` [S3388]. У сучасній російській мові аналогічного явища немає, там послідовно вживається закінчення `-у` (*брату, учителю*). Спроба представити українське `-ові` як щось архаїчне або необов'язкове є помилковою і часто є наслідком русифікаційного впливу.
    *   **УКР**: *Я написав листа **братові** / **брату**.*
    *   **РОС**: *Я написал письмо **брату**.* (лише один варіант)

2.  **Чітке розрізнення давального та місцевого відмінків.** В українській мові конструкції з прийменником `по` для позначення руху по поверхні вимагають місцевого відмінка: *йти по дорозі (М. в.), плавати по морю (М. в.)*. У російській мові в аналогічних конструкціях вживається давальний відмінок: *идти по дороге (Д. в.), плавать по морю (Д. в.)*. Це одна з ключових граматичних розбіжностей. Помилкове вживання давального відмінка тут є яскравим маркером російського впливу.

3.  **Відсутність омонімії з прийменниковим відмінком у множині.** В українській мові давальний (`-ам, -ям`) та місцевий (`-ах, -ях`) відмінки множини мають різні закінчення.
    *   **УКР**: *Я допомагаю **студентам** (Д. в.). Я розповів про **студентів** (Р. в.), але мова йшла про їхні успіхи на іспитах, які були в різних **університетах** (М. в.).*
    *   **РОС**: *Я помогаю **студентам** (Д. в.). Я рассказал о **студентах** (П. в.).*
    Навіть у тих випадках, де в українській місцевий, у російській часто прийменниковий. Наприклад, *читати по складах* (М. в., укр.) vs *читать по складам* (Д. в., рос.) [S3388].

## Рекомендації для вправ

Для ефективного засвоєння давального відмінка рекомендується трифазна послідовність вправ.

**Фаза 1: Розпізнавання та ідентифікація**
*   **Тип вправи:** Знайти іменники в давальному відмінку в тексті, поставити до них питання.
*   **Мета:** Навчити учня ідентифікувати функцію давального відмінка (адресат).
*   **Приклад:** *Прочитайте речення. Знайдіть іменники, що відповідають на питання «кому? чому?». Визначте їх відмінок.* (за аналогією з вправою 135 [S1]).

**Фаза 2: Формотворення**
*   **Тип вправи:** Поставити іменники з називного відмінка у давальний.
*   **Мета:** Відпрацювати правила відмінювання: закінчення та чергування приголосних.
*   **Приклади:**
    1.  *Змініть подані слова так, щоб відбувалося чергування приголосних звуків (рука → руці, нога → нозі)*. (за аналогією з вправою 140 [S1]).
    2.  *Запишіть два варіанти форми іменників давального відмінка однини: Президент, батько, Юрій, програміст*. (вправа 393 [S361]).
    3.  *Поставте іменники у формі давального відмінка: (дівчинка), (учитель), (кінь), (радість), (ім'я)*.

**Фаза 3: Контрольована та вільна практика**
*   **Тип вправи:** Заповнення пропусків у реченнях; відповіді на запитання; побудова власних речень.
*   **Мета:** Інтегрувати знання у зв'язне мовлення.
*   **Приклади:**
    1.  *Спишіть речення, уживаючи подані в дужках іменники в правильній формі: «Після вистави публіка влаштувала овацію (соліст балету)»*. (вправа 395 [S361]).
    2.  *Складіть речення, кому ви допомагаєте / дзвоните / даруєте подарунки.*
    3.  *Напишіть електронний лист (звернення), використовуючи іменники в давальному відмінку для позначення адресата (за аналогією з вправою 394 [S361]).*

## Зв'язки з іншими темами

-   **Передумови:** Для розуміння давального відмінка учень повинен знати поняття **роду**, **числа** та володіти базовим уявленням про **відміни** та **групи** іменників [S3387].
-   **Місцевий відмінок:** Давальний відмінок тісно пов'язаний із місцевим через омонімію форм в І відміні. Їх розрізнення є однією з перших тем при вивченні відмінків [S1].
-   **Родовий відмінок:** Протиставляється давальному при вираженні призначення (конструкція *для + родовий відмінок*) та напрямку руху (*до + родовий відмінок*).
-   **Займенники:** Особові займенники в давальному відмінку (*мені, тобі, йому, їй, нам, вам, їм*) є високочастотними і часто засвоюються як лексичні одиниці ще до формального вивчення відмінків.
-   **Безособові конструкції:** Давальний відмінок є ключовим для вираження стану, модальності, віку: *Мені сумно. Їй потрібно вчитися. Хлопчику десять років.*
