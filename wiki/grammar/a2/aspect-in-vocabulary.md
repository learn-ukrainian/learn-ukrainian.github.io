## Як це пояснюють у школі

Поняття виду дієслова є фундаментальним в українській граматиці і вводиться поетапно. Вперше з ним знайомляться у 6-му класі, де закладаються основи: розрізнення доконаного та недоконаного виду через питання `що робити?` та `що зробити?` [S4]. У старших класах, наприклад, у 10-му, ця тема поглиблюється, систематизуються способи творення видових пар, розглядаються одновидові та двовидові дієслова [S1].

Основний дидактичний підхід полягає в тому, щоб представити вид не як ізольовану категорію, а як властивість дієслова, що виражає характер перебігу дії в часі — її тривалість, повторюваність, незавершеність (недоконаний вид) або ж її завершеність, результативність, обмеженість у часі (доконаний вид) [S1, S4]. Шкільні підручники наголошують, що більшість дієслів існують у вигляді **видових пар**, які є, по суті, одним і тим самим дієсловом у двох його видових іпостасях: `писати` (процес) і `написати` (результат) [S4, S6].

## Повна парадигма творення

Видові пари в українській мові утворюються кількома регулярними способами. Розуміння цих моделей дозволяє учням не зазубрювати кожну пару, а розпізнавати та утворювати їх за аналогією.

| Спосіб творення | Механізм | Приклад творення (Недоконаний → Доконаний) | Джерело |
| :--- | :--- | :--- | :--- |
| **1. Префіксація** | Додавання префікса до дієслова недоконаного виду. Це найпоширеніший спосіб. | `читати → про- + читати → прочитати` | [S1, S4] |
| | | `робити → з- + робити → зробити` | [S4, S3] |
| | | `вітати → при- + вітати → привітати` | [S4] |
| **2. Суфіксація** | Зміна або додавання суфікса. Часто відбувається у зворотному напрямку: від доконаного виду до недоконаного. | `стукнути → стук- + -а- → стукати` | [S1, S4] |
| | (доконаний → недоконаний) | `відчинити → відчин- + -я- → відчиняти` | [S1] |
| | (доконаний → недоконаний) | `розв'язати → розв'яз- + -ува- → розв'язувати` | [S1] |
| **3. Чергування звуків** | Зміна голосного або приголосного в корені, часто супроводжується суфіксацією. | `допомогти → допомаг- + -а- → допомагати` | [S8] |
| | | `видерти → вид**и**р- + -а- → вид**и**рати` | [S1] |
| | | `перевезти → перев**о**з- + -и- → перев**о**зити` | [S4] |
| **4. Зміна наголосу** | Перенесення наголосу. Часто цей спосіб єдиний розрізняє видову пару. | `розріза́ти → розрі́зати` | [S1] |
| | | `вино́сити → ви́нести` | [S4] |
| | | `засипа́ти → заси́пати` | [S1] |
| **5. Суплетивізм** | Утворення пари від абсолютно різних коренів (основ). Такі пари потрібно запам'ятовувати. | `брати → взяти` | [S1, S6] |
| | | `ловити → піймати` (або `зловити`) | [S1, S4] |
| | | `шукати → знайти` | [S1] |
| | | `говорити → сказати` | <!-- VERIFY --> |

### Одновидові та двовидові дієслова

Не всі дієслова утворюють видові пари [S1, S4].

1.  **Одновидові дієслова** існують лише в одній формі — або доконаного, або недоконаного виду.
    *   **Тільки недоконаний вид**: позначають тривалі, нерезультативні процеси, стани, стосунки. Приклади: `існувати`, `заважати`, `сподіватися` [S4], `мріяти`, `гомоніти`, `ворогувати`, `стежити` [S1].
    *   **Тільки доконаний вид**: позначають дію, що миттєво досягає результату, або дію, що означає "провести час у якійсь дії". Приклади: `завітати`, `розговоритися`, `насидітися` [S4], `розпитатися`, `натерпітися`, `провчитися` [S1].

2.  **Двовидові дієслова** — це одна й та сама форма, що може виражати значення як доконаного, так і недоконаного виду. Вид визначається з контексту. Це переважно слова іншомовного походження на `-увати`, `-ізувати`.
    *   Приклади: `атакувати`, `телефонувати`, `організувати`, `гарантувати`, `наслідувати` [S1].
    *   *Контекст недоконаного виду*: "Командир щодня **атакував** ворожі позиції" (що робив? — процес, повторювана дія).
    *   *Контекст доконаного виду*: "Сьогодні вранці командир успішно **атакував** ворожі позиції" (що зробив? — завершена, результативна дія).

## Частотність і пріоритети

Для рівнів А2-В1 пріоритетним є засвоєння найуживаніших моделей.

1.  **Найвищий пріоритет:** Творення доконаного виду за допомогою найпродуктивніших префіксів: **`з-/с-`, `про-`, `на-`, `по-`**. Учні мають у першу чергу навчитися розпізнавати й утворювати пари типу `робити/зробити`, `писати/написати`, `читати/прочитати`, `їхати/поїхати`.
2.  **Середній пріоритет:** Суплетивні пари `брати/взяти`, `говорити/сказати`, `дивитися/подивитися`. Вони є високочастотними і не підпадають під загальні правила, тому потребують окремого запам'ятовування.
3.  **Важливо для розуміння:** Розрізнення процесу і результату в минулому часі (`я читав` vs `я прочитав`) і в майбутньому (`я буду читати` vs `я прочитаю`). Це ключова функція виду в мовленні [S8].
4.  **Менш пріоритетно для активного вживання (але важливо для пасивного):** Менш продуктивні моделі творення (чергування, зміна наголосу), а також більшість одновидових та двовидових дієслів. Учні мають їх розпізнавати, але не обов'язково активно продукувати на етапі А2.

## Типові помилки L2 (англомовні учні)

Англомовні учні часто роблять помилки з видами, оскільки в англійській мові ця граматична категорія відсутня і передається лексично або через часові форми (Simple vs. Continuous/Perfect).

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я **прочитаю** книгу кожного дня. | Я **читаю** книгу кожного дня. | Для повторюваної, звичайної дії в теперішньому часі завжди використовується **недоконаний** вид. Дієслова доконаного виду взагалі не мають форм теперішнього часу [S6, S8]. |
| Завтра я **буду написати** лист. | Завтра я **напишу** лист. <br/> _або_ <br/> Завтра я **буду писати** лист. | Складена форма майбутнього часу (`буду` + інфінітив) утворюється **тільки з інфінітивом недоконаного виду**. Для доконаного виду існує тільки проста форма майбутнього часу [S8]. |
| Коли ти **робив** домашнє завдання? | Коли ти **зробив** домашнє завдання? | Питання про завершений результат (`коли?`) вимагає доконаного виду. Недоконаний вид (`робив`) більше пасував би питанню «Як довго ти робив...?» або «Що ти робив учора ввечері?». |
| Я **подивився** телевізор три години. | Я **дивився** телевізор три години. | Коли вказується тривалість дії, а не її завершення, перевага надається недоконаному виду. `Подивився` означає короткий акт перегляду, а не тривалий процес. |
| Мені подобається **зробити** фотографії. | Мені подобається **робити** фотографії. | Для позначення хобі, уподобань, загальних дій, які не мають конкретного кінцевого результату в даному контексті, використовується недоконаний вид. |
| Він **прийшов** додому і зразу **почав робити** вечерю. | Він **прийшов** додому і зразу **почав готувати** вечерю. | Хоча `робити` є універсальним дієсловом, у багатьох контекстах існують більш точні видові пари. `Готувати/приготувати` є кращою парою для їжі. Це лексична, а не суто граматична помилка, але вона пов'язана з вибором правильної видової пари. |

## Деколонізаційні застереження

Попри значну схожість, українська та російська системи дієслівного виду мають суттєві відмінності. Покладання на знання російської мови може призвести до системних помилок.

1.  **Форми майбутнього часу недоконаного виду:** В українській мові активно вживаються обидві форми: складена (`буду читати`) та синтетична (`читатиму`) [S8]. Синтетична форма (`читатиму`) є питомо українською і дуже поширеною в літературній та офіційній мові. У російській мові аналогічна синтетична форма відсутня.
2.  **Суплетивні пари:** Деякі базові пари відрізняються. Наприклад, укр. `говорити/сказати` vs. рос. `говорить/сказать`, але укр. `казати` є стилістично іншим.
3.  **Наголос:** Наголос у видових парах може суттєво відрізнятися, навіть якщо слова виглядають схоже. Порівняйте укр. `розріза́ти (НД) / розрі́зати (Д)` [S1] з російськими відповідниками. Неправильний наголос може не тільки звучати неприродно, але й змінювати вид дієслова.
4.  **Вплив суфіксів:** Суфікс `-ну-` в українській мові часто вказує на одноразову, раптову дію (доконаний вид): `стукати (НД) → стукнути (Д)` [S1], `кричати (НД) → крикнути (Д)` [S8]. Хоча подібне явище є і в російській, моделі його застосування не завжди збігаються.
5.  **Педагогічний підхід:** Українську граматику слід вивчати як самодостатню систему. Пояснення українських явищ через порівняння з російськими ("в українській так, а от в російській інакше") є хибним методичним підходом, який закріплює колоніальну оптику. Українські граматичні явища мають власну внутрішню логіку, яку й слід пояснювати [S9].

## Природні приклади

Приклади згруповано за моделлю вживання виду.

**Група 1: Минулий час — Процес (НД) vs. Результат (Д)**

*   `Учні цілий урок **розв’язували** завдання, але не могли **розв’язати** правильно.` [S4] (Показано тривалий процес і відсутність результату, а потім очікуваний результат).
*   `Чілла **читала** повість впродовж тижня, а я **прочитав** за три дні.` [S4] (Тривалість дії проти факту її завершення).
*   `Я **прибирав** у своїй кімнаті.` (НД) vs. `Прибирати й **прибрати** — різні дії.` (НД/Д) [S6] (Діалог ілюструє різницю у сприйнятті: син наголошує на процесі, мати очікує результату).

**Група 2: Майбутній час — Намір/Процес (НД) vs. План/Результат (Д)**

*   `Ще нам, браття молодії, **усміхнеться** доля.` [S9] (Доконаний вид, обіцянка конкретного результату в майбутньому).
*   `Згинуть наші воріженьки, як роса на сонці. **Запануєм** і ми, браття, у своїй сторонці.` [S9] (Доконаний вид, впевненість у майбутньому результаті).
*   `Я **буду робити** уроки завтра. / Я **робитиму** уроки завтра.` [S8] (НД, наголос на процесі, який відбуватиметься в майбутньому).
*   `Що ти **будеш робити**?` [S7] (Питання про плани на майбутнє, часто з відтінком тривалості).

**Група 3: Інфінітив у спонуканні та модальних конструкціях**

*   `Кожен із вас повинен **зробити** щось таке, що робить людей щасливими.` [S4] (Модальне дієслово "повинен" + інфінітив доконаного виду, бо йдеться про досягнення результату).
*   `Розпрягайте, хлопці, коней та й лягайте **спочивать**.` [S6] (Спонукання до дії-процесу, відпочинку).
*   `Треба **з’їсти**, щоб знову не забула.` [S7] (Необхідність виконати результативну дію).

**Група 4: Вид у загальних твердженнях та описах**

*   `Дієслова недоконаного виду **виражають** тривалу дію або стан.` [S1] (Загальне правило, постійна властивість — недоконаний вид).
*   `Дієслова доконаного виду **виражають** завершену, результативну дію.` [S1] (Аналогічно, опис постійної характеристики).
*   `Любити — пр..красне почуття.` [S6] (Недоконаний вид використовується для узагальнень).

## Рекомендації для вправ

Автор-письменник має будувати вправи за принципом поступового ускладнення.

1.  **Фаза 1: Ідентифікація та групування.**
    *   **Вправа "Дві колонки"**: Учні отримують список дієслів і мають розподілити їх на доконаний та недоконаний вид. Приклад: вправа 2 у [S6].
    *   **Вправа "Знайди пару"**: Дано таблицю з дієсловами, потрібно з'єднати видові пари.

2.  **Фаза 2: Утворення видових пар.**
    *   **Вправа "Утвори пару"**: Дано дієслово одного виду, потрібно утворити інший вид за допомогою префікса або суфікса. Приклад: вправа 516 у [S1].
    *   **Вправа "Трансформація"**: Дано речення з дієсловом недоконаного виду. Потрібно переписати речення, замінивши дієслово на доконаний вид і внісши необхідні зміни (наприклад, додавши слова `вже`, `нарешті`).

3.  **Фаза 3: Контекстуальний вибір.**
    *   **Вправа "Заповни пропуск"**: У реченні пропущено дієслово, а в дужках подано видову пару. Учень має вибрати правильний вид залежно від контексту (наявність слів `довго`, `завжди`, `вже`, `нарешті`, загальний зміст речення). Приклад: вправа 440 у [S4].
    *   **Вправа "Відповідь на питання"**: Учні відповідають на питання, використовуючи дієслово в правильному виді. `Що ти робив учора? — Я читав книгу.` vs. `Ти вже зробив домашнє завдання? — Так, я вже зробив.` Приклад: вправа 439 у [S4].
    *   **Складання коротких оповідань**: Учням дається ланцюжок дій (наприклад, `прокинутися → вмитися → поснідати → піти до школи`). Вони мають написати кілька речень, правильно вживаючи минулий час доконаного виду.

## Зв'язки з іншими темами

*   **Часи дієслова:** Вид нерозривно пов'язаний з часом. Як зазначено, дієслова доконаного виду не мають теперішнього часу, а майбутній час для різних видів утворюється по-різному [S8].
*   **Дієслівні префікси:** Тема виду є чудовою нагодою для знайомства з роллю префіксів, які не тільки змінюють вид, але й додають нових відтінків значення (`писати → написати, переписати, записати, виписати`) [S3, S5].
*   **Дієприкметники та дієприслівники:** Утворення цих форм також залежить від виду вихідного дієслова. Наприклад, дієприслівники теперішнього часу (`читаючи`) утворюються лише від дієслів недоконаного виду.
*   **Дієслова руху:** Ця група дієслів має особливо складну систему видових та префіксальних пар (`йти/ходити`, `їхати/їздити`), яка вивчається як окрема підтема.
