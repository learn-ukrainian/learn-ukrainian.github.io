## Як це пояснюють у школі

Тема пір року, місяців та свят є наскрізною в українській шкільній програмі, з'являючись у підручниках з мови та літератури з початкових класів. Вона слугує природним контекстом для введення та відпрацювання лексики, а також ключових граматичних конструкцій, пов'язаних із часом.

У 5 класі тема свят вводиться через тексти про Різдво та пов'язані з ним традиції, як-от колядки та Святвечір [S1, S2]. Учні знайомляться з лексикою на зразок: *звичай, традиція, Різдво, Великдень, Святвечір, колядка, кутя* [S2]. На цьому етапі основна увага приділяється читанню, розумінню тексту та збагаченню словникового запасу. Граматичні конструкції для позначення часу подаються у вигляді таблиць для запам'ятовування: дні тижня (`у понеділок`), місяці (`у січні`), частини доби (`вранці`, `ввечері`) та конкретні дати з використанням родового відмінка (`першого січня`) [S4].

У 6 класі тема розширюється через вивчення календарно-обрядових пісень, що систематизує знання за циклами: зимовий (колядки, щедрівки), весняний (веснянки) та літній (русальні, купальські, жниварські) [S8]. Тексти про свято Івана Купала поглиблюють культурний контекст та вводять відповідну лексику [S3, S7]. Граматично, на цьому етапі фокусуються на категорії числа іменників, використовуючи назви свят як приклади іменників, що вживаються лише в однині (*Різдво*, *Великдень*) або лише у множині (*канікули*, *іменини*) [S6]. Також на прикладі назв свят відпрацьовується правило вживання великої літери [S6].

Таким чином, шкільний підхід є інтегрованим: лексика і граматика подаються не ізольовано, а в щільному зв'язку з культурними реаліями, традиціями та фольклором, що забезпечує їх природне засвоєння.

## Повна парадигма

### Пори року (Seasons)

Назви пір року вживаються з прийменниками для позначення часу. Найчастіше використовуються форми Орудного та Місцевого відмінків.

| Пора року (Що?) | Влітку / Взимку (Коли?) | Навесні / Восени (Коли?) | Інші форми (приклад) |
| :-------------- | :---------------------- | :----------------------- | :--------------------- |
| зима́ (f)        | **взи́мку** (adv.)       | —                        | зимо́ю (О. в.)         |
| весна́ (f)       | —                       | **навесні́** (adv.)       | весно́ю (О. в.)         |
| лі́то (n)        | **влі́тку** (adv.)       | —                        | лі́том (О. в.)          |
| о́сінь (f)       | —                       | **восени́** (adv.)        | о́сінню (О. в.)         |

**Творення прислівників часу:**
- Іменники `літо` та `зима` утворюють прислівники за допомогою прийменника `в` та суфікса `-ку`: `в + літ + ку`, `в + зим + ку`.
- Іменники `весна` та `осінь` утворюють прислівники з прийменниками `на` / `во` та закінченням `-і` (форма М. в.): `навесн + і`, `восен + и`.

Зверніть увагу, що прислівникові форми `взимку`, `влітку`, `навесні`, `восени` є стилістично кращими та більш поширеними для позначення дії в межах пори року, ніж конструкції з орудним відмінком (`зимою`, `літом` тощо) [S5, S8].

### Місяці та дні тижня (Months and Days of the Week)

Для відповіді на питання «Коли?» назви місяців і днів тижня вживаються у відповідних відмінкових формах з прийменниками `у/в` [S4].

**Місяці:**

| Називний (Що?) | Місцевий (Коли? `у/в ...`) |
| :-------------- | :-------------------------- |
| сі́чень         | **у сі́чні**               |
| лю́тий          | **у лю́тому**              |
| бе́резень       | **у бе́резні**             |
| кві́тень        | **у кві́тні**              |
| тра́вень        | **у тра́вні**              |
| че́рвень        | **у че́рвні**              |
| ли́пень         | **у ли́пні**               |
| се́рпень        | **у се́рпні**              |
| ве́ресень       | **у ве́ресні**             |
| жо́втень        | **у жо́втні**              |
| листопа́д       | **у листопа́ді**           |
| гру́день        | **у гру́дні**              |

**Дні тижня:**

| Називний (Що?) | Знахідний (Коли? `у/в ...`) |
| :-------------- | :--------------------------- |
| понеді́лок     | **у понеді́лок**            |
| вівто́рок      | **у вівто́рок**             |
| середа́         | **у се́реду**               |
| четве́р         | **у четве́р**               |
| п'я́тниця       | **у п'я́тницю**             |
| субо́та         | **у субо́ту**               |
| неді́ля         | **у неді́лю**               |

### Конструкції для позначення дати

Для позначення конкретної дати використовується конструкція **«порядковий числівник у родовому відмінку + назва місяця у родовому відмінку»**. У шкільній практиці це часто подається у спрощеному вигляді: `[число] [місяць]` з числівником у формі Н.в. або Р.в. [S4]. Однак повна і граматично правильна форма вимагає родового відмінка для обох компонентів.

**Приклад творення:**
`Перший (Н.в.) + січень (Н.в.) → Пе́ршого (Р.в.) + сі́чня (Р.в.)`

| Дата (приклад) | Повна форма в реченні (Р.в.) |
| :------------- | :---------------------------------- |
| 1 січня        | Свято відбудеться **пе́ршого сі́чня**. |
| 6 січня        | Святвечір відзначають **шо́стого сі́чня**. [S1] |
| 7 липня        | Свято Івана Купала — **сьо́мого ли́пня**. [S3] |
| 24 грудня       | В угорців Святвечір **два́дцять четве́ртого гру́дня**. [S4] |

### Назви свят (Holiday Names)

Назви свят в українській мові є власними назвами і пишуться з великої літери [S6]. Граматично вони поводяться як звичайні іменники, але деякі з них мають особливості у вживанні числа.

1.  **Тільки однина:** Назви, що позначають унікальне свято або абстрактне поняття.
    - *Різдво́, Вели́кдень, Нови́й рік, Водохре́ща* [S6, S9].

2.  **Тільки множина (pluralia tantum):** Назви, що позначають обряди, процеси або тривалі періоди.
    - *кані́кули, імени́ни, хрести́ни, весело́щі, рокови́ни, Зеле́ні свя́та*.
    - Приклад: *Я взимку їду туди. Адже у школі зимові канікули.* [S2]

3.  **Мають обидві форми:** Зазвичай це стосується свят, які можуть сприйматися як родове поняття.
    - *свя́то → свя́та*.

## Частотність і пріоритети

Для рівня А2 пріоритетним є засвоєння конструкцій, що дозволяють орієнтуватися в часі та розповідати про базові події.

**Високий пріоритет (must know):**
1.  **Назви місяців у місцевому відмінку:** `у січні`, `у лютому` і т.д. Це ключова конструкція для відповіді на питання «Коли?».
2.  **Назви пір року у прислівниковій формі:** `влітку`, `взимку`, `навесні`, `восени`. Це найпоширеніший спосіб позначення сезону.
3.  **Назви головних свят:** *Новий рік, Різдво, Великдень*. Учні повинні впізнавати їх і вміти використовувати в простих реченнях (`Я люблю Різдво`).
4.  **Дні тижня у знахідному відмінку:** `у понеділок`, `у суботу` для планування подій.
5.  **Питальні слова:** *Коли? Якого числа? Яке свято?*

**Середній пріоритет (should know):**
1.  **Конструкція для дати з родовим відмінком:** `першого січня`. Хоча вона складніша, вона необхідна для точного позначення дат (напр., дня народження).
2.  **Лексика традицій:** *колядка, щедрівка, кутя, писанка, вінок*. Ці слова часто зустрічаються в текстах про культуру [S1, S5].
3.  **Множинні іменники:** *канікули, свята*.

**Низький пріоритет (can wait):**
1.  **Рідковживані або історичні свята:** *Введення, Катерини* (згадані в [S5]).
2.  Повні парадигми відмінювання всіх святкових іменників. Достатньо засвоїти форми, що найчастіше вживаються.
3.  Деталі про походження обрядів (напр., зв'язок `ластівки` з давнім Новим роком [S9]), хоча це цікавий культурний аспект.

## Типові помилки L2 (англомовні учні)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Ми святкуємо Різдво *в грудень*. | Ми святкуємо Різдво **у грудні**. | Англомовні учні переносять модель "in December", але українська мова вимагає Місцевого відмінка (`у/в` + `-і`/`-у`) для назв місяців у відповідь на питання «Коли?». |
| Мій день народження *п'ять травень*. | Мій день народження **п'ятого травня**. | В англійській мові для дати використовується кардинальний або порядковий числівник з місяцем. В українській мові стандартна конструкція вимагає **родового відмінка** і для порядкового числівника, і для назви місяця. |
| *На зиму* я їду в Карпати. | **Взимку** я їду в Карпати. (Або **Зимою**...) | Помилка виникає через калькування англійської фрази "for the winter". Для позначення дії, що відбувається протягом пори року, використовується прислівник (`взимку`) або іменник в орудному відмінку (`зимою`). Конструкція `на зиму` означає мету або тривалість (напр., `залишитися на зиму`). |
| У нас скоро *різдвяні свято*. | У нас скоро **різдвяні свята**. | В англійській мові "Christmas holiday" може бути в однині. В українській мові, якщо мова йде про період святкувань (канікули), вживається множина. Слово `свято` (однина) узгоджується з `різдвяне`, а `свята` (множина) — з `різдвяні`. |
| — Коли Новий рік? <br> — *Перший січень*. | — Коли Новий рік? <br> — **Першого січня**. | Це та сама помилка, що й у другому пункті, але у форматі короткої відповіді. У відповідь на питання «Коли?» дата має стояти в родовому відмінку. |
| Дякую, *доброго вечора*! | Дякую, **добрий вечір**! | Поширений русизм, калька з російського `доброго вечера`. Українська традиційна етикетна формула — `добрий вечір` у називному відмінку, як це підкреслено в текстах про колядки [S9]. |

## Деколонізаційні застереження

При викладанні цієї теми важливо наголошувати на автентичних українських мовних і культурних рисах, щоб уникнути поширених пасток, пов'язаних із російським впливом.

1.  **Вітальні формули:** Обов'язково слід наголосити на традиційності форм називного відмінка `Добрий день!` та `Добрий вечір!`. У джерелах прямо вказується, що форма `Добрий вечір тобі, пане господарю` є давньою мовною традицією, зафіксованою в колядках [S9]. Форми родового відмінка (`Доброго дня`, `Доброго вечора`) поширилися під впливом російської мови і, хоча є поширеними, не є питомо українськими. Винятком є `Доброго ранку!`, що вкорінився в мові.

2.  **Назви свят:** Важливо використовувати виключно українські назви: `Різдво` (не *Рождество*), `Великдень` (не *Пасха* в загальному вжитку, хоча слово `паска` позначає великодній хліб [S2]). Свято `Івана Купала` має глибокі дохристиянські корені в українській культурі, що варто підкреслити [S3, S7, S8].

3.  **Історичний контекст Нового року:** Слід згадати, що святкування Нового року в березні, прив'язане до весняного рівнодення, було давньою українською традицією [S6]. Образ `ластівочки` у щедрівці «Щедрик» є прямим свідченням цього [S9]. Це розриває асоціативний зв'язок сучасного святкування Нового року виключно з радянською/російською традицією "Нового года".

4.  **Фокус на унікальних традиціях:** Вивчення таких понять, як `дідух`, `кутя`, `писанка`, `вертеп` [S5, S1], підкреслює самобутність української культури. Пояснення їх символізму (`кутя — символ урожаю` [S5]) допомагає учням зрозуміти глибинний зв'язок мови, звичаїв та національної ідентичності.

## Природні приклади

**Група 1: Різдвяні свята та традиції**
1.  Увечері 6 січня, на Святвечір, українці готують 12 пісних страв. [S1]
2.  Обов’язково на столі має бути кутя — традиційна різдвяна страва. [S1]
3.  Діти й дорослі ходять до родичів, знайомих і співають колядки. [S1]
4.  А другий же празник — Василя святого, а третій же празник — Святе Водохреща. [S9]

**Група 2: Літні свята та обряди**
5.  Свято Івана Купала відзначають у ніч із 6 на 7 липня. [S3]
6.  Парубки розпалювали багаття, а дівчата неподалік закопували принесену з лісу березову гілку. [S3]
7.  Після цього дівчата йшли до річки й пускали за течією по два віночки. [S3]

**Група 3: Календарні цикли та пори року**
8.  Узимку найпопулярнішими були щедрівки. [S8]
9.  Колись наші предки святкували язичницький новий рік 20 березня, під час весняного рівнодення. [S9]
10. Коли все життя наших предків було пов’язане зі змінами в природі, вони святкували Новий рік навесні. [S6]

**Група 4: Дати та особисті події**
11. Першого січня — Новий рік. [S4]
12. Я взимку їду туди. Адже у школі зимові канікули. [S2]

## Рекомендації для вправ

**Фаза 1: Розпізнавання та базове відтворення**
- **Вправа на співставлення:** З'єднати назву свята з його символом (Різдво → ялинка/зірка, Великдень → писанка/паска, Івана Купала → вінок/багаття).
- **Заповнення пропусків (лексика):** Дати короткий текст про свята, де пропущені ключові слова (*колядки, кутя, вінок* і т.д.).
- **Календар:** Дати учням календар на рік і попросити позначити свята, про які вони дізналися, та підписати місяці українською.

**Фаза 2: Контрольована практика (граматика)**
- **Трансформація речень:** Перетворити речення з моделлю «Свято / [назва місяця]» на модель «Ми святкуємо / у [назва місяця в М.в.]». *Зразок: Зараз січень. → У січні ми святкуємо Різдво.*
- **Заповнення пропусків (граматика):** Дати речення з пропущеними прийменниками або неправильними відмінками. *Зразок: Я народився \_\_\_ (травень). → Я народився **у травні**.* *Ми їдемо в гори \_\_\_ (зима). → Ми їдемо в гори **взимку**.*
- **Питання-відповідь:** Побудувати діалоги за моделлю: "Коли святкують Різдво?" – "Різдво святкують сьомого січня".

**Фаза 3: Вільна продукція**
- **Опис улюбленого свята:** Попросити учнів написати 3-5 речень про своє улюблене свято за планом: 1) Назва свята. 2) Коли його святкують? 3) Що люди роблять? (використовуючи лексику з текстів [S1, S3]).
- **Складання розкладу:** Дати завдання скласти свій розклад на тиждень, використовуючи конструкції `у понеділок`, `у вівторок` і т.д.
- **Діалог:** Розіграти діалог, де один учень запитує іншого про його день народження та про те, як він/вона зазвичай святкує, спираючись на приклади з діалогів у [S2].

## Зв'язки з іншими темами

- **Передумови:**
    - **Іменник:** базове розуміння роду та числа.
    - **Відмінки:** початкове знайомство з Називним та Місцевим відмінками.
    - **Прийменники:** знання базових прийменників `у/в`, `на`.

- **Теми, що вводяться або поглиблюються:**
    - **Місцевий відмінок (Locative Case):** Систематизується його використання для позначення часу з місяцями (`у січні`).
    - **Родовий відмінок (Genitive Case):** Вводиться його функція для позначення точної дати (`п'ятого травня`).
    - **Орудний відмінок (Instrumental Case):** Вводиться для позначення часу з порами року (`зимою`), хоча прислівникова форма є пріоритетною.
    - **Порядкові числівники (Ordinal Numbers):** Невід'ємна частина конструкції для позначення дати.
    - **Прислівники часу (Adverbs of Time):** Вводяться ключові прислівники `взимку, влітку, навесні, восени`.

- **Майбутні теми (що уможливлює):**
    - **Дієслова доконаного/недоконаного виду:** Розповіді про традиції (`готують` vs `приготували кутю`) стають більш деталізованими.
    - **Складні речення:** Тема свят є чудовим контекстом для побудови складних речень з причиною, часом та умовою (`Коли настає Різдво, ми...`).
    - **Пасивний стан:** Опис традицій (`страви готуються`, `пісні співаються`).
