## Як це пояснюють у школі (How Schools Teach This)
У шкільній програмі України тема "Навчання і робота" вводиться поступово, поєднуючи лексику з відповідними граматичними структурами. Це не стільки окремий граматичний розділ, скільки наскрізна практична тема, що розвивається від молодших до старших класів.

Основна прогресія виглядає так:

1.  **Початковий етап (5-6 класи):** Вводяться базові назви професій та навчальних закладів. Ключовою граматичною конструкцією стає **орудний відмінок** для вираження професії чи ролі. У підручнику Бетси для 6 класу (Source 2) ця тема розкривається через діалог, де персонажі обговорюють навчання батьків. Використовуються фрази «Вона хоче бути **лікаркою**», «Мріяв бути **архітектором**». Це показує, що конструкція `бути + [професія в орудному відмінку]` вводиться рано і на практичних прикладах.

2.  **Середній етап (7-9 класи):** Розширюється лексика: з'являються назви факультетів, спеціальностей, посад. Активно використовуються **місцевий** та **родовий** відмінки. У підручнику Заболотного (Source 15) пропонується розіграти діалог про майбутню спеціальність. В текстах про відомих людей (Вернадський, Патон, Грушевський) детально описується їхній освітній шлях: `навчання на природничому відділенні фізико-математичного факультету` (Source 1), `закінчив Київський політехнічний інститут` (Source 4), `вступив на історико-філологічний факультет Київського університету` (Source 13).

3.  **Старший етап (9-11 класи):** Акцент зміщується на практичне застосування мови в діловому контексті. Учні вчаться складати офіційні документи: автобіографію, резюме, заяву. Підручник Авраменка для 9 класу (Sources 5, 10, 33) містить зразки цих документів, де чітко видно граматичні конструкції: `вступила до Київського національного університету ... на економічний факультет за спеціальністю «Менеджмент та економіка»` (Source 5). У підручнику для 11 класу (Source 22) подається зразок характеристики, де оцінюються ділові якості особи, наприклад: `Мартиненко Сергій Степанович працює на ... заводі ... з 2012 року`.

Отже, українська шкільна методика навчає цієї теми не через абстрактні правила, а через контекст, моделювання реальних ситуацій (діалоги, розповіді про плани) та створення практичних документів. Граматика подається як інструмент для досягнення комунікативної мети.

## Повна парадигма (Full Paradigm)

Для обговорення навчання та роботи на рівні А2 ключовими є три відмінки: орудний, місцевий та родовий, а також фемінітиви для назв професій.

### 1. Орудний відмінок (Instrumental Case)
Використовується для відповіді на питання «Ким?» і є основним для позначення професії з дієсловами `бути`, `стати`, `працювати`.

| Рід | Закінчення основи | Однина (Ким?) | Множина (Ким?) | Приклади |
| :-- | :--- | :--- | :--- | :--- |
| **Чоловічий** | Приголосний | **-ом** | **-ами** | інженер → інженер**ом**, лікар → лікар**ем*** |
| | **-й**, **-ь** | **-єм** / **-ем** | **-ями** / **-ами** | водій → воді**єм**, учитель → учител**ем** |
| **Жіночий** | **-а** | **-ою** | **-ами** | медсестра → медсестр**ою** |
| | **-я** | **-ею** / **-єю** | **-ями** | вчителька → вчительк**ою**, швачка → швачк**ою** |

*Примітка: Чергування `о/е` в суфіксі `-ар` (`лікар` - `лікарем`).

### 2. Фемінітиви (Feminine Forms of Professions)
У сучасній українській мові активно вживаються жіночі форми назв професій. Їх утворення є важливою частиною теми.

| Чоловіча форма | Жіноча форма | Суфікс |
| :--- | :--- | :--- |
| вчитель | вчитель**ка** | -ка |
| студент | студент**ка** | -ка |
| письменник | письменни**ця** | -иця |
| лікар | лікар**ка** | -ка |
| директор | директор**ка** | -ка |
| продавець | продавч**иня** | -иня |
| перекладач | перекладач**ка** | -ка |
| архітектор | архітектор**ка** <!-- VERIFY --> | -ка |

### 3. Місцевий відмінок (Locative Case)
Відповідає на питання «Де?» і використовується для позначення місця роботи чи навчання.

| Конструкція | Приклади |
| :--- | :--- |
| `в / у + [іменник у місц. в.]` | навчатися **в університеті**, **в школі**, **в інституті** |
| `на + [іменник у місц. в.]` | працювати **на заводі**, **на пошті**, навчатися **на факультеті**, **на курсах** |

### 4. Родовий відмінок (Genitive Case)
Відповідає на питання «Чого? Кого?» і використовується для позначення приналежності, спеціалізації.

| Конструкція | Приклади |
| :--- | :--- |
| `вступити до + [іменник у род. в.]` | вступити **до університету**, **до коледжу**, **до школи** (Source 39) |
| `студент / викладач + [іменник у род. в.]` | студент **університету**, викладач **історії** |
| `факультет / відділення + [іменник у род. в.]` | факультет **економіки**, відділення **філології** |

## Частотність і пріоритети (Frequency & Priorities)

Для рівня А2 пріоритети слід розподілити так:

1.  **Найвищий пріоритет:** Конструкція **`бути / працювати + [професія в орудному відмінку]`**. Це основа для розповіді про себе та інших. `Ким ти хочеш бути?` — одне з перших складних питань, яке може поставити учень.
2.  **Високий пріоритет:** Вживання **місцевого відмінка з прийменниками `в/у` та `на`** для позначення місця навчання/роботи (`в школі`, `на заводі`). Це базова конструкція для відповіді на питання «Де?».
3.  **Середній пріоритет:** **Фемінітиви**. Учень має вміти розпізнавати та вживати найпоширеніші з них (`вчителька`, `студентка`, `лікарка`), оскільки вони є нормою в сучасному мовленні (Source 2).
4.  **Середній пріоритет:** Конструкція **`вступити до + [навчальний заклад у родовому відмінку]`**. Це важливий етап в описі освітнього шляху (Source 5, 39).
5.  **Низький пріоритет (для розпізнавання):** Складніші конструкції з родовим відмінком для назв факультетів та спеціальностей (`студент економічного факультету за спеціальністю "Менеджмент"`). На рівні А2 учень повинен їх розуміти в тексті (напр., в автобіографії), але активне вживання можна відкласти до В1.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Моя мама є **лікар**. | Моя мама є **лікарем** / Моя мама — **лікарка**. | В англійській мові `to be` поєднується з називним відмінком. В українській мові для означення професії/ролі використовується орудний відмінок (ким є?). |
| Я **навчаюся** українську мову. | Я **вивчаю** українську мову. | Дієслово `навчатися` означає "бути студентом" і вживається з місцем навчання (`навчатися в університеті`). Дієслово `вивчати` означає "вивчати предмет" і вживається з прямим додатком у знахідному відмінку. |
| Він хоче **поступити в** університет. | Він хоче **вступити до** університету. | Дієслово `вступити` (у значенні "бути зарахованим") вимагає прийменника `до` та родового відмінка. Конструкція `поступити в` є поширеною помилкою, частково через вплив російської мови та плутанину з `вступити в` у значенні "увійти". (Source 19). |
| Вона працює **на** школі. | Вона працює **в** школі. | Вибір прийменника `в` чи `на` з місцем роботи є ідіоматичним. Зі словами `школа`, `університет`, `лікарня`, `офіс` вживається `в`. Зі словами `завод`, `фабрика`, `пошта`, `факультет`, `ринок` — `на`. |
| Я буду **інженер**. | Я буду **інженером**. | Як і з `є`, дієслово `бути` в майбутньому часі з назвою професії вимагає орудного відмінка. |
| Моя сестра — **програміст**. | Моя сестра — **програмістка**. | Ігнорування фемінітивів. Хоча в розмовній мові чоловіча форма іноді може вживатися для жінок, у сучасній літературній мові та офіційному стилі фемінітиви є нормою і важливою частиною мовної культури. |

## Деколонізаційні застереження (Decolonization Notes)

**MANDATORY — never omit.** Це критично важливий розділ для правильного викладання української мови.

1.  **Активне вживання фемінітивів:** На відміну від сучасної російської мови, де використання чоловічих форм для позначення професій жінок є поширеним (особливо в офіційному мовленні), в українській мові спостерігається стійка тенденція до нормалізації та активного вживання фемінітивів (`директорка`, `професорка`, `міністерка`). Підручники (Source 2: `лікарка`, `терапевтка`) та офіційні документи відображають цю норму. Навчання фемінітивів — це не просто граматична деталь, а частина сучасної української мовної ідентичності та поваги.

2.  **Лексичні відмінності:** Слід наголошувати на питомо українській лексиці, уникаючи поширених русизмів. Наприклад, `навчальний заклад` (Source 6) або `заклад освіти` (Source 19) замість кальки `учбовий заклад`. Дієслово `вступити **до** вишу` замість `поступити в...`. Слід підкреслювати, що `вдача` (характер) і `удача` (успіх) — це різні слова, а не варіанти одного (Source 27).

3.  **Внутрішня логіка, а не винятки:** Українська граматика повинна викладатися як самостійна, логічна система. Неприпустимо пояснювати українські форми як "відхилення від російських норм". Наприклад, вибір прийменників `в/на` з місцем роботи має свої історичні та логічні підстави в українській мові і не повинен порівнюватися з російською.

4.  **Культурний контекст:** Приклади для ілюстрації теми мають бути з українського життя, історії та культури. Джерела надають чудові приклади біографій українських вчених та діячів (Вернадський, Патон, Грушевський, Крушельницька), що дозволяє вивчати граматику в осмисленому, деколонізованому контексті.

## Природні приклади (Natural Examples)

**Група 1: Орудний відмінок для позначення професії/ролі**
*   Вона хоче бути **лікаркою**. (Source 2)
*   Він цікавився архітектурою. Мріяв бути **архітектором**. (Source 2)
*   Його вважають **основоположником** радіогеології як самостійної наукової галузі. (Source 1)
*   2016 року, закінчивши названий вищий навчальний заклад, був призначений на посаду **нотаріуса** Київської нотаріальної контори № 1. (Source 39)

**Група 2: Місцевий відмінок для позначення місця навчання/роботи**
*   Мама навчається **в Ужгородському національному університеті на медичному факультеті**. (Source 2)
*   Мій тато навчався **у Львівському політехнічному університеті**. (Source 2)
*   Дитинство Володимир Вернадський провів **у Петербурзі**, а потім — **Харкові**, де вступив до гімназії. (Source 1)

**Група 3: Вступ та закінчення навчальних закладів**
*   2015 року **вступила до Київського національного університету** імені Тараса Шевченка на економічний факультет. (Source 5)
*   Закінчив **Київський політехнічний інститут** за спеціалізацією «системи керування літальними апаратами». (Source 4)
*   Прошу **зарахувати мене на перший курс** Інституту людини. (Source 33)

**Група 4: Розповідь про спеціальність та навчання**
*   Я навчаюся **сумлінно й старанно**. (Source 3)
*   Майбутній академік займається **мінералогією і кристалографією, ґрунтознавством, філософією**. (Source 1)
*   Протягом навчання в школі й університеті виконувала **обов’язки старости** класу та групи. (Source 5)

## Рекомендації для вправ (Activity Concepts)

**Phase 1: Розпізнавання та базове відтворення**
*   **Matching:** Поєднати зображення людей різних професій з їхніми назвами (чоловіча та жіноча форми).
*   **Identification:** У короткому тексті (напр., біографії) знайти речення, що описують, де людина навчалася і ким працювала.
*   **Simple Q&A:** `Це вчитель?` - `Так, це вчитель.` / `Ні, це лікарка.`.

**Phase 2: Контрольована практика**
*   **Fill-in-the-blanks (Орудний відмінок):** Дати речення типу `Мій брат — ... (студент). Він хоче бути ... (програміст).` Учень має поставити слова в правильну форму (`студент`, `програмістом`).
*   **Transformation Drills:** Перетворити речення. `Вона — лікар.` → `Вона працює лікарем.` `Він вчиться в університеті.` → `Він — студент.`
*   **Preposition Choice (`в/на`):** Дати список місць (школа, університет, факультет, завод, пошта) і попросити скласти речення з `працювати/навчатися`, вибравши правильний прийменник.

**Phase 3: Вільне мовлення**
*   **Role-Play:** Розіграти діалог за ситуацією Б зі (Source 15): "Ви зустріли свого сусіда, який вступив до університету. Розпитайте в нього про його майбутню спеціальність, повідомте про свої плани на майбутнє, обґрунтуйте свій вибір."
*   **Mini-CV:** Заповнити спрощену форму резюме або автобіографії за зразком (Source 10, 39), вказавши вигадану школу, університет та майбутню професію.
*   **Presentation:** Підготувати коротку розповідь (2-3 хвилини) про члена родини: де він/вона навчався/навчалася, ким працює.

## Зв'язки з іншими темами (Connections)

*   **Передумови (Prerequisites):**
    *   **Називний відмінок:** Учень вже повинен знати назви професій у словниковій формі.
    *   **Дієслова:** Знати відмінювання базових дієслів `бути`, `працювати`, `хотіти`, `жити`.
    *   **Роди іменників:** Вміти розрізняти чоловічий, жіночий та середній рід для правильного узгодження.

*   **Наступні кроки (What this enables):**
    *   **Вищий рівень (В1):** Складання повноцінних автобіографій та резюме, проходження співбесід, обговорення кар'єрних планів та освітніх шляхів у деталях.
    *   **Складніші відмінки:** Ця тема є природним входом до вивчення функцій **родового відмінка** (назви факультетів, спеціальностей) та **давального відмінка** (напр., `мені подобається моя робота`).
    *   **Дієслова руху:** Поєднання теми роботи з дієсловами руху (`їздити на роботу`, `ходити в університет`).
    *   **Минулий час:** Розповідь про минулий досвід роботи чи навчання (`Я працював...`, `Я навчалася...`).
