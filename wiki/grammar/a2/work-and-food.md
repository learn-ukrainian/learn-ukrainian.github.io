## Як це пояснюють у школі

В українській шкільній програмі граматичні теми, пов'язані з професіями та кулінарією, не виділяються в окремий лексичний блок, як це часто буває в комунікативних курсах іноземних мов. Натомість ця лексика органічно інтегрована у вивчення морфології, зокрема іменника та його відмінків [S10]. 

Підхід українських підручників ґрунтується на поступовому введенні відмінкових закінчень з опорою на фонетичні закономірності. Наприклад, орудний відмінок для опису професій та інструментів починають системно вивчати у 4 класі. Учням пропонують аналізувати закінчення `-ом`, `-ем`, `-єм` на прикладі слів на кшталт *ветеринар*, *лікар*, *господар* [S1]. Підручники застосовують покроковий алгоритмічний підхід: «Крок 1. Якого роду іменники... Крок 2. На який приголосний закінчується основа... Крок 3. Які закінчення... Крок 4. Зроби висновок» [S1]. Це формує в учнів глибоке аналітичне розуміння того, що вибір закінчення безпосередньо залежить від кінцевого звука основи (твердого, м'якого чи шиплячого).

Згодом, у 6 класі, парадигма розширюється за рахунок іменників ІІІ відміни (наприклад, *відповідь*, *якість*, *ніч*), де запроваджується фундаментальне фонетичне правило — подовження приголосних у формі орудного відмінка однини (*ніччю*, *подорожжю*) [S3, S4]. У 9 класі увага приділяється складнішим орфограмам, таким як розрізнення голосних `-о-` та `-е-` в закінченнях іменників чоловічого роду (наприклад, *знахідка*, *бар'єр*, *лікар*, *окріп*, *зброяр*) [S2]. Крім того, вивчається правопис складних слів, що позначають сучасні багатокомпонентні професії чи поняття: *лікар-еколог*, *бізнес-план*, *вебсайт* [S6].

Тема кулінарії та їжі у шкільному курсі найчастіше слугує практичним контекстом для відпрацювання родового відмінка [S10]. Особлива увага приділяється правилам вибору закінчень `-а/-я` чи `-у/-ю` в родовому відмінку однини іменників чоловічого роду ІІ відміни, що традиційно є однією з найскладніших тем. Наприклад, іменники, що означають речовину, масу, матеріал (інгредієнти), набувають закінчення `-у/-ю` (*цукру*, *борщу*, *чаю*), тоді як злічувані предмети чи плоди — `-а/-я` (*помідора*, *хліба*) [S10]. Цей самий поділ активно використовується в ресурсах для вивчення української мови як іноземної, наприклад, в епізодах про продуктові магазини та гастрономічні фестивалі [S7, S8, S9].

## Повна парадигма

### Орудний відмінок іменників (Instrumental Case)
Орудний відмінок є ключовим для вираження професії (конструкція «бути + іменник в орудному відмінку») та інструмента дії [S10, S1, S4].

#### І відміна (жіночий, чоловічий рід на -а/-я)
Закономірність: вибір закінчення залежить від групи основи (тверда, м'яка, мішана).

| Група | Ланцюжок творення | Приклади |
|---|---|---|
| Тверда | `{називний} → {основа на твердий} → {+ою} → {орудний}` | мама → мам- → +ою → **мамою** [S4]<br>хата → хат- → +ою → **хатою** [S4] |
| М'яка | `{називний} → {основа на м'який} → {+ею/єю} → {орудний}` | лілія → лілі- → +єю → **лілією** [S4]<br>пісня → пісн- → +ею → **піснею** [S4] |
| Мішана | `{називний} → {основа на шиплячий} → {+ею} → {орудний}` | груша → груш- → +ею → **грушею** [S4, S2]<br>площа → площ- → +ею → **площею** [S2] |

#### ІІ відміна (чоловічий рід на приголосний, середній рід)
Закономірність: іменники чоловічого роду утворюють орудний відмінок за допомогою закінчень `-ом`, `-ем`, `-єм` [S1]. Особливу увагу слід звертати на іменники чоловічого роду на `-р`, які історично поділяються на тверду, м'яку та мішану групи [S1].

| Група | Ланцюжок творення | Приклади |
|---|---|---|
| Тверда | `{називний} → {основа на твердий} → {+ом} → {орудний}` | ветеринар → ветеринар- → +ом → **ветеринаром** [S1]<br>шпак → шпак- → +ом → **шпаком** [S1]<br>професор → професор- → +ом → **професором** [S2] |
| М'яка | `{називний} → {основа на м'який} → {+ем/єм} → {орудний}` | лікар → лікар- → +ем → **лікарем** [S1]<br>обрій → обрі- → +єм → **обрієм** [S1]<br>зброяр → зброяр- → +ем → **зброярем** [S2] |
| Мішана | `{називний} → {основа на шиплячий} → {+ем} → {орудний}` | гараж → гараж- → +ем → **гаражем** [S1]<br>оповідач → оповідач- → +ем → **оповідачем** [S2]<br>борщ → борщ- → +ем → **борщем** [S1] |

*Обмеження:* Іменники чоловічого роду з основою на `-р` (ветеринар, лікар, зброяр, майстер [S2]) не мають єдиного правила визначення групи лише за називним відмінком. Учням слід запам'ятовувати форму родового та орудного відмінків для кожного такого слова індивідуально. Наприклад: *майстер → майстром* [S2] (тверда група), але *зброяр → зброярем* [S2] (м'яка група).

#### ІІІ відміна (жіночий рід на приголосний)
Закономірність: відбувається фонетичне подовження кінцевого приголосного основи, якщо він стоїть між двома голосними звуками [S3].

| Умова | Ланцюжок творення | Приклади |
|---|---|---|
| Один приголосний у кінці основи | `{називний} → {основа} → {подовження приголосного + ю} → {орудний}` | ніч → ніч- → чч + ю → **ніччю** [S3]<br>осінь → осін- → нн + ю → **осінню** [S3]<br>подорож → подорож- → жж + ю → **подорожжю** [S4] |
| Збіг приголосних у кінці основи | `{називний} → {основа} → {без подовження + ю} → {орудний}` | якість → якіст- → +ю → **якістю** [S3]<br>честь → чест- → +ю → **честю** [S3, S4]<br>молодість → молодіст- → +ю → **молодістю** `<!-- VERIFY -->` |
| Основа на губний (б, п, в, м, ф) або р | `{називний} → {основа} → {апостроф + ю} → {орудний}` | верф → верф- → +'ю → **верф'ю** [S3]<br>кров → кров- → +'ю → **кров'ю** [S4]<br>кіновар → кіновар- → +'ю → **кіновар'ю** [S3] |

### Родовий відмінок: Кулінарія та Інгредієнти (Genitive Case)
Для вираження кількості, відсутності та партитивності (частини від цілого) використовується родовий відмінок [S10]. Найбільшу складність для іноземців становить вибір закінчення `-а/-я` чи `-у/-ю` для іменників чоловічого роду ІІ відміни [S10].

| Категорія значення | Закінчення | Приклади (Кулінарія та їжа) |
|---|---|---|
| Речовини, маси, рідини, матеріали (незлічувані) | **-у / -ю** | борщ → **борщу** [S1]<br>цукор → **цукру** [S10]<br>чай → **чаю** [S10]<br>окріп → **окропу** [S2] |
| Злічувані предмети, чітко окреслені плоди, одиниці виміру | **-а / -я** | хліб → **хліба** [S10]<br>помідор → **помідора** [S10]<br>гарбуз → **гарбуза** `<!-- VERIFY -->` |

*Обмеження:* Правило семантичного вибору `-а/-я` чи `-у/-ю` стосується ВИКЛЮЧНО іменників чоловічого роду ІІ відміни в однині. Іменники жіночого та середнього роду мають свої чіткі закінчення в родовому відмінку (наприклад, вода → води, молоко → молока) і не підпадають під цю градацію [S10].

## Частотність і пріоритети

Для рівня А2 не всі аспекти цих відмінків є однаково пріоритетними:
1. **Топ-пріоритет:** Конструкція `Я (працюю) + професія в орудному відмінку` (наприклад, *Я працюю лікарем* [S1, S10]). Вона використовується в щоденній побутовій комунікації. Займенники в орудному відмінку (*зі мною*, *з тобою*) [S5] є абсолютно необхідними для базової соціалізації.
2. **Високий пріоритет:** Партитивний родовий відмінок для незлічуваних продуктів (*додати цукру*, *склянка молока*, *без капусти*) [S10]. Це структурна основа для орієнтування у продуктовому магазині [S8] та на гастрономічних фестивалях [S7].
3. **Середній пріоритет:** Подовження приголосних в іменниках ІІІ відміни (наприклад, *сіллю*, *ніччю*) [S3]. Хоча це важлива орфоепічна риса, що демонструє милозвучність української мови, кількість частотних кулінарних чи професійних термінів ІІІ відміни на рівні А2 обмежена.
4. **Низький пріоритет:** Складні орфографічні випадки розмежування закінчень орудного відмінка для рідкісних професій з основою на `-р` (наприклад, *зброярем* [S2] чи *оповідачем* [S2]). Їх доцільно залишити на пасивне засвоєння, натомість активно відпрацьовуючи високочастотні *лікар*, *вчитель*, *менеджер*, *кухар*.

## Деколонізаційні застереження

Авторам модулів необхідно категорично уникати лексичних та граматичних русизмів, суржику та кальок, які штучно вкоренилися в мовлення через тривалу русифікацію. У контексті професій та кулінарії це питання стоїть особливо гостро:

1. **Термінологія професій:**
   Слід використовувати автентичні українські назви:
   - ❌ *повар* → ✅ **кухар** `<!-- VERIFY -->`
   - ❌ *строітель* → ✅ **будівельник** `<!-- VERIFY -->`
   - ❌ *парікмахер* → ✅ **перукар** `<!-- VERIFY -->`
   
2. **Вираження фаху:**
   Російська синтаксична модель *«он по профессии врач»* часто неправомірно калькується як ❌ *«він по професії лікар»*. Природна і літературна українська модель звучить як ✅ **«він за фахом лікар»**, ✅ **«він за професією лікар»** або просто безособово ✅ **«він лікар»**.

3. **Кулінарна лексика (Суржик):**
   Багато англомовних студентів, які раніше контактували з російською, можуть вживати суржик. Його треба активно виправляти:
   - ❌ *первое блюдо* → ✅ **перша страва** `<!-- VERIFY -->` (в українській мові слово «блюдо» означає велику посудину, таріль, а не саму їжу).
   - ❌ *жарена картошка* → ✅ **смажена картопля** `<!-- VERIFY -->`.
   - ❌ *сахар* → ✅ **цукор** [S10].
   - ❌ *столова* → ✅ **їдальня** `<!-- VERIFY -->`.

4. **Частковий родовий відмінок (Partitive):**
   У російській мові часто використовується специфічний розмовний партітив із зменшувально-пестливими суфіксами (наприклад, *чайку*, *сахарку*). В українській літературній нормі використовується чистий родовий відмінок речовини: **«додати цукру»** [S10], **«випити чаю»** [S10]. Хоча форми на кшталт *чайку* існують у розмовній українській, для іноземців на рівні А2 слід твердо фіксувати літературну норму без демінутивів.

5. **Граматичні кальки:**
   Абсолютно заборонено використовувати конструкцію ❌ *являти собою* (пряма калька з російського *представлять собой*). В українській мові слід вживати дієслово **є** або інші синонімічні конструкції (наприклад, *Борщ — це страва, яка...*). 
   Також заборонено використовувати ❌ *приймати участь* (наприклад, у гастрономічному фестивалі [S7]). Єдиний правильний український відповідник — ✅ **брати участь**.

Для успішного й глибокого засвоєння тем орудного та родового відмінків (у контексті професій та кулінарії) авторам модулів слід застосовувати таку методичну послідовність:

*   **Фаза 1: Рецептивна ідентифікація (Notice).**
    Розпочніть із введення текстів або автентичних діалогів (наприклад, ситуація вибору покупок у продуктовому магазині [S8] або інтерв'ю про вибір професії). Учні мають знайти і підкреслити всі іменники, що позначають професії або інгредієнти їжі. Після цього запропонуйте завдання на класифікацію закінчень: наприклад, виписати всі іменники орудного відмінка з закінченнями `-ом` та `-ем` у дві окремі колонки [S2]. Це активує аналітичну увагу до фонетичних особливостей основи.

*   **Фаза 2: Керована трансформація (Controlled Practice).**
    *   *Вправа на цілеспрямовану трансформацію:* Надайте речення, де іменник стоїть у називному відмінку в дужках. Завдання учня — поставити слово в правильну форму орудного або родового відмінка з огляду на контекст. Наприклад: `Він працює (лікар) [S1]` → `Він працює лікарем`; `Улюблена пісня (Оксана) [S4]` → `Улюблена пісня Оксани`.
    *   *Вправа на семантичний вибір (Multiple Choice):* Запропонуйте вибір між родовим і знахідним або називним відмінками у кулінарному контексті. Наприклад: `Дайте мені, будь ласка, (чай / чаю) [S10]`.
    *   *Вправа на синтагматичне поєднання:* Завдання поєднати одиницю міри та відповідну речовину (наприклад, `склянка` + `молоко` → `склянка молока` [S10]).

*   **Фаза 3: Спонтанне продукування (Free Production).**
    *   *Комунікативна рольова гра:* «На гастрономічному фестивалі» [S7] або «У продуктовому магазині» [S8]. Учень має скласти список покупок або замовлення і озвучити його партнеру, правильно використовуючи партитивний родовий відмінок для незлічуваних продуктів (*Дайте мені трохи хліба, багато сиру і склянку води*).
    *   *Презентаційний проєкт:* «Професії моєї родини». Учень має скласти усну розповідь, використовуючи базові конструкції *Я працюю...*, *Мій тато працює...*, *Я хочу бути...*, обов'язково і свідомо застосовуючи орудний відмінок [S1]. На цьому етапі слід заохочувати використання складних слів для позначення сучасних реалій (наприклад, *вебдизайнер* `<!-- VERIFY -->`, *бізнес-аналітик* `<!-- VERIFY -->`, утворених за моделлю *лікар-еколог* [S6]).

## Зв'язки з іншими темами

*   **Що було раніше (Передумови):** Для успішного засвоєння орудного відмінка учні вже повинні автоматично розпізнавати категорію роду іменників (чоловічий, жіночий, середній) та впевнено розрізняти тверді, м'які й шиплячі приголосні в кінці основи слова. Це критично важливо для ухвалення рішення про вибір між закінченнями `-ом` та `-ем` [S1]. Крім того, вони повинні володіти парадигмою відмінювання особових займенників у родовому та знахідному відмінках (наприклад, *мене*, *тебе* [S5]), щоб без стресу перейти до їхніх форм в орудному відмінку.
*   **Що буде далі (Перспективи):** Засвоєння орудного відмінка як інструмента дії (наприклад, конструкції на кшталт *писати мечем, працювати плугом* [S4]) відкриває шлях до розуміння і конструювання складних пасивних речень на рівні В1 та В2, де реальний суб'єкт дії граматично виражається саме орудним відмінком (наприклад, *книга написана відомим письменником*). Вивчення ж партитивного родового відмінка (особливо для інгредієнтів [S10]) створює міцне підґрунтя для подальшого опанування складніших семантичних функцій родового відмінка, таких як родовий часу (наприклад, *п'ятого жовтня*) та родовий причини.
