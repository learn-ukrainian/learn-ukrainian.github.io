## Як це пояснюють у школі (How Schools Teach This)

Українські шкільні програми вводять теми професій та кулінарії поступово, пов'язуючи їх із базовими граматичними концепціями.

**1. Назви професій та словотворення (Grades 2-3):**
У початкових класах учні знайомляться з назвами поширених професій. У підручнику Большакової [S41] для 2 класу наводяться такі приклади: `таксист`, `учителька`, `лікар`, `будівельник`, `кухар`, `кондитер`. У цей же період закладаються основи словотворення. Наприклад, підручник для 2-го класу [S42] показує зв'язок між іменником та похідним прикметником (`овочі` → `овочевий`, `картопля` → `картопляне`, `сир` → `сирна`), що є ключовим для опису страв. Також обговорюється спільний корінь `вар-` у словах `вареник` та `варення` [S42], що пояснює їхнє походження від дієслова "варити".

**2. Рецепти як практичне застосування граматики (Grades 3-7):**
Рецепти використовуються як практичний інструмент для вивчення кількох граматичних тем:
*   **Числівники та Родовий відмінок:** У 3 класі учні вже стикаються з рецептами, де кількість інгредієнтів вказана числівниками (`один банан`, `два яблука`, `три груші`, `п’ять столових ложок сметани`) [S43]. Це створює природний контекст для подальшого вивчення Родового відмінка для позначення кількості, як це детально пояснено в подкасті ULP [S44] на прикладі рецепту сирників (`500 грамів сиру`, `дві ложки цукру`).
*   **Наказовий спосіб дієслів:** Інструкції в рецептах є ідеальним прикладом наказового способу. У підручниках з розвитку мовлення [S45][S46] та автентичних матеріалах [S47] учні бачать дієслова у формі `ви` (`візьміть`, `додайте`, `перемішайте`).
*   **Орудний відмінок з прийменником `з`:** Для опису складу страв використовується Орудний відмінок, як показано у підручнику Заболотного (5 клас): `вареники з вишнями`, `вареники із сиром`, `вареники зі сметаною` [S41].

**3. Професії та соціальні ролі (Grades 5-9):**
У старших класах тема професій розширюється. Учні вчаться не тільки називати професії, але й описувати діяльність, як-от `технолог із приготування їжі` [S48] або `шеф-кухар` [S49]. Історичний контекст [S50] показує розподіл обов'язків (`робота на полі`, `пряла, ткала, шила сорочки`), вводячи більш специфічну лексику.

Отже, шкільний підхід є інтегрованим: лексика професій та кулінарії слугує практичною основою для вивчення словотворення, відмінків (Родовий, Орудний) та дієслівних форм (наказовий спосіб).

## Повна парадигма (Full Paradigm)

### 1. Назви професій (Nouns for Professions)

Більшість назв професій в українській мові — іменники чоловічого роду. Жіночі відповідники часто утворюються за допомогою суфікса **-к(а)**.

**Чоловічий рід (Masculine)**
| Суфікс | Приклад | Інші приклади |
| :--- | :--- | :--- |
| **-ар/-яр** | `ку́хар` | `лі́кар`, `сто́ляр` |
| **-ник** | `будіве́льник` | `письме́нник`, `керівни́к` |
| **-ець** | `продаве́ць` | `плаве́ць` |
| **-ист** | `журналі́ст` | `такси́ст`, `економі́ст` |
| **-ер** | `інжене́р` | `конди́тер`, `дизайнер` |
| **-ор** | `дире́ктор` | `істо́рик`, `а́ктор` |
| **(без суфікса)** | `воді́й` | `суддя́` |

**Жіночий рід (Feminine)**
| Форма (Чол.) | Форма (Жін.) | Примітка |
| :--- | :--- | :--- |
| `вчи́тель` | `вчи́телька` | Дуже поширений суфікс **-ка**. |
| `письме́нник` | `письме́нниця` | Суфікс **-иця** для слів на **-ник**. |
| `студе́нт` | `студе́нтка` | |
| `журналі́ст` | `журналі́стка` | |
| `ку́хар` | `ку́харка` | |
| `лі́кар` | `лі́карка` | |
| `дире́ктор` | `дире́кторка` | Сучасна норма, стає дедалі поширенішою. |

**Важливо:** У формальному та офіційному мовленні для позначення посади жінки може вживатися іменник чоловічого роду (`пані директор`, `вона — професійний історик`). Однак у розмовному мовленні та сучасній публіцистиці фемінітиви є нормою.

### 2. Родовий відмінок однини для кількості (Genitive Singular for Quantity)

Використовується для незлічуваних іменників (рідини, сипучі продукти) після слів, що позначають міру (`склянка`, `ложка`, `кілограм`).

| Рід | Закінчення | Приклади з рецептів |
| :--- | :--- | :--- |
| **Чоловічий** | **-у / -ю** | `кілограм цу́кр**у**`, `склянка со́к**у**`, `ложка ме́д**у**` |
| **Середній** | **-а / -я** | `кілограм ма́сл**а**`, `ложка боро́шн**а**`, `трохи молок**а́**` |
| **Жіночий** | **-и / -і** | `дрібка со́л**і**`, `склянка вод**и́**` |

### 3. Числівники + Родовий відмінок множини (Numerals + Genitive Plural)

Це ключова структура для рецептів зі злічуваними інгредієнтами. Правило залежить від числівника.

| Числівник | Форма іменника | Приклад (інгредієнти) |
| :--- | :--- | :--- |
| `оди́н`, `одна́`, `одне́` | Називний відм. однини | `оди́н огіро́к`, `одна́ цибу́ля` |
| `два`, `три`, `чоти́ри` | Називний відм. множини | `два помідо́р**и**`, `три я́блук**а**`, `чотири яйця́` |
| `п'ять` – `два́дцять` (і всі, що закінчуються на 5-9, 0) | **Родовий відмінок множини** | `п'ять помідо́р**ів**`, `шість я́блук`, `десять яє́ць` |

**Парадигма Родового множини (Genitive Plural)**
| Рід | Закінчення | Приклад |
| :--- | :--- | :--- |
| Чоловічий | **-ів / -їв** | `п'ять огірк**і́в**`, `шість помідо́р**ів**` |
| Середній | (нульове), **-ів** | `п'ять я́блук`, `десять яє́ць` (нерегулярна форма) |
| Жіночий | (нульове), **-ей** | `п'ять груш`, `шість ви́шень` |

### 4. Наказовий спосіб дієслів (Imperative Mood: `ви` form)

Використовується для інструкцій.

| Правило утворення (`ти` → `ви`) | Основа (`ти`) | Форма `ви` | Приклад з рецепту |
| :--- | :--- | :--- | :--- |
| Основа закінчується на **-й** | `ми́й`, `ріж` (чергування `ж`→`з`) | `ми́йте`, `рі́жте` | `Поми́йте та порі́жте кубиками яблука.` [S43] |
| Основа закінчується на **голосний** | `готу́й`, `міша́й`, `чита́й` | `готу́йте`, `міша́йте`, `чита́йте` | `потім лийте воду та вимішуйте` [S47] |
| Основа закінчується на **приголосний** | `печ-и́` | `печі́ть` | `...печуть пироги, торти, тістечка` (форма теперішнього часу, але корінь той самий) [S48] |
| Нерегулярні / з чергуванням | `дода́й`, `ві́зьм-и`, `поклади́` | `дода́йте`, `візьмі́ть`, `покладі́ть` | `Дода́йте ще трохи води` [S47] |

## Частотність і пріоритети (Frequency & Priorities)

1.  **Назви професій:** Найчастотнішими є базові професії: `вчи́тель/вчи́телька`, `лі́кар/лі́карка`, `продаве́ць`, `воді́й`, `ку́хар`, `інжене́р`. Їх слід засвоїти в першу чергу. Менш поширені, як-от `сто́ляр` чи `істо́рик`, можуть вивчатися на вищих рівнях.
2.  **Genitive of Quantity:** Це найважливіша граматична конструкція в кулінарному контексті. Пріоритети:
    *   **Must-know:** `склянка води/молока`, `ложка цукру/солі`, `кілограм борошна/м'яса`. Це основи будь-якого рецепту.
    *   **Should-know:** Конструкція з числівниками. `два яблука`, `три яйця` (називний множини) та `п'ять помідорів` (родовий множини) є абсолютно необхідними. Правило "2-4 vs 5+" має бути засвоєне на рівні A2.
3.  **Imperative Verbs:** Форма на `ви` (`-йте`, `-іть`) є пріоритетною для розуміння та відтворення інструкцій. Найчастотніші дієслова: `візьміть`, `додайте`, `поріжте`, `змішайте`, `варіть`, `смажте`, `печіть`.
4.  **Instrumental with `з`:** Конструкція `страва + з + інгредієнт` (в орудному відмінку) є дуже частотною: `салат з овочів`, `суп з грибами`, `вареники з картоплею`.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я — повар.` | `Я — ку́хар.` | Пряма калька з російської (`повар`). Українською мовою `ку́хар`. Це також стосується `жарити` (рос.) замість `сма́жити` (укр.). [S48] |
| `Дайте мені два кілограми картоплів.` | `Дайте мені два кілограми карто́плі.` | Після числівників `два`, `три`, `чотири` вживається називний відмінок множини (`дві картоплини`), але після слів `кілограм`, `літр` вживається родовий відмінок. `Картопля` — збірний іменник, тому використовується родовий однини. |
| `Рецепт потребує п'ять яйця.` | `Рецепт потребує п'ять яє́ць.` | Після числівника `п'ять` (і вище) іменник має стояти в родовому відмінку множини. [S44] |
| `Добавте сіль.` | `Дода́йте сіль.` | Неправильне утворення наказового способу через вплив російської (`добавьте`). Правильна українська форма — `додайте`. [S47] |
| `Я люблю вареники з сиром (творогом).` | `Я люблю вареники з си́ром.` | В українській мові немає розрізнення `сир` (твердий) і `творог` (кисломолочний). Обидва поняття називаються словом `сир`. Уточнити можна як `твердий сир` або `домашній сир` / `кисломолочний сир`. [S48] |
| `Приємного апетиту!` | `Смачно́го!` | `Приємного апетиту` є калькою з російської та французької. Автентичне українське побажання — `Смачного!`. [S48] |

## Деколонізаційні застереження (Decolonization Notes)

**MANDATORY:** Українська кулінарна та професійна лексика має суттєві відмінності від російської. Ігнорування цих відмінностей веде до суржику та зміцнення колоніальних стереотипів.

1.  **Лексика професій:** Не вживайте російські назви.
    *   `повар` → **`ку́хар`**
    *   `уборщиця` → **`прибира́льниця`**
    *   `парикмахер` → **`перука́р`**

2.  **Кулінарна лексика:** Уникайте поширених русизмів.
    *   `творог` → **`сир`** (або `домашній сир`, `кисломолочний сир` для уточнення) [S48].
    *   `жарити` → **`сма́жити`** [S48].
    *   `тушене м'ясо` → **`тушко́ване м'ясо`** [S48].
    *   `яйце всмятку` → **`рідке́/некруте́ яйце́`** [S48].
    *   `блюдо` (страва) → **`стра́ва`**. Слово `блюдо` в українській мові означає велику тарілку. [S51].
    *   `масло` (про соняшникову) → **`олі́я`**. `Ма́сло` — це вершкове масло (butter). [S51][S52].

3.  **Етикет:**
    *   `Приємного апетиту!` → **`Смачно́го!`** [S48].

4.  **Граматика:** Не застосовуйте російські граматичні моделі. Наприклад, утворення наказового способу: рос. `добавьте` → укр. **`дода́йте`**.

Представлення української мови як самостійної системи, а не як "варіанту російської з деякими відмінностями" є ключовим принципом деколонізованої педагогіки.

## Природні приклади (Natural Examples)

**Група 1: Назви професій**
1.  `Мене звати акім галімов І це реальна історія друзі людина яку не потрібно представляти... дуже крутий неймовірний **шеф-кухар** Євген Клопотенко.` [S49]
2.  `Пані Олена дослідниця гастрономічної культури України і засновниця проекту їжакультура і **історикиня**.` [S49]
3.  `Я проживаю зараз в торонті в Канаді і ... я працюю в середній школі **викладаю** французьку мову вже 10 років.` (Імпліцитно - `вчителька` або `викладачка`) [S53]
4.  `Дідусь із лимонів зробив лимонад. А ми разом із сестричкою Аліною мили фрукти.` (Не професія, але опис родинних обов'язків) [S42]

**Група 2: Родовий відмінок для кількості**
5.  `На 500 **грамів сиру** я беру два **яйця**, дві столові **ложки цукру**, дві-три столові **ложки борошна**.` [S44]
6.  `Для його приготування тобі потрібно один стиглий **банан**, два **яблука**, три **груші**, п’ять столових **ложок сметани**, чотири чайні **ложки цукру**.` [S43]
7.  `Візьміть трохи **борошна**, приблизно одну **склянку**.` [S47]

**Група 3: Наказовий спосіб (`ви` форма)**
8.  `**Візьміть** одну склянку борошна, **висипайте** борошно в миску та **підсоліть**, потім **лийте** воду та **вимішуйте**.` [S47]
9.  `**Спробуй** так, тобі теж сподобається.` (форма на `ти`, але показує контекст поради) [S54]
10. `**Май на увазі**, коли приїду додому, буду тебе вчити.` (форма на `ти`) [S54]

**Група 4: Орудний відмінок з інгредієнтами**
11. `...приготувала вареники **з картоплею і сиром**.` [S54]
12. `Вареники **з вишнями**, вареники **із сиром**, вареники **зі сметаною**.` [S41]
13. `Одеська версія плову **з мідіями**...` [S55]

## Рекомендації для вправ (Activity Concepts)

*   **Phase 1 (Recognition & Basic Forms):**
    *   **Matching:** З'єднайте професію (`кухар`, `лікар`, `вчитель`) з місцем роботи (`ресторан`, `лікарня`, `школа`).
    *   **Gap-fill (Genitive Singular):** `Дайте, будь ласка, склянку \_\_\_\_\_ (вода), кілограм \_\_\_\_\_ (цукор) і пляшку \_\_\_\_\_ (олія).`
    *   **Multiple Choice:** `Вареники з (картопля / картоплею / картоплі).`

*   **Phase 2 (Structured Production):**
    *   **Transformation Drill:** Перетворіть речення з `2-4` на `5+`. Зразок: `Я купив два помідори.` → `Я купив п'ять помідорів.`
    *   **Recipe Writing (Guided):** Надайте список інгредієнтів (`2 яйця, 200 г борошна, 1 склянка молока, 50 г цукру`) та список дієслів у формі інфінітиву (`змішати`, `додати`, `пекти`). Завдання: написати 3-4 речення інструкцій у наказовому способі (`Змішайте...`, `Додайте...`).
    *   **Role-play script:** Написати короткий діалог "У магазині", використовуючи лексику продуктів та конструкції кількості.

*   **Phase 3 (Free Production):**
    *   **My Favorite Recipe:** Написати простий рецепт улюбленої страви, використовуючи наказовий спосіб, родовий відмінок для кількості та орудний для інгредієнтів.
    *   **My Family's Professions:** Написати короткий текст (3-5 речень) про те, ким працюють члени родини (`Мій тато — інженер. Моя мама — лікарка...`).
    *   **Problem Solving:** Дати "поганий" рецепт, повний русизмів (`жарити`, `творог`, `добавте`). Завдання: знайти та виправити помилки.

## Зв'язки з іншими темами

*   **Попередні теми (Prerequisites):**
    *   **A1 Noun Genders & Cases:** Розуміння роду іменників є необхідним для правильного утворення прикметників (`овочевий салат`) та відмінкових закінчень.
    *   **A1 Present Tense Conjugation:** Основа для утворення наказового способу.
*   **Наступні теми (What this enables):**
    *   **B1 Verbs of Motion:** Тема "Кулінарія" природно веде до обговорення покупок (`ходити на ринок`, `їхати в супермаркет`).
    *   **B1 Aspect (Perfective/Imperfective):** Рецепти — ідеальний контекст для введення видових пар: `різати/порізати`, `варити/зварити`, `мішати/змішати`.
    *   **B1 Adjectival Participles:** Для опису страв: `смажена картопля`, `варені яйця`, `тушковані овочі`.
