## Як це пояснюють у школі

В українській шкільній програмі граматичні теми, пов'язані з професіями та кулінарією, не виділяються в окремий блок, а інтегровані у вивчення морфології, зокрема іменника та його відмінків.

Ключовим для вираження професії є **орудний відмінок** іменника. Ця концепція закріплюється через численні вправи на трансформацію та побудову речень. Наприклад, у підручниках для 6-9 класів учням пропонують поставити іменники в орудному відмінку або відредагувати речення, де цей відмінок вжито неправильно [S1, S2]. Конструкція «бути + іменник в орудному відмінку» (напр., *бути лікарем*) є центральною.

Тема кулінарії слугує практичним контекстом для відпрацювання **родового відмінка**, особливо при переліку інгредієнтів (напр., *додати цукру, склянка молока*) або для вираження відсутності чогось (*борщ без капусти*) [S1]. Особлива увага приділяється закінченням `-а/-я` та `-у/-ю` в родовому відмінку однини іменників II відміни, що є складною темою для учнів [S1].

Також активно використовуються прислівники для опису процесів, як-от приготування їжі (`ретельно розтираючи`, `поступово додаючи`) [S3]. Загалом, підхід полягає у вивченні граматичних форм через практичне застосування в знайомих життєвих сценаріях.

## Повна парадигма

На рівні А2 у темах професій та кулінарії центральними є два відмінки: **орудний** (для позначення ролі, професії) та **родовий** (для інгредієнтів, кількості, відсутності).

### 1. Професії: Орудний відмінок (Instrumental Case)

Основна конструкція: `[хтось] + є/працює + [професія в орудному відмінку]`.

**Чоловічий рід (II відміна)**

| Називний (Хто?) | Орудний (Ким?) | Приклад речення |
| :--- | :--- | :--- |
| директор | директор**ом** | Він працює директором. |
| лікар | лікар**ем** | Мій брат хоче стати лікарем. [S1] |
| кухар | кухар**ем** | Смачну страву приготував кухар. |
| менеджер | менеджер**ом** | Вона мріє бути топменеджером. |
| програміст | програміст**ом** | Тарас працює програмістом у великій компанії. |
| учитель | учител**ем** | Пан Степан є нашим учителем математики. |
| водій | воді**єм** | Мій сусід працює водієм таксі. |

**Правило творення:**
- Іменники на твердий приголосний: `{основа} + -ом`. (`директор → директором`)
- Іменники на м'який або шиплячий приголосний: `{основа} + -ем (-єм)`. (`учитель → учителем`, `водій → водієм`)

**Жіночий рід (І відміна)**

| Називний (Хто?) | Орудний (Ким?) | Приклад речення |
| :--- | :--- | :--- |
| директорка | директорк**ою** | Пані Ірина є директоркою нашої школи. |
| лікарка | лікарк**ою** | Моя сестра працює лікаркою в дитячій лікарні. |
| вчителька | вчительк**ою** | Вона була моєю першою вчителькою. |
| менеджерка | менеджерк**ою** | Анна стала успішною менеджеркою проєктів. |
| програмістка | програмістк**ою** | Ольга працює програмісткою. |
| листоноша | листонош**ею** | Жінка працювала листоношею. [S1] |

**Правило творення:**
- Іменники на `-а`: `{основа} + -ою`. (`директорка → директоркою`)
- Іменники на `-я`: `{основа} + -ею (-єю)`. (`листоноша → листоношею`)

### 2. Кулінарія: Родовий відмінок (Genitive Case)

Використовується для позначення інгредієнтів, міри, відсутності.

**Чоловічий рід (II відміна) — закінчення `-у/-ю` та `-а/-я`**

Це одна з найскладніших тем в українській граматиці. На рівні А2 важливо запам'ятати найчастотніші слова.

**Закінчення `-у / -ю` (речовини, збірні поняття, абстракції):**

| Називний (Що?) | Родовий (Чого?) | Приклад речення |
| :--- | :--- | :--- |
| борщ | борщ**у** | Я хочу тарілку гарячого борщу. |
| сир | сир**у** | Для піци потрібно багато сиру. |
| цукор | цурк**у** | Поклади до чаю менше цукру. [S3] |
| мед | мед**у** | Твоїми б устами та мед пити. [S2] |
| хліб | хліб**а** | *Виняток.* Купи шматок чорного хліба. |
| горох | горох**у** | Не можна ні килим підняти, ні гороху зібрати. [S1] |
| вогонь | вогн**ю** | Варив кухар без вогню. [S1] |

**Закінчення `-а / -я` (конкретні предмети, плоди, овочі):**

| Називний (Що?) | Родовий (Чого?) | Приклад речення |
| :--- | :--- | :--- |
| помідор | помідор**а** | Для салату потрібно три помідора. |
| огірок | огірк**а** | У холодильнику не залишилось жодного огірка. |
| апельсин | апельсин**а** | Сік одного апельсина. |
| лимон | лимон**а** | Скибочка лимона. |

**Жіночий та середній рід (простіші правила):**

| Називний (Що?) | Родовий (Чого?) | Приклад речення |
| :--- | :--- | :--- |
| капуста | капуст**и** | Салат з капусти. |
| картопля | картопл**і** | Кілограм картоплі. |
| сметана | сметан**и** | Подаємо зі сметаною. [S3] |
| сіль | сол**і** | Додайте дрібку солі. |
| вода | вод**и** | Пляшка води. |
| молоко | молок**а** | Пакет молока. [S3] |
| масло | масл**а** | Пачка вершкового масла. [S3] |
| м'ясо | м'яс**а** | Шматок м'яса. |
| тісто | тіст**а** | Готуємо з тіста. [S3] |
| яйце | яйц**я** | Розбиваємо туди яйце. [S3] |

## Частотність і пріоритети

Для рівня А2 пріоритетним є засвоєння найуживаніших конструкцій:

1.  **Професії:** Конструкція `Я працюю + [професія в Орудному відмінку однини]`. Студенти повинні впевнено називати свою професію та професії членів родини. Множина на цьому етапі менш важлива.
2.  **Кулінарія (Інгредієнти):** Конструкції з родовим відмінком:
    *   `додати/треба + [інгредієнт в Родовому відмінку]` (напр., *додати цукру, треба молока*).
    *   `без + [інгредієнт в Родовому відмінку]` (напр., *кава без молока*).
    *   `багато/мало + [інгредієнт в Родовому відмінку]` (напр., *багато сиру, мало солі*).
3.  **Кулінарія (Страви та інструменти):** Орудний відмінок для знаряддя дії: *різати ножем*, *їсти ложкою*. Конструкція `з + [Орудний відмінок]` для начинки: *вареники з сиром*, *пиріг з яблуками* [S3].

Закінчення `-у/-ю` в родовому відмінку для речовин (цукор, сир, мед, борщ) є більш частотним у кулінарному контексті, ніж закінчення `-а/-я`, тому його слід відпрацьовувати в першу чергу.

## Типові помилки L2 (англомовні учні)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я працюю **вчитель**. | Я працюю **вчителем**. | Пряме перенесення англійської конструкції "I work as **a teacher**", де іменник стоїть у називному відмінку. В українській мові для позначення ролі, функції чи професії використовується орудний відмінок. [S1] |
| Вона — моя **кращий друг**. | Вона — мій **кращий друг**. / Вона — моя **найкраща подруга**. | Помилка узгодження роду присвійного займенника (*моя* вимагає жіночого роду) та сплутування понять друг/подруга. Англійське "friend" гендерно-нейтральне. [S1] |
| Рецепт для **борща**. | Рецепт **борщу**. | Неправильний вибір закінчення родового відмінка для іменників II відміни. Назви страв та рідин, як правило, мають закінчення `-у/-ю`. [S1] |
| Я п'ю каву без **молоко**. | Я п'ю каву без **молока**. | Після прийменника `без` завжди вживається родовий відмінок. Студенти часто забувають змінити форму іменника, залишаючи його в називному відмінку. |
| Приготувати з **яйце** і **мука**. | Приготувати з **яйця** і **муки**. | Неправильне відмінювання інгредієнтів після прийменника `з`. Вимагається родовий відмінок, а не називний. [S3] |
| Скільки **цукор** додати? | Скільки **цукру** додати? | Після питального слова `скільки` іменник має стояти в родовому відмінку. |

## Деколонізаційні застереження

Важливо підкреслювати відмінності української граматики від російської, щоб уникнути поширених помилок, спричинених хибними аналогіями.

1.  **Фемінітиви (Feminine Nouns for Professions):** В сучасній українській мові активно використовуються фемінітиви: *лікарка, директорка, менеджерка, програмістка, вчителька*. Це є нормою і відображає повагу до гендерної рівності. У російській мові використання фемінітивів менш поширене, часто перевага надається чоловічим формам (*она хороший врач*). Слід заохочувати учнів вживати фемінітиви, де це доречно.
2.  **Закінчення родового відмінка:** Правила закінчень `-у/-ю` та `-а/-я` в українській мові відрізняються від російських. Наприклад, укр. *чашка ча**ю*** — рос. *чашка ча**я***; укр. *кілограм піск**у*** — рос. *килограмм песк**а***. Не можна покладатися на аналогію.
3.  **Прийменникові конструкції:**
    *   Українська конструкція **`на адресу`** (напрямок дії: *листи надсилайте на адресу...*) відрізняється від **`за адресою`** (місцезнаходження: *магазин розташований за адресою...*) [S4]. У російській мові в обох випадках частіше вживається *по адресу*.
    *   Слід уникати кальки з російської `приймати участь`. Правильно українською — **`брати участь`** [S1].
    *   Конструкція `згідно з` вимагає орудного відмінка (`згідно з планом`), на відміну від російської `согласно`, що вимагає давального (`согласно плану`) [S4].
4.  **Лексика:** Слід використовувати автентичну українську кулінарну лексику, уникаючи русизмів. Наприклад, `овочі` (а не *фрукти* для всього), `тістечко` (а не *пирожное*), `крамниця` (а не *магазин* у розмовному стилі) [S5].

## Природні приклади

**Група 1: Називання професії (Орудний відмінок)**
1.  З Вами говорить **помічник директора** школи. [S5]
2.  Ким я був раніше? [S2] (Питання про професію)
3.  Мій добрий **друже**, що з тобою? [S1] (Звертання, але ілюструє соціальну роль).
4.  Він без вагань заявив: «Це ж наш **гімназист** — Сашко Шаргей!» [S1]

**Група 2: Інгредієнти та відсутність (Родовий відмінок)**
5.  Розісланий килим, розкиданий горох — не можна ні **килима** підняти, ні **гороху** зібрати. [S1]
6.  Летіло без крил, упало без ніг, варив кухар без **вогню**. [S1]
7.  Що біжить без **відпочинку**, не стає ні на хвилинку? [S1]
8.  До **сиру** додаємо яєчний жовток і **цукор**. [S3] (Тут "цукор" в знахідному, але контекст родового очевидний).

**Група 3: Приготування їжі (Орудний та інші відмінки)**
9.  Вареники — фірмова страва української **кухні**. [S3]
10. Подаємо з **вершковим маслом** і **сметаною**. [S3]
11. На стіл просіваємо гіркою **борошно**. [S3]

**Група 4: Офіційне та неофіційне спілкування про роботу**
12. Шановний **заступнику директора**… [S5]
13. Ми будемо вдячні, якщо Ви організуєте для нього зустріч з Вашими **клієнтами**. [S5]

## Рекомендації для вправ

**Фаза 1: Рецептивна практика та контрольоване відтворення**
*   **Вправа "Хто є хто?":** З'єднати зображення людей різних професій з їхніми назвами. Потім скласти речення за зразком: "Це лікар. Він працює **лікарем**."
*   **Вправа "Заповни пропуски (рецепт)":** Дати простий рецепт (напр., салату) з пропущеними інгредієнтами. Учні мають поставити слова з довідки в родовому відмінку. *Зразок: "Для салату нам потрібно: 2 ____ (огірок), 3 ____ (помідор), трохи ____ (сіль) і ____ (олія)."*
*   **Трансформація:** Перетворити речення. *Зразок: "Мій брат — інженер." → "Мій брат працює **інженером**."*

**Фаза 2: Продуктивна практика в обмеженому контексті**
*   **Вправа "Діалог у ресторані":** Учні розігрують діалог, де один замовляє страву "без" якогось інгредієнта. *Зразок: "— Можна мені, будь ласка, каву? — Так, з молоком? — Ні, без **молока** і без **цукру**."*
*   **Вправа "Моя родина":** Учні пишуть 3-4 речення про професії своїх рідних, використовуючи орудний відмінок.
*   **Складання списку покупок:** Учні складають список продуктів для приготування певної страви (борщу, вареників), використовуючи родовий відмінок. *Зразок: "Купити: пакет **молока**, пачку **масла**, кілограм **картоплі**."*

**Фаза 3: Вільна творча практика**
*   **Проєкт "Моя улюблена страва":** Учень готує коротку розповідь або презентацію про свою улюблену страву: як вона називається, з яких інгредієнтів складається, чому вона йому подобається.
*   **Есе "Ким я хочу стати":** Написати короткий текст про професію мрії, пояснюючи, чому вона цікава.

## Зв'язки з іншими темами

*   **Передумови:**
    *   **Іменник (A1):** Учні повинні знати роди іменників та основи відмінювання.
    *   **Прикметник (A1-A2):** Для опису професій (*хороший лікар*) та їжі (*смачний борщ*) необхідне узгодження прикметників з іменниками.
    *   **Дієслова (A1):** Знання дієслів `бути`, `працювати`, `хотіти`, `готувати` є обов'язковим.

*   **Наступні теми:**
    *   **Дієслова руху та аспектологія (B1):** Для опису складних процесів приготування їжі (напр., *нарізати* vs. *різати*, *додати* vs. *додавати*).
    *   **Прислівники (A2-B1):** Розширює можливості опису дій (*швидко готувати, ретельно перемішувати*) [S3].
    *   **Складні речення (B1):** Дозволяє створювати детальніші описи роботи або кулінарних рецептів, використовуючи сполучники та сполучні слова.
