## Як це пояснюють у школі (How Schools Teach This)

Українські шкільні підручники вводять поняття вираження професії поступово, починаючи з простих конструкцій і ускладнюючи їх у середніх та старших класах.

**Початкова школа (2-4 класи):**
На початковому етапі учні знайомляться з назвами професій у називному відмінку в простих реченнях, де присудок виражений іменником. Це конструкція "Хтось є кимось".
- **Ідентифікація:** `Підмет — Присудок (іменник у Н.в.)`. Тире між підметом і присудком, вираженими іменниками, є стандартною структурою.
  - *Приклад:* `Тато — вчитель.` (Джерело: `5-klas-ukrmova-uhor-2022-1_s0062`, Source 45)
  - *Приклад:* `Головна професія — школяр!` (Джерело: `2-klas-ukrmova-vashulenko-2019-2_s0002`, Source 38)

**Середня школа (5-8 класи):**
У 5-6 класах вводиться поняття орудного відмінка для позначення роду діяльності. Конструкція "працювати (ким?)" стає центральною.
- **Дія/Функція:** `Дієслово + Іменник в Ор. в.`
  - *Зразок речення:* `Батько Івана, Петро Павлович, працює лікарем у дитячій лікарні.` (Джерело: `6-klas-ukrmova-betsa-2023_s0058`, Source 25)
  - *Вправа:* У підручнику для 6 класу пропонується скласти речення за схемою `Хто? + працює (ким?) + де?`, що закріплює використання орудного відмінка. (Джерело: `6-klas-ukrmova-betsa-2023_s0058`, Source 25)

**Старша школа (8-11 класи):**
У старших класах ця тема розглядається в межах синтаксису, а саме через **складений іменний присудок**. Це дає теоретичне обґрунтування, чому використовується орудний відмінок.
- **Складений іменний присудок** складається з дієслова-зв'язки (`бути`, `стати`, `робитися`, `вважатися` тощо) та іменної частини (іменника, прикметника і т.д.).
- **Пояснення:** Підручник для 11 класу (Source 15) зазначає, що іменна частина ставиться в **орудному відмінку**, коли вона називає **непостійну, набуту ознаку**. Професія є класичним прикладом такої ознаки.
  - *Приклад:* `Оксана була лікаркою.` (Джерело: `8-klas-ukrmova-zabolotnyi-2025_s0066`, Source 18)
  - *Приклад:* `Назар буде програмістом.` (Джерело: `8-klas-ukrmova-zabolotnyi-2025_s0066`, Source 18)
- Дієслово-зв'язка `бути` у формі теперішнього часу (`є`) зазвичай опускається, що повертає нас до початкової конструкції з тире (`Вода є ласкава` -> `Вода — ласкава`). (Джерело: `8-klas-ukrmova-zabolotnyi-2025_s0066`, Source 18)

Таким чином, шкільна програма рухається від простої ідентифікації (`Тато — інженер`) до опису функції (`Тато працює інженером`) і, нарешті, до зміни стану чи характеристики в часі (`Тато став інженером`, `Він буде інженером`), об'єднуючи це під синтаксичним поняттям складеного іменного присудка.

## Повна парадигма (Full Paradigm)

Для вираження професії в орудному відмінку використовуються іменники I та II відмін.

### Іменники II відміни (Чоловічий рід)
Більшість назв професій чоловічого роду належать до II відміни. Закінчення залежить від кінцевого приголосного основи.

| Тип основи | Закінчення | Називний (Хто?) | Орудний (Ким?) | Джерело прикладу |
| :--- | :--- | :--- | :--- | :--- |
| **Тверда (не на -р)** | `-ом` | програміст | програміст**ом** | Source 7 |
| | | художник | художник**ом** | Source 8 |
| | | бухгалтер | бухгалтер**ом** | Source 6 |
| **Тверда на -р** | `-ом` | директор | директор**ом** | Source 23 |
| | | архітектор | архітектор**ом** | Source 23 |
| | | лікар | лікар**ом** | Source 9 |
| **М'яка та мішана** | `-ем` / `-єм` | вчитель | вчител**ем** | Source 9 |
| | | слюсар | слюсар**ем** | <!-- VERIFY --> |
| | | водій | воді**єм** | Source 6 |
| | | кресляр | кресляр**ем** | Source 38 |

**Примітка:** Назви професій з основою на `-р` можуть мати закінчення `-ом` (тверда група) або `-ем` (м'яка та мішана). Правильне написання слід перевіряти за орфографічним словником (Джерело: `4-klas-ukrayinska-mova-ponomarova-2021-1_s0055`, Source 23). Наприклад: `лікар` -> `лікарем`, але `директор` -> `директором`.

### Іменники I відміни (Жіночий рід)
Фемінітиви для назв професій належать до I відміни.

| **Основа на -ія** | `-єю` | <!-- VERIFY --> | <!-- VERIFY --> | <!-- VERIFY --> |

### Іменники спільного роду
Деякі іменники на `-а, -я` можуть позначати осіб обох статей (напр. `суддя`, `листоноша`). Вони відмінюються як іменники I відміни.
- `Він став добрим суддею.`
- `Вона стала доброю суддею.`

## Частотність і пріоритети

1.  **Пріоритет №1: Конструкція `[Дієслово] + [Професія в Ор. в.]`**. Це найважливіша конструкція для засвоєння на рівні A2/B1.
    *   `працювати (ким?)`: Найчастотніший спосіб описати поточну роботу. `Мій тато працює вчителем.` (Джерело: `6-klas-ukrmova-betsa-2023_s0062`, Source 32).
    *   `бути (ким?)` (в минулому/майбутньому часі): Для опису минулої або майбутньої ролі. `Я буду вчителькою.` `Він був лікарем.` (Джерело: `8-klas-ukrmova-zabolotnyi-2025_s0066`, Source 18).
    *   `стати (ким?)`: Для опису зміни професії або досягнення мети. `Вона хоче стати терапевткою.` (Джерело: `6-klas-ukrmova-betsa-2023_s0100`, Source 24).

2.  **Пріоритет №2: Конструкція `[Підмет] — [Професія в Н. в.]`**. Граматично простіша, але не менш важлива для базової ідентифікації.
    *   `Моя мама — лікар, тато — викладач.` (Джерело: `ext-ulp_youtube-296`, Source 7).
    *   Це основний спосіб відповідати на питання "Хто твій тато за професією?".

3.  **Пріоритетні закінчення:** Слід зосередитися на найпоширеніших закінченнях орудного відмінка:
    *   Чоловічий рід: `-ом` (для більшості) та `-ем` (для слів типу `вчитель`).
    *   Жіночий рід: `-ою` (для слів на `-ка`) та `-ею` (для слів на `-иня`, `-ниця`).

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Вона хоче стати `вчителька`. | Вона хоче стати `вчителькою`. | Прямий переклад з англійської ("She wants to become a teacher"), де не змінюється відмінок. Після дієслів `бути` та `стати` професія вживається в орудному відмінку. (Джерело: Source 18, 24). |
| Я працюю `як` програміст. | Я працюю `програмістом`. | Калька з англійської "I work *as a* programmer". В українській мові функція "as a" передається самим орудним відмінком, без прийменника `як`. (Джерело: Source 25, 32). |
| Він був `директор`. | Він був `директором`. | Помилкове вживання називного відмінка замість орудного в минулому часі. Правило складеного іменного присудка вимагає орудного відмінка для позначення набутої ролі. (Джерело: Source 18, 15). |
| Мій тато є `інженером`. | Мій тато — `інженер`. | Хоча граматично можлива, конструкція `є + Орудний відмінок` для простої констатації факту звучить неприродно або занадто формально. Для простої ідентифікації "A is B" використовується тире і називний відмінок. (Джерело: Source 45, 7). |
| Він працює `вчительом`. | Він працює `вчителем`. | Плутанина між закінченнями твердої (`-ом`) та м'якої (`-ем`) груп II відміни. Основа слова `вчитель` закінчується на м'який `л`, тому вимагає закінчення `-ем`. (Джерело: Source 9, 23). |

## Деколонізаційні застереження (Decolonization Notes)

1.  **Активне вживання фемінітивів:** Сучасна українська мова активно використовує та створює жіночі форми для назв професій (фемінітиви). Це є важливою рисою, яка відрізняє її від сучасної російської, де часто для жінок у формальному мовленні вживають маскулінітиви (`она врач`, `она директор`).
    *   **Приклади:** `вчителька` (Source 6), `лікарка` (Source 6), `програмістка` (Source 6), `дизайнерка` (Source 6), `журналістка` (Source 8), `фотографиня` (Source 8), `письменниця` (Source 6), `дослідниця` (Source 6).
    *   **Рекомендація:** Завжди надавати перевагу фемінітивам, коли йдеться про жінку, оскільки це відповідає сучасній мовній нормі та суспільним тенденціям в Україні. Не варто представляти фемінітиви як "розмовний" або "необов'язковий" варіант.

2.  **Уникнення кальок з російської:** Не слід пояснювати українські граматичні явища через порівняння з російською мовою. Українська граматична система є самодостатньою.
    *   **Приклад:** Конструкція `працювати (ким?)` є нормою. Не варто використовувати конструкцію з прийменником, як-от `працювати в якості...`, що є канцеляризмом, частково поширеним під впливом російської.

3.  **Специфічна лексика:** Деякі назви професій можуть бути унікальними або мати інші стилістичні відтінки. Наприклад, слово `митець` (чол.) / `мисткиня` (жін.) (Source 6) для митця є питомо українським і стилістично виразним.

## Рекомендації для вправ (Activity Concepts)

-   **Phase 1 (Recognition & Basic Forms):**
    -   **Вправа 1 (Вибір):** Дати речення з пропуском і два варіанти: в називному та орудному відмінках. `Вона працює ___ (лікарка / лікаркою)`.
    -   **Вправа 2 (Створення фемінітивів):** Дати список чоловічих професій і попросити утворити жіночі. `вчитель -> вчителька`, `фотограф -> фотографиня`. (Джерело: Source 8, 37).
    -   **Вправа 3 (Формування Орудного відмінка):** Дати список професій в називному відмінку і попросити написати їх в орудному. `Директор -> директором`, `вчителька -> вчителькою`.

-   **Phase 2 (Controlled Production):**
    -   **Вправа 4 (Доповнення речень):** Дати речення з дієсловом і професією в дужках, яку потрібно поставити в правильну форму. `Мій друг хоче стати (програміст)`. -> `...стати програмістом`. (Джерело: Source 24).
    -   **Вправа 5 (Трансформація):** Перетворити речення з конструкцією "Хтось є..." на "Хтось працює як...". `Він — інженер.` -> `Він працює інженером.`

-   **Phase 3 (Free Production):**
    -   **Вправа 6 (Питання-відповідь):** Поставити учням відкриті запитання про них та їхню родину. `Ким працюють твої батьки?` (Source 8), `Ким ти хочеш стати, коли виростеш?` (Source 23).
    -   **Вправа 7 (Творче завдання):** Написати короткий текст (3-4 речення) про професію мрії, використовуючи конструкції `я хочу бути/стати...`, `я буду працювати...`. (За мотивами Джерела: `4-klas-ukrayinska-mova-ponomarova-2021-1_s0055`, Source 23).
