## Як це пояснюють у школі (How Schools Teach This)

Українські шкільні підручники вводять поняття вираження професії поступово, починаючи з простих конструкцій і ускладнюючи їх у середніх та старших класах.

**Початкова школа (2-4 класи):**
На початковому етапі учні знайомляться з назвами професій у називному відмінку в простих реченнях, де присудок виражений іменником. Це конструкція "Хтось є кимось".
- **Ідентифікація:** `Підмет — Присудок (іменник у Н.в.)`. Тире між підметом і присудком, вираженими іменниками, є стандартною структурою.
  - *Приклад:* `Тато — вчитель.` [S45][S46]
  - *Приклад:* `Головна професія — школяр!` [S47][S48]

**Середня школа (5-8 класи):**
У 5-6 класах вводиться поняття орудного відмінка для позначення роду діяльності. Конструкція "працювати (ким?)" стає центральною.
- **Дія/Функція:** `Дієслово + Іменник в Ор. в.`
  - *Зразок речення:* `Батько Івана, Петро Павлович, працює лікарем у дитячій лікарні.` [S49][S50]
  - *Вправа:* У підручнику для 6 класу пропонується скласти речення за схемою `Хто? + працює (ким?) + де?`, що закріплює використання орудного відмінка. [S49][S50]

**Старша школа (8-11 класи):**
У старших класах ця тема розглядається в межах синтаксису, а саме через **складений іменний присудок**. Це дає теоретичне обґрунтування, чому використовується орудний відмінок.
- **Складений іменний присудок** складається з дієслова-зв'язки (`бути`, `стати`, `робитися`, `вважатися` тощо) та іменної частини (іменника, прикметника і т.д.).
- **Пояснення:** Підручник для 11 класу [S51] зазначає, що іменна частина ставиться в **орудному відмінку**, коли вона називає **непостійну, набуту ознаку**. Професія є класичним прикладом такої ознаки.
  - *Приклад:* `Оксана була лікаркою.` [S52]
  - *Приклад:* `Назар буде програмістом.` [S52]
- Дієслово-зв'язка `бути` у формі теперішнього часу (`є`) зазвичай опускається, що повертає нас до початкової конструкції з тире (`Вода є ласкава` -> `Вода — ласкава`). [S52]

Таким чином, шкільна програма рухається від простої ідентифікації (`Тато — інженер`) до опису функції (`Тато працює інженером`) і, нарешті, до зміни стану чи характеристики в часі (`Тато став інженером`, `Він буде інженером`), об'єднуючи це під синтаксичним поняттям складеного іменного присудка.

## Повна парадигма (Full Paradigm)

Для вираження професії в орудному відмінку використовуються іменники I та II відмін.

### Іменники II відміни (Чоловічий рід)
Більшість назв професій чоловічого роду належать до II відміни. Закінчення залежить від кінцевого приголосного основи.

| Тип основи | Закінчення | Називний (Хто?) | Орудний (Ким?) | Джерело прикладу |
| :--- | :--- | :--- | :--- | :--- |
| **Тверда (не на -р)** | `-ом` | програміст | програміст**ом** | Source 7 |
| | | художник | художник**ом** | Source 8 |
| | | бухгалтер | бухгалтер**ом** | Source 6 |
| **Тверда на -р** | `-ом` | директор | директор**ом** | Source 23 |
| | | архітектор | архітектор**ом** | Source 23 |
| | | лікар | лікар**ом** | Source 9 |
| **М'яка та мішана** | `-ем` / `-єм` | вчитель | вчител**ем** | Source 9 |
| | | слюсар | слюсар**ем** | <!-- VERIFY --> |
| | | водій | воді**єм** | Source 6 |
| | | кресляр | кресляр**ем** | Source 38 |

**Примітка:** Назви професій з основою на `-р` можуть мати закінчення `-ом` (тверда група) або `-ем` (м'яка та мішана). Правильне написання слід перевіряти за орфографічним словником [S53][S54]. Наприклад: `лікар` -> `лікарем`, але `директор` -> `директором`.

### Іменники I відміни (Жіночий рід)
Фемінітиви для назв професій належать до I відміни.

| Тип основи | Закінчення | Називний (Хто?) | Орудний (Ким?) | Джерело прикладу |
| :--- | :--- | :--- | :--- | :--- |
| **Тверда** | `-ою` | вчителька | вчительк**ою** | Source 6, Source 21 |
| | | лікарка | лікарк**ою** | Source 6 |
| | | менеджерка | менеджерк**ою**| Source 6 |
| **М'яка** | `-ею` | будівельниця | будівельниц**ею** | Source 6, Source 21 |
| | | засновниця | засновниц**ею** | Source 6 |
| **Мішана** | `-ею` | продавчиня | продавчин**ею** | Source 8, Source 21 |
| **Основа на -ія** | `-єю` | <!-- VERIFY --> | <!-- VERIFY --> | <!-- VERIFY --> |

### Іменники спільного роду
Деякі іменники на `-а, -я` можуть позначати осіб обох статей (напр. `суддя`, `листоноша`). Вони відмінюються як іменники I відміни.
- `Він став добрим суддею.`
- `Вона стала доброю суддею.`

## Частотність і пріоритети

1.  **Пріоритет №1: Конструкція `[Дієслово] + [Професія в Ор. в.]`**. Це найважливіша конструкція для засвоєння на рівні A2/B1.
    *   `працювати (ким?)`: Найчастотніший спосіб описати поточну роботу. `Мій тато працює вчителем.` [S55].
    *   `бути (ким?)` (в минулому/майбутньому часі): Для опису минулої або майбутньої ролі. `Я буду вчителькою.` `Він був лікарем.` [S52].
    *   `стати (ким?)`: Для опису зміни професії або досягнення мети. `Вона хоче стати терапевткою.` [S50][S53].

2.  **Пріоритет №2: Конструкція `[Підмет] — [Професія в Н. в.]`**. Граматично простіша, але не менш важлива для базової ідентифікації.
    *   `Моя мама — лікар, тато — викладач.` [S56].
    *   Це основний спосіб відповідати на питання "Хто твій тато за професією?".

3.  **Пріоритетні закінчення:** Слід зосередитися на найпоширеніших закінченнях орудного відмінка:
    *   Чоловічий рід: `-ом` (для більшості) та `-ем` (для слів типу `вчитель`).
    *   Жіночий рід: `-ою` (для слів на `-ка`) та `-ею` (для слів на `-иня`, `-ниця`).

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Вона хоче стати `вчителька`. | Вона хоче стати `вчителькою`. | Прямий переклад з англійської ("She wants to become a teacher"), де не змінюється відмінок. Після дієслів `бути` та `стати` професія вживається в орудному відмінку. [S52][S53]. |
| Я працюю `як` програміст. | Я працюю `програмістом`. | Калька з англійської "I work *as a* programmer". В українській мові функція "as a" передається самим орудним відмінком, без прийменника `як`. [S50][S55]. |
| Він був `директор`. | Він був `директором`. | Помилкове вживання називного відмінка замість орудного в минулому часі. Правило складеного іменного присудка вимагає орудного відмінка для позначення набутої ролі. [S52][S51]. |
| Мій тато є `інженером`. | Мій тато — `інженер`. | Хоча граматично можлива, конструкція `є + Орудний відмінок` для простої констатації факту звучить неприродно або занадто формально. Для простої ідентифікації "A is B" використовується тире і називний відмінок. [S46][S56]. |
| Він працює `вчительом`. | Він працює `вчителем`. | Плутанина між закінченнями твердої (`-ом`) та м'якої (`-ем`) груп II відміни. Основа слова `вчитель` закінчується на м'який `л`, тому вимагає закінчення `-ем`. [S57][S54]. |

## Деколонізаційні застереження (Decolonization Notes)

1.  **Активне вживання фемінітивів:** Сучасна українська мова активно використовує та створює жіночі форми для назв професій (фемінітиви). Це є важливою рисою, яка відрізняє її від сучасної російської, де часто для жінок у формальному мовленні вживають маскулінітиви (`она врач`, `она директор`).
    *   **Приклади:** `вчителька` [S58], `лікарка` [S58], `програмістка` [S58], `дизайнерка` [S58], `журналістка` [S59], `фотографиня` [S59], `письменниця` [S58], `дослідниця` [S58].
    *   **Рекомендація:** Завжди надавати перевагу фемінітивам, коли йдеться про жінку, оскільки це відповідає сучасній мовній нормі та суспільним тенденціям в Україні. Не варто представляти фемінітиви як "розмовний" або "необов'язковий" варіант.

2.  **Уникнення кальок з російської:** Не слід пояснювати українські граматичні явища через порівняння з російською мовою. Українська граматична система є самодостатньою.
    *   **Приклад:** Конструкція `працювати (ким?)` є нормою. Не варто використовувати конструкцію з прийменником, як-от `працювати в якості...`, що є канцеляризмом, частково поширеним під впливом російської.

3.  **Специфічна лексика:** Деякі назви професій можуть бути унікальними або мати інші стилістичні відтінки. Наприклад, слово `митець` (чол.) / `мисткиня` (жін.) [S58] для митця є питомо українським і стилістично виразним.

## Природні приклади (Natural Examples)

### Група 1: Конструкція з дієсловом `бути` (минулий і майбутній час)

1.  `Христина ... буде працювати вчителькою в школі.` [S60] - *Поєднання `буде працювати` чітко вказує на функцію в майбутньому.*
2.  `Оксана була лікаркою.` [S52] - *Класичний приклад опису минулої професії.*
3.  `Назар буде програмістом.` [S52] - *Простий опис майбутньої професії.*
4.  `Ким ти мрієш стати?` [S61] - *Хоча тут дієслово `стати`, відповідь передбачає `Я мрію бути/стати...`*

### Група 2: Конструкція з дієсловом `стати`

5.  `Однак я стану актрисою!` [S62] - *Вираження твердого наміру змінити свій статус.*
6.  `Вона хоче стати терапевткою.` [S50][S53] - *Опис бажання здобути професію.*

### Група 3: Конструкція з дієсловом `працювати`

7.  `Батько Івана, Петро Павлович, працює лікарем у дитячій лікарні.` [S49][S50] - *Типовий опис поточної роботи.*
8.  `А зараз він працює за фахом?` [S50][S53] - *Питання про відповідність роботи освіті.*
9.  `Мій брат програміст, він працює у міжнародній компанії.` [S55] - *Спочатку ідентифікація (Н.в.), потім опис діяльності (дієслово).*

### Група 4: Конструкція ідентифікації з тире (Називний відмінок)

10. `Моя мама — лікар, тато — викладач, а бабуся і дідусь — пенсіонери.` [S56] - *Швидке перелічення ролей членів родини.*
11. `Мама Наталки — журналістка, а тато — художник.` [S59] - *Проста ідентифікація.*
12. `Піфагор — це давньогрецький філософ і математик.` [S55] - *Історична ідентифікація.*

## Рекомендації для вправ (Activity Concepts)

-   **Phase 1 (Recognition & Basic Forms):**
    -   **Вправа 1 (Вибір):** Дати речення з пропуском і два варіанти: в називному та орудному відмінках. `Вона працює ___ (лікарка / лікаркою)`.
    -   **Вправа 2 (Створення фемінітивів):** Дати список чоловічих професій і попросити утворити жіночі. `вчитель -> вчителька`, `фотограф -> фотографиня`. [S59][S63].
    -   **Вправа 3 (Формування Орудного відмінка):** Дати список професій в називному відмінку і попросити написати їх в орудному. `Директор -> директором`, `вчителька -> вчителькою`.

-   **Phase 2 (Controlled Production):**
    -   **Вправа 4 (Доповнення речень):** Дати речення з дієсловом і професією в дужках, яку потрібно поставити в правильну форму. `Мій друг хоче стати (програміст)`. -> `...стати програмістом`. [S53].
    -   **Вправа 5 (Трансформація):** Перетворити речення з конструкцією "Хтось є..." на "Хтось працює як...". `Він — інженер.` -> `Він працює інженером.`

-   **Phase 3 (Free Production):**
    -   **Вправа 6 (Питання-відповідь):** Поставити учням відкриті запитання про них та їхню родину. `Ким працюють твої батьки?` [S59], `Ким ти хочеш стати, коли виростеш?` [S54].
    -   **Вправа 7 (Творче завдання):** Написати короткий текст (3-4 речення) про професію мрії, використовуючи конструкції `я хочу бути/стати...`, `я буду працювати...`. (За мотивами [S53], [S54]).
