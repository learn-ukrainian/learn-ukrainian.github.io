## Як це пояснюють у школі

Українська шкільна граматика вводить поняття часу та виду дієслова поступово, причому **вид** є фундаментальною категорією, що передує повному розумінню системи часів.

Спочатку учні знайомляться з трьома основними часами: теперішнім, минулим і майбутнім [S3]. Проте ключовий зсув у розумінні відбувається з уведенням категорії **виду дієслова** (aspect), як правило, у 6–7 класах [S4]. Підручники розрізняють два види:
1.  **Недоконаний вид** (imperfective): означає тривалу, незавершену, повторювану дію. Відповідає на питання *що робити?* (напр., `читати`, `писати`, `будувати`) [S1, S2].
2.  **Доконаний вид** (perfective): означає завершену, результативну, одноразову дію. Відповідає на питання *що зробити?* (напр., `прочитати`, `написати`, `збудувати`) [S1, S2].

Після встановлення цієї дихотомії, система часів уточнюється. Найважливіше правило, яке засвоюють учні: **дієслова в теперішньому часі можуть бути лише недоконаного виду** [S3]. Дія, що відбувається *зараз*, за визначенням не може бути завершеною. Дієслова доконаного виду форми теперішнього часу не мають [S1].

Наступним кроком є вивчення **способів дієслів**, які вказують на відношення дії до реальності [S6]:
*   **Дійсний спосіб** (`indicative`): констатує реальну дію, яка відбувалася, відбувається або відбуватиметься. Це єдиний спосіб, що має часові форми [S6].
*   **Умовний спосіб** (`conditional`): виражає бажану або можливу за певних умов дію (`я хотів би`, `ми зробили б`) [S6].
*   **Наказовий спосіб** (`imperative`): виражає наказ, прохання, спонукання до дії (`роби`, `ходімо`, `нехай знають`) [S5, S6].

Для рівня А2, контрольна точка фокусується на впевненому розрізненні видів, правильному вживанні трьох часів дійсного способу та основних форм наказового способу для комунікативних завдань.

## Повна парадигма

### 1. Творення видових пар

Більшість українських дієслів існують у видових парах, де від однієї основи різними засобами утворюються обидві форми [S2].

| Засіб творення | Механізм | Приклад (недоконаний → доконаний) | Джерело |
| :--- | :--- | :--- | :--- |
| **Префіксація** | Додавання префікса до дієслова недоконаного виду. Найпоширеніший спосіб. | `робити → з- + робити → зробити` <br> `писати → на- + писати → написати` <br> `вітати → при- + вітати → привітати` | [S1], [S2] |
| **Суфіксація** | Зміна або додавання суфікса. Часто відбувається у зворотному напрямку: від префіксованого доконаного виду утворюється недоконаний. | `розв'язати → розв'яз-**ува**-ти` <br> `викинути → викид-**а**-ти` <br> `стукати → стук-**ну**-ти` | [S1], [S2] |
| **Чергування звуків** | Зміна голосного або приголосного в корені. | `видирати → ви**дер**ти` <br> `перевозити → пере**вез**ти` <br> `допомагати → домо**мог**ти` | [S1], [S2] |
| **Зміна наголосу** | Перенесення наголосу. | `розріза́ти → розрі́зати` <br> `вино́сити → ви́нести` | [S1], [S2] |
| **Суплетивізм** | Утворення пари від різних коренів. Це винятки, які треба запам'ятовувати. | `брати → взяти` <br> `ловити → піймати` <br> `шукати → знайти` <br> `говорити → сказати` | [S1], [S2] |

Існують також **одновидові дієслова**, що мають лише одну форму (або доконану, або недоконану), та **двовидові**, де одна форма може означати обидва види залежно від контексту [S1].
*   **Одновидові (недоконані)**: `існувати`, `сподіватися`, `мріяти`, `гомоніти`, `стежити` [S1, S2].
*   **Одновидові (доконані)**: `завітати`, `розговоритися`, `натерпітися`, `провчитися` [S1, S2].
*   **Двовидові**: `атакувати`, `гарантувати`, `женити`, `ранити`, `телефонувати` [S1].

### 2. Часи дієслів (Дійсний спосіб)

#### Теперішній час
Виражає дію, що відбувається зараз або регулярно. **Тільки для дієслів недоконаного виду** [S3]. Змінюються за особами й числами.

| Особа | І дієвідміна (-уть/-ють) | ІІ дієвідміна (-ать/-ять) |
| :--- | :--- | :--- |
| 1 одн. (я) | чита**ю**, зна**ю** | говор**ю**, сто**ю** |
| 2 одн. (ти) | чита**єш**, зна**єш** | говор**иш**, сто**їш** |
| 3 одн. (він/вона/воно) | чита**є**, зна**є** | говор**ить**, сто**їть** |
| 1 мн. (ми) | чита**ємо**, зна**ємо** | говор**имо**, сто**їмо** |
| 2 мн. (ви) | чита**єте**, зна**єте** | говор**ите**, сто**їте** |
| 3 мн. (вони) | чита**ють**, зна**ють** | говор**ять**, сто**ять** |

*Чергування приголосних* в 1-й особі однини (іноді в усіх особах) є поширеним явищем [S3]:

| Чергування | Інфінітив | 1 ос. одн. (я) |
| :--- | :--- | :--- |
| `г → ж` | мо**г**ти | мо**ж**у |
| `к → ч` | пе**к**ти | пе**ч**у |
| `з → ж` | ма**з**ати | ма**ж**у |
| `с → ш` | пи**с**ати | пи**ш**у |
| `д → дж` | хо**д**ити | хо**дж**у |
| `т → ч` | хо**т**іти | хо**ч**у |
| `ст → щ` | мо**ст**ити | мо**щ**у |
| губний + `л` | лю**б**ити | лю**бл**ю |

#### Минулий час
Виражає дію, що відбулася до моменту мовлення. Утворюється від дієслів **обох видів** [S3]. Змінюється за родами (в однині) та числами.

**Творення:** `Основа інфінітива (без -ти) + суфікс` [S3].

| Число | Рід | Суфікс | Приклад |
| :--- | :--- | :--- | :--- |
| Однина | чол. | **-в** | `читав`, `зробив` |
| | жін. | **-л(а)** | `читала`, `зробила` |
| | сер. | **-л(о)** | `читало`, `зробило` |
| Множина | | **-л(и)** | `читали`, `зробили` |

**Особливість**: якщо основа інфінітива закінчується на приголосний, у формі чоловічого роду однини суфікс `-в` відсутній (`нульовий суфікс`) [S3].
*   `нес-ти → ніс` (відбулося чергування `е → і`)
*   `мог-ти → міг` (відбулося чергування `о → і`)
*   `пек-ти → пік` (відбулося чергування `е → і`)

#### Майбутній час
Виражає дію, що відбудеться після моменту мовлення. Форми залежать від **виду** дієслова [S3, S7].

**А. Доконаний вид (проста форма)**
Майбутня завершена дія. Форми утворюються від дієслова доконаного виду і збігаються із закінченнями теперішнього часу дієслів недоконаного виду [S3, S7].

| Особа | Приклад: `зробити` | Приклад: `написати` |
| :--- | :--- | :--- |
| 1 одн. (я) | зробл**ю** | напи**шу** |
| 2 одн. (ти) | зроб**иш** | напи**шеш** |
| 3 одн. (він/вона/воно) | зроб**ить** | напи**ше** |
| 1 мн. (ми) | зроб**имо** | напи**шемо** |
| 2 мн. (ви) | зроб**ите** | напи**шете** |
| 3 мн. (вони) | зробл**ять** | напи**шуть** |

**Б. Недоконаний вид (дві форми)**
Майбутня тривала або повторювана дія. Має дві паралельні форми: складену і просту [S3, S7].

| Особа | Складена форма (`бути` + інфінітив) | Проста форма (інфінітив + `-м-` + закінчення) |
| :--- | :--- | :--- |
| 1 одн. (я) | **буду** читати | читати**му** |
| 2 одн. (ти) | **будеш** читати | читати**меш** |
| 3 одн. (він/вона/воно) | **буде** читати | читати**ме** |
| 1 мн. (ми) | **будемо** читати | читати**мемо** |
| 2 мн. (ви) | **будете** читати | читати**мете** |
| 3 мн. (вони) | **будуть** читати | читати**муть** |

### 3. Наказовий спосіб
Виражає спонукання до дії. Не має часу, змінюється за особами і числами [S5, S6].

| Особа | Форма | Приклади |
| :--- | :--- | :--- |
| 2 одн. (ти) | проста | `читай`, `пиши`, `сядь`, `ріж` |
| 1 мн. (ми) | проста | `читаймо`, `пишімо`, `сядьмо`, `ріжмо` |
| 2 мн. (ви) | проста | `читайте`, `пишіть`, `сядьте`, `ріжте` |
| 3 одн. (він/вона/воно) | складена (`хай`/`нехай` + теп. час) | `хай читає`, `нехай пише` |
| 3 мн. (вони) | складена (`хай`/`нехай` + теп. час) | `хай читають`, `нехай пишуть` |

**Важливо**: у 2-й особі `ь` пишеться тільки після `д, т, з, с, л, н` (`сядь`), але не після губних, шиплячих та `р` (`став`, `ріж`) [S5, S6].

### 4. Дієслова руху
В українській граматиці дієслова руху не виділяють в таку ж окрему, ізольовану категорію, як у російській. Вони підпорядковуються загальним правилам творення видових пар через префіксацію. Однак для L2-учнів корисно розрізняти базові пари "односпрямованих" (unidirectional) та "різноспрямованих" (multidirectional) дієслів, від яких утворюються префіксальні видові пари. `<!-- VERIFY -->`

| Односпрямований (недок.) | Різноспрямований (недок.) | Доконаний (з префіксом `при-`) |
| :--- | :--- | :--- |
| іти (to go on foot, now) | ходити (to go on foot, generally) | прийти (to arrive on foot) |
| їхати (to go by vehicle, now) | їздити (to go by vehicle, generally) | приїхати (to arrive by vehicle) |
| летіти (to fly, now) | літати (to fly, generally) | прилетіти (to arrive by flying) |
| бігти (to run, now) | бігати (to run, generally) | прибігти (to arrive running) |

## Частотність і пріоритети

Для рівня A2/B1 пріоритетність засвоєння форм така:

1.  **Високий пріоритет (основа комунікації):**
    *   Впевнене розрізнення питань *що робити?* / *що зробити?* для вибору правильного виду.
    *   Усі форми **теперішнього часу** (тільки недоконаний вид) [S3].
    *   Усі форми **минулого часу** (обидва види), з особливою увагою до родових закінчень [S3].
    *   **Майбутній час**: складена форма недоконаного виду (`буду робити`) та проста форма доконаного виду (`зроблю`). Вони є найбільш частотними та концептуально зрозумілими [S3, S7].
    *   **Наказовий спосіб**: 2-га особа однини та множини (`роби`, `робіть`) [S5].

2.  **Середній пріоритет (розширення виразності):**
    *   **Майбутній час**: проста форма недоконаного виду (`робитиму`).
    *   **Наказовий спосіб**: 1-ша особа множини (`робімо`) та 3-тя особа (`хай робить`) [S5].
    *   Найбільш уживані **суплетивні пари**: `брати/взяти`, `говорити/сказати`. [S1]

3.  **Низький пріоритет (для подальшого вивчення на В1+):**
    *   Нюанси **двовидових дієслів** (`атакувати`) [S1].
    *   Запам'ятовування списків **одновидових дієслів** [S2].
    *   Стилістичне вживання форм одного способу в значенні іншого (напр., дійсний в значенні наказового: `Ти зараз підеш і зробиш це!`) [S6].

## Типові помилки L2 (англомовні учні)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| *Я вчора читав цю книгу і пішов додому.* | *Я вчора **прочитав** цю книгу і пішов додому.* | **Видова сліпота.** Англійська мова не має граматичної категорії виду. Для завершеної, результативної дії в минулому потрібен доконаний вид. Недоконаний (`читав`) означає процес ("I was reading"). |
| *Я зараз зроблюю домашнє завдання.* | *Я зараз **роблю** домашнє завдання.* | **Неправильне утворення теперішнього часу.** Учень намагається утворити теперішній час від дієслова доконаного виду (`зробити`), що неможливо в українській мові. Теперішній час утворюється лише від недоконаного виду (`робити`) [S3]. |
| *Завтра я буду прочитати статтю.* | *Завтра я **прочитаю** статтю.* <br> *Завтра я **буду читати** статтю.* | **Плутанина з формами майбутнього часу.** Допоміжне дієслово `буду` вживається лише з інфінітивом **недоконаного** виду. З доконаним видом `буду` ніколи не використовується [S3, S7]. |
| *Давайте поговоримо про це.* | ***Поговорімо** про це.* <br> ***Ходімо** в парк.* | **Калька з російської мови.** Конструкція `давайте + дієслово` для утворення наказового способу є русизмом. В українській мові використовуються форми на `-імо`/`-їмо` або спеціальні дієслова (`ходімо`, `нумо`) [S5]. |
| *Кожен день я **їду** на роботу на автобусі.* | *Кожен день я **їжджу** на роботу на автобусі.* | **Плутанина з дієсловами руху.** Для повторюваної, регулярної дії в обидва боки вживається різноспрямоване дієслово (`їздити`). Односпрямоване (`їхати`) використовується для конкретної поїздки в одному напрямку, що відбувається зараз. `<!-- VERIFY -->` |
| *Моя сестра **написав** листа.* | *Моя сестра **написала** листа.* | **Ігнорування роду в минулому часі.** В англійській мові немає узгодження дієслова з родом підмета в минулому часі ("she wrote"). В українській це обов'язково [S3]. |

## Деколонізаційні застереження

Вкрай важливо викладати українську граматику як самостійну систему, а не як діалект російської з "дивацтвами".

1.  **Наказовий спосіб: `ходімо`, а не `давайте підемо`**. Найпомітніший маркер суржику та російського впливу — це використання конструкції `давайте + дієслово`. Слід наголошувати на автентичних українських формах: `зробімо`, `поговорімо`, `сядьмо`. Для спонукання до руху є окреме дієслово `ходімо` [S5].
2.  **Майбутній час на `-му`, `-меш` (`писатиму`)**. Ця синтетична (проста) форма майбутнього часу недоконаного виду є унікальною рисою української мови, яка відсутня в російській (де є лише аналітична `буду писать`). Її слід подавати як повноцінний, літературний і природний варіант, а не як архаїзм [S3, S7].
3.  **Чергування `г`, `к`, `х` зі шиплячими**. Хоча подібні чергування є в інших слов'янських мовах, в українській вони відбуваються в дієсловах ІІ дієвідміни у всіх особах (крім 1 і 3 ос. мн.), тоді як у російській — часто лише в 1-й особі однини. Порівняй: укр. `я можу`, `ти можеш`, `він може` vs. рос. `я могу`, `ты можешь`, `он может`. Викладання української парадигми має відбуватися без огляду на російську.
4.  **Викладання дієслів руху**. Методика викладання дієслів руху в російських підручниках (чітке розділення на `глаголы движения I и II группы`) не є канонічною для української педагогіки. В українських підручниках ці дієслова органічно інтегровані в загальну тему виду та префіксального словотворення [S1, S2, S3]. Нав'язування російської методичної рамки є помилкою.
5.  **Вимова `-шся` і `-ться`**. Вимова `[с':а]` на місці `-шся` (`вмиваєшся` → `[вмивайес':а]`) та `[ц':а]` на місці `-ться` (`вмивається` → `[вмивайец':а]`) є характерною рисою української фонетики [S7]. Не можна допускати російської вимови `[шся]` чи `[ца]`.

## Природні приклади

**Група 1: Теперішній час (регулярна дія або стан)**
*   `На столі стоїть сільниця, у сільниці — сіль.` [S7]
*   `Я чую твій голос, пшеничний твій колос у душу мені засіває зерно.` [S7]
*   `Дієслова у формі теперішнього часу позначають дію, що відбувається в момент мовлення.` [S3]

**Група 2: Минулий час (процес vs. результат)**
*   `Учні цілий урок розв’язували завдання, але не могли розв’язати правильно.` [S2] (Процес і відсутність результату).
*   `Учора я побував у гостях у бабусі. Вона була дуже щаслива, коли я пішов.` [S2] (Завершені дії).
*   `Софійський собор стояв у Києві з давніх-давен. Його збудували київські князі.` [S3] (Тривалий стан `стояв` і завершена дія `збудували`).

**Група 3: Майбутній час (різні форми)**
*   `Виросте Тарасик, піде у світ і почне писати золотим пером...` [S7] (Проста доконанa форма).
*   `Тепер, якщо хтось вимкне рубильник, машина повідомить йому про це.` [S3] (Проста доконанa форма).
*   `Я сказав, що піду з ним до околиці, потім повернусь на головну дорогу і поїду автостопом.` [S3] (Проста доконанa форма).
*   `Що робитимеш? Що зробиш?` [S3] (Проста недоконана і проста доконана форми).

**Група 4: Наказовий спосіб (прохання, наказ)**
*   `Губами говори, а руками роби!` [S5] (2 ос. одн.).
*   `Розгорніть, будь ласка, зошит, запишіть число.` [S5] (2 ос. мн.).
*   `Спиніться під деревом, звідки доноситься голос вивільги.` [S7] (2 ос. мн.).
*   `Ходімо разом!` [S5] (1 ос. мн.).
*   `Хай кричить, хай пробачить.` [S5] (3 ос. одн.).

## Рекомендації для вправ

**Фаза 1: Розпізнавання та ідентифікація**
1.  **"Знайди дієслово":** Учень читає текст і підкреслює всі дієслова, визначаючи їх вид, час, особу, число, рід (для минулого часу). Приклад вправи: [S3, впр. 50].
2.  **"Сортування":** Розподілити список дієслів на три колонки: минулий, теперішній, майбутній час. Приклад вправи: [S3, впр. 49].

**Фаза 2: Формотворення**
1.  **"Утвори пару":** Дано дієслово недоконаного виду, учень має утворити доконаний за допомогою префіксів чи інших засобів, і навпаки. Приклад вправи: [S2, впр. 441], [S1, впр. 515].
2.  **"Зміни час":** Поставити дієслова з речення в іншому часі (напр., теперішній → минулий). Приклад вправи: [S3, впр. 56].
3.  **"Утвори наказовий спосіб":** Від інфінітивів утворити форми 2-ї особи однини та множини наказового способу. Приклад вправи: [S6, впр. 527].
4.  **"Усі форми майбутнього":** Від дієслова недоконаного виду (`писати`) утворити обидві форми майбутнього часу в усіх особах. Від дієслова доконаного (`написати`) — одну форму. Приклад вправи: [S3, впр. 67].

**Фаза 3: Контекстуальне застосування**
1.  **"Вибери правильний вид":** Речення з пропуском, де треба обрати один з двох видів дієслова, поданих у дужках. Контекст (наявність слів `довго`, `цілий день` vs. `нарешті`, `раптом`) має підказувати вибір. Приклад вправи: [S2, впр. 440].
2.  **"Дай відповідь на питання":** Питання `Що ти робиш?` вимагає відповіді в теперішньому часі. Питання `Що ти зробив?` — у минулому доконаному. Питання `Чому ти не пишеш?` провокує відповідь `Тому що я вже написав`. Приклад вправи: [S2, впр. 439].
3.  **"Склади розповідь":** Описати свій день, плануючи майбутній тиждень або переказуючи минулу подію, фокусуючись на правильному використанні часів та видів. Приклад вправи: [S3, впр. 61, 69].

## Зв'язки з іншими темами

*   **Передумови:**
    *   **Особові займенники:** Необхідні для розуміння дієвідмінювання за особами.
    *   **Відмінки іменників:** Особливо знахідний відмінок (`Accusative`), щоб розрізняти перехідні/неперехідні дієслова, що впливає на творення, наприклад, пасивних дієприкметників.

*   **Наступні теми:**
    *   **Дієприкметник (Participle):** Його вид (активний/пасивний) і час напряму залежать від виду і перехідності твірного дієслова. Наприклад, активні дієприкметники теперішнього часу (`читаючий`) утворюються лише від дієслів недоконаного виду [S4].
    *   **Дієприслівник (Gerund):** Форми дієприслівників недоконаного (`читаючи`) та доконаного (`прочитавши`) виду є прямим продовженням теми виду дієслова [S4].
    *   **Умовний спосіб:** Конструкції `...в би/...ла б` використовують форму минулого часу, тому її засвоєння є обов'язковим [S6].
    *   **Складні речення:** Правильне узгодження часів і видів у головній та підрядній частинах речення є основою складного синтаксису.
