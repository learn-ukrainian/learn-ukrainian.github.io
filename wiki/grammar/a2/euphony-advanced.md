## Як це пояснюють у школі (How Schools Teach This)
Милозвучність, або **евфонія** (грецьк. *euphone* — милозвучність), є однією з фундаментальних, визначальних рис української мови (Джерело: 10-klas-ukrajinska-mova-avramenko-2018_s0115). Українські шкільні підручники вводять це поняття вже у 5-му класі, підкреслюючи, що це не просто набір правил, а свідоме прагнення мови уникати складного для вимови нагромадження звуків (Джерело: 10-klas-ukrmova-glazova-2018_s0098).

Педагогічний підхід зазвичай починається з простого практичного спостереження. Учням пропонують прочитати дві версії речення і самостійно визначити, яку з них вимовляти "легше й зручніше" (Джерело: 5-klas-ukrmova-avramenko-2022_s0120).

**Приклад із підручника (5 клас):**
*   *Неправильно:* Брат й син ввечері привітали з святом.
*   *Правильно:* Брат **і** син **у**вечері привітали **зі** святом. (Джерело: 5-klas-ukrmova-avramenko-2022_s0120)

Через такі порівняння учні доходять висновку, що українська мова має вбудовані механізми для уникнення збігу кількох голосних або приголосних підряд. Ці механізми є основними засобами милозвучності.

Основні засоби, що вивчаються в школі (починаючи з 5-го класу), це:
1.  **Чергування прийменників/префіксів `У` — `В`**.
2.  **Чергування сполучників/початкових літер `І` — `Й`**.
3.  **Чергування прийменників `З` — `ІЗ` — `ЗІ` (`ЗО`)**. (Джерело: 5-klas-ukrmova-litvinova-2022_s0007)

У старших класах (10 клас) це поняття розширюється, і учні дізнаються, що милозвучність впливає не тільки на вимову, а й на письмо, зумовлюючи також:
*   **Спрощення в групах приголосних** (напр., *проїзд* -> *проїзний*). (Джерело: 10-klas-ukrajinska-mova-avramenko-2018_s0115)
*   **Зміни приголосних при словотворенні** (напр., *Прага + -ськ(ий)* -> *празький*).
*   **Поява вставних голосних у прийменниках/префіксах** (напр., *піді*, *наді*). (Джерело: 10-klas-ukrmova-glazova-2018_s0100)

Термінологія, що використовується, є прямою: **милозвучність**, **евфонія**, **чергування у–в, і–й**, **засоби милозвучності**. Відхилення від цих норм називається **орфоепічною помилкою** (Джерело: 10-klas-ukrajinska-mova-avramenko-2018_s0115).

## Повна парадигма (Full Paradigm)
Милозвучність в українській мові досягається через систему чергувань, яка працює на стику слів і всередині них. Це не стільки парадигма відмінювання, скільки набір фонетично зумовлених правил вибору.

### 1. Чергування `У` — `В`
Це стосується прийменників `у`/`в` та початкових літер слів (префіксів) `у-`/`в-`. Мета — уникнути збігу голосних або приголосних.

| Ситуація | Вживаємо `У` | Приклади | Вживаємо `В` | Приклади | Джерело |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Між приголосними** | **У** | наш **у**читель, вечір **у** місті, син **у** спортзалі | | | 10-klas-ukrmova-zabolotnyi-2018_s0105 |
| **Між голосними** | | | **В** | риба **в** акваріумі, успіхи **в** економіці, мешкає **в** Івано-Франківську | 10-klas-ukrajinska-mova-zabolotnij-2018_s0105, 5-klas-ukrmova-litvinova-2022_s0179 |
| **Після приголосного, перед голосним** | | | **В** | брат **в** Одесі, він **в** автобусі | 10-klas-ukrajinska-mova-avramenko-2018_s0117 |
| **Після голосного, перед приголосним** | | | **В** | пішла **в** садок, тепло **в** травні | 10-klas-ukrmova-zabolotnyi-2018_s0105 |
| **Початок речення** | **У** (перед приголосним) | **У** лісі багато грибів. **У** школі свято. | **В** (перед голосним) | **В** Одесі тепло. **В** озері знайшли човен. | 10-klas-ukrajinska-mova-zabolotnij-2018_s0105, 10-klas-ukrajinska-mova-avramenko-2018_s0117 |
| **Після паузи (розділового знака)** | **У** (перед приголосним) | У полі, **у** лісі волого. Ольга Іллівна — **у**чителька. | **В** (перед голосним) | Мабуть, **в** Одесі холодно. | 6-klas-ukrmova-avramenko-2023_s0242, 10-klas-ukrajinska-mova-avramenko-2018_s0117 |

### 2. Чергування `І` — `Й`
Схоже на `у-в`, це чергування стосується сполучника `і`/`й` та початкових літер слів `і-`/`й-`.

| Ситуація | Вживаємо `І` | Приклади | Вживаємо `Й` | Приклади | Джерело |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Між приголосними** | **І** | син **і** донька, урок **і**де | | | 5-klas-ukrmova-litvinova-2022_s0181 |
| **Між голосними** | | | **Й** | діти **й** дорослі, життя **й**де | 5-klas-ukrmova-litvinova-2022_s0181 |
| **Після голосного, перед приголосним** | | | **Й** | вона **й** не знала, річки **й** моря | 5-klas-ukrmova-litvinova-2022_s0181, 6-klas-ukrmova-avramenko-2023_s0242 |
| **Початок речення** | **І** (перед приголосним) | **І** дайте відповідь. **І**дуть дощі. | **Й** (перед голосним) | **Й** учимося грамотно писати. | 5-klas-ukrmova-litvinova-2022_s0181 |
| **Після паузи (розділового знака)** | **І** (перед приголосним) | Озирнувся — **і** подався вперед. | | | 10-klas-ukrajinska-mova-avramenko-2018_s0117 |
| **Перед `й, я, ю, є, ї`** | **І** (оскільки ці букви починаються зі звука [й]) | Одеса **і** Ялта, лисиця **і** їжак, сосни **і** ялинки, мені **і** йому | | | 6-klas-ukrmova-avramenko-2023_s0242, 5-klas-ukrmova-litvinova-2022_s0181 |
| **Виняток: зіставлення/протиставлення** | **І** (не чергується) | батьки **і** діти, добро **і** зло, Шевченко **і** сучасність | | | 6-klas-ukrmova-avramenko-2023_s0242, 10-klas-ukrajinska-mova-avramenko-2018_s0117 |

### 3. Чергування `З` — `ІЗ` — `ЗІ` (`ЗО`)
Це чергування прийменника є більш складним і залежить від комбінації звуків на межі слів.

| Форма | Коли вживається | Приклади | Джерело |
| :--- | :--- | :--- | :--- |
| **З** | 1. Перед словом, що починається з **голосного** (`з` + Г).<br>2. Перед **приголосним** (крім шиплячих/свистячих), якщо немає складного збігу (`з` + П). | 1. вареники **з** вишнями, **з** однокласницями, **з** Одеси<br>2. **з** нагоди святкування, чай **з** лимоном | 5-klas-ukrmova-litvinova-2022_s0182, 5-klas-ukrmova-zabolotnyi-2023_s0126 |
| **ІЗ** | 1. **Між приголосними** (П + `із` + П).<br>2. Перед або після певних **шиплячих і свистячих** (`з, с, ц, ч, ш, щ`). | 1. Максим **із** Семеном, вареники **із** сиром, біг **із** бар'єрами<br>2. повернувся **із** села, товариш **із** Житомира | 5-klas-ukrmova-litvinova-2022_s0182, 5-klas-ukrmova-avramenko-2022_s0122 |
| **ЗІ** | 1. Перед словом, що починається **збігом кількох приголосних**, особливо якщо перші з них `з, с, ш, щ`.<br>2. Обов'язково перед деякими словами для уникнення збігу. | 1. **зі** святом, **зі** швидкістю, **зі** школи, **зі** Львова<br>2. **зі** мною (завжди) | 5-klas-ukrmova-litvinova-2022_s0182, 5-klas-ukrmova-avramenko-2022_s0122 |
| **ЗО** | З числівниками **`два`, `три`**. | **зо** два, **зо** три, **зо** дві сотні | 5-klas-ukrmova-avramenko-2022_s0122, 5-klas-ukrmova-zabolotnyi-2023_s0126 |

**Примітка:** Іноді `із` та `зі` можуть бути взаємозамінними, напр. *разом із сходом сонця* / *разом зі сходом сонця* (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0126).

## Частотність і пріоритети (Frequency & Priorities)
Для рівнів A2-B1 пріоритети засвоєння мають бути такими:

1.  **Найвищий пріоритет (A2 Core):** Чергування **`у/в`** та **`і/й`**. Це найчастотніші сполучники та прийменники, і їх неправильне вживання одразу робить мовлення неприродним. Ці правила мають бути доведені до автоматизму, оскільки вони стосуються не лише окремих слів, а й початкових звуків дієслів та інших частин мови (*узяв/взяв*, *уже/вже*, *іде/йде*). (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0120).

2.  **Середній пріоритет (A2+):** Чергування **`з/із/зі`**. Правила тут складніші та мають більше винятків, що залежать від фонетичного оточення. Спершу варто засвоїти найуживаніші випадки: `з` + голосний (*з Оленою*), `зі` + збіг приголосних (*зі Львова*), та сталі вирази (*зі мною*). Форма `із` між двома приголосними (*лист із Києва*) також є високочастотною. Форма `зо` є рідкісною і обмежується кількома виразами.

3.  **Нижчий пріоритет (B1+):** Інші засоби милозвучності. Вживання `піді`, `-сь`, `б`, `ж` та розуміння спрощення є ознакою більш вільного володіння мовою. На рівні А2 учень може обійтися без них, хоча пасивно буде зустрічати їх у текстах. Активне використання цих форм — мета рівня В1 і вище. Наприклад, уміння сказати `я б хотів` замість `я би хотів` після голосного свідчить про глибше відчуття мовного ритму.

## Типові помилки L2 (Common L2 Errors)
Англомовні студенти часто роблять помилки, переносячи фонетичні звички своєї мови або ігноруючи системність української милозвучності.

| ❌ Помилково (Неправильно) | ✅ Правильно | Чому (Why the error occurs) |
| :--- | :--- | :--- |
| *Він був **в** школі.* (збіг `в-ш-к`) | Він був **у** школі. | Англійська мова толерує збіги приголосних. Учень ігнорує правило "приголосний + **У** + приголосний". (Джерело: 10-klas-ukrmova-glazova-2018_s0098) |
| *Я бачу Оксану **й** Івана.* (збіг `у-й-і`) | Я бачу Оксану **та** Івана. / Я бачу Оксану **й** Олега. | Учень механічно чергує `і/й`, але після голосного перед приголосним краще вжити `й`. Однак, якщо далі йде `І`, виникає незручний збіг. Підручники пропонують у таких випадках сполучник `та` (Джерело: 5-klas-ukrmova-litvinova-2022_s0181). Правило: після голосного перед `і` вживається `й` (`вона й іде`), але в цілому краще уникати такої конструкції. |
| *Це історія про добро **й** зло.* | Це історія про добро **і** зло. | Учень застосовує стандартне чергування `і/й` до випадку, який є винятком. При зіставленні або протиставленні понять завжди вживається сполучник `і`. (Джерело: 6-klas-ukrmova-avramenko-2023_s0242) |
| *Я розмовляв **з** своїм другом.* (збіг `з-с-в`) | Я розмовляв **зі** своїм другом. | Збіг трьох приголосних (`[зсв]`) є складним для вимови. Учень обирає базову форму прийменника `з`, не знаючи, що в таких випадках для розриву збігу використовується `зі`. (Джерело: 5-klas-ukrmova-litvinova-2022_s0182) |
| *Моя сестра **у** Одесі.* (збіг `а-у-о`) | Моя сестра **в** Одесі. | Учень, намагаючись уникнути збігу приголосних, створює збіг голосних. Правило "голосний + **В** + голосний" ігнорується. (Джерело: 10-klas-ukrmova-glazova-2018_s0098) |
| *Він хоче **і** яблуко, **і** йогурт.* | Він хоче **і** яблуко, **і** йогурт. (Правильно, але...) | Хоча формально це правильно (`і` перед `[й]`), природніше для уникнення повторення `і-і` було б сказати: "Він хоче яблуко **та** йогурт". Це вже стилістичний, а не суто фонетичний вибір. |

## Деколонізаційні застереження (Decolonization Notes)
**Це обов'язковий розділ. Милозвучність — одна з ключових відмінностей між українською та російською фонетичними системами.**

1.  **Українська евфонія — це система, а не виняток.** У російській мові також існують варіанти (`о/об`, `с/со`), але їх використання є менш системним і часто лексично зумовленим (напр., `со мной`). В українській мові чергування `у/в`, `і/й`, `з/із/зі` є всеохопним **фонетичним законом**, який діє автоматично на межі будь-яких слів (Джерело: 10-klas-ukrajinska-mova-avramenko-2018_s0115). Це не "стилістична перевага", а базова орфоепічна норма.

2.  **Уникайте "російської кальки".** Багато помилок у мовленні українців, які зазнали русифікації, походять саме з ігнорування правил милозвучності. Речення `Він був в школі` — типовий приклад впливу російської конструкції `Он был в школе`, де збіг `[фшк]` є нормою. Для української мови це ненормативно.

3.  **Прийменник `з` не є еквівалентом російського `с`.** Навчаючи прийменника `з` та його варіантів, не можна проводити пряму паралель з російським `с/со`. Правила їх вживання кардинально відрізняються. Наприклад, рос. `со мной`, `со школы` vs укр. `зі мною`, `зі школи`. Аналогія тут лише заплутає.

4.  **Милозвучність — ознака живої народної мови.** Прагнення до плавності та мелодійності мовлення є глибоко закоріненим у фольклорі та живій розмовній мові. Ігнорування цих правил створює штучне, "книжне" і часто суржикізоване мовлення. Мова стає "незграбною" (Джерело: 10-klas-ukrmova-glazova-2018_s0098).

5.  **Навчальний пріоритет.** Пояснюючи цю тему, слід наголосити, що дотримання цих правил для української мови є настільки ж важливим, як правильне використання артиклів в англійській. Це не "дрібниця", а маркер грамотності та поваги до мови.

## Природні приклади (Natural Examples)

**Група 1: Чергування У/В**
*   *Між приголосними:* Вона була **у** гарній формі. (Джерело: 10-klas-ukrajinska-mova-avramenko-2018_s0117)
*   *Після голосного:* Вони живуть **в** Ужгороді.
*   *Перед специфічними приголосними:* Я хочу вчитися **у** Львові. (Джерело: 5-klas-ukrmova-uhor-2022-1_s0177)
*   *Початок речення:* **У** селі було тихо, по-хмуро. (Джерело: 7-klas-ukrmova-zabolotnyi-2024_s0204)

**Група 2: Чергування І/Й**
*   *Між приголосними:* Праця чоловіка годує, а лінь марнує. (Примітка: тут `і` було б немилозвучним між `а` та `л`). Краще `...годує, а лінь **його** марнує` або `...а лінь **же** марнує`.
*   *Між голосними:* Річки **й** озера. (Джерело: 6-klas-ukrmova-avramenko-2023_s0242)
*   *Після голосного перед приголосним:* Ольга **й** Назар. (Джерело: 5-klas-ukrmova-avramenko-2022_s0122)
*   *При протиставленні (без чергування):* Це війна **і** мир, батьки **і** діти. (Джерело: 5-klas-ukrmova-litvinova-2022_s0181)

**Група 3: Чергування З/ІЗ/ЗІ/ЗО**
*   *Перед голосним:* Я приїхав **з** Одеси. (Джерело: 5-klas-ukrmova-litvinova-2022_s0182)
*   *Між приголосними:* Чай **із** лимоном. / Зустріч **із** письменником. (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0126, 10-klas-ukrmova-zabolotnyi-2018_s0205)
*   *Перед збігом приголосних:* Вітаю **зі** святом! / Розмовляли **зі** мною. (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0126)
*   *З числівниками:* Витратив **зо** три сотні. (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0126)

**Група 4: Інші засоби (В1+)**
*   *Вставний голосний:* Зійшли**сь** учні. (Джерело: 10-klas-ukrmova-glazova-2018_s0100)
*   *Спрощення:* Не брудни криниці, бо **ще** схочеш водиці напитись. (тут `ще` замість `іще` після `бо`) (Джерело: 7-klas-ukrmova-zabolotnyi-2024_s0204)
*   *Варіанти часток:* Хоч мороз припікає, зате комарів немає. (`хоч` замість `хоча` перед голосною) (Джерело: 7-klas-ukrmova-zabolotnyi-2024_s0204)

## Рекомендації для вправ (Activity Concepts)

*   **Phase 1: Ізольований вибір (A2)**
    *   **Вправа "Вибери правильний варіант".** Надаються речення з дужками: `Подарунок (з/із/зі) Сумщини`, `Ярмарок (у/в) Полтаві`. Студент має обрати правильну форму, керуючись безпосереднім фонетичним контекстом. (За мотивами Джерела: 8-klas-ukrmova-zabolotnyi-2025_s0026).
    *   **Вправа "Класифікація".** Дати список словосполучень і попросити розкласти їх у дві колонки: 1) з `у` та `і`; 2) з `в` та `й`. (За мотивами Джерела: 10-klas-ukrajinska-mova-avramenko-2018_s0117).

*   **Phase 2: Редагування та перебудова (A2+)**
    *   **Вправа "Виправ помилки".** Надати текст з навмисними порушеннями милозвучності (як у "поганих" прикладах з підручників) і попросити відредагувати його, обґрунтовуючи виправлення. Напр.: `Шановні українки і українці! Вітаємо вас з святом! Нехай в ваших родинах...` (Джерело: 5-klas-ukrmova-litvinova-2022_s0179).
    *   **Вправа "Об'єднай речення".** Дати пари простих речень і завдання об'єднати їх у складне, правильно вживаючи сполучники `і`/`й`/`та`. Напр.: `Надійшла весна. У саду зацвіли яблуні.` -> `Надійшла весна, і в саду зацвіли яблуні.`

*   **Phase 3: Вільне мовлення та аудіювання (B1)**
    *   **Вправа "Склади розповідь".** Дати набір слів та фраз і попросити скласти коротку зв'язну розповідь, звертаючи особливу увагу на плавне поєднання слів за допомогою `у/в`, `і/й`, `з/із/зі`.
    *   **Аудіювання.** Прослухати короткий текст (пісню, вірш) і виписати всі випадки чергувань. Це тренує "чуття мови". (За мотивами Джерела: 5-klas-ukrmova-zabolotnyi-2023_s0120).
    *   **Усні відповіді.** Ставити запитання, які провокують відповіді з використанням цих прийменників/сполучників. Наприклад: "Де ти був учора ввечері?", "З ким ти розмовляв?". Оцінювати не лише зміст, а й милозвучність відповіді.

## Зв'язки з іншими темами

*   **Попередні теми (Prerequisites):**
    *   **Фонетика:** Учень повинен чітко розрізняти голосні та приголосні звуки, а також розуміти поняття збігу звуків.
*   **Наступні теми (Enabled Topics):**
    *   **Синтаксис:** Правильне вживання прийменників та сполучників є основою для побудови будь-яких речень, особливо складних.
    *   **Стилістика:** Розуміння милозвучності є першим кроком до відчуття ритму та мелодики української мови, що важливо для читання поезії та написання виразних текстів.
    *   **Словотвір:** Явище спрощення в групах приголосних (напр., `честь-чесний`) напряму пов'язане з милозвучністю і є частиною цієї великої теми. (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0110).
