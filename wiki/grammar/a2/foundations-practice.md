## Як це пояснюють у школі (How Schools Teach This)

На рівні A2 українські освітні підходи зосереджуються на практичному застосуванні граматики, що вже була введена на A1, та розширенні її до більш складних, але частотних конструкцій. Основний фокус робиться на **видах дієслів** (доконаний/недоконаний) та розширенні вжитку **знахідного відмінка**.

1.  **Вид дієслова (Verbal Aspect):** Це ключова тема A2. Замість глибокого теоретичного аналізу, підручники та подкасти для учнів вводять цю концепцію через протиставлення (Source 4, 28).
    *   **Недоконаний вид (Imperfective):** Позначає процес, повторювану або звичайну дію. Часто використовується зі словами `завжди`, `часто`, `зазвичай`, `кожного дня` (Source 4). Приклад: `Я там завжди **купую** овочі` (Source 4).
    *   **Доконаний вид (Perfective):** Позначає завершену, одноразову дію або результат. Часто використовується для послідовності дій у минулому або для вираження наміру зробити щось один раз (Source 4, 11). Приклад: `Де можна **купити** фрукти й овочі?` (Source 4).
    У шкільній практиці пари дієслів часто вводяться разом, щоб учні запам'ятовували їх як єдине ціле: `купувати/купити`, `робити/зробити`, `читати/прочитати`.

2.  **Знахідний відмінок (Accusative Case):** На A2 вивчення виходить за межі прямого додатка.
    *   **Час:** Знахідний відмінок використовується з прийменниками `у/в` для позначення днів тижня: `у понеділок`, `у вівторок`, але `у середу`, `у п'ятницю` (Source 5). Це подається як правило для запам'ятовування.
    *   **Напрямок руху:** З прийменниками `в/у` та `на` для позначення мети або напрямку руху, на відміну від місцевого відмінка, що позначає місцезнаходження (Source 5). Приклад: `Ми підемо **в бар**` (напрямок) vs. `Ми сидимо **в барі**` (місце).
    *   **Особові займенники:** Вводиться відмінювання особових займенників у знахідному відмінку (`мене`, `тебе`, `його`) та їхні форми з прийменниками, що вимагають генітива, де з'являється приставний `н-` (`у нього`, `без неї`) (Source 7).

3.  **Числові конструкції:** Учні вчаться правильно узгоджувати іменники з числівниками, особливо з такими словами, як `кілограм` або `метр`. Правило, яке відпрацьовується: `2, 3, 4 + називний відмінок множини` (`два кілограми`), `5+ + родовий відмінок множини` (`п'ять кілограмів`) (Source 8, 40).

4.  **Незмінювані іменники:** Наголошується, що деякі слова іншомовного походження (`метро`, `кіно`, `кафе`, `таксі`) не змінюють своєї форми в різних відмінках (Source 9). Це подається як просте правило для запам'ятовування.

5.  **Зменшувально-пестливі форми (Diminutives):** Хоча це більше стосується лексики, ніж граматики, ця тема часто з'являється на A2, щоб зробити мову учнів більш природною та емоційно забарвленою. Учні знайомляться з найпоширенішими суфіксами: `-ик`, `-очка`, `-еньк-` (Source 1). Приклад: `мандарини -> мандаринки`, `борщ -> борщик`.

## Повна парадигма (Full Paradigm)

### 1. Види дієслів (Verbal Aspect)

Формування доконаного виду від недоконаного відбувається кількома шляхами:

**A. Додавання префікса (найпоширеніший спосіб)** (Source 28)

| Недоконаний (Imperfective) | Префікс | Доконаний (Perfective) | Значення |
| :------------------------ | :------ | :--------------------- | :------- |
| робити                    | з-        | **з**робити            | to do    |
| писати                    | на-       | **на**писати           | to write |
| читати                    | про-      | **про**читати          | to read  |
| готувати                  | при-      | **при**готувати        | to cook  |
| їсти                      | з'-       | **з'**їсти             | to eat   |
| телефонувати              | за-       | **за**телефонувати     | to call  |

**Б. Зміна суфікса та/або основи (часто з чергуванням)** (Source 3, 11)

| Недоконаний (Imperfective) | Доконаний (Perfective) | Тип зміни                            |
| :------------------------ | :--------------------- | :----------------------------------- |
| вивч**а**ти               | в**и**вчити            | `-а-` → `-и-` + зміна наголосу        |
| замовл**я**ти             | зам**о**вити           | `-я-` → `-и-` + вставний `-л-` + наголос |
| допомаг**а**ти            | допомог**ти**          | Випадання суфікса `-а-`              |
| прикраш**а**ти            | прикрас**и**ти         | Чергування `ш` → `с` + суфікс `-и-`   |
| запрош**ува**ти           | запрос**и**ти          | `-ува-` → `-и-` + чергування `ш` → `с` |
| куп**ува**ти              | куп**и**ти             | Суфікс `-ува-` зникає                 |
| розпові**д**ати           | розповіс**ти**         | Чергування `д` → `с`                 |

### 2. Знахідний відмінок з прийменниками часу (Дні тижня)

Правило: Masculine дні (`понеділок, вівторок, четвер`) не змінюються. Feminine (`середа, п'ятниця, субота`) та neuter (`неділя`) змінюють закінчення на `-у`. (Source 5)

| День тижня | Питання | Приклад у знахідному відмінку |
| :---------- | :------ | :----------------------------- |
| Понеділок   | Коли?   | **у понеділок**                |
| Вівторок    | Коли?   | **у вівторок**                 |
| Середа      | Коли?   | **у серед**у**                 |
| Четвер      | Коли?   | **у четвер**                   |
| П'ятниця    | Коли?   | **у п'ятниц**ю**              |
| Субота      | Коли?   | **у субот**у**                 |
| Неділя      | Коли?   | **у неділ**ю**                 |

### 3. Знахідний та Родовий відмінки особових займенників

| Називний | Знахідний (кого?) | Родовий (з прийменником) | Приклад (Знахідний)            | Приклад (Родовий)             |
| :------- | :---------------- | :----------------------- | :----------------------------- | :---------------------------- |
| я        | **мене**          | без **мене**             | Я хочу запросити **тебе** (Source 7) | Приходь, бо без **тебе** буде сумно (Source 7) |
| ти       | **тебе**          | для **тебе**             |                                |                               |
| він/воно | **його**          | у **нього**              | Всі будуть раді бачити **тебе** (Source 7) | Я був у **тебе** недавно (Source 7) |
| вона     | **її**            | біля **неї**             |                                |                               |
| ми       | **нас**           | про **нас**              |                                |                               |
| ви       | **вас**           | до **вас**               |                                |                               |
| вони     | **їх**            | без **них**              |                                |                               |

**Важливо:** Приставний `н-` з'являється у формах родового відмінка займенників `він, вона, вони` після прийменників. (Source 7)

## Частотність і пріоритети

1.  **Вид дієслова є найвищим пріоритетом на A2.** Без розуміння різниці між процесом (недоконаний) та результатом (доконаний) неможливо правильно будувати речення про минулі події або майбутні плани. Усному мовленню притаманна постійна зміна видів.
2.  **Знахідний відмінок для напрямку та часу.** Конструкції `їхати в Україну`, `йти в магазин`, `у понеділок`, `у п'ятницю` є надзвичайно частотними в щоденному спілкуванні про плани та розклад (Source 5, 13).
3.  **Особові займенники `тебе`, `мене`, `його`**. Ці форми є базовими для будь-якої розмови. Форми з приставним `н-` (`у нього`, `до неї`) також є високочастотними.
4.  **Зменшувально-пестливі форми (`-ик`, `-очка`).** Важливі для розуміння живого, неформального мовлення (наприклад, на ринку чи в розмові з друзями), але на етапі A2 важливіше розпізнавання, а не активне вживання (Source 1).
