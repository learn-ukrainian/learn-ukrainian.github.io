## Як це пояснюють у школі

В українській шкільній програмі поняття виду дієслова вводять у 6-му класі [S3, S2]. Основний принцип пояснення полягає в розрізненні дії за її завершеністю чи незавершеністю. У підручниках послідовно використовується метод запитань: дієслова **недоконаного виду** (НД) відповідають на питання *«що робити?»*, а дієслова **доконаного виду** (ДВ) — на питання *«що зробити?»* [S1, S3, S5]. Цей підхід є центральним і повторюється в підручниках для 7-го [S4, S5] та 10-го класів [S1, S8].

Навчальний процес будується за такою логікою:
1.  **Введення поняття:** Пояснення різниці між тривалою, незавершеною дією (НД) і завершеною, результативною дією (ДВ) [S1, S5]. Часто це ілюструється парними малюнками або реченнями, де контекст підкреслює різницю (напр., *«Я розв'язував задачу...»* проти *«Я розв'язав задачу!»* [S5]).
2.  **Формальні ознаки:** Учні знайомляться з основними способами творення видових пар: за допомогою префіксів, суфіксів, чергування звуків в основі, зміни наголосу та зміни основи (суплетивізм) [S1, S3, S5].
3.  **Зв'язок із часами:** Одразу наголошується на важливому обмеженні: дієслова доконаного виду не мають форми теперішнього часу [S1, S8]. Усі дієслова в теперішньому часі завжди недоконаного виду [S4]. Це ключова відмінність, на якій будуються подальші вправи. Майбутній час, у свою чергу, має три форми, які безпосередньо залежать від виду дієслова [S10].
4.  **Окремі категорії:** Вводяться поняття **двовидових дієслів** (одна форма може означати обидва види залежно від контексту, напр., *атакувати, гарантувати*) та **одновидових дієслів** (мають форму лише одного виду, напр., *прагнути* (НД), *розговоритися* (ДВ)) [S1, S3, S5].

Термінологія, що використовується в підручниках, є сталою: *доконаний вид*, *недоконаний вид*, *видові пари*, *одновидові* та *двовидові дієслова* [S1, S3, S4, S5].

## Повна парадигма

Видові пари — це фундаментальна риса української граматики. Більшість дієслів існують у таких парах [S3, S11]. Нижче наведено 30 пріоритетних пар для рівня А2, згрупованих за основним способом творення.

| Недоконаний вид (НД) _(що робити?)_ | Доконаний вид (ДВ) _(що зробити?)_ | Спосіб творення |
| :--- | :--- | :--- |
| **Префіксація** (найпоширеніший спосіб [S11]) | | |
| робити | **з**робити | Префікс `з-`: `робити → з-робити` [S3] |
| писати | **на**писати | Префікс `на-`: `писати → на-писати` [S1, S5] |
| читати | **про**читати | Префікс `про-`: `читати → про-читати` [S1, S3] |
| бачити | **по**бачити | Префікс `по-`: `бачити → по-бачити` [S11] |
| слухати | **по**слухати | Префікс `по-`: `слухати → по-слухати` [S11] |
| думати | **по**думати | Префікс `по-`: `думати → по-думати` [S11] |
| снідати | **по**снідати | Префікс `по-`: `снідати → по-снідати` [S11] |
| обідати | **по**обідати | Префікс `по-`: `обідати → по-обідати` [S11] |

| готувати | **при**готувати | Префікс `при-`: `готувати → при-готувати` [S11] |
| платити | **за**платити | Префікс `за-`: `платити → за-платити` [S11] |
| телефонувати | **за**телефонувати | Префікс `за-`: `телефонувати → за-телефонувати` [S11] |

| **Суфіксація та чергування** | | |
| допомагати | допомогти | Суфікс `-ага-` (НД) зникає, чергування `г/ж`: `допомогти → допомог-а-ти` |
| вивчати | вивчити | Суфікс `-а-` (НД) vs. `-и-` (ДВ), чергування `ч/т`: `вивчити → вивч-а-ти` |
| запитувати | запитати | Суфікс `-ува-` (НД) vs. `-а-` (ДВ): `запитати → запит-ува-ти` |
| розповідати | розповісти | Суфікс `-а-` (НД) vs. відсутність, чергування `д/ст`: `розповісти → розповід-а-ти` |
| закінчувати | закінчити | Суфікс `-ува-` (НД) vs. `-и-` (ДВ): `закінчити → закінч-ува-ти` |
| повторювати | повторити | Суфікс `-юва-` (НД) vs. `-и-` (ДВ): `повторити → повтор-юва-ти` [S3] |
| витирати | витерти | Суфікс `-а-` (НД) vs. відсутність, чергування `и/е`: `витерти → витир-а-ти` [S5] |
| збирати | зібрати | Суфікс `-а-` (НД) vs. відсутність, чергування `и/і`: `зібрати → збир-а-ти` [S5] |
| починати | почати | Суфікс `-ина-` (НД) vs. відсутність: `почати → поч-ина-ти` |
| **Зміна наголосу** | | |
| розрі́зати | розріза́ти | Наголос на корені (НД) vs. наголос на суфіксі (ДВ): `розрі́зати → розріза́ти` [S1, S5] |
| заси́пати | засипа́ти | Наголос на префіксі (НД) vs. наголос на суфіксі (ДВ): `заси́пати → засипа́ти` [S5] |
| **Суплетивізм (зміна основи)** | | |
| брати | **взяти** | Повна зміна основи: `бра-` vs. `взя-` [S1, S3, S5] |
| говорити | **сказати** | Повна зміна основи: `говор-` vs. `сказ-` (або `казати` [S11]) |
| шукати | **знайти** | Повна зміна основи: `шука-` vs. `знай-` [S1, S3, S5] |
| ловити | **піймати** | Повна зміна основи: `лов-` vs. `пійма-` [S1, S3, S5] |

**Обмеження:**
*   Дієслова доконаного виду **не мають** форми теперішнього часу [S1, S4, S8].
*   Складена форма майбутнього часу (`буду + інфінітив`) утворюється **тільки** від дієслів недоконаного виду [S10].
*   Складна форма майбутнього часу (`-му, -меш`) утворюється **тільки** від дієслів недоконаного виду [S10].
*   Проста форма майбутнього часу утворюється **тільки** від дієслів доконаного виду (зазвичай шляхом префіксації) [S8, S10].

## Частотність і пріоритети

Для рівня А2 пріоритетним є засвоєння найуживаніших пар, що позначають базові щоденні дії. Найбільш частотним і продуктивним способом творення доконаного виду є **префіксація** [S2, S11].

**Пріоритет 1: Префіксальні пари з `по-`, `з-/с-`, `на-`, `про-`**
Ці пари є найбільш прозорими за значенням (додавання префікса = завершення дії).
*   **робити / зробити:** основа будь-якої розмови про плани та результати.
*   **писати / написати, читати / прочитати:** ключові для навчання.
*   **бачити / побачити, слухати / послухати, думати / подумати:** базові дієслова сприйняття та мислення.
*   **їсти / з'їсти, снідати / поснідати, готувати / приготувати:** тема «Їжа».

**Пріоритет 2: Суплетивні пари**
Ці пари є нерегулярними, але надзвичайно частотними. Їх потрібно завчити як окремі лексичні одиниці [S3].
*   **брати / взяти**
*   **говорити / сказати**
*   **шукати / знайти**

**Пріоритет 3: Пари із суфіксацією та чергуванням**
Ці пари часто позначають процес і результат.
*   **допомагати / допомогти**
*   **починати / почати**
*   **закінчувати / закінчити**

Пари, що утворюються зміною наголосу, є найменш пріоритетними для активного засвоєння на рівні А2, але важливі для розпізнавання на слух.

## Типові помилки L2 (англомовні учні)

Англійська мова не має граматичної категорії виду, тому англомовні учні часто роблять передбачувані помилки, намагаючись перенести логіку англійських часів на українську систему.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| *«Вчора я **читав** цю книгу і тепер знаю сюжет».* | *«Вчора я **прочитав** цю книгу і тепер знаю сюжет».* | Для одноразової завершеної дії в минулому, що має результат, потрібен доконаний вид. Англійське "I read" може означати і процес, і результат, що призводить до плутанини. |
| *«Я люблю **прочитати** детективи».* | *«Я люблю **читати** детективи».* | Для вираження загальної вподобання або повторюваної дії використовується недоконаний вид. Доконаний вид *прочитати* вказує на конкретну, одноразову дію [S11]. |
| *«Завтра я **буду зробити** домашнє завдання».* | *«Завтра я **зроблю** домашнє завдання»* або *«Завтра я **буду робити** домашнє завдання».* | Складена форма майбутнього часу `буду + інфінітив` вживається тільки з інфінітивом **недоконаного** виду [S10]. Для доконаного виду існує тільки проста форма майбутнього часу. |
| *«Що ти **зробив**?»* (у значенні «What were you doing?») | *«Що ти **робив**?»* | Англійське "What did you do?" може запитувати як про результат, так і про процес. В українській мові це чітко розрізняється: `зробив` (ДВ) — питає про результат, `робив` (НД) — про тривалу дію в минулому. |
| *«Він **запитав** мене багато питань, але я не відповіла».* | *«Він **пи́тав** (або **запитував**) мене багато питань, але я не відповіла».* | Якщо дія повторювалась або була тривалою (він ставив питання одне за одним), потрібен недоконаний вид. `Запитав` (ДВ) означало б, що він поставив одне питання (або серію як одну дію) і все. |

## Деколонізаційні застереження

Через історичні умови та тривалу русифікацію, учні, знайомі з російською мовою, можуть робити хибні припущення щодо українського виду. Важливо наголошувати, що українська граматична система є самостійною.

1.  **Різні префікси:** Не завжди префікс, що утворює видову пару в російській мові, буде таким самим в українській.
    *   укр. **чекати (НД) — почекати (ДВ)** vs. рос. *ждать (НСВ) — **подо**ждать (СВ)*.
    *   укр. **дякувати (НД) — подякувати (ДВ)** vs. рос. *благодарить (НСВ) — **по**благодарить (СВ)*.

2.  **Різні суплетивні форми:** Основи в недоконаному виді можуть відрізнятися.
    *   укр. **шукати (НД) — знайти (ДВ)** vs. рос. ***искать** (НСВ) — найти (СВ)*.
    *   укр. **брати (НД) — взяти (ДВ)** vs. рос. *брать (НСВ) — взять (СВ)*. Хоча тут схоже, але відмінності в інших парах є системними.

3.  **Різний статус дієслова:** Дієслово, що є одновидовим в одній мові, може мати пару в іншій. Також двовидові дієслова не завжди збігаються. Наприклад, укр. *слідувати* є двовидовим, але його відповідники в російській мають різні форми.

4.  **Фонетичні та морфологічні відмінності:** Навіть коли префікси однакові, основи слів і чергування в них підпорядковуються законам української, а не російської мови (напр., `з'їсти` з апострофом [S11], `з-` перед дзвінкими, `с-` перед «кафе птах» [S5]).

**Методична настанова:** Завжди вивчати українські видові пари як самостійні лексичні одиниці. Не намагатися «вивести» українську форму з російської. Спиратися на авторитетні українські джерела [S1, S3, S5], а не на аналогії з іншими мовами.

## Природні приклади

Приклади згруповано за типовими ситуаціями вживання виду.

**Група 1: Завершена дія в минулому (результат) — Доконаний вид**
1.  Я **написала** твір. [S5] (Дія завершена, є результат — написаний твір).
2.  Дмитро **помив** посуд. [S5] (Дія завершена, посуд чистий).
3.  Ольга **навчилася** грати на гітарі. [S5] (Є результат — вона тепер уміє грати).
4.  Учора я **побував** у гостях у бабусі. [S3] (Візит завершено).

**Група 2: Процес, тривала або незавершена дія — Недоконаний вид**
1.  Ольга **вчилася** грати на гітарі. [S5] (Вона була в процесі навчання, результат невідомий).
2.  Я **писала** твір. [S5] (Я була в процесі написання).
3.  Учні цілий урок **розв'язували** завдання, але не могли розв’язати правильно. [S3] (Процес тривав, але не дав результату).
4.  Сьогодні мій сусід довго **косив** траву. [S3] (Дія тривала в часі).

**Група 3: Майбутній час**
1.  Діти **намалюють** картину. [S5] (ДВ, проста форма майбутнього — дія завершиться в майбутньому).
2.  Ми **займатимемось** волонтерством на канікулах. [S10] (НД, складна форма — це буде тривалий процес у майбутньому).
3.  Завтра я **буду надсилати** листи. [S10] (НД, складена форма — процес у майбутньому).
4.  Наступного року режисер **зніме** байопік. [S10] (ДВ, проста форма — результат у майбутньому).

**Група 4: Узагальнення, повторювана дія — Недоконаний вид**
1.  Я дуже люблю **читати**! [S11] (Загальна вподобання).
2.  Наша вчителька щодня **питає** правило. [S3] (Дія повторюється щодня).
3.  На вихідних татові подобається **робити** барбекю. [S11] (Повторювана дія, звичка).

## Рекомендації для вправ

Автор-письменник має створювати вправи, що ведуть учня від простого розпізнавання до вільного використання.

*   **Фаза 1: Розпізнавання та групування.**
    *   **Вправа:** Дати список дієслів і попросити розділити їх на дві колонки: НД і ДВ [S4].
    *   **Вправа:** Дати речення і попросити визначити вид підкресленого дієслова, поставивши до нього питання «що робити?» / «що зробити?» [S1, S3].

*   **Фаза 2: Утворення пар.**
    *   **Вправа:** Дати дієслово НД і попросити утворити ДВ за допомогою префікса (напр., *робити → зробити*) [S3, S5].
    *   **Вправа:** Дати дієслово ДВ і попросити утворити НД за допомогою суфікса (напр., *розказати → розказувати*) [S1].
    *   **Вправа:** «Знайди пару». Дати два стовпчики дієслів НД і ДВ у довільному порядку і з'єднати їх.

*   **Фаза 3: Контекстний вибір.**
    *   **Вправа:** Дати речення з пропуском і двома варіантами в дужках (НД/ДВ). Учень має обрати правильний варіант, аналізуючи контекст (одноразова дія, процес, результат, повторювана дія) [S3].
        *   *Приклад:* «Він довго (писав / написав) листа і нарешті (писав / написав) його».

*   **Фаза 4: Контрольоване продукування.**
    *   **Вправа:** Поставити запитання в ДВ, вимагаючи відповіді в ДВ. *Приклад:* «Ти вже **прочитав** статтю?» — «Так, я її вже **прочитав**» / «Ні, я її ще **читаю** / **буду читати**». [S3].
    *   **Вправа:** Дати короткий сценарій і попросити описати його двома реченнями, використовуючи НД для процесу і ДВ для результату. *Приклад:* Сценарій: "Мама готувала вечерю 2 години. О 7 вечора вечеря була готова". → *Мама довго **готувала** вечерю. О сьомій годині вона **приготувала** вечерю.*

## Зв'язки з іншими темами

*   **Передумови:**
    *   **Інфінітив:** Поняття виду вводиться через питання до інфінітива (`що робити?` / `що зробити?`) [S7].
    *   **Часи дієслів (основи):** Учень має знати про існування минулого, теперішнього і майбутнього часів.

*   **Наступні теми:**
    *   **Часи дієслів (поглиблено):** Категорія виду є ключем до розуміння трьох форм майбутнього часу [S10, S8]. Без розуміння виду неможливо правильно вживати `буду робити` / `робитиму` / `зроблю`.
    *   **Дієслова руху:** Це окрема складна підтема, де префікси не лише змінюють вид, а й додають просторове значення (в-, ви-, при-, за- тощо). Розуміння виду є основою для їх вивчення.
    *   **Дієприкметники та дієприслівники:** Творення цих форм також часто залежить від виду вихідного дієслова.
