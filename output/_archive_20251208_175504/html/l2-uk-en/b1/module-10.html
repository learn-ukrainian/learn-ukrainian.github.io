Arriving, exiting, and entering

Warm-up

The Moment When Everything Changes

You already know basic motion verbs ‚Äî —ñ—Ç–∏/—Ö–æ–¥–∏—Ç–∏, —ó—Ö–∞—Ç–∏/—ó–∑–¥–∏—Ç–∏. But how do you say "to come," "to go out," "to enter"?

The answer: prefixes. When you add a prefix to a motion verb, magic happens:

  1. The verb becomes perfective
  2. It gains a specific direction
  3. A new aspect pair is formed

This isn't a rule with exceptions ‚Äî it's an absolute law. No exceptions.

Did you know?

The prefix system in Slavic languages is one of the oldest in the world. These same prefixes (–ø—Ä–∏-, –≤–∏-, –≤-) existed in Proto-Slavic over 2000 years ago! Latin had a similar system (in-, ex-, ad-), but Romance languages lost it. Slavic languages preserved it. When you say "–ø—Ä–∏–π—Ç–∏," you're using the same structure your ancestors used thousands of years ago.

Presentation

Key Rule: Prefix = Perfective Aspect

When you add ANY prefix to a motion verb, it becomes PERFECTIVE.

Base (imperfective) + Prefix = Perfective
—ñ—Ç–∏ (–π—Ç–∏) –ø—Ä–∏- –ø—Ä–∏–π—Ç–∏ (to come/arrive) ‚úì pf
—ñ—Ç–∏ (–π—Ç–∏) –≤–∏- –≤–∏–π—Ç–∏ (to exit) ‚úì pf
—ñ—Ç–∏ (–π—Ç–∏) –≤-/—É- —É–≤—ñ–π—Ç–∏ (to enter) ‚úì pf

Then from the perfective, a new imperfective is formed:

–ø—Ä–∏–π—Ç–∏ (pf) ‚Üí –ø—Ä–∏—Ö–æ–¥–∏—Ç–∏ (impf)

Myth vs Fact

Myth: "You need to memorize which prefixed motion verbs are perfective and which aren't."

Fact: You don't need to memorize anything! All prefixed motion verbs are perfective. No exceptions. This is an automatic rule that always works.

Did you know?

The alternation –≤/—É is an ancient phonetic feature of Ukrainian. The rule is simple: if the previous word ends in a consonant, use —É (–≤—ñ–Ω —É–≤—ñ–π—à–æ–≤), if in a vowel ‚Äî –≤ (–≤–æ–Ω–∞ –≤–≤—ñ–π—à–ª–∞). This makes speech smoother. Ukrainian is one of few languages where this euphony is codified in grammar!
