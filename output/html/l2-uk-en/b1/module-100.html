What to Remember

Aspect:

  • Perfective = single, completed, result
  • Imperfective = duration, repetition, process
  • Negation generally uses imperfective
  • "–©–µ –Ω–µ" uses perfective

Motion:

  • Unidirectional = one way, now
  • Multidirectional = round trip, habitual
  • Prefixes add direction + perfectivize

Communication:

  • Hedge opinions: –ú–µ–Ω—ñ –∑–¥–∞—î—Ç—å—Å—è, –ú–æ–∂–ª–∏–≤–æ
  • Disagree politely: –Ø –Ω–µ –∑–æ–≤—Å—ñ–º –∑–≥–æ–¥–µ–Ω
  • Give reasons: —Ç–æ–º—É —â–æ, –æ—Å–∫—ñ–ª—å–∫–∏
