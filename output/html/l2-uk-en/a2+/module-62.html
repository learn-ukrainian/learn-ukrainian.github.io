### Beyond Motion Verbs

In Module 61, you learned prefixes with motion verbs. Now let's see how the same prefixes work with everyday action verbs!

The pattern continues:

  • –Ω–∞- + –ø–∏—Å–∞—Ç–∏ = –Ω–∞–ø–∏—Å–∞—Ç–∏ (to write ‚Üí to finish writing)
  • –ø—Ä–æ- + —á–∏—Ç–∞—Ç–∏ = –ø—Ä–æ—á–∏—Ç–∞—Ç–∏ (to read ‚Üí to read through)

üí° Key Insight

The same prefix often has the same core meaning with different verbs:

  • –ø—Ä–æ- = "through" ‚Üí –ø—Ä–æ—á–∏—Ç–∞—Ç–∏ (read through), –ø—Ä–æ–π—Ç–∏ (go through)
