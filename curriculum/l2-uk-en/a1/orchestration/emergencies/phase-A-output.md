# Дослідження: Emergencies

## State Standard Reference
§4.2.3.4: "Vocative case — address forms"
Alignment: Direct address for emergency personnel (Лікарю!, Поліціє!)

§4.2.4.2: "Imperative mood — 2nd person only at A1"
Alignment: Polite emergency requests in imperative forms («Скажіть, будь ласка», «Допоможіть!»).

§1.3.2: Ознайомлення з базовим викликом «Допоможіть!»
Alignment: Core communicative intention for asking for help and stating an emergency.

## Cultural Hooks
1. Ukraine is transitioning to the unified European 112 emergency number as a sign of digitalization and Euro-integration, while still maintaining 101, 102, and 103 for specific dispatches.
2. The "Дія" ecosystem: Ukraine's state-in-a-smartphone app provides digital documents and emergency services, demonstrating world-class digital resilience during crises.

## Common Learner Errors
1. «дзвонити поліцію» → «викликати поліцію» — Learners confuse "ringing/dialing" (дзвонити) with "summoning" (викликати) services.
2. «визивати швидку» → «викликати швидку» — "Визивати" is a Russism; standard Ukrainian strictly uses "викликати".
3. «Лікар!» → «Лікарю!» — Learners use the Nominative instead of the Vocative when directly calling out for help.

## Notes for Content Writing
- Decolonization: When addressing the Russism "визивати", do so by simply stating it is non-standard/incorrect and emphasizing the authentic Ukrainian "викликати". Do not frame it as "unlike Russian...".
- English Scaffolding: Since this is A1, use English explicitly to explain the mechanics of the Vocative case and polite Imperative forms before diving into Ukrainian examples.
- Sentences should remain short (max 8-10 words) for A1 level.
