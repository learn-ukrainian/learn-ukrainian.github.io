# Дослідження: The Cyrillic Code II

## State Standard Reference
§4.1.1: "Ukrainian alphabet" and §4.1.4: "Vowel and consonant sounds"
Alignment: Introduces 7 letters of the Cyrillic alphabet (К, И, І, Р, В, Т, Е) and fundamental vowel/consonant distinctions, directly addressing the foundational literacy prerequisite for A1.

## Cultural Hooks
1. The dotted "І" is deeply meaningful in Ukraine; it is a recognizable symbol of Ukrainian typography and linguistic identity, distinguishing the language visually from Russian.
2. The word "метро" (metro) is a recognizable internationalism that immediately helps learners navigate major Ukrainian cities like Kyiv, Kharkiv, and Dnipro.

## Common Learner Errors
1. Visual interference with 'Р' → Pronouncing it as English /p/ instead of the rolled /r/. This happens because the shape is identical to Latin P.
2. Visual interference with 'В' → Pronouncing it as English /b/ instead of /v/. This happens because the shape is identical to Latin B.
3. Merging 'И' and 'І' → English speakers often struggle to produce the lower, relaxed 'И' and default to the familiar high 'І' (/ee/) sound, losing a critical phonemic distinction.
