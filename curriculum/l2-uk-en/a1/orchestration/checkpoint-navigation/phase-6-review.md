## Critical Issues Found

### Issue 1: Immersion Overshoot (Safety Violation)
- **Location**: Entire Module
- **Original**: «Ми завершуємо блок А1.2. Це був довгий шлях. Це був важливий етап. Ми вивчили багато нового.»
- **Problem**: This is the introduction to an A1.2 module. A student at this level likely knows ~150-200 words. Using B1-level narrative text for *instructions* and *explanations* makes the content inaccessible and discouraging. The plan requires English scaffolding.
- **Fix**: Rewrite all meta-text, introductions, grammar explanations, and instructions in English. Only examples and dialogue should be in Ukrainian.

### Issue 2: Abstract Metaphors (LLM Fingerprint/Pedagogy)
- **Location**: Section "Огляд: Навігація" / "Навичка 2" / "Навичка 4"
- **Original**: «Українська мова як будинок...» / «Прийменник — це якір...» / «Це як танець...»
- **Problem**: Using complex metaphors ("foundation/walls/cracks", "anchor", "dance") to explain grammar adds cognitive load. Beginners need concrete rules, not philosophical comparisons. This is a hallmark of LLM "creative writing" filler.
- **Fix**: Remove metaphors. Explain the rules directly. E.g., "Prepositions connect words. Without them, the sentence falls apart." instead of "Preposition is an anchor."

### Issue 3: Advanced Vocabulary (Level Appropriateness)
- **Location**: Section "Огляд: Навігація"
- **Original**: «Вітаємо на екваторі рівня А1!... Тріщини треба залатати зараз.»
- **Problem**: Words like "екватор" (equator), "тріщини" (cracks), "залатати" (patch up) are essentially useless for a beginner trying to ask for directions. They create noise.
- **Fix**: Simplify to standard A1 vocabulary or switch to English.

### Issue 4: Robotic Staccato Syntax (Language Quality)
- **Location**: Section "Огляд: Навігація"
- **Original**: «Спочатку ми вчили назви предметів. Це Називний відмінок. Потім ми вчили взаємодію. Ми бачимо об'єкти. Це Знахідний відмінок.»
- **Problem**: The sentence structure is unnatural, repetitive, and robotic (`Subject-Verb-Object. This is X.`). It feels like an AI trying too hard to be "simple" but ending up sounding unnatural.
- **Fix**: Use natural flowing English for these summaries.
