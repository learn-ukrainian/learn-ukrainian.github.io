## Critical Issues Found

### Issue 1: Content-Activity Gap — "фото" Never Taught

- **Location**: Activities file lines 184-185 (anagram) and lines 207-208 (match-up); Vocabulary file line 50
- **Original**: The word **фото** appears in:
  - Anagram activity: `answer: фото` / `scrambled: ф о т о`
  - Match-up activity: `left: фото` / `right: photo`
  - Vocabulary file: `lemma: фото`
- **Problem**: The word "фото" is NEVER introduced, used, or mentioned anywhere in the content file. Students encounter it in activities without ever seeing it in the lesson. For Module 1 absolute beginners, this is disorienting — they will scramble letters of a word they've never seen.
- **Fix**: Either (a) add "фото" to the cognate drill table in Section «Практика: Читаємо міжнародні слова» (e.g., `| **фото** | [ˈfɔtɔ] | photo | **зробити фото** (take a photo) |`), or (b) remove "фото" from activities and vocabulary.

### Issue 2: Untaught Grammar Forms in Examples Exceed A1.1 Scope

- **Location**: Lines 193, 200, 206, 222 in Section «Практика: Читаємо міжнародні слова»
- **Original**: «У банку» (line 193), «Моя мама найкраща.» (line 200), «Так, чудовий план.» (line 206), «каву» (line 222)
- **Problem**: These phrases use locative case (банку), superlative adjective (найкраща), untaught adjective (чудовий), and accusative case (каву) — all grammar concepts taught in later modules. For Module 1 where the learner just started recognizing letters, these complex forms may trigger anxiety: "Am I supposed to understand this grammar?"
- **Fix**: Replace with simpler constructions using only taught vocabulary: (a) Line 200: change «Моя мама найкраща.» to «Це моя мама.» (b) Line 206: change «Так, чудовий план.» to «Так! Кафе!» (c) Line 222: rephrase to avoid accusative — "**Кафе** is where you sit; **кава** is what you order."

### Issue 3: Activity-Content Inconsistency for О Reference Sound

- **Location**: Activities file line 10; Content file line 55
- **Original**: Content: "Similar to the 'o' in 'd**o**g' or '**o**ff.'" (line 55). Activity: `right: o (more)` (activities line 10).
- **Problem**: The reference words are inconsistent. While both "dog" and "more" contain [ɔ], a beginner who memorized "О = dog/off" from the lesson will be confused when the activity says "О = more". Consistency builds confidence at A1.
- **Fix**: Change activity line 10 from `right: o (more)` to `right: o (off)` to match the content.

### Issue 4: Vocabulary Form Mismatch — друг vs друзі

- **Location**: Content line 205; Vocabulary file line 78
- **Original**: Content uses «друг» (singular): "You see a friend (**друг**)." Vocabulary lists «друзі» (plural): `lemma: друзі`.
- **Problem**: The vocabulary file should list the lemma form actually used in the content. "Друг" is the singular nominative (what a dictionary would list); "друзі" is the nominative plural. Since the content introduces the singular form, the vocabulary should match.
- **Fix**: Change vocabulary entry from `lemma: друзі` to `lemma: друг` with translation "friend" (singular).
