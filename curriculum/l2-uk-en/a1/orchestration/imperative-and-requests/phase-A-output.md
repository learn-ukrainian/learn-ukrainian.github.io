# Дослідження: Imperative and Requests

## State Standard Reference
§1.1: "Сприймає прості усні вказівки/інструкції та адекватно реагує на них" (Understands simple oral directions/instructions and responds adequately).
Alignment: This module provides the foundational grammar for giving and understanding instructions, requests, and commands, which is heavily utilized in classroom environments and daily life.

## Vocabulary Frequency
| Word | Notes | Key collocations |
|------|-------|------------------|
| читай / читайте | Regular formation, imperfective | читай текст, читайте вголос |
| прочитай / прочитайте | Perfective counterpart, highly frequent in textbooks | прочитай і розкажи, прочитай завдання |
| пиши / пишіть | Regular formation | пиши правильно, пишіть записку |
| скажи / скажіть | Very common for requests | скажіть, будь ласка; скажи мені |
| дай / дайте | Irregular stem | дайте каву, дай ручку |
| дивись / дивіться | Reflexive ending (-сь/-ся) | дивись сюди, дивіться на дошку |
| слухай / слухайте | Regular formation | слухай уважно, слухайте вчителя |

## Cultural Hooks
1. **Classroom Instructions**: As seen in native Ukrainian textbooks, the imperative is the default mode of instruction for children and students (e.g., *Прочитай і розкажи*, *Розгляньте малюнки*, *Напиши записку*). This creates a direct bridge to authentic learning materials.
2. **Proverbs with Imperatives**: Ukrainian folklore is rich with imperative advice, such as "Вік живи — вік учись" (Live a century, learn a century) and "Сім разів відмір, один раз відріж" (Measure seven times, cut once).
3. **Politeness Marker**: Adding "будь ласка" (please) is culturally essential when turning an imperative command into a polite request (e.g., *Скажіть, будь ласка*).

## Common Learner Errors
1. **Using the infinitive instead of imperative** → *Ти читати* instead of *Читай*. English relies on the base form of the verb for imperatives (You read → Read!), which leads learners to wrongly assume the Ukrainian infinitive works the same way.
2. **Dropping the formal/plural marker (-те)** → Addressing an elder or a group as *скажи* instead of *скажіть*. This is a critical sociolinguistic error that sounds disrespectful.
3. **Ignoring aspect** → Using imperfective (e.g., *читай*) when a single, completed action is requested (*прочитай*). Textbook exercises specifically target this by using perfective imperatives for concrete tasks (*Прочитай текст. Визнач...*).

## Teaching Strategy
- **Discovery exercise**: Present a short dialogue of a teacher giving instructions to a class (*Читайте текст! Пишіть!*), followed by instructions to a single student (*Читай текст! Пиши!*). Ask learners to identify the difference between the verb endings and deduce the rule for singular vs. plural/formal.
- **Best context**: A classroom setting or a café/market where the learner has to ask for things (*Дайте, будь ласка, каву*).
- **Contrastive pairs**: *Читай* (informal/singular) vs. *Читайте* (formal/plural) to highlight the social and numerical difference.

## Notes for Content Writing
- Focus on the practical application of imperatives. The provided textbook excerpts clearly show that imperatives are not just theoretical; they are the language of action (*Роз’єднай*, *Досліджуй*, *Міркуй*).
- Highlight the use of perfective vs. imperfective in a simple way (e.g., *прочитай* for "read this specific thing now" vs. *читай* for "read in general").
- Ensure all imperative examples include the correct stress marks to aid pronunciation, as stress can sometimes shift in the imperative mood.
