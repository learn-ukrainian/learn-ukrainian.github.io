## Critical Issues Found

### Issue 1: Outdated Cultural Fact (Calendar Reform)
- **Location**: Section "Культурний контекст: Друзі та імена" / Subsection "Name Days (Іменини)"
- **Original**: «If your name is **Андрій**, your name day is December 13th.»
- **Problem**: Ukraine switched to the New Julian calendar in September 2023. St. Andrew's Day is now celebrated on November 30th. Providing the old date confuses learners interacting with modern Ukraine.
- **Fix**: Change «December 13th» to «November 30th».

### Issue 2: Confusing Linguistic Explanation (Vowels)
- **Location**: Section "Теорія: Як змінюються слова" / Subsection "The Fleeting Vowel (Біглий голос)"
- **Original**: «Sometimes, when we add an ending, a vowel inside the word "runs away" (flees). ... **Кіт** (cat): If we say "Кіта", it sounds okay. But historically, it changed to **Кота**.»
- **Problem**: The text defines "fleeting vowel" as one that "disappears/flees" (correct for *пес* → *пса*). However, it immediately uses *кіт* → *кота* as the primary example. In *кіт*, the vowel *і* does not "flee" (disappear); it changes to *о* (alternation). This contradicts the definition just given ("runs away") and confuses the learner.
- **Fix**: Rename the header to "**Vowel Magic: Fleeting & Changing**" and clarify: "Sometimes a vowel disappears (**Пес** → **Пса**), and sometimes it changes (**Кіт** → **Кота**)."

### Issue 3: Dangling Reference
- **Location**: Section "Культурний контекст: Друзі та імена" / Callout "Foreign Names"
- **Original**: «**David** (ends in consonant) acts like Taras: Я люблю **Девіда**.»
- **Problem**: The name "Taras" appears nowhere else in the module. The learner has no model for how "Taras" acts.
- **Fix**: Replace «acts like Taras» with «acts like Ivan» (a name used extensively in the lesson) or «acts like a regular noun».
