## Critical Issues Found

### Issue 1: Activity Ambiguity
- **Location**: Activities File / Fill-in "Complete the Sentences" / Item "___ людина."
- **Original**: Sentence: "___ людина." Answer: "Вона". Options: "Він", "Вона", ...
- **Problem**: "Людина" (person) is grammatically feminine, but can refer to a man or a woman. Syntactically, "Він — людина" (He is a human/person) is just as correct as "Вона — людина". The item is ambiguous without context (e.g., a picture or a preceding sentence).
- **Fix**: Replace "людина" with a clearly gendered noun like "жінка" (woman) or "дівчина" (girl) where "Вона" is the only logical answer among options. Or change the item to "___ гарна людина" (She is a good person) if adjectives were known, but they aren't.
- **Action**: Change sentence to: `___ жінка.` Answer: `Вона`.

### Issue 2: Content-Activity Gap (Unexplained Cultural Rule)
- **Location**: Activities File / Group-sort "Formal vs Informal" / Item "Talking to God"
- **Original**: Item: "Talking to God" -> Group: "Use Ти (Informal)"
- **Problem**: The content explains usage with "friends, family, children, pets". It does *not* mention that addressing God uses the informal "Ти" (which is a specific religious/cultural convention in Ukrainian, distinct from some other languages). A beginner might guess "Formal" out of respect.
- **Fix**: Either add a small note in the content (e.g., "Note: In prayers, Ukrainians address God as 'Ти'") OR remove this item from the activity as it wasn't taught.
- **Action**: Remove "Talking to God" from the activity items.

### Issue 3: Content-Activity Gap (Unexplained Item)
- **Location**: Activities File / Group-sort "Formal vs Informal" / Item "Talking to yourself in a mirror"
- **Original**: Item: "Talking to yourself in a mirror" -> Group: "Use Ти (Informal)"
- **Problem**: Not explicitly taught. While intuitive, it might confuse literal thinkers.
- **Fix**: Replace with a clear example taught in text, e.g., "Talking to a cat" (pets are mentioned).
- **Action**: Change to `Talking to your cat`.
