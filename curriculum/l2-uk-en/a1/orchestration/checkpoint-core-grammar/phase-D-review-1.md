## Critical Issues Found

### Issue 1: Pedagogy (Verb Conjugation Oversimplification)
- **Location**: Section "Навичка 3: Система дієслів"
- **Original**: «1. Group -ати (читати, знати)... 2. Group -ити (робити, говорити)»
- **Problem**: This rule implies all -ати verbs are Class 1 (conjugate with -є-). This leads to errors with common A1 verbs like *писати* (which is -ати but conjugates *пише*, not *писає*) or *хотіти* (*хоче*, not *хотіє*).
- **Fix**: Clarify that these are *examples* of patterns, or add a note about stem changes.

### Issue 2: Pedagogy (Future Tense vs Idiomatic Usage)
- **Location**: Section "Навичка 5: Практичні ситуації"
- **Original**: «Я буду **каву** і **салат**.»
- **Problem**: The student just learned «Future = БУДУ + Infinitive». Suddenly seeing «Я буду + Noun» is confusing without a note that this is an idiom for "I will have/order".
- **Fix**: Add a brief explanation or use the full phrase «Я буду замовляти...» / «Я буду їсти...» if keeping it strict, OR explain the idiom.
