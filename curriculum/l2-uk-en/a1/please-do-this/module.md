# Будь ласка, зроби це

You can now call a person naturally: **Олено! Тарасе! Друже!** This module
adds the next useful piece: asking someone to do something.

Ukrainian uses short command/request forms:

- **Читай, будь ласка.** - Read, please.
- **Читайте, будь ласка.** - Please read. (polite or plural)
- **Дай мені меню, будь ласка.** - Give me the menu, please.
- **Дайте рахунок, будь ласка.** - Please give the bill.

By the end, you can:

- recognize the school term **наказовий спосіб**;
- use common **ти** commands with one familiar person;
- use common **ви** commands with a teacher, server, stranger, or group;
- soften a command with **будь ласка**;
- keep the lesson practical: commands and polite requests, not a full verb
  paradigm.

:::tip[Polite habit]
For A1, build the whole line: **Олено, прочитай, будь ласка.** Address +
action + **будь ласка** is clear and friendly.
:::

## Діалоги

<!-- INJECT_ACTIVITY: act-1 -->

### У класі

```text
Учителька: Відкрийте підручники, будь ласка.
Студент: Вибачте, яку сторінку?
Учителька: Сторінку двадцять три. Читайте текст.
Учителька: Тепер пишіть. Напишіть три речення.
Студент: Можна запитати?
Учителька: Так, запитуйте!
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Учителька: Відкрийте підручники, будь ласка.** | Teacher: Open your textbooks, please. |
| **Студент: Вибачте, яку сторінку?** | Student: Excuse me, which page? |
| **Учителька: Сторінку двадцять три. Читайте текст.** | Teacher: Page twenty-three. Read the text. |
| **Учителька: Тепер пишіть. Напишіть три речення.** | Teacher: Now write. Write three sentences. |
| **Студент: Можна запитати?** | Student: May I ask? |
| **Учителька: Так, запитуйте!** | Teacher: Yes, ask! |

Classroom Ukrainian gives you many polite **ви** commands:
**відкрийте**, **читайте**, **пишіть**, **напишіть**, **запитуйте**. They are
normal instructions, especially with **будь ласка**.

:::note[Classroom tone]
In a class, **Читайте текст** and **Пишіть речення** are normal instructions.
They become warmer when the teacher adds **будь ласка** or a name.
:::

### Між друзями

```text
Олена: Тарасе, слухай, ходімо в кафе!
Тарас: Добре. Іди, я зараз.
Олена: Подивись, яка гарна погода!
Тарас: Так! Сідай тут.
Олена: Дай мені меню, будь ласка.
Тарас: Ось, дивись.
Олена: Скажи, що ти хочеш?
Тарас: Я хочу каву.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Олена: Тарасе, слухай, ходімо в кафе!** | Olena: Taras, listen, let's go to a cafe! |
| **Тарас: Добре. Іди, я зараз.** | Taras: All right. Go, I am coming. |
| **Олена: Подивись, яка гарна погода!** | Olena: Look, what nice weather! |
| **Тарас: Так! Сідай тут.** | Taras: Yes! Sit here. |
| **Олена: Дай мені меню, будь ласка.** | Olena: Give me the menu, please. |
| **Тарас: Ось, дивись.** | Taras: Here, look. |
| **Олена: Скажи, що ти хочеш?** | Olena: Tell me what you want. |
| **Тарас: Я хочу каву.** | Taras: I want coffee. |

Friend language uses **ти** forms: **слухай**, **іди**, **подивись**,
**сідай**, **дай**, **дивись**, **скажи**. These can sound warm, not rude,
when the relationship and intonation fit.

### На тренуванні

```text
Тренер: Олено, принеси м'яч, будь ласка.
Тренер: Тарасе, розстав конуси.
Тренер: Андрію, натягни сітку.
Тренер: Маріє, поклади рушники на лавку.
Тренер: Друзі, відкрийте двері.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Тренер: Олено, принеси м'яч, будь ласка.** | Coach: Olena, bring the ball, please. |
| **Тренер: Тарасе, розстав конуси.** | Coach: Taras, set out the cones. |
| **Тренер: Андрію, натягни сітку.** | Coach: Andrii, tighten the net. |
| **Тренер: Маріє, поклади рушники на лавку.** | Coach: Mariia, put the towels on the bench. |
| **Тренер: Друзі, відкрийте двері.** | Coach: Friends, open the door. |

The volleyball scene from the plan is useful because every command has a
visible action: **принеси**, **розстав**, **натягни**, **поклади**,
**відкрийте**. Keep them as practical sports instructions.

## Наказовий спосіб

The school term is **наказовий спосіб**. It covers commands, requests,
instructions, and invitations. At A1, use only two active forms:

| Use | One person | Polite or plural |
| --- | --- | --- |
| read | **читай** | **читайте** |
| write | **пиши** | **пишіть** |
| listen | **слухай** | **слухайте** |
| speak | **говори** | **говоріть** |
| look | **дивись** | **дивіться** |
| go | **іди** | **ідіть** |
| give | **дай** | **дайте** |
| say/tell | **скажи** | **скажіть** |

The social choice is the important part:

- Use **ти** with a friend, child, close family member, or one teammate:
  **Тарасе, читай.**
- Use **ви** with a teacher, server, stranger, older person, or more than one
  person: **Пане Іване, читайте.**
- Add **будь ласка** to make the request polite:
  **Дайте меню, будь ласка.**

This is direct Ukrainian, not rude Ukrainian. **Читай!** can be a normal class
instruction. **Дай, будь ласка** can be a friendly request. Tone, relationship,
and **будь ласка** do the politeness work.

:::tip[Two choices]
Before you speak, choose only two things: **ти** or **ви**, then whether to add
**будь ласка**. That is enough for active A1 requests.
:::

## Як утворити?

For A1, learn the common forms as pairs.

### The ti form

Use **ти** commands with one familiar person:

| Infinitive | ти command | A1 line |
| --- | --- | --- |
| **читати** | **читай** | **Олено, читай.** |
| **слухати** | **слухай** | **Тарасе, слухай.** |
| **писати** | **пиши** | **Пиши речення.** |
| **говорити** | **говори** | **Говори повільно.** |
| **дивитися** | **дивись** | **Подивись на меню.** |
| **іти** | **іди** | **Іди додому.** |
| **дати** | **дай** | **Дай меню, будь ласка.** |
| **сказати** | **скажи** | **Скажи, що ти хочеш.** |
| **сісти** | **сядь** | **Сядь тут.** |
| **відкрити** | **відкрий** | **Відкрий двері.** |

### The vy form

For polite speech or a group, the form often adds **-те** or **-іть**:

| ти | ви |
| --- | --- |
| **читай** | **читайте** |
| **слухай** | **слухайте** |
| **пиши** | **пишіть** |
| **говори** | **говоріть** |
| **дивись** | **дивіться** |
| **ходи** | **ходіть** |
| **дай** | **дайте** |
| **скажи** | **скажіть** |
| **іди** | **ідіть** |
| **сядь** | **сядьте** |
| **відкрий** | **відкрийте** |

Do not overbuild the rule. Many forms are short and frequent. Memorize the
ones you need for real requests: **дайте**, **скажіть**, **відкрийте**,
**пишіть**, **читайте**.

### Recognition-only repairs

:::caution[A1 boundary]
This module does not teach every Ukrainian imperative. Keep active production
to **ти** and **ви** request lines: **читай / читайте**, **пиши / пишіть**,
**дай / дайте**, **скажи / скажіть**, **іди / ідіть**.
:::

Use the repair ideas only to protect the active A1 pattern:

| If the learner tries... | A1 repair |
| --- | --- |
| an infinitive after **будь ласка** | choose **читай** or **читайте** |
| polite address with a **ти** command | choose **дайте**, **скажіть**, or another **ви** form |
| a copied shortcut with **давай** | use the Ukrainian request form you already know |
| an infinitive after **хай** | recognize the finite repair **Хай він працює** |
| a copied soft sign in a command | keep the Ukrainian spelling: **постав**, **відкрий** |
| a mechanical command from **доповісти** | recognize the fixed repair **доповідай** |
| a mixed inclusive ending | recognize fixed phrases like **Читаймо разом!** later; do not improvise the form |

## Підсумок

<span id="підсумок"></span>

Use this as your active A1 request map:

| Situation | Say this |
| --- | --- |
| to a friend | **Олено, читай, будь ласка.** |
| to a teacher or stranger | **Читайте, будь ласка.** |
| asking for a menu | **Дайте меню, будь ласка.** |
| asking a friend for the menu | **Дай меню, будь ласка.** |
| asking someone to say something | **Скажи / скажіть, будь ласка.** |
| class instruction | **Відкрийте підручники. Пишіть речення.** |
| cafe request | **Дайте рахунок, будь ласка.** |

Common repairs:

| Problem | A1 repair |
| --- | --- |
| infinitive as request | choose **читай / читайте** |
| copied shortcut with **давай** | recognize **Робімо це зараз!** as a fixed repair; active A1 still uses **зроби / зробіть** requests |
| infinitive after **хай** | recognize **Хай він працює** as the repair |
| soft sign copied into a command | write **постав**, not a copied spelling |
| mechanical form from **доповісти** | recognize **доповідай** as the repair |
| mixed inclusive ending | use fixed **Читаймо разом!** only as a recognition repair |

The active lesson remains simple: **читай / читайте**, **пиши / пишіть**,
**дай / дайте**, **скажи / скажіть**, plus **будь ласка**. Use the wider
repairs only to avoid common interference.

Keep the decolonized grammar habit: Ukrainian does not need **«давай»** as a
form-building shortcut for commands. Use the Ukrainian command form itself:
**слухай / слухайте** here. For spelling, do not explain Ukrainian by saying
"like another language but without a soft sign." Treat forms such as **постав**
and **відкрий** as Ukrainian patterns in their own right. The learner builds
Ukrainian from Ukrainian.

Next module links short ideas. You will be able to say:
**Олено, читай і пиши, будь ласка.** For now, keep one clear action at a time.
