## Expressing Ability: –º–æ–≥—Ç–∏ vs –≤–º—ñ—Ç–∏

> [!observe] Pattern Discovery
>
> Look at these sentences:
>
> - –Ø **–º–æ–∂—É** –≥–æ–≤–æ—Ä–∏—Ç–∏ —É–∫—Ä–∞—ó–Ω—Å—å–∫–æ—é. (I can speak Ukrainian.) ‚Äî general ability
> - –Ø **–≤–º—ñ—é** –ø–ª–∞–≤–∞—Ç–∏. (I know how to swim.) ‚Äî learned skill
>
> **–º–æ–≥—Ç–∏** = physical/circumstantial ability
> **–≤–º—ñ—Ç–∏** = acquired skill or knowledge

### –º–æ–≥—Ç–∏ ‚Äî Can (Physical Ability)

## Expressing Obligation: –ø–æ–≤–∏–Ω–µ–Ω, —Ç—Ä–µ–±–∞, –º—É—Å–∏—Ç–∏

> [!observe] Pattern Discovery
>
> - –Ø **–ø–æ–≤–∏–Ω–µ–Ω** –ø—Ä–∞—Ü—é–≤–∞—Ç–∏. (I must work.) ‚Äî personal obligation
> - **–¢—Ä–µ–±–∞** –ø—Ä–∞—Ü—é–≤–∞—Ç–∏. (One must work.) ‚Äî impersonal necessity
> - –Ø **–º—É—à—É** –π—Ç–∏. (I have to go.) ‚Äî strong obligation

### –ø–æ–≤–∏–Ω–µ–Ω/–ø–æ–≤–∏–Ω–Ω–∞ ‚Äî Must (Personal)

This is an adjective that agrees in gender:

| Gender    | Form    | Example                                      |
| --------- | ------- | -------------------------------------------- |
| masculine | –ø–æ–≤–∏–Ω–µ–Ω | –í—ñ–Ω **–ø–æ–≤–∏–Ω–µ–Ω** –≤—á–∏—Ç–∏—Å—è. (He must study.)    |
| feminine  | –ø–æ–≤–∏–Ω–Ω–∞ | –í–æ–Ω–∞ **–ø–æ–≤–∏–Ω–Ω–∞** –ø—Ä–∞—Ü—é–≤–∞—Ç–∏. (She must work.) |
| neuter    | –ø–æ–≤–∏–Ω–Ω–µ | ‚Äî (rare)                                     |
| plural    | –ø–æ–≤–∏–Ω–Ω—ñ | –í–æ–Ω–∏ **–ø–æ–≤–∏–Ω–Ω—ñ** –ø—Ä–∏–π—Ç–∏. (They must come.)   |

### —Ç—Ä–µ–±–∞ / –ø–æ—Ç—Ä—ñ–±–Ω–æ ‚Äî Need To (Impersonal)

These are impersonal ‚Äî the "subject" is optional:

- **–¢—Ä–µ–±–∞** –ø—Ä–∞—Ü—é–≤–∞—Ç–∏. (One needs to work. / It's necessary to work.)
- **–ü–æ—Ç—Ä—ñ–±–Ω–æ** –≤—á–∏—Ç–∏—Å—è. (One needs to study.)
- **–ù–µ–æ–±—Ö—ñ–¥–Ω–æ** –≤—ñ–¥–ø–æ–≤—ñ—Å—Ç–∏. (It's essential to answer.)

### –º—É—Å–∏—Ç–∏ ‚Äî Must, Have To

| Person   | –º—É—Å–∏—Ç–∏ | Example                                  |
| -------- | ------ | ---------------------------------------- |
| —è        | –º—É—à—É   | –Ø –º—É—à—É –π—Ç–∏. (I must go.)                 |
| —Ç–∏       | –º—É—Å–∏—à  | –¢–∏ –º—É—Å–∏—à –∑—Ä–æ–±–∏—Ç–∏ —Ü–µ. (You must do this.) |

## Expressing Permission: –º–æ–∂–Ω–∞ / –Ω–µ –º–æ–∂–Ω–∞

> [!observe] Pattern Discovery
>
> - **–ú–æ–∂–Ω–∞** —É–≤—ñ–π—Ç–∏? (May I enter?)
> - **–ù–µ –º–æ–∂–Ω–∞** –ø–∞–ª–∏—Ç–∏ —Ç—É—Ç. (Smoking is not allowed here.)
