# Підсумок: Мій світ

This checkpoint brings Modules 8-13 together. You are not learning a new
grammar rule today. You are checking whether the familiar pieces can work in
one small Ukrainian scene: things, gender, descriptions, colors, numbers,
pointing words, and simple plurals.

By the end, you can:

- choose noun gender with **він / вона / воно** and use **мій / моя / моє / мої**;
- describe objects with adjectives and colors: **великий стіл**, **нова книга**, **біле вікно**;
- ask and answer prices with familiar number chunks;
- point with **цей / ця / це / ці** and **той / та / те / ті**;
- form familiar plurals such as **столи**, **книги**, **вікна**, **зошити**;
- make a 6-10 sentence description of your own small world.

:::tip
Use the noun as the control word. First ask: **він**, **вона**, **воно**, or
plural? Then choose the pointing word, possessive, and adjective.
:::

## Що ми знаємо?

<!-- INJECT_ACTIVITY: act-1 -->

Here is your A1.2 toolkit from Modules 8-13.

| Toolkit | Quick check | Safe example |
| --- | --- | --- |
| Gender | Is the noun **він**, **вона**, or **воно**? | **стіл** -> **він**, **книга** -> **вона**, **вікно** -> **воно** |
| Possession | Does **my** match the noun? | **мій стіл**, **моя книга**, **моє вікно**, **мої книги** |
| Description | Does the adjective match? | **великий стіл**, **нова книга**, **чисте вікно** |
| Colors | Can you use the color form, including both blue words? | **червона сумка**, **синій зошит**, **блакитний зошит**, **біле вікно** |
| Numbers and prices | Can you ask a price? | **Скільки коштує?** -> **Двісті гривень.** |
| Pointing | Is it near, far, or plural? | **ця лампа**, **та сумка**, **ці книги**, **ті зошити** |
| Plural | Can you name more than one? | **столи**, **книги**, **вікна**, **ручки** |

Notice the pattern: one noun controls several small choices. If you know
**книга** is feminine, then **моя книга**, **ця книга**, **нова книга**, and
**червона книга** all feel connected. If you know **вікно** is neuter, then
**моє вікно**, **це вікно**, and **чисте вікно** use the same idea.

:::note
This is a checkpoint, so mistakes are useful. If one form feels wrong, go back
to the noun and rebuild the phrase from there.
:::

## Читання

Read this text aloud. It uses recycled A1.2 vocabulary only: room objects,
colors, prices, pointing words, and plural nouns.

```text
Це моя кімната.
Мій стіл великий і новий.
Ця лампа біла, а та лампа жовта.
У мене є три книги.
Ці книги нові.
Ті зошити сині.
Моє вікно чисте.
Ці вікна чисті.
Та сумка чорна.
Один зошит коштує двадцять гривень.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Це моя кімната.** | This is my room. |
| **Мій стіл великий і новий.** | My table is big and new. |
| **Ця лампа біла, а та лампа жовта.** | This lamp is white, and that lamp is yellow. |
| **У мене є три книги.** | I have three books. |
| **Ці книги нові.** | These books are new. |
| **Ті зошити сині.** | Those notebooks are blue. |
| **Моє вікно чисте.** | My window is clean. |
| **Ці вікна чисті.** | These windows are clean. |
| **Та сумка чорна.** | That bag is black. |
| **Один зошит коштує двадцять гривень.** | One notebook costs twenty hryvnias. |

Now do one more reading step. This is the same grammar, but in a Ukrainian
street-market scene. Four cultural object words are recognition labels here:
**вишиванка**, **глечик**, **намисто**, **писанки**. You do not need to make
new grammar with them yet; just notice the gender and plural cues around them.

| Market word | English support | Control cue |
| --- | --- | --- |
| **вишиванка** | embroidered shirt | **вона**, **ця**, **біла** |
| **глечик** | clay jug | **він**, **той**, **старий** |
| **намисто** | necklace | **воно / це**, **коштує** |
| **писанки** | decorated Easter eggs | **вони**, **ці**, **червоні** |

:::tip
For cultural words, you can understand more than you can produce. Recognition
first is normal at A1.
:::

## Граматика

The checkpoint grammar is a set of small control questions.

| Question | Use it for | Examples |
| --- | --- | --- |
| **Що це?** | naming a thing | **Це стіл. Це книга. Це вікно.** |
| **Він, вона, воно?** | noun gender | **стіл** -> **він**, **книга** -> **вона**, **вікно** -> **воно** |
| **Мій, моя, моє, мої?** | possession | **мій стіл**, **моя лампа**, **моє вікно**, **мої книги** |
| **Який / яка / яке / які?** | description | **великий**, **нова**, **чисте**, **сині** |
| **Цей / ця / це / ці?** | this / these | **цей зошит**, **ця сумка**, **це вікно**, **ці книги** |
| **Той / та / те / ті?** | that / those | **той стіл**, **та лампа**, **те вікно**, **ті зошити** |

Adjectives and colors match the noun.

| Noun type | Possessive | Adjective | Color |
| --- | --- | --- | --- |
| Masculine | **мій стіл** | **великий стіл** | **синій зошит** |
| Feminine | **моя книга** | **нова книга** | **червона сумка** |
| Neuter | **моє вікно** | **чисте вікно** | **біле вікно** |
| Plural | **мої книги** | **нові книги** | **сині зошити** |

The hard and soft adjective contrast is small but important.

| Type | Masculine | Feminine | Neuter | Plural |
| --- | --- | --- | --- | --- |
| hard pattern | **червоний** | **червона** | **червоне** | **червоні** |
| soft pattern | **синій** | **синя** | **синє** | **сині** |

Plural review stays simple. Familiar nouns change, and the adjective uses
**-і**.

| One | More than one |
| --- | --- |
| **це стіл** | **це столи** |
| **ця книга** | **ці книги** |
| **це вікно** | **ці вікна** |
| **мій зошит синій** | **мої зошити сині** |

Numbers are chunks here, not a new grammar topic.

| Question | Safe answer |
| --- | --- |
| **Скільки коштує ця сумка?** | **Двісті гривень.** |
| **Скільки коштує один зошит?** | **Двадцять гривень.** |
| **Скільки книг?** | **Три книги.** |

:::warning
Do not add new case rules today. This checkpoint is about nominative gender,
agreement, pointing words, familiar plurals, and price chunks.
:::

## Діалог

Use the full checkpoint conversation at a Ukrainian street market. Ivanko is
visiting with Katia, who helps him point to handmade items, describe them, and
ask prices.

```text
Іванко: Добрий день! Що це?
Катя: Це вишиванка. Вона біла і синя.
Іванко: А той глечик? Він новий?
Катя: Ні, той глечик старий.
Іванко: Ці писанки червоні чи жовті?
Катя: Ці писанки червоні і жовті.
Іванко: Скільки коштує це намисто?
Катя: Сто гривень.
Іванко: У вас є сумки?
Катя: Так. Ця червона чи та синя?
Іванко: Та синя. Скільки вона коштує?
Катя: Двісті гривень.
Іванко: Добре. А ці зошити?
Катя: Один зошит коштує двадцять гривень.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Іванко: Добрий день! Що це?** | Ivanko: Good afternoon! What is this? |
| **Катя: Це вишиванка. Вона біла і синя.** | Katia: This is an embroidered shirt. It is white and dark blue. |
| **Іванко: А той глечик? Він новий?** | Ivanko: And that clay jug? Is it new? |
| **Катя: Ні, той глечик старий.** | Katia: No, that clay jug is old. |
| **Іванко: Ці писанки червоні чи жовті?** | Ivanko: Are these pysanky red or yellow? |
| **Катя: Ці писанки червоні і жовті.** | Katia: These pysanky are red and yellow. |
| **Іванко: Скільки коштує це намисто?** | Ivanko: How much does this necklace cost? |
| **Катя: Сто гривень.** | Katia: One hundred hryvnias. |
| **Іванко: У вас є сумки?** | Ivanko: Do you have bags? |
| **Катя: Так. Ця червона чи та синя?** | Katia: Yes. This red one or that dark blue one? |
| **Іванко: Та синя. Скільки вона коштує?** | Ivanko: That dark blue one. How much does it cost? |
| **Катя: Двісті гривень.** | Katia: Two hundred hryvnias. |
| **Іванко: Добре. А ці зошити?** | Ivanko: Good. And these notebooks? |
| **Катя: Один зошит коштує двадцять гривень.** | Katia: One notebook costs twenty hryvnias. |

This dialogue checks the whole A1.2 set:

- gender: **вона**, **він**, **це**;
- agreement: **біла**, **синя**, **старий**, **червоні**, **жовті**;
- pointing: **той**, **ці**, **ця**, **та**;
- prices: **сто гривень**, **двісті гривень**, **двадцять гривень**;
- plural nouns: **писанки**, **сумки**, **зошити**.

## Підсумок

You can now describe a small part of your world in Ukrainian. A1.2 gave you
the pieces for objects, gender, descriptions, colors, numbers, pointing, and
simple plurals. That is enough for a short, useful description.

Make your own 6-10 sentence checkpoint text. Keep it small: your room, your
desk, your bag, your notebook, or a few things in a shop.

Use this frame if you want structure:

1. **Це моя кімната.**
2. **Мій стіл великий.**
3. **Моє вікно чисте.**
4. **Ця лампа біла, а та сумка синя.**
5. **У мене є три книги.**
6. **Ці книги нові.**
7. **Ті зошити сині.**
8. **Один зошит коштує двадцять гривень.**

One safe model:


### Виправ пастки

Keep these repairs close:

| Avoid | Use | Why |
| --- | --- | --- |
| **Мій книга.** | **Моя книга.** | **книга** is feminine |
| **Цей лампа.** | **Ця лампа.** | **лампа** is feminine |
| **Великий вікно.** | **Велике вікно.** | **вікно** is neuter |
| **Ці книга.** | **Ці книги.** | **ці** needs plural |
| **Ті зошит.** | **Ті зошити.** | **ті** needs plural |
| **Двадцять гривня.** | **Двадцять гривень.** | price chunk |
