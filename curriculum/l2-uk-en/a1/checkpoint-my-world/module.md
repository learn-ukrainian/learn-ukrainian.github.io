# Checkpoint: My World

This checkpoint ties together the A1.2 world: things, gender, descriptions,
colors, numbers, pointing words, and simple plurals. Nothing here needs a new
grammar rule. You are testing whether the pieces can work together in a short
real-life scene.

By the end, you can:

- choose noun gender with **він / вона / воно** and use **мій / моя / моє**;
- describe objects with **великий / велика / велике / великі** and colors;
- count and ask prices with familiar number chunks;
- point with **цей / ця / це / ці** and **той / та / те / ті**;
- use familiar plural forms such as **столи**, **книги**, **вікна**;
- read and produce a 6-10 sentence description of your own small world.

## Що ми знаємо?

You have six small toolkits from Modules 8-13.

| Toolkit | You can now do this |
| --- | --- |
| Gender | connect **стіл** with **він**, **книга** with **вона**, **вікно** with **воно** |
| Possession | choose **мій**, **моя**, **моє**, **мої** |
| Description | say **великий стіл**, **нова книга**, **чисте вікно** |
| Colors | use **червоний**, **синій**, **блакитний**, **білий**, **чорний**, **жовтий** |
| Numbers and prices | ask **Скільки коштує...?** and answer with **гривень** |
| Pointing and plural | use **цей / ця / це / ці**, **той / та / те / ті**, **столи**, **книги**, **вікна** |

The older first-contact chunks still matter. You can say **Мене звати...**,
not **Моє ім'я є...**, and **Я студент / студентка** without adding **є**.
The pronouns **я, ти, він, вона, воно, ми, ви, вони** help you keep track of
people and things.

This checkpoint uses Ukrainian directly. Learn **українська мова** on its own
terms. Do not use Russian as a sound bridge, spelling bridge, or cultural
standard. In course examples, keep the clean Ukrainian choices:
**Добрий день**, **До побачення**, **тато**, **прізвище**.

:::tip
Use the noun as the control word. Ask: is it **він**, **вона**, **воно**, or
plural? Then choose the pointing word, possessive, and adjective.
:::

## Граматика

Review the control questions.

| Question | Use it for | Examples |
| --- | --- | --- |
| **Хто?** | people | **студент**, **мама**, **вчителька** |
| **Що?** | things | **стіл**, **книга**, **вікно** |
| **Який?** | masculine description | **великий стіл**, **синій зошит** |
| **Яка?** | feminine description | **нова книга**, **чиста кімната** |
| **Яке?** | neuter description | **біле вікно**, **моє місто** |
| **Які?** | plural description | **великі кімнати**, **вечірні міста** |

Nouns that name things still have gender. Store a new noun with a cue:
**словник** is **він**, **кімната** is **вона**, **місто** is **воно**.
The nominative singular habit lets you build **білий сніг**, **біла шубка**,
and **біле козенятко**. The checkpoint uses the same idea with familiar words:
**великий стіл**, **велика кімната**, **велике вікно**.

Possessives behave like adjectives:

| English | Masculine | Feminine | Neuter | Plural |
| --- | --- | --- | --- | --- |
| my | **мій телефон** | **моя кімната** | **моє місто** | **мої книги** |
| your | **твій зошит** | **твоя лампа** | **твоє вікно** | **твої ручки** |

Adjectives use the endings you already practiced:

- masculine: **який? великий**, **чистий**, **синій**; endings **-ий**, **-ій**
- feminine: **яка? велика**, **чиста**, **синя**; endings **-а**, **-я**
- neuter: **яке? велике**, **чисте**, **синє**; endings **-е**, **-є**
- plural: **які? великі**, **чисті**, **сині**; shared ending **-і**

Plural-only words, or **множинні іменники**, remain whole-word items for now:
**окуляри**, **двері**, **ліки**, **шахи**. Collective singular words such as **дітвора** and
**студентство** are passive recognition only.

## Підсумок

Make your own 6-10 sentence checkpoint text. Keep it personal and small. This
is your connected production task: write about a normal day or favorite things,
show noun-adjective agreement, and use pronouns clearly.

In Ukrainian terms, this is a **зв'язне висловлювання**: a **коротку
розповідь** of **6–10 речень** about your **звичайний день** or **улюблені
речі**. Show that you can **узгоджувати іменники з прикметниками** and use
**займенники** clearly.

Use this frame:

1. **Мене звати...**
2. **Це моя кімната / мій стіл / моє місто.**
3. **Мій / моя / моє ... великий / велика / велике.**
4. **Ця ... біла, а та ... синя.**
5. **У мене є три...**
6. **Ці ... нові.**
7. **Скільки коштує...?**
8. **... гривень.**

### Repair traps

Keep these repairs close:

| Avoid | Use | Why |
| --- | --- | --- |
| **Я є студент.** | **Я студент.** | present identity normally drops **є** |
| **Моє ім'я є Джон.** | **Мене звати Джон.** | name chunk |
| **Мій книга на столі.** | **Моя книга на столі.** | **книга** is feminine |
| **Вона має один окуляри.** | **Вона має окуляри.** / **У неї є окуляри.** | **окуляри** is plural-only |
| **Я бачу великий кімнату.** | **Я бачу велику кімнату.** | feminine object chunk |
| **Мені 20 роки.** | **Мені 20 років.** | age chunk |

Historical and cultural facts stay precise when they appear later:
the Holodomor is **1932-1933**, and Ukrainian literature examples belong to
the Ukrainian canon. Do not present Russian-imperial figures as Ukrainian
authors.
