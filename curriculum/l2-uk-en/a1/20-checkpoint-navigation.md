### Section 7: Phase A1.2 Summary

You started Phase A1.2 knowing only "This is a table".
Now you can:
1.  **See** the table (BaÄu stil).
2.  **Put** something on the table (Na stoli).
3.  **Take** something from the table (Z stolu - Gen).
4.  **Buy** the table (Kupuyu stil).
5.  **Say** there is no table (Nemaye stolu).

This "Case System" is the engine of the language.
In Phase A1.3, we add **Time** (Past/Future).
The static picture becomes a movie.
