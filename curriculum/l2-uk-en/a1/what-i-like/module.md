# Що я люблю

Today starts A1.3: actions. You already know how to point to things and
describe them. Now you can say what you like doing.

There are two safe preference lanes:

- **Я люблю + action word in -ти** for things you like to do;
- **Мені подобається + noun** for things, places, music, books, or films.

You do not need a case table today. Treat **мені подобається** as one ready
phrase, and treat the **-ти** ending as the sign of an action word.

By the end, you can:

- ask **Що ти любиш робити?**;
- answer with **Я люблю читати** or another **-ти** action;
- recognize common infinitives such as **читати**, **гуляти**, **готувати**;
- say **Мені подобається музика / ця книга / цей фільм**;
- make a simple negative sentence with **не люблю** and **не подобається**.

:::tip
Keep the two lanes separate. Actions go after **я люблю**. Things go after
**мені подобається**.
:::

## Діалоги

<!-- INJECT_ACTIVITY: act-1 -->

Anna and Viktor are at a language exchange and talk about hobbies over tea.
The first dialogue uses **люблю + -ти** for actions.

```text
Анна: Добрий день, Вікторе!
Віктор: Добрий день, Анно!
Анна: Що ти любиш робити?
Віктор: Я люблю читати і слухати музику.
Анна: А я люблю готувати.
Віктор: Правда? Що ти готуєш?
Анна: Суп і чай.
Віктор: Чудово!
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Анна: Добрий день, Вікторе!** | Anna: Good afternoon, Viktor! |
| **Віктор: Добрий день, Анно!** | Viktor: Good afternoon, Anna! |
| **Анна: Що ти любиш робити?** | Anna: What do you like to do? |
| **Віктор: Я люблю читати і слухати музику.** | Viktor: I like to read and listen to music. |
| **Анна: А я люблю готувати.** | Anna: And I like to cook. |
| **Віктор: Правда? Що ти готуєш?** | Viktor: Really? What do you cook? |
| **Анна: Суп і чай.** | Anna: Soup and tea. |
| **Віктор: Чудово!** | Viktor: Great! |

The second dialogue uses **мені подобається** for things.

```text
Анна: Тобі подобається ця книга?
Віктор: Так, мені подобається.
Анна: А цей фільм?
Віктор: Ні, мені не подобається цей фільм.
Анна: Що тобі подобається?
Віктор: Мені подобається музика.
Анна: Мені теж.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Анна: Тобі подобається ця книга?** | Anna: Do you like this book? |
| **Віктор: Так, мені подобається.** | Viktor: Yes, I like it. |
| **Анна: А цей фільм?** | Anna: And this film? |
| **Віктор: Ні, мені не подобається цей фільм.** | Viktor: No, I do not like this film. |
| **Анна: Що тобі подобається?** | Anna: What do you like? |
| **Віктор: Мені подобається музика.** | Viktor: I like music. |
| **Анна: Мені теж.** | Anna: Me too. |

:::note
You will learn the full verb patterns later. Today, use **я люблю** and
**ти любиш** actively.
:::

## Я люблю...

Use **Я люблю + infinitive** for actions. In Ukrainian, an infinitive is the
dictionary form of a verb. In this module, your action words end in **-ти**.

| English | Ukrainian action word | Safe sentence |
| --- | --- | --- |
| to read | **читати** | **Я люблю читати.** |
| to walk | **гуляти** | **Я люблю гуляти.** |
| to cook | **готувати** | **Я люблю готувати.** |
| to listen | **слухати** | **Я люблю слухати музику.** |
| to watch | **дивитися** | **Я люблю дивитися фільм.** |
| to play | **грати** | **Я люблю грати.** |
| to draw | **малювати** | **Я люблю малювати.** |
| to travel | **подорожувати** | **Я люблю подорожувати.** |
| to sing | **співати** | **Я люблю співати.** |

The second action does not change:

| Avoid | Use |
| --- | --- |
| **Я люблю читаю.** | **Я люблю читати.** |
| **Я люблю малюю.** | **Я люблю малювати.** |
| **Ти любиш готуєш?** | **Ти любиш готувати?** |

English can say "I like reading" or "I like to read." Ukrainian gives you one
safe A1 habit:

```text
Я люблю читати.
Я люблю гуляти.
Я люблю слухати музику.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Я люблю читати.** | I like to read. |
| **Я люблю гуляти.** | I like to walk. |
| **Я люблю слухати музику.** | I like to listen to music. |

:::warning
Do not build a full verb-conjugation lesson here. The checkpoint skill is
simple: **люблю / любиш + -ти**.
:::

## Мені подобається...

Use **Мені подобається + noun** when a thing, place, book, film, or music is
pleasing to you. At A1, this is a fixed phrase. Do not analyze why **мені**
has that form yet.

```text
Мені подобається музика.
Мені подобається ця книга.
Мені подобається цей фільм.
Мені подобається Київ.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Мені подобається музика.** | I like music. |
| **Мені подобається ця книга.** | I like this book. |
| **Мені подобається цей фільм.** | I like this film. |
| **Мені подобається Київ.** | I like Kyiv. |

Compare the two lanes:

| If you like doing something | If you like a thing |
| --- | --- |
| **Я люблю читати.** | **Мені подобається ця книга.** |
| **Я люблю готувати.** | **Мені подобається суп.** |
| **Я люблю слухати музику.** | **Мені подобається музика.** |

Add **не** before the preference verb:

- **Я не люблю готувати.**
- **Я не люблю дивитися фільм.**
- **Мені не подобається цей фільм.**
- **Тобі не подобається ця книга?**

Questions stay short:

| Question | Safe answer |
| --- | --- |
| **Що ти любиш робити?** | **Я люблю читати.** |
| **Ти любиш готувати?** | **Так, люблю.** / **Ні, не люблю.** |
| **Тобі подобається ця книга?** | **Так, мені подобається.** |
| **Тобі подобається цей фільм?** | **Ні, мені не подобається.** |

## Підсумок

You now have two useful A1 preference patterns.

| Use this | For this | Example |
| --- | --- | --- |
| **Я люблю + -ти** | actions | **Я люблю читати.** |
| **Мені подобається + noun** | things, places, media | **Мені подобається музика.** |
| **не** before the verb | negatives | **Я не люблю готувати.** |

Build a short personal answer:

```text
Я люблю читати.
Я люблю гуляти.
Я не люблю готувати.
Мені подобається музика.
Мені подобається ця книга.
А ти?
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Я люблю читати.** | I like to read. |
| **Я люблю гуляти.** | I like to walk. |
| **Я не люблю готувати.** | I do not like to cook. |
| **Мені подобається музика.** | I like music. |
| **Мені подобається ця книга.** | I like this book. |
| **А ти?** | And you? |

### Виправ пастки

| Avoid | Use | Why |
| --- | --- | --- |
| **Я люблю малюю.** | **Я люблю малювати.** | the action after **люблю** stays in **-ти** |
| **Ти любиш готуєш?** | **Ти любиш готувати?** | **любиш** also takes an infinitive |
| **Я подобається музика.** | **Мені подобається музика.** | use the fixed phrase **мені подобається** |
| **Мені подобається дивлюся.** | **Я люблю дивитися.** | use **я люблю + -ти** for actions |
| **Я люблю не готувати.** | **Я не люблю готувати.** | **не** goes before **люблю** |
| **Мені подобається не цей фільм.** | **Мені не подобається цей фільм.** | **не** goes before **подобається** |

Next you will start learning more verb forms. For now, these two lanes are
enough for a real conversation about hobbies and interests.
