# Екстрені ситуації

This module gives you **emergency language** for a short clear call or for
asking another person to call.

Official Ukrainian emergency numbers checked on 2026-06-03:

| Number | Ukrainian label | English check |
| --- | --- | --- |
| **112** | **єдиний номер екстреної допомоги** | single emergency number |
| **101** | **служба порятунку / пожежна допомога** | rescue / fire |
| **102** | **поліція** | police |
| **103** | **швидка медична допомога** | ambulance / emergency medical help |
| **104** | **аварійна газова служба** | gas emergency service |

For A1, you can use **112** as the main number and still recognize the direct
service numbers.

The core call frame is:

**Допоможіть! Тут аварія. Я на вулиці Шевченка, будинок п'ять. Мене звати
Анна.**

| Step | Phrase |
| --- | --- |
| call for help | **Допоможіть!** |
| say the problem | **Тут аварія / пожежа.** |
| say the place | **Я на вулиці... біля...** |
| say your name | **Мене звати...** |
| repair | **Повторіть, будь ласка.** |

:::tip[A1 boundary]
Stay with language. Say the problem, place, name, and phone number in short A1
phrases.
:::

This call uses only four learner jobs:

- **Допоможіть!**
- **Тут аварія.**
- **Я на вулиці...**
- **Мене звати...**

If the operator speaks quickly, use **Повторіть, будь ласка** or
**Повільніше, будь ласка**.

Learn the numbers as chunks. Do not translate every word during a call.

| Number | Say it | Service word |
| --- | --- | --- |
| **112** | **один один два** | **екстрена допомога** |
| **101** | **один нуль один** | **служба порятунку / пожежна допомога** |
| **102** | **один нуль два** | **поліція** |
| **103** | **один нуль три** | **швидка** |

## Що сталося?

Use one short problem sentence.

| Problem | A1 sentence |
| --- | --- |
| accident | **Тут аварія.** |
| fire | **Тут пожежа.** |
| police help | **Потрібна поліція.** |
| ambulance | **Потрібна швидка.** |
| gas problem | **Потрібна газова служба.** |
| lost passport | **Я загубив / загубила паспорт.** |
| I feel unwell | **Мені погано.** |
| someone feels unwell | **Людині погано.** |

Use **загубив** if you speak as a man or boy. Use **загубила** if you speak as
a woman or girl.

For an A1 call, one problem sentence is enough. Add location next. Keep each
line separate: help, problem, place, identity, repair.



## Де ви?

Emergency language needs place words you already know from the city modules.

| Need | Phrase |
| --- | --- |
| street | **Я на вулиці Шевченка.** |
| building | **Будинок п'ять.** |
| near | **Я біля метро.** |
| in a place | **Я в готелі.** |
| opposite | **Я навпроти аптеки.** |
| city | **Я в Києві.** |

You can combine them:

**Я в Києві, на вулиці Франка, будинок десять.**

**Я біля метро, навпроти аптеки.**

**Я в готелі. Адреса: вулиця Лесі Українки, будинок два.**

Say numbers slowly. If you need time to repeat, use:

**Повільніше, будь ласка. Я повторюю адресу.**


### Особиста інформація

The operator or helper may ask for basic personal information. These are A1
identity chunks.

| Question | Answer frame |
| --- | --- |
| **Як вас звати?** | **Мене звати...** |
| **Ваше прізвище?** | **Моє прізвище...** |
| **Ваш номер телефону?** | **Мій номер телефону...** |
| **Ви звідки?** | **Я з Канади / Польщі / США.** |
| **Ви розумієте?** | **Я трохи розумію. Повторіть, будь ласка.** |

Keep the answer slow and separate:

**Мене звати Сара. Моє прізвище Браун. Мій номер телефону: нуль дев'яносто
сім, триста десять, двадцять два, сорок один.**

This is not a story. It is a clear information block.
