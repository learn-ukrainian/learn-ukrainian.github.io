## Critical Issues Found

### Issue 1: Activity Sentences Exceed A1 Complexity (CRITICAL)
- **Location:** Activities file, quiz "Оберіть правильний прийменник" (lines 49-136)
- **Evidence:** Line 49: «Я зараз йду ... магазин, щоб купити свіжий хліб.» — "щоб + infinitive" is a B1 structure (purpose clause). Line 71: «Моя нова цікава книга лежить ... великому столі.» — three stacked adjectives is excessive for A1. Line 311: «Швидкий нічний поїзд ... до Львова через Київ.» — "через" (through) is not taught in this module. 6 of 8 quiz items in activity 3 use padding clauses beyond A1 level.
- **Impact:** Learners will be overwhelmed by sentence complexity when the task should only test preposition choice. This fails the "Did I feel overwhelmed?" test.
- **Fix:** Simplify all quiz sentences to A1 structures. E.g., «Я йду ... магазин.» instead of «Я зараз йду ... магазин, щоб купити свіжий хліб.»

### Issue 2: Missing Warmth and Encouragement (CRITICAL for Tier 1)
- **Location:** Throughout content file — no encouragement phrases anywhere
- **Evidence:** No "Great!", "Well done!", "You've got this!" or equivalent markers. No warm greeting at the start. Line 16 opens with «Imagine you are at the railway station in Kyiv.» — no welcome. Line 459 closes with «Тепер можна йти далі!» — functional but impersonal. Zero "don't worry" moments despite teaching three cases (which is intimidating).
- **Impact:** Fails beginner emotional safety mapping. Required: ≥3 encouragement phrases, ≥2 "don't worry" moments, ≥1 welcome/orientation. Current: 0, 0, 0.
- **Fix:** Add warm opening ("Привіт! Today we're tackling something big — and you're ready for it."). Add encouragement after each case section. Add "You can now..." celebration at closing.

### Issue 3: Slipper Culture Hook Missing from Plan
- **Location:** Content file, section «Культурний аспект: Вдома (At Home)» (lines 159-166)
- **Evidence:** The plan explicitly requires "Slipper Culture: Exploring the linguistic and physical boundary between the outside world (shoes on) and the home «вдома» (shoes off)." The content only says «**Вдо́ма** is an adverb. It means "at home".» — no cultural dimension whatsoever.
- **Impact:** Plan compliance failure. Missing a key cultural hook that would connect grammar to real Ukrainian life.
- **Fix:** Expand the «Культурний аспект: Вдома» subsection with the slipper culture hook: entering a Ukrainian home means removing shoes — the physical boundary between outside (динаміка, рух) and inside (статика, вдома).

### Issue 4: Naturalness Error in Activity — "студенти до школи"
- **Location:** Activities file, quiz "Йти чи Їхати?", line 300
- **Evidence:** «Вранці всі студенти ... до школи на уроки.» — In standard Ukrainian, "студенти" attend "університет/інститут", while "учні" attend "школа". A student going "до школи на уроки" is a register mismatch.
- **Impact:** Teaches unnatural Ukrainian. A real speaker would say «учні йдуть до школи» or «студенти йдуть до університету».
- **Fix:** Change "студенти" to "учні" in this item.

### Issue 5: IPA Inconsistency for "звідки"
- **Location:** Content file line 26 vs vocabulary file line 12
- **Evidence:** Content: [ˈzʋʲidkɪ]. Vocabulary: [ˈzʲʋʲidkɪ]. The "з" palatalization differs between files.
- **Impact:** Learner sees two different pronunciations for the same word.
- **Fix:** Standardize to [ˈzʋʲidkɪ] (without palatalized з) in the vocabulary file — regressive palatalization of "з" before "ʋʲ" is not standard in careful speech.

### Issue 6: IPA Stress Error for "вокзал"
- **Location:** Vocabulary file, line 84
- **Evidence:** [ʋɔˈɡzɑl] — the stress mark is placed before "ɡ", but "вокзал" is stressed on the final syllable (вок-ЗАЛ). The correct transcription should be [ʋɔɡˈzɑl].
- **Impact:** Incorrect stress guidance.
- **Fix:** Change to [ʋɔɡˈzɑl].

### Issue 7: Misleading "Exception" Label
- **Location:** Content file, line 200
- **Evidence:** «Робота -> Я йду **на роботу** (exception: uses "на").» — "На роботу" is not an exception. The module systematically teaches that "на" is used for activities and open spaces (на роботі, на концерті, на пошті). Using "на" for "робота" follows this exact rule. Labeling it "exception" contradicts the module's own framework.
- **Impact:** Confuses learners who just learned the в/на distinction based on container vs. activity logic.
- **Fix:** Remove "(exception: uses "на")" and instead add a note like "(remember: робота is an activity, so we use 'на')".

### Issue 8: Vocabulary File Missing Key Items
- **Location:** Vocabulary file
- **Evidence:** "додому" is used 6+ times in content and activities but is absent from the vocabulary file. "центр" and "робота" are in the plan's recommended vocabulary list and heavily used throughout the content but absent from the vocabulary file.
- **Impact:** Learners cannot look up high-frequency words introduced in this module.
- **Fix:** Add "додому", "центр", and "робота" to the vocabulary file.

## Verdict

**NEEDS REVISION**

The module delivers solid grammar instruction with accurate rules, effective visual aids (tables, mirror rules), and good cultural hooks (Ukrzaliznytsia, ринок vs супермаркет). The core pedagogical framework (Де?/Куди?/Звідки? → three cases) is well-structured and clearly presented.

However, three issues require revision before passing:

1. **Activity complexity** (Critical): Quiz sentences systematically use B1 structures (purpose clauses, stacked adjectives, untaught prepositions). These must be simplified to A1 level.
2. **Warmth deficit** (Critical for Tier 1): Zero encouragement phrases, zero "don't worry" moments, no warm welcome. For a module teaching three cases simultaneously — which is intimidating for beginners — the complete absence of emotional support is a significant gap.
3. **Plan compliance**: Missing Slipper Culture hook and 3 vocabulary items (додому, центр, робота).

Secondary fixes: IPA inconsistency/errors, misleading "exception" label, "студенти"→"учні" naturalness fix, "із" characterization.

The content quality is adequate but the module reads more like a reference grammar than a patient tutor's lesson. With warmth injection and activity simplification, this can reach passing quality.
