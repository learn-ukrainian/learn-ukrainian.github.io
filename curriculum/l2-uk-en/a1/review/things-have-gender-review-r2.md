## Findings

**[LINGUISTIC ACCURACY] [CRITICAL]**
Location: "Він, вона, воно" section, paragraph 2: "Now try **книга** (book): **вона книжка**? Yes — **жіночий рід** (feminine)."
Issue: The gender test starts with книга but tests книжка — a different lexeme (confirmed by Антоненко-Давидович ad-038: книга = folio/register, книжка = common book). A1 learners will think the він/вона/воно test transforms the word, or that книга and книжка are interchangeable. This directly undermines the gender test being taught.
Fix: Change "вона книжка" to "вона книга" to match the word being tested.

**[LINGUISTIC ACCURACY] [MAJOR]**
Location: "Предмети навколо" section, feminine list: "**Жіночий рід** (вона, моя) — **книжка** (book)"
Issue: The objects list uses книжка, but the example sentence immediately below says "**Це моя книга.**" and the summary table uses "книга (book)". The plan's vocabulary_hints specify "книга". This inconsistency between list and example is confusing.
Fix: Standardize to "книга" throughout to match plan vocabulary_hints.

**[LINGUISTIC ACCURACY] [MAJOR]**
Location: "Він, вона, воно" section, paragraph 3: "**У мене є книжка.**"
