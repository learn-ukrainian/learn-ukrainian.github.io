## Findings

**[EXERCISE QUALITY] [MAJOR]**
Location: Quiz 2, Підсумок section — `:::quiz title: "В чи на?..."`
Issue: All 10 correct answers are at index 0. The answer key is `a: 0` for every item. A learner who always picks the first option gets 100%.
Fix: Swap option order for roughly half the items so correct answers appear at varied indices.

**[EXERCISE QUALITY] [MAJOR]**
Location: Fill-in, В чи на? section — `:::fill-in title: "Де?..."`
Issue: The plan's activity_hint 2 specifies *"Answer Де?: Олена ___ (школа). Тарас ___ (робота)"* — testing locative form production. The actual exercise pre-declines nouns ("Олена ___ школі") and only tests preposition choice, duplicating quiz 2's skill. Locative form production (the core grammar skill of this module) is never tested in isolation.
Fix: Rewrite fill-in items to require the full locative form: "Олена ___ (школа)" → answer: "в школі".

**[PLAN ADHERENCE] [MINOR]**
Location: Dialogue 2 — Богдан's workplace description
Issue: Plan specifies "На вулиці Хрещатик. — А де ти працюєш? — В офісі, на другому поверсі." The module drops the floor detail ("на другому поверсі"), reducing the variety of locative examples.
