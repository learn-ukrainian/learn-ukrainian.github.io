## Linguistic Scan

**1. Grammatical error — verb agreement:**
- **Location:** Fill-in exercise #5 in "Бо і тому що" section
- **Text:** `"— Чому ти сміється? — ___ це смішно!"`
- **Issue:** `сміється` is 3rd person singular (`verb:rev:imperf:pres:s:3` per VESUM). With pronoun `ти`, the correct form is `смієшся` (`verb:rev:imperf:pres:s:2`). This is a subject-verb agreement error.
- **Fix:** `сміється` → `смієшся`

## Findings

```
[LINGUISTIC ACCURACY] [SEVERITY: critical]
Location: Fill-in exercise #5 in "Бо і тому що" section — "— Чому ти сміється? — ___ це смішно!"
Issue: "сміється" is 3rd person singular (VESUM: verb:rev:imperf:pres:s:3). With "ти", the correct conjugation is "смієшся" (verb:rev:imperf:pres:s:2). Subject-verb agreement error.
Fix: Replace "сміється" with "смієшся".
```

```
[EXERCISE QUALITY] [SEVERITY: major]
Location: Fill-in exercise "З'єднай причину: бо / тому що" — items 2, 4, 6
Issue: The module explicitly teaches "Both are correct. Both are Ukrainian. бо is NOT informal or wrong." Yet the exercise forces specific answers (e.g., item 2 "Він не прийшов на урок, ___ він захворів" requires "тому що" when "бо" is equally valid). The implicit pattern (question-response = бо, mid-sentence = тому що) is not taught in the prose. This contradicts the lesson and will confuse learners.
Fix: Either (a) add explicit teaching about when each tends to be preferred (бо in short responses, тому що in longer written sentences), or (b) accept both answers. Since this is a fill-in exercise with single answers, option (a) is better — add a short teaching note before the exercise, and adjust the sentence frames to make the expected answer more natural/unambiguous.
```

```
[PEDAGOGICAL QUALITY] [SEVERITY: minor]
Location: Quiz exercise placed after "Діалоги" section, before "Сполучники" formal instruction
Issue: In strict PPP methodology, Practice comes after Present. The quiz tests conjunction knowledge before conjunctions are formally taught. Learners can infer from the dialogues, but this is a discovery approach rather than PPP.
Fix: No change needed — the dialogues provide sufficient implicit input, and this functions as a transitional "noticing" activity. Acceptable variation within PPP.
```

## Verdict: REVISE

Two fixable issues: one critical grammatical error (сміється→смієшся) and one major exercise logic flaw (бо/тому що fill-in forces arbitrary answers). Both are surgically fixable without rewrite.
