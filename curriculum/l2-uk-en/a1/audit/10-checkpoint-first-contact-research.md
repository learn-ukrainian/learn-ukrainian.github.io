## State Standard Reference

**Checkpoint Purpose**: A checkpoint (контрольна точка) is a standardized assessment moment that verifies whether learner has achieved specific level requirements according to CEFR-aligned State Standard.

**A1 Assessment Components** (per State Standard):
1. **Аудіювання** (Listening) — understanding simple messages, announcements, instructions
2. **Читання** (Reading) — understanding short, simple texts (signs, announcements, simple letters)
3. **Письмо** (Writing) — writing short phrases, filling forms, simple greetings
4. **Говоріння** (Speaking) — simple self-introduction, answering basic questions

## Assessment Methodology for Checkpoints

**Theoretical Framework** (State Standard 2024):

### 1. Test-Teach-Test (TTT) Approach for Checkpoints

Unlike regular modules (PPP — Present-Practice-Produce), **checkpoints use TTT**:
- **Test** — Diagnostic assessment (what do you know?)
- **Teach** — Targeted review of weak areas
- **Test** — Summative assessment (can you do it now?)

### 2. Checkpoint vs Regular Module

**Regular Module**:
- Introduces NEW material
- Scaffolded instruction
- Practice activities build skills
- Production shows learning

**Checkpoint Module**:
- Tests EXISTING knowledge
- Reviews common errors
- Self-assessment tools
- Integration tasks (apply multiple skills)

### 3. Formative vs Summative Assessment

**A1 Checkpoint Purpose**:
- Formative: Help learner identify gaps before moving to A1.2
- Summative: Verify readiness for next phase
- Diagnostic: Show which areas need more practice
- Motivational: Celebrate what learner CAN do

### 4. Integration Tasks

**Key Principle**: Checkpoint should include **integration task** that requires using MULTIPLE skills from the phase:
- Example: Café dialogue uses Cyrillic reading (menu), conjugation (Я хочу), questions (Скільки коштує?), vocabulary (їжа), and politeness (будь ласка, дякую)

**Source**: [Standardized Requirements UCU](https://er.ucu.edu.ua/bitstreams/b5b6ee90-495e-45c3-ae73-8e03decfbcba/download)

## Pedagogical Notes for A1.1 Checkpoint

### Skill 1: Cyrillic Reading
- **Common Errors**: и/і confusion, е/є mixing, г/ґ substitution
- **Test Format**: Read word list, identify letters, match sound to letter
- **Success Criterion**: 90%+ accuracy on all 33 letters

### Skill 2: Gender Identification
- **Pattern Recognition**: -а/-я (fem), consonant (masc), -о/-е (neut)
- **Exceptions**: тато, батько, Михайло (masculine despite -о)
- **Test Format**: Classify 20 nouns by gender
- **Success Criterion**: 85%+ accuracy

### Skill 3: Verb Conjugation
- **Coverage**: First (-ати), Second (-ити), Reflexive (-ся/-сь)
- **Common Errors**: Mixing conjugation patterns, wrong endings for person
- **Test Format**: Conjugate verbs in sentences, choose correct form
- **Success Criterion**: 80%+ accuracy across all three types

### Skill 4: Questions and Negation
- **Question Words**: хто, що, де, коли, як, чому
- **Intonation**: Rising intonation for так/ні questions
- **Negation**: не + verb
- **Test Format**: Form questions from statements, negate sentences
- **Success Criterion**: 85%+ accuracy

### Skill 5: Practical Communication
- **Context**: Café ordering (authentic A1 scenario)
- **Functions**: Request (Можна мені...?), desire (Я хочу...), politeness
- **Test Format**: Role-play dialogue, respond to prompts
- **Success Criterion**: Can complete transaction with comprehensible Ukrainian

## Cultural Context

**Checkpoint Philosophy**: In Ukrainian educational tradition, контрольна робота (control work) is:
- Serious but supportive
- Opportunity for self-assessment, not punishment
- Shows respect for learner's progress
- Cultural value: честь (honor) — you prove to yourself you know it

**Pedagogical Tone**: Warm encouragement + clear standards. "Ти вже багато знаєш!" (You already know a lot!) + concrete feedback on gaps.

## Notes for Content Generation

- **DO NOT teach new material** — checkpoint reviews only
- Use TTT structure explicitly
- Include self-assessment rubrics (Can I...? Yes/Mostly/Not Yet)
- Provide **diagnostic feedback** for each skill
- Integration task should feel authentic (café dialogue is perfect)
- Celebrate progress: "Ти пройшов A1.1 — це 25% шляху до базового володіння!"
