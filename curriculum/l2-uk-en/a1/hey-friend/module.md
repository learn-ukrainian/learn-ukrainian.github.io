A1.7 begins with a very common Ukrainian habit: when you speak **to** a
person, the name or role often has a calling form.

You already know how to talk **about** people:

- **Це Олена.**
- **Це Тарас.**
- **Це мій друг.**
- **Це моя мама.**

Now learn how to call them:

- **Олено, привіт!**
- **Тарасе, ходи сюди!**
- **Гей, друже!**
- **Мамо, де телефон?**

By the end, you can:

- recognize the practical calling form, the **кличний відмінок**;
- call common names and family words: **Олено**, **Тарасе**, **мамо**,
  **бабусю**;
- use friend and polite address phrases: **друже**, **пане**, **пані**;
- choose the reliable A1 endings **-о**, **-е**, and **-ю**;
- avoid copied or shortened forms that do not fit standard Ukrainian address.

Keep this lesson practical. You are not learning a full case lecture. You are
learning what to say when you want a real person to look at you.

:::tip[Fast check]
If the person should answer you, use the calling form: **Олено! Тарасе!
Друже! Мамо!**
:::

## Діалоги

<!-- INJECT_ACTIVITY: act-1 -->

### На вечірці

```text
Олена: Привіт, Тарасе!
Тарас: Привіт, Олено! Як справи?
Олена: Добре, дякую. А це мій брат Андрій.
Тарас: Андрію, привіт!
Андрій: Привіт, друже!
Олена: Тарасе, ходи сюди. Тут наша подруга Марія.
Тарас: Маріє, радий тебе бачити!
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Олена: Привіт, Тарасе!** | Olena: Hi, Taras! |
| **Тарас: Привіт, Олено! Як справи?** | Taras: Hi, Olena! How are you? |
| **Олена: Добре, дякую. А це мій брат Андрій.** | Olena: Good, thanks. And this is my brother Andrii. |
| **Тарас: Андрію, привіт!** | Taras: Andrii, hi! |
| **Андрій: Привіт, друже!** | Andrii: Hi, friend! |
| **Олена: Тарасе, ходи сюди. Тут наша подруга Марія.** | Olena: Taras, come here. Our friend Mariia is here. |
| **Тарас: Маріє, радий тебе бачити!** | Taras: Mariia, glad to see you! |

Notice the small switch:

| About the person | To the person |
| --- | --- |
| **Олена** тут. | **Олено**, привіт! |
| **Тарас** тут. | **Тарасе**, ходи сюди! |
| Це **Андрій**. | **Андрію**, привіт! |
| Це **Марія**. | **Маріє**, радий тебе бачити! |

The English sentence keeps the same name. Ukrainian changes the form when the
speaker addresses the person directly. Treat the form as a ready calling
shape: **Олено**, **Тарасе**, **Андрію**, **Маріє**.

### Удома

```text
Син: Мамо, де мій телефон?
Мама: На столі, синку.
Дочка: Тату, а де ключі?
Тато: У кишені, дочко.
Олена: Бабусю, ми йдемо!
Бабуся: Добре, діти. Будьте обережні!
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Син: Мамо, де мій телефон?** | Son: Mom, where is my phone? |
| **Мама: На столі, синку.** | Mom: On the table, son. |
| **Дочка: Тату, а де ключі?** | Daughter: Dad, and where are the keys? |
| **Тато: У кишені, дочко.** | Dad: In the pocket, daughter. |
| **Олена: Бабусю, ми йдемо!** | Olena: Grandma, we are leaving! |
| **Бабуся: Добре, діти. Будьте обережні!** | Grandma: All right, children. Be careful! |

Family words are especially useful as whole phrases:

- **мамо** - Mom!
- **тату** - Dad!
- **синку** - son!
- **дочко** - daughter!
- **бабусю** - Grandma!
- **дідусю** - Grandpa!

Do not shorten **мамо** to a clipped form. Use the full Ukrainian address
phrase: **Мамо, де телефон?**

## Кличний відмінок

The **кличний відмінок** is the Ukrainian calling form. Use it when you speak
directly to a person.

Compare:

| Sentence | Meaning | What is happening? |
| --- | --- | --- |
| **Олена прийшла.** | Olena came. | We talk about Olena. |
| **Олено, привіт!** | Olena, hi! | We speak to Olena. |
| **Тарас тут.** | Taras is here. | We talk about Taras. |
| **Тарасе, ходи сюди!** | Taras, come here! | We call Taras. |
| **Друг чекає.** | A friend is waiting. | We talk about a friend. |
| **Гей, друже!** | Hey, friend! | We call a friend. |

This is why the module title is **Гей, друже!** The dictionary word is
**друг**, but the friendly address is **друже**.

At A1, use a small set of trusted address patterns.

| Pattern | Dictionary form | Calling form |
| --- | --- | --- |
| feminine **-а** | **Олена**, **мама**, **сестра**, **подруга** | **Олено**, **мамо**, **сестро**, **подруго** |
| feminine **-ія** | **Марія**, **Надія** | **Маріє**, **Надіє** |
| feminine **-ся** | **бабуся** | **бабусю** |
| masculine hard consonant | **Тарас**, **Іван**, **брат**, **пан** | **Тарасе**, **Іване**, **брате**, **пане** |
| masculine **-й** or soft sound | **Андрій**, **дідусь**, **вчитель** | **Андрію**, **дідусю**, **вчителю** |
| friend phrase | **друг** | **друже** |

You do not need to explain every Ukrainian name. You need to notice the common
shape and use the common line. If you are speaking to the person, choose the
calling form.

:::note[A1 boundary]
Treat **ходи сюди**, **ходімо**, and **Будьте обережні** as ready phrases here.
The next module teaches polite commands more directly; this module's job is the
address form before the phrase.
:::

## Закінчення кличного відмінка

Start with the three endings you will meet most often.

### -о

Many feminine names and family words on **-а** change to **-о**:

- **Олена -> Олено**
- **Оксана -> Оксано**
- **мама -> мамо**
- **сестра -> сестро**
- **подруга -> подруго**

Use them in short lines:

- **Олено, привіт!**
- **Мамо, де телефон?**
- **Сестро, ходи сюди.**
- **Подруго, дякую!**

### -е

Many masculine names and words with a hard final consonant add **-е**:

- **Тарас -> Тарасе**
- **Іван -> Іване**
- **брат -> брате**
- **пан -> пане**

Use them in short lines:

- **Тарасе, як справи?**
- **Іване, це моя сестра.**
- **Брате, ходімо.**
- **Пане Іване, добрий день.**

### -ю

Masculine names and words with **-й** or a soft ending often use **-ю**:

- **Андрій -> Андрію**
- **Сергій -> Сергію**
- **дідусь -> дідусю**
- **вчитель -> вчителю**

Use them in short lines:

- **Андрію, привіт!**
- **Сергію, дякую.**
- **Дідусю, ми вдома.**
- **Вчителю, можна питання?**

Some calling forms are best learned as phrases. **Друг -> друже** has a sound
change, but at A1 you do not need a sound-change rule. Just keep **друже** as
the friend-address word: **Привіт, друже!**

### Ввічливе звертання

For polite address, use **пан**, **пані**, and the person's name carefully.

| About the person | To the person |
| --- | --- |
| **пан Іван** | **пане Іване** |
| **пан Сергій** | **пане Сергію** |
| **пані Оксана** | **пані Оксано** |
| **пані Марія** | **пані Маріє** |

The role word and the name both need attention:

- **Добрий день, пане Іване!**
- **Дякую, пані Оксано.**
- **Пане Сергію, ходімо.**
- **Пані Маріє, ось меню.**

:::tip[Polite address]
With **пан** and **пані**, keep the line short and respectful: **Добрий день,
пане Іване!** or **Пані Оксано, ось меню.**
:::

At A1, do not build long formal titles or patronymics. Recognize that
**Володимире Олексійовичу** exists, but do not drill it here. Your active
goal is the useful daily line: **пане Іване**, **пані Оксано**.

Also keep the adjective simple if it appears before the address. Say
**Шановний пане Іване!**, not a changed adjective form. This is useful for
recognition, but the everyday A1 path is shorter: **Добрий день, пане Іване!**

### Типові помилки

The main learner error is using the dictionary form when speaking directly to
someone. Repair it immediately:

| Problem | Use |
| --- | --- |
| <bad>Привіт, Антон!</bad> | **Привіт, Антоне!** |
| <bad>Добрий ранок, мій друг!</bad> | **Добрий ранок, мій друже!** |
| <bad>Пане Сергій!</bad> | **Пане Сергію!** |
| <bad>Привіт, Надія!</bad> | **Привіт, Надіє!** |
| <bad>Шановному пане Іване!</bad> | **Шановний пане Іване!** |
| <bad>Я є Джон.</bad> | **Мене звати Джон.** |

The last line is not a vocative ending. It is a familiar introduction repair:
use **Мене звати...** for "My name is...". A greeting lesson often brings this
back because learners meet names and address at the same time.

Keep this A1 lesson active, not encyclopedic:

- use **Олено**, **Маріє**, **мамо**, **подруго** for the common feminine
  address shapes here;
- use **Тарасе**, **Іване**, **брате**, **пане** for the common masculine
  hard-consonant shapes here;
- use **Андрію**, **Сергію**, **дідусю**, **вчителю** for the soft or **-й**
  shapes here;
- keep **друг -> друже** as a memorized friend-address phrase.

For Ukrainian cultural and decolonized language, use standard Ukrainian names
and address phrases. Say **Маріє** for **Марія** in a neutral address, and use
**Марійко** only when the relationship is close and the diminutive fits. Say
**Світлано** for **Світлана**. Say **тітко Світлано** or **дядьку Петре** when
the family role is part of the address. Avoid copied non-Ukrainian name shapes
and clipped family calls.

:::caution[Clean Ukrainian choices]
Use Ukrainian etiquette words cleanly:

- **«Добрий день»**, not <bad>«Здрастуйте»</bad> or
  <bad>«Здравствуйте»</bad>.
- **«До побачення»**, not <bad>«Пока»</bad>.
- **«тато»**, not <bad>«папа»</bad> for a family father.
- **«прізвище»**, not <bad>«фамілія»</bad> for surname.
:::

## Підсумок

Use this small map when you need to call someone:

| Dictionary form | Calling form | A1 line |
| --- | --- | --- |
| **Олена** | **Олено** | **Олено, привіт!** |
| **Марія** | **Маріє** | **Маріє, дякую.** |
| **Тарас** | **Тарасе** | **Тарасе, ходи сюди.** |
| **Андрій** | **Андрію** | **Андрію, як справи?** |
| **мама** | **мамо** | **Мамо, де телефон?** |
| **тато** | **тату** | **Тату, я вдома.** |
| **бабуся** | **бабусю** | **Бабусю, ми йдемо.** |
| **друг** | **друже** | **Гей, друже!** |
| **пан Іван** | **пане Іване** | **Добрий день, пане Іване!** |

Self-check:

- Are you talking **about** the person? Use the dictionary form:
  **Олена тут.**
- Are you speaking **to** the person? Use the calling form:
  **Олено, привіт!**
- Is the address a phrase you already know? Use it whole:
  **мамо**, **тату**, **друже**, **пане**, **пані**.

Keep the clean Ukrainian etiquette set in the same place as your address
forms: **«Добрий день»**, **«До побачення»**, **«тато»**, and
**«прізвище»**.

Next module adds small polite commands and requests. That will pair naturally
with address forms: **Олено, зроби це, будь ласка.** For now, keep the command
simple and focus on the address: **Олено! Тарасе! Друже! Мамо!**

### Ремонти

| Problem | Use |
| --- | --- |
| dictionary name as address | use **Олено, привіт! / Тарасе, ходи сюди!** |
| friend addressed with the dictionary word | use **Гей, друже!** |
| clipped family call | use **Мамо, де телефон?** |
| borrowed name shape for Марія | use **Маріє** or close **Марійко** |
| borrowed name shape for Світлана | use **Світлано** |
| missing double address | use **пане Іване / пані Оксано** |
| over-explaining at A1 | learn **Олено**, **Тарасе**, **Андрію**, **мамо**, **друже** as useful calling forms |
