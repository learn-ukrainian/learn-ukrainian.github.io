# Дослідження: Adjective Case Forms

## State Standard Reference
§4.2.1.2: "Adjective declension — hard/soft groups, gender agreement"
Alignment: This module introduces Accusative and Locative adjective forms — the two cases already taught at A1. It applies §4.2.1.2 in a narrowed scope: hard-group adjectives only (soft group -ній deferred to A2), agreement across three genders.

## Cultural Hooks

1. **Ukrainian riddle (загадка)** using adjective agreement: *Він росте у чистім полі / На високих ніжках, / У зелених панчішках.* (from Avramenko Grade 6 textbook — a cornflower riddle). Strip adjectives and the description collapses — a natural discovery of adjective importance.
2. **Café/market descriptions**: Ordering smачну каву (acc.) vs asking Де смачна кава? (nom.) — everyday Ukrainian situations where the same adjective takes different forms. Kyiv café culture makes this feel real.
3. **City navigation**: *Я живу у старому місті* / *Я бачу старе місто* — Lviv's old city (Старе Місто) as concrete cultural anchor. Learners can immediately use these forms.

## Common Learner Errors

1. **Frozen adjective** → *красивий книгу* instead of *красиву книгу* — happens because English adjectives never inflect; learners change the noun but forget the adjective must follow. Textbook model: Avramenko Grade 6 exercises contrasting Н.в. vs Зн.в. pairs (*їде автомобіль* vs *ремонтують автомобіль*) — the adjective analog is the same drill.
2. **Animate masculine Accusative** → *Я бачу новий студент* instead of *нового студента* — English speakers don't distinguish animate/inanimate grammatically; this is the hardest A1 adjective rule.
3. **Locative default to Nominative** → *у новий місто* instead of *у новому місті* — "in + [unchanged adjective]" mirrors English "in new city"; -ому ending feels arbitrary without drilling.
4. **Feminine Locative -ій vs -у confusion** → *у новій книгу* or *у нову книзі* — mixing the two feminine oblique endings (-у for Acc., -ій for Loc.) because both are changes from -а.

## Teaching Strategy
- **Discovery exercise**: Show three café sentences in a table with no explanation — *Це нова кав'ярня.* / *Я люблю нову кав'ярню.* / *Я в новій кав'ярні.* Ask: "Same café, same adjective — why do the endings change?" Let learners articulate the pattern (same noun, different role in sentence) before labeling it. This mirrors the Zabolotnyi Grade 6 Н.в./Зн.в. comparison format.
- **Best context**: Shopping + café ordering (Accusative: "I want смачну каву") and giving locations (Locative: "I work у великому офісі"). Both are immediately practical.
- **Contrastive pairs**: Run a 3×3 grid — Nom/Acc/Loc × Masc/Fem/Neut — using one adjective (новий) across all cells. Visual pattern recognition beats memorizing rules. Highlight the zero-change cells (Masc/Neut Acc inanimate = Nom) to reduce load.

## Notes for Content Writing
- The plan's persona is "Fashion Advisor" — use clothing and style context to anchor examples (красивий светр, нову сукню, у модному магазині) alongside city/café contexts.
- Soft-group adjectives (синій → синьому) appear in the plan's vocabulary_hints (на синьому морі) — include 1-2 soft examples but don't teach the rule; mark them as "you'll see this pattern more in A2."
- Hard cap: inanimate masc. Acc = Nom is a *relief* — stress this reduces workload. Learner morale matters.
- Gender-agreement reminder: if the noun gender was taught in earlier modules (the-gender-code), remind learners that knowing gender is now paying off.
