# Вправи

## quiz: Частина 1 — Граматика

> Перевірте граматичні знання.

1. Виберіть правильну форму: «Якби я ___ більше часу, я б поїхав у Карпати».
   - [x] мав
   - [ ] маю
   - [ ] матиму
   > Умовний спосіб: «якби + минулий час».

2. Який відмінок після «біля»?
   - [ ] називний
   - [x] родовий
   - [ ] давальний
   > Біля + родовий: «біля столу», «біля хати».

3. Виберіть доконаний вид: «Вчора я ___ цю книгу».
   - [ ] читав
   - [x] прочитав
   - [ ] читаю
   > Прочитав (pf) = завершена дія.

4. Яке закінчення орудного відмінка для «лікар»?
   - [ ] -а
   - [ ] -у
   - [x] -ем
   > Лікар → лікарем (орудний).

5. Виберіть правильний сполучник: «Він прийшов, ___ допомогти».
   - [x] щоб
   - [ ] що
   - [ ] який
   > Щоб + infinitive для мети.

6. Яка форма пасивного стану: «Цей роман ___ Франком»?
   - [x] був написаний
   - [ ] написав
   - [ ] пише
   > Пасивний: «був написаний» (was written).

7. Виберіть правильну форму: «Я бачу ___ (він)».
   - [ ] він
   - [x] його
   - [ ] йому
   > Знахідний відмінок: бачу кого? — його.

8. Що означає «читаючи книгу»?
   - [x] while reading a book
   - [ ] having read a book
   - [ ] to read a book
   > Дієприслівник теперішнього часу.

## quiz: Частина 2 — Лексика

> Перевірте словниковий запас.

1. Що таке «вакансія»?
   - [x] job opening
   - [ ] vacation
   - [ ] vacancy (hotel)
   > Вакансія = відкрита позиція на роботі.

2. Як сказати «I have a headache»?
   - [x] У мене болить голова
   - [ ] У мене біль голови
   - [ ] Моя голова біль
   > «Болить» + частина тіла.

3. Що означає «заголовок»?
   - [x] headline
   - [ ] header
   - [ ] goal
   > Заголовок = headline (стаття, новина).

4. Яке слово означає «to book (a hotel)»?
   - [ ] бронза
   - [x] забронювати
   - [ ] бронхіт
   > Забронювати номер = to book a room.

5. Що таке «співбесіда»?
   - [ ] conversation
   - [x] job interview
   - [ ] meeting
   > Співбесіда = job interview.

6. Як сказати «discount»?
   - [ ] скидка
   - [x] знижка
   - [ ] зниження
   > Знижка (not скидка — that's Russian).

7. Що означає «підписник»?
   - [ ] subscriber (newspaper)
   - [x] follower (social media)
   - [ ] signature
   > У соцмережах підписник = follower.

8. Яке слово означає «environment»?
   - [x] довкілля
   - [ ] середовище
   - [ ] оточення
   > Довкілля = environment (ecology).

## quiz: Частина 3 — Культура

> Перевірте знання української культури.

1. Хто написав «Кобзар»?
   - [x] Тарас Шевченко
   - [ ] Іван Франко
   - [ ] Леся Українка
   > «Кобзар» — найвідоміший збірник Шевченка.

2. Яке місто відоме кав'ярнями та австрійською архітектурою?
   - [ ] Київ
   - [x] Львів
   - [ ] Одеса
   > Львів — «кавова столиця» України.

3. Що таке Буковель?
   - [ ] місто
   - [x] лижний курорт
   - [ ] національний парк
   > Буковель — найбільший лижний курорт України.

4. Хто такий Олександр Усик?
   - [ ] футболіст
   - [x] боксер
   - [ ] співак
   > Усик — чемпіон світу з боксу.

5. Що таке вишиванка?
   - [ ] вишневий торт
   - [x] традиційна сорочка з вишивкою
   - [ ] тип танцю
   > Вишиванка — традиційний одяг з вишивкою.

6. Яке свято відзначають 24 серпня?
   - [x] День Незалежності
   - [ ] Великдень
   - [ ] Різдво
   > 24 серпня 1991 — проголошення незалежності України.

7. Що таке Ukrainer?
   - [ ] соціальна мережа
   - [x] медіапроєкт з документальними відео
   - [ ] новинний сайт
   > Ukrainer документує Україну через відео.

8. Яка головна страва української кухні?
   - [ ] вареники
   - [x] борщ
   - [ ] голубці
   > Борщ — символ української кухні.

## quiz: Частина 4 — Практичні ситуації

> Перевірте практичні навички.

1. Як запитати ціну в магазині?
   - [x] Скільки це коштує?
   - [ ] Яка вартість?
   - [ ] Що за гроші?
   > «Скільки це коштує?» — стандартне питання.

2. Як сказати «Bill, please» у ресторані?
   - [x] Рахунок, будь ласка
   - [ ] Білль, будь ласка
   - [ ] Чек, будь ласка
   > Рахунок = bill (у ресторані).

3. Який номер екстреної допомоги?
   - [ ] 911
   - [x] 112
   - [ ] 999
   > 112 — єдиний номер екстреної допомоги в Україні.

4. Як запитати «What time is the departure?»
   - [x] О котрій відправлення?
   - [ ] Коли їхати?
   - [ ] Яка година відправлення?
   > «О котрій відправлення?» — стандартна фраза.

5. Як сказати «I need a doctor»?
   - [ ] Я хочу лікаря
   - [x] Мені потрібен лікар
   - [ ] Дайте лікаря
   > «Мені потрібен/потрібна» — стандартна конструкція.

6. Як запитати про Wi-Fi в готелі?
   - [x] Який пароль від Wi-Fi?
   - [ ] Є інтернет?
   - [ ] Wi-Fi працює?
   > Пряме питання про пароль.

7. Як сказати «Can I try it on?» (одяг)
   - [ ] Можна одягнути?
   - [x] Можна приміряти?
   - [ ] Можна взяти?
   > Приміряти = to try on (clothes).

8. Як попросити зупинку в таксі?
   - [x] Зупиніть тут, будь ласка
   - [ ] Стоп тут
   - [ ] Тут вихід
   > «Зупиніть тут, будь ласка» — ввічлива форма.
