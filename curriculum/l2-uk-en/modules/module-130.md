# Зміст уроку | Lesson Content

## warm-up | Розминка

### Зв'язність мови | Language Coherence

Слова-зв'язки роблять мову плавною та зрозумілою:

Discourse markers make language smooth and clear:

**Без зв'язок:** Я хотів поїхати. Погода була погана. Я залишився вдома.

**Зі зв'язками:** Я хотів поїхати, **але** погода була погана, **тому** я залишився вдома.

## practice | Практика

### Choose the Correct Marker

1. Він хотів прийти, ___ (але/тому) не зміг.
   → **але**

2. Я не люблю каву. ___ (Навпаки/Тому), я п'ю чай.
   → **Тому**

3. ___ (По-перше/Нарешті), це дорого. ___ (По-друге/Отже), це непотрібно.
   → **По-перше**, **По-друге**

4. ___ (Крім того/Проте), він ще й запізнився.
   → **Крім того**

5. ___ (Завдяки/Через) йому ми виграли.
   → **Завдяки**

### Connect the Sentences

Use appropriate markers:

1. Погода погана. Ми підемо на прогулянку.
   → **Хоча погода погана, ми підемо на прогулянку.**

2. Він багато працює. Він заробляє добре.
   → **Він багато працює, тому заробляє добре.**

3. Це цікаво. Це корисно.
   → **Це не тільки цікаво, а й корисно.**

4. Він не прийшов. Він захворів.
   → **Він не прийшов, тому що захворів.**

### Complete the Text

Fill in discourse markers:

(1)___, я хочу розповісти про свою поїздку. (2)___, ми поїхали до Львова. (3)___ погода була погана, (4)___ ми чудово провели час. (5)___, ми побачили багато цікавого. (6)___, Львів — прекрасне місто.

**Answers:**
1. По-перше / Спочатку
2. Отже
3. Хоча
4. проте / але
5. Загалом
6. Словом / У підсумку

### Translate Using Markers

1. However, he didn't come.
   → **Проте він не прийшов.**

2. In my opinion, this is wrong.
   → **На мою думку, це неправильно.**

3. In conclusion, we succeeded.
   → **У підсумку, ми досягли успіху.**

4. Not only is it expensive, but also unnecessary.
   → **Це не тільки дорого, а й непотрібно.**

## production | Продукція

### Structure an Argument

Present your opinion on a topic using markers:

**Тема:** Чи потрібно вивчати іноземні мови?

**Structure:**
- По-перше... (first argument)
- По-друге... (second argument)
- Крім того... (additional point)
- Хоча... (counterargument)
- У підсумку... (conclusion)

### Write a Paragraph

Write about your favorite city using:
- addition markers (також, крім того)
- contrast markers (але, проте)
- conclusion markers (отже, словом)

# Summary

## Discourse Markers: Key Points

### By Function

| Function | Markers |
|----------|---------|
| Addition | також, крім того, до того ж |
| Contrast | але, проте, однак, хоча |
| Cause | тому що, через те що |
| Effect | тому, отже, таким чином |
| Sequence | по-перше, потім, нарешті |
| Example | наприклад, зокрема |
| Conclusion | отже, у підсумку, словом |

### Usage Tips

1. Use markers to structure arguments
2. Vary your markers (don't repeat)
3. Match formality level to context
4. Place markers at sentence start or mid-sentence

> **Key Takeaways**
>
> - Discourse markers make speech coherent
> - Different markers for different functions
> - Essential for academic and formal writing
> - Practice combining markers in paragraphs
> - Markers show logical relationships between ideas
