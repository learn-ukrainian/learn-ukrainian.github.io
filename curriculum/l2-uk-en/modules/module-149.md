# Вступ

Ви хочете взяти відпустку. Підходите до директора й кажете: «Слухайте, мені треба відпочити. Давайте я два тижні не приходитиму?» Директор дивиться на вас, як на інопланетянина: «Напишіть заяву.» І ось тут починається квест.

«Заява» — це не просто папірець. Це **ритуал**. Зверху праворуч — кому. Нижче — від кого. Посередині великими літерами — «ЗАЯВА». Далі — магічні слова «Прошу надати мені...». І підпис знизу. Якщо щось не так — повертають переписувати.

Офіційно-діловий стиль — це мова, де **форма важливіша за зміст**. Ви можете геніально пояснити, чому вам потрібна відпустка, але якщо заява оформлена неправильно — її не приймуть. Це стиль, де панують **шаблони, кліше та канцеляризми** — і без них нікуди.

> **Чи знали ви?**
>
> Слово **«канцелярія»** походить від латинського *cancelli* — «решітка». У Римській імперії так називали загородку, за якою сиділи писарі. Згодом слово стало означати установу, де пишуть документи. А «канцеляризм» — це мовний штамп, народжений за тією решіткою. «Вищезазначений», «нижчепідписаний», «належним чином» — усі ці слова живуть тільки в документах.

# Типи офіційних документів

| Тип | Функція | Типова формула |
|-----|---------|----------------|
| **Заява** | Прохання | Прошу надати... |
| **Довідка** | Підтвердження | Довідка видана... |
| **Наказ** | Розпорядження | Наказую... |
| **Протокол** | Фіксація засідання | Слухали... Ухвалили... |
| **Договір** | Угода сторін | Сторони домовилися... |
| **Скарга** | Претензія | Прошу розглянути скаргу... |
| **Акт** | Фіксація факту | Цим актом засвідчується... |

# Структура заяви

## Шаблон

```
                                    Директору ТОВ «Компанія»
                                    Петренку Івану Михайловичу
                                    від менеджера відділу продажу
                                    Коваленко Олени Петрівни

                        ЗАЯВА

Прошу надати мені щорічну оплачувану відпустку з 01.07.2024
по 14.07.2024 тривалістю 14 календарних днів.

25.06.2024                          _________ О.П. Коваленко
                                    (підпис)
```

## Анатомія заяви

| Елемент | Де | Приклад |
|---------|-----|---------|
| Адресат | Вгорі праворуч | Директору... Петренку І.М. |
| Заявник | Під адресатом | від... Коваленко О.П. |
| Назва | По центру | ЗАЯВА |
| Текст | Основна частина | Прошу надати... |
| Дата | Знизу зліва | 25.06.2024 |
| Підпис | Знизу справа | _________ О.П. Коваленко |

> **Міф vs Факт**
>
> ❌ **Міф:** «Головне — написати, що хочеш. Форма не важлива.»
>
> ✅ **Факт:** В офіційному стилі **форма — це закон**. Заяву з неправильною структурою можуть не прийняти. Тому шаблони — не бюрократія, а необхідність. Вони гарантують, що документ буде зрозумілий і дійсний.

# Магічні формули

## Початок

| Формула | Коли вживати |
|---------|--------------|
| **Прошу...** | заяви, прохання |
| **Повідомляю...** | інформування |
| **Звертаюся з проханням...** | листи |
| **Відповідно до...** | посилання на закон |
| **У зв'язку з...** | пояснення причини |
| **З метою...** | пояснення мети |

## Середина

| Формула | Приклад |
|---------|---------|
| **відповідно до** | відповідно до статті 5 Закону України |
| **у зв'язку з** | у зв'язку з сімейними обставинами |
| **на підставі** | на підставі поданих документів |
| **згідно з** | згідно з договором № 123 |
| **з огляду на** | з огляду на вищевикладене |

## Завершення

| Формула | Коли |
|---------|------|
| **Прошу розглянути...** | заяви, скарги |
| **Очікую на Вашу відповідь...** | листи |
| **Заздалегідь вдячний(-а)...** | прохання |
| **З повагою...** | усі листи |

> **Чи знали ви?**
>
> Формула **«Прошу надати»** — це не просто ввічливість. Юридично це **волевиявлення** — офіційне вираження вашого бажання. Без цієї формули заява може бути визнана недійсною. Тому «Хочу відпустку» ніколи не замінить «Прошу надати мені відпустку».

# Кліше офіційного стилю

| Кліше | Значення | Де вживається |
|-------|----------|---------------|
| **вищезазначений** | mentioned above | посилання на попередній текст |
| **нижчепідписаний** | the undersigned | про автора документа |
| **в установленому порядку** | in due course | про процедуру |
| **належним чином** | properly | про оформлення |
| **за наявності** | if available | умова |
| **за відсутності** | in the absence of | умова |
| **довести до відома** | to inform | про інформування |
| **взяти до уваги** | to take into account | про врахування |
| **у найкоротший термін** | as soon as possible | про терміновість |

# Офіційний лист

## Структура

1. **Шапка** — адресат, відправник
2. **Звертання** — «Шановний пане/пані!»
3. **Вступ** — мета листа
4. **Основна частина** — деталі
5. **Завершення** — очікування, подяка
6. **Підпис** — посада, ім'я

## Приклад

```
                                    Директору ТОВ «Партнер»
                                    Шевченку Віктору Івановичу

Шановний пане Шевченко!

Звертаюся до Вас з приводу укладання договору
про співпрацю між нашими компаніями.

Відповідно до попередніх домовленостей, пропонуємо
розпочати співпрацю з 01.08.2024. Умови договору
надсилаємо у додатку.

Прошу повідомити про Ваше рішення до 15.07.2024.

З повагою,
Директор ТОВ «Компанія»
_________________ І.М. Петренко
```

# Скарга: структура

## Елементи

1. **Адресат** — кому скаржитесь
2. **Заявник** — хто скаржиться (+ контакти)
3. **Назва** — СКАРГА
4. **Суть** — що трапилось (факти, дати)
5. **Вимоги** — чого хочете
6. **Додатки** — докази

## Приклад

```
                                    Директору магазину «Техніка»
                                    від покупця Коваленко О.П.
                                    тел.: +380 XX XXX XX XX

                        СКАРГА

15.06.2024 я придбав у Вашому магазині телевізор Samsung
(чек № 12345). Через тиждень виявилася несправність екрана.

Відповідно до Закону України «Про захист прав споживачів»,
прошу:
1. Замінити товар на аналогічний належної якості.
2. Або повернути кошти у повному обсязі.

Додатки:
1. Копія чека.
2. Гарантійний талон.
3. Фото несправності.

22.06.2024                          _________ О.П. Коваленко
```

> **Чи знали ви?**
>
> Посилання на **Закон України «Про захист прав споживачів»** — не просто формальність. Це юридичний аргумент, що зобов'язує продавця реагувати. Без цього посилання скарга — просто «хотілка». З посиланням — правова вимога.

# Безособовість: ключ до офіційності

В офіційному стилі **«я» майже не існує**:

| Особова форма | Безособова форма |
|---------------|------------------|
| Я прошу | **Прошу** |
| Я наказую | **Наказую** / **Наказано** |
| Ми ухвалили | **Ухвалено** |
| Вони підписали | **Підписано** |
| Мене звільнили | **Працівника звільнено** |

> **Міф vs Факт**
>
> ❌ **Міф:** «Канцеляризми — це застаріла совкова мова, їх треба уникати.»
>
> ✅ **Факт:** Канцеляризми **необхідні в офіційних документах**. «Здійснити оплату» в заяві — правильно. «Здійснити оплату» в розмові з другом — смішно. Проблема не в канцеляризмах, а в їх **недоречному вживанні**.

# Практика

## Напишіть заяву

**Ситуація:** Ви хочете взяти 3 дні відпустки без збереження зарплати через сімейні обставини.

**Рішення:**
```
                                    Директору ТОВ «Компанія»
                                    [ПІБ директора]
                                    від [посада]
                                    [Ваше ПІБ]

                        ЗАЯВА

Прошу надати мені відпустку без збереження заробітної плати
з 01.07.2024 по 03.07.2024 тривалістю 3 календарних дні
у зв'язку з сімейними обставинами.

[Дата]                              _________ [Підпис, ПІБ]
```

## Трансформуйте в офіційний стиль

| Розмовне | Офіційне |
|----------|----------|
| Хочу відпустку. | **Прошу надати мені відпустку.** |
| Мені треба довідка. | **Прошу видати мені довідку.** |
| Телевізор зламався, поверніть гроші! | **Прошу повернути кошти за несправний товар.** |
| Я згоден. | **Погоджено.** |
| Бо захворів. | **У зв'язку з хворобою.** |

# Підсумок

## Структура документів

| Елемент | Функція |
|---------|---------|
| Адресат | Кому |
| Заявник | Від кого |
| Назва | ЗАЯВА, СКАРГА |
| Текст | Суть справи |
| Дата, підпис | Засвідчення |

## Ключові формули

| Функція | Формула |
|---------|---------|
| Прохання | Прошу надати/видати... |
| Причина | У зв'язку з... |
| Посилання | Відповідно до... |
| Завершення | З повагою, |

> **Запам'ятайте:**
>
> ✓ Офіційний стиль = **форма + шаблон + кліше**
> ✓ «Прошу надати» — не ввічливість, а **юридичне волевиявлення**
> ✓ Безособові форми (**-но/-то**) — норма
> ✓ Канцеляризми **доречні в документах**, не в розмові
> ✓ Посилання на закон — **правова сила** скарги
