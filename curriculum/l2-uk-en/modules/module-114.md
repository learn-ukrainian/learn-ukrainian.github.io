## presentation | Презентація

### Complex Sentence Summary

| Type | Conjunction | Example |
|------|-------------|---------|
| **Relative** | який, яка | Людина, яка тут працює... |
| **Relative** | де, куди, коли | Місто, де я живу... |
| **Purpose (same subj)** | щоб + inf | Я прийшов, щоб допомогти. |
| **Purpose (diff subj)** | щоб + past | Я хочу, щоб ти прийшов. |
| **Real conditional** | якщо | Якщо матиму час, прийду. |
| **Unreal conditional** | якби...б | Якби мав час, прийшов би. |
| **Concessive** | хоча | Хоча йде дощ, я піду. |
| **Concessive** | незважаючи на | Незважаючи на дощ... |
| **Causal** | тому що, бо | ...тому що захворів. |
| **Causal** | через + noun | ...через хворобу. |
| **Result** | тому, отже | ...тому не прийшов. |
| **Temporal** | коли | Коли він прийшов... |
| **Temporal** | після того як | Після того як закінчив... |
| **Temporal** | поки (не) | Чекай, поки він не прийде. |

### Relative Clauses Review

**Який declines for gender/number/case:**

| Function | Example |
|----------|---------|
| Subject | Людина, **яка** прийшла... |
| Direct object | Книга, **яку** я читаю... |
| Possession | Друг, **якого** батько... |
| After preposition | Місто, в **якому** я живу... |

**Alternatives:**
- де = в якому/якій (location)
- куди = до якого (direction)
- коли = у який (time)

### Purpose Clauses Review

| Structure | When | Example |
|-----------|------|---------|
| **щоб + infinitive** | Same subject | Я прийшов, щоб допомогти. |
| **щоб + past** | Different subjects | Я хочу, щоб ти допоміг. |

### Conditionals Review

| Type | Structure | Example |
|------|-----------|---------|
| **Real** | якщо + future | Якщо прийдеш, поговоримо. |
| **Unreal** | якби + past + б | Якби прийшов, поговорили б. |
| **Unless** | якщо не | Прийду, якщо не захворію. |
| **Even if** | навіть якщо/якби | Навіть якщо піде дощ... |

### Other Clause Types Review

**Concessive (although):**
- хоча + clause: Хоча йде дощ, я піду.
- незважаючи на + noun: Незважаючи на дощ...

**Causal (because):**
- тому що: ...тому що захворів.
- через + acc (negative): ...через хворобу.
- завдяки + dat (positive): ...завдяки допомозі.

**Temporal (when):**
- коли: Коли він прийшов...
- після того як: Після того як закінчив...
- перед тим як: Перед тим як піти...
- поки не (until): Поки він не прийде...

## practice | Практика

### Combine Using Correct Conjunction

**Choose:** який/яка, тому що, якщо, хоча, щоб, після того як

1. Це людина. Людина допомогла мені.
   > [!answer] Це людина, **яка** допомогла мені.

2. Я не прийшов. Я захворів.
   > [!answer] Я не прийшов, **тому що** захворів.

3. Він матиме час. Він зателефонує.
   > [!answer] **Якщо** він матиме час, він зателефонує.

4. Йде дощ. Я піду на прогулянку.
   > [!answer] **Хоча** йде дощ, я піду на прогулянку.

5. Я прийшов. Я хотів допомогти.
   > [!answer] Я прийшов, **щоб** допомогти.

6. Я закінчу роботу. Я відпочину.
   > [!answer] **Після того як** закінчу роботу, я відпочину.

# Summary

## B1.2 Grammar Review

### Relative Clauses
- який/яка/яке/які — agrees with noun, declines for case
- де, куди, звідки, коли — simpler alternatives

### Purpose Clauses
- Same subject: щоб + infinitive
- Different subjects: щоб + past tense

### Conditionals
- Real: якщо + future
- Unreal: якби + past + б

### Other Clauses
- Concessive: хоча, незважаючи на
- Causal: тому що, через, завдяки
- Temporal: коли, після того як, перед тим як, поки (не)

## B1.2 Vocabulary Review

- Media vocabulary for discussing news and journalism
- Political vocabulary for civic discussions
- Environmental vocabulary for global issues
