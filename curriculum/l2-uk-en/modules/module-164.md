# Зміст уроку

## Пасивний стан: чотири форми

### Огляд форм

| Форма | Структура | Приклад | Стиль |
|-------|-----------|---------|-------|
| Дієприкметник | підмет + дієприкм. + (діяч в Ор.) | Книга написана. | Нейтральний |
| -но/-то | додаток + -но/-то | Книгу написано. | Офіційний |
| -ся | підмет + дієсл.-ся | Книга пишеться. | Науковий |
| 3-я ос. мн. | додаток + 3 ос. мн. | Книгу написали. | Розмовний |

### Вибір форми за контекстом

| Контекст | Краща форма |
|----------|-------------|
| Офіційний документ | -но/-то |
| Новини | -но/-то / дієприкметник |
| Наукова стаття | -ся |
| Розмова | 3-я ос. мн. |
| Опис стану | дієприкметник |

### Практика: виберіть форму

1. Протокол засідання: «Рішення ___»
   → **прийнято** (-но/-то, офіційний)

2. Розмова з другом: «Мені ___ про це.»
   → **сказали** (3-я ос. мн., розмовний)

3. Наукова стаття: «Проблема ___ у дослідженні.»
   → **розглядається** (-ся, науковий)

4. Опис кімнати: «Вікно ___»
   → **відкрите** (дієприкметник, стан)

## Регістри: п'ять стилів

### Огляд стилів

| Стиль | Сфера | Маркери |
|-------|-------|---------|
| Розмовний | Щоденне спілкування | частки, неповні речення |
| Художній | Література | метафори, образність |
| Публіцистичний | ЗМІ | оцінність, риторика |
| Науковий | Наука | терміни, безособовість |
| Офіційно-діловий | Документи | кліше, шаблони |

### Ти vs Ви

| Ти | Ви |
|----|-----|
| Друзі, родина | Незнайомці, старші |
| Неформальні ситуації | Офіційні ситуації |
| 2 ос. однини | 2 ос. множини |

### Практика: визначте стиль

1. «Ну, слухай, я думаю, що...»
   → **Розмовний** (частки)

2. «Відповідно до законодавства України...»
   → **Офіційно-діловий** (кліше)

3. «Чи готова Україна до нових викликів?»
   → **Публіцистичний** (риторичне питання)

4. «У роботі аналізуються дані...»
   → **Науковий** (-ся, терміни)

## Дієприкметники та дієприслівники

### Пасивні дієприкметники (-ний/-тий)

| Утворення | Приклад |
|-----------|---------|
| -ати → -аний | написати → написаний |
| -ити → -ений | зробити → зроблений |
| -ути → -утий | забути → забутий |
| -ти → -тий | відкрити → відкритий |

### Активні дієприкметники

| Тип | Статус | Приклад |
|-----|--------|---------|
| -учий/-ючий | Обмежено | працюючий (книжно) |
| -лий | Дуже обмежено | зів'ялий |
| *-вший/-ший | РУСИЗМ! | *прочитавший ❌ (*having read) |

**Альтернатива:** підрядне речення
- ❌ студент, прочитавший книгу
- ✓ студент, який прочитав книгу

### Дієприслівники

| Тип | Утворення | Приклад |
|-----|-----------|---------|
| Теперішній | -учи/-ючи | читаючи |
| Минулий | -вши/-ши | прочитавши |

**Правило:** підмет дієприслівника = підмет речення!

## Комплексна практика

### Трансформуйте текст

**Завдання:** Перепишіть розмовний текст офіційним стилем.

**Розмовний:**
«Ну, вчора мені сказали, що компанія закривається. Кажуть, гроші закінчилися. Я думаю, що це через погане управління.»

**Офіційний:**
«Повідомляємо, що вчора було оголошено про припинення діяльності компанії. Згідно з наявною інформацією, причиною є вичерпання фінансових ресурсів. На нашу думку, це є наслідком неефективного управління.»

### Виправте помилки

1. ❌ «Студент, прочитавший книгу...»
   ✓ **«Студент, який прочитав книгу...»**

2. ❌ «Читаючи книгу, мені зателефонували.»
   ✓ **«Коли я читав книгу, мені зателефонували.»**

3. ❌ «Привіт! Відповідно до наших домовленостей...»
   ✓ **«Добрий день! Відповідно до наших домовленостей...»** (стиль)
