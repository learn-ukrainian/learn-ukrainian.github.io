# Вступ

Сьома ранку. Метро «Лук'янівська». **Софія**, 28 років, дизайнерка з Києва, вставляє навушники. «Ukrainian Lessons, епізод 247», — читає вона на екрані. Голос в навушниках починає: «Привіт! Сьогодні ми поговоримо про українські традиції...»

За 40 хвилин дороги на роботу Софія вивчила 15 нових слів. Без підручника. Без класу. Просто слухаючи.

> «Подкасти змінили все», — каже вона. — «Раніше я вивчала українську три роки і боялася говорити. Після року подкастів — я розумію 80% новин і нарешті почала думати українською».

Подкасти — це ваш кишеньковий вчитель. У метро, на прогулянці, під час готування. Жива українська — просто у ваших навушниках.

## Частина 1: Історії слухачів

### Шлях Марка

**Марк**, 35 років, американець, Чикаго:

> «Мої бабуся і дідусь з України. Вони говорили українською, але я не розумів. Після їхньої смерті я вирішив вивчити мову. Знайшов подкаст Ukrainian Lessons — і слухав щодня по дорозі на роботу. Через два роки я приїхав до Львова і замовив каву українською. Бабуся б пишалася».

### Історія Анни

**Анна**, 42 роки, британка, Лондон:

> «Я волонтерка. Допомагаю українським біженцям у Британії. Щоб краще спілкуватися, почала вчити українську. Подкасти — мій головний інструмент. Слухаю під час пробіжок. Спочатку розуміла тільки 'привіт' і 'дякую'. Через рік — можу вести базову розмову».

### Відгук Томаша

**Томаш**, 29 років, поляк, Варшава:

> «Польська і українська схожі, але не однакові. Подкасти допомогли мені почути різницю. Особливо рекомендую 'Простими словами' — там пояснюють новини простою мовою. Ідеально для мого рівня».

## Частина 3: Як слухати ефективно

### Історія успіху

**Мартін**, 31 рік, німець:

> «Спочатку я слухав подкасти і нічого не розумів. Це було frustrierend — дуже розчаровувало. Потім знайшов 'техніку сендвіча'. За три місяці мій рівень розуміння виріс з 20% до 60%».

### Техніка «сендвіч»

| Крок | Дія | Мета |
|------|-----|------|
| 1 | Слухаєте БЕЗ тексту | Загальне розуміння |
| 2 | Читаєте транскрипт | Вивчаєте нові слова |
| 3 | Слухаєте З текстом | Бачите зв'язок |
| 4 | Слухаєте БЕЗ тексту | Перевірка розуміння |

### Рівні слухання

| Рівень | Мета | Приклад |
|--------|------|---------|
| Загальне | Про що? | «Подкаст про подорожі» |
| Детальне | Факти | «Вони говорили про Львів» |
| Інтенсивне | Мова | «Він сказав 'мальовничий' — нове слово!» |

### Поради від вчителів

**Оксана**, 45 років, викладачка української у Торонто:

> «Моя порада студентам: слухайте один епізод п'ять разів. Не десять епізодів по одному разу. Повторення — ключ до успіху. Перший раз — хаос. П'ятий — усе зрозуміло».

## Частина 4: Практика — слухаємо разом

### Фрагмент подкасту (Ukrainian Lessons)

**Транскрипт епізоду:**

> Ведуча: «Сьогодні ми поговоримо про українську кухню. Що ви знаєте про борщ?»
>
> Гість: «О, борщ — це класика! Моя бабуся готувала найкращий борщ у світі».
>
> Ведуча: «Цікаво! А який рецепт?»
>
> Гість: «Ну, перш за все — буряк. Без буряка немає борщу. Потім капуста, картопля, морква...»

### Аналіз фрагменту

| Фраза | Переклад | Примітка |
|-------|----------|----------|
| Сьогодні ми поговоримо про... | Today we'll talk about... | типовий вступ |
| Що ви знаєте про...? | What do you know about...? | питання до гостя |
| О, ... — це класика! | Oh, ... is a classic! | емоційна реакція |
| Цікаво! | Interesting! | відповідь ведучого |
| Ну, перш за все... | Well, first of all... | початок пояснення |

### Дискусійні фрази

| Фраза | Використання |
|-------|--------------|
| «Наш гість сьогодні...» | представлення |
| «Як ви вважаєте?» | питання до гостя |
| «Розкажіть детальніше» | прохання пояснити |
| «Підсумовуючи...» | завершення |
| «Дякуємо за увагу!» | прощання |

## Частина 5: Ваш план на місяць

### Тиждень 1-2: Старт

**Завдання:**
- Виберіть ОДИН подкаст вашого рівня
- Слухайте 15-20 хвилин щодня
- Не хвилюйтеся, якщо розумієте мало

**Рекомендація для початківців:**
| Рівень | Подкаст |
|--------|---------|
| A1-A2 | Ukrainian Lessons (повільні епізоди) |
| A2-B1 | «Простими словами» |
| B1+ | «Книга-Гак», «Toloking» |

### Тиждень 3-4: Розвиток

**Завдання:**
- Використовуйте техніку «сендвіч»
- Записуйте 5 нових слів щодня
- Спробуйте повторювати фрази вголос

### Тиждень 5+: Прогрес

**Завдання:**
- Додайте складніший подкаст
- Слухайте без субтитрів
- Обговорюйте почуте (з учителем, партнером, або самі з собою!)

# Вправи

## quiz: Перевірте себе

> Що ви запам'ятали про подкасти?

1. Що таке техніка «сендвіч»?
   - [x] слухати → читати → слухати → слухати
   - [ ] слухати три рази поспіль
   - [ ] тільки читати транскрипт
   > Техніка «сендвіч» чергує слухання без тексту, читання транскрипту, і знову слухання.

2. Хто такий «ведучий»?
   - [ ] guest
   - [x] host
   - [ ] listener
   > Ведучий = host (той, хто веде подкаст).

3. Скільки хвилин рекомендовано слухати щодня?
   - [ ] 5-10
   - [x] 15-20
   - [ ] 60+
   > 15-20 хвилин — оптимальний час для початку.

4. Який подкаст найкращий для початківців?
   - [x] Ukrainian Lessons
   - [ ] «Наш формат»
   - [ ] «Toloking»
   > Ukrainian Lessons спеціально створений для тих, хто вчить українську.

5. Що краще: слухати багато епізодів чи повторювати один?
   - [ ] багато епізодів
   - [x] повторювати один
   - [ ] однаково
   > Повторення одного епізоду ефективніше, ніж багато епізодів по одному разу.

## true-false: Правда чи неправда

> Визначте, чи правильні твердження.

1. Подкасти можна слухати тільки вдома.
   - [ ] правда
   - [x] брехня
   > Подкасти можна слухати будь-де: у метро, на прогулянці, під час готування.

2. Техніка «сендвіч» включає читання транскрипту.
   - [x] правда
   - [ ] брехня
   > Так, другий крок техніки — читання транскрипту.

3. Краще слухати один епізод п'ять разів, ніж п'ять епізодів по одному.
   - [x] правда
   - [ ] брехня

# Підсумок

## Що ви вивчили

У цьому уроці ви дізналися про:

**Українські подкасти:**
- Ukrainian Lessons — найкращий для початківців
- «Простими словами», «Книга-Гак» — для середнього рівня

**Стратегії слухання:**
- Техніка «сендвіч» (слухати-читати-слухати-слухати)
- Три рівні: загальне → детальне → інтенсивне
- Повторення ефективніше за кількість

**Історії успіху:**
- Люди з усього світу вчать українську через подкасти
- 15-20 хвилин щодня = великий прогрес

# Виробництво

## speak: Обговоріть

1. Чи слухаєте ви подкасти? Якими мовами?
2. Яка техніка вивчення мов вам підходить найкраще?
3. Де і коли ви могли б слухати українські подкасти?

## write: Напишіть

**Завдання:** Напишіть короткий відгук (5-7 речень) про подкаст, який ви послухали. Використовуйте фрази:
- «Подкаст називається...»
- «Ведучий говорить про...»
- «Мені сподобалося / не сподобалося...»
- «Я дізнався/дізналася, що...»

> **Домашнє завдання:** Виберіть один подкаст і слухайте щодня цього тижня!

> **Наступний урок:** Короткі оповідання I — читаємо українську літературу.
