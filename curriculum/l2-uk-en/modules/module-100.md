## Presentation

### Key reminders

**Aspect:**
- Perfective = single, completed, result
- Imperfective = duration, repetition, process
- Negation usually uses imperfective
- "Still not yet" (—â–µ –Ω–µ) uses perfective

**Motion verbs:**
- Unidirectional = one direction, now
- Multidirectional = round trip, habit
- Prefixes add direction + make perfective

**Communication:**
- Soften opinions: –ú–µ–Ω—ñ –∑–¥–∞—î—Ç—å—Å—è, –ú–æ–∂–ª–∏–≤–æ
- Disagree politely: –Ø –Ω–µ –∑–æ–≤—Å—ñ–º –∑–≥–æ–¥–µ–Ω
- Give reasons: —Ç–æ–º—É —â–æ, –æ—Å–∫—ñ–ª—å–∫–∏
