# Зміст уроку

## Вступ

«Український стартап розробив застосунок на основі штучного інтелекту для аналізу медичних зображень. Алгоритм машинного навчання показує точність 95%.» — такі заголовки ви зустрінете в українських технологічних новинах. І це не просто слова: Україна — справжній IT-гігант!

Подивіться на цифри: **понад 300 000** IT-спеціалістів, **$7+ мільярдів** експорту IT-послуг щороку, сотні продуктових компаній зі світовим ім'ям. **Grammarly** — той самий сервіс перевірки англійської граматики, яким користуються 30 мільйонів людей — створили українці з Києва! **MacPaw** робить програми для Mac, які люблять у всьому світі. **Readdle** — одні з найкращих застосунків для продуктивності на iOS. **Ajax Systems** — провідна система безпеки «розумного дому». І це далеко не весь список!

Чому саме Україна? По-перше, потужна математична та інженерна освіта — спадок радянської технічної школи, але без ідеологічного баласту. По-друге, конкурентні ціни та європейська робоча етика. По-третє, географічне розташування: часові пояси зручні і для Європи, і для США.

А тепер — мовна загадка. Як правильно: **застосунок** чи **додаток**? Обидва слова — переклади англійського "application", але з різними відтінками. «Застосунок» — це програма, яку ви *застосовуєте* для певної мети. «Додаток» — щось, що *додається* до чогось іншого. Офіційно рекомендують «застосунок», але в розмовній мові часто кажуть просто «апка» (від англ. app)!

> **Чи знали ви?**
>
> Українці — піонери кібербезпеки! Під час повномасштабного вторгнення 2022 року Україна відбила сотні потужних кібератак на критичну інфраструктуру. Міністерство цифрової трансформації створило **«IT-армію України»** — унікальний феномен, коли тисячі волонтерів-хакерів з усього світу об'єдналися для кіберзахисту країни.
>
> А ще Україна — одна з перших країн, яка впровадила **Дію** — мобільний застосунок, де всі документи (паспорт, права, свідоцтва) доступні в цифровому форматі. У 2022 році через Дію вперше в історії провели цифрові вибори!

---

> **Міф vs Факт**
>
> ❌ **Міф:** «Хакер» — це завжди злочинець.
>
> ✅ **Факт:** В українській (як і в англійській) слово «хакер» має нейтральне значення — це просто людина, яка глибоко розуміє комп'ютерні системи. «Білі хакери» (white hat hackers) тестують безпеку систем легально. «Чорні хакери» (black hat) — зловмисники. Україна має потужну спільноту етичних хакерів, які захищають критичну інфраструктуру.

---

Цей урок дасть вам словник для розуміння технологічних новин, спілкування з IT-спеціалістами та участі в цифровому суспільстві. Навіть якщо ви не програміст — ці терміни стали частиною повсякденної мови!

## Презентація

### Програмне забезпечення

#### Базові терміни

| Термін | English | Приклад |
|--------|---------|---------|
| **програмне забезпечення (ПЗ)** | software | Оновіть ПЗ. |
| **застосунок** | application, app | Мобільний застосунок. |
| **додаток** | app (informal) | Завантажив новий додаток. |
| **апка** | app (slang) | Класна апка! |
| **операційна система (ОС)** | operating system | Windows, macOS, Linux. |
| **браузер** | browser | Chrome, Firefox, Safari. |
| **база даних** | database | Зберегти в базі даних. |
| **код** | code | Писати код. |
| **програмування** | programming | Вивчати програмування. |
| **розробка** | development | Веб-розробка. |

**Стиль:**
- Офіційно: «застосунок», «програмне забезпечення»
- Розмовно: «апка», «софт»
- IT-сленг: «прога», «апа»

### Штучний інтелект

#### Основні поняття

| Термін | English | Пояснення |
|--------|---------|-----------|
| **штучний інтелект (ШІ)** | AI | Загальний термін. |
| **машинне навчання (МН)** | machine learning | Тип ШІ. |
| **нейронна мережа** | neural network | Імітує мозок. |
| **глибинне навчання** | deep learning | Складні нейромережі. |
| **алгоритм** | algorithm | Набір інструкцій. |
| **модель** | model | Навчена система. |
| **великі мовні моделі (ВММ)** | LLM | ChatGPT, Claude. |
| **генеративний ШІ** | generative AI | Створює контент. |

**Етимологія:** «Штучний» = artificial (створений людиною, не природний). «Інтелект» — від латинського *intelligere* (розуміти). Цікаво, що українське «розум» — від «роз-уміти», теж «розуміння»!

#### Застосування ШІ

| Зображення | **розпізнавання образів** | image recognition |
| Голос | **розпізнавання мовлення** | speech recognition |
| Текст | **обробка природної мови** | NLP |
| Спілкування | **чат-бот** | chatbot |
| Рекомендації | **рекомендаційна система** | recommendation system |
| Процеси | **автоматизація** | automation |

|--------|---------|---------|
| **дані** | data | Збирати дані. |
| **великі дані** | big data | Аналіз великих даних. |
| **аналітика** | analytics | Бізнес-аналітика. |

| Термін | English | Приклад |

| **кібербезпека** | cybersecurity | Фахівець з кібербезпеки. |
