# Вступ

Документальні фільми — чудовий спосіб пізнавати Україну. Проєкт Ukrainer створив сотні відео про людей, регіони та традиції. У цьому уроці ви навчитеся ефективно дивитися документальні фільми українською.

Documentary films are a wonderful way to discover Ukraine. The Ukrainer project has created hundreds of videos about people, regions, and traditions. In this lesson, you'll learn to effectively watch documentaries in Ukrainian.

> [!note] Ukrainer.net — безкоштовно, з субтитрами українською та англійською!

# Зміст уроку | Lesson Content

## Частина 1: Ukrainer

### Що таке Ukrainer?

**Ukrainer** — це медіапроєкт, який документує Україну:

| Характеристика | Опис |
|----------------|------|
| Засновано | 2016 |
| Формат | короткі документальні відео |
| Теми | люди, регіони, традиції |
| Мова | українська + субтитри |
| Доступ | безкоштовно на YouTube |

### Типи відео

| Серія | Про що | Тривалість |
|-------|--------|------------|
| Люди | історії звичайних українців | 5-15 хв |
| Регіони | подорожі Україною | 10-20 хв |
| Ремесла | традиційні професії | 5-10 хв |
| Війна | життя під час війни | різна |

> [!tip] **Порада:** Почніть з коротких відео (5-7 хвилин) і поступово переходьте до довших.

## Частина 2: Як дивитися

### Стратегія перегляду

**Перший перегляд (без субтитрів):**
1. Дивіться відео повністю
2. Намагайтеся зрозуміти загально
3. Не хвилюйтеся через незнайомі слова

**Другий перегляд (з українськими субтитрами):**
1. Увімкніть субтитри
2. Зупиняйте на незрозумілих місцях
3. Виписуйте нові слова

**Третій перегляд (без субтитрів):**
1. Перевірте, скільки тепер розумієте
2. Повторіть цікаві фрази

### Що виписувати

| Категорія | Приклади |
|-----------|----------|
| Нові слова | професії, місця |
| Фрази | «у нас кажуть...», «це означає...» |
| Граматика | цікаві конструкції |
| Культура | традиції, звичаї |

## Частина 3: Рекомендовані відео

### Для початку (прості)

| «Ткаля» | ткацтво | прості речення |

| Назва | Тема | Регіон |
|-------|------|--------|

| «Полісся» | болота, традиції | Північ |

### Складніші

| Назва | Тема | Виклик |
|-------|------|--------|

| Діалекти | регіональна мова | незвична лексика |

## Частина 4: Словник документалістики

### Технічні терміни

| Українська | English |
|------------|---------|
| документальний фільм | documentary |
| режисер | director |
| оператор | cameraman |
| монтаж | editing |
| субтитри | subtitles |
| закадровий голос | voiceover |
| інтерв'ю | interview |
| кадр | shot, frame |

### Описові фрази

| Фраза | Використання |
|-------|--------------|
| «Мене звати...» | представлення героя |
| «Я народився/народилася в...» | біографія |
| «Я займаюся...» | професія |
| «Для мене це означає...» | значення |
| «Найважливіше — це...» | цінності |
