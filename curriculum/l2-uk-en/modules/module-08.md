# Lesson Content

## warm-up

### The Second Pattern

In Module 06, you learned Class I verbs (-–∞—Ç–∏/-—è—Ç–∏) with their -—î- pattern:
—á–∏—Ç–∞—é, —á–∏—Ç–∞—î—à, —á–∏—Ç–∞—î, —á–∏—Ç–∞—î–º–æ, —á–∏—Ç–∞—î—Ç–µ, —á–∏—Ç–∞—é—Ç—å

Now meet Class II! These verbs end in **-–∏—Ç–∏/-—ñ—Ç–∏** and have a completely different pattern with **-–∏-**:
–≥–æ–≤–æ—Ä—é, –≥–æ–≤–æ—Ä–∏—à, –≥–æ–≤–æ—Ä–∏—Ç—å, –≥–æ–≤–æ—Ä–∏–º–æ, –≥–æ–≤–æ—Ä–∏—Ç–µ, –≥–æ–≤–æ—Ä—è—Ç—å

> üîç **Myth Buster: "Too Many Patterns!"**
>
> Good news: these are the ONLY two main patterns! Once you know Class I and Class II, you can conjugate about 97% of all Ukrainian verbs. The remaining 3% are irregulars that you'll memorize individually.

## presentation

### Class II Conjugation: -–∏—Ç–∏/-—ñ—Ç–∏ Verbs

**Pattern: –≥–æ–≤–æ—Ä–∏—Ç–∏ (to speak)**

| Person | Pronoun | Ending | Full Form | English |
|--------|---------|--------|-----------|---------|
| 1st sing. | —è | -—é/-–ª—é | –≥–æ–≤–æ—Ä**—é** | I speak |
| 2nd sing. | —Ç–∏ | -–∏—à | –≥–æ–≤–æ—Ä**–∏—à** | you speak |
| 3rd sing. | –≤—ñ–Ω/–≤–æ–Ω–∞ | -–∏—Ç—å | –≥–æ–≤–æ—Ä**–∏—Ç—å** | he/she speaks |
| 1st pl. | –º–∏ | -–∏–º–æ | –≥–æ–≤–æ—Ä**–∏–º–æ** | we speak |
| 2nd pl. | –≤–∏ | -–∏—Ç–µ | –≥–æ–≤–æ—Ä**–∏—Ç–µ** | you speak |
| 3rd pl. | –≤–æ–Ω–∏ | -—è—Ç—å/-–ª—è—Ç—å | –≥–æ–≤–æ—Ä**—è—Ç—å** | they speak |

**Key endings to memorize: -—é, -–∏—à, -–∏—Ç—å, -–∏–º–æ, -–∏—Ç–µ, -—è—Ç—å**

> üí° **Did You Know?**
>
> The difference between Class I and Class II comes from ancient Slavic verb classes. Class II verbs originally had a thematic vowel **-i-**, which is why you see -–∏—à, -–∏—Ç—å, -–∏–º–æ, -–∏—Ç–µ. Class I had **-e-**, giving us -—î—à, -—î, -—î–º–æ, -—î—Ç–µ. These patterns are over 1,000 years old!

> üîó **Language Link: Class I vs Class II at a Glance**
>
> | | Class I (-–∞—Ç–∏) | Class II (-–∏—Ç–∏) |
> |---|----------------|-----------------|
> | —Ç–∏ | —á–∏—Ç–∞**—î—à** | –≥–æ–≤–æ—Ä**–∏—à** |
> | –≤—ñ–Ω | —á–∏—Ç–∞**—î** | –≥–æ–≤–æ—Ä**–∏—Ç—å** |
> | –º–∏ | —á–∏—Ç–∞**—î–º–æ** | –≥–æ–≤–æ—Ä**–∏–º–æ** |
> | –≤–∏ | —á–∏—Ç–∞**—î—Ç–µ** | –≥–æ–≤–æ—Ä**–∏—Ç–µ** |
> | –≤–æ–Ω–∏ | —á–∏—Ç–∞**—é—Ç—å** | –≥–æ–≤–æ—Ä**—è—Ç—å** |
>
> Quick check: **-—î-** = Class I, **-–∏-** = Class II

### Consonant Changes in Class II

Many Class II verbs have consonant changes in **—è** and **–≤–æ–Ω–∏** forms:

| Infinitive | Change | —è form | –≤–æ–Ω–∏ form |
|------------|--------|--------|-----------|
| —Ä–æ–±–∏—Ç–∏ | –± ‚Üí –±–ª | —Ä–æ–±**–ª—é** | —Ä–æ–±**–ª—è—Ç—å** |
| –ª—é–±–∏—Ç–∏ | –± ‚Üí –±–ª | –ª—é–±**–ª—é** | –ª—é–±**–ª—è—Ç—å** |
| —Ö–æ–¥–∏—Ç–∏ | –¥ ‚Üí –¥–∂ | —Ö–æ**–¥–∂—É** | —Ö–æ**–¥—è—Ç—å** |
| –±–∞—á–∏—Ç–∏ | —á ‚Üí —á | –±–∞**—á—É** | –±–∞**—á–∞—Ç—å** |
| –ø—Ä–æ—Å–∏—Ç–∏ | —Å ‚Üí —à | –ø—Ä–æ**—à—É** | –ø—Ä–æ**—Å—è—Ç—å** |

> ‚ö° **Pro Tip: Predict the Change**
>
> Common consonant mutations:
> - **–± ‚Üí –±–ª**: —Ä–æ–±–∏—Ç–∏ ‚Üí —Ä–æ–±–ª—é, –ª—é–±–∏—Ç–∏ ‚Üí –ª—é–±–ª—é
> - **–≤ ‚Üí –≤–ª**: –ª–æ–≤–∏—Ç–∏ ‚Üí –ª–æ–≤–ª—é
> - **–ø ‚Üí –ø–ª**: –∫—É–ø–∏—Ç–∏ ‚Üí –∫—É–ø–ª—é
> - **–º ‚Üí –º–ª**: –ª–∞–º–∞—Ç–∏ ‚Üí –ª–∞–º–ª—é
> - **–¥ ‚Üí –¥–∂**: —Ö–æ–¥–∏—Ç–∏ ‚Üí —Ö–æ–¥–∂—É, —Å–∏–¥—ñ—Ç–∏ ‚Üí —Å–∏–¥–∂—É
> - **—Å ‚Üí —à**: –ø—Ä–æ—Å–∏—Ç–∏ ‚Üí –ø—Ä–æ—à—É, –Ω–æ—Å–∏—Ç–∏ ‚Üí –Ω–æ—à—É
> - **–∑ ‚Üí –∂**: –≤–æ–∑–∏—Ç–∏ ‚Üí –≤–æ–∂—É
>
> These changes only happen in —è and –≤–æ–Ω–∏ forms!

> üéØ **Fun Fact: Why Consonants Change**
>
> These consonant changes happened centuries ago when certain sounds couldn't appear before specific vowels. The sound [j] (like "y" in yes) caused consonants to "soften" and change. Even though the [j] sound disappeared, the changed consonants remained. You're speaking with sounds shaped by medieval Slavic phonology!

### Common Irregular Verbs

Some very common verbs don't follow either pattern perfectly:

**—Ö–æ—Ç—ñ—Ç–∏ (to want)** - mixed pattern

| | Form |
|-|------|
| —è | —Ö–æ—á—É |
| —Ç–∏ | —Ö–æ—á–µ—à |
| –≤—ñ–Ω/–≤–æ–Ω–∞ | —Ö–æ—á–µ |
| –º–∏ | —Ö–æ—á–µ–º–æ |
| –≤–∏ | —Ö–æ—á–µ—Ç–µ |
| –≤–æ–Ω–∏ | —Ö–æ—á—É—Ç—å |

**—ó—Å—Ç–∏ (to eat)** - irregular

| | Form |
|-|------|
| —è | —ó–º |
| —Ç–∏ | —ó—Å–∏ |
| –≤—ñ–Ω/–≤–æ–Ω–∞ | —ó—Å—Ç—å |
| –º–∏ | —ó–º–æ |
| –≤–∏ | —ó—Å—Ç–µ |
| –≤–æ–Ω–∏ | —ó–¥—è—Ç—å |

**–ø–∏—Ç–∏ (to drink)** - irregular

| | Form |
|-|------|
| —è | –ø'—é |
| —Ç–∏ | –ø'—î—à |
| –≤—ñ–Ω/–≤–æ–Ω–∞ | –ø'—î |
| –º–∏ | –ø'—î–º–æ |
| –≤–∏ | –ø'—î—Ç–µ |
| –≤–æ–Ω–∏ | –ø'—é—Ç—å |

> ‚ö° **Pro Tip: The -—É–≤–∞—Ç–∏ Trick (Class I)**
>
> Verbs ending in **-—É–≤–∞—Ç–∏** (like –∫—É–ø—É–≤–∞—Ç–∏, –¥—è–∫—É–≤–∞—Ç–∏) are actually **Class I**, but they have a trick!
> The **-–≤–∞-** drops out and becomes **-—É-**:
> - **–∫—É–ø—É–≤–∞—Ç–∏** (to buy) ‚Üí —è –∫—É–ø**—É—é**, —Ç–∏ –∫—É–ø**—É—î—à**, –≤–æ–Ω–∏ –∫—É–ø**—É—é—Ç—å**
> - **–¥—è–∫—É–≤–∞—Ç–∏** (to thank) ‚Üí —è –¥—è–∫**—É—é**, —Ç–∏ –¥—è–∫**—É—î—à**, –≤–æ–Ω–∏ –¥—è–∫**—É—é—Ç—å**
>
> If you see -—É–≤–∞—Ç–∏, conjugate it like -—É—é/-—É—î—à!

> üìú **History Bite: Ancient Irregulars**
>
> The most common verbs in any language tend to be irregular because they're used so often they resist regularization. **–á—Å—Ç–∏** (eat) and **–ø–∏—Ç–∏** (drink) have been irregular since Proto-Slavic times - humans have been conjugating these verbs "wrong" for over 2,000 years!
