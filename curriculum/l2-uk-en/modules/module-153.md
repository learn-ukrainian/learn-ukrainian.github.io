# Вступ

Ви читаєте науковий текст і бачите: «Працюючі в даній галузі спеціалісти...» Ви читаєте художній роман і бачите: «Сплячий під деревом чоловік...» Потім ви намагаєтесь використати ці конструкції в розмові — і ваш українець-співрозмовник дивиться на вас трохи дивно.

Що сталось? Ви щойно зіткнулись із **граматичним привидом** — формою, яка існує в українській мові, але майже не вживається в живому мовленні. Активні дієприкметники теперішнього часу («читаючий», «працюючий», «сплячий») — це **книжна мова**, що живе в наукових статтях та офіційних документах, але рідко з'являється в повсякденній розмові.

Для вас на рівні B2 це означає: **розпізнавати — так, активно вживати — обережно**. Ці форми потрібні для розуміння складних текстів, але для власного мовлення є кращі альтернативи.

> **Чи знали ви?**
>
> Українські активні дієприкметники на **-учий/-ючий** — давній спадок **церковнослов'янської мови**. Саме тому вони звучать книжно й піднесено. У XVII-XVIII століттях українські письменники навмисно використовували їх для надання текстам вагомості. Сьогодні ця «вагомість» сприймається як **надмірна офіційність** у розмовному контексті.

# Пасивний vs Активний

Пригадаємо різницю:

| Тип | Значення | Приклад | Хто діє? |
|-----|----------|---------|----------|
| **Пасивний** | хтось зробив над об'єктом | написан**ий** (був написаний кимось) | Хтось інший |
| **Активний** | сам виконує дію | пишуч**ий** (той, хто пише) | Сам об'єкт |

Пасивні дієприкметники ви вже знаєте (модулі 142-146). Тепер — активні.

# Утворення

## Формула

**Від недоконаних дієслів теперішнього часу:**

| Дієвідміна | Формула |
|------------|---------|
| I дієвідміна | Основа 3 ос. мн. + **-учий/-ючий** |
| II дієвідміна | Основа 3 ос. мн. + **-ачий/-ячий** |

## Приклади

| Інфінітив | 3 ос. мн. | Дієприкметник |
|-----------|-----------|---------------|
| читати | чита**ють** | читаюч**ий** |
| писати | пиш**уть** | пишуч**ий** |
| працювати | працю**ють** | працююч**ий** |
| говорити | говор**ять** | говоряч**ий** |
| любити | любл**ять** | любляч**ий** |
| спати | спл**ять** | спляч**ий** |

> **Міф vs Факт**
>
> ❌ **Міф:** «Активні дієприкметники — це те саме, що англійські present participles (-ing).»
>
> ✅ **Факт:** Англійські -ing форми універсальні та вживаються скрізь. Українські активні дієприкметники — **книжні** та **стилістично марковані**. «The reading student» — нормально. «Студент, читаючий книгу» — надто офіційно для розмови.

# Стилістичні обмеження

## Де вживаються

| Контекст | Приклад |
|----------|---------|
| Наукові тексти | «Фактори, впливаючі на процес...» |
| Офіційні документи | «Особи, працюючі на підприємстві...» |
| Художня література | «Сплячий лицар не чув грому...» |
| Переклади | (часто калька з інших мов) |

## Де НЕ вживаються

- ❌ Розмовна мова
- ❌ Неформальне листування
- ❌ SMS, чати

> **Чи знали ви?**
>
> У **радянські часи** активні дієприкметники масово проникали в українську мову під впливом російської, де вони вживаються значно ширше. Багато форм, які тоді нав'язувались («завідуючий», «керуючий»), сьогодні визнані **русизмами**. Сучасна українська мова повернулась до більш природних підрядних речень.

---

# Краща альтернатива: підрядні речення

У сучасній українській мові **підрядні речення з «який»** — завжди кращий вибір:

| Дієприкметник (книжний) | Підрядне речення (нейтральний) |
|-------------------------|--------------------------------|
| Студент, **читаючий** книгу... | Студент, **який читає** книгу... |
| Людина, **працююча** тут... | Людина, **яка працює** тут... |
| Діти, **граючі** на подвір'ї... | Діти, **які грають** на подвір'ї... |
| Птах, **співаючий** на гілці... | Птах, **який співає** на гілці... |

# Субстантивовані дієприкметники

Деякі дієприкметники стали **іменниками**:

| Дієприкметник → Іменник | Значення |
|-------------------------|----------|
| працюючий → **працюючий** | working person |
| віруючий → **віруючий** | believer |
| завідуючий → ❌ | РУСИЗМ! |
| керуючий → ❌ | РУСИЗМ! |

## Правильні форми

| ❌ Русизм | ✅ Українська |
|----------|--------------|
| завідуючий | **завідувач** |
| керуючий | **керівник** |
| командуючий | **командувач** |
| проводжаючий | **проводжач** |

---

# Усталені вживання

Деякі дієприкметники стали усталеними виразами — їх вчіть як цілісні одиниці:

| Вираз | Значення | Контекст |
|-------|----------|----------|
| **чинне** законодавство | current legislation | юридичні тексти |
| **діючий** президент | incumbent president | політика |
| **існуюча** проблема | existing problem | наукові тексти |
| **зростаючий** попит | growing demand | економіка |
| **пануюча** думка | prevailing opinion | публіцистика |
| **домінуюча** культура | dominant culture | соціологія |

> **Чи знали ви?**
>
> Вираз **«діюча особа»** (character in a play/book) — один із небагатьох випадків, де активний дієприкметник абсолютно природний і не має альтернативи. Шекспірівські «діючі особи» драми — це Гамлет, Офелія, Клавдій. Спроба замінити на «особи, які діють» змінила б значення.

## quiz: Дієприкметник чи підрядне речення?
> Виберіть природнішу українську форму.

1. Для розмови: «студент, який читає книгу» чи «студент, читаючий книгу»?
   - [x] студент, який читає книгу
   - [ ] студент, читаючий книгу
   > У розмові → підрядне речення (природніше)

2. Як правильно: «завідуючий» чи «завідувач»?
   - [ ] завідуючий
   - [x] завідувач
   > «Завідуючий» — русизм, правильно: завідувач

3. Як правильно: «керуючий» чи «керівник»?
   - [ ] керуючий
   - [x] керівник
   > «Керуючий» — русизм, правильно: керівник

4. Усталений вираз: «чинне...» що?
   - [x] законодавство
   - [ ] право
   - [ ] кодекс
   > «Чинне законодавство» — усталений вираз

# Словник

| Слово | Переклад | Примітки |
|-------|----------|----------|
| активний дієприкметник | active participle | книжна форма |
| підрядне речення | subordinate clause | нейтральна альтернатива |
| книжний | bookish, literary | офіційний стиль |
| субстантивований | substantivized | перехід у іменник |
| усталений | established, set | фіксований вираз |
| чинний | current, valid | про закони |
| існуючий | existing | про проблеми |
| зростаючий | growing | про попит |
| пануючий | prevailing | про думку |
| завідувач | manager (correct) | не «завідуючий»! |

# Підсумок

## Активні дієприкметники теперішнього часу

| Аспект | Характеристика |
|--------|----------------|
| **Утворення** | Основа 3 ос. мн. + -учий/-ючий, -ачий/-ячий |
| **Значення** | Той, хто виконує дію |
| **Стиль** | Книжний, офіційний, науковий |
| **Альтернатива** | Підрядне речення з «який» |

## Що запам'ятати

> **Запам'ятайте:**
>
> ✓ Активні дієприкметники — **для розпізнавання**, не для активного вживання
> ✓ У розмові завжди краще: «**який читає**», не «читаючий»
> ✓ Форми «завідуючий», «керуючий» — **русизми** (правильно: завідувач, керівник)
> ✓ Усталені вирази («чинне законодавство») вчіть як **цілі одиниці**
> ✓ Це **церковнослов'янська спадщина** — тому звучить піднесено
