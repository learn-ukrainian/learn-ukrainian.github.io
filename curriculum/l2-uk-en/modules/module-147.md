# Зміст уроку

## Розминка

### Що таке регістр?

**Регістр** (стиль мовлення) — це варіант мови, який ми вибираємо залежно від ситуації.

Порівняйте:
- «Привіт! Як справи?» (неформальний)
- «Доброго дня! Як ваші справи?» (нейтральний)
- «Шановний пане! Маю честь звернутися...» (офіційний)

**Та сама ідея — різні слова, різна граматика, різний тон.**

## Презентація

### П'ять функціональних стилів

Українська мова має **п'ять основних стилів:**

| Стиль | Сфера | Приклад |
|-------|-------|---------|
| **Розмовний** | Щоденне спілкування | «Слухай, я запізнююсь!» |
| **Художній** | Література | «Тихо місяць плив над степом...» |
| **Публіцистичний** | ЗМІ, публічні виступи | «Сьогодні парламент ухвалив...» |
| **Науковий** | Наука, освіта | «Дослідження показує, що...» |
| **Офіційно-діловий** | Документи, закони | «Відповідно до статті 5...» |

### 1. Розмовний стиль

**Характеристики:**
- Неповні речення
- Скорочення
- Емоційність
- Просторіччя

| Риса | Приклад |
|------|---------|
| Скорочення | «Що?» → «Шо?» |
| Неповні речення | «Куди?» (замість «Куди ти йдеш?») |
| Частки | «ну», «от», «та» |
| Вигуки | «Ого!», «Ой!», «Ех...» |

**Приклад:**
«Ну, слухай, я от думав... Може, підемо кудись? Ну, не знаю, в кіно чи ще кудись.»

### 2. Художній стиль

**Характеристики:**
- Образність
- Метафори, порівняння
- Ритм, звукопис
- Емоційна виразність

| Риса | Приклад |
|------|---------|
| Метафора | «серце співає» |
| Порівняння | «білий як сніг» |
| Епітет | «золоте сонце» |
| Персоніфікація | «вітер шепоче» |

**Приклад:**
«Сонце, втомлене за день, повільно сідало за обрій, фарбуючи небо в кольори полум'я.»

### 3. Публіцистичний стиль

**Характеристики:**
- Актуальність
- Оцінність
- Риторичні засоби
- Суспільна значущість

| Риса | Приклад |
|------|---------|
| Суспільна лексика | «демократія», «реформа» |
| Оцінні слова | «важливий», «критичний» |
| Риторичні питання | «Чи можемо ми це ігнорувати?» |
| Заклики | «Маємо діяти зараз!» |

**Приклад:**
«Сьогодні Україна стоїть на порозі важливих змін. Чи готові ми до викликів майбутнього?»

### 4. Науковий стиль

**Характеристики:**
- Точність
- Логічність
- Термінологія
- Безособовість

| Риса | Приклад |
|------|---------|
| Терміни | «морфема», «синтаксис» |
| Безособовість | «розглядається», «аналізується» |
| Логічні зв'язки | «отже», «таким чином» |
| Складні речення | підрядні, вставні |

**Приклад:**
«У статті розглядається проблема впливу соціальних факторів на мовну поведінку. Аналізуються дані, зібрані протягом 2020-2023 років.»

### 5. Офіційно-діловий стиль

**Характеристики:**
- Стандартизованість
- Безособовість
- Точність формулювань
- Кліше та шаблони

| Риса | Приклад |
|------|---------|
| Кліше | «відповідно до», «у зв'язку з» |
| Безособові форми | «наказано», «затверджено» |
| Канцеляризми | «вищезазначений», «нижчепідписаний» |
| Стандартні формули | «Шановний пане/пані» |

**Приклад:**
«Відповідно до наказу № 123 від 15.03.2024, наказую затвердити штатний розпис на 2024 рік.»

### Порівняльна таблиця

| Аспект | Розмовний | Офіційний |
|--------|-----------|-----------|
| Звертання | Привіт, Петре! | Шановний пане Петренко! |
| Прохання | Дай, будь ласка | Прошу надати |
| Подяка | Дякую! | Висловлюю вдячність |
| Відмова | Не можу | На жаль, не маю змоги |
| Погодження | Добре | Ухвалено / Погоджено |

## Практика

### Визначте стиль

1. «Рішення про затвердження бюджету прийнято одноголосно.»
   → **Офіційно-діловий**

2. «Ну, я не знаю... Може, завтра?»
   → **Розмовний**

3. «Дослідження показує статистично значущу кореляцію.»
   → **Науковий**

4. «Срібний місяць виплив із-за хмар.»
   → **Художній**

5. «Чи готова Україна до нових викликів?»
   → **Публіцистичний**

### Трансформуйте стиль

**Розмовний → Офіційний:**

1. «Мені потрібна відпустка.»
   → «Прошу надати мені відпустку.»

2. «Я згоден.»
   → «Погоджено.»

3. «Дякую за допомогу!»
   → «Висловлюю вдячність за надану допомогу.»

## Продукція

### Напишіть повідомлення

Напишіть одне повідомлення трьома стилями:

**Тема:** Запрошення на зустріч

**Розмовний:**
«Привіт! Слухай, можеш прийти завтра о третій? Треба поговорити.»

**Нейтральний:**
«Доброго дня! Запрошую вас на зустріч завтра о 15:00.»

**Офіційний:**
«Шановний пане! Маю честь запросити Вас на зустріч, яка відбудеться завтра о 15:00.»

# Вправи

## quiz: Стилі мовлення

> Визначте стиль.

1. «Відповідно до законодавства України...»
   - [ ] розмовний
   - [ ] науковий
   - [x] офіційно-діловий
   > Кліше «відповідно до» → офіційний.

2. «Ну от, я ж казав!»
   - [x] розмовний
   - [ ] публіцистичний
   - [ ] художній
   > Частки «ну», «от» → розмовний.

3. «У роботі аналізуються фактори впливу...»
   - [ ] публіцистичний
   - [x] науковий
   - [ ] офіційно-діловий
   > Безособовість + терміни → науковий.

4. «Золоті коси сонця падали на землю.»
   - [ ] публіцистичний
   - [ ] розмовний
   - [x] художній
   > Метафора → художній.

5. «Чи зможе Україна подолати ці виклики?»
   - [x] публіцистичний
   - [ ] офіційно-діловий
   - [ ] науковий
   > Риторичне питання + суспільна тема → публіцистичний.

# Підсумок

## П'ять стилів української мови

| Стиль | Сфера | Ключові риси |
|-------|-------|--------------|
| Розмовний | Побут | Емоційність, неповнота |
| Художній | Література | Образність, виразність |
| Публіцистичний | ЗМІ | Оцінність, актуальність |
| Науковий | Наука | Точність, термінологія |
| Офіційно-діловий | Документи | Стандартність, кліше |

> **Головне**
>
> - Вибір стилю залежить від ситуації
> - Кожен стиль має свої маркери
> - Змішування стилів може бути помилкою
> - B2 = вміння розрізняти та використовувати різні стилі
