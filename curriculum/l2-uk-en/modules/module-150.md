# Зміст уроку

## Розминка

### Мова новин

Порівняйте:

| Розмовний | Публіцистичний |
|-----------|----------------|
| Учора такий мітинг був! | Учора відбулася масова акція протесту. |
| Люди хочуть змін. | Громадяни вимагають реформ. |
| Влада щось вирішила. | Парламент ухвалив історичне рішення. |

**Публіцистичний стиль** — мова ЗМІ, публічних виступів, реклами.

## Презентація

### Характеристики публіцистичного стилю

| Риса | Опис |
|------|------|
| Актуальність | Про важливе тут і зараз |
| Оцінність | Авторська позиція |
| Риторичність | Засоби переконання |
| Експресивність | Емоційний вплив |
| Доступність | Зрозуміло широкій аудиторії |

### Жанри

| Жанр | Характеристика |
|------|----------------|
| Новина | Факти без оцінки |
| Репортаж | Опис події |
| Інтерв'ю | Розмова з героєм |
| Стаття | Аналіз проблеми |
| Коментар | Думка автора |
| Колонка | Особиста позиція |

### Лексика публіцистики

**Суспільно-політична:**

| Слово | Переклад |
|-------|----------|
| демократія | democracy |
| реформа | reform |
| парламент | parliament |
| вибори | elections |
| коаліція | coalition |
| опозиція | opposition |
| санкції | sanctions |
| дипломатія | diplomacy |

**Оцінна лексика:**

| Позитивна | Негативна |
|-----------|-----------|
| прорив | криза |
| успіх | провал |
| перемога | поразка |
| розквіт | занепад |
| стабільність | хаос |

### Заголовки

**Типи заголовків:**

| Тип | Приклад |
|-----|---------|
| Інформативний | Парламент ухвалив бюджет на 2025 рік |
| Питальний | Чи готова Україна до ЄС? |
| Закличний | Час діяти! |
| Цитатний | Президент: «Це історичний день» |
| Інтригувальний | Що ховається за реформою? |

**Особливості:**
- Теперішній час замість минулого: «Парламент ухвалює» (не «ухвалив»)
- Відсутність дієслова: «Новий закон — новий етап»
- Скорочення: «ВР» (Верховна Рада), «МЗС» (Міністерство закордонних справ)

### Засоби впливу

**Риторичні запитання:**
- «Чи можемо ми це ігнорувати?»
- «Хто несе відповідальність?»
- «До чого це призведе?»

**Звертання до читача:**
- «Уявіть собі...»
- «Кожен із нас...»
- «Ми всі розуміємо...»

**Емоційні вигуки:**
- «Неприпустимо!»
- «Це вражає!»
- «Нарешті!»

**Метафори:**
- «хвиля змін»
- «стіна нерозуміння»
- «коріння проблеми»
- «світло в кінці тунелю»

### Структура новини

**Перевернута піраміда:**

1. **Лід** (головне) — хто, що, де, коли
2. **Деталі** — як, чому
3. **Фон** — контекст, історія
4. **Цитати** — коментарі експертів

**Приклад:**

**Заголовок:** Парламент ухвалив закон про освіту

**Лід:** Верховна Рада України у четвер ухвалила новий закон про освіту, який набуде чинності з вересня 2025 року.

**Деталі:** За законопроєкт проголосували 256 народних депутатів. Документ передбачає реформу середньої освіти та збільшення фінансування шкіл.

**Фон:** Це вже третя спроба провести освітню реформу за останні п'ять років.

**Цитата:** «Це історичний крок для української освіти», — заявив міністр освіти.

### Нейтральність vs Оцінність

**Нейтральна новина:**
«Учора в Києві відбувся мітинг. За даними поліції, взяли участь близько 5 тисяч осіб.»

**Оцінна стаття:**
«Учора Київ став свідком масової демонстрації народного невдоволення. Тисячі громадян вийшли на вулиці, щоб висловити протест проти скандального рішення влади.»

**Маркери оцінності:**
- прикметники: «масова», «скандальний»
- іменники: «невдоволення», «протест»
- дієслова з конотацією: «вийшли» (нейтральне) vs «повстали» (емоційне)

### Сталі вирази публіцистики

| Вираз | Контекст |
|-------|----------|
| вийти на новий рівень | досягнення |
| стати на порозі | початок змін |
| зробити крок назустріч | компроміс |
| дати зелене світло | дозволити |
| поставити крапку | завершити |
| відкрити нову сторінку | почати знову |
| бути в епіцентрі | бути в центрі подій |

## Практика

### Проаналізуйте заголовки

Визначте тип заголовка:

1. «Уряд схвалив новий бюджет»
   → **Інформативний**

2. «Чи витримає економіка новий удар?»
   → **Питальний**

3. «Настав час змін!»
   → **Закличний**

4. «Прем'єр: "Ми готові до викликів"»
   → **Цитатний**

### Знайдіть засоби впливу

**Текст:**
«Чи можемо ми й далі закривати очі на цю проблему? Кожен громадянин має зрозуміти: хвиля змін неминуча. Настав час діяти!»

**Засоби:**
- Риторичне запитання: «Чи можемо ми...?»
- Метафора: «хвиля змін»
- Звертання до читача: «Кожен громадянин»
- Заклик: «Настав час діяти!»

### Напишіть новину

**Факти:** Відкриття нового музею в Києві, 15 травня, присвячений історії України, директор — Олена Петренко.

**Новина:**
«У Києві відкрито новий музей історії України. Урочиста церемонія відбулася 15 травня за участі представників влади та культурних діячів. "Цей музей — наша пам'ять і наше майбутнє", — заявила директор музею Олена Петренко.»

# Підсумок

## Публіцистичний стиль

| Риса | Приклад |
|------|---------|
| Актуальність | Про сьогодення |
| Оцінність | «історичний», «скандальний» |
| Риторичність | «Чи можемо ми...?» |
| Метафоричність | «хвиля змін» |

## Структура новини

1. **Лід** — хто, що, де, коли
2. **Деталі** — як, чому
3. **Фон** — контекст
4. **Цитати** — коментарі

> **Головне**
>
> - Публіцистика поєднує інформацію та вплив
> - Заголовки — у теперішньому часі
> - Риторичні засоби привертають увагу
> - Оцінна лексика виражає позицію автора
