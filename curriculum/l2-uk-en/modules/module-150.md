# Вступ

Відкрийте будь-який новинний сайт. Перший заголовок: «Україна на порозі історичних змін». Другий: «Шокуюча правда про...». Третій: «Чи готові ми до майбутнього?» Помічаєте щось спільне?

Публіцистичний стиль — це мова, яка **не просто інформує, а впливає**. Вона бере звичайний факт («Парламент прийняв закон») і перетворює його на подію («Парламент ухвалив ІСТОРИЧНИЙ закон!»). Вона ставить питання, на які сама ж відповідає. Вона переконує вас, що ви маєте думати так, а не інакше.

Це не погано і не добре — це просто **природа публіцистики**. Журналіст — не машина для передачі фактів. Він обирає слова, і кожне слово несе оцінку. «Протестувальники» чи «бунтівники»? «Реформа» чи «експеримент»? «Криза» чи «тимчасові труднощі»? Різні слова — різна картина світу.

> **Чи знали ви?**
>
> Слово **«публіцистика»** походить від латинського *publicus* — «суспільний». Це мова, яка звертається до **публіки** — широкого загалу. Тому вона мусить бути доступною, емоційною та переконливою. Академічна стаття може бути нудною — її читатимуть фахівці. Газетна стаття має «чіпляти» — інакше читач перегорне сторінку.

# Характеристики публіцистичного стилю

| Риса | Що це означає | Приклад |
|------|---------------|---------|
| **Актуальність** | Про важливе тут і зараз | «Сьогодні», «учора», «щойно» |
| **Оцінність** | Авторська позиція | «історичний», «скандальний» |
| **Риторичність** | Засоби переконання | «Чи можемо ми...?» |
| **Експресивність** | Емоційний вплив | «Неприпустимо!», «Нарешті!» |
| **Доступність** | Зрозуміло всім | Прості речення, пояснення |

# Жанри публіцистики

| Жанр | Характеристика | Оцінність |
|------|----------------|-----------|
| **Новина** | Факти без оцінки | Мінімальна |
| **Репортаж** | Опис події «зсередини» | Помірна |
| **Інтерв'ю** | Розмова з героєм | Залежить |
| **Стаття** | Аналіз проблеми | Висока |
| **Коментар** | Думка автора | Максимальна |
| **Колонка** | Особиста позиція | Максимальна |

> **Міф vs Факт**
>
> ❌ **Міф:** «Новини — це об'єктивна правда.»
>
> ✅ **Факт:** Новини — це **відібрані факти**. Журналіст вирішує, що показати, а що ні. Вибір фото, порядок інформації, підбір слів — усе це формує враження. Повної об'єктивності не існує — є різні ступені суб'єктивності.

# Лексика публіцистики

## Суспільно-політична лексика

| Українська | English | Контекст |
|------------|---------|----------|
| демократія | democracy | політика |
| реформа | reform | зміни |
| парламент | parliament | законодавство |
| вибори | elections | політика |
| коаліція | coalition | уряд |
| опозиція | opposition | політика |
| санкції | sanctions | міжнародні відносини |
| дипломатія | diplomacy | міжнародні відносини |
| суверенітет | sovereignty | державність |

## Оцінна лексика

| Позитивна | Негативна |
|-----------|-----------|
| прорив | криза |
| успіх | провал |
| перемога | поразка |
| розквіт | занепад |
| стабільність | хаос |
| досягнення | загроза |
| надія | небезпека |

> **Чи знали ви?**
>
> Слова **«криза»** і **«можливість»** китайською нібито пишуться одним ієрогліфом — це популярний міф. Але в журналістиці ця ідея працює реально: одну й ту саму ситуацію можна назвати «кризою» (негативно) або «викликом» (нейтрально), або «можливістю для змін» (позитивно). **Вибір слова = вибір оцінки**.

# Заголовки: мистецтво привернення уваги

## Типи заголовків

| Тип | Приклад | Ефект |
|-----|---------|-------|
| **Інформативний** | Парламент ухвалив бюджет | Факт |
| **Питальний** | Чи готова Україна до ЄС? | Інтрига |
| **Закличний** | Час діяти! | Емоція |
| **Цитатний** | Президент: «Це історичний день» | Авторитет |
| **Інтригувальний** | Що ховається за реформою? | Цікавість |

## Граматичні особливості заголовків

- **Теперішній час замість минулого:** «Парламент ухвалює» (не «ухвалив») — створює ефект «тут і зараз»
- **Відсутність дієслова:** «Новий закон — новий етап» — стислість
- **Скорочення:** ВР (Верховна Рада), МЗС (Міністерство закордонних справ)

# Засоби впливу

## Риторичні запитання

Запитання, що не потребують відповіді, бо відповідь очевидна:

- «Чи можемо ми це ігнорувати?» (= Ні, не можемо)
- «Хто несе відповідальність?» (= Ті, про кого стаття)
- «До чого це призведе?» (= До поганого)

## Звертання до читача

Створює ілюзію діалогу:

- «Уявіть собі...»
- «Кожен із нас...»
- «Ми всі розуміємо...»

## Метафори

Абстрактне стає конкретним:

| Метафора | Значення |
|----------|----------|
| **хвиля змін** | масові зміни |
| **стіна нерозуміння** | глибокий конфлікт |
| **коріння проблеми** | глибинні причини |
| **світло в кінці тунелю** | надія |
| **вийти на новий рівень** | прогрес |

> **Чи знали ви?**
>
> Метафора **«залізна завіса»** (*iron curtain*) стала політичним терміном після промови Черчилля в 1946 році. Одна влучна метафора може змінити те, як люди бачать цілі явища. Тому журналісти полюбляють метафори — вони **фреймують** (обрамляють) реальність.

# Структура новини: перевернута піраміда

Класична структура новини — від найважливішого до найменш важливого:

```
┌─────────────────────────────────────────────┐
│  ЛІД: Хто? Що? Де? Коли?                    │ ← Найважливіше
├─────────────────────────────────────────────┤
│  ДЕТАЛІ: Як? Чому?                          │ ← Важливе
├─────────────────────────────────────────────┤
│  ФОН: Контекст, історія                     │ ← Корисне
├─────────────────────────────────────────────┤
│  ЦИТАТИ: Коментарі експертів                │ ← Додаткове
└─────────────────────────────────────────────┘
```

**Чому «перевернута»?** Бо в звичайному тексті кульмінація — в кінці. А в новині — на початку. Якщо читач перестане читати на середині — він уже знатиме головне.

# Приклад новини

**Заголовок:** Парламент ухвалив закон про освіту

**Лід (головне):**
Верховна Рада України у четвер ухвалила новий закон про освіту, який набуде чинності з вересня 2025 року.

**Деталі (як, чому):**
За законопроєкт проголосували 256 народних депутатів. Документ передбачає реформу середньої освіти та збільшення фінансування шкіл на 15%.

**Фон (контекст):**
Це вже третя спроба провести освітню реформу за останні п'ять років. Попередні два законопроєкти було відхилено через брак голосів.

**Цитата (авторитет):**
«Це історичний крок для української освіти», — заявив міністр освіти Оксан Лисенко.

# Нейтральність vs Оцінність

Порівняйте два тексти про одну подію:

**Нейтральна новина:**
«Учора в Києві відбувся мітинг. За даними поліції, взяли участь близько 5 тисяч осіб. Учасники висловили незгоду з рішенням уряду.»

**Оцінна стаття:**
«Учора Київ став свідком масової демонстрації народного невдоволення. Тисячі громадян вийшли на вулиці, щоб висловити рішучий протест проти скандального рішення влади.»

**Маркери оцінності в другому тексті:**
- «масова демонстрація» (не «мітинг»)
- «народного невдоволення» (емоційно забарвлене)
- «рішучий протест» (підсилення)
- «скандального рішення» (негативна оцінка)

# Сталі вирази публіцистики

| Вираз | Значення |
|-------|----------|
| **вийти на новий рівень** | досягти прогресу |
| **стати на порозі** | бути напередодні |
| **зробити крок назустріч** | піти на компроміс |
| **дати зелене світло** | дозволити |
| **поставити крапку** | завершити |
| **відкрити нову сторінку** | почати знову |
| **бути в епіцентрі** | бути в центрі подій |
| **взяти курс на** | обрати напрямок |

# Практика

## Проаналізуйте заголовки

1. «Уряд схвалив новий бюджет»
   > [!answer] **Інформативний**
   > [!explanation] факт
2. «Чи витримає економіка новий удар?»
   > [!answer] **Питальний**
   > [!explanation] інтрига
3. «Настав час змін!»
   > [!answer] **Закличний**
   > [!explanation] емоція
4. «Прем'єр: "Ми готові до викликів"»
   > [!answer] **Цитатний**
   > [!explanation] авторитет
## Знайдіть засоби впливу

**Текст:**
«Чи можемо ми й далі закривати очі на цю проблему? Кожен громадянин має зрозуміти: хвиля змін неминуча. Настав час діяти!»

**Засоби:**
- Риторичне запитання: «Чи можемо ми...?»
- Метафора: «хвиля змін»
- Звертання до читача: «Кожен громадянин»
- Заклик: «Настав час діяти!»

# Підсумок

## Публіцистичний стиль

| Риса | Приклад |
|------|---------|
| Актуальність | «Сьогодні», «учора» |
| Оцінність | «історичний», «скандальний» |
| Риторичність | «Чи можемо ми...?» |
| Метафоричність | «хвиля змін» |

## Структура новини

1. **Лід** — хто, що, де, коли (головне)
2. **Деталі** — як, чому
3. **Фон** — контекст
4. **Цитати** — коментарі

> **Запам'ятайте:**
>
> ✓ Публіцистика **інформує + впливає**
> ✓ Заголовки — у **теперішньому часі**
> ✓ **Риторичні засоби** привертають увагу
> ✓ Оцінна лексика **виражає позицію** автора
> ✓ Один факт можна подати **по-різному**
