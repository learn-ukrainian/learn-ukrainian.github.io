# Зміст уроку | Lesson Content

## warm-up | Розминка

### Що таке дієприкметник? | What is a Participle?

Дієприкметник — це слово, яке поєднує властивості дієслова та прикметника:

A participle is a word that combines properties of a verb and an adjective:

- **Написана** книга - A **written** book (participle from "написати")
- **Закрите** вікно - A **closed** window (participle from "закрити")
- **Зроблена** робота - **Done** work (participle from "зробити")

## presentation | Презентація

### Пасивні дієприкметники | Passive Participles

**Passive participles** describe a state resulting from an action done TO something:

| Verb | Participle | Meaning |
|------|------------|---------|
| читати → прочитати | прочитан**ий** | read (having been read) |
| писати → написати | написан**ий** | written |
| відкривати → відкрити | відкрит**ий** | opened |
| закривати → закрити | закрит**ий** | closed |

### Утворення з -ний | Formation with -ний

**Most common pattern:** perfective verb stem + **-н-** + adjective ending

| Verb | Stem | Participle |
|------|------|------------|
| зроби**ти** | зробл- | зробл**ений** |
| напис**ати** | напис- | напис**аний** |
| прочит**ати** | прочит- | прочит**аний** |
| вивч**ити** | вивч- | вивч**ений** |
| побуд**увати** | побудов- | побудов**аний** |
| приготув**ати** | приготов- | приготов**лений** |

**Stem changes to watch:**

| Pattern | Example |
|---------|---------|
| -ити → -ений | зробити → зроблений |
| -ати → -аний | написати → написаний |
| -увати → -ований | намалювати → намальований |
| -ювати → -йований | організувати → організований |

### Утворення з -тий | Formation with -тий

**For verbs with stems ending in vowel or certain consonants:**

| Verb | Stem | Participle |
|------|------|------------|
| відкри**ти** | відкри- | відкри**тий** |
| закри**ти** | закри- | закри**тий** |
| поми**ти** | поми- | поми**тий** |
| взя**ти** | взя- | взя**тий** |
| поча**ти** | поча- | поча**тий** |
| забу**ти** | забу- | забу**тий** |
| одягну**ти** | одягну- | одягну**тий** |
| прочита**ти** | прочита- | прочита**ний** |

**Pattern:** -ти verbs with vowel before -ти usually get **-тий**

### Відмінювання дієприкметників | Participle Declension

Participles decline like adjectives:

| Case | Masculine | Feminine | Neuter | Plural |
|------|-----------|----------|--------|--------|
| Nom | написаний | написана | написане | написані |
| Gen | написаного | написаної | написаного | написаних |
| Dat | написаному | написаній | написаному | написаним |
| Acc | написаний/ого | написану | написане | написані/их |
| Inst | написаним | написаною | написаним | написаними |
| Loc | написаному | написаній | написаному | написаних |

### Дієприкметники як прикметники | Participles as Adjectives

Participles function exactly like adjectives:

**Before noun:**
- **Написана** книга лежить на столі. - The written book is on the table.
- Я бачу **відкриті** двері. - I see the open(ed) door.

**After noun (predicate):**
- Книга **написана**. - The book is written.
- Двері **відкриті**. - The door is open.

### Порівняння: дієприкметник vs прикметник | Participle vs Adjective

Some participles have become regular adjectives:

| Participle (action) | Adjective (quality) |
|---------------------|---------------------|
| відкритий (opened by someone) | відкритий (open personality) |
| закритий (closed by someone) | закритий (closed system) |
| зайнятий (occupied) | зайнятий (busy) |

## practice | Практика

### Form Participles

1. написати → **написаний**
2. зробити → **зроблений**
3. відкрити → **відкритий**
4. прочитати → **прочитаний**
5. забути → **забутий**
6. купити → **куплений**
7. приготувати → **приготовлений**
8. збудувати → **збудований**

### Complete with Correct Form

1. ___ (написати) лист лежить на столі.
   → **Написаний** лист лежить на столі.

2. Двері ___ (закрити).
   → Двері **закриті**.

3. Я прочитав ___ (рекомендувати) книгу.
   → Я прочитав **рекомендовану** книгу.

4. ___ (запросити) гості вже прийшли.
   → **Запрошені** гості вже прийшли.

5. Вечеря вже ___ (приготувати).
   → Вечеря вже **приготовлена**.

### Match Verb to Participle

| Verb | Participle |
|------|------------|
| знайти | знайдений |
| втратити | втрачений |
| продати | проданий |
| організувати | організований |
| вимити | вимитий |

## production | Продукція

### Describe Your Room

Use participles to describe objects:

- Ліжко застелене. - The bed is made.
- Книги розставлені на полиці. - Books are arranged on the shelf.
- Вікно відкрите. - The window is open.

### What Was Done?

Describe completed actions using participles:

- Домашнє завдання зроблене.
- Лист написаний і відправлений.
- Вечеря приготовлена.

# Summary

## Past Passive Participle Formation

### Two Main Suffixes

| Suffix | When | Example |
|--------|------|---------|
| **-ний/-ений/-аний** | Most verbs | зроблений, написаний |
| **-тий** | Vowel stems | відкритий, забутий |

### Common Patterns

| Verb Type | Participle | Example |
|-----------|------------|---------|
| -ити | -ений | зробити → зроблений |
| -ати | -аний | написати → написаний |
| -увати | -ований | намалювати → намальований |
| -ти (vowel) | -тий | відкрити → відкритий |

### Usage
- Participles decline like adjectives
- Can be used before or after nouns
- Agree in gender, number, case

> **Key Takeaways**
>
> - Passive participles = verb meaning + adjective form
> - Two main suffixes: -ний and -тий
> - Decline exactly like adjectives
> - Formed from perfective verbs
> - Very common in written Ukrainian
