# Lesson Content

## Warm-up

### Your voice matters

Until now, you've been learning to describe things ‚Äî what exists, what happened, where to go. But at B1 level, you need to be able to say **what you think**.

Is learning languages difficult? What's the best season? Is it better to live in a city or a village?

There's no "right" answer to these questions. There's only your **–¥—É–º–∫–∞** (opinion). Today you'll learn to express it ‚Äî clearly, politely, persuasively.

> üí° **Did you know?**
>
> In Ukrainian culture, a **justified** opinion is highly valued. A simple "I think so" usually isn't enough ‚Äî an explanation with "because..." is expected. This differs from some cultures where simply stating your position is sufficient. When a Ukrainian asks "—á–æ–º—É?" (why?) after your opinion ‚Äî it's not a challenge, but an invitation to a deeper conversation!

## Presentation

### Expressing opinions

Ukrainian has many ways to express opinions, ranging from casual to formal:

**Direct expressions:**

| Ukrainian | English | Register |
|-----------|---------|----------|
| –Ø –¥—É–º–∞—é, —â–æ... | I think that... | neutral |
| –Ø –≤–≤–∞–∂–∞—é, —â–æ... | I consider/believe that... | neutral-formal |
| –Ø –≥–∞–¥–∞—é, —â–æ... | I think/suppose that... | neutral |
| –ù–∞ –º–æ—é –¥—É–º–∫—É, ... | In my opinion, ... | neutral |
| –ù–∞ –º—ñ–π –ø–æ–≥–ª—è–¥, ... | In my view, ... | formal |
| –ó –º–æ—î—ó —Ç–æ—á–∫–∏ –∑–æ—Ä—É, ... | From my point of view, ... | formal |
| –ú–µ–Ω—ñ –∑–¥–∞—î—Ç—å—Å—è, —â–æ... | It seems to me that... | softer |
| –Ø–∫ –Ω–∞ –º–µ–Ω–µ, ... | As for me, ... | informal |

**Examples:**

- **–Ø –¥—É–º–∞—é**, —â–æ —Ü–µ –≥–∞—Ä–Ω–∞ —ñ–¥–µ—è. ‚Äî I think it's a good idea.
- **–Ø –≤–≤–∞–∂–∞—é**, —â–æ –≤—ñ–Ω –º–∞—î —Ä–∞—Ü—ñ—é. ‚Äî I believe he's right.
- **–ù–∞ –º–æ—é –¥—É–º–∫—É**, —Ü–µ –Ω–∞–π–∫—Ä–∞—â–µ —Ä—ñ—à–µ–Ω–Ω—è. ‚Äî In my opinion, this is the best solution.
- **–ú–µ–Ω—ñ –∑–¥–∞—î—Ç—å—Å—è**, —â–æ –º–∏ –ø–æ–º–∏–ª—è—î–º–æ—Å—è. ‚Äî It seems to me we're wrong.
- **–Ø–∫ –Ω–∞ –º–µ–Ω–µ**, —Ü–µ –∑–∞–Ω–∞–¥—Ç–æ –¥–æ—Ä–æ–≥–æ. ‚Äî As for me, it's too expensive.

> üîç **Myth vs Fact**
>
> ‚ùå **Myth:** "–î—É–º–∞—Ç–∏ and –≤–≤–∞–∂–∞—Ç–∏ are the same."
>
> ‚úÖ **Fact:** There's a subtle difference! **–î—É–º–∞—Ç–∏** is the general, neutral option. **–í–≤–∞–∂–∞—Ç–∏** is slightly stronger, implying a considered position. "–Ø –¥—É–º–∞—é, –±—É–¥–µ –¥–æ—â" (spontaneous) vs "–Ø –≤–≤–∞–∂–∞—é, —â–æ –æ—Å–≤—ñ—Ç–∞ –≤–∞–∂–ª–∏–≤–∞" (considered position). In formal contexts (work, presentations), –≤–≤–∞–∂–∞—Ç–∏ sounds more professional.

### Expressing certainty

When you're confident about something, use these expressions:

**High certainty:**

| Ukrainian | English |
|-----------|---------|
| –Ø –≤–ø–µ–≤–Ω–µ–Ω–∏–π/–∞, —â–æ... | I'm sure that... |
| –Ø –ø–µ—Ä–µ–∫–æ–Ω–∞–Ω–∏–π/–∞, —â–æ... | I'm convinced that... |
| –ë–µ–∑ —Å—É–º–Ω—ñ–≤—É, ... | Without a doubt, ... |
| –ë–µ–∑–ø–µ—Ä–µ—á–Ω–æ, ... | Undoubtedly, ... |
| –û—á–µ–≤–∏–¥–Ω–æ, —â–æ... | Obviously, ... |
| –ó–≤—ñ—Å–Ω–æ, —â–æ... | Of course, ... |

**Examples:**

- **–Ø –≤–ø–µ–≤–Ω–µ–Ω–∏–π**, —â–æ —Ü–µ –ø—Ä–∞–≤–¥–∞. ‚Äî I'm sure this is true.
- **–ë–µ–∑ —Å—É–º–Ω—ñ–≤—É**, –≤—ñ–Ω –Ω–∞–π–∫—Ä–∞—â–∏–π –∫–∞–Ω–¥–∏–¥–∞—Ç. ‚Äî Without a doubt, he's the best candidate.
- **–û—á–µ–≤–∏–¥–Ω–æ**, —â–æ –ø–æ—Ç—Ä—ñ–±–Ω—ñ –∑–º—ñ–Ω–∏. ‚Äî Obviously, changes are needed.

> üí° **Did you know?**
>
> The word **–≤–ø–µ–≤–Ω–µ–Ω–∏–π** comes from "–ø–µ–≤–Ω–∏–π" (certain), which shares a root with "–≤—ñ—Ä–∞" (faith). To be –≤–ø–µ–≤–Ω–µ–Ω–∏–π literally means "to have faith in something." Interestingly, the English "sure" is also related to Latin "securus" (without worry) ‚Äî certainty as the absence of doubt!

### Expressing uncertainty (Hedging)

When you're less sure, use hedging expressions to soften your statements:

**Lower certainty:**

| Ukrainian | English |
|-----------|---------|
| –ú–æ–∂–ª–∏–≤–æ, ... | Perhaps, ... |
| –ù–∞–ø–µ–≤–Ω–æ, ... | Probably, ... |
| –ú–∞–±—É—Ç—å, ... | Probably, maybe... |
| –ó–¥–∞—î—Ç—å—Å—è, ... | It seems... |
| –ô–º–æ–≤—ñ—Ä–Ω–æ, ... | Probably, likely... |
| –ù–µ –≤–ø–µ–≤–Ω–µ–Ω–∏–π/–∞, –∞–ª–µ... | Not sure, but... |
| –í–∞–∂–∫–æ —Å–∫–∞–∑–∞—Ç–∏, –∞–ª–µ... | Hard to say, but... |

**Examples:**

- **–ú–æ–∂–ª–∏–≤–æ**, —Ü–µ –ø—Ä–∞–≤–¥–∞. ‚Äî Perhaps this is true.
- **–ú–∞–±—É—Ç—å**, –≤—ñ–Ω –∑–∞–ø—ñ–∑–Ω–∏—Ç—å—Å—è. ‚Äî He'll probably be late.
- **–ù–µ –≤–ø–µ–≤–Ω–µ–Ω–∏–π**, –∞–ª–µ –∑–¥–∞—î—Ç—å—Å—è, —â–æ —Ç–∞–∫. ‚Äî Not sure, but it seems so.

### Justifying your opinion

To support your opinion with reasons, use these conjunctions:

**Causal conjunctions:**

| Ukrainian | English |
|-----------|---------|
| —Ç–æ–º—É —â–æ | because |
| –æ—Å–∫—ñ–ª—å–∫–∏ | since, as |
| —á–µ—Ä–µ–∑ —Ç–µ, —â–æ | because of the fact that |
| –∑–∞–≤–¥—è–∫–∏ —Ç–æ–º—É, —â–æ | thanks to the fact that |
| –∞–¥–∂–µ | for, since, because |
| –±–æ | because (informal) |

**Examples:**

- –Ø —Ç–∞–∫ –¥—É–º–∞—é, **—Ç–æ–º—É —â–æ** —Ü–µ –ª–æ–≥—ñ—á–Ω–æ. ‚Äî I think so because it's logical.
- –í–æ–Ω–∞ –ø—Ä–∞–≤–∞, **–æ—Å–∫—ñ–ª—å–∫–∏** –º–∞—î –¥–æ—Å–≤—ñ–¥. ‚Äî She's right since she has experience.
- –¶–µ –≤–∞–∂–ª–∏–≤–æ, **–∞–¥–∂–µ** –≤–ø–ª–∏–≤–∞—î –Ω–∞ –≤—Å—ñ—Ö. ‚Äî This is important because it affects everyone.

> üí° **Did you know?**
>
> The conjunction **–∞–¥–∂–µ** is a favorite in journalism and formal speech. It comes from "–∞" + "–∂" + "—î" and carries a nuance of "because it's just so" ‚Äî as if you're appealing to a commonly known fact. "–ê–¥–∂–µ –≤—Å—ñ –∑–Ω–∞—é—Ç—å, —â–æ..." ("After all, everyone knows that...") is a rhetorical device that makes your opinion more persuasive.

### Comparing viewpoints

Use these expressions to present different perspectives:

| Ukrainian | English |
|-----------|---------|
| –ó –æ–¥–Ω–æ–≥–æ –±–æ–∫—É, ... –∑ —ñ–Ω—à–æ–≥–æ –±–æ–∫—É, ... | On one hand, ... on the other hand, ... |
| –ù–∞ –≤—ñ–¥–º—ñ–Ω—É –≤—ñ–¥... | Unlike... |
| –ü–æ—Ä—ñ–≤–Ω—è–Ω–æ –∑... | Compared to... |
| –ü—Ä–æ—Ç–µ / –û–¥–Ω–∞–∫ | However |
| –ù–µ–∑–≤–∞–∂–∞—é—á–∏ –Ω–∞ —Ç–µ, —â–æ... | Despite the fact that... |

**Examples:**

- **–ó –æ–¥–Ω–æ–≥–æ –±–æ–∫—É**, —Ü–µ –¥–æ—Ä–æ–≥–æ. **–ó —ñ–Ω—à–æ–≥–æ –±–æ–∫—É**, —Ü–µ —è–∫—ñ—Å–Ω–æ. ‚Äî On one hand, it's expensive. On the other hand, it's quality.
- **–ù–∞ –≤—ñ–¥–º—ñ–Ω—É –≤—ñ–¥** —ñ–Ω—à–∏—Ö, –≤—ñ–Ω –∑–∞–≤–∂–¥–∏ –ø—Ä–∏—Ö–æ–¥–∏—Ç—å –≤—á–∞—Å–Ω–æ. ‚Äî Unlike others, he always comes on time.
- –¶–µ –¥–æ–±—Ä–µ. **–ü—Ä–æ—Ç–µ** —î –Ω–µ–¥–æ–ª—ñ–∫–∏. ‚Äî This is good. However, there are drawbacks.

### Strengthening your argument

Use these expressions to emphasize key points:

| Ukrainian | English |
|-----------|---------|
| –ì–æ–ª–æ–≤–Ω–µ, —â–æ... | The main thing is that... |
| –ù–∞–π–≤–∞–∂–ª–∏–≤—ñ—à–µ, —â–æ... | The most important thing is that... |
| –ü–µ—Ä—à –∑–∞ –≤—Å–µ, ... | First of all, ... |
| –î–æ —Ç–æ–≥–æ –∂, ... | Moreover, ... |
| –ö—Ä—ñ–º —Ç–æ–≥–æ, ... | Besides, ... |
| –û—Å–æ–±–ª–∏–≤–æ... | Especially... |

**Examples:**

- **–ì–æ–ª–æ–≤–Ω–µ**, —â–æ –≤—Å—ñ –∑–¥–æ—Ä–æ–≤—ñ. ‚Äî The main thing is that everyone's healthy.
- **–ü–µ—Ä—à –∑–∞ –≤—Å–µ**, —Ü–µ –±–µ–∑–ø–µ—á–Ω–æ. ‚Äî First of all, it's safe.
- **–î–æ —Ç–æ–≥–æ –∂**, —Ü–µ —â–µ –π –¥–µ—à–µ–≤–æ. ‚Äî Moreover, it's also cheap.

### Summarizing

Use these to wrap up your argument:

| Ukrainian | English |
|-----------|---------|
| –û—Ç–∂–µ, ... | So, therefore, ... |
| –ó–∞–≥–∞–ª–æ–º, ... | In general, ... |
| –ü—ñ–¥—Å—É–º–æ–≤—É—é—á–∏, ... | Summarizing, ... |
| –û–¥–Ω–∏–º —Å–ª–æ–≤–æ–º, ... | In a word, ... |
| –í —Ü—ñ–ª–æ–º—É, ... | On the whole, ... |

**Examples:**

- **–û—Ç–∂–µ**, –º–∏ –≤–∏—Ä—ñ—à–∏–ª–∏ –ø–æ–≥–æ–¥–∏—Ç–∏—Å—è. ‚Äî So, we decided to agree.
- **–ó–∞–≥–∞–ª–æ–º**, —è –∑–∞–¥–æ–≤–æ–ª–µ–Ω–∏–π. ‚Äî In general, I'm satisfied.
- **–ü—ñ–¥—Å—É–º–æ–≤—É—é—á–∏**, —Ü–µ –±—É–≤ —É—Å–ø—ñ—Ö. ‚Äî Summarizing, it was a success.
