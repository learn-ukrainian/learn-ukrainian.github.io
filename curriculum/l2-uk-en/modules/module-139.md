### Vocabulary Application

#### Exercise 1: Express Opinion

Express opinions on these topics using discourse markers:

1. Дистанційна робота
   > [!answer] **На мою думку, дистанційна робота має як переваги, так і недоліки. З одного боку, вона дає більше свободи. З іншого боку, важко відокремити роботу від особистого життя.**
2. Вивчення мов
3. Технології в освіті

#### Exercise 2: Describe Changes

Describe trends using change vocabulary:

1. Population in cities
   > [!answer] **Населення міст постійно зростає. За останні роки воно значно збільшилося.**
2. Use of technology
3. Environment

#### Exercise 3: Use Collocations

Complete with correct collocations:

1. Я мушу ___ (make a decision) до завтра.
   > [!answer] **прийняти рішення**
2. Вона ___ (gave advice), яка мені допомогла.
   > [!answer] **дала пораду**
3. Це ___ (doesn't make sense).
   > [!answer] **не має сенсу**
4. Ми ___ (conducted negotiations) з партнерами.
   > [!answer] **вели переговори**
#### Exercise 4: Use Idioms

Respond with appropriate idioms:

1. «Він нічого не робить!»
   > [!answer] **Так, він цілий день байдики б'є.**
2. «Вона дуже щаслива!»
   > [!answer] **Так, вона на сьомому небі!**

4. «Він справжній експерт.»
   > [!answer] **Так, він собаку з'їв на цій справі.**
### Integrated Communication

#### Scenario 1: Business Discussion

**Setting:** You're discussing a new project with colleagues.

Use:
- Discourse markers (по-перше, крім того, отже)
- Business vocabulary (проєкт, зустріч, дедлайн)
- Opinion expressions (я вважаю, на мою думку)

**Example:**
«По-перше, я вважаю, що цей проєкт має великий потенціал. Крім того, ми маємо досвідчену команду. Проте потрібно визначити чіткий дедлайн. Отже, пропоную провести додаткову зустріч.»

#### Scenario 2: Travel Story

**Setting:** Tell about a trip using travel vocabulary.

Use:
- Travel vocabulary (кордон, віза, бронювання)
- Change vocabulary (покращитися, змінитися)
- Idioms where appropriate

## quiz: Vocabulary Integration

> Choose the best answer.

1. Для твердої думки вживаємо:
   - [x] вважаю
   - [ ] гадаю
   > Strong opinion = вважаю.

2. «Прийняти ___» — правильна колокація:
   - [x] рішення
   - [ ] помилку
   > приймати рішення.

3. «Він собаку з'їв» означає:
   - [x] він експерт
   - [ ] він їсть собак
   > He's an expert.

4. Слово для контрасту:
   - [x] проте
   - [ ] крім того
   > However = contrast.

5. Зростання = ___:
   - [x] increase
   - [ ] decrease
   > зростання = growth/increase.

6. «Вести ___» — правильна колокація:
   - [x] переговори
   - [ ] рішення
   > вести переговори.
