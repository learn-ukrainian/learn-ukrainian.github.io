# Lesson Content

## warm-up

### Why Cyrillic Isn't Scary

Cyrillic was designed for Slavic sounds. Many letters come from Greek, which is why some look familiar. The key: don't trust your eyes—trust the logic.

## presentation

### Group 1: True Friends

These letters look AND sound like their Latin equivalents:

• А а = A (as in 'father')
• Е е = E (as in 'bet')
• І і = I (as in 'machine')
• О о = O (as in 'hot')
• К к = K
• М м = M
• Т т = T

### Group 2: False Friends (DANGER!)

These letters LOOK Latin but sound DIFFERENT:

• В в = V (not B!)
• Н н = N (not H!)
• Р р = R (not P!) - trilled
• С с = S (not C!)
• У у = U (not Y!)
• Х х = KH (not X!) - like 'Bach'

### Group 3: New but Easy

Letters that don't exist in Latin but are straightforward:

• Б б = B (like a belly with a hat)
• Д д = D
• З з = Z
• Л л = L
• П п = P (like a doorway/portal)
• Ф ф = F

## practice

### Reading Practice

Let's read international words. Sound out each letter using what you've learned.

## production

### Real-World Reading

Try reading these words without looking at the transliteration first:

1. БАНК
2. ПАРК
3. МЕТРО
4. ПІЦА
5. СТУДЕНТ

### Remember

П = P, Р = R, В = V, Н = N, С = S, Х = KH

### Italic Warning

Cyrillic italic/cursive looks VERY different from print!

Dangerous letters:
• т → looks like "m" in italic
• п → looks like "n" in italic
• д → looks like "g" in italic
• и → looks like "u" in italic
• л → looks like "л" with a hook
• г → looks like "∂" in italic

We'll cover this in Module 02!

## quiz: Decipher International Words

> Choose the correct English meaning for each Cyrillic word.

1. What does ТАКСІ mean?
   - [x] Taxi
   - [ ] Task
   - [ ] Text
   - [ ] Talk
   > ТАКСІ (taksi) = Taxi. Remember: Т=T, А=A, К=K, С=S, І=I

2. What does БАНК mean?
   - [x] Bank
   - [ ] Back
   - [ ] Book
   - [ ] Bunch
   > БАНК (bank) = Bank. Б=B (not В!), А=A, Н=N (not H!), К=K

3. What does МЕТРО mean?
   - [x] Metro
   - [ ] Meter
   - [ ] Mother
   - [ ] Metal
   > МЕТРО (metro) = Metro/Subway. М=M, Е=E, Т=T, Р=R (not P!), О=O

4. What does ПАРК mean?
   - [x] Park
   - [ ] Pork
   - [ ] Part
   - [ ] Pack
   > ПАРК (park) = Park. П=P, А=A, Р=R (not P!), К=K

5. What does ПІЦА mean?
   - [x] Pizza
   - [ ] Peace
   - [ ] Piece
   - [ ] Price
   > ПІЦА (pitsa) = Pizza. П=P, І=I, Ц=TS, А=A

6. What does СТУДЕНТ mean?
   - [x] Student
   - [ ] Stadium
   - [ ] Study
   - [ ] Street
   > СТУДЕНТ (student) = Student. С=S (not C!), Т=T, У=U, Д=D, Е=E, Н=N (not H!)

7. What does КАФЕ mean?
   - [x] Café
   - [ ] Coffee
   - [ ] Cave
   - [ ] Calf
   > КАФЕ (kafe) = Café. К=K, А=A, Ф=F, Е=E

8. What does РАДІО mean?
   - [x] Radio
   - [ ] Ready
   - [ ] Radar
   - [ ] Ratio
   > РАДІО (radio) = Radio. Р=R (not P!), А=A, Д=D, І=I, О=O
