# Зміст уроку | Lesson Content

## warm-up | Розминка

### Світ змін | A World of Changes

Світ постійно змінюється. Як описати ці зміни?

The world constantly changes. How do we describe these changes?

- Населення **зростає** | The population is growing
- Ціни **підвищуються** | Prices are increasing
- Якість повітря **погіршується** | Air quality is worsening

## practice | Практика

### Describe the Changes

1. Population: 1 million → 1.5 million
   → **Населення зросло на 50% / на півмільйона.**

2. Prices: $100 → $80
   → **Ціни знизилися на 20%.**

3. Quality: bad → good
   → **Якість покращилася.**

4. Temperature: 20°C → 20°C
   → **Температура залишилася стабільною.**

5. Sales: rapid increase
   → **Продажі різко зросли.**

### Complete with Correct Verb

1. Ціни на бензин ___ (increase) удвічі.
   → **зросли / підвищилися**

2. Безробіття ___ (decrease) на 5%.
   → **зменшилося / знизилося**

3. Ситуація поступово ___ (improve).
   → **покращується**

4. Економіка ___ (recover) після кризи.
   → **відновлюється**

5. Населення міста ___ (remain stable).
   → **залишається стабільним**

### Translate to Ukrainian

1. Prices increased significantly.
   → **Ціни значно зросли.**

2. The situation gradually improved.
   → **Ситуація поступово покращилася.**

3. Sales fell sharply compared to last year.
   → **Продажі різко впали порівняно з минулим роком.**

4. The economy has been developing steadily.
   → **Економіка стабільно розвивається.**

## production | Продукція

Talk about changes over the past years:

- Як змінилося ваше життя за останні 5 років?

- Що залишилося незмінним?

### Discuss Global Trends

Describe trends using vocabulary:

- Як змінюється клімат?
- Як розвиваються технології?
- Як змінюється населення?

# Summary

## Describing Changes: Key Points

### Increase
зростати, підвищуватися, збільшуватися

### Decrease
зменшуватися, знижуватися, падати

### Quality Changes
покращуватися, погіршуватися

### Intensity
різко (sharply), значно (significantly), поступово (gradually)

### Comparisons
удвічі (twice), на ... % (by ... %), порівняно з (compared to)

> **Key Takeaways**
>
> - Rich vocabulary for describing changes
> - Important for academic and business contexts
> - Combine verbs with intensity adverbs
> - Use comparisons for precise descriptions
> - Time expressions add context to changes
