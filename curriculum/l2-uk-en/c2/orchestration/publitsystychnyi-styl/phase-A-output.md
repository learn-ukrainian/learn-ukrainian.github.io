## Common Learner Errors
1. **Passive Voice Overuse**: "Було повідомлено про подію..." → *Active*: "ЗМІ повідомили про подію..." (Ukrainian publicistic style values active agency).
2. **Lexical Calques**: "Приймати міри" (from Rus. принимать меры) → *Correct*: "Вживати заходів"; "Згідно закону" → *Correct*: "Згідно із законом".
3. **Register Confusion**: Mixing bureaucratic clichés (kantseliaryzmy) into lively reportage (e.g., using "з метою здійснення" instead of "щоб зробити").
