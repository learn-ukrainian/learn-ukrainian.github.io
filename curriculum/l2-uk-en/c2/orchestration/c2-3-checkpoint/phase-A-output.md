# Дослідження: Контрольна точка C2.3 — Оцінювання професійної спеціалізації

## State Standard Reference
§4.3.1.1, §4.3.3, §4.3.8: "Стилі української мови: ... офіційний; науковий ... Професіоналізми (у межах обраного фаху) ... Канцеляризми: згідно з...; відповідно до...; у зв’язку з... ... Уміння утворювати вторинні тексти, зокрема реферати, повідомлення, довідково-інформаційні тексти."
Alignment: Цей модуль перевіряє здатність учня оперувати професійною лексикою, створювати вторинні тексти (звіти, огляди) та дотримуватися норм офіційно-ділового та наукового стилів відповідно до вимог стандарту для рівня C2.

## Cultural Hooks
1. **ЄДКІ (Єдиний державний кваліфікаційний іспит)**: Реальний аналог для українських студентів (медиків, юристів) — ідеальний контекст для симуляції оцінювання.
2. **Атестація держслужбовців**: Процедура оцінювання результатів службової діяльності державних службовців в Україні, яка включає оцінку компетентностей та результативності.

## Common Learner Errors
1. **Приймати участь** → **Брати участь** — Типова калька з російської (принимать участие) або англійської (take part).
2. **Згідно інструкції** → **Згідно з інструкцією** — Граматична помилка (прийменник "згідно з" вимагає орудного відмінка, але часто помилково вживають родовий без прийменника).
3. **Компетенція vs Компетентність** → **Компетенція** (повноваження, сфера діяльності) vs **Компетентність** (рівень обізнаності, вміння). Часто плутають у професійних резюме.
