# –ß–∞—Å—Ç–∏–Ω–∞ II: –ì–µ–æ–ø–æ–ª—ñ—Ç–∏–∫–∞ –û–ª—ñ–º–ø—É

## 1. –ó–µ–≤—Å: –ü–æ—Ä—Ç—Ä–µ—Ç –°–∞–º–æ–¥—É—Ä–∞

## 2. –ö–∞–Ω—Ü–µ–ª—è—Ä—Å—å–∫–∞ –ú–æ–≤–∞ –ù–µ–±–∞
–ë–æ–≥–∏ –≥–æ–≤–æ—Ä—è—Ç—å —Å—É—Ä–∂–∏–∫–æ–º –∫–∞–Ω—Ü–µ–ª—è—Ä–∏—Å—Ç—ñ–≤.

–û–ª—ñ–º–ø –ø–µ—Ä–µ—Ç–≤–æ—Ä—é—î—Ç—å—Å—è –Ω–∞ **–î–µ–ø–∞—Ä—Ç–∞–º–µ–Ω—Ç**. –Æ–Ω–æ–Ω–∞ –ø–∏—à–µ "–¥–æ–Ω–æ—Å–∏". –í–µ–Ω–µ—Ä–∞ "–∫–ª–æ–ø–æ—á–µ". –ú–µ—Ä–∫—É—Ä—ñ–π "—Ä–æ–∑–Ω–æ—Å–∏—Ç—å –ø–æ–≤—ñ—Å—Ç–∫–∏".

## 3. –í–µ–Ω–µ—Ä–∞: –°–∏–ª–∞ –°–ª–∞–±–∫–∏—Ö
–í–µ–Ω–µ—Ä–∞, –º–∞—Ç–∏ –ï–Ω–µ—è, –¥—ñ—î —è–∫ —Ç–∏–ø–æ–≤–∞ —É–∫—Ä–∞—ó–Ω—Å—å–∫–∞ –∂—ñ–Ω–∫–∞, —è–∫–∞ –∑–∞—Ö–∏—â–∞—î —Å–≤–æ—é –¥–∏—Ç–∏–Ω—É –ø–µ—Ä–µ–¥ –∂–æ—Ä—Å—Ç–æ–∫–∏–º —Å–≤—ñ—Ç–æ–º.

## 4. –ë–æ–≥–∏-–ß–∏–Ω–æ–≤–Ω–∏–∫–∏
–ü–æ–¥–∏–≤—ñ—Ç—å—Å—è –Ω–∞ —ñ–Ω—à–∏—Ö –º–µ—à–∫–∞–Ω—Ü—ñ–≤ –Ω–µ–±–∞. –¶–µ —Ç–∏–ø–∞–∂—ñ —ñ–º–ø–µ—Ä—Å—å–∫–æ—ó –∞–¥–º—ñ–Ω—ñ—Å—Ç—Ä–∞—Ü—ñ—ó:

# –ß–∞—Å—Ç–∏–Ω–∞ III: –°–∞–∫—Ä–∞–ª—å–Ω–∞ –ì–µ–æ–≥—Ä–∞—Ñ—ñ—è

## 1. –ß–æ—Ä–Ω–µ –ú–æ—Ä–µ —è–∫ "–¢—É—Ä–µ—Ü—å–∫–∏–π –°—Ç–∞–≤"

# –ß–∞—Å—Ç–∏–Ω–∞ IV: –ó–≤—É–∫ –°–≤–æ–±–æ–¥–∏ (–§—ñ–ª–æ–ª–æ–≥—ñ—á–Ω–∏–π –ê–Ω–∞–ª—ñ–∑)

## 2. –ê–ª—ñ—Ç–µ—Ä–∞—Ü—ñ—è: –ó–≤—É–∫–æ–ø–∏—Å –ë—É—Ä—ñ

> *"–Ø–∫ –ø—ñ–¥—Ö–æ–ø–∏–ª–∏—Å—å –≤—ñ—Ç—Ä–∏-—Å—É–∫–∏*

## 3. –†–∏—Ç–º —è–∫ –ü–∞–º'—è—Ç—å
