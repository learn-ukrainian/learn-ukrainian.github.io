## Examples of Excellence

### Example 1: Contrast Table (lines 135-141)

```markdown
| Одноразова дія (ДВ) | Повторювана дія (НДВ) |
|---------------------|----------------------|
| Одного разу я **написав** листа. | Я часто **писав** листи додому. |
| Того разу вона **приготувала** обід. | Вона завжди **готувала** обід о першій. |
| Якось він **забув** ключі. | Він постійно **забував** ключі. |
```

**Why Excellent:**
- Minimal pairs: Same verb, different aspect
- Time markers reinforce distinction
- Parallel structure aids memory
- Efficient learning design

### Example 2: Decision Algorithm (lines 189-203)

```markdown
Задайте собі ці питання:

1. Скільки разів відбулася дія?
   - Один раз → ДВ
   - Багато разів → НДВ

2. Чи є маркер частоти?
   - Одного разу, нарешті → ДВ
   - Завжди, щодня → НДВ

3. Чи розповідаєте ви про звичку чи конкретну подію?
   - Звичка → НДВ
   - Подія → ДВ
```

**Why Excellent:**
- Three diagnostic questions
- Concrete decision-making framework
- Empowers self-correction
- Replicable strategy for future use

### Example 3: Nuanced Explanation (lines 162-168)

```markdown
Слово "колись" може означати:
- "Одного разу в далекому минулому" → ДВ
- "У минулому взагалі" → НДВ

Приклади:
- Колись **я жив** у Києві. (НДВ, загальна характеристика)
- Колись **я зустрів** її. (ДВ, конкретна подія)

Як зрозуміти? Залежить від контексту:
- Стан чи звичка → НДВ
- Конкретна подія → ДВ
```

**Why Excellent:**
- Addresses ambiguity directly
- Provides both examples
- Gives decision logic
- Doesn't oversimplify
