# Дослідження: Вид у наказовому способі

## State Standard Reference
§4.2.3.2: "Наказовий спосіб. Утворення форм наказового способу: мрій, мріймо; кажіть, скажіть, скажімо; пий, пиймо, пийте; борися, борімося, боріться, хай / нехай бореться, хай / нехай борються."
Alignment: While the B1 standard specifically mandates the formation of these imperative paradigms (including perfective/imperfective pairs like кажіть/скажіть), this module fulfills the standard by focusing on the functional usage and aspectual choice between these forms for polite invitations vs. direct commands.

## Cultural Hooks
1. The "Politeness Paradox" in Ukrainian: The use of the imperfective aspect (НДВ) natively functions to soften a command into an invitation (e.g., «Сідайте», «Заходьте»), establishing politeness through grammar rather than just appending words like "please".
2. The proverb «Вік живи — вік учись» highlights the ingrained cultural connection between continuous, lifelong processes and the mandatory use of the imperfective aspect for general advice.

## Common Learner Errors
1. Using Perfective for general prohibitions: Learners often say *«Не скажи»* instead of *«Не кажи»* for "don't say". Prohibition almost always mandates Imperfective (НДВ).
2. Using Imperfective for a single result: Learners sometimes say *«Відчиняй вікно»* when asking for a one-time action, instead of the correct *«Відчини вікно»*.
