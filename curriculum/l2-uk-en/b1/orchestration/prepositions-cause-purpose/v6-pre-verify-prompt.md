## Sections to research

- **Причина: через та завдяки**: через + Зн.в. — negative or neutral cause (Заболотний Grade 7 p.193): через хворобу (because of illness), через неуважність (due to carelessness), через погану погоду (due to bad weather), через війну (because of the war). 'Прийменник через вказує на причину, що заважає здійсненню чогось.'; завдяки + Д.в. — positive cause: завдяки допомозі друзів (thanks to friends' help), завдяки наполегливості (thanks to perseverance), завдяки вчителям (thanks to teachers). 'Прийменник завдяки містить вказівку на причину, яка СПРИЯЄ здійсненню.' Critical rule: NEVER use завдяки for negative outcomes. *Завдяки хворобі він не прийшов — WRONG. Через хворобу він не прийшов — CORRECT.; Practice: 12 sentences — learners choose через or завдяки based on whether the cause is positive or negative. 'Я запізнився _____ затор на дорозі.' (через) 'Ми перемогли _____ вашій підтримці.' (завдяки)
- **Причина: від та з**: від + Р.в. — physical or emotional cause (external stimulus): від радості (from joy), від сорому (from shame), від болю (from pain), від холоду (from cold), від страху (from fear). 'Він почервонів від сорому. Вона тремтіла від холоду.' These express involuntary reactions to a cause.; з + Р.в. — inner cause (motivation, character): з цікавості (out of curiosity), з повагою (with respect — manner, not cause), зі страху (from fear — close to від страху but more internal). 'Він зробив це з цікавості (He did it out of curiosity).' Note: з + Р.в. often overlaps with від + Р.в. for emotions, but з emphasises MOTIVATION while від emphasises REACTION.; Additional cause prepositions: унаслідок/внаслідок + Р.в. (as a result of — внаслідок аварії), у результаті + Р.в. (as a result of — у результаті перемовин), у зв'язку з + Ор.в. (in connection with — у зв'язку з реформою). These are formal/written register. From Авраменко Grade 9 p.86: складені прийменники причини.
- **Мета: для, заради, за, щоб**: для + Р.в. — 'for (the purpose/benefit of)': для друга (for a friend), для здоров'я (for health), для роботи (for work), для навчання (for studying). This is the most common purpose preposition.; заради + Р.в. — 'for the sake of' (emphatic purpose): заради миру (for the sake of peace), заради дітей (for the sake of children), заради справедливості (for the sake of justice). More emotionally charged than для.; за + Ор.в. — 'to fetch/get' (purpose of going): піти за хлібом (go for bread), зайти за другом (drop by for a friend), поїхати за покупками (go shopping). From Заболотний Grade 7 p.193: 'Зайти за книжкою' means to go and pick up a book; 'зайти по книжку' is an equivalent (по + Зн.в. for purpose).
- **Складні конструкції причини і мети**: Compound prepositions for cause: через те що + clause: Через те що він захворів, він не прийшов. завдяки тому що + clause: Завдяки тому що вона допомогла, ми встигли. у зв'язку з + Ор.в. (in connection with — formal): у зв'язку з карантином.; Compound prepositions for purpose: для того щоб + infinitive: Для того щоб зрозуміти, треба прочитати. з метою + Р.в. (with the aim of — formal): з метою покращення. заради того щоб + infinitive (emphatic purpose).; Practice: combine simple sentences into complex ones using compound cause/purpose constructions. 'Він не прийшов. Він захворів.' → 'Він не прийшов через те, що захворів.'
- **Причина vs мета: розрізнення**: Cause answers ЧОМУ? (why did it happen?): Він запізнився через затор. Purpose answers НАВІЩО? ДЛЯ ЧОГО? (what for?): Він прийшов для зустрічі. Same preposition, different meaning: Він прийшов через парк (path — not cause or purpose!). Він не прийшов через хворобу (cause). Context and the question word (чому/навіщо) disambiguate.; Practice: classify 12 sentences as expressing cause, purpose, or path. Then rewrite each using an alternative construction: 'Завдяки вашій допомозі ми перемогли' → 'Ми перемогли, бо ви допомогли' (subordinate clause alternative).; Reading passage: a news article about a community project using cause and purpose constructions throughout. Learners identify each cause/purpose expression, classify it (через/завдяки/від/для/щоб), and explain why that particular preposition was chosen over alternatives.
- **Підсумок**: Reference table: CAUSE: через + Зн.в. (negative), завдяки + Д.в. (positive), від + Р.в. (reaction), з + Р.в. (motivation). PURPOSE: для + Р.в. (general), заради + Р.в. (emphatic), за + Ор.в. (to fetch), щоб + inf. (in order to).; Golden rule: завдяки = GOOD cause, через = BAD cause. Never mix them.; Self-check: 1. Express negative cause (через + Зн.в.). 2. Express positive cause (завдяки + Д.в.). 3. Express purpose with для + Р.в. and щоб + inf. 4. Use compound constructions (через те що, для того щоб). 5. Distinguish від (reaction) from з (motivation). Preview: порядкові числівники і відмінки.
