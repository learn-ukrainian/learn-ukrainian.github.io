# Граматика B1: Творення прикметників



## Як це пояснюють у школі (How Schools Teach This)

Творення прикметників (словотвір прикметників) є центральною темою в українській шкільній програмі, яку починають вивчати в 5-6 класах. Підхід базується на розумінні морфемної будови слова та твірної основи.

**Клас 5:** Учні знайомляться з основними поняттями словотвору: основа (похідна і непохідна), твірне слово, і значущі частини слова (корінь, префікс, суфікс). (Джерело: `5-klas-ukrmova-uhor-2022-1_s0223`). Основні способи словотворення — префіксальний, суфіксальний, префіксально-суфіксальний — представлені як загальні механізми поповнення мови (Джерело: `5-klas-ukrmova-uhor-2022-1_s0245`).

**Клас 6:** Це ключовий рік для вивчення прикметників. Учні систематично вивчають:
1.  **Групи прикметників за значенням:** Якісні, відносні та присвійні. Це фундаментальний поділ, на якому будуються подальші пояснення (Джерела: `6-klas-ukrmova-avramenko-2023_s0130`, `6-klas-ukrmova-zabolotnyi-2020_s0135`, `6-klas-ukrmova-betsa-2023_s0118`).
2.  **Способи творення:** Тема розглядається в контексті кожної групи. Підручники детально пояснюють, які суфікси є продуктивними для творення відносних (`-н-`, `-ов-`, `-ськ-`) та присвійних (`-ин`, `-ів`) прикметників (Джерела: `6-klas-ukrmova-avramenko-2023_s0144`, `6-klas-ukrmova-betsa-2023_s0119`).
3.  **Фонетичні зміни:** Особлива увага приділяється змінам приголосних при додаванні суфіксів `-ськ-` (наприклад, `Прага` -> `празький`) та `-ств-` (Джерело: `5-klas-ukrmova-uhor-2022-1_s0238`, `7-klas-ukrmova-avramenko-2024_s0039`).
4.  **Орфографія:** Правила написання `-н-` та `-нн-` у прикметниках є важливою орфографічною темою, яка закріплюється через вправи (Джерела: `6-klas-ukrmova-zabolotnyi-2020_s0157`, `6-klas-ukrmova-litvinova-2023_s0218`).

**Старші класи (10-11):** Тема повторюється та поглиблюється. Учні систематизують знання про суфікси, розглядають менш продуктивні моделі та аналізують перехід прикметників з однієї групи в іншу (`залізний стовп` (відносний) -> `залізна воля` (якісний)) (Джерела: `10-klas-ukrmova-karaman-2018_s0284`, `11-klas-ukrajinska-mova-avramenko-2019_s0035`).

Загалом, шкільний підхід є структурним: від визначення груп за значенням до аналізу морфемних засобів їх творення, з акцентом на орфографічні та фонетичні правила.

## Повна парадигма (Full Paradigm)

Творення прикметників в українській мові відбувається переважно **суфіксальним** та **префіксально-суфіксальним** способами. Твірними основами найчастіше виступають іменники.

### 1. Творення якісних прикметників (Formation of Qualitative Adjectives)

Якісні прикметники, що виражають ознаку безпосередньо, творяться від інших частин мови або від інших прикметників, додаючи емоційного забарвлення чи відтінку якості.

| Спосіб | Суфікс/Префікс | Твірне слово (Основа) | → | Утворений прикметник | Значення/Примітка | Джерело |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| Суфіксальний | `-лив-` | `щастя` | → | `щасливий` | Схильність до чогось | `6-klas-ukrmova-litvinova-2023_s0215` |
| | `-н-` | `сила` | → | `сильний` | Наявність ознаки | `ext-other_blogs-61` |
| | `-к-` | `солодити` | → | `солодкий` | | |
| | `-ав-` / `-яв-` | `чорний` | → | `чорнявий` | Відтінок, схожість | `6-klas-ukrmova-litvinova-2023_s0216` |
| | `-уват-` | `синій` | → | `синюватий` | Неповний вияв ознаки | `6-klas-ukrmova-litvinova-2023_s0216` |
| | `-еньк-` / `-есеньк-` | `малий` | → | `маленький`, `малесенький` | Зменшено-пестливе | `ext-other_blogs-61` |
| | `-езн-` / `-енн-` | `здоровий`, `високий` | → | `здоровенний`, `височенний` | Збільшення, найвища міра | `5-klas-ukrmova-uhor-2022-1_s0188`, `6-klas-ukrmova-zabolotnyi-2020_s0157` |
| | `-ущ-` / `-ющ-` | `злий` | → | `злющий` | Дуже високий ступінь ознаки | `6-klas-ukrmova-litvinova-2023_s0216` |
| Префіксальний | `пре-` | `гарний` | → | `прегарний` | Найвищий ступінь ознаки | `10-klas-ukrmova-karaman-2018_s0271` |

| Префіксально-суфіксальний | `не-`...`-нн(ий)` | `здолати` | → | `нездоланний` | Неможливість виконання дії | `5-klas-ukrmova-uhor-2022-1_s0188` |
| | `без-`...`-н(ий)` | `межа` | → | `безмежний` | Відсутність ознаки | `6-klas-ukrmova-litvinova-2023_s0215` |

### 2. Творення відносних прикметників (Formation of Relative Adjectives)

Це найпродуктивніший спосіб творення. Прикметники виражають ознаку через відношення до предмета, матеріалу, місця, часу тощо.

| Ознака за відношенням | Суфікс | Твірний іменник | → | Утворений прикметник | Джерело |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Матеріал** | `-ан-` / `-ян-` | `глина` | → | `глиняний` | `6-klas-ukrmova-avramenko-2023_s0150` |
| | `-ов-` / `-ев-` / `-єв-` | `береза`, `слива` | → | `березовий`, `сливовий` | `10-klas-ukrmova-karaman-2018_s0284` |
| | `-н-` | `мідь` | → | `мідний` | `10-klas-ukrmova-karaman-2018_s0284` |
| **Час** | `-н-` | `осінь` | → | `осінній` | `10-klas-ukrmova-karaman-2018_s0284` |
| **Місце/Призначення** | `-н-` | `стіна`, `школа` | → | `стінний`, `шкільний` | `10-klas-ukrmova-karaman-2018_s0284` |
| **Належність (місце, особа)** | `-ськ-` / `-зьк-` / `-цьк-` | `Київ`, `Париж`, `ткач` | → | `київський`, `паризький`, `ткацький`| `10-klas-ukrmova-karaman-2018_s0284` |
| **Відношення до дії** | `-льн-` | `навчати` | → | `навчальний` | `6-klas-ukrmova-betsa-2023_s0119` |

#### Зміни приголосних при творенні з суфіксом `-ськ-`

Це ключове фонетичне правило.

| Кінцевий приголосний основи | + | Суфікс | = | Результат | Приклад | Джерело |
| :--- | :-: | :--- | :-: | :--- | :--- | :--- |
| `г`, `ж`, `з` | + | `-ськ-` | = | `-зьк-` | `Прага` → `празький`; `Париж` → `паризький` | `5-klas-ukrmova-uhor-2022-1_s0238` |
| `к`, `ч`, `ц` | + | `-ськ-` | = | `-цьк-` | `козак` → `козацький`; `ткач` → `ткацький` | `5-klas-ukrmova-uhor-2022-1_s0238` |
| `х`, `ш`, `с` | + | `-ськ-` | = | `-ськ-` | `чех` → `чеський`; `товариш` → `товариський` | `5-klas-ukrmova-uhor-2022-1_s0238` |
| *Інші приголосні* | + | `-ськ-` | = | `-ськ-` | `студент` → `студентський`; `брат` → `братський` | `5-klas-ukrmova-uhor-2022-1_s0238` |

**Винятки:** `казахський`, `тюркський`, `баскський`, `нью-йоркський`, `герцогський` (Джерело: `7-klas-ukrmova-avramenko-2024_s0039`).

#### Написання `-н-` та `-нн-` у прикметниках

| Правило | Приклади | Пояснення | Джерело |
| :--- | :--- | :--- | :--- |
| **Одна `-н-`** | `орлиний`, `гречаний`, `жаданий`, `шалений` | Утворено суфіксами `-ин-`, `-ян-`, `-їн-` від основ, що не закінчуються на `н`. Або в суфіксах зі значенням звичайної міри. | `5-klas-ukrmova-uhor-2022-1_s0188`, `6-klas-ukrmova-zabolotnyi-2020_s0157` |
| **Подвоєна `-нн-`** | `осінній` (`осінь`+`н`), `вагонний` (`вагон`+`н`) | Основа іменника закінчується на `-н-` + додається суфікс `-н-`. | `6-klas-ukrmova-zabolotnyi-2020_s0157` |
| | `височенний`, `здоровенний`, `несказанний` | У наголошених суфіксах `-енн-`, `-анн-` (`-янн-`) зі значенням перебільшення або неможливості дії. | `6-klas-ukrmova-zabolotnyi-2020_s0157`, `6-klas-ukrmova-litvinova-2023_s0218`|
| | `священний`, `благословенний`, `огненний` | Слова старослов'янського походження. | `6-klas-ukrmova-zabolotnyi-2020_s0157` |

### 3. Творення присвійних прикметників (Formation of Possessive Adjectives)

Присвійні прикметники вказують на належність особі чи тварині.

| Твірна основа (Іменник) | + Суфікс | → | Утворений прикметник | Примітка | Джерело |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Іменники I відміни** (ч.р. та ж.р. на `-а`/`-я`) | `-ин` | `сестра`, `Микола` | → | `сестрин`, `Миколин` | Основа на твердий приголосний. | `11-klas-ukrajinska-mova-avramenko-2019_s0035` |
| | `-їн` | `Марія`, `Софія` | → | `Маріїн`, `Софіїн` | Основа закінчується на `[й]`. | `6-klas-ukrmova-betsa-2023_s0119` |
| **Іменники II відміни** (ч.р. на приголосний або `-о`) | `-ів` (`-їв`) | `батько`, `Сергій`, `брат` | → | `батьків`, `Сергіїв`, `братів` | У Н.в. однини ч.р. Суфікс чергується з `-ов-`/`-ев-` в інших формах: `батькового`, `Сергієва`. | `6-klas-ukrmova-betsa-2023_s0119`, `11-klas-ukrajinska-mova-avramenko-2019_s0035` |
| **Назви тварин** | `-ин-`/`-їн-` | `бджола`, `соловей` | → | `бджолиний`, `солов'їний` | | `5-klas-ukrmova-uhor-2022-1_s0188`, `11-klas-ukrajinska-mova-avramenko-2019_s0035` |
| | `-ач-`/`-яч-` | `собака`, `заєць` | → | `собачий`, `заячий` | | `6-klas-ukrmova-litvinova-2023_s0200` |

## Частотність і пріоритети

Для рівня B1/B2 пріоритети у вивченні творення прикметників такі:

1.  **Відносні прикметники (високий пріоритет):** Це найпоширеніший спосіб розширити словниковий запас.
    *   **Must-know:** Суфікси `-н-` (`літній`), `-ов-` (`квітковий`), `-ськ-` (`український`). Вони використовуються постійно для опису предметів.
    *   Правила чергування приголосних з `-ськ-` (`-зьк-`, `-цьк-`) є обов'язковими для правильного написання назв, пов'язаних з географією та національностями (`французький`, `німецький`).

2.  **Присвійні прикметники (високий пріоритет):** Незамінні для вираження належності у повсякденному мовленні.
    *   **Must-know:** Моделі `батьків` / `батькова` та `сестрин` / `сестрина` є високочастотними. Розуміння різниці між творенням від іменників І та ІІ відмін є ключовим.

3.  **Якісні прикметники (середній пріоритет):**
    *   Суфікси емоційної оцінки (`-еньк-`, `-езн-`) є дуже поширеними у розмовній мові, надаючи мовленню природності. Їх варто засвоїти для розуміння та відтворення неформального спілкування.
    *   Суфікси `-нн-` зі значенням перебільшення (`незрівнянний`, `здоровенний`) важливі для експресивного мовлення рівня B2.

4.  **Складні випадки (нижчий пріоритет):**
    *   Прикметники, утворені від прислівників (`вчорашній`) або через складання основ (`чорнобровий`), менш частотні для активного вживання, але важливі для розуміння.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `козакський` | `козацький` | Неправильне застосування правила чергування приголосних. `к` + `-ськ-` дає `-цьк-`. (Джерело: `5-klas-ukrmova-uhor-2022-1_s0238`) |
| `Ольгин` / `Ольгова` | `Ольжин` | Відсутність чергування `г` → `ж` перед суфіксом `-ин-` при творенні присвійних прикметників від імен на `-га`. (Джерело: `10-klas-ukrmova-karaman-2018_s0284`) |
| `Миколаїв зошит` | `Миколин зошит` | Неправильний вибір суфікса. Для іменників І відміни (як `Микола`) використовується суфікс `-ин`, а не `-ів`. (Джерело: `11-klas-ukrajinska-mova-avramenko-2019_s0035`) |
| `дерев'яний характер` | `залізний характер` | Буквальний переклад англійського "wooden character". В українській мові для передачі негнучкості характеру використовується метафора `залізний`, а `дерев'яний` вживається для опису незграбних рухів (`дерев'яні ноги`). (Джерело: `6-klas-ukrmova-betsa-2023_s0120`) |
| `туманий ранок` | `туманний ранок` | Помилка в орфограмі "-н- і -нн-". Прикметник `туманний` утворений від іменника `туман` (основа на `-н`) за допомогою суфікса `-н-`, тому відбувається подвоєння. (Джерело: `6-klas-ukrmova-zabolotnyi-2020_s0157`) |
| `Надіїн олівець` | `Надіїн олівець` | Це не помилка, але важливе уточнення. Хоча формально `Надія` є І відміною, при творенні присвійного прикметника від основи на `-ій` використовується суфікс `-їн`. (Джерело: `11-klas-ukrajinska-mova-avramenko-2019_s0039`). Помилка виникає, якщо застосувати правило для імен типу `Микола` -> `Надиин`. |

## Деколонізаційні застереження (Decolonization Notes)

**Обов'язково для письменника:** Українська мова має власну, самобутню систему словотвору. Неприпустимо пояснювати українські форми як "варіанти" чи "винятки" з російських норм. Українська є точкою відліку.

1.  **Присвійні прикметники від імен І відміни:** Це одна з найяскравіших відмінностей. В українській мові від чоловічих імен на `-а` (`Микола`, `Ілля`) та жіночих (`сестра`, `Галя`) присвійні прикметники утворюються за допомогою суфікса **`-ин` / `-їн`**: `Миколин`, `Іллин`, `сестрин`, `Галинин`. Це високопродуктивна модель. Російська мова тут часто використовує суфікс `-ов` (`Николов`) або описові конструкції. Акцент на українській моделі є критично важливим. (Джерела: `11-klas-ukrajinska-mova-avramenko-2019_s0035`, `10-klas-ukrmova-karaman-2018_s0235`).

2.  **Чергування голосних в суфіксах:** Суфікс **`-ів`** у присвійних прикметниках (`батьків`, `Андріїв`) є формою називного відмінка однини. В інших відмінках він закономірно чергується з **`-ов-`** або **`-ев-`** (`батькового`, `Андрієвого`). Це унікальна риса, пов'язана з історичним розвитком української фонетики (`о`, `е` в нових закритих складах). Пояснювати це треба через внутрішню логіку української мови, а не через порівняння. (Джерело: `6-klas-ukrmova-betsa-2023_s0119`).

3.  **Вплив російської на наголос:** Як зазначено в `ext-ulp_youtube-29`, під впливом російської мови та суржику українці можуть неправильно наголошувати слова (напр., `нена́видіти` замість `ненави́діти`). При вивченні новоутворених прикметників слід завжди перевіряти наголос за словником, оскільки він може не збігатися з російським аналогом.

4.  **Уникання кальок:** При перекладі метафоричних прикметників слід шукати українські відповідники, а не калькувати з російської чи англійської. Наприклад, `золоті руки` (якісний) — це вправні руки, і ця ідіома є спільною для багатьох мов. Однак для інших понять існують унікальні українські метафори (`залізна воля`, а не `стальна`). (Джерело: `11-klas-ukrajinska-mova-avramenko-2019_s0039`).
