# V6 Pre-Write Verification — Tool-Forced Fact Checking

You MUST verify facts using the tools below BEFORE writing begins. This is a verification-only step — do NOT write any module content.

## Module

**Module:** 35: Найвищий ступінь прикметників (B1, B1.4 [Degrees of Comparison & Word Formation])

## Plan vocabulary to verify

- найвищий ступінь (superlative degree)
- префікс (prefix — word-forming element before the root)
- найбільш (most — used in compound superlative)
- найменш (least — used in compound superlative)
- щонайкращий (the very best — intensified superlative with що-)
- якнайшвидший (as fast as possible — intensified superlative with як-)
- вельми (very, exceedingly — formal synonym of дуже)
- вкрай (extremely — degree intensifier)
- надзвичайно (extraordinarily — degree intensifier)
- підсилений (intensified — describing що-/як- forms)
- синонімічний (synonymic — alternative way of expressing the same idea)
- стилістичний (stylistic — relating to register and style choices)
- прегарний (very beautiful — emphatic prefix пре-)
- білісінький (absolutely white — emphatic suffix -ісіньк-)
- занадто (too much — degree word showing excess)

## Sections to research

- **Проста форма найвищого ступеня**: Formation rule from Литвінова Grade 6 p.198: prefix най- + проста форма вищого ступеня. сильніший → найсильніший, молодший → наймолодший, дорожчий → найдорожчий, вищий → найвищий. The superlative inherits all alternations from the comparative.; The simple superlative declines like a regular adjective: Н. найсильніший / найсильніша / найсильніше / найсильніші Р. найсильнішого / найсильнішої / найсильнішого / найсильніших Practice with declined forms in sentences: Це найцікавіша книжка, яку я читав. Ми живемо в найгарнішому місті.; Common formation patterns — practice batch: дорогий → дорожчий → найдорожчий високий → вищий → найвищий широкий → ширший → найширший тихий → тихіший → найтихіший 10-12 adjectives for learners to form full chains.
- **Складена форма найвищого ступеня**: Formation from Заболотний Grade 8 p.18: найбільш/найменш + початкова форма прикметника. найбільш цікавий, найменш відомий, найбільш поширений. The adjective declines; найбільш/найменш is invariable.; Register difference: складена form is formal, academic, official. 'найбільш поширена хвороба' (medical text) vs 'найпоширеніша хвороба' (everyday speech). Practice: rewrite formal sentences into informal and vice versa.; Critical error prevention: NEVER combine най- with більш (*найбільш найкращий). NEVER use самий (*самий кращий — this is a Russicism from самый лучший). Correct: найкращий or найбільш добрий.
- **Підсилені форми: що- та як-**: Ukrainian has a unique intensifying mechanism absent in English: що- + найвищий ступінь: щонайкращий, щонайшвидший, щонайвищий. як- + найвищий ступінь: якнайкращий, якнайшвидший, якнайвищий. Meaning: 'the very best possible,' 'as fast as absolutely possible.'; Usage context: що- forms express absolute maximum/minimum; як- forms express desire or effort toward the maximum. Зроби це якнайшвидше (do it as fast as you can — effort implied). Це щонайкращий варіант (this is the absolute best option — factual).; Practice sentences in context: instructions, advice, evaluations. Приходьте якнайскоріше. Це щонайменша ціна. Він обрав якнайпростіший шлях.
- **Синонімічні засоби**: From Заболотний Grade 8 p.21: alternatives that express the superlative idea without formal superlative forms: дуже, вельми, занадто, вкрай, особливо, надзвичайно + adjective. вельми чемний, особливо уважний, вкрай виважений.; Emphatic repetition and prefixes (Заболотний Grade 8 p.21): гарний-прегарний, білий-білісінький, синій-синій. Prefix пре-: предобрий, прегарний, прекрасний. These are stylistically richer than formal superlatives.; Practice: rewrite sentences using different synonymic means. 'Це найкращий фільм' → 'Цей фільм надзвичайно гарний.' Discussion: which version sounds more natural in which context?
- **Ступені порівняння в контексті**: Combined practice using вищий and найвищий ступені together: a product comparison text (choosing a phone, a laptop, or a car), where the writer systematically compares options and concludes with a superlative recommendation.; Cultural note: Ukrainian proverbs and folk sayings using comparisons: 'Що темніша ніч, то ясніші зорі.' (Заболотний Grade 6 p.138) 'Свій хліб більш ситний.' 'З усіх скарбів найбільш цінним є молодість.' Discussion of comparison as a rhetorical device in Ukrainian culture.; Self-check questions: 1. Утворіть найвищий ступінь: тихий, широкий, близький, дешевий. 2. Чим відрізняється щонайкращий від найкращий? 3. Чому *самий кращий — помилка?
- **Типові помилки з найвищим ступенем**: Systematic error catalogue from Авраменко Grade 11 p.29 and Заболотний Grade 8 p.21: 1. Russicism *самий кращий (→ найкращий). 2. Double marking *найбільш найкращий (→ найкращий OR найбільш добрий). 3. Wrong base *найдорогіший (→ найдорожчий — must apply alternation first, then най-). 4. Hyphenation *най-кращий (→ найкращий — prefix written together).; Diagnostic drill: 8-10 sentences each containing one error from the catalogue. Learners identify, classify (Russicism / double marking / wrong base / spelling), and correct each error.
- **Підсумок**: Complete comparison chain diagram: початкова форма → вищий ступінь → найвищий ступінь. проста: гарний → гарніший → найгарніший. складена: — → більш гарний → найбільш гарний. підсилена: — → — → щонайгарніший / якнайгарніший.; Summary table consolidating all formation patterns: проста (най- + comparative), складена (найбільш/найменш + base), підсилена (що-/як- + superlative), синонімічні (дуже, вельми, пре-, повтор). Include register guidance for each.; Preview: next module covers суплетивні (irregular) comparison forms where the comparative comes from a completely different root (гарний-кращий, поганий-гірший).
