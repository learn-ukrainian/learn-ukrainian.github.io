## Grammar Rules
- **Degrees of Comparison**: Правопис (confirmed via textbooks) — Вищий ступінь (проста: -ш-, -іш-; складена: більш/менш + adj). Найвищий ступінь (проста: най- + вищий ступінь; складена: найбільш/найменш + adj). Alternations: [г, ж, з] + -ш- → -жч-; [к, ч, ц] + -ш- → -чч- (often drop [к]); [с, ш] + -ш- → -щ-.
- **Word Formation (-нн-)**: Правопис §33 — Double -нн- occurs when the base ends in -н and the suffix -н- is added (основа на -н- + суфікс -н- = -нн-), e.g., туман + н = туманний.
- **Suffixal changes**: Правопис §19 — к, ц, ч + -ськ- → -цьк-; г, ж, з + -ськ- → -зьк-; х, ш, с + -ськ- → -ськ-.

## Calque Warnings
- **самий кращий**: Calque (Russicism) — Correct form: **найкращий**. Using "самий" for the superlative degree is a gross error in Ukrainian.
- **приймати участь**: Calque (Russicism) — Correct form: **брати участь**. (Common B1 level error).
- **вірний**: Often used incorrectly for "правильний" (correct answer) or "певний" (certain). In B1, should be restricted to "loyal/faithful."
