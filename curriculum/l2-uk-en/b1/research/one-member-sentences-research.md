## State Standard Reference

**Actual reference (B2):** §4.3.3 — "Просте односкладне речення" (lines 3409–3418):
> «4.3.3. Просте односкладне речення:
> 4.3.3.1. Односкладні речення з головним членом – дієсловом:
> означено-особові речення: Іду повз пам'ятник Богданові Хмельницькому;
> неозначено-особові речення: По радіо повідомили про погіршення погоди;
> безособові речення: Було приємно дивитися на веселих дітей. Там усе правильно записано.
> 4.3.3.2. Односкладні речення з головним членом – іменником (номінативні речення): Вечір. Омріяна прохолода. Спокій…»

**Reference discrepancy alert:** The plan cites "§4.3.2" as a B1 State Standard reference for односкладні речення. This is incorrect — §4.3.2 at B1 is "Adverb comparison" (word formation). The actual State Standard treatment of односкладні речення is at **B2 §4.3.3** (lines 3409–3418). At B1 level, односкладні appear only *implicitly* in the declarative sentence examples (§4.4.1.1, line 2345: "Люблю ранкову тишу"). This module teaches B2-codified grammar content at B1 level — which is pedagogically valid as preview/enrichment, but content writers should cite **B2 §4.3.3** or state "prepares for B2 §4.3.3" rather than claiming a nonexistent B1 reference.

**Alignment:** The module fully addresses all four constructions listed in §4.3.3.1–4.3.3.2. Content writers should note that the State Standard's own examples (Іду повз пам'ятник, По радіо повідомили, Вечір. Омріяна прохолода.) are excellent anchor points.

## ⚠️ Critical Plan Gap: Missing 5th Type

Ukrainian textbooks (Авраменко Grade 8, 2025; Заболотний Grade 8, 2025) teach **5 types**, not 4. The plan omits:

**Узагальнено-особові речення (Generalized-Personal):** verb in 2nd person singular or 3rd person plural, referring to anyone/everyone — used in proverbs and general truths.
- Examples: *Бережи честь змолоду.* / *Лежачого не б'ють.* / *Що посієш, те й пожнеш.*
- Avramenko's classification table explicitly lists it as a distinct type between неозначено-особові and безособові.

**Recommendation for content writers:** Briefly acknowledge узагальнено-особові as a recognized 5th type (common in proverbs/sayings), note that this curriculum groups them with означено-особові for pedagogical simplicity at B1, and provide 2–3 proverb examples to satisfy learner curiosity. Do NOT ignore it entirely — Ukrainian grade 8 students learn all 5, so adult learners will encounter this classification in references.

## Cultural Hooks

1. **Ukrainian impersonal expression as cultural value:** Ukrainian extensively uses безособові конструкції to describe states and experiences without naming an agent — *Мені сумно* (lit. "To me it is sad") rather than an explicit "I am sad." This reflects a language pattern where the experiencer is marked by dative case rather than nominative, distributing agency differently than English. This is an authentic feature of Ukrainian syntax documented in Авраменко (2025) §44 and distinguishes Ukrainian from English's "I"-centering structure.

2. **Naming-free agency in news and public discourse:** Неозначено-особові речення (*По радіо повідомили*, *Стало відомо*) are characteristic of Ukrainian journalism and official communication — the State Standard's own example (§4.3.3.1) uses a news context. This connects grammar to real Ukrainian media consumption. Learners who watch Ukrainian news will encounter this pattern constantly.

3. **Називні речення in contemporary Ukrainian writing:** The State Standard example "Вечір. Омріяна прохолода. Спокій…" reflects a literary tradition of nominal sentences in Ukrainian prose and poetry (Сосюра: "Ніч. Тиша. Біле, біле поле"). These also appear in social media — Заболотний (2025) uses Maks Barskikh's Instagram posts as authentic material for односкладні rechenia exercises, making this directly relevant to contemporary digital Ukrainian.

## Common Learner Errors

1. **Inserting "воно" into безособові sentences** → *\*Воно холодно.* → Correct: *Холодно.* — Happens because English requires "It is cold" and learners transfer the dummy subject. Also appears as *\*Воно треба* (from Russian *Надо*) → Correct: *Треба.*

2. **Using nominative pronoun with безособові dative** → *\*Я сумно.* → Correct: *Мені сумно.* — Learners know the experience is theirs but place it in nominative (English "I feel sad") rather than dative (Ukrainian "To me it is sad"). This is one of the most persistent errors at B1.

3. **Confusing означено-особові with неозначено-особові** → marking "Кажуть, що..." as означено-особове because the verb form feels definite. Diagnostic: apply the substitution test (can you add ти/ви? → означено; can you only add вони? → неозначено).

4. **Misclassifying -но/-то passives as двоскладні** → *Написано листа* treated as having a hidden subject. Correct classification: безособове (головний член — дієслово на -но/-то, no subject recoverable).

## Notes for Content Writing

- **Cite §4.3.3 (B2)** rather than the plan's §4.3.2 (which is incorrect), or frame as "prepares learners for B2 State Standard §4.3.3"
- **Include узагальнено-особові** as a brief acknowledgment (2–3 proverb examples) even though the plan's outline lists only 4 types — Grade 8 textbooks teach 5 and learners will encounter this in references
- **Lexical sandbox** will likely reject `означено-особовий` and `неозначено-особовий` as not in VESUM — these should be treated as terminological metalanguage and handled accordingly
- **Register chart** would be highly effective: which sentence types dominate in which registers (news → неозначено-особові; poetry/prose → називні; conversation → означено-особові; technical/weather → безособові)
- **Strong two-member ↔ one-member contrastive pairs** (from Заболотний): *Я хочу спати → Мені хочеться спати*, *Сніг замів → Снігом замело* — use these as the backbone of the practice section
- The State Standard's own B2 examples are excellent anchors: *Іду повз пам'ятник Богданові Хмельницькому* (означено-особове), *По радіо повідомили* (неозначено-особове), *Вечір. Омріяна прохолода. Спокій…* (номінативне)
