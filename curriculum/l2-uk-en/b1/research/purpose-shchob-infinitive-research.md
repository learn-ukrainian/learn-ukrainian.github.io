# Дослідження: Підрядні речення мети: щоб + інфінітив

## State Standard Reference
§4.4.3: "складнопідрядні речення з підрядною частиною мети (сполучник щоб): Ми ходимо в музеї Львова, щоб краще знати історію цього старовинного міста"
Alignment: The module directly addresses this B1 requirement by teaching the formation of complex sentences with purpose clauses using "щоб" and the infinitive, specifically focusing on same-subject structures as demonstrated in the standard's example.

## Cultural Hooks
1. **Lesia Ukrainka's Poetry:** The poem "Як дитиною, бувало..." (1897) features the iconic line «Щоб не плакать, я сміялась» and its reversal «Плачу я, щоб не сміятись». This is a perfect, authentic example of negative purpose and infinitive usage.
2. **Folk Wisdom:** The proverb «Щоб рибу їсти, треба у воду лізти» perfectly encapsulates the grammar structure (щоб + infinitive, main clause + infinitive) while teaching Ukrainian work ethic.

## Common Learner Errors
1. **Over-conjugation (Same Subject):** Learners often say *«Я йду, щоб я купив»* instead of the correct *«Я йду, щоб купити»*. If the subject is the same in both clauses, the infinitive MUST be used.
2. **Subject Mismatch with Infinitive:** Saying *«Я хочу, щоб ти читати»* instead of *«щоб ти читав»*. The infinitive is strictly for same-subject purpose.
3. **Lexical Interference (Russicism):** Using the literal translation *«переслідувати мету»* (from Russian преследовать цель) instead of the authentic Ukrainian *«мати на меті»* or *«ставити за мету»*.

## Notes for Content Writing
- **Decolonized Framing:** Focus on the concise elegance of the Ukrainian infinitive purpose clause. Emphasize authentic verbs like *мати на меті* over colonial calques. 
- **Immersion:** 100% Ukrainian prose. Zero English explanations.
- **Pedagogy (TTT):** Start with a diagnostic, explain the same-subject rule clearly with a Mermaid logic flow, and use rich tables for the synonyms (щоб/аби/для того щоб).
- **Phonetics:** Ensure any phonetic representation uses IPA (e.g., [ʃt͡ʃɔb]), no Latin transliteration.
