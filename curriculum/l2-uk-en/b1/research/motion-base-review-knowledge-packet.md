# Граматика B1: Рух: іти, їхати, бігти



## Як це пояснюють у школі (How Schools Teach This)

Українська педагогічна традиція вводить дієслова руху через систему протиставлень, що допомагає учням зрозуміти їхню функціональність. Основні дихотомії:

1.  **Напрямок руху:** Це ключовий аспект. Дієслова поділяються на ті, що позначають **односпрямований** рух (в один бік, в конкретний момент) і **багатоспрямований** або **регулярний** рух (в обидва боки, повторювана дія, звичка) (Source 2, 7).
    *   Односпрямовані: `іти`, `їхати`, `бігти`.
    *   Багатоспрямовані: `ходити`, `їздити`, `бігати`.

2.  **Спосіб пересування:** Проводиться чітке розрізнення між рухом **пішки** та рухом **транспортом**.
    *   Пішки: `іти` / `ходити`, `бігти` / `бігати`.
    *   Транспортом: `їхати` / `їздити` (Source 2, 11). Це стосується будь-якого транспорту: машина, автобус, поїзд, велосипед (Source 2, 16).

Шкільні підручники для 6-7 класів вводять правила дієвідмінювання та чергування приголосних, які є характерними для цих дієслів (Source 25, 39). Вправи зазвичай фокусуються на виборі правильного дієслова залежно від контексту та на правильному утворенні особових форм (Source 10). На наступному етапі вводяться префікси (`по-`, `при-`, `ви-` тощо), які додають дієсловам руху нових значень (відправлення, прибуття, вихід) і є основою для творення доконаного виду (Source 3, 8).

## Повна парадигма (Full Paradigm)

Дієслова руху в українській мові належать до І та ІІ дієвідмін і часто мають чергування приголосних в основі.

### 1. Рух пішки: іти / ходити

| Форма | Односпрямований (Unidirectional) | Багатоспрямований (Multidirectional) |
| :--- | :--- | :--- |
| **Інфінітив** | **іти́** (І дієвідміна) | **ходи́ти** (ІІ дієвідміна) |
| **Теперішній час** | | |
| я | іду́ | **ходжу́** (`д` → `дж`) |
| ти | іде́ш | хо́диш |
| він / вона / воно | іде́ | хо́дить |
| ми | ідемо́ | хо́димо |
| ви | ідете́ | хо́дите |
| вони | іду́ть | хо́дять |
| **Минулий час** | **йшов**, **йшла**, йшло, йшли | ходи́в, ходи́ла, ходи́ло, ходи́ли |
| **Майбутній час (НДВ)** | буду йти / йти́му | буду ходи́ти / ходи́тиму |
| **Майбутній час (ДВ)** | **піду́**, пі́деш, пі́де... (префікс `по-` або `при-` та ін.) | (утворюється від іншого дієслова, напр. `сходити`) |
| **Наказовий спосіб** | іди́, іді́мо, іді́ть | ходи́, ході́мо, ході́ть |

*Примітка:* Чергування `д` → `дж` у дієслові `ходити` відбувається тільки в 1-й особі однини (Source 39). Минула форма `йшов` є суплетивною.

### 2. Рух транспортом: їхати / їздити

| Форма | Односпрямований (Unidirectional) | Багатоспрямований (Multidirectional) |
| :--- | :--- | :--- |
| **Інфінітив** | **ї́хати** (І дієвідміна) | **ї́здити** (ІІ дієвідміна) |
| **Теперішній час** | | |
| я | ї́ду | **ї́жджу** (`зд` → `ждж`) |
| ти | ї́деш | ї́здиш |
| він / вона / воно | ї́де | ї́здить |
| ми | ї́демо | ї́здимо |
| ви | ї́дете | ї́здять |
| вони | ї́дуть | ї́здять |
| **Минулий час** | ї́хав, ї́хала, ї́хало, ї́хали | ї́здив, ї́здила, ї́здило, ї́здили |
| **Майбутній час (НДВ)** | буду ї́хати / ї́хатиму | буду ї́здити / ї́здитиму |
| **Майбутній час (ДВ)** | **пої́ду**, пої́деш, пої́де... (префікс `по-`, `при-` та ін.) | (утворюється від іншого дієслова, напр. `з'їздити`) |
| **Наказовий спосіб** | їдь, ї́дьмо, ї́дьте | їздь, ї́здьмо, ї́здьте |

*Примітка:* Чергування `зд` → `ждж` у дієслові `їздити` відбувається тільки в 1-й особі однини (Source 26, 33).

### 3. Рух бігом: бігти / бігати

| Форма | Односпрямований (Unidirectional) | Багатоспрямований (Multidirectional) |
| :--- | :--- | :--- |
| **Інфінітив** | **бі́гти** (І дієвідміна, змішаний тип) | **бі́гати** (І дієвідміна) |
| **Теперішній час** | | |
| я | **біжу́** | бі́гаю |
| ти | **біжи́ш** | бі́гаєш |
| він / вона / воно | **біжи́ть** | бі́гає |
| ми | **біжимо́** | бі́гаємо |
| ви | **біжите́** | бі́гаєте |
| вони | **біжа́ть** | бі́гають |
| **Минулий час** | **біг**, **бі́гла**, бі́гло, бі́гли | бі́гав, бі́гала, бі́гало, бі́гали |
| **Майбутній час (НДВ)** | буду бі́гти / бі́гтиму | буду бі́гати / бі́гатиму |
| **Майбутній час (ДВ)** | **побіжу́**, побіжи́ш... (префікс `по-`, `при-` та ін.) | (утворюється від іншого дієслова, напр. `збігати`) |
| **Наказовий спосіб** | біжи́, біжі́мо, біжі́ть | бі́гай, бі́гаймо, бі́гайте |

*Примітка:* Дієслово `бігти` є складним. Хоча воно належить до І дієвідміни, в особових формах відбувається чергування `г` → `ж` (Source 25), а закінчення схожі на ІІ дієвідміну (`-иш`, `-ить`, `-ать`). У минулому часі чоловічого роду відсутній суфікс `-в` (Source 9).

## Частотність і пріоритети

Для рівня B1 пріоритети вивчення наступні:

1.  **Основна концепція:** Найважливіше — засвоїти функціональну різницю між односпрямованими (`іти`, `їхати`) та багатоспрямованими (`ходити`, `їздити`) дієсловами. Це фундаментальна концепція, без якої неможливе природне мовлення.
2.  **Форми теперішнього часу:** Усі шість дієслів є високочастотними в теперішньому часі. Особливу увагу слід приділити формам 1-ї особи однини (`ходжу`, `їжджу`, `біжу`) через їхні чергування.
3.  **Минулий час:** Форми минулого часу є обов'язковими для опису минулих подій. Форми `йшов`/`йшла` та `біг`/`бігла` потребують окремого запам'ятовування.
4.  **Простий майбутній час (доконаний):** Найбільш вживаними є форми з префіксами `по-` (початок/намір руху: `піду`, `поїду`) та `при-` (прибуття: `прийду`, `приїду`) (Source 8). Вони є ключовими для вираження планів.
5.  **Складні форми майбутнього (недоконаний):** Форми `буду ходити` або `ходитиму` менш пріоритетні для активного вжитку на початковому етапі B1, але важливі для розуміння тривалих або повторюваних дій у майбутньому.

## Типові помилки L2 (Common L2 Errors)

Англомовні студенти часто роблять помилки через те, що в англійській мові одне дієслово "to go" охоплює багато значень, які в українській розрізняються.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| `Я іду в спортзал щовівторка.` | `Я ходжу в спортзал щовівторка.` | Для регулярної, повторюваної дії використовується багатоспрямоване дієслово `ходити`, а не `іти`, що позначає конкретний рух в один бік зараз (Source 7). |
| `Ми ідемо в Карпати на машині.` | `Ми їдемо в Карпати на машині.` | Якщо рух відбувається за допомогою будь-якого транспорту, обов'язково вживаються дієслова `їхати` або `їздити` (Source 2). |
| `Вчора він бігв марафон.` | `Вчора він біг марафон.` | У дієсловах з основою інфінітива на приголосний (`біг-ти`, `нес-ти`) суфікс `-в` у формі минулого часу чоловічого роду випадає (Source 9). |
| `Я ходю на роботу пішки.` | `Я ходжу на роботу пішки.` | В 1-й особі однини дієслів ІІ дієвідміни відбувається чергування приголосних: `д` → `дж` (`ходити` → `ходжу`), `зд` → `ждж` (`їздити` → `їжджу`) (Source 25, 26). |
| `Завтра я буду йти в кіно.` | `Завтра я піду в кіно.` | Хоча граматично форма `буду йти` правильна, для вираження одноразової запланованої дії в майбутньому значно природніше і частіше вживається доконана форма `піду` (Source 8). |

## Деколонізаційні застереження (Decolonization Notes)

При викладанні українських дієслів руху важливо уникати порівнянь з російською мовою як з нормою. Українська мова має власну, самодостатню систему.

1.  **Самостійність системи:** Система протиставлення односпрямованих/багатоспрямованих дієслів є спільною для багатьох слов'янських мов, але реалізація в кожній мові унікальна. Форми, наголоси та чергування в українських дієсловах (`їжджу`, `біжиш`) слід вивчати як норму української мови, а не як "відмінність від російської".
2.  **Наголос:** Український наголос є рухомим і часто не збігається з російським. Наприклад, у формах `несемо́`, `ведемо́`, `ідемо́` наголос падає на закінчення (Source 17). Це потрібно вивчати як самостійне правило.
3.  **Лексика:** Слід послідовно використовувати питомо українську лексику. Наприклад, `квиток` на поїзд (Source 8), а не `білет`.
4.  **Фразеологія:** Використовуйте автентичні українські фразеологізми, що пов'язані з рухом, наприклад: `з усіх ніг` (дуже швидко бігти), `куди вода несе` (пасивно слідувати обставинам), `іти своєю дорогою` (Source 13).
5.  **Дієслово `бігти`:** Це дієслово має змішаний тип дієвідмінювання в українській мові. Це унікальна риса, а не "схожість" чи "відмінність" від російського аналога `бежать`, який також має свої особливості. Пояснювати його слід через внутрішню логіку української граматики.

## Природні приклади (Natural Examples)

Приклади згруповано за семантичною парою для демонстрації контрасту.

#### `іти` / `ходити` (пішки)

1.  **іти** (зараз, в один бік): `«Навесні в гайок піду, першу квіточку знайду»` (Source 21). (про заплановану майбутню дію)
2.  **іти** (зараз, в один бік): `Я зараз іду в магазин, тобі щось купити?` <!-- VERIFY -->
3.  **ходити** (регулярно): `«Інколи я ходжу в кіно чи в ресторан»` (Source 7).
4.  **ходити** (регулярно): `Діти ходять до школи п'ять днів на тиждень.` <!-- VERIFY -->

#### `їхати` / `їздити` (транспортом)

5.  **їхати** (зараз, в один бік): `«Цього року ми з дружиною їдемо в Європу на Різдво»` (Source 2).
6.  **їхати** (послідовність поїздки): `«Спочатку ми їдемо в Угорщину в Будапешт, тоді в Словаччину»` (Source 2).
7.  **їздити** (регулярно, звичка): `«Оксано, Ви часто їздите у відпустку?»` (Source 2).
8.  **їздити** (регулярно, звичка): `«Зазвичай я їжджу на дачу влітку»` (Source 2).

#### `бігти` / `бігати` (бігом)

9.  **бігти** (зараз, в один бік): `«Хмарки біжать по небу вдалечінь»` (Source 34).
10. **бігти** (опис процесу): `«На першій доріжці біжить чемпіонка з легкої атлетики»` (Source 25).
11. **бігати** (регулярно, хобі): `«Якщо я вдома, то я бігаю в парку чи займаюся спортом»` (Source 7).
12. **бігати** (як діяльність): `«Я дуже люблю займатись спортом, наприклад... бігати»` (Source 16).

## Рекомендації для вправ (Activity Concepts)

Для ефективного засвоєння матеріалу рекомендується триетапний підхід:

*   **Фаза 1: Розпізнавання та вибір.**
    *   **Вправа "або/або":** Студенти обирають правильне дієслово з пари.
        *   _Приклад:_ `Минулого літа ми (їхали / їздили) в Карпати тричі.`
        *   _Приклад:_ `Дивись, до нас (іде / ходить) бабуся!`
    *   **Вправа "Пішки чи транспортом?":** Студенти заповнюють пропуски дієсловами `іти/ходити` або `їхати/їздити` залежно від контексту (`...на автобусі`, `...через парк`).

*   **Фаза 2: Відтворення форм.**
    *   **Дієвідмінювання в таблиці:** Заповнення таблиць дієвідмінювання для всіх шести дієслів у теперішньому часі.
    *   **Трансформація речень:** Переписати речення, змінивши особу або час.
        *   _Приклад:_ `Я ходжу на ринок. → Ми ... . Вони ... .`
        *   _Приклад:_ `Він біжить. → (Минулий час) Він ... . Вона ... .`

*   **Фаза 3: Вільне мовлення та письмо.**
    *   **Опис тижня:** Студенти описують свій типовий тиждень, використовуючи `ходити`, `їздити`, `бігати` для регулярних дій.
    *   **Планування подорожі:** Студенти складають план уявної подорожі, використовуючи доконані форми майбутнього часу (`поїдемо`, `підемо`, `прилетимо`).
        *   _Приклад:_ `«На канікулах ми поїдемо до Львова. Спочатку ми підемо на площу Ринок...»` (за зразком Source 8).

## Зв'язки з іншими темами

*   **Дієвідміни (Conjugations):** Дієслова руху є яскравими представниками І (`іти`, `їхати`, `бігати`) та ІІ (`ходити`, `їздити`) дієвідмін. `Бігти` — приклад змішаного типу, що є корисним для ілюстрації складності системи.
*   **Аспект дієслова (Verb Aspect):** Пари односпрямованих/багатоспрямованих дієслів є фундаментом для розуміння недоконаного виду. Додавання префіксів (`по-`, `при-`) до односпрямованих дієслів (`по+іти` → `піти`) є основним способом творення доконаного виду.
*   **Майбутній час (Future Tense):** Ці дієслова демонструють усі три форми майбутнього часу: просту (`піду`, `поїду`), складну (`йтиму`, `їхатиму`) і складену (`буду йти`, `буду їхати`).
*   **Відмінки (Cases):** Дієслова руху керують іменниками в різних відмінках для позначення напрямку (`в/у` + Знахідний), мети (`по` + Знахідний), засобу (`...автобусом` — Орудний).

# Граматика B1: Контрольна робота 3



## Як це пояснюють у школі (How Schools Teach This)

На рівні B1 учні вже знайомі з основами словотвору, які вивчаються в середній школі. Концепція **префікса** як значущої частини слова, що стоїть перед коренем і слугує для утворення нових слів, вводиться ще в 3-4 класах (Джерело: 3-klas-ukrainska-mova-kravtsova-2020-1_s0038, 3-klas-ukrainska-mova-vashulenko-2020-1_s0082). У 5-му класі ця тема поглиблюється, і учні вчаться розрізняти префікси за значенням, яке вони надають слову, наприклад, просторове значення (Джерело: 5-klas-ukrmova-zabolotnyi-2023_s0056).

Ключові орфографічні правила, що є фундаментом для цієї теми, також закладаються в середній школі:
1.  **Правопис префіксів `з-` ( `с-` )**: Правило "Кафе «Птах»" (пишемо `с-` перед `к, п, т, ф, х`) є одним з основних орфографічних правил (Джерело: 3-klas-ukrainska-mova-kravtsova-2020-1_s0045, 5-klas-ukrmova-zabolotnyi-2023_s0107).
2.  **Правопис префіксів `пре-`, `при-`, `прі-`**: Учні вчаться розрізняти `пре-` (вища міра ознаки, = *дуже*), `при-` (наближення, приєднання, неповнота дії) та `прі-` (у трьох словах: `прізвище, прізвисько, прірва`) (Джерело: 10-klas-ukrmova-zabolotnyi-2018_s0154).
3.  **Розрізнення префіксів і прийменників**: Шкільні підручники наголошують, що прийменник — це окреме слово, між яким і наступним словом можна вставити інше, тоді як префікс є частиною слова і пишеться разом (Джерело: 4-klas-ukrayinska-mova-ponomarova-2021-1_s0025).

На рівні B1 фокус зміщується від простого розпізнавання префіксів до їх системного застосування, зокрема з **дієсловами руху**. Це найскладніший аспект теми. Учні переходять від базових пар (напр. `іти`/`ходити`) до розуміння того, як префікси не просто створюють нове слово, а й **завершують дію (перфективізують дієслово)** і надають їй конкретного напрямку (прибуття, відбуття, перетин тощо). Це пояснюється на прикладах подорожей та планів (Джерело: ext-ulp_youtube-206, ext-ulp_youtube-244).

## Повна парадигма (Full Paradigm)

Центральним елементом цієї теми є взаємодія дієслів руху з префіксами.

### 1. Базові пари дієслів руху (без префіксів)

Ці пари розрізняються за ознакою односпрямованості (unidirectional) та різноспрямованості/повторюваності (multidirectional/habitual). У теперішньому часі вони позначають різні типи дії.

| Тип | Односпрямовані (`-іти`, `-їхати`) | Різноспрямовані (`-ходити`, `-їздити`) |
| :-- | :--- | :--- |
| **Значення** | Рух в одному напрямку, відбувається зараз | Рух у різних напрямках, повторювана, регулярна дія, або здатність до руху |
| **Пішки** | я **іду**, ти **ідеш**, він/вона/воно **іде** | я **ходжу**, ти **ходиш**, він/вона/воно **ходить** |
| **Транспортом** | я **їду**, ти **їдеш**, він/вона/воно **їде** | я **їжджу**, ти **їздиш**, він/вона/воно **їздить** |
| **Летячи** | я **лечу**, ти **летиш**, він/вона/воно **летить** | я **літаю**, ти **літаєш**, він/вона/воно **літає** |
| **Бігом** | я **біжу**, ти **біжиш**, він/вона/воно **біжить** | я **бігаю**, ти **бігаєш**, він/вона/воно **бігає** |

*(Джерело: ext-ulp_youtube-244, ext-ulp_youtube-243)*

### 2. Роль префіксів у творенні доконаних дієслів руху

Додавання будь-якого префікса до дієслова руху недоконаного виду (як односпрямованого, так і різноспрямованого) утворює **нове дієслово доконаного виду**. Це нове дієслово має своє специфічне значення, надане префіксом.

| Префікс | Значення | Приклад (з `їхати`/`іти`) | Приклад (з `летіти`/`бігти`) |
|:---|:---|:---|:---|
| **по-** | Початок руху, відправлення | `поїхати`, `піти` | `полетіти`, `побігти` |
| **при-** | Прибуття, наближення | `приїхати`, `прийти` | `прилетіти`, `прибігти` |
| **ви-** | Рух назовні, вихід | `виїхати`, `вийти` | `вилетіти`, `вибігти` |
| **в- (у-)** | Рух всередину, вхід | `в'їхати`, `ввійти` | `влетіти`, `вбігти` |
| **від- (віді-)** | Рух на відстань від чогось | `від'їхати`, `відійти` | `відлетіти`, `відбігти` |
| **до-** | Рух до кінцевої точки | `доїхати`, `дійти` | `долетіти`, `добігти` |
| **з- (зі-)** | Рух згори вниз; рух звідкись | `з'їхати`, `зійти` | `злетіти`, `збігти` |
| **за-** | Рух за щось; короткий візит | `заїхати`, `зайти` | `залетіти`, `забігти` |
| **пере-** | Рух через щось; з одного місця в інше | `переїхати`, `перейти` | `перелетіти`, `перебігти` |
| **об-** | Рух навколо чогось | `об'їхати`, `обійти` | `облетіти`, `оббігти` |
| **про-** | Рух повз щось або крізь щось | `проїхати`, `пройти` | `пролетіти`, `пробігти` |
| **роз- (розі-)** | Рух у різні боки | `роз'їхатися`, `розійтися` | `розлетітися`, `розбігтися` |

*(Джерела: ext-ulp_youtube-206, 5-klas-ukrmova-zabolotnyi-2023_s0056, 6-klas-ukrmova-litvinova-2023_s0052, ext-other_blogs-26)*

### 3. Час і вид у дієсловах руху

Префіксація кардинально змінює використання дієслова в часових формах.

| Час | Недоконаний вид (процес, повторюваність) | Доконаний вид (результат, одноразова дія) |
|:---|:---|:---|
| **Теперішній** | `Я **їду** в Київ.` (зараз, в процесі) <br> `Я **їжджу** в Київ.` (регулярно) | *(Не існує)* |
| **Минулий** | `Я довго **їхав** у Київ.` (опис процесу) <br> `Я раніше **їздив** у Київ.` (повторювана дія в минулому) | `Я **приїхав** у Київ.` (факт прибуття, результат) <br> `Я **поїхав** з Києва.` (факт відправлення) |
| **Майбутній** | `Я **буду їхати** в Київ цілий день.` (складена форма: процес у майбутньому) | `Я **приїду** в Київ завтра.` (проста форма: результат у майбутньому) <br> `Я **поїду** з Києва завтра.` |

*(Джерело: ext-ulp_youtube-206, ext-ulp_youtube-244)*

## Частотність і пріоритети

Для рівня B1 найважливішим є впевнене володіння базовими парами та найуживанішими префіксами.

1.  **Найвищий пріоритет:** Впевнено розрізняти **`іти/ходити`** та **`їхати/їздити`**. Це основа, без якої подальше вивчення неможливе.
2.  **Частотні префікси:**
    *   `по-` (для вираження планів на майбутнє: `поїду, піду`)
    *   `при-` (для позначення прибуття: `приїхав, прилетить`)
    *   `ви-` / `в- (у-)` (для позначення виходу/входу: `вийшов з кімнати`, `зайшов у кафе`)
    *   `до-` (для позначення досягнення мети: `добіг до фінішу`, `доїхали до готелю`)
3.  **Майбутній час:** Конструкція **`по-` + дієслово** (`поїду, піду, полечу`) є надзвичайно частотною для вираження майбутніх планів (Джерело: ext-ulp_youtube-206). Її слід засвоїти в першу чергу.
4.  **Орфографія:** Правила на `пре-/при-` та `з-/с-` є базовими і мають бути доведені до автоматизму.

## Типові помилки L2 (Common L2 Errors)

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| Я **їду** на роботу кожен день. | Я **їжджу** на роботу кожен день. | Для регулярної, повторюваної дії використовується різноспрямоване дієслово (`їздити`, `ходити`). `Їду` означає рух в одному напрямку просто зараз. (Джерело: ext-ulp_youtube-244) |
| Завтра ми **будемо їхати** в Карпати. (коли йдеться про факт поїздки, а не процес) | Завтра ми **поїдемо** в Карпати. | Для позначення одноразової завершеної дії в майбутньому (результату) використовується проста форма доконаного виду (`поїдемо`), а не складена недоконаного. (Джерело: ext-ulp_youtube-206) |
| Коли я **поїхав** додому, я тобі подзвонив. (значення: "коли я прибув додому") | Коли я **приїхав** додому, я тобі подзвонив. | Плутанина префіксів. `Поїхати` означає початок руху, від'їзд, а `приїхати` — прибуття в кінцеву точку. |
| Він **в їхав** у гараж. | Він **в'їхав** у гараж. | Плутанина прийменника `в` та префікса `в-`. Префікс є частиною слова. Після префіксів, що закінчуються на приголосний, перед `я, ю, є, ї` ставиться апостроф. (Джерело: 4-klas-ukrayinska-mova-kravtsova-2021-1_s0025, 4-klas-ukrayinska-mova-ponomarova-2021-1_s0025) |
| Я хочу **спитати** тебе. (написання `зпитати`) | Я хочу **спитати** тебе. | Орфографічна помилка. Перед `к, п, т, ф, х` завжди пишеться префікс `с-`. (Джерело: 3-klas-ukrainska-mova-kravtsova-2020-1_s0045) |
| Це було **пре-** цікаво. | Це було **дуже** цікаво. / Це було **прецікаво**. | `Пре-` є префіксом і пишеться разом зі словом, а не як окреме слово. Він синонімічний до `дуже`. (Джерело: 10-klas-ukrmova-zabolotnyi-2018_s0154) |

## Деколонізаційні застереження (Decolonization Notes)

1.  **Уникати порівнянь з російською:** Хоча російська мова має схожу систему дієслів руху (`ехать/ездить`, `идти/ходить`), пояснювати українську граматику слід через її власну внутрішню логіку, а не як "варіант" російської.
2.  **Фонетика:** Наголошуйте на чіткій вимові українського звука **[йі]** в `їхати`, `їжджу`, `приїхав`, який не має аналога в російській літературній мові (`ехать`).
3.  **Префікс `по-`:** В українській мові префікс `по-` з дієсловами руху найчастіше означає **початок руху** (`поїхали` - let's go, `побіг` - he started running). Це самостійна риса, а не запозичення.
4.  **Лексика:** Слід використовувати автентичну українську лексику. Наприклад, `відпустка` (vacation), а не `отпуск`; `квиток` (ticket), а не `білет` (хоча `білет` теж вживається, `квиток` є більш питомим для транспорту).
5.  **Прийменник `на` з назвами країн:** В українській мові правильно казати `в Україні`. Вживання `на Україні` є застарілим і асоціюється з російським колоніальним дискурсом, що розглядав Україну як територію, а не суверенну державу (Джерело: 11-klas-ukrajinska-mova-avramenko-2019_s0093).

## Природні приклади (Natural Examples)

**Група 1: Односпрямовані vs. Різноспрямовані**
*   `Зазвичай я **їжджу** на дачу влітку, але цього року ми з дружиною **їдемо** в Європу на Різдво.` (Source 4, ext-ulp_youtube-244)
*   `Оксано, Ви часто **їздите** у відпустку?` (Source 4, ext-ulp_youtube-244)

**Група 2: Доконаний майбутній час для планів**

*   `А потім **поїдемо** в Карпати, **підемо** в невеличкий похід.` (Source 2, ext-ulp_youtube-206)

**Група 3: Префікси прибуття та відбуття**
*   `Коли твоя мама **прилітає**? – **Прилітає** 28 грудня, а назад **полетить** 8 січня.` (Source 2, ext-ulp_youtube-206)
*   `Мій тато дуже швидко **повернувся** назад до України і з тих пір ми не бачилися.` (Source 1, ext-ulp_youtube-38)
*   `На другий день я і Софія **сіли** на літак до Варшави, а мій чоловік **поїхав** сам на машині.` (Source 1, ext-ulp_youtube-38)

**Група 4: Орфографія префіксів `з-`/`с-` та `пре-`/`при-`**
*   `Я хочу **сходити** в супермаркет, хочеш зі мною?` (Source 3, ext-ulp_youtube-215)

**Група 5: Вживання з прийменниками та відмінками**
*   `У суботу **підемо в театр на виставу** "Наталка Полтавка".` (Source 14, ext-ulp_youtube-246)
*   `Спочатку ми **їдемо в Угорщину, в Будапешт**.` (Source 4, ext-ulp_youtube-244)
*   `Наша дорога **зі Швеції... до України** тривала чотири дні.` (Source 1, ext-ulp_youtube-38)

## Рекомендації для вправ (Activity Concepts)

**Phase 1: Засвоєння базових пар**
*   **Вправа "Коли?":** Студенти обирають правильне дієслово в залежності від маркера часу.
    *   *Приклад:* `Я (їду / їжджу) до Львова (зараз / щоліта).`
*   **Вправа "Опиши свій день/тиждень":** Студенти описують свої регулярні переміщення, використовуючи `ходжу, їжджу, літаю`.

**Phase 2: Введення префіксів та доконаного виду**
*   **Вправа "Конструктор":** Студенти додають різні префікси (`при-, по-, ви-, за-`) до дієслова `їхати` і пояснюють, як змінилося значення.
    *   *Приклад:* `їхати -> приїхати` (arrival), `їхати -> поїхати` (departure).
*   **Вправа "Що сталося?":** Короткі ситуації, де треба вжити правильне дієслово доконаного виду в минулому часі.
    *   *Приклад:* `Поїзд був у дорозі 5 годин. Зараз він на вокзалі. Поїзд ... (приїхав).`

**Phase 3: Плани на майбутнє**
*   **Вправа "Мої канікули":** Студенти пишуть/розповідають про свої плани на вихідні чи відпустку, використовуючи `поїду, піду, полечу`.
*   **Вправа "Вибір форми":** Студенти обирають між складеною формою недоконаного виду (`буду їхати`) і простою доконаного (`поїду`).
    *   *Приклад:* `Завтра о 10-й я (буду їхати / поїду) в поїзді, тому не зможу говорити.` -> `буду їхати` (процес у конкретний момент). `Завтра я (буду їхати / поїду) до батьків.` -> `поїду` (факт поїздки).

## Зв'язки з іншими темами

*   **Передумови:**
    *   **Відмінки:** Впевнене володіння знахідним (`в/на` + куди?), родовим (`з/від` + звідки?) та орудним (`чим?` - транспортом) відмінками.
    *   **Часи дієслів:** Знання того, як утворюються теперішній, минулий і складений майбутній часи.
    *   **Поняття виду:** Базове розуміння різниці між доконаним (результат) і недоконаним (процес) видом.
*   **Наступні теми:**
    *   **Дієприкметники та дієприслівники:** Утворення активних та пасивних дієприкметників (`приїхавший`, `привезений`) та дієприслівників (`приїхавши`) базується на розумінні виду та основи дієслова.
    *   **Складні речення:** Вміння правильно вживати вид і час дієслів руху є ключовим для побудови складних речень з послідовними або одночасними діями (`Коли я приїду, я тобі зателефоную`).
