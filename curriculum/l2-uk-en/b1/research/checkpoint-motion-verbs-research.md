# Дослідження: Контрольна точка: Дієслова руху

## State Standard Reference
§4.3.8: "Утворення префіксальних дієслів руху: іти – прийти, перейти, дійти, відійти, зайти, вийти; їхати – приїхати, переїхати, доїхати, від’їхати, заїхати, поїхати, виїхати, об’їхати."
Alignment: This module directly targets the formation and usage of prefixed motion verbs as explicitly listed in the B1 word formation standards. It also reinforces §4.2.2.4 (Accusative case for direction) and §4.2.3.1 (Indicative mood/Aspect).

## Cultural Hooks
1. **Kyiv Metro (Київський метрополітен)**: The Arsenalna station is the deepest in the world (105.5m), requiring a long descent (сходити вниз) which is a perfect context for motion verbs. The "Zoloti Vorota" station is an artistic masterpiece depicting Kyivan Rus' history.
2. **Ukrzaliznytsia (Укрзалізниця)**: The "Iron People" (залізні люди) and the evacuation trains of 2022. The concept of "евуакуаційні рейси" (evacuation trips) heavily uses motion verbs like *виїжджати*, *покидати*, *рятуватися*.
3. **Idioms**:
    - *Час іде / біжить / летить* (Time goes / runs / flies).
    - *Справа йде на лад* (Things are getting better).
    - *Далеко піти* (To achieve great success).

## Common Learner Errors
1. **Unidirectional vs. Multidirectional**: Using *йти* for habitual actions (e.g., *Я йду в офіс щодня* instead of *Я ходжу*).
2. **Transport vs. Foot**: Using *йти* when using a vehicle (e.g., *Я йду до Львова поїздом* instead of *Я їду*).
3. **Prefix Confusion (Ви- vs. Від-)**: Confusing *вийти* (exit from inside) with *відійти* (move away from an object).
4. **Case Mismatch**: Using Locative instead of Accusative for destination (e.g., *Я йду в магазині* instead of *Я йду в магазин*).
