## State Standard Reference

**§4.4.2** — Просте ускладнене речення (lines 2367–2378):
> "просте речення з однорідними членами: У конференції беруть участь українські й зарубіжні фахівці з теоретичної фізики. [...] просте речення з дієприкметниковим зворотом: Три години тому ми повернулися з екскурсії, організованої для учасників конференції; просте речення з дієприслівниковим зворотом: Повернувшись з екскурсії, ми смачно пообідали в ресторані; просте речення зі вставними словами мабуть, можливо, здається: Мабуть, ніхто не зможе забути нашу зустріч."

**§4.4.3** — Складне речення (lines 2380–2431):
> Covers: складносурядне (і / а, але / або...або), складнопідрядне зі з'ясувальною (що, хто, де, куди, звідки, щоб), означальною (який у всіх відмінках), часовою (коли — одночасність та різночасовість), причиновою (тому що, бо, через те що), умовною (якщо / якби, коли б), мети (щоб), допустовою (хоч, дарма що, незважаючи на те що) підрядними частинами; пряма і непряма мова.

**Alignment**: This module introduces the syntactic metalanguage (члени речення, граматична основа, типи речень) that the learner needs to work with §4.4.2–§4.4.3 constructions. It is an explicit B1.0 Bridge module — learners cannot parse "просте ускладнене речення" or label clause types without this foundational terminology. Modules b1-30 (підрядні означальні) and b1-38 (допустові речення) both build directly on the §4.4.3 system introduced here.

Also relevant — **§4.4.1** declarative sentences (lines 2344–2352) include the standard examples that anchor subject/predicate identification:
> "Студент усе зрозумів. Люблю ранкову тишу." — Note that "Люблю ранкову тишу" exemplifies pro-drop (null subject) with the predicate encoding the subject, a key error point for English speakers.

## Cultural Hooks

1. **The Underlining Ritual as Pan-Ukrainian Schoolroom Experience**: The visual code (one line for підмет, two for присудок, wavy for означення, dotted for додаток, dash-dot for обставина) is confirmed across multiple generations of Ukrainian school textbooks from Grade 3 onward — Kravtsova (Grade 3, 2020), Zabolotnyi (Grade 5, tier 1), Avramenko (Grade 5, tier 1), Zabolotnyi (Grade 8, tier 1). Every Ukrainian who attended a Ukrainian school recognizes this system immediately. For a B1 learner, framing this as "the shared visual language of Ukrainian classrooms" creates an authentic cultural connection — this is what Ukrainian children do during *розбір* exercises from age 9. Source: Textbook RAG, multiple tier-1 and tier-2 sources confirmed.

2. **Ukrainian Syntax as an Independent Scholarly Tradition**: Wikipedia (uk.wikipedia.org/wiki/Синтаксис_української_мови) describes Ukrainian syntax as a full, independently developed branch of linguistics: "частина граматики української мови, яка вивчає словосполучення та речення, їхню будову, типи та об'єднання в надфразні одиниці, роль і значення частин мови та граматичних форм у реченні в синхронному й діахронному (історичному) вимірах." Ukrainian syntactic science has its own rich scholarly tradition (Олексій Потебня, Олена Курило, Іван Огієнко), developed in parallel with — not derived from — other Slavic traditions. This decolonized framing is essential: Ukrainian sentence analysis is not "like Russian grammar" — it predates Soviet standardization attempts.

## Common Learner Errors

1. **Confusing Part of Speech with Sentence Function** → *"іменник" ≠ "підмет"*: English speakers map "noun = subject" and short-circuit the analysis. In Ukrainian, the predicate can be expressed by an adjective ("Вода холодна" — присудок = adjective), a noun ("Він — лікар" — присудок = noun), or a participle. The predicate-subject relationship is about *function in the sentence*, not category. Textbook evidence: Grade 11 (Voron, tier 2, p. 108) dedicates explicit attention to non-standard predicate forms.

2. **Pro-Drop Panic: "Missing" Subject** → *"Люблю каву" is complete, not a fragment*: English has obligatory subjects; Ukrainian is a null-subject (pro-drop) language. The verb conjugation encodes person and number, making an explicit subject redundant but optional. The State Standard itself uses "Люблю ранкову тишу" as a canonical example (§4.4.1.1, line 2345). English speakers often mark this as "incorrect" or search for a missing subject. Content must address this directly.

3. **Object vs. Adverbial Confusion in Prepositional Phrases** → *"Він поклав книгу на стіл"*: "на стіл" answers "куди?" (обставина місця/напрямку) not "що?" (додаток). The English prepositional phrase "on the table" does not map cleanly to Ukrainian case grammar. The diagnostic question system (кого? чого? → додаток; де? куди? звідки? → обставина) is the primary tool. Textbook evidence: Zabolotnyi Grade 5 tier 1 (p. 168, 174), Grade 8 tier 1 (p. 78–79).

4. **Bonus error — складносурядне vs. складнопідрядне**: English speakers often conflate coordination (і, але) and subordination (що, який, тому що) because English "and/but/because/which" feel similar in use. The distinction between рівноправні частини (координація) and залежна частина (субординація) needs explicit framing with visual clause-diagram support.

## Notes for Content Writing

- **The GRAC asymmetry** (речення = 11.12/million vs. підмет/присудок = 0.35/million) should be used rhetorically in the introduction: these metalanguage terms are rare in the wild *because* native speakers internalized them in school and no longer need to say "підмет" consciously. The learner is accessing the hidden analytical layer of a language they're building.
- **"Додаток" false friend**: the word also means "appendix/supplement" in general Ukrainian. Content should acknowledge this briefly — "додаток у реченні" vs. "додаток до документа."
- **Zabolotnyi Grade 5 and 8 (tier 1)** are the primary textbook sources for sentence part terminology — quotes from these can anchor explanations as "this is how Ukrainian schoolchildren first encounter these terms."
- **State Standard examples are re-usable**: "Студент усе зрозумів," "Люблю ранкову тишу," "Мабуть, ніхто не зможе забути нашу зустріч" — these are canonical State Standard examples that can appear in exercises as authentic reference material.
- **Decolonized framing**: Frame Ukrainian word order flexibility (SVO/OVS/VOS) as a feature of Ukrainian's rich case system — never as "freedom" relative to some other language's constraints. Ukrainian syntax reflects its own internal logic of topic-comment structuring (тема-рема).
- **B1.0 Bridge**: English scaffolding is permitted for abstract concepts (граматична основа, узгодження), but the visual code (underlining) should be introduced entirely in Ukrainian — it's a practical skill, not an abstract concept.
