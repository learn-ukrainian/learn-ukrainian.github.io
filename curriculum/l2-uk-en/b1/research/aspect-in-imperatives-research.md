# Дослідження: Вид у наказовому способі

## State Standard Reference
§4.2.3.2: "Наказовий спосіб. Утворення форм наказового способу: мрій, мріймо; кажіть, скажіть, скажімо; пий, пиймо, пийте; борися, борімося, боріться, хай / нехай бореться, хай / нехай борються."
Alignment: This module directly addresses the B1 requirement by teaching the semantic distinction between Aspect pairs in the imperative (e.g., *кажіть* vs. *скажіть*), moving beyond simple formation to functional usage (process vs. result, politeness vs. command).

## Cultural Hooks
1. **Politeness Paradox**: In Ukrainian, using the **Imperfective** aspect is often *more polite* for invitations (*Сідайте, заходьте, беріть*), whereas the **Perfective** can sound curt or like a military order (*Сядьте, зайдіть, візьміть*). This contrasts with English where "Sit down" is neutral.
2. **Proverbial Wisdom**: The imperative is central to Ukrainian folk wisdom. Example: *"Вік живи — вік учись"* (Live a century — learn a century). The use of imperfective verbs *живи* and *учись* emphasizes a continuous, lifelong process, not a single completed act.

## Common Learner Errors
1. **The Politeness Trap** → Using Perfective (*Сядьте!*) for polite offers.
   - *Why it happens*: Learners assume "result" = "specific command", missing the nuance that imperfective softens the command into an invitation.
2. **Prohibition vs. Warning** → Using Perfective (*Не впади*) for a general prohibition (*Don't fall*).
   - *Why it happens*: Confusion between "Do not do X" (Imperfective: *Не роби*) and "Be careful not to X" (Perfective Warning: *Гляди, не зроби*).
3. **Repeated vs. Single Command** → Using Imperfective (*Зачиняй вікно*) for a one-time specific request (*Close the window*).
   - *Why it happens*: Overusing the imperfective form because it's often shorter or learned first, instead of the required Perfective (*Зачини*).

## Notes for Content Writing
- **Critical Distinction**: Emphasize that Negation flips the logic (Negated Imperative almost always reverts to Imperfective for prohibition).
- **Pedagogical Hook**: Use the "Invitation vs. Order" concept early to grab attention — it's a high-value pragmatic skill for learners to avoid sounding rude.
- **Visual Aid**: A decision flowchart is essential for this topic: Affirmative? → Result (Pf) / Process (Imp); Negative? → Prohibition (Imp) / Warning (Pf).
