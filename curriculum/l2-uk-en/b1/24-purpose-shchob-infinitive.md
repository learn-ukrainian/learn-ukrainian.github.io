## The Same-Subject Pattern

### When to Use —â–æ–± + Infinitive

Use **—â–æ–± + infinitive** when:
1. The main clause has a subject doing an action
2. The purpose clause describes WHY that same subject does it
3. Both actions have the SAME performer

**Formula:** Subject + verb, —â–æ–± + infinitive

**Examples:**
- **–Ø** –ø—Ä–∏–π—à–æ–≤, —â–æ–± **–¥–æ–ø–æ–º–æ–≥—Ç–∏**. (I came to help.)

### The Logic Behind Same-Subject Infinitive

The grammatical principle is straightforward: when the person who performs the main verb is the same person who will perform the purpose verb, Ukrainian uses the simple infinitive. This parallels English "to + verb" constructions but with an explicit purpose marker (—â–æ–±). Compare: "I came to help" ‚Äî the subject "I" both came and will help. Ukrainian follows the same logic with its infinitive construction.

### The Logic Behind Same-Subject Infinitive

Ukrainian purpose clauses follow a simple but powerful rule: when the same person performs both actions, the second verb appears as an infinitive. This is fundamentally different from purpose clauses with different subjects, which require a different construction (covered in Module 25).

The logic is intuitive. In "I came to help," there's only one actor: "I." The coming and the helping are both performed by the same person. English uses "to" plus the base verb; Ukrainian uses "—â–æ–±" plus the infinitive. Both languages recognize that same-subject purpose simplifies the grammatical structure ‚Äî no need to repeat or specify who's doing the second action.

This same-subject rule creates clear, efficient sentences. Compare the alternatives: you could say "I came so that I would help," but that's unnecessarily complex. The infinitive construction acknowledges that the actor is already established and doesn't need re-identification.

English speakers sometimes try to force this pattern where it doesn't belong. The critical test is: are there two different people involved in the two actions? "I want to help" (same subject: I want, I help) uses infinitive. But "I want him to help" involves two people, requiring a completely different structure in Ukrainian. This module focuses on the same-subject cases; the different-subject pattern has its own logic and grammar.

Negative purpose works identically: "I speak quietly so as not to wake the baby" has one actor (I), so it uses infinitive: ¬´–Ø –≥–æ–≤–æ—Ä—é —Ç–∏—Ö–æ, —â–æ–± –Ω–µ —Ä–æ–∑–±—É–¥–∏—Ç–∏ –¥–∏—Ç–∏–Ω—É.¬ª The "–Ω–µ" simply negates the infinitive that follows —â–æ–±.
