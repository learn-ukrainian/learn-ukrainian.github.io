# Warm-up

Welcome back to the Word Factory!
Last time, we made Nouns. Today, we make **Adjectives**.
Adjectives describe *what kind* of thing something is.

In English, you use:
- **-al** (Music -> Musical)
- **-ic** (History -> Historic)
- **-an** (America -> American)
- **-y** (Sun -> Sunny)

In Ukrainian, we have standard suffixes too.
Input: **Музика** [Noun] -> Machine -> Output: **Музич-ний** [Adj].
Input: **Україна** [Noun] -> Machine -> Output: **Україн-ський** [Adj].

> [!question] The Universal Endings
> Remember that the suffix changes the *meaning*, but the *ending* (flexion) must still match the Gender!
> - Музичн**ий** (Masc)
> - Музичн**а** (Fem)
> - Музичн**е** (Neut)
> - Музичн**і** (Plural)
>
> This is exactly like regular adjectives (Good -> Гарний, Гарна, Гарне).
> The suffix (-н-, -ов-, -ськ-) just sits between the Root and the Ending.
> Root + Suffix + Ending.
> Think of it like a sandwich: Root is the meat, Suffix is the cheese, Ending is the bread.

# Grammar Breakdown: 4 Rules to Remember

Before we look at more examples, let's clarify the logic.
Unlike English, which uses many different patterns (-al, -ic, -y, -ous, -an), Ukrainian is surprisingly consistent.

**Rule 1: The Generalist (-ний) is King**
If you don't know which suffix to use, guess **-ний**. It is the most common adjective suffix in the Ukrainian language.
It works for almost all abstract concepts (Theory -> Theoretical).

**Rule 2: Materials use -овий**
If something is made of distinct parts or materials, it often uses **-овий**.
Think of "Fruit" -> "Fruit-based cake". That's a "Fruit-ov-y" cake.
Think of "Science" -> "Science-based fact". That's a "Science-ov-y" fact.

**Rule 3: Softness Matters**
Sometimes **-овий** becomes **-евий** or **-євий**. Why?
Because of Ukrainian phonetics. After soft sounds or 'j' sounds (like in Життя), we need a soft vowel.
So *Житт-ов-ий* sounds hard. *Житт-єв-ий* flows like water.

**Rule 4: People and Places use -ськ-**
This is the "Belonging" suffix.
It answers: "Whose is it?" or "Where is it from?"
It is NOT about what it is made of. It is about *identity*.
- Kyiv -> Kyiv-an identity -> Київський.
- Student -> Student identity -> Студентський.

# The Big Three: -Н, -ОВ, -СЬК

These three suffixes create 80% of Ukrainian adjectives.

## Machine 1: The Generalist (-н-ий)

This is the default suffix. Use it for general qualities.

| Noun | Adjective | English | Note |
| :--- | :--- | :--- | :--- |
| **Музика** | Музич**н**ий | Musical | к -> ч mutation |
| **Матема**тика | Математич**н**ий | Mathematical | |
| **Фізик**а | Фізич**н**ий | Physical | |
| **Смак** (Taste) | Смач**н**ий | Tasty | к -> ч |
| **Холод** (Cold) | Холод**н**ий | Cold | |

## Machine 2: The Relationship (-ов-ий)

Use this for "made of" or "related to".

| Noun | Adjective | English | Note |
| :--- | :--- | :--- | :--- |
| **Наук**а | Наук**ов**ий | Scientific | |
| **Колір** | Кольор**ов**ий | Colorful | |
| **Бізнес** | Бізнес**ов**ий | Business-related | |
| **Життя** | Житт**єв**ий | Vital / Life-related | -ов becomes -єв after soft consonant |

## Machine 3: The Origin (-ськ-ий)

Use this for places, groups, and people.

| Noun | Adjective | English | Note |
| :--- | :--- | :--- | :--- |
| **Україна** | Україн**ськ**ий | Ukrainian | |
| **Міст**о | Мі**ськ**ий | Urban / City | ст -> ськ |
| **Сел**о | Сіль**ськ**ий | Rural / Village | е -> і |
| **Люд**и | Люд**ськ**ий | Human | |
| **Чоловік** | Чолові**ч**ий | Male / Men's | Exception: -чий here |
| **Жінк**а | Жіно**ч**ий | Female / Women's | Exception: -чий here |

> [!tip] Sound Mutations
> When attaching suffixes, consonants often change to make pronunciation smoother:
> - **К, Ц, Ч** + ськ -> **-цьк-** (Козак -> Козацький).
> - **Г, Ж, З** + ськ -> **-зьк-** (Прага -> Празький).
> - **Х, Ш, С** + ськ -> **-ськ-** (Чех -> Чеський).

# Deep Dive: Living Adjectives (The Active State)

Some adjectives describe a state or feeling.
- **Гарячий** (Hot) — from "to burn".
- **Свіжий** (Fresh).
- **Чужий** (Foreign/Alien).
- **Справжній** (Real/Genuine).

> [!note] Why -ій?
> Soft adjectives end in **-ій** (masculine).
> - **Синій**: Blue
> - **Вечірній**: Evening
> - **Останній**: Last
> They decline differently: *Синього*, NOT *Синого*.

---

# Deep Dive: The Diminutive (-еньк)

Just like nouns have **-ок**, adjectives have **-еньк**.
It means "sweet", "little", or "pleasant".

- **Малий** -> **Маленький** (Little/Small).
- **Гарний** -> **Гарненький** (Pretty/Nice).
- **Теплий** -> **Тепленький** (Nice and warm).
- **Свіжий** -> **Свіженький** (Fresh).
- **Чистий** -> **Чистенький** (Clean).

*Мама купила свіженький хліб.* (Mom bought fresh-little-bread).
*Це мій маленький кіт.* (This is my little cat).

# Language in Action

## Dialogue: Shopping

**Customer:** Добрий день. У вас є **дитячий** одяг?
**Seller:** Так, звісно. Ось відділ для хлопців і дівчат.
**Customer:** Мені потрібен **теплий** светр.
**Seller:** Ось дуже гарний, **вовняний** (woolen).
**Customer:** А він м'який?
**Seller:** Так, він дуже **м'якенький** (soft-diminutive).
**Customer:** А який це розмір?
**Seller:** Це **стандартний** розмір.
**Customer:** Добре. А ціна?
**Seller:** Ціна **доступна**.
**Customer:** Чудово. Я беру.

## Monologue: The Artist

Я люблю цей світ. Він **кольоровий**.
**Синє** небо, **зелена** трава, **жовте** сонце.
Моя робота — **творча** (creative).
Я малюю **міські** пейзажі.
Я люблю **старий** Львів і **сучасний** Київ.
Кожне місто має свій **унікальний** смак.
Львів — **кавовий** (coffee-flavored).
Київ — **каштановий** (chestnut).
Одеса — **морська** (marine).
Усе це — моя **рідна** Україна.

## Dialogue: At the Hotel

**Guest:** Добрий вечір. У вас є **вільний** номер?
**Receptionist:** Добрий вечір. Так, є **одномісний** (single) і **двомісний** (double).
**Guest:** Мені потрібен **тихий** номер. Я дуже **сонний** (sleepy).
**Receptionist:** У нас є **чудовий** номер на другому поверсі.
**Guest:** Там є **гаряча** вода?
**Receptionist:** Звісно. І **свіжі** рушники.
**Guest:** А сніданок?
**Receptionist:** Сніданок **континентальний**. Дуже **смачний**.
**Guest:** Супер. Я беру.

## Dialogue: Describing People

**Anna:** Хто цей **молодий** чоловік?
**Oleh:** Це мій **новий** сусід.
**Anna:** Він дуже **симпатичний** (cute). У нього **сині** очі.
**Oleh:** Так, і він дуже **розумний**. Він — **науковий** співробітник.
**Anna:** А хто ця **гарна** жінка поруч?
**Oleh:** Це його сестра. У неї **довге** волосся.
**Anna:** Вона має **модний** вигляд.
**Oleh:** Так, вона — **відома** актриса.
**Anna:** Який **цікавий** світ!

# English theory

We have covered the main tools for building adjectives.
Let's review the "machine parts" one last time.

1.  **-н-ий** = The Generalist. Use this for general qualities (Musical, Tasty, Cold). It connects a noun to a quality.
2.  **-ов-ий** = The Materialist. Use this for things made of something or related to a field (Scientific, Colorful, Apple-flavored).
3.  **-ськ-ий** = The Identity Maker. Use this for geography, origin, and social groups (Ukrainian, Urban, Human).
4.  **-еньк-ий** = The Sweetener. Use this to make anything cute, small, or pleasant (Small-sweet, Fresh-sweet).
5.  **Active Adjectives** = These describe a state, like "Hot" (Burning) or "Fresh" (Blooming). They often behave like verbs frozen in time.

Remember: Adjectives must always agree with their noun in Gender, Number, and Case!
A "Ukrainian book" is *Українська книга* (Fem), but a "Ukrainian passport" is *Український паспорт* (Masc).

---

# Summary

1.  **-н-ий** = General quality (Музичний, Смачний).
2.  **-ов-ий** = Material/Relational (Науковий, Кольоровий).
3.  **-ський** = Origin/Group (Український, Міський).
4.  **-еньк-ий** = Diminutive (Маленький, Гарненький).
5.  **Active Adjectives** = Гарячий, Свіжий (Describe state).
