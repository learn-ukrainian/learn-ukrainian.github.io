# Багато книг, мало студентів

Родовий відмінок множини - одна з найскладніших форм в українській.
Це нормально. Навіть носії мови іноді перевіряють рідкісні слова.

На А2 ваша мета практична:

- розпізнавати найчастіші моделі;
- правильно казати поширені фрази з кількістю;
- не боятися винятків, а вчити їх маленькими групами.

## Діалог: у маленькому магазині

Продавець і помічник перевіряють, що залишилося на полицях.

**Діалог 1 - скільки є?**

> **Продавець:** Скільки пляшок води?
> **Помічник:** П'ять пляшок.
> **Продавець:** А банок меду?
> **Помічник:** Три банки. Але немає маленьких пакетів.
> **Продавець:** Скільки булок?
> **Помічник:** Немає булок. І мало свіжих яблук.
> **Продавець:** Добре. Запишіть: багато студентів питають про воду, але немає булок.

Карта діалогу:

| Фраза | Називний однини | Родовий множини |
| --- | --- | --- |
| п'ять пляшок | пляшка | пляшок |
| банок меду | банка | банок |
| немає пакетів | пакет | пакетів |
| немає булок | булка | булок |
| мало яблук | яблуко | яблук |
| багато студентів | студент | студентів |

## Коли потрібен родовий множини?

Родовий множини часто з'являється після слів кількості:

> багато студентів
>
> мало книг
>
> кілька днів
>
> скільки людей?
>
> немає квитків

Після числівників від 5 і більше теж часто потрібен родовий множини:

> п'ять столів
>
> шість книжок
>
> десять міст
>
> двадцять студентів

На А2 просто запам'ятайте контраст:

| Фраза | Форма |
| --- | --- |
| один студент | однина |
| два студенти | інша модель, не фокус цього модуля |
| п'ять студентів | родовий множини |
| багато студентів | родовий множини |

## Чоловічий рід: найчастіше -ів

Для багатьох іменників чоловічого роду родовий множини має закінчення
`-ів`.

| Називний однини | Родовий множини |
| --- | --- |
| студент | студентів |
| брат | братів |
| стіл | столів |
| будинок | будинків |
| підручник | підручників |
| олівець | олівців |

Деякі слова змінюють основу:

| Називний | Родовий множини | Коментар |
| --- | --- | --- |
| день | днів | голосний зникає |
| хлопець | хлопців | е зникає |
| українець | українців | е зникає |

Є невелика група людей на `-ин/-їн`, де закінчення нульове:

> громадянин - громадян
>
> селянин - селян

Але це не основна модель. Для більшості звичайних чоловічих слів на А2
безпечна перша гіпотеза: `-ів`.

## Жіночий рід: часто нульове закінчення

Багато іменників жіночого роду в родовому множини втрачають `-а` або
`-я`.

| Називний однини | Родовий множини |
| --- | --- |
| книга | книг |
| школа | шкіл |
| сестра | сестер |
| банка | банок |
| сумка | сумок |
| зупинка | зупинок |
| пісня | пісень |

Іноді з'являється вставний голосний `о` або `е`, щоб кінець слова було
легше вимовити:

> банка - банок
>
> сумка - сумок
>
> сестра - сестер
>
> зупинка - зупинок

Деякі часті слова мають форму на `-ей`:

> стаття - статей
>
> сім'я - сімей
>
> миша - мишей

Слова на `-ія` часто мають форму без `-я`:

> лекція - лекцій
>
> станція - станцій
>
> історія - історій

Не намагайтесь вивести все з одного правила. Вивчайте модель плюс
кілька частих прикладів.

## Середній рід: нульове, -ів або особлива основа

У середньому роді є кілька дуже частих моделей.

| Називний однини | Родовий множини |
| --- | --- |
| місто | міст |
| слово | слів |
| вікно | вікон |
| яблуко | яблук |
| море | морів |
| поле | полів |
| завдання | завдань |
| питання | питань |
| ім'я | імен |

Зверніть увагу:

> багато міст
>
> мало слів
>
> кілька вікон
>
> немає питань

Форми `завдань` і `питань` дуже корисні для навчання:

> У мене багато завдань.
>
> У вас є кілька питань?
>
> Немає питань.

## Кількість у реальному мовленні

Найкраще вчити родовий множини в готових фразах.

| Слово кількості | Приклад |
| --- | --- |
| багато | багато студентів, багато книг |
| мало | мало місць, мало питань |
| кілька | кілька днів, кілька зупинок |
| скільки | скільки людей, скільки квитків |
| немає | немає булок, немає пакетів |

Міні-діалог:

> **Покупець:** Скільки є яблук?
> **Продавець:** Десять кілограмів.
> **Покупець:** А булок?
> **Продавець:** Немає булок, але є багато пакетів печива.

Пам'ятайте: після `багато`, `мало`, `кілька`, `скільки`, `немає` ви
дуже часто питаєте себе: «яка форма множини в родовому?»

## Міні-рольова гра: список запасів

**Діалог 2 - що треба купити?**

> **Ірина:** У нас мало яблук і немає булок.
> **Павло:** Скільки треба пляшок води?
> **Ірина:** Шість пляшок. І кілька пакетів рису.
> **Павло:** А для уроку?
> **Ірина:** Багато зошитів і десять олівців.

Фрази для вашої відповіді:

- `У нас багато ...`
- `У нас мало ...`
- `Немає ...`
- `Треба п'ять/шість/десять ...`

## Підсумок

Родовий множини має багато варіантів, але A2-стратегія проста:

| Тип | Частий результат | Приклади |
| --- | --- | --- |
| чоловічий рід | -ів | студентів, столів, підручників |
| жіночий рід | нульове закінчення, часто зі вставним голосним | книг, банок, сестер |
| середній рід | нульове, -ів або особлива форма | міст, морів, питань |
| кількість | часто родовий множини | багато книг, п'ять студентів |

Самоперевірка:

- Як сказати `many students`?
- Як сказати `no buns`?
- Яка форма після `скільки`: `квитки` чи `квитків`?
