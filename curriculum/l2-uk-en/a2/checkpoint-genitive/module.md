# Контрольна робота: родовий відмінок

Це checkpoint для всього блоку A2.2 про родовий відмінок. Це не нова
велика тема, а спокійна перевірка: що вже працює, а що треба повторити.

Ви перевіряєте чотири вміння:

- прийменники з родовим: `з/із/зі`, `від`, `після`, `для`, `без`,
  `біля`, `навпроти`, `коло`, `до`;
- повні фрази: `мого нового друга`, `цієї старої книги`;
- родовий множини: `багато студентів`, `немає булок`;
- реальні ситуації: покупки, здоров'я, місто, час.

## Діалог: екскурсія Києвом

Гід пояснює маршрут групі. Тут зібрані різні моделі родового.

**Діалог 1 - біля собору**

> **Гід:** Ми стоїмо біля Софійського собору.
> **Турист 1:** До Хрещатика далеко?
> **Гід:** До Хрещатика п’ять хвилин пішки.
> **Турист 2:** А без квитка можна зайти?
> **Гід:** Ні, без квитка не можна. Для групи з десяти людей є знижка.
> **Турист 1:** Після екскурсії ми йдемо до музею?
> **Гід:** Так, музей навпроти старої школи, недалеко від парку.

Карта діалогу:

| Фраза | Модель | Функція |
| --- | --- | --- |
| біля Софійського собору | біля + родовий | розташування |
| до Хрещатика | до + родовий | напрямок |
| без квитка | без + родовий | відсутність |
| для групи | для + родовий | призначення |
| з десяти людей | з + родовий | склад групи |
| після екскурсії | після + родовий | час |
| навпроти старої школи | навпроти + родовий | розташування |
| від парку | від + родовий | відстань |

## Частина 1: впізнавання

Спочатку не творіть форму, а впізнайте її.

Питання:

1. Який прийменник у реченні?
2. Яке значення він має?
3. Яка форма після нього?

Приклади:

| Речення | Прийменник | Значення | Форма |
| --- | --- | --- | --- |
| Я йду до аптеки. | до | напрямок | аптеки |
| Ми сидимо біля вікна. | біля | місце | вікна |
| Після уроку я вільний. | після | час | уроку |
| Це подарунок для мого брата. | для | призначення | мого брата |

Якщо бачите `немає`, теж думайте про родовий:

> Немає квитків.
>
> Немає температури.
>
> Немає часу.

## Частина 2: вибір правильної форми

Тепер перевіряйте повну фразу.

| Називний | Родовий |
| --- | --- |
| мій новий друг | мого нового друга |
| ця стара книга | цієї старої книги |
| наш великий будинок | нашого великого будинку |
| ваша нова лікарня | вашої нової лікарні |

Міні-алгоритм:

1. Знайдіть іменник.
2. Визначте рід: чоловічий, жіночий, середній.
3. Поставте іменник у родовий.
4. Узгодьте прикметник або займенник.

> без мого нового друга
>
> біля цієї старої школи
>
> до вашої нової лікарні

## Частина 3: родовий множини

Родовий множини треба вчити групами.

| Модель | Приклади |
| --- | --- |
| -ів | студентів, столів, зошитів |
| нульове закінчення | книг, булок, міст, яблук |
| зміна основи | днів, слів, людей |
| м'який знак | питань, завдань, пісень |

Після кількості:

> багато студентів
>
> мало книг
>
> кілька днів
>
> п'ять пляшок води
>
> скільки людей?

Після `немає`:

> немає квитків
>
> немає булок
>
> немає питань

## Частина 4: реальні ситуації

Перевірте себе в коротких сценаріях.

**На ринку**

> Дайте, будь ласка, кілограм яблук і пляшку води.

**У лікаря**

> Температури немає, але немає сил.

**У місті**

> Як пройти до аптеки? Вона біля цього нового ринку?

**Про час**

> Після роботи я йду до лікаря. Чекайте до шостої.

## Самооцінка

Позначте для себе:

## Підсумок

Родовий відмінок у цьому блоці має кілька різних дверей, але всі вони
ведуть до практичного мовлення.

| Двері | Приклад |
| --- | --- |
| прийменники | до аптеки, біля школи, без квитка |
| узгодження | мого нового друга, цієї старої книги |
| множина | багато книг, мало студентів |
| ситуації | кілограм яблук, ліки від кашлю |

Самоперевірка:

- Опишіть три місця у кімнаті з `біля`, `навпроти`, `коло`.
- Складіть одну фразу з `мого`, одну з `цієї`, одну з `нашої`.
- Скажіть три кількості: `багато ...`, `кілька ...`, `п'ять ...`.
