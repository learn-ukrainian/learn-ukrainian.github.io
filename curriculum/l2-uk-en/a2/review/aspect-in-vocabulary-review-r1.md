However, зрозуміти is formed by prefix addition: **з-** + розуміти → зрозуміти. The root розумі- is identical in both forms. This is a Method 1 (prefix) pair, not Method 2. Ukrainian textbooks confirm: Avramenko Grade 7 §25 lists "зміна основи слова" (suppletion) and "prefix" as separate categories. зрозуміти/розуміти is straightforward prefixation. This pair was added by the writer (not in the plan) and is placed in the wrong category. **This is a factual error about Ukrainian linguistics that learners will memorize.**

**B. ловити / зловити classified as suppletion (Method 3).** The module states: *"Linguists still group ловити / зловити here because the prefix з- does not create the perfective meaning the way на- creates написати from писати — it merely reinforces a root that already implies catching."* This claim is **factually false**. зловити is transparently formed by prefix з- + ловити (shared root лов-). Ukrainian textbooks (Avramenko Grade 7 §25, Karaman Grade 10 §72) list suppletion examples as брати/взяти, шукати/знайти — pairs with zero shared root material. ловити/зловити is a prefix pair. The true suppletive partner would be ловити/піймати (VESUM-confirmed). **Note: this error originates in the plan**, which lists "ловити / зловити (to catch)" under suppletion. The writer noticed the contradiction but rationalized it with a false linguistic claim instead of flagging it. **Plan requires version bump to fix the root cause.**

**C. забути / забувати placed under "vowel alternation" sub-section.** забути → забувати is formed by inserting the suffix **-ва-** (забу + ва + ти). No root vowel changes.
