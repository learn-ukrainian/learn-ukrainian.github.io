## Linguistic Scan

**CRITICAL ERROR found:**

In Section 4 (Порівняння відмінків), the dialogue contains:

> "Я зараз покажу школу **нашому нового студенту**."

This mixes dative **нашому** with genitive **нового**. In a module that teaches dative adjective agreement, this is a devastating error — the learner sees an example labeled "Dative" that uses a genitive adjective. The correct form is: **нашому новому студентові**.

VESUM confirms: **нового** = genitive of новий; **новому** = dative of новий. Both exist in VESUM, so this is a case-mixing error, not a spelling error.
