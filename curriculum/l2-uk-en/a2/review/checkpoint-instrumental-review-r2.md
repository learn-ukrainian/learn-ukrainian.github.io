**6. Factually wrong grammar claim — CRITICAL:**

In the **Підсумок** section, Q1 asks: *"Які закінчення мають іменники чоловічого роду в орудному відмінку після м'яких приголосних?"* The answer uses **лікар → лікарем** as an example:

> «Вони мають закінчення -ем (наприклад, лікар стає лікарем)»

**The consonant -р in лікар is HARD, not soft.** VESUM confirms `лікарем` is the correct (and only) instrumental form, and `лікаром` does NOT exist in VESUM. However, the reason лікар takes -ем is NOT soft consonant — it's the **-ар suffix pattern**. The Захарійчук Grade 4 textbook (с. 63, §134) explicitly teaches this as a separate rule: *"Іменники чоловічого роду з основою на -ар, -яр в орудному відмінку -ом, -ем. Правопис закінчень таких іменників треба перевіряти за орфографічним словником."* The textbook table shows: слюсар → слюсарем, вівчар → вівчарем, BUT столяр → столяром — confirming these are irregular and need dictionary verification. Using лікар as an example of "soft consonant → -ем" teaches a **wrong generalization** that will confuse learners.

**7. Missing -ар/-яр pattern in Part 1:** The plan references Захарійчук Grade 4, с. 62-69. That textbook explicitly covers -ар/-яр instrumental endings as a special case (§134). The module omits this entirely, presenting only the hard/soft/sibilant rule, which doesn't account for words like лікар, слюсар, вівчар.
