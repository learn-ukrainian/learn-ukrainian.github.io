## Linguistic Scan

### Gender agreement errors (CRITICAL — 3 errors in one dialogue)

In the final consolidation dialogue (Section 4), the speaker **Олена** (female name) uses masculine verb/adjective forms throughout:

1. **«На цьому тижні я дуже зайнятий на роботі.»** — `зайнятий` is masculine. Must be `зайнята` (feminine).
2. **«Я почув ці цікаві новини по радіо.»** — `почув` is masculine past tense. Must be `почула` (feminine).
3. **Максим asks Олена: «Як ти дізнався про це свято?»** — `дізнався` is masculine. Must be `дізналася` (feminine), since Максim is addressing Олена.

VESUM confirms: `зайнята` ← adj form of `зайнятий`; `почула` ← verb `почути`; `дізналася` ← verb `дізнатися`. All correct feminine forms exist.

**This is a 3-for-3 gender agreement failure in a single dialogue in a language teaching module. A learner memorizing these sentences will internalize wrong agreement patterns.**

### Minor linguistic note

The module states: *"Some modern usage shifts to по + dative for distribution (по одному), but for communication means, locative is standard."* — The framing of по + dative as a "modern shift" is misleading. По + dative for distribution (дати по яблуку) is a standard, long-established Ukrainian pattern documented in Avramenko Grade 11 (§72), not a modern innovation. This is a factual inaccuracy about Ukrainian grammar, though in a parenthetical note rather than the core teaching.

### No Russianisms, Surzhyk, calques, or Russian characters found.

The phrase «по телефону» was checked against Антоненко-Давидович — no flag. VESUM confirms `телефону` as valid locative (v_mis). All other Ukrainian forms verified correct.
