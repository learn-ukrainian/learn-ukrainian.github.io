# Дослідження: In Order To

## State Standard Reference
§4.4.2: "складнопідрядні реченні зі сполучником щоб для вираження мети певної дії суб’єкта: Ми вчимо українську мову, щоб розмовляти з друзями." (Lines 1415-1416)
Alignment: This module directly addresses the requirement to form complex sentences expressing purpose using "щоб" and "для того щоб", which is a core syntax competency for A2. It also reinforces Genitive case usage with "для" (§4.2.2.2.2).

## Vocabulary Frequency
| Word | Frequency / Source | Key collocations |
|------|-------------------|------------------|
| **щоб** | High (Top 100) | Я хочу, щоб... / Він пішов, щоб... / Для того, щоб... |
| **для** | High (Top 50) | для тебе / для того / для роботи / для здоров'я |
| **аби** | Medium | аби тільки / аби не спізнитися (often implies "just so that" or "if only") |
| **мета** | Medium | моя мета / з метою (formal) |

## Cultural Hooks
1. **The "Щоб" Toast Culture**: In Ukraine, toasts almost always start with "Щоб..." (May we have...). A classic playful toast is: *"Щоб елось і пилося, щоб хотілося і моглося!"* (May we eat and drink, may we want and be able!). This makes the grammar point instantly social and useful.
2. **Proverb**: *"На те коня кують, щоб не спотикався."* (One shoes a horse so it doesn't stumble). Illustrates purpose and planning.

## Common Learner Errors
1. **The "For To" Calque**: *Я йду в магазин для купити хліб.* (Wrong) → *Я йду в магазин, **щоб** купити хліб.* (Correct). Learners often translate "for" directly as "для" with an infinitive. "Для" is ONLY for nouns/pronouns.
2. **Subject Mismatch**: *Я хочу тебе читати.* (Wrong) → *Я хочу, **щоб ти читав**.* (Correct). Learners fail to switch to the past tense (subjunctive) when the subject changes.
3. **Omitting the Comma**: *Він прийшов щоб допомогти.* (Wrong) → *Він прийшов, **щоб** допомогти.* (Correct). Purpose clauses in Ukrainian are strictly separated by commas.

## Cross-References
- **Builds on**: A1/A2 Verb Infinitive, Past Tense (for subjunctive formation), Genitive Case (for *для*).
- **Prepares for**: B1 Subjunctive Mood, B1 Complex Syntax, C1 Stylistics (Official Style *з метою*).

## Notes for Content Writing
- **Visual Logic**: Use a clear flowchart or decision tree: "Is the doer the same?" → YES (Infinitive) / NO (Past Tense).
- **Register Check**: Keep *аби* informal/spoken and *для того, щоб* emphatic/formal. Focus 80% of practice on standard *щоб*.
