## State Standard Reference
§4.2.2.2: "Родовий відмінок: ... Із прийменниками:"

## Cultural Hooks
1. Since 2022, Ukraine has widely implemented the «е-рецепт» (electronic prescription) system via SMS for antibiotics and other prescription drugs to regulate dispensing and modernize healthcare (Source: moz.gov.ua).
2. Many pharmacies in Ukraine operate as a «чергова аптека» (duty pharmacy), remaining open 24/7 for nighttime emergencies, identifiable by a prominent green cross. Additionally, there is a strong cultural preference for «фітотерапія» (herbal remedies) as a first line of defense for minor ailments.

## Common Learner Errors
1. **Ліки для кашлю** → **Ліки від кашлю** — English speakers directly translate "medicine for a cough" using "для", but Ukrainian requires "від" (against/from) + Genitive.
2. **Це ліки** → **Ці ліки** — Learners treat "ліки" as a singular noun, but it is pluralia tantum (always plural) and requires plural agreement for adjectives and demonstratives.
3. **Лікарство** → **Ліки** — Using the Russianism "лікарство" instead of the standard Ukrainian word "ліки".
