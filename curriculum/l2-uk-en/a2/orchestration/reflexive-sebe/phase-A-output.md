# Дослідження: Reflexive Pronoun Sebe

## State Standard Reference
§4.2.1.4 (B1 level line 2107): "зворотний займенник себе (собі, собою, на собі);"
Alignment: While the explicit standard for `себе` declension is formalized in B1, this module introduces it in A2 (Phase A2.2) to support the expanded communicative intentions of A2, such as describing oneself and using Dative/Instrumental cases functionally. It serves as a crucial bridge, shifting learners away from English-style reflexive pronouns.

## Cultural Hooks
1. **The Philosophy of -ся vs себе:** Ukrainian inherently streamlines routine daily actions (washing, dressing) into the verb itself using the `-ся` suffix. The explicit pronoun `себе` is reserved for emphasis, reflection, and psychological states. This reflects a linguistic economy where the self is naturally integrated into the action unless special attention is needed.
2. **Proverbial Wisdom:** Ukrainian proverbs frequently use `себе` to emphasize self-respect and self-reliance, e.g., «Хто сам себе не поважає, того й інші не поважатимуть» (Whoever does not respect himself, others will not respect him either). 

## Common Learner Errors
1. **Overusing `себе` for routine actions** → *Я мию себе* instead of *Я вмиваюся*. English speakers map "myself" directly to `себе`, missing the natural `-ся` suffix.
2. **Searching for a Nominative subject** → Learners try to use `себе` as the subject of a sentence, failing to realize it only exists in oblique cases (Genitive through Locative).
3. **Confusing `себе` and `свій`** → Using `себе` when possession is meant (e.g., *Я люблю себе собаку* instead of *свою собаку*).

## Notes for Content Writing
- **Decolonized Framing:** Frame the Ukrainian system as an elegant simplification. English requires six distinct reflexive pronouns (myself, yourself, himself, etc.), whereas Ukrainian uses a single, unifying pronoun (`себе`) across all persons and numbers. Do not compare to Russian.
- **Immersion:** The module is A2.2 (50-60% Band 1). Use English scaffolding for the theoretical explanation of the missing Nominative case and the `-ся` contrast. All examples, practice, and dialogues must be in Ukrainian.
- **Scaffolding:** Explicitly define grammatical terms (e.g., "Reflexive", "Target") before using them. Ensure each case form gets its own H3.
