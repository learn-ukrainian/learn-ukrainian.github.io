## Grammar Rules
- Accusative Plural (Animate): Правопис §99 — Inanimate nouns have Acc. equal to Nom., while animate nouns (masculine/feminine) have Acc. equal to Gen. (лікарі -> бачу лікарів).
- Genitive Plural Endings: §92 — Masculine nouns usually take -ів (лікарів, пацієнтів, подарунків), but some soft/mixed nouns like "гість" take -ей (гостей).
- Locative Plural: §99 — Ending -ах / -ях (у лікарнях, на вечірках, у потягах).

## Calque Warnings
- приймати участь: Calque — Use **брати участь** (Confirmed by style guide ad-172).
- подорожувати по Україні: Stylistic preference — Use **подорожувати Україною** (Instrumental case for space).
- приймати ліки: OK — But style guide ad-220 suggests using "ліки проти" (instead of "від").
- визначне місце: OK — But **пам'ятка** is more precise for cultural/historical landmarks (ad-052).
