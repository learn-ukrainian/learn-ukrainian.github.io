# Граматика A2: Перші кроки в А2



## Як це пояснюють у школі (How Schools Teach This)
На рівні А2 українські методичні підходи, орієнтовані на іноземців, вводять граматику через практичні, комунікативні ситуації. Замість абстрактних правил, учні спочатку стикаються з граматикою в реалістичних діалогах.

Наприклад, подкаст "Ukrainian Lessons" (ULP) демонструє цей підхід:
1.  **Контекст перш за все:** Урок починається з діалогу, що моделює реальну життєву ситуацію: покупки на ринку (Source 1), замовлення в піцерії (Source 3), планування тижня (Source 5) або прохання про пораду (Source 4). Учням дають просте завдання на розуміння (наприклад, "спробувати запам'ятати, що купила Христина" (Source 1)).
2.  **Виокремлення патернів:** Після прослуховування вчитель звертає увагу на конкретні граматичні явища, що з'явилися в діалозі. Наприклад, після діалогу про поради, учнів просять знайти пари дієслів доконаного і недоконаного виду, які вони щойно почули: "купити і купувати", "поїсти і їсти", "надіслати і надсилати" (Source 4).
3.  **Пояснення "на пальцях":** Правила пояснюються через функцію, а не форму. Різниця між видами дієслів подається як "One Time Action versus a regular Action" або "result and process" (Source 4). Знахідний відмінок пояснюється як відмінок для прямого об'єкта ("Я люблю Україну") та для позначення напрямку руху ("підемо в бар") чи часу ("у п'ятницю") (Source 5, 9).
4.  **Фокус на "милозвучності":** Підкреслюється, що багато граматичних особливостей української мови, як-от використання прийменників `у`/`в`, слугують для гарного звучання (`милозвучність`). Це робить мову менш механічною і більш живою для учня (Source 5).
5.  **Введення зменшувально-пестливих форм:** Такі форми, як *картопелька*, *борщик*, *сонечко*, вводяться не як окрема граматична тема, а як невід'ємна частина живої мови, що передає емоції та ставлення (Source 1). Це показує, що граматика — це не лише правила, а й інструмент для вираження відтінків почуттів.

Такий підхід (від контексту до правила) дозволяє учням рівня А2 засвоювати складні концепції (як-от види дієслів) органічно, бачачи їхню безпосередню користь у спілкуванні.

## Повна парадигма (Full Paradigm)
На початковому етапі А2 основна увага приділяється найбільш контрастним і частотним граматичним явищам.

### 1. Види дієслів: Недоконаний vs. Доконаний (Imperfective vs. Perfective)
Це головний виклик на рівні А2. Учень має навчитися розрізняти дію як **процес/регулярну дію** (недоконаний вид) і дію як **завершений результат/одноразову подію** (доконаний вид).

**Основні пари з діалогів (Sources 3, 4):**
| Недоконаний (Процес) | Доконаний (Результат) | Приклад (що робив?) / (що зробив?) |
| :--- | :--- | :--- |
| купувати | купити | Я завжди **купую** овочі на ринку. / Вчора я **купив** овочі. |
| їсти | поїсти | Я зазвичай **їм** вдома. / Де можна **поїсти**? |
| надсилати | надіслати | Я рідко **надсилаю** листи. / Треба **надіслати** листівку. |
| знаходити | знайти | Я часто **знаходжу** там щось цікаве. / Де можна **знайти** фотоапарат? |
| пригощати | пригостити | Ти вже вдруге мене **пригощаєш**. / Можеш ти мене **пригостити**? |
| замовляти | замовити | Що ви будете **замовляти**? / Що **замовили** дівчата? |
| вибирати | вибрати | У нас немає часу **вибирати**. / Допоможіть нам **вибрати** піцу. |

**Чергування звуків при утворенні виду (Source 3):**
При переході від недоконаного до доконаного виду (або навпаки) часто відбуваються чергування приголосних у корені.
| Чергування | Приклад (недок./док.) |
| :--- | :--- |
| `ш` ↔ `с` | прикра**ш**ати / прикра**с**ити |
| `щ` ↔ `ст` | приго**щ**ати / приго**ст**ити |
| `ж` ↔ `з` | заморо**ж**увати / заморо**з**ити |
| `ч` ↔ `т` | скоро**ч**увати / скоро**т**ити |
| `ч` ↔ `к` | кри**ч**ати / кри**к**нути |
| `дж` ↔ `д` | наро**дж**увати / наро**д**ити |
| `вл` ↔ `в` | замо**вл**яти / замо**в**ити |

### 2. Знахідний відмінок (Accusative Case)
На А2 знахідний відмінок є ключовим для побудови простих речень.

**Використання з днями тижня (Source 5):**
| Номінатив (хто? що?) | Знахідний з `у`/`в` (коли?) |
| :--- | :--- |
| понеділок | **у** понеділок |
| вівторок | **у** вівторок |
| середа | **у** серед**у** |
| четвер | **у** четвер |
| п'ятниця | **у** п'ятниц**ю** |
| субота | **у** субот**у** |
| неділя | **у** неділ**ю** |
*Патерн: чоловічий рід не змінюється, жіночий отримує закінчення `-у` або `-ю`.*

**Особові займенники: Прямий об'єкт і прийменники (Source 9)**
| Називний | Знахідний (кого? що?) | Родовий (з прийменником `без`, `у`, `для`) |
| :--- | :--- | :--- |
| я | мене | без **мéне** |
| ти | тебе | без **тéбе** |
| він | **його** | без **ньóго** (з'являється `н-`) |
| вона | **її** | без **нéї** (з'являляється `н-`) |
| ми | нас | без **нáс** |
| ви | вас | без **вáс** |
| вони | **їх** | без **ни́х** (з'являється `н-`) |
*Ключове правило: Після прийменників у займенниках 3-ї особи (`він`, `вона`, `вони`) з'являється приставний `н-`.*

## Частотність і пріоритети (Frequency & Priorities)
1.  **Види дієслів (Найвищий пріоритет):** Без розуміння аспектної системи неможливо коректно говорити про минулі події або будувати плани. На А2 слід зосередитись на 15-20 найбільш уживаних парах, особливо пов'язаних з повсякденними діями: *робити/зробити, писати/написати, читати/прочитати, їсти/з'їсти, пити/випити, купувати/купити, дзвонити/подзвонити, бачити/побачити*.
2.  **Знахідний відмінок (Високий пріоритет):** Це другий за важливістю відмінок після називного. Пріоритетними є:
    *   Прямий об'єкт: *Я читаю **книгу**.*
    *   Напрямок з `в`/`на`: *Я йду **в магазин**, **на роботу**.*
    *   Час з `в`/`у`: *Я прийду **в суботу**.*
3.  **Зменшувально-пестливі форми (Середній пріоритет):** На рівні А2 учень повинен їх **пасивно розуміти**, оскільки вони дуже поширені в живому спілкуванні (Source 1). Активне використання не є обов'язковим, але знання кількох форм (*кава/кавуся, дякую/дякуючки, смачно/смачненько*) значно підвищує природність мовлення.
4.  **Незмінювані іменники (Низький пріоритет):** Такі слова як *кіно, метро, таксі, меню, какао* (Source 11) потрібно просто запам'ятати як винятки, що не змінюють форму. Це питання лексики, а не складної граматики.

| Я вчора **читав** книгу. (Якщо книгу дочитано) | Я вчора **прочитав** книгу. | Англомовні учні часто використовують недоконаний вид для будь-якої минулої дії. Українська мова вимагає доконаного виду, якщо є результат (книга дочитана). "Читав" означає процес, можливо, не завершений (Source 4). |
| Я люблю **Україна**. | Я люблю **Україну**. | Прямий переклад з англійської без урахування відмінків. У знахідному відмінку іменники жіночого роду на `-а` змінюють закінчення на `-у` (Source 5). |
| Я був у **його** в гостях. | Я був у **нього** в гостях. | Забувають додати приставний `н-` до займенників 3-ї особи (`він, вона, вони`) після прийменників `у, в, до, без, з` і т.д. (Source 9). |
| Ми зустрінемось **п'ятниця**. | Ми зустрінемось **у п'ятницю**. | Калькування англійської конструкції "See you Friday". В українській мові для позначення дня тижня, коли щось відбудеться, використовується прийменник `у`/`в` зі знахідним відмінком (Source 5). |
| Дайте мені **два кілограм** картоплі. | Дайте мені **два кілограми** картоплі. | Неправильне узгодження числівників з іменниками. Після `два`, `три`, `чотири` іменник чоловічого роду має форму називного відмінка множини (закінчення `-и`) (Source 10). |
| Де знаходиться **метро**? / **Метро** є новий. | Де знаходиться **метро**? / **Метро** **нове**. | Помилки в узгодженні роду для незмінюваних іменників. Слова іншомовного походження на `-о` (кіно, метро, таксі) є середнього роду, тому прикметники та дієслова мають узгоджуватися з ними в середньому роді (Source 11). |

## Деколонізаційні застереження (Decolonization Notes)
**Це обов'язковий розділ.** Навчання української мови на пострадянському просторі часто страждає від інерційного порівняння з російською, що є педагогічно хибним і культурно шкідливим. Українська — самостійна, повноцінна мова, а не "діалект російської з польськими словами".

1.  **Не "як у російській, але...":** Заборонено пояснювати українські явища через їхню схожість або відмінність від російських. Правило про приставний `н-` у займенниках (`у нього`) є внутрішньою нормою української мови, а не "відмінністю від русского 'у него'". Пояснення має йти від української логіки.
2.  **Уникайте "Суржику":** Навчальні матеріали не повинні містити лексичного чи граматичного суржику. Наприклад, вживання "да" замість "так", "канешно" замість "звичайно". Акцент на чистоті мови є актом культурного самозбереження.
3.  **Види дієслів — не однакові:** Хоча категорія виду існує в обох мовах, багато аспектних пар, їхні значення та префікси не збігаються. Наприклад, українське `шукати` (недок.) / `знайти` (док.) відрізняється від російського `искать / найти`. Вивчення українських пар має відбуватися з нуля, без опори на російські аналоги.
4.  **Зменшувальні форми — це ДНК мови:** Українська мова має значно багатшу і більш вживану систему димінутивів, ніж сучасна російська. Форми як *кавуся, борщик, сонечко, здоров'ячко, їстоньки* (Source 1) є маркером української ідентичності і "милозвучності" (Source 5). Це не "дитяча мова", а повноцінний емоційний регістр. Ігнорування цього пласту робить мовлення учня штучним і "дерев'яним".
5.  **Культурний контекст має значення:** Як чітко зазначено в (Source 2), сувеніри на кшталт матрьошок чи радянської символіки є чужими й образливими для українського контексту. Натомість варто просувати автентичні українські символи: трав'яний чай, мед, вишиванку. Це стосується і мовних прикладів: замість нейтральних "стіл/стілець" краще використовувати культурно значущі лексеми.
6.  **"Російська колія":** Вислів Віктора Ющенка про "глибоку російську колію" в політиці (Source 6) можна екстраполювати на лінгводидактику. Мета сучасного викладання — зійти з цієї колії, тобто перестати розглядати українську мову через російську призму.

## Природні приклади (Natural Examples)
**Мінімум 12 речень. Приклади взяті безпосередньо з наданих джерел.**

**Група 1: Види дієслів у питаннях і відповідях (Source 4)**
1.  — Де можна **купити** фрукти? (Доконаний: одноразова дія)
2.  — Я там завжди **купую** овочі. (Недоконаний: регулярна дія)
3.  — Де можна **надіслати** листівку? (Доконаний: одноразова дія)
4.  — Давненько я не **надсилала** паперові листи. (Недоконаний: процес/тривала відсутність дії)

**Група 2: Знахідний відмінок для часу і мети (Source 5, 9)**
5.  Хочу запросити тебе в гості **у п'ятницю** на маленьку **вечірку**.
6.  **У суботу** підемо **в театр** на **виставу** "Наталка-Полтавка".
7.  Це наш розклад **на тиждень**.

**Група 3: Зменшувально-пестливі форми в живій мові (Source 1)**
8.  А у вас мандарини солодкі? — Дуже добрі **Мандаринки**.
9.  Візьміть до **борщику** зелений **пучечок**, **цибульку**, **часничок**.
10. Їж на **здоров'ячко**, **сонечко**.

**Група 4: Особові займенники з прийменниками (Source 9)**
11. Приходь, бо без **тебе** буде сумно.
12. Всі будуть раді **тебе** бачити.
13. Я ж був у **тебе** недавно.

## Рекомендації для вправ (Activity Concepts)
*   **Фаза 1 (Розпізнавання):**
    *   **"Знайди пару":** Дати учням діалог (Source 3, 4) і попросити виписати всі пари дієслів недоконаного/доконаного виду.
    *   **"Сортувальник":** Дати список дій (наприклад, *щодня читати газету, прочитати книгу, завжди купувати молоко, купити хліб*) і попросити розсортувати їх у дві колонки: "Процес" (недоконаний) і "Результат" (доконаний).
    *   **"Чутливе вухо":** Учень слухає речення і піднімає картку "процес" або "результат". Наприклад: "Вчора я цілий день *писав* лист" (процес) vs. "Нарешті я *написав* цей лист!" (результат).

*   **Фаза 2 (Контрольована практика):**
    *   **"Вибери правильне слово":** Вправи типу "Вчора я (купував/купив) хліб і молоко".
    *   **"Плануємо тиждень":** Заповнити таблицю-розклад, використовуючи конструкції з `у/в` + знахідний відмінок: "У понеділок я буду...", "У середу я піду..." (на основі Source 5).
    *   **"Запроси друга":** Трансформаційні вправи з займенниками: "Я запрошую (ти)" → "Я запрошую **тебе**". "Я йду без (він)" → "Я йду без **нього**".

*   **Фаза 3 (Вільна практика):**
    *   **Рольова гра "На ярмарку" (Source 1):** Один учень — продавець, інший — покупець. Покупець має щось *купити* (док.), а продавець розхвалює свій товар, активно використовуючи зменшувальні форми (*картопелька, морквочка*).
    *   **"Що ти робив на вихідних?":** Усна розповідь, де учень має коректно використовувати минулий час доконаного ("Я *подивився* фільм") і недоконаного ("Потім я довго *гуляв* у парку") виду.
    *   **"Складаємо меню для вечірки":** Завдання, де потрібно написати список покупок, використовуючи правильні форми родового відмінка множини (напр. "два кілограми помідорів", але "п'ять кілограмів помідорів") (на основі Source 10).

## Зв'язки з іншими темами
*   **Передумови:**
    *   **Граматика А1:** Базове відмінювання дієслів у теперішньому часі, знання родів іменників (чоловічий, жіночий, середній), прості речення.
    *   **Лексика А1:** Назви продуктів, днів тижня, базові дієслова дії.
*   **Що ця тема відкриває:**
    *   **Дієслова руху з префіксами (B1):** Поняття виду є основою для розуміння пар *йти/піти, їхати/поїхати*, а згодом і *прийти, вийти, зайти*.
    *   **Минулі часи:** Коректне використання видів дозволяє розповідати історії, розрізняючи фон (недоконаний) та ключові події (доконаний).
    *   **Складні речення (B1/B2):** Розуміння аспектів є ключем до правильного використання часових конструкцій у складнопідрядних реченнях (напр. "*Коли я прийшов*, він уже *читав* книгу").
    *   **Усі інші відмінки:** Знахідний відмінок слугує моделлю для вивчення інших непрямих відмінків, оскільки вводить поняття зміни закінчення залежно від функції слова в реченні.

# Граматика A2: Пробний іспит



## Як це пояснюють у школі (How Schools Teach & Assess This)

Пробний іспит наприкінці рівня А2 є комплексним інструментом оцінювання, що перевіряє не ізольовані граматичні знання, а загальну комунікативну компетентність учня. Українська шкільна система, як і сучасні європейські методики, оцінює володіння мовою через чотири основні види мовленнєвої діяльності: **аудіювання, читання, говоріння та письмо** (Джерело: 5-klas-ukrmova-avramenko-2022_s0009, 6-klas-ukrmova-zabolotnyi-2020_s0228).

На відміну від простого тесту з граматики, пробний іспит симулює реальні життєві ситуації спілкування. Оцінювання відбувається за 12-бальною шкалою, де вищі бали (10-12) відповідають високому рівню володіння (Джерело: ext-ulp_youtube-60).

Ключові принципи шкільного оцінювання, які мають бути відображені в іспиті:
1.  **Інтегрованість:** Завдання поєднують кілька навичок. Наприклад, учень слухає текст (аудіювання), а потім письмово відповідає на запитання (письмо).
2.  **Контекстуальність:** Граматика перевіряється не в відриві від контексту, а через завдання на розуміння текстів, написання власних висловлювань та аналіз ситуацій спілкування (Джерело: 6-klas-ukrmova-zabolotnyi-2020_s0228).
3.  **Продуктивність:** Перевага надається завданням, що вимагають від учня самостійно конструювати відповідь, а не просто вибирати з варіантів. Дослідники підтверджують, що "ефективніші тести — це ті, коли вам треба самим написати або сказати відповідь, а не вибирати варіант" (Джерело: ext-ulp_youtube-90).

## Ключові граматичні теми для тестування (Key Grammar Topics for Testing)

Іспит рівня А2 має перевіряти впевнене володіння наступними темами:

1.  **Фонетика і Орфографія:**
    *   **Наголос:** Розуміння ролі вільного і рухомого наголосу. Здатність розрізняти слова-омографи, значення яких залежить від наголосу (напр. `орган` (музичний інструмент) vs. `орган` (частина організму); `ніколи` (never) vs. `ніколи` (no time)). Знання слів з подвійним наголосом (`алфавіт`/`алфавіт`, `помилка`/`помилка`) (Джерело: ext-ulp_youtube-29).
    *   **Вживання м'якого знака (`ь`):** Правильне написання для пом'якшення приголосних та в дієслівних закінченнях (Джерело: 5-klas-ukrmova-avramenko-2022_s0131).

2.  **Іменник:**
    *   **Рід, число, відмінок:** Узгодження з іншими частинами мови.
    *   **Істоти/Неістоти:** Правильна постановка питання `хто?` чи `що?` та відповідна форма знахідного відмінка (Джерело: 3-klas-ukrainska-mova-kravtsova-2020-1_s0053).
    *   **Шість основних відмінків:** Впевнене використання іменників з найуживанішими прийменниками у знахідному, родовому, давальному, орудному та місцевому відмінках. Кличний відмінок для звертань.

3.  **Прикметник:**
    *   Узгодження з іменником за родом, числом і відмінком.
    *   Створення простих описів предметів та людей (Джерело: 5-klas-ukrmova-avramenko-2022_s0022).

4.  **Числівник:**
    *   Кількісні числівники (відповідають на питання `скільки?`) для позначення віку, часу, кількості предметів (Джерело: 3-klas-ukrainska-mova-ponomarova-2020-1_s0102).
    *   Узгодження числівників 1-4 з іменниками.

5.  **Займенник:**
    *   Правильне використання особових, присвійних, вказівних займенників.
    *   Розуміння потенційної двозначності при неправильному вживанні ("Дмитрик попрохав друга покласти зошит у свій портфель. (У чий портфель?)") (Джерело: 10-klas-ukrmova-karaman-2018_s0298).

6.  **Дієслово:**
    *   **Часи:** Теперішній (I та II дієвідміни), минулий (узгодження за родом та числом) та майбутній (проста і складена форми).
    *   **Особливі дієслова:** Відмінювання дієслів `бути`, `їсти`, `дати` та похідних від `-вісти` (`розповісти`, `відповісти`) є обов'язковим для перевірки (Джерело: 7-klas-ukrmova-avramenko-2024_s0084).
    *   **Наказовий спосіб** для простих команд та прохань.

## Частотність і пріоритети (Frequency & Priorities)

Іспит має зосереджуватися на найбільш функціональних та частотних аспектах граматики.

*   **Найвищий пріоритет:**
    1.  **Відмінки іменників з прийменниками:** Родовий (для позначення відсутності, власності), місцевий (для позначення місця) та знахідний (для позначення прямого об'єкта) є найважливішими для базового спілкування.
    2.  **Минулий час дієслів:** Узгодження за родом (`він сказав`, `вона сказала`, `воно сказало`) та числом (`вони сказали`) — критично важлива навичка.
    3.  **Відмінювання дієслів `бути` та `дати`:** Ці дієслова є основою для безлічі конструкцій.
*   **Середній пріоритет:**
    1.  **Теперішній час дієслів:** Розрізнення I та II дієвідмін.
    2.  **Наголос:** Перевірка знання наголосу в 10-15 найбільш уживаних словах з рухомим або подвійним наголосом (напр., `книжки`, `завжди`, `помилка`, `також`).
    3.  **Прикметники:** Узгодження в називному та знахідному відмінках.
*   **Нижчий пріоритет (але бажано включити):**
    1.  Складніші випадки відмінювання числівників.
    2.  Давальний та орудний відмінки у менш частотних конструкціях.

## Типові помилки L2 (Common L2 Errors)

Іспит має містити завдання, що виявляють типові помилки, зокрема ті, що виникають під впливом російської мови.

| ❌ Помилково | ✅ Правильно | Чому |
| :--- | :--- | :--- |
| **здавати** іспит | **складати** іспит | Калька з російського "сдавать экзамен". В українській мові використовується дієслово `складати`. (Джерела: 5-klas-ukrmova-avramenko-2022_s0131, 6-klas-ukrmova-avramenko-2023_s0026, 7-klas-ukrmova-avramenko-2024_s0084, 6-klas-ukrmova-avramenko-2023_s0078) |
| **вірна** відповідь | **правильна** відповідь | `Вірний` означає "loyal, faithful" (вірний друг). Для правильності використовується `правильний`. Калька з рос. "верный ответ". (Джерело: 6-klas-ukrmova-avramenko-2023_s0026) |
| **приймати** участь | **брати** участь | `Приймати` щось можна (подарунок), але участь `беруть`. Калька з рос. "принимать участие". (Джерело: 6-klas-ukrmova-avramenko-2023_s0026) |
| **учбовий** процес | **навчальний** процес | `Учбовий` — це суржикізм. Літературна норма — `навчальний`. (Джерела: 5-klas-ukrmova-avramenko-2022_s0131, 6-klas-ukrmova-avramenko-2023_s0078) |
| Говорити **на** українській мові | Говорити **українською** мовою | Вплив англійської ("speak **in** a language") або російської (" говорить **на** русском"). В українській мові для позначення мови мовлення використовується орудний відмінок. <!-- VERIFY --> |
| Я вибачаю**сь** | **Вибачте** / **Перепрошую** | Зворотна частка `-ся` означає дію, спрямовану на себе. "Я вибачаюсь" буквально означає "я пробачаю себе". Для вибачення перед іншими використовуються незворотні форми. <!-- VERIFY --> |

## Деколонізаційні застереження (Decolonization Notes)

**Це обов'язковий розділ.** Пробний іспит має активно утверджувати українські мовні норми та уникати пасток, пов'язаних з російським мовним впливом.

1.  **Пріоритет на унікальних українських формах:** Іспит повинен свідомо тестувати явища, що відрізняють українську від російської. Це стосується наголосу (`донька`, а не `дочка`), лексики (`складати іспит`), кличного відмінка.
2.  **Суржик як індикатор помилки:** Завдання на виправлення помилок мають включати типові русизми (`вірна відповідь`, `приймати участь`). Це не просто помилки, а маркери недостатнього занурення в українське мовне середовище.
3.  **Жодних порівнянь з російською:** В інструкціях та формулюваннях завдань слід уникати будь-яких згадок чи порівнянь з російською мовою. Українська мова є самодостатньою системою і має вивчатися як така.
4.  **Тестування лексичних "фальшивих друзів":** Завдання можуть включати слова, що мають різне значення в українській та російській мовах (напр., `неділя` — "Sunday" в українській, "week" в російській; `луна` — "echo" в українській, "moon" в російській).
