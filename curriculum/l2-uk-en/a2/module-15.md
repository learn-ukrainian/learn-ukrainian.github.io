# Summary

This module teaches **comparative adjectives** ‚Äî how to say something is bigger, better, faster, more expensive. You'll learn three formation methods and two comparison structures.

# Presentation: Comparative Adjectives

## üéØ Formation Method 1: Synthetic (-—à–∏–π/-—ñ—à–∏–π)

Add **-—à–∏–π** or **-—ñ—à–∏–π** to the stem:

| Positive | Comparative | Translation |
|----------|-------------|-------------|
| –Ω–æ–≤–∏–π | –Ω–æ–≤**—ñ—à–∏–π** | newer |
| —Å—Ç–∞—Ä–∏–π | —Å—Ç–∞—Ä**—ñ—à–∏–π** | older |
| –º–æ–ª–æ–¥–∏–π | –º–æ–ª–æ–¥**—à–∏–π** | younger |
| –¥–æ–≤–≥–∏–π | –¥–æ–≤**—à–∏–π** | longer |
| –∫–æ—Ä–æ—Ç–∫–∏–π | –∫–æ—Ä–æ—Ç**—à–∏–π** | shorter |
| —à–∏—Ä–æ–∫–∏–π | —à–∏—Ä**—à–∏–π** | wider |
| –≤—É–∑—å–∫–∏–π | –≤—É–∂**—á–∏–π** | narrower |
| –¥–æ—Ä–æ–≥–∏–π | –¥–æ—Ä–æ–∂**—á–∏–π** | more expensive |
| –¥–µ—à–µ–≤–∏–π | –¥–µ—à–µ–≤**—à–∏–π** | cheaper |
| —à–≤–∏–¥–∫–∏–π | —à–≤–∏–¥**—à–∏–π** | faster |
| –ø–æ–≤—ñ–ª—å–Ω–∏–π | –ø–æ–≤—ñ–ª—å–Ω**—ñ—à–∏–π** | slower |

> ‚ö° **Consonant Changes:** Before -—à–∏–π, some consonants change: –≥‚Üí–∂, –∫‚Üí—á, —Ö‚Üí—à. –¥–æ—Ä–æ–≥–∏–π ‚Üí –¥–æ—Ä–æ–∂—á–∏–π.

## üéØ Formation Method 2: Compound (–±—ñ–ª—å—à/–º–µ–Ω—à)

Use **–±—ñ–ª—å—à** or **–º–µ–Ω—à** + adjective:

| Compound | Translation |
|----------|-------------|
| **–±—ñ–ª—å—à** –≤–∞–∂–ª–∏–≤–∏–π | more important |
| **–º–µ–Ω—à** —Ü—ñ–∫–∞–≤–∏–π | less interesting |
| **–±—ñ–ª—å—à** –ø–æ–ø—É–ª—è—Ä–Ω–∏–π | more popular |
| **–º–µ–Ω—à** —Å–∫–ª–∞–¥–Ω–∏–π | less complicated |

> ‚ö° **When to Use:** Compound forms work for any adjective, especially longer ones or when emphasizing the degree.

| Positive | Comparative | Translation |
|----------|-------------|-------------|
| –¥–æ–±—Ä–∏–π | **–∫—Ä–∞—â–∏–π** | good ‚Üí better |
| –ø–æ–≥–∞–Ω–∏–π | **–≥—ñ—Ä—à–∏–π** | bad ‚Üí worse |
| –≤–µ–ª–∏–∫–∏–π | **–±—ñ–ª—å—à–∏–π** | big ‚Üí bigger |
| –º–∞–ª–∏–π | **–º–µ–Ω—à–∏–π** | small ‚Üí smaller |
| –≤–∏—Å–æ–∫–∏–π | **–≤–∏—â–∏–π** | tall ‚Üí taller |

## Comparison Structures

### Structure 1: X + comparative + –Ω—ñ–∂ + Y

| Example | Translation |
|---------|-------------|
| –ö–∏—ó–≤ **–±—ñ–ª—å—à–∏–π –Ω—ñ–∂** –õ—å–≤—ñ–≤. | Kyiv is bigger than Lviv. |
| –í–æ–Ω–∞ **—Å—Ç–∞—Ä—à–∞ –Ω—ñ–∂** —è. | She is older than me. |

### Structure 2: X + comparative + –∑–∞ + Accusative

| Example | Translation |
|---------|-------------|
| –ö–∏—ó–≤ **–±—ñ–ª—å—à–∏–π –∑–∞** –õ—å–≤—ñ–≤. | Kyiv is bigger than Lviv. |
| –í–æ–Ω–∞ **—Å—Ç–∞—Ä—à–∞ –∑–∞** –º–µ–Ω–µ. | She is older than me. |

> ‚ö° **Difference:** Both structures are correct. "–ù—ñ–∂" is more formal; "–∑–∞ + Acc" is more colloquial.
