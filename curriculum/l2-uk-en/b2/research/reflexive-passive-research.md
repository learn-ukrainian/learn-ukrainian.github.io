# Дослідження: Зворотний пасив із -ся

## State Standard Reference

§4.1.3.1 (lines 3297–3317): "Дійсний спосіб: дієвідмінювання дієслів недоконаного виду в теперішньому часі: зоставатися (зостаюся, зостаєшся, зостається…); дієвідмінювання дієслів недоконаного й доконаного виду в минулому часі…"

§4.1.3.1 mapping description (b2 block): "Indicative — complex aspect usage, passive voice (participles), pluperfect."

The standard does not use the label "зворотний пасив" explicitly. It covers imperfective -ся verb conjugation as part of §4.1.3.1 indicative mood, and this module's focus on imperfective -ся as morphological base for passive constructions is a valid curricular interpretation of that section.

**Alignment**: Module teaches imperfective -ся verb usage in passive function (процесна пасивна конструкція), directly serving the B2 indicative mood requirement. The aspect restriction (imperfective only) maps to §4.2.3 aspect pair formation (lines 3348–3350).

**Textbook corroboration** (Glazova Grade 11, §21, tier 2):
> "Засобом вираження пасивного стану є формотворчий афікс -ся, що додається до перехідних дієслів: будується, передається, продовжується. Українська мова уникає пасивних конструкцій. Тому перевагу, де це можливо, слід надавати дієсловам активного стану."

**Critical stylistic norm** (Zabolotnyi Grade 11, §, tier 2 — on -но/-то vs. passive):
> "Порушенням є введення в речення діяча у формі іменника або займенника в орудному відмінку" — cited for -но/-то forms. For -ся constructions, agent expression in Instrumental is grammatically permitted but stylistically constrained to contexts where the agent is essential for meaning.

## Common Learner Errors

1. **Agent preposition error** — `від` instead of bare Instrumental:
   - Error: *«Книга читається від студентів»*
   - Correct: *«Книга читається студентами»*
   - Why: Calque from English "by" or Polish "przez"; Ukrainian marks agent with bare Instrumental case (no preposition). The preposition `від` marks source/origin, not agent.

2. **Perfective aspect in -ся passive** (aspect mismatch):
   - Error: *«Будинок збудується» / «Задача розв'яжеться вченими»*
   - Correct: *«Будинок будується»* (ongoing process) or *«Будинок збудовано»* (-но/-то for completed result)
   - Why: Perfective -ся verbs in 3rd person are lexically reflexive or inchoative (*він збадьорився = he perked up*), not passive. The -ся passive construction is **exclusively imperfective** — this is a defining constraint of the construction.

3. **Register mismatch — using -ся passive in conversational speech**:
   - Error: *«Піца їсться мною»* / *«Цей фільм дивиться мною»*
   - Correct: *«Я їм піцу»* / *«Я дивлюся цей фільм»*
   - Why: Ukrainian -ся passive is a formal/written register marker. In spoken registers, -ся with animate subjects is typically lexically reflexive (self-directed action). Using it as passive in conversation sounds unnatural and bookish — textbooks explicitly note *«Українська мова уникає пасивних конструкцій»*.

## Notes for Content Writing

- **Lead with the Ukrainian-specific stylistic norm**: Ukrainian has a documented preference for active over passive voice. The -ся passive is a *marked* form used specifically in scientific, official-business, and journalistic registers. Frame this as a positive feature of Ukrainian stylistic design — Ukrainian prioritises clarity and directness — not as a limitation. Do not compare to any other language.

- **The -ся / -сь alternation rule**: -ся after consonants, -сь after vowels in personal endings. Examples: *будується* (3sg, ends in consonant) → -ся; *будуються* (3pl, ends in vowel) → -ся (exception — 3pl ending -ть + ся always keeps -ся). In practice for passive, only 3rd person forms appear: *будується / будуються*. The alternation matters more for 1st/2nd person reflexive forms which are NOT passive.

- **Aspect restriction is the module's central constraint**: Only imperfective -ся verbs function as passives. This should be demonstrated contrastively: *будується* (ongoing process, passive ✅) vs. *збудується* (will-be-built, NOT passive, inchoative or perfective completion sense only).

- **Agent in Instrumental is optional, not required**: In scientific texts the agent is often omitted entirely (*Результати аналізуються*). When the agent IS expressed, it goes in Instrumental without a preposition. This optional nature is a key B2 stylistic competency — knowing when to include or drop the agent.

- **Zabolotnyi (Grade 11) explicitly warns** against agent in Instrumental for -но/-то forms. Phase B content should clarify the asymmetry: -ся passive CAN take an Instrumental agent (though rarely); -но/-то forms in formal Ukrainian should NOT.

- **Puliui cultural hook is pedagogically stronger** than Sikorsky for this module because his work involves scientific process descriptions (*досліджується, встановлюється, вимірюється*) which directly model the target grammar. Ющенко provides the programming/tech register (*розробляється, впроваджується, застосовується*). Use both.
