## State Standard Reference

**§4.1.1.3 Числівник** (lines 3131–3149, B2 рівень):

> «відмінювання числівників на позначення часу: о дванадцятій годині, пів на першу, двадцять [хвилин] по вісімнадцятій [годині]; відмінювання числівників для позначення дати: перше вересня, першого вересня, після двадцять восьмого лютого; відмінювання родових форм порядкових числівників: тисячний (тисячного, тисячному, тисячний / тисячного, тисячним, на тисячному / на тисячнім...), тисячна (тисячної, тисячній, тисячну, тисячною, на тисячній...); відмінювання складених порядкових числівників: сто третій (сто третього, сто третьому, сто третій / сто третього, сто третім, на сто третьому / на сто третім...), сто третя (сто третьої, сто третій, сто третю, сто третьою, на сто третій...); відмінювання кількісних числівників один (одна, одне, одні), два (дві), три, чотири, від 5 до 20, 30, 50, 60, 70, 80, 40, 90, 100: двадцяти (двадцятьох) восьми (вісьмох), без сорока одного, шістдесятьома (шістдесятьма) двома, на чотирнадцятьох, вісімдесятьома (вісімдесятьма) чотирма, до ста семи (сімох).»

**Alignment**: The module covers all three Standard requirements directly:
1. Time declension (пів на першу, по вісімнадцятій) — Section 2
2. Date declension (першого вересня, після двадцять восьмого лютого) — Section 3
3. Compound ordinal declension (у дві тисячі двадцять шостому році) — Sections 3–4

The Standard explicitly provides the exact preposition constructions used in this module (по, пів на, за + ordinal) confirming full scope alignment.

## Cultural Hooks

1. **Etymological month names as living history** — Ukrainian month names are derived from natural phenomena of the agricultural calendar, not Latin saints' days. This is documented in Slavic comparative linguistics (Wikipedia: Слов'янські назви місяців). Key etymologies: **січень** (від сікти — the frost "cuts"/bites); **лютий** (від лютий — fierce cold); **березень** (від береза — birch, sap-rising time); **квітень** (від квіти — blossoming); **травень** (від трава — grass growth); **червень** (від черв — dye-worm/red berries); **липень** (від липа — linden blooming); **серпень** (від серп — sickle, harvest); **вересень** (від верес — heather blooming); **жовтень** (від жовтий — yellowing leaves); **листопад** (from листя падає — leaf-fall); **грудень** (від груди — frozen clods of earth). This system is shared with Belarusian, Polish, Czech, and Croatian — not Russian and Bulgarian, which use Latin borrowings. This is an ideal mnemonic anchor for genitive case forms since the month names encode seasonal logic.

2. **The Slavic time-naming heritage vs. Latin-loan systems** — Ukrainian, along with Polish, Czech and Croatian, preserves the Proto-Slavic natural-observation naming system (reconstructed by V. Shaur). Lithuanian shares partial cognates, possibly borrowed from Old Ukrainian when it was the official language of the Grand Duchy of Lithuania. This underscores Ukrainian's deep, independent linguistic tradition — contrasting with languages that adopted Latin month names wholesale.

## Common Learner Errors

1. **"Скільки годин?" замість "Котра година?"** → Correct: «Котра година? — Друга.» — This is a direct calque from the Germanic/Russian cardinal question ("Сколько времени?" / "What time is it?"). Ukrainian uses an ordinal-based system (порядковий числівник) for clock-time. The textbook source (Авраменко Grade 6, tier 1) makes this distinction explicitly. English speakers also tend to ask "how many hours?" and must learn the ordinal framing.

2. **Accusative instead of genitive for dates** → Error: «народився *перше* травня» (called знахідний відмінок error); Correct: «народився *першого* травня». The Nominative form (перше) is only used for stating a date in isolation ("Сьогодні перше."), never as a temporal-when answer. This is confirmed by State Standard §4.1.1.3: «першого вересня» as the canonical genitive-of-time form.

3. **"Без десяти сьома" instead of "За десять сьома"** → This is a Russicism modeled on «без десяти семь». Ukrainian normative usage for "before the hour" uses «за» (not «без»): «за десять (хвилин) сьома» or «десять до сьомої». The plan section 5 lists this as a focus item; the textbook source (Litvinova Grade 6, tier 1, p. 246) has an exercise explicitly correcting this error pattern.

4. **Conflating термін and строк in official register** → «термін» = specific deadline/date (термін подачі документів — deadline for submission); «строк» = duration/period (строк дії договору — contract duration). Both are high-frequency (73 and 51 IPM respectively) but used in different legal-document contexts.

## Notes for Content Writing

- **Ordinal-as-adjective principle**: Ordinal numerals decline like adjectives (hard type, except третій = soft type). Compound ordinals: only the LAST element changes (двісті двадцять п'ята → двісті двадцять п'ятої). This is the core structural insight to build around.
- **Time prepositions table is essential**: Content must include the systematic contrast: `по + Locative` (past the hour), `за + Accusative` (before the hour), `пів на + Accusative` (half past), `на + Accusative` (approaching). This is directly mandated by §4.1.1.3 examples.
- **Official format "14:30"**: The 24-hour spoken format (чотирнадцята тридцять) used in news, protocols, and transport schedules must be distinguished from colloquial (пів на третю). B2 register awareness is essential here.
- **Genitive month forms**: All month names are needed in genitive singular (місяця) — verify all 12: січня, лютого, березня, квітня, травня, червня, липня, серпня, вересня, жовтня, листопада, грудня. These have been VESUM-verified (вересня ✅ → вересень/noun; березня ✅ → березень/noun; лютого ✅ → лютий/adj-substantivized).
- **Decolonized framing**: Month name etymology section should be framed as "Ukrainian preserves the Proto-Slavic agricultural calendar system, shared with Polish, Czech, and Croatian" — not as contrast with Russian. The shared Slavic heritage is the positive story.
- **Word budget note**: Section 2 (1000 words) carries the heaviest grammatical load; ensure the preposition table + ordinal paradigm + colloquial vs. official contrast all have sufficient prose explanation to meet the section target.
