# Дослідження: Синоніми: Якість та Оцінка

## State Standard Reference
§4.4.1.2: "Стилістичні засоби лексики: ... синоніми: говорити, розмовляти, спілкуватися; ... епітети: теплий день, смачний торт;"
Alignment: This module fulfills the B2 stylistic requirement for lexical variety by expanding synonymous sets and epithets specifically for evaluative adjectives, enabling learners to express nuanced positive and negative evaluations.

## Vocabulary Frequency
| Word | Frequency / Source | Key collocations |
|------|-------------------|------------------|
| хороший | High / Everyday | хороший фахівець, хороший результат, хороший настрій |
| добрий | High / Moral & utility | добра людина, добрий урожай, добрий намір |
| гарний | High / Aesthetic | гарна дівчина, гарний краєвид, гарна сукня |
| чудовий | Mid / Emotive praise | чудовий день, чудова новина, чудова нагода |
| кепський | Mid / Colloquial negative | кепські справи, кепський жарт, мати кепський вигляд |
| огидний | Low / Strong negative | огидний вчинок, огидний смак, огидний запах |
| старовинний | Mid / Historical | старовинний замок, старовинна монета |

## Cultural Hooks
1. "Добре ім'я — найкраще багатство" (A good name is the best wealth) — emphasizes that the concept of "добрий" is deeply rooted in morality and reputation in Ukrainian culture.
2. "Добро не лихо, ходить тихо" (Goodness is not evil, it walks quietly) — reflects the Ukrainian cultural value of unassuming, modest quality, distinguishing it from boastful excellence.

## Common Learner Errors
1. Visual vs Functional: Using "гарний" (handsome/beautiful) to describe professional competence (e.g., «Він гарний студент») instead of "хороший" (diligent/skilled). → Correct: «Він хороший студент».
2. Emotion vs Quality: Using "злий" (angry/evil) to describe bad objects or situations (e.g., «злий фільм») instead of "поганий". → Correct: «поганий фільм». "Злий" applies to character or mood.

## Notes for Content Writing
- Decolonized framing: Ukrainian has a sophisticated division of evaluation («добрий» vs «хороший» vs «гарний») that differentiates aesthetic, functional, and moral goodness. Emphasize this richness instead of defaulting to a single catch-all translation. "Хороший" is a fully legitimate Ukrainian word with a specific functional scope, not a Russianism.
- Strict 100% immersion: Explain nuanced synonyms using Ukrainian definitions and contexts, without English scaffolding.
- The tone should be a Professional Language Coach with a QA Engineer flavor — precise, analytical, and structured.
