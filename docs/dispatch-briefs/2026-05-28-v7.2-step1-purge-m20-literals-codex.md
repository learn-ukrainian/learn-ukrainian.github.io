## New regression test

Create `tests/build/test_writer_prompt_no_lesson_specific_leakage.py`:

```python
"""Assert that the rendered writer prompt for lesson A does not contain
lesson B's signature vocabulary, textbook author, or chunk_id."""

import pytest
from pathlib import Path
from scripts.build.linear_pipeline import _render_writer_prompt_for_module

@pytest.mark.parametrize("self_slug, other_slug", [
    ("my-morning", "letters-and-sounds"),  # or whichever other A1 lesson exists
    # add more pairs as A1 grows
])
def test_no_cross_lesson_token_leakage(self_slug, other_slug):
    self_prompt = _render_writer_prompt_for_module(level="a1", slug=self_slug)
    other_specific = _extract_lesson_specific_tokens(level="a1", slug=other_slug)
    for tok in other_specific:
        assert tok not in self_prompt, (
            f"Lesson '{other_slug}' specific token '{tok}' leaked into "
            f"'{self_slug}' rendered writer prompt"
        )
```

You'll need to implement `_extract_lesson_specific_tokens(level, slug)` — it reads `wiki/pedagogy/{level}/{slug}.md` + `curriculum/l2-uk-en/plans/{level}/{slug}.yaml` and returns: vocabulary minimum lemmas, textbook author names from plan.references, chunk_ids from plan.references[*].notes, decolonization contrast pairs from wiki.

Place the helper alongside the test (or in `tests/build/_helpers.py`).
