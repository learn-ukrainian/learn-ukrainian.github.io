## What to build

A linter that scans dispatch briefs (`docs/dispatch-briefs/**/*.md`) for `.venv/bin/python` invocations that lack a `cd <main-checkout-path>` immediately before, OR documented worktree-venv-symlink. Catches the bug class that bit on 2026-05-07 (Codex worker hit "ModuleNotFoundError" because it was in a worktree without `.venv` symlinked).

## Acceptance criteria (numbered, all required)

1. **New script** `scripts/audit/lint_dispatch_brief.py` with:
   - `argparse` CLI: `--brief PATH` (required) or `--all` (scan all `docs/dispatch-briefs/**/*.md`)
   - Detection rule: any line containing `.venv/bin/python` MUST be preceded (within 5 lines, same fenced block) by either:
     - `cd ` to a path that contains `learn-ukrainian` and does NOT contain `.worktrees/`, OR
     - A line documenting the worktree's `.venv` is symlinked (regex like `ln -s .*\.venv` or comment `# venv symlinked`)
   - Output: per-violation report `path:line: missing cd-to-main or symlinked-venv before .venv/bin/python`
   - Exit 0 if clean, 1 if any violations
   - Conforms to `cli-help-standard.md` (description, epilog with 2-3 examples, arg help, exit codes, related)
2. **Tests** in `tests/audit/test_lint_dispatch_brief.py`:
   - 2 fixture briefs that should PASS (one with proper `cd`, one with documented venv symlink)
   - 2 fixture briefs that should FAIL (one with bare `.venv/bin/python`, one with `.venv/bin/python` after `cd .worktrees/...`)
   - Assert correct exit codes
3. **Pre-commit hook** in `.pre-commit-config.yaml` — local hook running the linter on staged `docs/dispatch-briefs/*.md` files. Read the existing config first to match style.
4. `docs/SCRIPTS.md` updated with the new script (one-line entry under audit section).
5. `ruff check` clean on touched files.
6. All existing tests pass: `.venv/bin/pytest tests/audit/ -x` (don't run the full suite, just the audit subset to keep it fast).

## Why this matters (one line)

If a brief tells the worker to run `.venv/bin/python` while in a worktree without `.venv` symlinked, the worker explodes with import errors and burns dispatch budget. Detect at brief-write time, not at dispatch time.
