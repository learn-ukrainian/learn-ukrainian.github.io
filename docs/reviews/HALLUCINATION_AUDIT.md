# Hallucinated & Suspicious Proverbs Audit

This report lists "proverbs", "sayings", or "folk wisdom" found in the curriculum that appear to be:
1.  **AI Hallucinations:** Invented phrases that sound authoritative but have no basis in real Ukrainian folklore.
2.  **Pedagogical Fabrications:** Phrases created to teach a specific grammar rule but presented as "ancient wisdom".
3.  **Misattributed Slogans:** Modern slogans or literary quotes presented as "folk proverbs".
4.  **Suspicious Modifications:** Real proverbs altered to fit a lesson theme (e.g., changing "tongue" to "language").
