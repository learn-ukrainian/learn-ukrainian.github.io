# B2 State Standard 2024 Gap Analysis

**Result: 6 genuine gaps, 5 partial gaps, and significant "extra" coverage (good).**

## 2. GAPS: State Standard topics NOT covered by any B2 plan

### GAP 1: Adjective/Adverb Comparison — Expanded Forms (SS 4.2.1, 4.2.2)
- **State Standard requires**: "Expanded comparison — irregular forms, stylistic choices" for adjectives AND "Adverb comparison — irregular, compound forms"
- **What B2 has**: NOTHING dedicated. The `word-formation-adjective-formation` plan mentions ступені порівняння in passing (one subsection line), and `word-formation-adverbs-integration` covers прислівники міри та ступеня but NOT comparison degree formation.
- **Severity**: HIGH. Irregular comparative/superlative forms (кращий/найкращий, гірший, більший, менший, вищий, нижчий) and the analytic forms (більш складний, менш важливий, найбільш значущий) are core B2 morphology. Ukrainian has unique patterns here (e.g., no "more" calques for synthetic comparatives).
- **Recommendation**: Dedicated module `adjective-adverb-comparison` or split into two.

### GAP 2: Prefixed Motion Verbs — Full System (SS 4.2.4)
- **State Standard requires**: "Prefixed motion verbs — full system with all prefixes"
- **What B2 has**: ZERO coverage. No plan mentions дієслова руху, verbs of motion, іти/ходити/їхати/їздити, or prefix systems.
- **Severity**: HIGH. This is one of the hardest topics in Ukrainian grammar. The full prefix system (ви-, за-, при-, від-, пере-, по-, про-, об-, під-, роз-, з-) applied to motion verbs with determinate/indeterminate base pairs is critical for B2.
- **Recommendation**: Dedicated module `prefixed-motion-verbs` covering all prefix+base combinations, determinate vs indeterminate, figurative extensions.

### GAP 3: Full Pronoun System (SS 4.1.1.4)
- **State Standard requires**: "Full pronoun system — reflexive, reciprocal, definitive (сам, самий, весь, кожний)"
- **What B2 has**: Pronouns appear incidentally in other modules (e.g., direct-indirect-speech covers pronoun shifts) but NO dedicated pronoun module. The reflexive pronoun себе, reciprocal один одного, and definitive pronouns сам/самий/весь/кожний/інший are not systematically taught.
- **Severity**: MEDIUM-HIGH. While some pronoun usage is implicit throughout, the systematic paradigm (especially the difference between сам vs самий, reflexive vs emphatic use of себе, кожний vs кожен) needs explicit treatment.
- **Recommendation**: Dedicated module `pronoun-system-advanced`.

### GAP 4: Advanced Noun Declension (SS 4.1.1.1)
- **State Standard requires**: "Advanced noun declension — rare forms, archaic endings, foreign noun adaptation"
- **What B2 has**: `advanced-case-semantics` covers case MEANINGS but not declension FORMS. No module covers: невідмінювані іменники (metro, кіно, кафе), іншомовні іменники (Рерасу, Мехіко), archaic endings (-ові/-еві in dative), second declension -у/-ю vs -а/-я genitive alternation at B2 level.
- **Severity**: MEDIUM. Some of this is covered at B1 level, but the State Standard explicitly calls for advanced forms at B2.
- **Recommendation**: Either expand `advanced-case-semantics` or add `advanced-noun-declension`.

### GAP 5: Zero-Suffix Deverbal Nouns (SS 4.2.7)
- **State Standard requires**: "Zero-suffix deverbal nouns (переїзд, розгляд)"
- **What B2 has**: NOTHING. The word-formation modules cover -ння/-ення nouns, person suffixes, place names, adjective formation, adverbs — but not zero-suffix derivation.
- **Severity**: LOW-MEDIUM. This is a productive pattern in Ukrainian (переїзд, розгляд, виклик, запис, вхід, вихід, напад, опис) important for formal/academic register.
- **Recommendation**: Add a subsection to `word-formation-abstract-nouns` or create a small focused module.

### GAP 6: Collective and Fractional Numerals (SS 4.1.1.3 — partial)
- **State Standard requires**: "Full numeral paradigm — compound, COLLECTIVE, FRACTIONAL with case agreement"
- **What B2 has**: `numeral-declension-compound-numbers` covers compound cardinals (50-900, тисяча, мільйон). `numeral-declension-time-dates` covers time/dates. But COLLECTIVE numerals (двоє, троє, четверо, п'ятеро) and FRACTIONAL numerals (одна друга, три четвертих, півтора) are not explicitly covered.
- **Severity**: MEDIUM. Collective numerals have unique agreement rules (двоє хлопців, but NOT двоє дівчат) and fractional numerals require different case patterns.
- **Recommendation**: Expand `numeral-declension-compound-numbers` to include collective and fractional sections, or add a third numeral module.

## 3. PARTIAL GAPS: Covered but insufficiently

### PARTIAL 1: Vocative stylistic usage (SS 4.1.2.7)
- State Standard specifies "stylistic usage, archaic forms, literary vocative"
- `advanced-case-semantics` covers all 7 cases but may not dedicate sufficient space to vocative literary/archaic forms (Боже мій, друже, козаче). Should be verified when the module is built.

### PARTIAL 2: Aspect pairs — suppletive (SS 4.2.3)
- State Standard requires suppletive aspect pairs (брати/взяти, класти/покласти, говорити/сказати)
- `aspect-nuances-secondary-imperfectivization` covers suffix-based and prefix-based pairs but suppletive pairs are a different mechanism. Likely assumed from B1 but should be verified.

### PARTIAL 3: Interrogative sentence types (SS 4.3.1.2)
- State Standard requires "deliberative, rhetorical, embedded" questions
- Rhetorical questions are in `syntactic-stylistic-devices`. Embedded questions may appear in `multi-clause-sentences`. But deliberative questions (internal monologue questions) may not be explicitly covered.

### PARTIAL 4: Complex sentence clause types (SS 4.3.4) — temporal, concessive
- State Standard lists: "attributive, explanatory, temporal, causal, conditional, concessive clauses"
- `multi-clause-sentences` and `advanced-conjunctions-i/ii` cover these, but temporal (поки, доки, коли, щойно) and concessive (хоча, дарма що, незважаючи на те що) clauses should be verified as explicitly taught, not just mentioned.

### PARTIAL 5: Thematic coverage gaps
- State Standard themes NOT directly addressed by a dedicated B2 module: **спорт**, **подорожі**, **здоров'я**, **харчування**, **купівля**, **послуги**, **місця**, **природа**, **дім/побут**
- These are likely covered at A1-B1 levels and assumed as known at B2, but the State Standard lists them for B2 as well (with higher complexity expected). The B2 modules focus heavily on professional, academic, and cultural content.
- **Assessment**: Not a true gap for a curriculum that builds progressively. The B2 thematic modules (tekhnolohii, nauka, mystetstvo, modern-diaspora, religion) add NEW domains appropriate for B2.

## 4. EXTRA: B2 plans beyond the State Standard

These modules go beyond what the State Standard requires at B2. This is generally positive — they provide practical skills and cultural knowledge.

### Register system (9 modules + checkpoint)
The State Standard does not explicitly require register analysis at B2 — it's more of a C1 topic. But the B2 curriculum dedicates 9 modules to it:
- `register-introduction`, `register-formal-informal`, `register-business-ukrainian`, `register-academic-technical`, `register-official-legal`, `register-literary-ukrainian`, `register-media-journalistic`, `register-colloquial-style`, `register-religious-ukrainian`, `register-practice-cross-register-rewriting`
- **Assessment**: This is a GOOD pedagogical choice. Register awareness is critical for practical language use, and introducing it at B2 prepares learners for C1 mastery. Not a problem.

### Professional communication block (10 modules)
- `professional-email-basics/advanced`, `professional-reports-basics/advanced`, `academic-writing`, `presentation-skills-basics/advanced`, `discussion-debate`, `news-analysis-basics/advanced`
- **Assessment**: Practical and valuable. Goes beyond pure linguistic competence into communicative competence.

### Passive voice depth (6 modules + checkpoint)
- The State Standard lists passive in one line under verb forms. B2 dedicates 6 full modules to different passive strategies.
- **Assessment**: This depth is warranted because Ukrainian passive works fundamentally differently from English passive, and learners need this exposure.

### Phraseology/idioms block (6 modules)
- `proverbs-work-wisdom-character`, `proverbs-nature-time-caution`, `set-expressions-combined`, `idioms-somatic`, `idioms-animals`, `idioms-nature`
- **Assessment**: The State Standard mentions phraseology at B2 implicitly. This dedicated block is valuable for natural language use.

### Cultural/thematic modules (5 modules)
- `tekhnolohii-ta-shi`, `nauka-i-doslidzhennia`, `mystetstvo-i-literatura`, `modern-diaspora`, `religion-in-ukraine`
- **Assessment**: These address the State Standard's thematic catalogue while building domain vocabulary.

## 5. Assessment Format Requirements

The State Standard (Section 1, lines 2452-2873) specifies these speech activities for B2:

| Activity | State Standard Requirement | B2 Plan Coverage |
|---|---|---|
| **Listening** (аудіювання) | Understand extended speech, lectures, complex argumentation | `listening_situations` in most plans |
| **Reading** (читання) | Read articles, reports, literary prose, understand implicit attitudes | `reading_situations` in most plans |
| **Writing** (письмо) | Write clear detailed text on complex subjects, essays, reports, letters | `writing_tasks` in most plans; professional email/reports/academic writing modules |
| **Speaking** (говоріння) | Present clear detailed descriptions, explain viewpoints, debate | `discussion_topics` in most plans; presentation/debate modules |

**Assessment formats are well-integrated into the plan structure** through the `reading_situations`, `listening_situations`, `writing_tasks`, and `discussion_topics` fields. The State Standard does not prescribe specific activity types beyond these four skills.

## 6. Priority Recommendations

Ranked by pedagogical impact:

1. **HIGH PRIORITY**: Add `prefixed-motion-verbs` module (SS 4.2.4) — this is a major gap for a topic that is notoriously difficult and explicitly required
2. **HIGH PRIORITY**: Add `adjective-adverb-comparison` module (SS 4.2.1-4.2.2) — core morphology missing entirely
3. **MEDIUM-HIGH PRIORITY**: Add `pronoun-system-advanced` module (SS 4.1.1.4) — systematic pronoun teaching needed
4. **MEDIUM PRIORITY**: Expand numeral modules to cover collective/fractional numerals (SS 4.1.1.3 partial)
5. **MEDIUM PRIORITY**: Add advanced noun declension content (SS 4.1.1.1) — either new module or expand existing
6. **LOW-MEDIUM PRIORITY**: Add zero-suffix deverbal nouns content (SS 4.2.7) — can be added to existing word-formation module

Adding 3 new modules (motions verbs, comparison, pronouns) and expanding 2 existing ones (numerals, word-formation-abstract-nouns) would close all gaps. Total module count would go from 89 to 92.
