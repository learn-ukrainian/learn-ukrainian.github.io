# Plan: Restructure A1 Track — State Standard 2024 Compliance

## Context

Gemini adversarial review of the l2-uk-en A1 track found **20 gaps** vs the Ukrainian State Standard 2024, **3 compressed modules**, and **3 ordering problems**. The l2-uk-direct track (PoC) already addresses these gaps with a 44-module design, but the user wants l2-uk-en fixed to the same standard.

**Goal**: Restructure A1 from 44 → 64 modules. Fill all State Standard gaps, decompress overloaded modules (including letters), fix ordering.

- `fill-in`, `match-up`, `quiz`, `true-false`, `anagram`, `unjumble`

## Implementation Steps

### Step 1: Create GH issue

Track this restructuring as a major task under #667 (pipeline hardening).

### Step 2: Update `curriculum/l2-uk-en/a1/curriculum.yaml`

Rewrite with new 64-module sequence. This is the source of truth for module ordering.

### Step 3: Update A1 activity schema

Add `watch_and_repeat`, `classify`, `image_to_letter` to `schemas/activities-A1.schema.json` (import from activities-direct.schema.json). These are used ONLY in Cyrillic modules 1-4.

### Step 4: Create 26 new plan files

In `curriculum/l2-uk-en/plans/a1/`:
- 14 new topic module plans + 1 utility (word_target: 2000)
- 4 new checkpoint plans (word_target: 1500)
- 2 Cyrillic split plans (word_target: 2000, with Anna Ohoiko video refs)
- 6 other split plans (word_target: 2000)

Each plan follows existing schema: module, level, sequence, version, title, focus, pedagogy, objectives, content_outline, vocabulary_hints, activity_hints, prerequisites, connects_to.

### Step 5: Update existing plan files

- Update `sequence` and `module` fields in all 36 kept plan files
- For 6 expanded modules, add new content points to `content_outline`
- For 2 existing Cyrillic plans, update to split scope (А-Ж and З-Н) + add Anna Ohoiko video refs

### Step 6: Remove deprecated plan files

- `food-and-shopping.yaml` → replaced by food-vocabulary + shopping-and-market
- `can-must-want-modals.yaml` → replaced by can-and-know-how + must-and-want
- `prepositions-direction-origin.yaml` → replaced by prepositions-of-place + direction-and-origin
- `at-the-cafe-practical.yaml` → dropped (redundant with at-the-cafe)
- `checkpoint-navigation.yaml` → replaced by checkpoint-sentences
- `checkpoint-core-grammar.yaml` → replaced by checkpoint-daily-life

### Step 7: Build new modules via pipeline

```bash

  syllables-and-transfer stress-and-intonation greetings-and-politeness \

  imperative-and-requests leisure-and-hobbies travel-and-transport \
  prohibitions-and-signs writing-skills euphony-and-polish \
  checkpoint-sentences checkpoint-cases checkpoint-daily-life checkpoint-communication \
  prepositions-of-place direction-and-origin food-vocabulary shopping-and-market \
  can-and-know-how must-and-want; do
  .venv/bin/python scripts/build_module.py a1 "$slug" --stop-before D
done
```
