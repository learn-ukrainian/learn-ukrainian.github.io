# Research: Ukrainian Bukvar Letter Introduction Sequences

## 1. Zaharijchuk Bukvar (2025, NUS Tier 1) — Letter Order

The Zaharijchuk bukvar is split into 2 parts. Part 1 covers vowels first (without consonant pairing), then introduces consonants one-by-one with immediate syllable/word reading. Part 2 continues with remaining consonants.

### Pre-letter phase (pp. 1-33)
Pages 1-33 contain pre-reading activities: sound awareness, syllable structure, sentence/word concepts. NO letters are formally introduced. Children learn to identify sounds orally.

| 2 | **А** | 36 | "Бачу А, а. Чую [а]." | Second vowel. Still no consonants = no syllable reading yet. |

## 4. Key Findings

### 4.1 How many letters before sentences?

**Zaharijchuk**: First recognizable words appear with М (letter #6, ~p.46): "ма-ма". First exclamatory quasi-sentences appear with Н (letter #7): "На!" "Ні!" First pronouns with В (letter #8): "він, во-на, во-но, во-ни." First multi-word sentences with К (letter #11, ~p.72): "Ку-кі і Ва-ко і-ме-ни-ни." So roughly **11 letters (5 vowels + 6 consonants) before real sentences**.

**Bolshakova**: First words with М (letter #3, p.14): "ма-ма, ма-мо, ма-ла." First sentences with С (letter #7, p.22): "У нас -- ананас." So roughly **7 letters (2 vowels + 5 consonants) before sentences**. Bolshakova is faster to sentences because it interleaves vowels and consonants from the start.

### 4.2 Why both use О early

**Zaharijchuk puts О FIRST** (before А!). This is striking because most curricula start with А. The reason: О is the **highest-frequency vowel in Ukrainian text** and appears in the most common words (молоко, око, вода, школа, дім/дом). Also, О in Ukrainian is stable (never reduces to [а] like in Russian), so it's pedagogically clean. Starting with О reinforces Ukrainian phonetic identity from day one.

**Bolshakova puts О second** (after А), which is the more traditional order, but still very early.

### 4.3 The vowel-first strategy (Zaharijchuk)

Zaharijchuk introduces ALL 5 basic vowels (О, А, У, И, І) BEFORE the first consonant. This is pedagogically significant:
- When М arrives (letter #6), students can immediately form syllables with all 5 vowels: ма, мо, му, ми + ам, ом, ум, им
- This creates an explosion of readable syllables with each new consonant
- By contrast, Bolshakova interleaves (А, О, М, Л, У, Н...) so each consonant initially pairs with only 2 vowels

### 4.4 Words used for syllable/word reading drills at each stage

**Stage 1 (М only, ~6 letters known):**
- Zaharijchuk: ма-ма, мо-ма, ми-ла, му-ла (syllable combos)
- Bolshakova: ма-ма, ма-мо, ма-ла, ма-ло

**Stage 2 (+Н, ~7 letters):**
- Zaharijchuk: На! Ні! ни-ми, на-ми
- Bolshakova: ан, он, ун, лу-на, нам

**Stage 3 (+Л/В, ~8-9 letters):**
- Zaharijchuk (+В): він, во-на, во-но, во-ни
- Bolshakova (+Л then +У): ала, ало, мала, мало, лама, ум, ул

**Stage 4 (+С, ~10 letters):**
- Zaharijchuk: оса, со-ва, са-ми, мас
- Bolshakova: сама, насос, сом, оса, смола, ананас, сон, сосна. FIRST SENTENCES: "У нас -- ананас."

**Stage 5 (+К, ~11 letters):**
- Zaharijchuk: ко-ник, кі-но, Лі-ка, Ні-ко, Ві-ка. First sentences.
- Bolshakova: око, коса, коло, колос, корова

**Stage 6 (+И/Р, ~12-13 letters):**
- Bolshakova (+И): соми, сани, малина, лимони, миски, калина, колиска, килими, сосиска. "У Клима -- лимони. У Килини -- малина."
- Bolshakova (+Р): кора, рано, нора, ранок, рак, краб, робот

## 5. Comparison with Current Code / Plans

### Current plan letter grouping (the-cyrillic-code-i through iv):
- **M1**: А, М, Л, У, Н, С (6 letters)
- **M2**: К, И, Р, Б, В, Д, І + О (8 letters, total 14)
- **M3**: П, Т, Г, Ґ, Е, З, Ж, Ш, Х (9 letters, total 22/23)
- **M4**: Й, Ч, Щ, Я, Ю, Є, Ь, Ї, Ц, Ф + ДЖ, ДЗ, ' (remaining)

### Current decodability charsets in code:
- M1: `АаМмЛлУуНнСс` (6 letters: А, М, Л, У, Н, С)
- M2: `АаМмЛлУуНнСсТтОоКкИиВвРрЕеІі` (14 letters: +Т, О, К, И, В, Р, Е, І)
- M3: `АаМмЛлУуНнСсТтОоКкИиВвРрЕеІіДдПпЗзБбГгХхЖжШшЧч` (23 letters: +Д, П, З, Б, Г, Х, Ж, Ш, Ч)

### Discrepancies between plans and code:
1. **Plans say M2 adds "К, И, Р, Б, В, Д, І + О"** but the **code charset for M2** is `Тт Оо Кк Ии Вв Рр Ее Іі` -- it includes **Т and Е** but NOT Б and Д. The M2 plan and M2 code disagree: plan has Б, Д; code has Т, Е.
2. **Plans say M3 adds "П, Т, Г, Ґ, Е, З, Ж, Ш, Х"** but the **code charset for M3** adds `Дд Пп Зз Бб Гг Хх Жж Шш Чч` -- it includes **Д, Б, Ч** but NOT Ґ. Also, Т and Е are already in M2 code but M3 plan. The plan and code are out of sync.

### How textbook data could improve the grouping:

The textbook evidence suggests:
- **О should move to M1** (both textbooks use it extremely early, Zaharijchuk makes it letter #1)
- **М, Н, Л, С, К** are the consensus first consonant group in both textbooks
- **В** is introduced very early in Zaharijchuk (letter #8) and could be M1 or early M2
- **Т** comes before **Е** in both textbooks; the code correctly places Т in M2
- **Б, Д** come later than the current M2 plan suggests; the code correctly defers them to M3

### A textbook-aligned M1 charset could be:
**О, А, У, И, І + М, Н, Л, С** = 9 letters (5 vowels + 4 consonants)
- This matches the Zaharijchuk vowel-first approach
- Decodable words: мама, нас, сам, луна, сума, він (if В added), масло, мала, мила, лимон, малина, ніс, сон, сом, лис

Or keep current M1 as 6 letters (А, М, Л, У, Н, С) but **add О** to make it 7 — which massively expands decodable vocabulary:
- +О enables: молоко, масло, сало, слово, мало, мола, сом, сон, лом, лоно, сум, лосось (most Ukrainian words need О)
