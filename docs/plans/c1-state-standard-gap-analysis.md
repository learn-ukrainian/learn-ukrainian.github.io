## 7. Summary

### Strengths
- **Syntax coverage is excellent**: 6 dedicated modules for complex sentence types, matching SS 4.3 precisely
- **Stylistics coverage is excellent**: 9 modules covering all SS 4.4 categories (phonetic, lexical, morphological, syntactic, phraseological)
- **Cultural depth far exceeds SS**: 33 modules of folk culture + performing arts + sociolinguistics
- **Professional skills well covered**: academic writing, presentations, career skills
- **Plan file hygiene**: 100% slug alignment between curriculum.yaml and plan files, 100% have references

### Weaknesses
- **4 critical/high gaps**: aspect pairs, conditional mood, pronoun paradigm, sport theme
- **5 moderate gaps**: agent nouns, abstract nouns, zero-suffix deverbals, narrative present, consumer vocabulary
- **Systemic: zero activity_hints** across all 111 plans
- **2 plan quality bugs**: `literature-review` and `research-article` missing `writing_tasks`
- **Word formation coverage thin**: SS 4.2 has 7 sub-requirements, only comparison and nominalization have explicit plans

### Priority Actions
1. **Add 2 modules**: `aspect-pairs-mastery-c1` and `conditional-mastery-c1` (critical linguistic gaps)
2. **Expand `morfolohichna-norma-c1`**: add dedicated pronoun paradigm section or create standalone module
3. **Add `activity_hints`** to all 111 plans before C1 build begins
4. **Fix `writing_tasks`** for `literature-review` and `research-article`
5. **Address sport/health themes** in existing or new modules
6. **Expand word formation coverage** in `leksykolohiia-c1` to cover agent nouns, abstract nouns, zero-suffix deverbals
