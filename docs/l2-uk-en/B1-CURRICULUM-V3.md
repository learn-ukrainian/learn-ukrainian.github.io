# B1 Curriculum V3 — State Standard 2024 Compliant

**Date:** 2026-03-22
**Status:** DRAFT
**Method:** State Standard 2024 (WHAT) + Ukrainian textbooks (HOW) + 100% Ukrainian Immersion (B1 Paradigm)

---

## Architecture & Principles

B1 represents a fundamental shift in the Learn Ukrainian curriculum. The training wheels come off. English is removed. The learner transitions from "learning about Ukrainian" to "learning IN Ukrainian."

### Key Design Decisions:
1. **The Metalanguage Bridge:** Modules 01-05 explicitly teach Ukrainian linguistic terminology (Звуки, Відмінки, Присудок). All subsequent grammar explanations use these terms.
2. **100% Ukrainian Immersion:** Prose, grammar explanations, and instructions are exclusively in Ukrainian.
3. **Deep Morphology:** A1/A2 taught the "normal" nouns. B1 tackles the exceptions: alternations (о/і, е/і), consonant shifts (к/ц, г/з, х/с), pluralia tantum, and zero-ending feminines.
4. **The Motion Verb Universe:** Dedicating a full 10-module phase to mastering spatial prefixes (в-, ви-, при-, пі-, пере-, про-, за-, до-, об-, від-, під-, з-).
5. **Participles & Gerunds:** A massive expansion in syntax capability via дієприкметниковий and дієприслівниковий звороти.
6. **4,000 Word Target per Module:** High-density immersion reading, mimicking B1 level authentic texts, blogs, and literature excerpts.

### What B1 adds (from State Standard §4):

| Category | A2 had | B1 adds |
|----------|--------|---------|
| Phonetics | Basic reading | Vowel/consonant alternations, simplification, abbreviations |
| Nouns | 4 main classes | Subclasses (-ар, -яр, -ин, -ість), pluralia tantum, zero-ending |
| Verbs | Aspect, simple tenses | Conditional mood (би/б), reflexive nuances, verbal nouns |
| Motion | Base pairs (іти, їхати) | 12+ prefixes (прийти, вийти, об'їхати, перейти...) |
| Adjectives | Full paradigm | Degrees of comparison, suppletive forms, derivation from nouns |
| Participles | None | Active/Passive participles, Gerunds (Дієприкметник, Дієприслівник) |
| Syntax | Basic complex | Participle phrases, unreal conditionals (якби), reported speech |
| Stylistics | Basic synonyms | Epithets, metaphors, ellipsis, repetition, double negation |

## Gap Analysis & Pedagogy Notes

- **Immersion Shock:** Transitioning to 100% Ukrainian is hard. The 5-module metalanguage bridge (M01-M05) is crucial. We teach the *words for words* (іменник, відмінок) using simple A2-level sentences so learners learn the concepts without grammar overload.
- **Participles vs. Gerunds:** Standard B1 introduces both дієприкметниковий зворот and дієприслівниковий зворот. Phase 7 devotes 9 modules to this because English speakers struggle immensely with Slavic participles.
- **Motion Verbs:** A full phase (Phase 4) is dedicated to prefixes because the State Standard specifically highlights "утворення префіксальних дієслів руху" as a B1 requirement. We map these spatially.
