## Quick Reference Checklist

Before submitting a B1 checkpoint module, verify all items from `b1-grammar-module-template.md` PLUS:

### Checkpoint-Specific Requirements
- [ ] **TTT pedagogy:** Test-Teach-Test structure (comprehensive review)
- [ ] **Word count:** 1200+ words (review content, not new teaching)
- [ ] **Vocabulary:** 30-40 items (review vocabulary from ALL modules in phase)
- [ ] **Activities:** 16+ comprehensive testing activities
- [ ] **Integration:** Complete review of ALL modules in the phase
- [ ] **Self-assessment:** "–ß–∏ –º–æ–∂–µ—Ç–µ –≤–∏..." checklist at end
- [ ] **Immersion:** 100% Ukrainian (English only in vocabulary translations)
- [ ] **No new content:** Checkpoints ONLY review and test existing knowledge

### Comprehensive Testing (16+ activities required)

Checkpoints must test ALL skill groups from the phase:

**Core Testing Activities (Skill-Targeted):**
1. **quiz** (2-3 activities, 10-14 items each) ‚Äî One quiz per skill group cluster
2. **match-up** (2 activities, 14+ items) ‚Äî Term-definition, concept-example
3. **fill-in** (2 activities, 12-14 items) ‚Äî Contextual application of specific skills
4. **group-sort** (1-2 activities, 14-18 items) ‚Äî Categorization by skill group

**Integration Activities:**
5. **error-correction** (2 activities, 10-12 items) ‚Äî Common mistakes across phase
6. **cloze** (1 activity, 16+ blanks) ‚Äî Extended passage integrating all concepts
7. **unjumble** (1 activity, 10+ items) ‚Äî Sentence formation with phase structures

**Production Activities:**
8. **Writing task with Model Answer** (1 activity, 100+ words)

## Common Pitfalls to Avoid

### 1. **Introducing New Content**
- ‚ùå Problem: Teaching new grammar or vocabulary in checkpoint
- ‚úÖ Solution: Checkpoints ONLY review and test. All content must come from prior modules.

### 2. **Insufficient Module Coverage**
- ‚ùå Problem: Testing only 2-3 modules out of 8-module phase
- ‚úÖ Solution: Create dedicated quiz for EACH major concept + integration quiz

### 3. **English in Content**
- ‚ùå Problem: English explanations or instructions
- ‚úÖ Solution: 100% Ukrainian immersion (English only in vocabulary translations)

### 4. **No Model Answers**
- ‚ùå Problem: Writing tasks without examples
- ‚úÖ Solution: Every writing task includes complete model answer

### 5. **Too Few Activities**
- ‚ùå Problem: 12 activities (same as regular modules)
- ‚úÖ Solution: 16+ activities for comprehensive testing

### 6. **Missing Self-Assessment**
- ‚ùå Problem: No "–ß–∏ –º–æ–∂–µ—Ç–µ –≤–∏..." checklist
- ‚úÖ Solution: Include specific criteria tied to each module in the phase
