## State Standard 2024 Gaps (Priority: Medium)

Per `STATE-STANDARD-COMPLIANCE-ANALYSIS.md`:

### Gap 1: Explicit Aspect Pairs

**Requirement:** Видова пара (писати-написати) with dedicated drilling

**Current state:** Aspect introduced but pairs not systematically drilled

**Missing:**
- Dedicated aspect pair vocabulary lists
- Systematic drilling of common pairs
- Clear pattern recognition exercises

**Examples needed:**
```
читати → прочитати (to read)
писати → написати (to write)
робити → зробити (to do/make)
говорити → сказати (to say)
дивитися → подивитися (to watch/look)
```

**Action:** Expand M16 (Aspect Mastery Pairs) with explicit pair drilling

**Effort:** 3 hours

### Gap 2: Verb Prefixes for Aspect

**Requirement:** Префіксальне творення виду (Prefixal aspect formation)

**Current state:** Prefixes covered in M36-38 (word formation) but not linked to aspect

**Missing:**
- Explicit teaching of how prefixes change aspect
- Pattern: imperfective + prefix → perfective
- Common prefix meanings (про-, на-, з-, по-, ви-)

**Action:** Add aspect-prefix connection to M15 (Aspect Morphology) or M36 (Basic Motion Prefixes)

**Effort:** 3 hours

### Gap 3: свій Contrastive

**Requirement:** Clear distinction свій vs його/її/їхній

**Current state:** M17 covers свій but contrastive use is light

**Missing:**
- Explicit contrastive examples
- When subject = possessor → свій
- When subject ≠ possessor → його/її/їхній

**Action:** Expand M17 (Possessive свій) with contrastive section

**Effort:** 2 hours

### Gap 4: Numerals with Cases

**Requirement:** Числівники + іменники в різних відмінках

**Current state:** M21 covers numerals but case agreement patterns need drilling

**Missing:**
- Practice with 1-2-5 rule across all cases
- Numeral-noun agreement in genitive, dative, instrumental
- Complex numeral phrases (21, 32, 45, etc.)

**Action:** Expand M21 (Numerals and Nouns) with case agreement exercises

**Effort:** 2 hours


**Deliverables:**
- [ ] Update M16 plan with pair drilling section
- [ ] Add 30+ aspect pairs to M16 vocabulary
- [ ] Add 3-4 pair-matching activities

### Phase 2: Prefix-Aspect Connection — 3 hours

**Expand M15 (Aspect Morphology):**

Add explicit prefix-aspect teaching:
```
Base verb (imperfective) + prefix → perfective
читати + про- → прочитати
писати + на- → написати
робити + з- → зробити
```

**Deliverables:**
- [ ] Update M15 plan with prefix-aspect section
- [ ] Add prefix pattern examples to content
- [ ] Add 2-3 prefix-aspect activities

### Phase 3: свій Contrastive — 2 hours

**Expand M17 (Possessive свій):**

Add contrastive section:
```
Same subject → свій:
Він читає свою книгу. (his own)

Different subject → його/її:
Він читає його книгу. (someone else's)
```

**Deliverables:**
- [ ] Update M17 plan with contrastive section
- [ ] Add contrastive examples to content
- [ ] Add 2-3 contrastive activities

### Phase 4: Numeral-Case Agreement — 2 hours

**Expand M21 (Numerals and Nouns):**

Add case agreement drilling:
```
Nominative: два студенти, п'ять студентів
Genitive: двох студентів, п'яти студентів
Dative: двом студентам, п'яти студентам
```

**Deliverables:**
- [ ] Update M21 plan with case agreement section
- [ ] Add agreement tables to content
- [ ] Add 2-3 agreement activities
