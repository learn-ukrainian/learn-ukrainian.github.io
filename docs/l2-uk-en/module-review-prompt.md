- ‚ùå No conjugated verbs in sentences
- ‚ùå No "–Ø —á–∏—Ç–∞—é", "–í—ñ–Ω –ø—Ä–∞—Ü—é—î" constructions
- ‚úÖ Infinitives in vocabulary lists are OK: "—á–∏—Ç–∞—Ç–∏ - to read"

### Before Module 08 (No Class II / Irregulars)
- ‚ùå **No Class II verbs:** –≥–æ–≤–æ—Ä–∏—Ç–∏, —Ä–æ–±–∏—Ç–∏, –±–∞—á–∏—Ç–∏, –ª—é–±–∏—Ç–∏, —Ö–æ–¥–∏—Ç–∏, –Ω–æ—Å–∏—Ç–∏, —Å–∏–¥—ñ—Ç–∏, —Å—Ç–æ—è—Ç–∏, –∂–∏—Ç–∏, —Å–ø–∞—Ç–∏

- ‚ùå **No motion verbs:** –π—Ç–∏, —ó—Ö–∞—Ç–∏, —ó–∑–¥–∏—Ç–∏, –ª–µ—Ç—ñ—Ç–∏, –±—ñ–≥—Ç–∏

- ‚ùå No location phrases: "—É –ö–∏—î–≤—ñ", "–≤ —à–∫–æ–ª—ñ", "–Ω–∞ —Å—Ç–æ–ª—ñ", "–≤ –æ—Ñ—ñ—Å—ñ"

- ‚ùå **Specific verbs to avoid:** –¥–∏–≤–∏—Ç–∏—Å—è, –≤—á–∏—Ç–∏—Å—è, –≤–º–∏–≤–∞—Ç–∏—Å—è, –æ–¥—è–≥–∞—Ç–∏—Å—è, –Ω–∞–∑–∏–≤–∞—Ç–∏—Å—è, –∑–Ω–∞—Ö–æ–¥–∏—Ç–∏—Å—è

- ‚ùå **In unjumble activities, avoid:** –∫—Ä–∞—Å–∏–≤–∏–π, –≤–µ–ª–∏–∫–∏–π, –Ω–æ–≤–∏–π, —Å—Ç–∞—Ä–∏–π, –¥–æ–±—Ä–∏–π, —Å–º–∞—á–Ω–∏–π, –≥–∞—Ä—è—á–∏–π

### Imperative Mood at A1 (Special Case)

**Imperatives are NOT formally taught in A1, but are needed for functional modules (M15 directions, M19 caf√©).**

**How to handle imperatives at A1:**
- ‚úÖ Treat as **lexical chunks** (memorized phrases), NOT conjugatable verbs
- ‚úÖ Present in fixed form: "–ô–¥—ñ—Ç—å –ø—Ä—è–º–æ" = "Go straight" (phrase to memorize)
- ‚ùå Do NOT list infinitives in vocabulary (implies full conjugation needed)
- ‚ùå Do NOT explain imperative formation rules
- ‚ùå Do NOT include imperative conjugation in activities

**Vocabulary table format for imperatives:**
```markdown
# WRONG - implies verb learning:
| –ø–æ–≤–µ—Ä–Ω—É—Ç–∏ | to turn | verb | - | pf |  ‚Üê Problem: learner thinks they need to conjugate

# RIGHT - presents as phrase:
| –ü–æ–≤–µ—Ä–Ω—ñ—Ç—å –ª—ñ–≤–æ—Ä—É—á | Turn left | phrase | - | Memorize as chunk |
| –ô–¥—ñ—Ç—å –ø—Ä—è–º–æ | Go straight | phrase | - | Memorize as chunk |
```

**Activity restrictions:**
- ‚úÖ Recognition: "What does '–ô–¥—ñ—Ç—å –ø—Ä—è–º–æ' mean?" ‚Üí "Go straight"
- ‚úÖ Matching: Match phrase to meaning
- ‚ùå Production: "How do you say 'Turn right'?" (requires knowing imperative formation)
- ‚ùå Transformation: "Change to imperative" (not taught)

### Verbal Aspect at A1 (AVOID)

**Aspect (perfective vs imperfective) is NOT taught at A1.** Learners have only seen imperfective verbs.

**Problem verbs to flag:**
| Imperfective | Perfective | Issue |
|--------------|------------|-------|
| –ø–æ–≤–µ—Ä—Ç–∞—Ç–∏ | –ø–æ–≤–µ—Ä–Ω—É—Ç–∏ | –ø–æ–≤–µ—Ä–Ω—É—Ç–∏ is perfective - confusing if conjugated |
| –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç–∏ | –ø–µ—Ä–µ–π—Ç–∏ | –ø–µ—Ä–µ–π—Ç–∏ is perfective |
| –π—Ç–∏ | –ø—ñ—Ç–∏ | –ø—ñ—Ç–∏ is perfective |

**Why this matters:**
- Perfective verbs in present tense = FUTURE meaning
- "–Ø –ø–æ–≤–µ—Ä–Ω—É" ‚â† "I turn" ‚Üí it means "I will turn"
- Learners don't know this yet ‚Üí major confusion

**A1 solution:**
- Use imperatives as chunks (avoid infinitive/conjugation entirely)
- If infinitive needed, prefer imperfective: –ø–æ–≤–µ—Ä—Ç–∞—Ç–∏, –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç–∏
- Never require conjugating perfective verbs in activities

## 5. Vocabulary Scope Rules

### Vocabulary Plan Compliance (CRITICAL)

**Every module has a specific vocabulary list in CURRICULUM-PLAN.md.** The module's vocabulary table MUST include all planned words.

**Audit steps:**
1. Look up the module in the relevant CURRICULUM-PLAN.md (A1, A2, B1, etc.)

3. Compare against the module's `# Vocabulary` table
4. Report:
   - **Missing words**: Words in plan but not in vocabulary table
   - **Extra words**: Words in vocabulary table but not in plan (may be OK if related)

### Vocabulary Bloat Detection (CRITICAL)

**Vocabulary tables must stay within target AND maintain thematic focus.**

**Bloat indicators:**
1. **Count exceeds plan by >10%**: If plan says 40 words, max is ~44
2. **Thematic drift**: Words unrelated to module's core function
3. **Synonym overload**: Multiple words for same concept (pick ONE for production)
4. **Scope creep**: Including "nice to have" vocabulary from adjacent topics

**Example - M15 City Navigation:**
```
Plan focus: "Finding your way" (walking, metro, asking directions)
‚ùå Bloat: –∞–≤—Ç–æ–±—É—Å, —Ç—Ä–∞–º–≤–∞–π, —Ç—Ä–æ–ª–µ–π–±—É—Å (transport modes = separate topic)
‚ùå Bloat: –ø—Ä–∞–≤–æ—Ä—É—á AND –Ω–∞–ø—Ä–∞–≤–æ (synonyms - pick one)
‚úÖ Core: –ø—Ä—è–º–æ, –Ω–∞–ª—ñ–≤–æ, –Ω–∞–ø—Ä–∞–≤–æ, –Ω–∞–∑–∞–¥ (direction adverbs)
‚úÖ Core: –º–µ—Ç—Ä–æ, —Å—Ç–∞–Ω—Ü—ñ—è, –≤—É–ª–∏—Ü—è (essential navigation nouns)
```

**Resolution:**
- Remove words that exceed target count
- Prioritize CURRICULUM-PLAN.md vocabulary
- For synonyms: keep the one in the plan, note the other as "also used"
- Move thematic drift words to their proper module

### Synonym/Doublet Detection

**At A1, learners need ONE word per concept for active production.**

**Common doublets to flag:**
| Concept | Option A | Option B | A1 Recommendation |
|---------|----------|----------|-------------------|
| to the right | –Ω–∞–ø—Ä–∞–≤–æ | –ø—Ä–∞–≤–æ—Ä—É—á | Pick –Ω–∞–ø—Ä–∞–≤–æ (plan term) |

| near | –±—ñ–ª—è | –ø–æ—Ä—É—á | Pick –ø–æ—Ä—É—á (no case needed) |
| here | —Ç—É—Ç | —Å—é–¥–∏ | —Ç—É—Ç (location), —Å—é–¥–∏ = direction |

**Rule:** If both appear in vocabulary:
1. Mark one as "active" (for production in activities)
2. Mark other as "recognition only" with note: "Also used: X"
3. Do NOT test both in quizzes - only test the active one

### Case Notation in Vocabulary Tables (A1 WARNING)

**At A1, vocabulary tables should NOT include case requirements that haven't been taught.**

**Problematic pattern:**
```markdown
| –±—ñ–ª—è | near, by | prep | - | + genitive |  ‚Üê PROBLEM!
| –∑–∞ | behind | prep | - | + instrumental |  ‚Üê PROBLEM!
```

**Why this is bad:**
- Learner sees "+ genitive" but hasn't learned genitive (M16)
- Implies they need to learn case endings NOW
- Creates anxiety and cognitive overload

## 10. The Review Process

### Step 1: Scope Compliance Audit (CRITICAL)

Before anything else, check for scope violations:

1. **Identify module number** ‚Üí determine what grammar/vocabulary is allowed
2. **Check immersion level** ‚Üí verify Ukrainian/English ratio matches the level
3. **Scan all sentences** for verbs, cases, adjectives not yet introduced
4. **Check vocabulary list** against CURRICULUM-PLAN.md
5. **Verify transliteration** matches the module range rules
6. **Report all violations** using the format below

### Step 2: Content Richness Audit

- **Example sentences**: Count all Ukrainian example sentences (min 12+ for A1)
- **Engagement boxes**: Count üí°üé¨üåçüéØ boxes (min 3+ for A1)
- **Content words**: Estimate total word count in lesson content (min 600+ for A1)
- **Immersion level**: Estimate Ukrainian vs English ratio (30% for A1)

### Step 3: Activity Audit

- Count activities (minimum met?)
- Count items per activity (minimum met?)
- Check sentence lengths (within range?)
- Verify all activity words exist in vocabulary section
- Verify answer format is correct

### Step 4: Vocabulary Audit

- Count total words (within ¬±10% of target?)
- Check against CURRICULUM-PLAN.md target list
- Identify any words from future modules (leakage)
- Check example/activity nouns against vocabulary table
- **NEW**: Check for vocabulary bloat (thematic drift, synonyms)
- **NEW**: Check for case notation on untaught cases

### Step 5: Coming Next Verification

- Read "Coming Next" section
- Cross-check against next module in CURRICULUM-PLAN.md
- Verify terminology matches exactly (no synonyms)

### Step 6: Advanced Topic Check

- Identify any grammar/vocabulary above module scope
- Verify appropriate "Preview" labels are present
- Ensure preview topics are NOT in quizzes/activities

### Step 7: Imperative/Aspect Check (A1 functional modules)

- Are imperatives presented as phrases or conjugatable verbs?
- Are perfective verbs avoided or handled correctly?
- Do activities require production of imperatives (bad) or just recognition (ok)?
