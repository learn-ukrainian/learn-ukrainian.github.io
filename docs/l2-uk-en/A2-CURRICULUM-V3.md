**Date:** 2026-03-20
**Status:** DRAFT — designed by Claude, reviewed by Gemini, pending plan writing
**Method:** State Standard 2024 (WHAT) + Ukrainian textbooks (HOW) + Gemini adversarial review

---

## Architecture

A2 builds on A1's 55 modules. A1 taught 4 cases (Nom, Acc, Loc, Voc) + present/past/future + basic syntax. A2 completes the case system, introduces aspect, and builds complex sentences.

### Key design decisions (Gemini-reviewed):
1. **Genitive FIRST** — highest utility case, unlocks negation, quantity, possession
2. **Aspect introduced early** — so all vocabulary lists present verb pairs from day 1
3. **Adjective/pronoun paradigms taught WITH their case** — not in a separate late phase
4. **Plural cases get dedicated space** — Gen.Pl. alone needs its own module
5. **Real conditionals (якщо) at A2** — unreal (якби) deferred to B1
6. **Motion verb prefixes deferred to B1** — only base pairs at A2
7. **No pure vocabulary phases** — vocab integrated into communication modules

### What A2 adds (from State Standard §4):

| Category | A1 had | A2 adds |
|----------|--------|---------|
| Cases | Nom, Acc, Loc, Voc | + Genitive, Dative, Instrumental (full grammar) |
| Verbs | Present, past, analytic future | + Aspect pairs, synthetic future, motion verb base pairs, complete imperative |
| Adjectives | Nominative agreement only | + Full 7-case paradigm, comparative, superlative |
| Pronouns | Nominative only | + Full paradigm for personal, possessive, demonstrative |
| Numerals | As vocabulary | + Declension (ordinals + cardinal один) |
| Syntax | і, а, але, бо, що, де, коли | + тому що, щоб, який (relative), якщо |
| Stylistics | — | + Synonyms, antonyms, epithets, diminutives |

### Gap analysis notes for plan writers
When writing plans, ensure:
1. Every case module includes examples from BOTH hard and soft noun groups
2. Every adjective module includes examples from BOTH hard (-ий) and soft (-ій) groups
3. Genitive modules explicitly name -ар nouns, mixed group, -ія nouns, -тя neuter as subclasses
4. M49 numerals distinguishes -ий ordinals from -ій ordinals (третій/третього)
5. M18 explicitly teaches подобатися as Dative experiencer + Nominative object
6. M38 includes казати→кажу and пити→п'ю as explicit conjugation models
