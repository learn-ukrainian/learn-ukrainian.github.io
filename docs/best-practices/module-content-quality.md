# Module Content Quality Standards

> **Scope:** What makes a Ukrainian language module good. Applies to all tracks.
> For technical validation thresholds, `scripts/audit/config.py` is authoritative.
> Full richness metrics: `docs/l2-uk-en/MODULE-RICHNESS-GUIDELINES-v2.md`

---

## The Mission

This curriculum teaches Ukrainian to English speakers. It is not neutral educational software — it is a tool for cultural preservation and decolonization during an existential war. Every module must reflect this.

**Quality means:** linguistically correct, culturally authentic, emotionally engaging, and unambiguously Ukrainian in perspective.

---

## Ukrainian Language Standards

### Zero tolerance
- **No Surzhyk** (Ukrainian-Russian mixed speech)
- **No Russianisms** — replace with Ukrainian equivalents:
  - `кушати` → `їсти`
  - `приймати участь` → `брати участь`
  - `робити сенс` → `мати сенс`
  - `брати місце` → `відбуватися`
- **No calques from Russian** — think in Ukrainian, not translated Russian
- **No passive voice** where active is possible
- **Cyrillic only** — no transliteration except A1 phonetics modules

### Source hierarchy for verification
1. Горох (Horoh) — modern Ukrainian language norms
2. Грінченко (Hrinchenko) — historical Ukrainian dictionary
3. Антоненко-Давидович (Antonenko-Davydovych) — usage guide

### Grammar quality
- Case agreement must be perfect
- Verb aspect choice must be semantically correct
- Gender agreement: adjectives, past-tense verbs
- Sentence length: max 35 words for seminar tracks, shorter for beginner

## Decolonization Perspective

Every history and biography module must actively counter imperial and Soviet framing.

### What to counter
| Imperial framing | Ukrainian reality |
|-----------------|-------------------|
| "Common Rus cradle" shared by Russians, Ukrainians, Belarusians | Distinct ethnic and cultural trajectories from the beginning |
| Slavic tribes were primitive before Varangians | Tribal unions had proto-state structures, veche democracy, metallurgy |
| Ukraine as "Little Russia" / peripheral | Ukraine as the cultural and geographic core |
| Soviet achievements as Ukrainian achievements | Soviet erasure of Ukrainian identity |

### How to frame it
- Write from the Ukrainian perspective, not a "neutral" imperial perspective
- Name the bias explicitly: "Нестор-літописець, як симпатик полян, описує..."
- Use decolonization callout boxes: `[!decolonization]`
- Do not over-preach — integrate naturally into narrative

### Forbidden sources
- Russian-language academic sources (biased framing)
- Soviet historiography as authoritative
- Wikipedia in Russian as primary source

### Preferred sources
- `uk.wikipedia.org` (Ukrainian Wikipedia)
- `history.org.ua` (Institute of History)
- `litopys.org.ua` (primary sources)
- `iananu.org.ua` (archaeology)

## Patriotic Content

Ukrainian patriotism in this curriculum means:
- Pride in Ukrainian cultural, scientific, and artistic achievement
- Recognition of historical Ukrainian figures as Ukrainian (not Soviet/Russian/Polish)
- Emotional connection to Ukrainian identity for heritage learners
- Resistance framing: Ukrainians who resisted Russian/Soviet/Polish domination

This is not propaganda — it is restoring context that was systematically erased.

## Engagement Standards

A module that is technically correct but boring fails. Apply the A+ seminar test:

**"Would I want to keep reading?"**

### Required engagement elements

| Element | Minimum |
|---------|---------|
| Opening hook (question, vivid scene, provocative fact) | 1 |
| `[!myth-buster]` or `[!history-bite]` callouts | 2+ |
| Direct address to reader (`Уявіть...`, `Чому це важливо?`) | 5+ |
| "Why this matters now" statements | 2+ |
| Surprising/counterintuitive fact | 1+ |
| Emotional peak (moment of pride, surprise, connection) | 1+ |

### Narrative arc
Every seminar module must have:
```
HOOK → TENSION → JOURNEY → CLIMAX → RESOLUTION → CALL TO ACTION
```

### Anti-patterns to eliminate
| Pattern | Fix |
|---------|-----|
| "In this module we will learn..." | Replace with hook or question |
| Wall of facts without interpretation | Add "why this matters" after every 2 facts |
| Textbook passive voice | Rewrite as direct narrative |
| Generic conclusion | End with resonance, quote, or call to action |
| Orphan facts (interesting but unconnected) | Connect to main narrative or cut |

| C2 | 3000w | — |

## Research-Based Content

### Seminar tracks require:
- 3+ cited Ukrainian-language academic sources
- 5+ dated chronological events
- 2+ primary source quotes (verified, not paraphrased)
- Section-mapped research notes
- No fabricated dates or quotes — mark uncertain info as `[needs verification]`

### Core tracks require:
- State Standard reference (§section from UKRAINIAN-STATE-STANDARD-2024.txt)
- Verified vocabulary frequency data
- 1-2 cultural hooks with sources
- Common learner error patterns

---

## Primary Sources

Primary sources give modules authority and authenticity. Requirements:

- Quote in original Ukrainian where possible
- Provide context for the quote (when, who, why it matters)
- Analyze the quote — don't just paste it
- Flag bias if the source has a political agenda (e.g. chronicle writers favoring one tribe)

---

## Activity Quality

Activities must test understanding, not just recognition. Seminar tracks use:
- `reading` — comprehension of the lesson text
- `essay-response` — open-ended reflection
- `critical-analysis` — evaluate a claim or primary source
- `comparative-study` — compare two figures/periods/perspectives

Core tracks use grammar drills, fill-in, matching — but always in authentic contexts.

**Forbidden for seminar tracks:** `match-up`, `fill-in`, `cloze`, `group-sort`, `unjumble`, `anagram`, `mark-the-words`

---

## Self-Check Before Submitting

Run through this before finalizing any module:

- [ ] No Russianisms or calques
- [ ] Ukrainian perspective throughout (no Soviet/Russian framing)
- [ ] Opening hook engages in first 50 words
- [ ] At least 2 surprising facts or myth-busters
- [ ] Primary sources cited and analyzed (seminar tracks)
- [ ] Word count meets or exceeds target
- [ ] Immersion % correct for level
- [ ] Activities test understanding, not just recall
- [ ] Closing resonates — not a generic summary
