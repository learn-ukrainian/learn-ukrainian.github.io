## Level C2: Proficiency

### 1. Style Transformation (Strong)
*   **Requirement:** State Standard §4.3.7 (C2): "Ability to transform text between styles".
*   **Status:** `style-transformation-i.yaml` (Scientific -> Popular) and `style-transformation-ii.yaml` (Official -> Narrative) provide excellent coverage.
*   **Recommendation:** Ensure `style-transformation-ii.yaml` includes "Bureaucratic to Human" translation exercises (debunking officialese).

### 2. Rhetoric & Public Speaking (Nuance Needed)
*   **Requirement:** State Standard §4.3.5 (C2): "Rhetoric — argumentation, persuasion, public speaking".
*   **Status:** `professional-oral-communication.yaml` covers the basics.
*   **Gap:** Advanced rhetorical figures (chiasmus, zeugma, anaphora) need explicit instruction.
*   **Action:** Create `curriculum/l2-uk-en/plans/c2/advanced-rhetoric.yaml`. Focus on analyzing and writing political speeches or sermons (high style).

### 3. Modernist Syntax (Verification)
*   **Requirement:** State Standard §4.3.4 (C2): "Stream of consciousness, modernist syntax".
*   **Status:** `contemporary-literature.yaml` exists.
*   **Action:** Verify this module explicitly teaches *how to write* stream of consciousness, not just analyze it. If not, add a section to `creative-writing-prose.yaml`.

### 4. Dialect Mastery (Strong)
*   **Requirement:** State Standard Morphology (C2): "Archaic, dialectal, neologisms".
*   **Status:** `rare-archaic-forms.yaml` and `regional-varieties.yaml` are present.
*   **Recommendation:** Ensure `regional-varieties.yaml` includes active production tasks (e.g., "Rewrite this standard text in Hutsul dialect style").
