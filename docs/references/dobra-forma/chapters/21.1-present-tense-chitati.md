# 21.1 Present Tense of Verbs (Читати Type)

## Overview

This chapter introduces the present tense conjugation of the most common Ukrainian verb type—the **читати** (to read) type. These verbs represent the first conjugation and use consistent endings across all persons.

## Key Concepts

### Infinitive Forms

Most читати-type verbs end in **-ати** or **-яти** in their infinitive form. However, some exceptions exist that must be memorized:

- **розуміти** (to understand)
- **вміти** (to know how to do something)
- **чути** (to hear)

### Present Tense Conjugation Pattern

The complete conjugation follows this pattern:

| Person | Ending | Example (читати) |
|--------|--------|------------------|
| я (I) | -ю | читаю |
| ти (you, singular) | -єш | читаєш |
| він/вона/воно (he/she/it) | -є | читає |
| ми (we) | -ємо | читаємо |
| ви (you, plural) | -єте | читаєте |
| вони (they) | -ють | читають |

### Important Notes

The infinitive ending is **replaced** with the appropriate present tense ending—not added to it.

## Usage with Specific Verbs

**For playing games:** Use preposition **в** (e.g., грати **в** футбол)

**For playing instruments:** Use preposition **на** (e.g., грати **на** гітарі)

## Common Verbs of This Type

- читати (read)
- плавати (swim)
- гуляти (walk)
- слухати (listen)
- відпочивати (rest)
- вивчати (study)
- дякувати (thank)
- думати (think)
- замовляти (order)
- обідати (have lunch)
