# 
			17.3 Genitive Case of Adjectives		

										
		

In this module, you will learn the genitive case endings of adjectives for all genders and for plural. You will practice agreement between adjectives and nouns in all the genitive case contexts covered in *Dobra forma*. To review these contexts, see [Module 5.2](https://opentext.ku.edu/dobraforma/chapter/5-2/), [Module 6.1](https://opentext.ku.edu/dobraforma/chapter/6-1/), [Module 7.1](https://opentext.ku.edu/dobraforma/chapter/7-1/), [Module 7.2](https://opentext.ku.edu/dobraforma/chapter/7-2/), [Module 7.3](https://opentext.ku.edu/dobraforma/chapter/7-3/) and [Module 7.4](https://opentext.ku.edu/dobraforma/chapter/7-4/).
