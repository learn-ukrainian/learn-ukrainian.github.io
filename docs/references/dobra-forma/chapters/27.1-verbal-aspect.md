# 27.1 Introduction to Verbal Aspect (Prefixed Perfective Verbs)

## Core Concept

Verbal aspect in Ukrainian describes the nature of an action expressed by a verb—specifically whether it represents a completed one-time action or a repeated/habitual action. Most Ukrainian verbs form "aspectual pairs" consisting of an imperfective and a perfective counterpart.

## Imperfective vs. Perfective

**Imperfective verbs** serve as the default form:
- Reference actions in general terms
- Describe repeated or habitual actions
- Example: "Я люблю читати" (I love to read [in general])

**Perfective verbs** emphasize completion:
- Denote one-time actions in specific circumstances
- Stress that an action is finished
- Example: "Я маю прочитати цю книжку" (I need to read this book [completely])

## Common Aspectual Pairs

| Imperfective | Perfective |
|---|---|
| читати | прочитати |
| працювати | попрацювати |
| писати | написати |
| бачити | побачити |
| пити | випити |
| платити | заплатити |

## Prefixes for Forming Perfectives

The most common method involves adding a prefix to the imperfective form. The **по-** prefix is most frequent:

### Po- Prefix
- ду́мати → поду́мати (to think briefly)
- чу́ти → почу́ти (to hear)
- сні́дати → посні́дати (to have breakfast)
- слу́хати → послу́хати (to listen)
- проси́ти → попроси́ти (to ask once)

*Meaning*: Often conveys "to do something once" or "to do something for some time."

### Other Common Prefixes

| Prefix | Examples |
|---|---|
| **про-** | чита́ти → прочита́ти; жи́ти → прожи́ти |
| **при-** | готува́ти → приготува́ти; паркува́ти → припаркува́ти |
| **на-** | малюва́ти → намалюва́ти; писа́ти → написа́ти |
| **ви-** | пи́ти → ви́пити |
| **с-** | каза́ти → сказа́ти; фотографува́ти → сфотографува́ти |
| **з-** | розумі́ти → зрозумі́ти; роби́ти → зроби́ти; ї́сти → з'ї́сти |
| **за-** | плати́ти → заплати́ти; співа́ти → заспіва́ти; губи́ти → загуби́ти |

## Special Cases

### Reflexive (-ся) Verbs
Perfective forms also exist for -ся verbs:
- диви́тися → подиви́тися
- знайо́митися → познайо́митися
- вчи́тися → навчи́тися

### Verbs Without Perfective Counterparts
Some verbs do not form perfectives:
- ма́ти (to have)
- ко́штувати (to cost)
- подорожува́ти (to travel)
- намага́тися (to try/strive)

## Key Takeaways

1. Verbal aspect conveys whether an action is completed or habitual
2. Most verbs exist in imperfective-perfective pairs
3. Prefixes are added to imperfective verbs to create perfectives
4. **По-** is the most common prefix
5. Memorizing pairs is the most reliable approach since prefix selection is not predictable
