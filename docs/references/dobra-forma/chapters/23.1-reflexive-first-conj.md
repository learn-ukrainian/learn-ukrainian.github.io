# 23.1 Present Tense of -ся Verbs (First Conjugation)

## Overview

This module introduces **-ся verbs** in Ukrainian and their present-tense conjugations within the first conjugation pattern.

## Key Concepts

### What Are -ся Verbs?

Ukrainian contains numerous verbs ending in **-ся** with varied meanings:

- **Reflexive verbs**: Actions performed on oneself (e.g., *мітися* — to wash oneself)
- **Reciprocal verbs**: Mutual actions (e.g., *бачитися* — to see each other)
- **Non-reflexive -ся verbs**: Those without obvious reflexive meaning (e.g., *усміхатися* — to smile)

### Present Tense Conjugation Pattern

The verb *усміхатися* (to smile) exemplifies first-conjugation **-ся** verbs:

| Person | читати | усміхатися |
|--------|--------|-----------|
| я | читаю | усміхаюся |
| ти | читаєш | усміхаєшся |
| він/вона/воно | читає | **усміхається** |
| ми | читаємо | усміхаємося |
| ви | читаєте | усміхаєтеся |
| вони | читають | усміхаються |

**Note**: The third-person singular differs notably: *усміхається* (not *усміхаєся*).

## Special Conjugation: -уватися/-юватися Verbs

Verbs like *запізнюватися* (to be late), *користуватися* (to use), *одружуватися* (to get married), and *захоплюватися* (to be passionate about) drop the **-ва-** suffix in all present-tense forms:

| Person | запізнюватися |
|--------|-------------|
| я | запізнююся |
| ти | запізнюєшся |
| він/вона/воно | запізнюється |
| ми | запізнюємося |
| ви | запізнюєтеся |
| вони | запізнюються |

**Exception**: Verbs with vowels other than *у* or *ю* before **-ва-** retain the suffix (e.g., *називатися* — to be called → *називається*).

## Summary Points

- All -ся verbs modify endings based on person; endings precede -ся
- First-conjugation -ся verbs follow *читати*-type patterns
- Third-person singular forms consistently use *-ється*
- -уватися/-юватися verbs drop the -ва- infix during present conjugation
