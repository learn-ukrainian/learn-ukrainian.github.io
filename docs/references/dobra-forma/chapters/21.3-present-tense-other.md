# 21.3 Present Tense of Verbs (Other First Conjugation Verbs)

## Overview

This chapter covers present-tense conjugation for first-conjugation verbs that deviate from the standard **читати** and **працювати** patterns, specifically focusing on verbs like **пити** (to drink), **жити** (to live), **писати** (to write), and **казати** (to say).

## Key Concepts

### Hard-Vowel vs. Soft-Vowel Endings

Unlike the soft-vowel pattern (читати type), certain first-conjugation verbs use hard-vowel endings:
- Hard vowels: **-у, -е, -ем, -ете, -ут**
- Soft vowels: **-ю, -є, -ємо, -єте, -ють**

### Common Verbs with Hard-Vowel Endings

Three important verbs follow this pattern:
- **жити** (to live)
- **писати** (to write)
- **казати** (to say)

### Conjugation Pattern for жити

| Person | Form |
|--------|------|
| я | живу́ |
| ти | живе́ш |
| він/вона/воно | живе́ |
| ми | живе́мо |
| ви | живе́те |
| вони | живу́ть |

### Additional Changes in писати

The verb **писати** undergoes consonant mutation: the suffix **-са-** becomes **-ш-** in all present-tense forms (пи́шу, пи́шеш, пи́ше, etc.).

### Changes in казати

When conjugated, **-за-** is replaced with **-ж-** throughout: кажу́, кажеш, каже́, кажемо, кажете, кажу́ть.

## Summary

These verbs maintain first-conjugation characteristics while employing hard-vowel endings distinct from the standard soft-vowel pattern, with several requiring consonant or stem modifications during conjugation.
