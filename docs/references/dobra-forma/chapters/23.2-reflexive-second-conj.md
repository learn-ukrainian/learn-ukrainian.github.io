# 23.2 Present Tense of -ся Verbs (Second Conjugation)

## Overview

This module teaches the present-tense conjugation of six common reflexive verbs (-ся verbs) belonging to the second conjugation in Ukrainian.

## Key Verbs Covered

**вчи́тися** (to study/learn) and **ба́читися** (to see each other) follow the лежа́ти type pattern:

| Person | лежа́ти | вчи́тися |
|--------|---------|----------|
| я | леж**у́** | вч**у́**ся |
| ти | леж**и́ш** | вч**и́ш**ся |
| він/вона/воно | леж**и́ть** | вч**і́ть**ся |
| ми | леж**имо́** | вч**имо́**ся |
| ви | леж**ите́** | вч**ité**ся |
| вони | леж**а́ть** | вч**а́ть**ся |

## Usage Distinctions

Three Ukrainian verbs convey "to study" with different applications:

- **вивча́ти**: Used exclusively with academic subjects (e.g., "я вивча́ю літерату́ру")
- **вчи́тися**: Applies to learning skills via infinitives and stating where one studies
- **навча́тися**: Formal register for stating educational location/year

## Consonant Mutation Verbs

**диви́тися** (to watch) and **ціка́витися** (to be interested) undergo stem mutation in first-person singular and third-person plural forms, adding **-л-** before the ending (similar to люби́ти type).

**Знайо́митися** (to get acquainted) similarly adds **-л-** in these two forms.

## Fear Verb Pattern

**боя́тися** (to be afraid) follows the стоя́ти type:

| Person | боя́тися |
|--------|----------|
| я | бо**ю́**ся |
| ти | бо**ї́ш**ся |
| він/вона/воно | бо**ї́ть**ся |
| ми | бо**їмо́**ся |
| ви | бо**їте́**ся |
| вони | бо**я́ть**ся |

This verb requires its object in the genitive case (e.g., "я бою́ся води́").
