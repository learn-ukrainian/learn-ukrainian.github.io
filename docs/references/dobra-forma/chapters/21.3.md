## Важли́во!

Unlike the verbs of the **чита́ти** and **працюва́ти** types, the verbs **жи́ти **and **писа́ти** represent the pattern of hard-vowel endings in the first conjugation. The two patterns, however, are very similar. Compare:

| 

| 
| чита́ти 
| писа́ти 
 |

| я 
| чита́ю 
| пишу́ 
 |

| ти 
| чита́єш 
| пи́шеш 
 |

| він/вона/воно 
| чита́є 
| пи́ше 
 |

| ми 
| чита́ємо 
| пи́шемо 
 |

| ви 
| чита́єте 
| пи́шете 
 |

| вони 
| чита́ють 
| пи́шуть 
 |

Phonetically, **ю = й + у** and **є = й + е**, so one can think of these patterns as being the same, with the addition of the “**й**” sound in the soft-vowel pattern.

Many more verbs belong to the **чита́ти** type. The ones that take the hard-vowel endings of the first conjugation are best memorized. For now, remember these three common ones:

**жи́ти** (to live)

**писа́ти** (to write)

**каза́ти** (to say)
