## 1. Textbook-to-CEFR Mapping: Approach C (Hybrid)

> **Decision (2026-02-27)**: Rigid grade-to-CEFR mapping abandoned. See `ARCHITECTURE.md` §10 for full rationale.

### Why the old mapping broke

The original mapping (A1=G1-2, A2=G3-4, B1=G5-6) assumed each grade range maps exclusively to one CEFR level. This fails because:
- Ukrainian schools use a **spiral curriculum** — every concept is introduced simply, then revisited with increasing depth
- Grade 5 restarts ALL grammar with formal terminology — same concepts as Grade 3-4, higher abstraction
- Grade 6 has the best declension tables but they're useful for A2 learners, not just B1
- L2 learners need daily-life vocabulary (transport, shopping) that textbooks never teach

### New approach: Three-layer architecture

| Layer | Source | Role |
|-------|--------|------|
| **Structure** | CEFR levels (A1→B1) | Progression framework |
| **Requirements** | State Standard 2024 | WHAT to teach per level |
| **Content library** | ALL textbooks (Grades 1–11) | HOW to teach — searchable by topic |

### Which grades are STRONGEST for each topic

| Topic | Best Grade(s) | Why |
|-------|--------------|-----|
| Alphabet, sounds | Grade 1 (Букvar) | Foundational, sounds-before-letters pedagogy |
| Question-word grammar | Grade 2–4 | ХТО?/ЩО?/ЯКИЙ? approach throughout |
| Word structure | Grade 3 | Explicit prefix/suffix/root/ending analysis |
| Case system (naming) | Grade 4 | All 7 cases first named with question words |
| Case paradigm tables | Grade 6 | Full 4 відміни × 3 groups tables |
| Verb tenses | Grade 4 | Signal words (учора/сьогодні/завтра) |
| Euphony tables | Grade 5 | Full rule tables for у/в, і/й, з/із/зі |
| Stress patterns | Grade 5 | Explicit rules with verified word lists |
| Formal terminology | Grade 5 | Лексикологія, фонетика, синтаксис |
| Full morphology | Grade 6 | 130+ pages of paradigms |
| Anti-суржик | Grade 5–6 | "Культура слова" boxes every section |
| Proverbs/riddles | All grades | Rich source across all textbooks |

### Per-level primary content sources

**A1 (750 words)** — draws primarily from: Grade 1 (alphabet, sounds), Grade 2 (question-word grammar, basic sentences), Grade 3 (phonetics, euphony basics)

**A2 (1800 cumulative)** — draws primarily from: Grade 4 (case naming, verb tenses), Grade 3 (word structure, folklore activities), Grade 6 (paradigm tables as reference), Grade 5 (euphony review)

**B1 (3300 cumulative)** — draws primarily from: Grade 5 (formal terminology, complex syntax, stress), Grade 6 (full morphology, phraseology), Grade 4 (case practice), Grade 3 (word formation basics)

### Key sequence insight: SPIRAL CURRICULUM

Every concept is introduced simply, then revisited with increasing depth:
1. **Grade 1**: Experience (hear sounds, see letters)
2. **Grade 2**: Identify (name parts of speech by question words)
3. **Grade 3**: Analyze (word structure, sentence members)
4. **Grade 4**: Systematize (all 7 cases, all tenses, all pronouns)
5. **Grade 5**: Formalize (linguistic terminology, rule tables)
6. **Grade 6**: Master (full paradigm tables, exceptions, nuances)

**Curriculum design principle**: Our A1→A2→B1 should mirror this spiral. Introduce through experience at A1, name and practice at A2, formalize at B1.

**у/в alternation**:
| Context | Use у | Use в |
|---------|-------|-------|
| Consonant_vowel | ✅ у | — |
| Consonant_consonant | ✅ у | — |
| Vowel_consonant | — | ✅ в |
| Vowel_vowel | — | ✅ в |
| Sentence-initial + consonant | ✅ У | — |
| Sentence-initial + vowel | — | ✅ В |

**і/й alternation**:
| Context | Use і | Use й |
|---------|-------|-------|
| Consonant_consonant | ✅ і | — |
| Consonant_vowel | — | ✅ й |
| Vowel_vowel | — | ✅ й |
| Vowel_consonant | either | either |
| Sentence-initial | ✅ І | — |

**з/із/зі alternation**:
| Form | Context |
|------|---------|
| з | Before vowels (mid-sentence) |

## 7. Key Pedagogical Patterns

### Pattern 1: Question-Word Grammar (Grades 1–4)
Children NEVER hear "nominative case" until Grade 4, and even then the question word comes first:
- ХТО? ЩО? → Називний
- КОГО? ЧОГО? → Родовий
- etc.

**Curriculum rule**: A1/A2 modules should use ONLY question words. Case terms can appear as optional labels at A2.

### Pattern 2: Mnemonic for Case Order (Grade 4)
"Нашого Ромчика Дивує Зебра — Оця Маленька Красуня" (Н-Р-Д-З-О-М-К)

**Curriculum action**: Include this mnemonic when teaching case system at A2.

### Pattern 3: Signal Words for Tenses (Grade 4)
| Tense | Signal words |
|-------|-------------|
| Past | учора, минулого тижня |
| Present | зараз, сьогодні |
| Future | завтра, наступного дня |

**Curriculum action**: A1 module `den` uses сьогодні; module `mynule` uses учора. Future tense signals in A2.

### Pattern 4: Hard/Soft Pairs Through Minimal Pairs (Grade 1)
Буквар shows: лис/ліс, дуб/дім, торт/тісто, сумка/сітка

**Curriculum action**: A1 module `abetka` already uses these pairs. Reinforce in `sklad`.

### Pattern 5: Anti-Суржик Correction (Grade 5+)
"Культура слова" boxes correct Russian interference patterns. Common errors:
- приймати участь → брати участь
- учбовий → навчальний
- по рішенню → за рішенням
- дані співпадають → дані збігаються

**Curriculum action**: B1+ should include error correction activities targeting these patterns.

### Pattern 6: Genitive -а/-я vs -у/-ю (Grade 6)
Semantic rule, not arbitrary:
- Concrete/countable → **-а/-я**: стола, учня, Києва
- Abstract/uncountable → **-у/-ю**: страху, розвитку, цукру

**Curriculum action**: A2/B1 case modules should teach this as a meaning-based distinction, not a memorization list.

### Pattern 7: Synonym Chains for Textual Cohesion (Grade 3)
Single text uses multiple synonyms for the same referent:
дятел → птах → лісовий лікар → незвичайний хірург

**Curriculum action**: A2+ reading activities should include synonym recognition exercises.
