## Pedagogical Patterns

### 1. Teaching Sequence Logic

**Phase 1 — Open vowels (А, О, У)**: Maximum mouth opening, easiest to produce. Vowels first = can form syllables immediately.

**Phase 2 — Sonants + С (М, С, Н, Л, К)**: Most common consonants. М first (мама = universal first word). С chosen early because [с] is common and the hard/soft contrast is clear (сумка vs сітка).

**Phase 3 — More vowels + common consonants (И, Р, В, Б, І, Е)**: Expand vowel inventory (И, І, Е), add frequent consonants (Р, В, Б).

**Phase 4 — Remaining voiced/voiceless pairs (П, Д, Т, Г, Ґ)**: Complete the main consonant set.

**Phase 5 — Semi-vowel and sibilants (Й, З, Ж, Ш, Х, Ч)**: More complex sounds.

**Phase 6 — Special letters (ь, Я, Ю, Є, Ї, Щ, Ц, Ф)**: Letters with special phonetic properties (softening, two-sound letters, rare letters).

### 2. Consonant Hard/Soft Pairs

When a consonant has distinct hard and soft pronunciations, the textbook shows TWO key words:

| Letter | Hard | Soft |
|--------|------|------|
| С | сýмка | сíтка |
| Н | намéт | нíрка |
| Л | лис [лис] | ліс [л'іс] |
| Р | рóбот | рікá |
| Д | дуб | дім |
| Т | торт | тісто |
| З | (implied) | зíрка |
| Ц | цирк | цікáвий |

**Always hard** (no soft variant): Ж, Ш
**Always soft**: Ч
**Not a sound**: ь (softens preceding consonant)

### 3. Two-Sound Letters (Йотовані)

All explained with the same pattern: "Буква X позначає два звуки [йX] на початку слова й складу."

| Letter | Sounds | Key Word | Visual Pattern |
|--------|--------|----------|---------------|
| Я | [йа] | яблуко | ЯК [йа] / МАЯК [йа] |
| Ю | [йу] | юшка | ЮШКА [йу] / МИЮ [йу] |
| Є | [йе] | єнот | ЄНОТ [йе] / МИЄ [йе] |
| Ї | [йі] | їжак | ЇЖАК [йі] / ПОЇЗД [йі] |
| Щ | [шч] | щука | ЩУКА [шч] |

**Ї is special**: "Буква ї **завжди** позначає два звуки [йі]" — ALWAYS two sounds (no single-sound variant).

### 4. Soft Sign (ь) — Detailed Softening Pairs

The ь page (p.101) systematically shows ALL consonant pairs that can be softened:

| Hard → Soft | Examples |
|-------------|----------|
| л → ль | сіль, міль, кисіль |
| с → сь | лось, рись, карась |
| н → нь | кінь, тінь, день |
| т → ть | сидить, лежить, біжить |
| з → зь | вузько, слизько, близько |
| д → дь | мідь, ведмідь, відповідь |
| ст → сть | радість, ніжність, гордість |

### 5. Standard Page Structure

Each letter page follows this consistent template:

1. **Header**: Large letter + key word (with syllable/stress marks) + theme illustration
2. **Exercise 1**: "Назви предмети. Визнач місце звука [X] у словах." — 4 thematic items with illustrations
3. **Exercise 2**: Syllable tables — consonant + each vowel, building complexity (CV → CVC → CVCC)
4. **Exercise 3**: "Знайди слово — підпис до малюнка й слово до схеми." — Word lists + sound schema matching
5. **Exercise 4**: Connected text (story or poem) using the target letter heavily

### 7. Special Teaching Moments

- **Ґ page**: ҐАВА vs КАВА shown side by side — minimal pair for Ґ vs К distinction
- **Й page**: МАЙКА vs ЙОД — shows Й in middle vs beginning of word
- **Ї page**: Possessive pronoun morphology (мій→моїі, твій→твої, свій→свої) + patriotic poem about Ukraine
- **Г page**: "Факти і думки" about Говерла (highest mountain in Ukraine) — first factual text
- **Щ page**: "Факти і думки" about щука (pike) — scientific text with зябрами (gills)
- **Я page**: Adjective agreement introduced (яблучний/яблучна/яблучне/яблучні)
- **Ю page**: 1st person verb forms (мрію, співаю, читаю, малюю, граю, дякую, шаную)
