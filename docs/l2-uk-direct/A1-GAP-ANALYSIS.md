# A1 Gap Analysis: Current Plan vs State Standard 2024

> Date: 2026-02-26
> Original plan: 25 modules → expanded to 41 modules (this analysis) → further expanded to **44 modules** (compression splits, 2026-02-26)
> Reference: `UKRAINIAN-STATE-STANDARD-2024.txt` lines 221–747
>
> **Note**: After this gap analysis produced a 41-module plan, a compression analysis identified 3 HIGH-risk modules that packed too many unrelated grammar concepts. These were split:
> - `rechennia` → `rechennia` + `spoluchnyky` (SVO/negation separated from connectors/Чи-questions)
> - `den` → `den` + `mynule` (daily routine+reflexive separated from past tense)
> - `nakazy-prohannia` → `nakazy` + `zaborony-dokonane` (imperative+requests separated from prohibitions+perfective future)
> See `CURRICULUM-PLAN.md` for the current 44-module sequence.

## Method

Every requirement in the State Standard A1 (§1–4, Catalogues A/B/C) was mapped against the current 25-module plan. Each item is marked:
- ✅ = covered
- ⚠️ = partially covered (needs expansion)
- ❌ = missing (needs new module or major addition)

## Summary: Gap Severity

### CRITICAL (must add — cannot pass A1 test without these)

1. **Greetings, farewells, politeness formulas** — Добрий день/ранок/вечір, До побачення, Дякую, Вибачте, Будь ласка — the most basic communicative functions, 100% missing
2. **Self-introduction** — Мене звати..., nationality, language, occupation, age — Theme 1 (Людина) core, 100% missing
3. **Likes/dislikes** — Мені подобається / не подобається, Я люблю / не люблю — communicative intention #13, 100% missing
4. **Personal pronoun declension** — мене/мені, тебе/тобі, нас/нам, вас/вам — required morphology, 100% missing
5. **Demonstrative pronouns** — цей/ця/це, той/та/те + plurals — required morphology, 100% missing
6. **Adjective declension in accusative/locative** — великого/великому, великий/великій (masc), велику/великій (fem), etc. — explicitly required, 100% missing
7. **Travel theme** (Catalogue B #7) — station, airport, border, directions — 100% missing
8. **Leisure theme** (Catalogue B #6) — hobbies, sports, sports venues — 100% missing
9. **Чи-questions** — required syntax type, 0% coverage
10. **Causal clauses (тому що, бо)** — required syntax, 0% coverage

### SIGNIFICANT (needs expansion)

11. **Writing skills** — forms, postcards, self-descriptions — whole skill area nearly absent
12. **Real-world reading** — signs, schedules, tickets, receipts, forms, SMS — absent
13. **Noun plural formation** — vowel alternation (рік→роки, будинок→будинки) — not systematically taught
14. **Imperative** — only 2 verbs; Standard lists 8 (читай, прочитай, скажи, дай, пиши, напиши, сідай, сиди)
15. **Perfective future** — зможу, скажете, прочитають — standard explicitly requires some forms
16. **Appearance vocabulary** — зовнішність (tall, short, blond, etc.) — Theme 1 subtopic, missing
17. **Transport vocabulary** — автобус, трамвай, метро, таксі, зупинка, квиток — missing
18. **Acc with prepositions за, через** — required, not covered
19. **Locative for time (months)** — у травні, у вересні — required, not covered
20. **звідки, куди** — question words required but not taught

### MINOR (should fix)

21. Tableware (посуд, столові прибори) — Theme 9 detail
22. Hygiene products (мило, шампунь, зубна паста) — Theme 8 detail
23. Office supplies (канцелярські товари) — Theme 8 detail

## Recommendation: Expand to ~45 modules

The current 25-module plan covers phonetics and core grammar well but has **critical gaps in communicative competence, thematic coverage, and all 4 speech activities**. A student completing the current plan would know grammar but not be able to introduce themselves, say hello, express preferences, ask for directions, or fill in a form.

**Phase 0: Script** (3 modules — no change)
