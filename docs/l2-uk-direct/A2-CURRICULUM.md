## What A2 Adds Over A1

| Feature | A1 (covered) | A2 (new) |
|---------|-------------|----------|
| Cases | 4 (Н, Зн, М, Кл) | **ALL 7** — adds Родовий, Давальний, Орудний |
| Verbs | Present, past, compound future, imperative 2nd person | Expanded aspect pairs, imperative with хай/нехай, **synthetic future**, motion verbs |
| Adjectives | Nominative + accusative + locative | **Comparison** (вищий, найвищий, більш) |
| Pronouns | Personal + possessive + demonstrative (nom/acc/loc) | **Full declension** through all 7 cases |
| Numerals | Cardinals + ordinals (nominative) | **Declension** basics (два/двох/двом) |
| Vocabulary | 750 words | 1800 cumulative (+1050 new) |
| Topics | 12 | 14 (adds навчання, робота) |
| Dialogue | 4–6 turns | **8–10 turns** |
| Writing | Short phrases, forms | **50-word texts**, letters, short biographies |
| Phonetics | Basics, euphony | **Alternations** (vowel, consonant, mutation) |

---

## Design Principles (additions to A1)

1. **Cases taught one at a time** — each new case gets 2–3 dedicated modules: concept → prepositions → practice
2. **Case questions come first, names second** — "КОГО? ЧОГО?" before "Родовий відмінок" (Grade 4 approach)
3. **Case mnemonic introduced** — "Нашого Ромчика Дивує Зебра — Оця Маленька Красуня" (Grade 4 p.38)
4. **Aspect pairs taught through meaning contrast** — читати/прочитати as process vs result, not as grammatical categories
5. **Phonetic alternations** (vowel, consonant) taught through examples, not rules — learner observes the pattern
6. **Two new thematic areas** (навчання, робота) each get dedicated modules
7. **Writing module every ~10 modules** — short bio, personal letter, description
8. **Folklore activities increase** — riddles, proverbs, legends from Grade 3 Part 2
9. **Content drawn from ALL textbook grades** — each module sources content from whichever grade has the best treatment. Grade 4 for case naming, Grade 6 for paradigm tables, Grade 5 for euphony, Grade 3 for folklore activities. No grade-locking. (Approach C)

#### Module 1: `rodovyi-koho-choho`
**Focus**: Genitive concept — КОГО? ЧОГО? — possession, absence, origin
**Type**: `grammar`
**State standard**: §4.2.3.2 (genitive), Catalogue A #17 (belong: "Це книга мого брата")

**The key question**: КОГО? ЧОГО? — who is it about? what is it from/of?

**Genitive formation — basic patterns** (Grade 4 p.31):

| Gender | Nominative | Genitive | Pattern |
|---|---|---|---|
| Masc hard | брат | брата | +а |
| Masc soft | учитель | учителя | +я |
| Fem -а | мама | мами | -а → -и |
| Fem -я | земля | землі | -я → -і |
| Neut -о | місто | міста | -о → -а |
| Neut -е | море | моря | -е → -я |

**Core meanings**:
- **Possession**: книга брат**а**, дім бабус**і**, вікно кімнат**и**
- **Absence** (немає + gen): Немає хліб**а**. Немає вод**и**. Немає час**у**.
- **Origin** (з/із + gen): Він із Львов**а**. Вона з Україн**и**.

**Vocabulary** (20 words — possession/absence context):
- власник/власниця (owner), річ (thing), ключ (key), гаманець (wallet), паспорт, документ
- Expansion of family: свекруха, тесть, зять, невістка, племінник, племінниця
- Absence phrases: немає часу, немає грошей, немає місця, немає відповіді

**Activities**:

1. `pattern_drill` — genitive formation: «брат → книга...» → «брата» (15 nouns across genders)
2. `build_sentence` — possession: «Це [річ] [person in genitive]» (Це машина батька. Це сумка сестри.)
3. `pattern_drill` — absence: «хліб → немає...» → «немає хліба» (10 items)
4. `true_false` — «Немає водо» → false (should be: немає води)
5. `classify` — sort genitive endings: -а/-я (masc/neut) vs -и/-і (fem)
6. `reading` — "Lost and found" scene: items belong to different people. | question: Чия це сумка? Чий це паспорт?
7. `proverb_drill` — «Без труда немає плода.» (Grade 3)

#### Module 2: `rodovyi-priymennyky`
**Focus**: Genitive with prepositions — від, до, з/із, без, для, після, біля
**Type**: `grammar`
**State standard**: §4.2.3.2 (genitive prepositions)

**Preposition + genitive patterns**:

| Preposition | Meaning | Example |
|---|---|---|
| від | from (person/cause) | лист **від** друг**а**, втомився **від** робот**и** |
| до | to (destination/limit) | іти **до** школ**и**, **до** вечор**а** |
| з / із | from (origin/material) | повернутися **з** подорож**і**, варення **з** вишень |
| без | without | кава **без** цукр**у**, день **без** дощ**у** |
| для | for (purpose) | подарунок **для** мам**и**, книга **для** дітей |
| після | after | **після** обід**у**, **після** урок**у** |
| біля | near | **біля** школ**и**, **біля** будинк**у** |

**Additional location prepositions**: навпроти (opposite), серед (among), недалеко від (not far from)

**Activities**:

1. `classify` — sort 12 sentences by preposition meaning: від (source) vs до (destination) vs без (absence) vs для (purpose) vs після (time) vs біля (location)
2. `pattern_drill` — preposition + genitive: «школа → іду до...» → «до школи» (12 items)
3. `build_sentence` — compose from: [preposition] + [noun in genitive] (8 combinations)
4. `true_false` — «Подарунок для мамо» → false (для мами)
5. `pattern_drill` — time expressions: «обід → після...» → «після обіду»
6. `reading` — letter from a friend: «Привіт! Я приїхав із Одеси до Львова. Після подорожі я відпочиваю біля готелю. Завтра піду до музею...» | questions: Звідки він? Де він відпочиває?

#### Module 3: `rodovyi-kilkist`
**Focus**: Genitive for quantity, numerals 2–4 + genitive, dates
**Type**: `grammar`
**State standard**: §4.2.3.2 (quantity), §4.2.1.3 (numeral declension)

**Quantity patterns**:
- 2, 3, 4 + genitive singular: два брат**и**, три книг**и**, чотири вікн**а**
- 5–20 + genitive plural: п'ять братів, десять книг, двадцять вікон
- багато/мало/кілька/декілька/скільки + genitive plural: багато людей, мало часу, скільки грошей?

**Date expressions (genitive for dates)**:
- Яке сьогодні число? — Сьогодні перше січня (Н.в. + Р.в.)
- Коли? — Першого січня (Р.в. + Р.в.)

**Month genitive forms**: січня, лютого, березня, квітня, травня, червня, липня, серпня, вересня, жовтня, листопада, грудня

**Activities**:

1. `pattern_drill` — numerals + gen: «3 + книга» → «три книги» / «7 + книга» → «сім книг» (15 items)
2. `classify` — genitive singular (2–4) vs genitive plural (5+): sort number phrases
3. `pattern_drill` — dates: «1 січень» → «першого січня» (12 months)
4. `build_sentence` — answer «Скільки?»: «Скільки учнів у класі?» → «У класі двадцять п'ять учнів.»
5. `true_false` — «Три книг» → false (три книги — genitive singular)
6. `reading` — birthday invitation: «Запрошую на день народження! Дата: десятого лютого. Час: о шостій годині. Адреса: вулиця Шевченка, 15. Буде багато гостей і смачної їжі!» | questions

#### Module 4: `rodovyi-zaperech`
**Focus**: Genitive of negation — немає, не було, не буде; negated object
**Type**: `grammar`
**State standard**: §4.2.3.2 (negated object in genitive)

**Key pattern**: Negated existence uses genitive, not nominative:
- Є книга (Н.в.) → Немає книги (Р.в.)
- Був учитель → Не було вчителя
- Буде урок → Не буде уроку

**Negated direct object** (genitive instead of accusative):
- Я бачу друга (Зн.в.) → Я не бачу друга (Р.в. = same form for animate)
- Я маю книгу (Зн.в.) → Я не маю книги (Р.в.)

**Vocabulary**: Not-having, absence, lack — useful for daily life:
- Немає інтернету. Немає зв'язку. Немає світла. Немає гарячої води.
- Не було дощу. Не буде снігу.

**Activities**:

1. `pattern_drill` — transform: «Є хліб» → «Немає хліба» (12 items)
2. `pattern_drill` — past/future negation: «Був дощ» → «Не було дощу» / «Буде урок» → «Не буде уроку»
3. `build_sentence` — describe what's missing from an image of a room
4. `true_false` — «Немає книга» → false (немає книги)
5. `classify` — nominative (є/був/буде) vs genitive (немає/не було/не буде)
6. `reading` — everyday complaint text: «Сьогодні поганий день. Немає інтернету. Не було гарячої води. Магазин зачинений — немає молока. Але є хліб і чай!» | questions: Що є? Чого немає?

#### Module 5: `rodovyi-praktyka`
**Focus**: Genitive consolidation — mixed practice across all genitive uses
**Type**: `practice`
**State standard**: §4.2.3.2 (all genitive uses)

**Consolidation across all 4 previous modules**:
- Possession (книга брата)
- Prepositions (від, до, з, без, для, після, біля)
- Quantity (два + gen sg, п'ять + gen pl)
- Negation (немає + gen)

**Activities**:

1. `classify` — sort 20 genitive phrases by function: possession / preposition / quantity / negation
2. `pattern_drill` — mixed genitive: all types interleaved (20 items)
3. `build_sentence` — complex sentences using multiple genitive functions: «Після уроку я іду до друга без книги.»
4. `reading` — narrative paragraph using all genitive types: «Марко повернувся з Києва. Після подорожі він дуже втомився. У нього немає сил. Він купив подарунок для сестри — два кілограми цукерок і букет квітів.» | comprehension

7. `tongue_twister` — «Летіли лелеки ліловими лелеками, летіли лелеки летіли.» (Grade 3 p.63)

#### Module 7: `davalnyi-komu-chomu`
**Focus**: Dative concept — КОМУ? ЧОМУ? — giving, communicating, age
**Type**: `grammar`
**State standard**: §4.2.3.3 (dative)

**The key question**: КОМУ? ЧОМУ? — to whom? for whom?

**Dative formation** (Grade 4 p.35):

| Gender | Nominative | Dative | Pattern |
|---|---|---|---|
| Masc hard | брат | братові / брату | +ові / +у |
| Masc soft | учитель | учителеві / учителю | +еві / +ю |
| Fem -а | мама | мамі | -а → -і |
| Fem -я | земля | землі | -я → -і |
| Neut -о | місто | місту | -о → -у |
| Neut -е | море | морю | -е → -ю |

**Note**: Masculine has TWO dative forms — -ові/-еві is preferred for animate nouns, -у/-ю is alternative.

**Core meanings**:
- **Giving/communicating**: Дати книгу брат**ові**. Написати лист мам**і**. Сказати друг**ові**.
- **Age**: Мені двадцять років. Бабус**і** вісімдесят. Дідус**еві** вісімдесят п'ять.
- **State/feeling** (impersonal): Мені холодно. Йому сумно. Їй весело.
- **Need**: Мені потрібно / треба + infinitive: Мені потрібно працювати.

**Vocabulary** (20 words — giving/communicating context):
- дарувати (to give as gift), допомагати (to help), показувати (to show), пояснювати (to explain), відповідати (to answer), обіцяти (to promise), повідомляти (to inform), дзвонити (to call)

**Activities**:

1. `pattern_drill` — dative formation: «брат → дати...» → «братові» (15 nouns)
2. `build_sentence` — giving: «Я дарую [що?] [кому?]» (8 sentences)
3. `pattern_drill` — age: «бабуся, 80» → «Бабусі вісімдесят років.»
4. `true_false` — «Мені холодно» → true (dative + state = correct)
5. `classify` — dative endings: -ові/-еві (masc) vs -і (fem) vs -у/-ю (neut)
6. `reading` — birthday party description: giving gifts to family members | questions: Що Оля подарувала мамі? Кому Тарас дав книгу?

#### Module 8: `davalnyi-osobovi`
**Focus**: Personal pronoun dative — мені, тобі, йому, їй, нам, вам, їм
**Type**: `grammar`
**State standard**: §4.2.1.4 (pronoun declension — dative)

**Personal pronoun dative forms**:

| Nominative | Dative | Example |
|---|---|---|
| я | мені | Мені подобається. |
| ти | тобі | Тобі потрібно відпочити. |
| він | йому | Йому двадцять років. |
| вона | їй | Їй холодно. |
| ми | нам | Нам треба йти. |
| ви | вам | Вам допомогти? |
| вони | їм | Їм весело. |

**Usage patterns**:
- Подобається (A1 review with dative pronouns): Мені подобається Київ. Тобі подобається музика?
- Потрібно / треба: Мені потрібно купити хліб. Їй треба працювати.
- State: Мені тепло. Йому сумно. Нам весело.
- Communication: Скажи мені. Я напишу тобі. Він подзвонив їй.

**Activities**:

1. `pattern_drill` — fill pronoun dative: «я → подобається...» → «мені подобається» (7 pronouns)
2. `build_sentence` — state expressions: «[pronoun-dat] + [стан]» (мені холодно, йому весело, etc.)
3. `pattern_drill` — need: «я + працювати» → «Мені потрібно працювати.»
4. `true_false` — «Він подзвонив вона» → false (should be: їй)
5. `classify` — мені/тобі/йому/їй/нам/вам/їм — match to nominative
6. `reading` — phone conversation: two friends discuss what they need to do, how they feel, what they like | comprehension

#### Module 9: `davalnyi-praktyka`
**Focus**: Dative consolidation + contrast with genitive
**Type**: `practice`
**State standard**: §4.2.3.3 (all dative uses)

**Cross-case practice**: genitive (КОГО? ЧОГО?) vs dative (КОМУ? ЧОМУ?):
- Книга брат**а** (Р.в. — whose?) vs Дати книгу брат**ові** (Д.в. — to whom?)

**Activities**:

1. `classify` — sort 16 phrases: genitive (КОГО? ЧОГО?) vs dative (КОМУ? ЧОМУ?)
2. `pattern_drill` — mixed: same noun in genitive vs dative (contrast drill)
3. `build_sentence` — complex: «Я купив подарунок для [мама-Р.в.] і подарував [мама-Д.в.]»
4. `reading` — market scene: buying, giving, asking — multiple cases in context
5. `proverb_drill` — «Друзі пізнаються в біді.» (Grade 3)
6. `pattern_drill` — verb government: which verbs take dative (дати, допомогти, сказати, подзвонити)?

#### Module 10: `orudnyi-kym-chym`
**Focus**: Instrumental concept — КИМ? ЧИМ? — means, instrument, profession
**Type**: `grammar`
**State standard**: §4.2.3.5 (instrumental)

**The key question**: КИМ? ЧИМ? — with what? as what?

**Instrumental formation** (Grade 4 p.37):

| Gender | Nominative | Instrumental | Pattern |
|---|---|---|---|
| Masc hard | брат | братом | +ом |
| Masc soft | учитель | учителем | +ем |
| Fem -а | мама | мамою | -а → -ою |
| Fem -я | земля | землею | -я → -ею |
| Neut -о | місто | містом | -о → -ом |
| Neut -е | море | морем | -е → -ем |

**Core meanings**:
- **Instrument/means**: писати ручк**ою**, різати нож**ем**, їхати автобус**ом**
- **Profession** (бути/стати + instr): Він є лікар**ем**. Вона стала вчительк**ою**.
- **Object of interest**: захоплюватися музик**ою**, цікавитися історі**єю**

**Vocabulary** (20 words — instrument/profession context):
- ніж, виделка, ложка, олівець, ручка, пензлик (brush), молоток (hammer)
- Transport as instrument: літаком, поїздом, автобусом, таксі, велосипедом
- Profession expansion: адвокат, архітектор, журналіст, перекладач

**Activities**:

1. `pattern_drill` — instrumental formation: «ручка → пишу...» → «ручкою» (15 nouns)
2. `build_sentence` — means: «Я їду [чим?]» — поїздом, автобусом, літаком, etc.
3. `pattern_drill` — profession: «лікар → він став...» → «лікарем»
4. `true_false` — «Я пишу ручка» → false (ручкою)
5. `classify` — instrumental meanings: instrument / transport / profession
6. `reading` — "A day at work" description using instrumental: tools used, transport taken, profession held | comprehension

#### Module 11: `orudnyi-priymennyky`
**Focus**: Instrumental with prepositions — з/із, за, між, над, під, перед
**Type**: `grammar`
**State standard**: §4.2.3.5 (instrumental prepositions)

**Preposition + instrumental patterns**:

| Preposition | Meaning | Example |
|---|---|---|
| з / із | with (companionship) | кава **з** молок**ом**, гуляти **з** друг**ом** |
| з / із | attribute | дівчина **з** довгим волосся**м** |
| за | behind | **за** будинк**ом**, **за** стол**ом** |
| між | between | **між** школ**ою** і парк**ом** |
| над | above | лампа **над** стол**ом** |
| під | under | кіт **під** стол**ом** |
| перед | in front of / before | **перед** будинк**ом**, **перед** обід**ом** |

**Key contrast** — companionship (з + instr) vs origin (з + gen):
- Я іду **з друг**ом** (О.в. — with a friend)
- Я повернувся **з Києв**а** (Р.в. — from Kyiv)

**Activities**:

1. `classify` — sort prepositions by meaning: companionship / location / time
2. `pattern_drill` — location: «стіл → кіт під...» → «під столом» (12 items)
3. `build_sentence` — describe a room: «Лампа над столом. Кіт під стільцем. Книга між вікном і стіною.»
4. `true_false` — «Він гуляє з друга» → false (should be: з другом — instrumental, not genitive)
5. `pattern_drill` — companionship vs origin: «друг → гуляю з...» → «другом» / «Київ → їду з...» → «Києва»
6. `reading` — room description with spatial prepositions | draw/identify positions

#### Module 12: `orudnyi-osobovi`
**Focus**: Personal pronoun instrumental — мною, тобою, ним, нею, нами, вами, ними
**Type**: `grammar`
**State standard**: §4.2.1.4 (pronoun declension — instrumental)

**Personal pronoun instrumental forms**:

| Nominative | Instrumental | Example |
|---|---|---|
| я | мною | зі мною |
| ти | тобою | з тобою |
| він | ним | з ним |
| вона | нею | з нею |
| ми | нами | з нами |
| ви | вами | з вами |
| вони | ними | з ними |

**н- prefix rule** (Grade 4 p.135): After preposition → нього, ній, ним, нею, них, ними:
- Він → з **н**им (not: з ім)
- Вона → з **н**ею (not: з ою)

**Activities**:

1. `pattern_drill` — «я → гуляти зі...» → «зі мною» (7 pronouns)

#### Module 14: `vyd-diyeslova`
**Focus**: Aspect pairs — imperfective (process) vs perfective (result)
**Type**: `grammar`
**State standard**: §4.3.1 (aspect pairs: робити–зробити, писати–написати)

**Aspect as meaning** (not grammar label):
- **Процес** (ongoing/repeated): Я **читаю** книгу. (reading, process)
- **Результат** (completed/single): Я **прочитаю** книгу. (will finish reading)

**Core aspect pairs** (Grade 4 p.97+):

| Imperfective | Perfective | Meaning |
|---|---|---|
| читати | прочитати | read / finish reading |

| їсти | з'їсти | eat / eat up |
| пити | випити | drink / drink up |
| вчити | вивчити | study / learn (master) |
| ходити | піти | walk (habitual) / go (depart) |
| їздити | поїхати | travel (habitual) / depart |

- Imperfective: завжди, часто, іноді, зараз, кожного дня, щоранку

#### Module 16: `rukh-diyeslova`
**Focus**: Motion verb pairs — іти/ходити, їхати/їздити, летіти/літати, нести/носити
**Type**: `grammar`
**State standard**: §4.3.1 (motion verb pairs: іти–ходити, їхати–їздити, летіти–літати)

**Determinate vs indeterminate motion**:
- **Determinate** (one direction, right now): Я **іду** в школу. Він **їде** до Києва.
- **Indeterminate** (habitual/multi-directional): Я **ходжу** в школу щодня. Він **їздить** до Києва часто.

| Determinate | Indeterminate | Meaning |
|---|---|---|
| іти | ходити | go on foot |
| їхати | їздити | go by vehicle |
| летіти | літати | fly |
| нести | носити | carry (on foot) |
| везти | возити | carry (by vehicle) |
| бігти | бігати | run |

**Activities**:

1. `classify` — sort 12 sentences: зараз (determinate) vs завжди/часто (indeterminate)
2. `pattern_drill` — choose: «Я ___ в школу зараз» → іду / «Я ___ в школу щодня» → ходжу
3. `build_sentence` — describe current vs habitual: «Зараз я їду... / Зазвичай я їздить...»
4. `true_false` — «"Я ходжу в магазин" — це зараз» → false (habitual)
5. `pattern_drill` — all pairs: determinate → indeterminate and vice versa
6. `reading` — travel blog: mixing determinate (this trip) and indeterminate (usually) | comprehension

#### Module 18: `maybutnie-syntetychne`
**Focus**: Synthetic (simple) future — -тиму/-тимеш/-тиме/-тимемо/-тимете/-тимуть

**State standard**: §4.2.3 (synthetic future), §4.3.1 (perfective simple future)
**Textbook source**: Grade 4 (Kravtsova p.94 — dual future forms), Grade 5 (expanded conjugation)

**Two ways to express future in Ukrainian**:

| Type | Formation | Example | When to use |
|---|---|---|---|
| Compound (A1) | буду + infinitive | буду читати | Imperfective — ongoing future |
| Synthetic | stem + -тиму/-меш... | читатиму | Imperfective — same meaning, more colloquial |
| Perfective simple | conjugated perfective | прочитаю | Perfective — completed future |

**Synthetic future formation** (imperfective verbs):

| Person | -ти + suffix | читати | писати | говорити |
|---|---|---|---|---|
| я | -тиму | читатиму | писатиму | говоритиму |
| ти | -тимеш | читатимеш | писатимеш | говоритимеш |
| він/вона | -тиме | читатиме | писатиме | говоритиме |
| ми | -тимемо | читатимемо | писатимемо | говоритимемо |
| ви | -тимете | читатимете | писатимете | говоритимете |
| вони | -тимуть | читатимуть | писатимуть | говоритимуть |

**Perfective simple future** (conjugates like present tense of perfective verbs):

| Perfective verb | я | ти | він/вона | ми | ви | вони |
|---|---|---|---|---|---|---|
| прочитати | прочитаю | прочитаєш | прочитає | прочитаємо | прочитаєте | прочитають |
| написати | напишу | напишеш | напише | напишемо | напишете | напишуть |
| сказати | скажу | скажеш | скаже | скажемо | скажете | скажуть |

**Vocabulary** (20 words — future-oriented):

**Activities**:

1. `pattern_drill` — synthetic future: «читати → я...» → «читатиму» (12 verbs × 6 persons)

#### Module 31: `cherhuvannia-holosni`
**Focus**: Vowel alternations — [і]↔[о], [і]↔[е], fleeting vowels
**Type**: `grammar`
**State standard**: §4.1 (vowel alternation: ніс–носа, річ–речі, [о]/[е] with [і])

**Key vowel alternation patterns**:
- [і] ↔ [о]: ніс → носа, рік → року, стіл → стола
- [і] ↔ [е]: річ → речі, осінь → осені
- Fleeting [о]/[е] with zero: сон → сну, день → дня, вікно → вікон

**Why it matters**: These alternations happen when you change case/number — learners must recognize that ніс and носа are the SAME WORD.

**Activities**:

1. `pattern_drill` — alternation pairs: «ніс → Р.в.» → «носа» (15 pairs)
2. `classify` — sort by alternation type: [і]↔[о] vs [і]↔[е] vs fleeting
3. `true_false` — «"Ніс" і "носа" — це різні слова» → false (one word, different forms)
4. `build_sentence` — use alternating forms in sentences
5. `pattern_drill` — reverse: «носа → Н.в.» → «ніс» (recognition drill)
6. `reading` — text with many alternating words | identify base forms

#### Module 32: `cherhuvannia-pryholosni`
**Focus**: Consonant alternations — [г]/[к]/[х] → [ж]/[ч]/[ш] and [з']/[ц']/[с']
**Type**: `grammar`
**State standard**: §4.1 (consonant alternation/mutation)

**Key consonant alternation patterns**:
- In locative/dative: рука → у руці [к→ц'], нога → на нозі [г→з'], муха → у мусі [х→с']
- In vocative: друг → друже [г→ж], козак → козаче [к→ч]
- In conjugation: ходити → ходжу [д→дж], носити → ношу [с→ш], сидіти → сиджу [д→дж]

**Why it matters**: These alternations are NOT optional — they happen automatically. Learners must produce them.

**Activities**:

1. `pattern_drill` — locative: «рука → на...» → «руці» / «нога → на...» → «нозі» (12 items)
2. `pattern_drill` — vocative: «друг → !» → «друже» / «козак → !» → «козаче» (8 items)
3. `pattern_drill` — conjugation: «ходити → я...» → «ходжу» (10 verbs)
4. `classify` — sort by alternation: [г→з'→ж] vs [к→ц'→ч] vs [х→с'→ш]
5. `true_false` — «У руку — це місцевий відмінок» → false (у руці)
6. `reading` — text featuring many alternating consonants | identify the base consonant

#### Module 33: `pryslivnyky-osnovy`
**Focus**: Adverbs — formation, common patterns, question word ЯК?
**Type**: `grammar`
**State standard**: §4.5 (stylistic expressiveness), implicit in thematic areas
**Textbook source**: Grade 5 (Avramenko — adverb formation from adjectives), Grade 3 (common adverbs in context)

**The key question**: ЯК? (How?) — extends the question-word grammar system

**Adverb formation from adjectives** (Grade 5):

| Adjective | Adverb | Pattern |
|---|---|---|
| швидкий (fast) | швидко | -ий → -о |
| повільний (slow) | повільно | -ий → -о |
| гарний (good/nice) | гарно | -ий → -о |
| поганий (bad) | погано | -ий → -о |
| тихий (quiet) | тихо | -ий → -о |
| голосний (loud) | голосно | -ий → -о |

**Common adverbs by category**:

| Category | Adverbs |
|---|---|
| ЯК? (manner) | добре, погано, швидко, повільно, тихо, голосно, гарно, правильно |
| ДЕ? (place) | тут, там, далеко, близько, вгорі, внизу, ліворуч, праворуч |
| КОЛИ? (time) | зараз, потім, раніше, пізніше, давно, завжди, ніколи, іноді |
| СКІЛЬКИ? (degree) | дуже, трохи, багато, мало, зовсім, майже, ледве |

**Vocabulary** (30 words — all the adverbs above + related):
- правильно/неправильно, уважно (attentively), обережно (carefully), спокійно (calmly)
- Comparative: швидше, повільніше, краще, гірше

**Activities**:

1. `pattern_drill` — adjective → adverb: «швидкий → ?» → «швидко» (15 pairs)
2. `classify` — sort 20 adverbs: ЯК? vs ДЕ? vs КОЛИ? vs СКІЛЬКИ?
3. `build_sentence` — add adverb to sentence: «Він говорить [як?]» → «Він говорить голосно.»
4. `true_false` — «"Швидкий" — це прислівник» → false (adjective; швидко = adverb)
5. `pattern_drill` — comparative adverbs: «швидко → ще...» → «швидше» (8 pairs)
6. `reading` — description of someone's day using many adverbs | identify all adverbs and categorize

#### Module 34: `skladni-rechennia`
**Focus**: Complex sentences — що, щоб, де, куди, тому що, бо
**Type**: `grammar`
**State standard**: §4.4 (complex sentences with що, щоб, де/куди/звідки, тому що/бо)

**Subordinate conjunctions** (expanding A1's і/а/але):

| Conjunction | Meaning | Example |
|---|---|---|
| що | that | Я знаю, **що** він прийде. |
| щоб | in order to | Я вчуся, **щоб** знати мову. |
| де | where | Я знаю, **де** він живе. |
| куди | where to | Скажи, **куди** ти йдеш. |
| звідки | where from | Я не знаю, **звідки** він. |
| тому що / бо | because | Я не пішов, **тому що** йшов дощ. |
| коли | when | Я прийду, **коли** закінчу. |

**Key punctuation**: Comma BEFORE the conjunction in Ukrainian complex sentences.

**Activities**:

1. `classify` — sort conjunctions: мета (purpose: щоб) vs причина (cause: бо/тому що) vs місце (place: де/куди) vs час (time: коли)
2. `build_sentence` — combine two simple sentences into one complex: «Я вчуся. Я хочу знати мову.» → «Я вчуся, щоб знати мову.»
3. `pattern_drill` — fill conjunction: «Я не прийшов, ___ був хворий» → «тому що»
4. `true_false` — «"Я знаю що він тут" — пунктуація правильна» → false (comma before що)
5. `build_sentence` — answer with complex sentence: «Чому?» → «Тому що...» / «Для чого?» → «Щоб...»
6. `reading` — passage with multiple subordinate clauses | identify main vs subordinate
