## Pattern Analysis

### Critical Issues Identified

#### 1. **Formulaic "Унікальний регіон" Pattern (M72-M75)**

**Pattern:**
```
[Region] — це унікальний регіон з [adjective] [noun]
```
**Occurrences:**
- M72: "унікальний регіон з надзвичайно багатою спадщиною"
- M73: "унікальний історичний регіон зі славною козацькою спадщиною"
- M74: "унікальний регіон широких степів"

**Problem:** Students will notice this exact template repeated across 3 consecutive modules.

**Fix approach:** Vary structures:
- "Це край [noun]"
- "Регіон відомий [instrumental]"
- Remove "унікальний" entirely from some sentences

---

#### 2. **Double Superlative Redundancy (M75)**

**Pattern:**
```
[Subject] — це [найA та найB] [noun]
```
**Occurrences:**
- "найвидатніший та найвідоміший поет"
- "найбільша та найголовніша річка"

**Problem:** Semantically redundant (if most outstanding, already most famous; if largest, already most important)

**Fix approach:**
- Use single superlative + qualifier: "найвидатніший український поет"
- Use comparative context: "найбільша річка України, яка..."

---

#### 3. **Missing Discourse Markers**

**Current state:** 90% of unjumble sentences = independent facts with no connectors

**Problem:** Trains students to produce disconnected declarative statements

**Fix approach:** Add discourse markers:
- **Opposition:** "але", "проте", "однак"
- **Addition:** "також", "крім того", "до того ж"
- **Causation:** "тому", "через це", "адже"
- **Sequence:** "спочатку", "потім", "нарешті"

**Example fix (M72):**
```
Before: Львів відомий усьому світу своєю унікальною кавовою культурою.
After: Львів також відомий своєю особливою кавовою культурою.
```

---

#### 4. **Excessive Intensifiers**

**Problematic words (overused):**
- "дуже" (5+ occurrences per module)
- "надзвичайно"
- "справжній/справжня"
- "унікальний"
- "особливий"

**Fix approach:**
- Remove 50% of intensifiers
- Replace with specific details: "дуже великий" → "площею 12 гектарів"

## Technical Notes

**Scan methodology:**
- Protocol: claude_extensions/protocols/a1-naturalness-scan.md (adapted for B1)
- Activity types analyzed: cloze, fill-in, unjumble (5+ sentences)
- Excluded: quiz, true-false, match-up, group-sort, select, error-correction, translate, mark-the-words
- Naturalness criteria:
  1. Subject consistency and variety
  2. Discourse markers (а, але, потім, тому, також)
  3. Topic coherence across sentences
  4. Avoidance of redundancy and pleonasms

**Vocabulary constraint:** All fixes must use ONLY words from B1 M01-M75 cumulative vocabulary.

**Grammar constraint:** All fixes must use ONLY structures introduced in B1 curriculum plan.
