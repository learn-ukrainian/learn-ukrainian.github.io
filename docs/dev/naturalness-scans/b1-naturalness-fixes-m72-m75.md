# B1 Naturalness Fixes - M72-M75 (Regional Modules)
**Date:** 2026-01-12
**Status:** COMPLETED
**Modules affected:** M72 (Західна Україна), M73 (Східна Україна), M74 (Південна Україна), M75 (Центральна Україна)

---

## Summary

Applied naturalness improvements to 4 regional modules (M72-M75) based on findings from the B1 M72-M86 naturalness scan. Total changes: **16 unjumble sentences** across 4 modules.

**Key improvements:**
- Eliminated the repetitive "унікальний регіон" template pattern
- Removed double superlatives (redundant intensifiers)
- Reduced excessive use of intensifiers ("дуже", "надзвичайно", "справжній", "заслужено")
- Added discourse marker "також" to create better flow
- Simplified overly ornate constructions

**Vocabulary constraint:** All changes use ONLY B1 M01-M75 cumulative vocabulary.

## M72: Західна Україна

### Changes Applied (5 sentences)

#### 1. Removed "по праву" + "справжньою"
**Before:**
```
Львів по праву вважається справжньою культурною столицею сучасної України.
```
**After:**
```
Львів вважається культурною столицею сучасної України.
```
**Rationale:** "По праву" and "справжньою" are redundant intensifiers. Simpler sentence achieves same meaning.

---

#### 2. Broke "унікальний регіон" template
**Before:**
```
Це унікальний регіон з надзвичайно багатою культурною та історичною спадщиною.
```
**After:**
```
Регіон має багату культурну та історичну спадщину.
```
**Rationale:**
- Eliminates "унікальний регіон" cliché
- Removes "надзвичайно" intensifier
- Uses active verb "має" instead of descriptive "це"
- More direct and natural

---

#### 3. Removed intensifier "дуже" + "сучасній"
**Before:**
```
Габсбурзький період залишив дуже глибокий культурний слід в сучасній Галичині.
```
**After:**
```
Габсбурзький період залишив глибокий культурний слід в Галичині.
```
**Rationale:** "Глибокий" already strong; "дуже" adds no value. "Сучасній" unnecessary context.

---

#### 4. Added discourse marker "також" + changed "унікальною"
**Before:**
```
Львів відомий усьому світу своєю унікальною кавовою культурою.
```
**After:**
```
Львів також відомий своєю особливою кавовою культурою.
```
**Rationale:**
- "Також" creates connection to previous sentence about Lviv
- "Особливою" less clichéd than "унікальною"
- Removed "усьому світу" (hyperbolic)

---

#### 5. Removed intensifiers "високих" + "унікальні"
**Before:**
```
У високих Карпатах живуть три унікальні етнографічні групи українців.
```
**After:**
```
У Карпатах живуть три етнографічні групи українців.
```
**Rationale:** Fact that Carpathians are high is implicit. "Унікальні" overused in this module.

## M73: Східна Україна

### Changes Applied (2 sentences)

#### 6. Broke "унікальний регіон" template (2nd occurrence)
**Before:**
```
Слобожанщина — це унікальний історичний регіон зі славною козацькою спадщиною.
```
**After:**
```
Слобожанщина — історичний регіон зі славною козацькою спадщиною.
```
**Rationale:** Eliminates second occurrence of "унікальний ... регіон" pattern. Keeps "історичний" as factual descriptor.

---

#### 7. Fixed pleonasm "головною основою"
**Before:**
```
Важка промисловість завжди була головною основою економіки цього індустріального регіону.
```
**After:**
```
Важка промисловість завжди була основою економіки цього індустріального регіону.
```
**Rationale:** "Головна основа" is redundant (if it's the basis, it's already the main one).

## M74: Південна Україна

### Changes Applied (5 sentences)

#### 8. Broke "унікальний регіон" template (3rd occurrence)
**Before:**
```
Південна Україна — це унікальний регіон широких степів і теплих морів.
```
**After:**
```
Південна Україна — це край широких степів і теплих морів.
```
**Rationale:**
- Third occurrence of "унікальний регіон" — critical to break pattern
- "Край" (land/region) is more poetic and varied vocabulary

---

#### 9. Removed bookish "заслужено"
**Before:**
```
Миколаїв заслужено називають містом корабелів і центром суднобудування.
```
**After:**
```
Миколаїв називають містом корабелів і центром суднобудування.
```
**Rationale:** "Заслужено" (deservedly) is bookish register for B1. Fact speaks for itself.

---

#### 10. Fixed pleonasm "солодкими та смачними"
**Before:**
```
Сонячний Херсон славиться своїми солодкими та смачними кавунами.
```
**After:**
```
Сонячний Херсон славиться своїми смачними кавунами.
```
**Rationale:** Watermelons are inherently sweet; "солодкими та смачними" is pleonastic. "Смачними" sufficient.

---

#### 11. Removed "унікальний" (4th use in module)
**Before:**
```
Це унікальний багатонаціональний регіон різних культур та народів.
```
**After:**
```
Це багатонаціональний регіон різних культур та народів.
```
**Rationale:** "Унікальний" used 4 times in M72-M74. "Багатонаціональний" already descriptive enough.

---

#### 12. Removed "справжній" intensifier
**Before:**
```
Кримські татари — це справжній корінний народ півострова Крим.
```
**After:**
```
Кримські татари — це корінний народ півострова Крим.
```
**Rationale:** "Корінний" (indigenous) already means genuine/authentic. "Справжній" redundant.

---

## M75: Центральна Україна

### Changes Applied (4 sentences)

#### 13. Removed tautology "Стародавній... сучасна"
**Before:**
```
Стародавній Київ — це сучасна столиця та серце України.
```
**After:**
```
Київ — це столиця та серце України.
```
**Rationale:** "Стародавній... сучасна" creates awkward old/modern contrast. Simple statement better.

---

#### 14. Removed "справжнє" intensifier
**Before:**
```
Черкащина — це справжнє історичне серце козацької України.
```
**After:**
```
Черкащина — це історичне серце козацької України.
```
**Rationale:** "Справжнє" (true/genuine) adds no semantic value to "історичне серце."

---

#### 15. Fixed double superlative (Critical!)
**Before:**
```
Тарас Шевченко — це найвидатніший та найвідоміший український поет.
```
**After:**
```
Тарас Шевченко — це найвидатніший український поет.
```
**Rationale:**
- **Semantically redundant:** If most outstanding, already most famous
- Double superlatives are unnatural in Ukrainian
- Single superlative "найвидатніший" carries full meaning

---

#### 16. Fixed double superlative + "всієї"
**Before:**
```
Дніпро — це найбільша та найголовніша річка всієї України.
```
**After:**
```
Дніпро — це найбільша річка України.
```
**Rationale:**
- **Semantically redundant:** If largest, already most important
- "Всієї" (entire) adds unnecessary emphasis
- Simpler statement more natural

## Pattern Analysis

### Issues Eliminated

#### 1. "Унікальний регіон" Template (3 occurrences)
- M72: "унікальний регіон з надзвичайно багатою..."
- M73: "унікальний історичний регіон зі славною..."
- M74: "унікальний регіон широких степів..."

**Impact:** Students would have noticed identical structure across 3 consecutive modules.

---

#### 2. Double Superlatives (2 occurrences, both M75)
- "найвидатніший та найвідоміший"
- "найбільша та найголовніша"

**Impact:** Trains students to produce redundant constructions (violates economy principle).

---

#### 3. Excessive Intensifiers (9 occurrences removed)
- **"унікальний/унікальна/унікальні":** 5 uses
- **"дуже":** 1 use
- **"надзвичайно":** 1 use
- **"справжній/справжнє/справжня":** 3 uses
- **"заслужено":** 1 use
- **"по праву":** 1 use

**Impact:** Creates inflated, unnatural prose that sounds like marketing copy.

---

#### 4. Pleonasms (2 occurrences)
- "головна основа" (main basis)
- "солодкі та смачні кавуни" (sweet and tasty watermelons)

**Impact:** Trains students to use redundant word combinations.
