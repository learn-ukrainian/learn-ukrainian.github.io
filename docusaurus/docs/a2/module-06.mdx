## Warm-up

In previous modules, you learned about the instrumental case: with the preposition **–∑** for accompaniment and without a preposition for transport and tools. Now it's time to discover another important use of the instrumental case ‚Äî with the verbs **–±—É—Ç–∏** (to be), **—Å—Ç–∞—Ç–∏** (to become), and **–ø—Ä–∞—Ü—é–≤–∞—Ç–∏** (to work as)!

When you want to say "He is a doctor," "She became a teacher," or "I work as a programmer," Ukrainian uses the instrumental case. This is one of the most practical uses of the instrumental ‚Äî talking about what people are and what they do!

**Why is this important?** Talking about professions is a fundamental skill in any language. By the end of this module, you'll be able to describe your job, discuss career changes, and ask others about their work ‚Äî all using proper Ukrainian grammar.

## Ukrainian Professions: Gender Forms

One of the distinctive features of Ukrainian is that it has separate feminine forms for most professions. While English uses "doctor" for both men and women, Ukrainian distinguishes: **–ª—ñ–∫–∞—Ä** for a male doctor and **–ª—ñ–∫–∞—Ä–∫–∞** for a female doctor.

This is not a modern invention ‚Äî Ukrainian has always had productive suffixes like -–∫–∞, -–∏—Ü—è, and -–∏–Ω—è that form feminine nouns from masculine ones. Using the correct gender form is considered grammatically proper and respectful.

The main suffixes for feminine professions are:

- **-–∫–∞**: –ª—ñ–∫–∞—Ä ‚Üí –ª—ñ–∫–∞—Ä–∫–∞, —ñ–Ω–∂–µ–Ω–µ—Ä ‚Üí —ñ–Ω–∂–µ–Ω–µ—Ä–∫–∞
- **-–∏—Ü—è**: –ø–∏—Å—å–º–µ–Ω–Ω–∏–∫ ‚Üí –ø–∏—Å—å–º–µ–Ω–Ω–∏—Ü—è

Here is a table of common professions with their masculine and feminine forms:

| Masculine  | Feminine     | English    |
| ---------- | ------------ | ---------- |
| –ª—ñ–∫–∞—Ä      | –ª—ñ–∫–∞—Ä–∫–∞      | doctor     |
| –≤—á–∏—Ç–µ–ª—å    | –≤—á–∏—Ç–µ–ª—å–∫–∞    | teacher    |
| –∂—É—Ä–Ω–∞–ª—ñ—Å—Ç  | –∂—É—Ä–Ω–∞–ª—ñ—Å—Ç–∫–∞  | journalist |
| —ñ–Ω–∂–µ–Ω–µ—Ä    | —ñ–Ω–∂–µ–Ω–µ—Ä–∫–∞    | engineer   |
| –ø—Ä–æ–≥—Ä–∞–º—ñ—Å—Ç | –ø—Ä–æ–≥—Ä–∞–º—ñ—Å—Ç–∫–∞ | programmer |
| —é—Ä–∏—Å—Ç      | —é—Ä–∏—Å—Ç–∫–∞      | lawyer     |
| –∞–∫—Ç–æ—Ä      | –∞–∫—Ç–æ—Ä–∫–∞      | actor      |
| –ø–∏—Å—å–º–µ–Ω–Ω–∏–∫ | –ø–∏—Å—å–º–µ–Ω–Ω–∏—Ü—è  | writer     |

When talking about a woman's profession, use the feminine form in both nominative and instrumental:

–í–æ–Ω–∞ –ø—Ä–∞—Ü—é—î –ª—ñ–∫–∞—Ä–∫–æ—é. (She works as a doctor.)

> üîç **Myth Buster**
>
> Some people claim feminine profession forms like "–ª—ñ–∫–∞—Ä–∫–∞" or "—ñ–Ω–∂–µ–Ω–µ—Ä–∫–∞" are "new" or "artificial." This is false! Ukrainian has always had productive feminine suffixes. Using –ª—ñ–∫–∞—Ä–∫–∞ instead of –ª—ñ–∫–∞—Ä for a woman is grammatically correct and culturally authentic!

## Practice

## Present vs. Past/Future

This is the key distinction you need to master: in present tense, Ukrainian doesn't use any verb for "to be" with professions. You simply state the subject and profession in nominative case. But in past and future tenses, you use –±—É—Ç–∏ followed by the profession in instrumental case.

Remember: Present tense = no verb, Nominative. Past/Future = –±—É—Ç–∏, Instrumental. This is a fundamental pattern in Ukrainian that differs from English where "is" appears in all tenses.

| -------------------- | ------------------------- | ------------------------- |
