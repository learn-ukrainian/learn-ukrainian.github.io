## Warm-up

In Module 31, we used prefixes to move (GPS).
Now, we use prefixes to **transform** actions.
Imagine the verb **–ø–∏—Å–∞—Ç–∏** (to write). It's just a neutral process.
But what if you want to say:
- "I wrote it **down**" (recorded it)?
- "I wrote it **again**" (redid it)?
- "I wrote **a little bit**"?
- "I **signed** it"?

In English, you use phrasal verbs or extra words.
In Ukrainian, you just click a LEGO piece (prefix) onto the verb!
**–ü–∏—Å–∞—Ç–∏** -> **–ó–∞**–ø–∏—Å–∞—Ç–∏, **–ü–µ—Ä–µ**–ø–∏—Å–∞—Ç–∏, **–ü–æ**–ø–∏—Å–∞—Ç–∏, **–ü—ñ–¥**–ø–∏—Å–∞—Ç–∏.

:::note[Think about it]
A prefix changes the "angle" of the action.
- **–ü–µ—Ä–µ-** usually means "re-" (redo) or "across".
- **–î–æ-** usually means "finish" (to the end).
- **–ó–∞-** usually means "close" or "fix/record".
:::

:::note[Cultural Insight: Hospitality]
Prefixes are everywhere in Ukrainian culture, especially hospitality.
When guests arrive, you **–≤—ñ–¥–∫—Ä–∏—Ç–∏** (open) the door and the wine.
You **–Ω–∞–∫—Ä–∏—Ç–∏** (cover/set) the table ("–Ω–∞–∫—Ä–∏—Ç–∏ –Ω–∞ —Å—Ç—ñ–ª").
You **–ø–æ–≥–æ–≤–æ—Ä–∏—Ç–∏** (have a chat).
And when they leave, you **–ø—Ä–æ–≤–µ—Å—Ç–∏** (lead through/escort) them to the door.
It's a sequence of prefixed actions!
:::

# Deep Dive: Open and Close (–í—ñ–¥–∫—Ä–∏—Ç–∏ / –ó–∞–∫—Ä–∏—Ç–∏)

These are the most useful practical verbs.
Unlike English ("turn on/off" vs "open/close"), Ukrainian logic is consistent.

- **Open:** **–í—ñ–¥**–∫—Ä–∏—Ç–∏ (Move *away* the cover).
- **Close:** **–ó–∞**–∫—Ä–∏—Ç–∏ (Move *behind* the cover).
- **Turn on:** **–í**–º–∏–∫–∞—Ç–∏ (Move *in*).
- **Turn off:** **–í–∏**–º–∏–∫–∞—Ç–∏ (Move *out*).

| Object | Action | Verb |
| :--- | :--- | :--- |
| Door/Window | Open | **–í—ñ–¥–∫—Ä–∏—Ç–∏** / **–í—ñ–¥—á–∏–Ω–∏—Ç–∏** |
| Door/Window | Close | **–ó–∞–∫—Ä–∏—Ç–∏** / **–ó–∞—á–∏–Ω–∏—Ç–∏** |
| Book | Open | **–†–æ–∑–∫—Ä–∏—Ç–∏** (Spread open) |
| Book | Close | **–ó–∞–∫—Ä–∏—Ç–∏** / **–ó–≥–æ—Ä–Ω—É—Ç–∏** |
| Eyes | Open | **–†–æ–∑–ø–ª—é—â–∏—Ç–∏** (Specific!) |
| Eyes | Close | **–ó–∞–ø–ª—é—â–∏—Ç–∏** |

:::note[Open vs Unlock]
**–í—ñ–¥–∫—Ä–∏—Ç–∏** usually means to open access (a store, a bottle, a new land).
**–í—ñ–¥—á–∏–Ω–∏—Ç–∏** is specifically for physical barriers like doors and windows.
However, in modern Ukrainian, **–≤—ñ–¥–∫—Ä–∏—Ç–∏ –¥–≤–µ—Ä—ñ** is also common, though purists prefer **–≤—ñ–¥—á–∏–Ω–∏—Ç–∏**.
:::

# Deep Dive: To Give (The "Give" Family)

The verb **–î–∞—Ç–∏** (to give) is a prefix magnet.

1.  **–î–∞—Ç–∏** = To give (simple).
2.  **–í—ñ–¥–¥–∞—Ç–∏** = To give back / return.
    *   *–Ø **–≤—ñ–¥–¥–∞–≤** –±–æ—Ä–≥.* (I returned the debt).
3.  **–ü–µ—Ä–µ–¥–∞—Ç–∏** = To pass (transmit).
    *   *–ë—É–¥—å –ª–∞—Å–∫–∞, **–ø–µ—Ä–µ–¥–∞–π—Ç–µ** —Å—ñ–ª—å.* (Please pass the salt).
4.  **–ó–¥–∞—Ç–∏** = To hand in / To betray / To rent out.
    *   *–°—Ç—É–¥–µ–Ω—Ç **–∑–¥–∞–≤** —ñ—Å–ø–∏—Ç.* (Handed in/Passed exam).
    *   *–í—ñ–Ω **–∑–¥–∞–≤** –∫–≤–∞—Ä—Ç–∏—Ä—É.* (He rented out the apartment).
5.  **–ü–æ–¥–∞—Ç–∏** = To serve / hand politely.
    *   *–û—Ñ—ñ—Ü—ñ–∞–Ω—Ç **–ø–æ–¥–∞–≤** –º–µ–Ω—é.* (served the menu).
6.  **–í–∏–¥–∞—Ç–∏** = To issue / publish / betray.
    *   *–ë–∞–Ω–∫ **–≤–∏–¥–∞–≤** –∫–∞—Ä—Ç–∫—É.* (Issued).
    *   *–í—ñ–Ω **–≤–∏–¥–∞–≤** —Å–µ–∫—Ä–µ—Ç.* (Betrayed/Leaked).

# Deep Dive: Connect and Disconnect

In the age of technology, these are crucial.
Base: **–Ñ–¥–Ω–∞—Ç–∏** (to unite).

- **–ü—Ä–∏**—î–¥–Ω–∞—Ç–∏ = To attach/connect TO something. (Connect printer).
- **–í—ñ–¥**'—î–¥–Ω–∞—Ç–∏ = To detach/disconnect FROM something. (Unplug).
- **–ó**'—î–¥–Ω–∞—Ç–∏ = To connect TWO things together. (Join wires).
- **–†–æ–∑**'—î–¥–Ω–∞—Ç–∏ = To separate two things. (Cut connection).

*–ú–∏ **–ø—ñ–¥'—î–¥–Ω–∞–ª–∏** —ñ–Ω—Ç–µ—Ä–Ω–µ—Ç!* (We connected the internet!).

# Deep Dive: The Logic of "Na-" (Accumulation)

The prefix **–ù–∞-** often means "to put on" or "to accumulate/create a lot of".
- **–ü–∏—Å–∞—Ç–∏** (write) -> **–ù–∞–ø–∏—Å–∞—Ç–∏** (create a written text).
- **–ú–∞–ª—é–≤–∞—Ç–∏** (paint) -> **–ù–∞–º–∞–ª—é–≤–∞—Ç–∏** (create a painting).
- **–†–æ–±–∏—Ç–∏** (do) -> **–ù–∞—Ä–æ–±–∏—Ç–∏** (make a massive amount/mess).
    *   *–ú–∏ **–Ω–∞—Ä–æ–±–∏–ª–∏** –ø–æ–º–∏–ª–æ–∫.* (We made a ton of mistakes).
- **–á—Å—Ç–∏** (eat) -> **–ù–∞—ó—Å—Ç–∏—Å—è** (To eat one's fill / accumulate food inside).

# Deep Dive: The "Za-" Trap

The prefix **–ó–∞-** is the trickiest one. It has three opposite meanings!

1.  **Close / Behind:**
    *   **–ó–∞–∫—Ä–∏—Ç–∏** (Close).
    *   **–ó–∞—Ö–æ–≤–∞—Ç–∏** (Hide behind).
2.  **Start (Suddenly):**
    *   **–ó–∞—Å–ø—ñ–≤–∞—Ç–∏** (Start singing).
    *   **–ó–∞–ø–ª–∞–∫–∞—Ç–∏** (Burst into tears).
    *   **–ó–∞—Ö–≤–æ—Ä—ñ—Ç–∏** (Get/Fall sick).
3.  **Too Much (Excess):**
    *   **–ó–∞–ø—Ä–∞—Ü—é–≤–∞—Ç–∏—Å—è** (Work too much / lose track of time).
    *   **–ó–∞–≥—É–ª—è—Ç–∏—Å—è** (Party/Walk too long).
    *   **–ó–∞—Å–∏–¥—ñ—Ç–∏—Å—è** (Sit too long / stay up late).

:::warning[Context Matters]
- *–í—ñ–Ω **–∑–∞–≥–æ–≤–æ—Ä–∏–≤**.* = He started speaking.
- *–í—ñ–Ω –º–µ–Ω–µ **–∑–∞–≥–æ–≤–æ—Ä–∏–≤**.* = He tired me with too much talking.
The context tells you if it's "Start" or "Too Much".
:::

# Anti-Propaganda Showcase

:::note[Myth: "Ukrainian vocabulary is small."]
**Fact:** This myth comes from Soviet dictionaries that artificially limited defining Ukrainian words to match Russian ones.
In reality, Ukrainian prefixation allows for nuances that English needs sentences to explain.
Example: **–ó–∞–≥–æ–≤–æ—Ä–∏—Ç–∏** (to start speaking / to charm someone / to tire someone with talk).
**–ù–∞–≥–æ–≤–æ—Ä–∏—Ç–∏** (to say a lot of things / to slander).
**–í—ñ–¥–≥–æ–≤–æ—Ä–∏—Ç–∏** (to dissuade).
**–ü—ñ–¥–≥–æ–≤–æ—Ä–∏—Ç–∏** (to instigate).
One root "–≥–æ–≤–æ—Ä–∏—Ç–∏" creates dozens of distinct concepts. This is linguistic wealth, not poverty.
:::

1.  **Prefixes transform meaning.** They act like modifiers (Re-, Sub-, Out-).
2.  **–ü–µ—Ä–µ-** = Redo / Across.
3.  **–î–æ-** = Finish.
4.  **–ó–∞-** = Close / Record / Start / Excess.
5.  **–í—ñ–¥-** = Open / Away.
6.  **–ü–æ-** = A bit / Start.
7.  **–ù–∞-** = Accumulate / Create.
