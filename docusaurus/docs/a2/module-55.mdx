**Grammar Check** "Aspect"
Remember the difference?
- **–Ø —á–∏—Ç–∞–≤ –∫–Ω–∏–≥—É** (Imperfective) = Process (I was reading).
- **–Ø –ø—Ä–æ—á–∏—Ç–∞–≤ –∫–Ω–∏–≥—É** (Perfective) = Result (I have finished reading).
In stories, use Imperfective for background (*–°–æ–Ω—Ü–µ —Å–≤—ñ—Ç–∏–ª–æ...*) and Perfective for main actions (*–í—ñ–Ω –≤—ñ—Å—Ç–∞–≤, –ø–æ–º–∏–≤—Å—è, –ø—ñ—à–æ–≤...*).
