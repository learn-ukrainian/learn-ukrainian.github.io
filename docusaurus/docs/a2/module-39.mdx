## Warm-up

Welcome back to the Word Factory!
Last time, we made Nouns. Today, we make **Adjectives**.
Adjectives describe *what kind* of thing something is.

In English, you use:
- **-al** (Music -> Musical)
- **-ic** (History -> Historic)
- **-an** (America -> American)
- **-y** (Sun -> Sunny)

In Ukrainian, we have standard suffixes too.
Input: **–ú—É–∑–∏–∫–∞** [Noun] -> Machine -> Output: **–ú—É–∑–∏—á-–Ω–∏–π** [Adj].
Input: **–£–∫—Ä–∞—ó–Ω–∞** [Noun] -> Machine -> Output: **–£–∫—Ä–∞—ó–Ω-—Å—å–∫–∏–π** [Adj].

:::note[The Universal Endings]
Remember that the suffix changes the *meaning*, but the *ending* (flexion) must still match the Gender!
- –ú—É–∑–∏—á–Ω**–∏–π** (Masc)
- –ú—É–∑–∏—á–Ω**–∞** (Fem)
- –ú—É–∑–∏—á–Ω**–µ** (Neut)
- –ú—É–∑–∏—á–Ω**—ñ** (Plural)

This is exactly like regular adjectives (Good -> –ì–∞—Ä–Ω–∏–π, –ì–∞—Ä–Ω–∞, –ì–∞—Ä–Ω–µ).
The suffix (-–Ω-, -–æ–≤-, -—Å—å–∫-) just sits between the Root and the Ending.
Root + Suffix + Ending.
Think of it like a sandwich: Root is the meat, Suffix is the cheese, Ending is the bread.
:::

## Grammar Breakdown: 4 Rules to Remember

Before we look at more examples, let's clarify the logic.
Unlike English, which uses many different patterns (-al, -ic, -y, -ous, -an), Ukrainian is surprisingly consistent.

**Rule 1: The Generalist (-–Ω–∏–π) is King**
If you don't know which suffix to use, guess **-–Ω–∏–π**. It is the most common adjective suffix in the Ukrainian language.
It works for almost all abstract concepts (Theory -> Theoretical).

**Rule 2: Materials use -–æ–≤–∏–π**
If something is made of distinct parts or materials, it often uses **-–æ–≤–∏–π**.
Think of "Fruit" -> "Fruit-based cake". That's a "Fruit-ov-y" cake.
Think of "Science" -> "Science-based fact". That's a "Science-ov-y" fact.

**Rule 3: Softness Matters**
Sometimes **-–æ–≤–∏–π** becomes **-–µ–≤–∏–π** or **-—î–≤–∏–π**. Why?
Because of Ukrainian phonetics. After soft sounds or 'j' sounds (like in –ñ–∏—Ç—Ç—è), we need a soft vowel.
So *–ñ–∏—Ç—Ç-–æ–≤-–∏–π* sounds hard. *–ñ–∏—Ç—Ç-—î–≤-–∏–π* flows like water.

**Rule 4: People and Places use -—Å—å–∫-**
This is the "Belonging" suffix.
It answers: "Whose is it?" or "Where is it from?"
It is NOT about what it is made of. It is about *identity*.
- Kyiv -> Kyiv-an identity -> –ö–∏—ó–≤—Å—å–∫–∏–π.
- Student -> Student identity -> –°—Ç—É–¥–µ–Ω—Ç—Å—å–∫–∏–π.

:::tip[Sound Mutations]
When attaching suffixes, consonants often change to make pronunciation smoother:
- **–ö, –¶, –ß** + —Å—å–∫ -> **-—Ü—å–∫-** (–ö–æ–∑–∞–∫ -> –ö–æ–∑–∞—Ü—å–∫–∏–π).

- **–•, –®, –°** + —Å—å–∫ -> **-—Å—å–∫-** (–ß–µ—Ö -> –ß–µ—Å—å–∫–∏–π).
:::

## Deep Dive: Living Adjectives (The Active State)

Some adjectives describe a state or feeling.
- **–ì–∞—Ä—è—á–∏–π** (Hot) ‚Äî from "to burn".
- **–°–≤—ñ–∂–∏–π** (Fresh).
- **–ß—É–∂–∏–π** (Foreign/Alien).
- **–°–ø—Ä–∞–≤–∂–Ω—ñ–π** (Real/Genuine).

:::note[Why -—ñ–π?]
Soft adjectives end in **-—ñ–π** (masculine).
- **–°–∏–Ω—ñ–π**: Blue
- **–í–µ—á—ñ—Ä–Ω—ñ–π**: Evening
- **–û—Å—Ç–∞–Ω–Ω—ñ–π**: Last
They decline differently: *–°–∏–Ω—å–æ–≥–æ*, NOT *–°–∏–Ω–æ–≥–æ*.
:::

## Deep Dive: The Diminutive (-–µ–Ω—å–∫)

Just like nouns have **-–æ–∫**, adjectives have **-–µ–Ω—å–∫**.
It means "sweet", "little", or "pleasant".

- **–ú–∞–ª–∏–π** -> **–ú–∞–ª–µ–Ω—å–∫–∏–π** (Little/Small).
- **–ì–∞—Ä–Ω–∏–π** -> **–ì–∞—Ä–Ω–µ–Ω—å–∫–∏–π** (Pretty/Nice).
- **–¢–µ–ø–ª–∏–π** -> **–¢–µ–ø–ª–µ–Ω—å–∫–∏–π** (Nice and warm).
- **–°–≤—ñ–∂–∏–π** -> **–°–≤—ñ–∂–µ–Ω—å–∫–∏–π** (Fresh).
- **–ß–∏—Å—Ç–∏–π** -> **–ß–∏—Å—Ç–µ–Ω—å–∫–∏–π** (Clean).

*–ú–∞–º–∞ –∫—É–ø–∏–ª–∞ —Å–≤—ñ–∂–µ–Ω—å–∫–∏–π —Ö–ª—ñ–±.* (Mom bought fresh-little-bread).
*–¶–µ –º—ñ–π –º–∞–ª–µ–Ω—å–∫–∏–π –∫—ñ—Ç.* (This is my little cat).
