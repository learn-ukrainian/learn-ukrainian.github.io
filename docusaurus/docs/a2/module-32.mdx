# Summary

This module teaches **action verb prefixes** ‚Äî how prefixes modify non-motion verbs like "–ø–∏—Å–∞—Ç–∏" (write), "—Ä–æ–±–∏—Ç–∏" (do), "–¥–∞–≤–∞—Ç–∏" (give). These prefixes add meanings like completion, beginning, repetition, and distribution.

## Cultural Insight: Precision üá∫üá¶

:::
