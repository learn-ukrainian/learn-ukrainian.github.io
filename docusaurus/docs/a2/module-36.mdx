## Aspect: Process ‚Üí Result

Why use prefixed verbs instead of just the base verb? The prefix changes the aspect from imperfective (process) to perfective.

**Base verbs:**
- **–π—Ç–∏** = walking process
- **—ó—Ö–∞—Ç–∏** = driving process
- –Ø **–π—à–æ–≤** —É –±—É–¥–∏–Ω–æ–∫. (I was walking... but did I arrive? Unknown!)

**Prefixed verbs:**
- **–≤–≤—ñ–π—Ç–∏** = RESULT: now inside!

- –Ø **–≤–≤—ñ–π—à–æ–≤** —É –±—É–¥–∏–Ω–æ–∫. (I entered = I'm now inside! Result achieved!)

:::warning[Prefix = Result Machine]

**–≤–∏–π—à–æ–≤** (exited) ‚Üí Listener understands: you are ALREADY outside, action complete!
