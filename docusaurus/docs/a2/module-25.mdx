# Summary

# Warm-up

> üí° **Emotion + —â–æ:** Most emotion expressions are followed by "—â–æ" + a clause: –Ø —Ä–∞–¥–∏–π, **—â–æ** —Ç–∏ –ø—Ä–∏–π—à–æ–≤. (I'm happy that you came.) Some are impersonal: **–®–∫–æ–¥–∞**, —â–æ –≤—ñ–Ω –Ω–µ –ø—Ä–∏–π—à–æ–≤. (It's a pity that he didn't come.)

:::important
