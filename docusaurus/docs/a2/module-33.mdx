# Summary

This module teaches **noun-forming suffixes** ‚Äî how to create nouns from verbs (—á–∏—Ç–∞–Ω–Ω—è, life), adjectives (–≤–∞–∂–ª–∏–≤—ñ—Å—Ç—å), and other nouns (—á–∏—Ç–∞—á, –∫–æ—Ç–∏–∫). These patterns multiply your vocabulary!

# Warm-up

> üí° **Suffix Magic:** From "—á–∏—Ç–∞—Ç–∏" (to read): —á–∏—Ç–∞–Ω–Ω—è (reading - action), —á–∏—Ç–∞—á (reader - person). From "–≤–∞–∂–ª–∏–≤–∏–π" (important): –≤–∞–∂–ª–∏–≤—ñ—Å—Ç—å (importance). From "–∫—ñ—Ç" (cat): –∫–æ—Ç–∏–∫ (kitty). Suffixes transform words!
