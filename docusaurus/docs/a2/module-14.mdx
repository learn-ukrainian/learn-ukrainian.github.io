# Summary

This module dives deep into **how aspect pairs are formed**. You'll learn the most common **prefixes** that create perfective verbs, **suffix patterns**, and recognize the **suppletive pairs** that have completely different roots.

# Warm-up

> üí° **Three Formation Types:** Aspect pairs form through: 1) Adding a prefix (most common), 2) Changing/removing a suffix, 3) Different roots entirely (suppletive pairs).

## Formation Overview

| Method | Example | Note |
|--------|---------|------|
| **Prefix** | —á–∏—Ç–∞—Ç–∏ ‚Üí **–ø—Ä–æ**—á–∏—Ç–∞—Ç–∏ | Most common |
| **Suffix** | –¥–∞–≤–∞—Ç–∏ ‚Üí –¥–∞—Ç–∏ (remove -–≤–∞-) | Remove suffix |
| **Suppletive** | –≥–æ–≤–æ—Ä–∏—Ç–∏ ‚Üí —Å–∫–∞–∑–∞—Ç–∏ | Different roots |

## Cultural Insight: Ukrainian Verbal System üá∫üá¶

:::important
**Linguistic Heritage:** The aspect system is one of the oldest features of Slavic languages, dating back over 2000 years to Proto-Slavic. When you learn aspect pairs, you're learning a grammatical system that connects Ukrainian to its ancient roots. This system gives Ukrainian verbs enormous expressive power ‚Äî one verb can have dozens of derived forms.
:::


:::tip
**Learning Strategy:** Don't try to memorize every pair. Learn the PATTERNS! If you know –ø—Ä–æ- means "through/completion," you can predict that —á–∏—Ç–∞—Ç–∏ ‚Üí –ø—Ä–æ—á–∏—Ç–∞—Ç–∏, —Ö–æ–¥–∏—Ç–∏ ‚Üí –ø—Ä–æ–π—Ç–∏, —ó—Ö–∞—Ç–∏ ‚Üí –ø—Ä–æ—ó—Ö–∞—Ç–∏ all work the same way. Pattern recognition is the key.
:::
