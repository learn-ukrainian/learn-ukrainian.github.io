# Warm-up

Welcome to the **Word Factory**!
In the last module, we put prefixes on the *front* of words to change direction.
Now, we put suffixes on the *end* of words to change their **class**.

Imagine a machine:
1.  Input: **–ß–∏—Ç–∞-—Ç–∏** [Verb: To read] -> [Machine] -> Output: **–ß–∏—Ç–∞-—á** [Noun: Reader].
2.  Input: **–ß–∏—Ç–∞-—Ç–∏** -> [Machine] -> Output: **–ß–∏—Ç–∞-–Ω–Ω—è** [Noun: Reading].
3.  Input: **–î–æ–±—Ä-–∏–π** [Adj: Good/Kind] -> [Machine] -> Output: **–î–æ–±—Ä-–æ—Ç–∞** [Noun: Kindness].

This is how you multiply your vocabulary instantly.

:::note[Think about it]
English does this too:
- Teach -> Teach**er** (-er = Person)
- Teach -> Teach**ing** (-ing = Process)
- Kind -> Kind**ness** (-ness = Abstract Quality)
Ukrainian has the exact same logic!
:::

# The Suffix System

Let's look at the three main machines in our factory.

## Machine 1: The Person Maker (-–∞—á / -–µ—Ü—å / -–Ω–∏–∫)

Turns an **Action** into a **Doer**.

| Input (Verb) | Suffix | Output (Person) | Note |
| :--- | :--- | :--- | :--- |
| **–ß–∏—Ç–∞**—Ç–∏: Read | -–∞—á | **–ß–∏—Ç–∞—á**: Reader | Also: –ì–ª—è–¥–∞—á: Viewer |
| **–ü—Ä–æ–¥–∞–≤–∞**—Ç–∏: Sell | -–µ—Ü—å | **–ü—Ä–æ–¥–∞–≤–µ—Ü—å**: Seller | Also: –°–ø—ñ–≤–µ—Ü—å: Singer |
| **–ü–∏—Å–∞**—Ç–∏: Write | -–Ω–∏–∫ | **–ü–∏—Å—å–º–µ–Ω–Ω–∏–∫**: Writer | Double '–Ω'! |
| **–í—á–∏**—Ç–∏: Teach | -—Ç–µ–ª—å | **–í—á–∏—Ç–µ–ª—å**: Teacher | |

## Machine 2: The Process Maker (-–Ω–Ω—è / -—Ç—Ç—è)

Turns an **Action** into a **Thing/Process** (Gerund).
*Usually requires double consonants.*

| Input (Verb) | Suffix | Output (Process) | English |
| :--- | :--- | :--- | :--- |
| **–ß–∏—Ç–∞**—Ç–∏ | -–Ω–Ω—è | **–ß–∏—Ç–∞–Ω–Ω—è** | Reading |
| **–ü–∏—Å–∞**—Ç–∏ | -–Ω–Ω—è | **–ü–∏—Å–∞–Ω–Ω—è** | Writing (Scripture) |
| **–ù–∞–≤—á–∞**—Ç–∏ | -–Ω–Ω—è | **–ù–∞–≤—á–∞–Ω–Ω—è** | Learning/Studies |
| **–ñ–∏**—Ç–∏ | -—Ç—Ç—è | **–ñ–∏—Ç—Ç—è** | Life |
| **–ü–∏—Ç–∞**—Ç–∏ | -–Ω–Ω—è | **–ü–∏—Ç–∞–Ω–Ω—è** | Question |
| **–ó–∞–≤–¥–∞–≤–∞**—Ç–∏ | -–Ω–Ω—è | **–ó–∞–≤–¥–∞–Ω–Ω—è** | Task/Assignment |

:::tip[Spelling Rule: Double N]
Notice that **–ß–∏—Ç–∞–Ω–Ω—è**, **–ü–∏—Ç–∞–Ω–Ω—è**, **–ñ–∏—Ç—Ç—è** all have double consonants (**–Ω–Ω**, **—Ç—Ç**).
This is typical for verbal nouns ending in -—è.
Just remember: Double N = Process!
:::

# Cultural Insight: Last Names

Many Ukrainian surnames tell you what an ancestor did for a living!
They use the exact suffixes we just learned.

- **-–Ω–∏–∫**:
    - **–ú–µ–ª—å–Ω–∏–∫** (Miller) -> Melnyk
    - **–û–ª—ñ–π–Ω–∏–∫** (Oil manufacturer) -> Oliynyk
- **-–µ—Ü—å**:
    - **–ö—Ä–∞–≤–µ—Ü—å** (Tailor) -> Kravets
    - **–®–≤–µ—Ü—å** (Shoemaker) -> Shvets
- **-–∞—á**:
    - **–¢–∫–∞—á** (Weaver) -> Tkach
- **-–∞—Ä**:
    - **–ö–æ–≤–∞–ª—å** (Blacksmith) -> Koval
    - **–õ—ñ–∫–∞—Ä** (Doctor) -> Likar

:::note[Cultural Note: Identity]
If you meet a Ukrainian with the last name **–ö—Ä–∞–≤—á—É–∫**, you know their ancestor was a Tailor (–ö—Ä–∞–≤–µ—Ü—å).
If **–ö–æ–≤–∞–ª–µ–Ω–∫–æ**, they were a Blacksmith (–ö–æ–≤–∞–ª—å).
Surnames preserve history.
:::

# Deep Dive: The Logic of -ISTYH

The suffix **-—ñ—Å—Ç—å** is the equivalent of English **-ness** or **-ity**.
It creates abstract nouns from adjectives.
It is almost always **Feminine** and ends in a soft sign (**—å**).

- **Reality:** –†–µ–∞–ª—å–Ω-–∏–π -> –†–µ–∞–ª—å–Ω-—ñ—Å—Ç—å.
- **Popularity:** –ü–æ–ø—É–ª—è—Ä–Ω-–∏–π -> –ü–æ–ø—É–ª—è—Ä–Ω-—ñ—Å—Ç—å.
- **Activity:** –ê–∫—Ç–∏–≤–Ω-–∏–π -> –ê–∫—Ç–∏–≤–Ω-—ñ—Å—Ç—å.
- **Quality:** –Ø–∫—ñ—Å–Ω-–∏–π -> –Ø–∫-—ñ—Å—Ç—å.

:::tip[Soft Sign Magic]
Don't forget the soft sign at the end!
It's not *—Ä–∞–¥—ñ—Å—Ç*, it's **—Ä–∞–¥—ñ—Å—Ç—å**.
It sounds soft and gentle.
:::

# Deep Dive: Feminitives (The Rise of -KA)

Ukrainian is a gendered language. In the past, many professions were used only in the masculine form (–õ—ñ–∫–∞—Ä, –î–∏—Ä–µ–∫—Ç–æ—Ä), even for women.
But in modern Ukrainian, we prefer **Feminitives**!
The magic machine for this is the suffix **-–∫–∞**.

| Profession (Masc) | Feminitive (Fem) |
| :--- | :--- |
| **–í—á–∏—Ç–µ–ª—å** (Teacher) | **–í—á–∏—Ç–µ–ª—å–∫–∞** |
| **–õ—ñ–∫–∞—Ä** (Doctor) | **–õ—ñ–∫–∞—Ä–∫–∞** |
| **–ê–≤—Ç–æ—Ä** (Author) | **–ê–≤—Ç–æ—Ä–∫–∞** |
| **–î–∏—Ä–µ–∫—Ç–æ—Ä** (Director) | **–î–∏—Ä–µ–∫—Ç–æ—Ä–∫–∞** |
| **–ñ—É—Ä–Ω–∞–ª—ñ—Å—Ç** (Journalist) | **–ñ—É—Ä–Ω–∞–ª—ñ—Å—Ç–∫–∞** |
| **–ü–∏—Å—å–º–µ–Ω–Ω–∏–∫** (Writer) | **–ü–∏—Å—å–º–µ–Ω–Ω–∏—Ü—è** (Note: -—Ü—è) |

:::note[Why it matters]
Using feminitives is respectful and precise.
Saying "–í–æ–Ω–∞ ‚Äî —Ö–æ—Ä–æ—à–∏–π –ª—ñ–∫–∞—Ä" is grammatically okay, but "–í–æ–Ω–∞ ‚Äî —Ö–æ—Ä–æ—à–∞ –ª—ñ–∫–∞—Ä–∫–∞" highlights her identity.
It validates women's visibility in professional roles.
:::

# Deep Dive: The Crafter Suffix (-–∞—Ä)

Another common suffix for professions is **-–∞—Ä** (or -—è—Ä).
It usually indicates a skilled craftsman or worker.

- **–õ—ñ–∫-–∏** (Medicine) -> **–õ—ñ–∫-–∞—Ä** (Doctor).
- **–ö—É—Ö-–Ω—è** (Kitchen) -> **–ö—É—Ö-–∞—Ä** (Cook).
- **–ü–µ—Ä—É–∫-–∞** (Wig) -> **–ü–µ—Ä—É–∫-–∞—Ä** (Hairdresser).
- **–ö–æ–±–∑-–∞** (Instrument) -> **–ö–æ–±–∑-–∞—Ä** (Bard).

:::note[Stress Shift]
Often the stress falls on the suffix: –ü–µ—Ä—É–∫**–∞—Ä**, –ö–æ–±–∑**–∞—Ä**.
But: –õ**—ñ**–∫–∞—Ä, –ö**—É**—Ö–∞—Ä.
:::

:::note[Cultural Insight: Diminutives]
Ukrainians LOVE diminutives. We use them not just for small things, but to show **affection** and **politeness**.

It makes the language sound softer and warmer.
:::

# Deep Dive: Diminutives Logic

To make things small or cute, we use suffixes:
- Masc: **-–æ–∫**, **-–∏–∫** (–ö—ñ—Ç -> –ö–æ—Ç**–∏–∫**, –î—ñ–º -> –ë—É–¥–∏–Ω**–æ–∫** -> –ë—É–¥–∏–Ω**–æ—á–æ–∫**)
- Fem: **-–∫–∞**, **-—á–∫–∞** (–ö–Ω–∏–≥–∞ -> –ö–Ω–∏–∂**–∫–∞** -> –ö–Ω–∏–∂**–µ—á–∫–∞**, –†—É–∫–∞ -> –†—É—á**–∫–∞**)
- Neut: **-–∫–æ**, **-—á–∫–æ** (–°–æ–Ω—Ü–µ -> –°–æ–Ω–µ**—á–∫–æ**, –í—ñ–∫–Ω–æ -> –í—ñ–∫–æ–Ω**—Ü–µ**)

:::tip[Magic of K/CH]
Notice how **–ö** often changes to **–ß** or **–¶**?
- –†—É**–∫**–∞ -> –†—É**—á**–∫–∞.
- –°–æ–Ω**—Ü**–µ -> –°–æ–Ω–µ**—á**–∫–æ.
This is sound mutation (palatalization). It makes words easier to pronounce softly.
:::
