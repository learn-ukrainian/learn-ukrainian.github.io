## Introduction

Module 4 introduced the Instrumental case with the preposition **–∑** for accompaniment: –∑ –¥—Ä—É–≥–æ–º, –∑ —Å–µ—Å—Ç—Ä–æ—é. Now it's time to learn the Instrumental case **without a preposition** ‚Äî the true ¬´instrumental¬ª use of this case!

**What you'll learn:**

- **Bare Instrumental for transport:** –∞–≤—Ç–æ–±—É—Å–æ–º, –ø–æ—ó–∑–¥–æ–º, –ª—ñ—Ç–∞–∫–æ–º
- **Bare Instrumental for tools:** —Ä—É—á–∫–æ—é, –Ω–æ–∂–µ–º, –ª–æ–∂–∫–æ—é
- **Bare Instrumental for body parts:** —Ä—É–∫–æ—é, –≥–æ–ª–æ–≤–æ—é
- **Bare Instrumental for manner:** –≥–æ–ª–æ—Å–æ–º, –ø–æ—à–µ–ø–∫–∏

**The key distinction:**

- **–ó + Instrumental** = accompaniment
- **Bare Instrumental** = means, tools, manner (by what means, with what tool, in what way)

**Why this matters:**
English uses ¬´by¬ª for transport and ¬´with¬ª for tools, but Ukrainian uses the same grammatical case for both ‚Äî the Instrumental without a preposition. This is the Instrumental in its most literal sense: the case of instruments and means.
