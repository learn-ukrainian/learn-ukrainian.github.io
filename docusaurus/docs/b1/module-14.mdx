### –ß–æ–º—É —Ü–µ –≤–∞–∂–ª–∏–≤–æ

English uses different constructions for these meanings:

Ukrainian prefixes encode these meanings directly into the motion verb.

> üí° **Did You Know**
>

> üîó **Language Link**
>
> English uses different verbs: "leave," "depart," "set off." Ukrainian uses ONE verb stem with the –ø–æ- prefix. This makes the system more economical but requires learning when to use each prefix.
