### The Style Editor's Challenge

One common real-world task is editing texts for stylistic consistency. A draft might mix registers inappropriately. This is especially important in professional contexts where register violations can undermine credibility.

Native speakers make these choices automatically, but learners must consciously select appropriate forms.

- Adding diminutives to formal texts (makes you sound unprofessional)
- Using complex participles in casual speech (sounds stilted)
- Mixing statistics language with emotional vocabulary (creates cognitive dissonance)
