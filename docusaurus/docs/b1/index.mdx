# ✅ B1 - Середній

**Завершено**

## Від граматики до справжньої розмови!

Цей рівень — ваш перехід від «я вивчаю українську» до «я говорю українською». B1 поглиблює розуміння видів дієслова, вводить систему дієслів руху та розвиває комунікативні навички.

**Чого ви навчитеся:**
- Повне опанування видів дієслова — вибір між доконаним і недоконаним
- Система дієслів руху — «йти/ходити», «їхати/їздити» та префікси
- Складнопідрядні речення — причина, час, умова
- Комунікативні навички — написання листів, подкасти, обговорення

**Занурення:** 75-90% українською — ви вже готові!

## Тематичний план

### Phase B1.0: Метамова
*Опанування граматичної термінології українською мовою.*

- **Основи:** Частини мови, назви відмінків, структура речення.
- **Дієслова:** Термінологія для видів, часів та заперечення.
- **Контроль:** [Контрольна точка — Готові до занурення](./module-05).

### Phase B1.1-2: Вид та Рух
*Мастерність у вживанні дієслів.*

- **Вид дієслова:** Процес vs Результат, вид у минулому, майбутньому та запереченні. 40 найважливіших пар.
- **Дієслова руху:** Повна система (14 пар), усі префікси (при-, ви-, пере-, за- та ін.), переносні значення.
- **Контроль:** [Контрольна точка — Опанування виду](./module-15), [Дієслова руху](./module-25).

### Phase B1.3: Складні речення
*Побудова логічних зв'язків.*

- **Типи підрядних:** Означальні (який), місця (де, куди), часу та умови (якщо, якби).
- **Складні ідеї:** Причина та наслідок, допуск (хоча), мета (щоб), непряма мова.
- **Контроль:** [Контрольна точка — Складні речення](./module-34), [Синтез складних речень](./module-41).

### Phase B1.4: Просунута граматика
*Дієприкметники, пасив та демінутиви.*

- **Форми:** Дієприслівники (-ючи, -вши), пасивні конструкції (-но, -то), дієприкметники.
- **Стиль:** Демінутиви (емоційне забарвлення), збірні числівники та дроби.
- **Контроль:** [Контрольна точка — Просунута граматика](./module-51).

### Phase B1.5-6: Розширення лексики
*Абстрактні поняття та професійна комунікація.*

- **Ідеї:** Абстрактні концепції, процеси та зміни.
- **Думки:** Вираження думок, згода/незгода, дискурсивні маркери.
- **Сфери:** Медіа, суспільство, екологія, здоров'я, бізнес, стосунки.
- **Тонкощі:** Синонімія дієслів мислення та мовлення, сталі вирази.

### Phase B1.7-9: Сучасна Україна та Навички
- **Культура:** Регіони України (Захід, Схід, Південь, Центр), музика, кіно, технології, кухня.
- **Спорт:** Активний відпочинок, біг, гори, зимові види.
- **Скілзи:** Робота з новинами, інтерв'ю та подкасти.
- **Комунікація:** Написання імейлів та офіційних листів, дискусії та дебати.
- **Фінал:** [Підсумковий іспит B1](./module-92).
