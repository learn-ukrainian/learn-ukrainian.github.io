# A1 - Beginner Level

**In Development**

A1 level modules introduce you to Ukrainian from scratch. This level covers:

- The Cyrillic alphabet and pronunciation
- Basic greetings and introductions
- Numbers, colors, and common objects
- Simple sentence structures
- Gender and basic case introduction
- ~750 vocabulary words

**Prerequisites:** None - complete beginners welcome!

**Module count:** 34 modules planned
