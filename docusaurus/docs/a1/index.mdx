# A1 - Beginner

**Level:** A1 (Beginner)
**Modules:** 34 modules
**Prerequisite:** None
**Focus:** Cyrillic alphabet, basic grammar, everyday vocabulary

---

## Level Overview

A1 is designed for **complete beginners** with no prior Ukrainian knowledge. By the end of this level, you will read Cyrillic fluently, form basic sentences, and handle everyday interactions like ordering food, asking directions, and introducing yourself.

### Learning Outcomes

- Read and write Cyrillic alphabet (print and italic)
- Understand noun gender and basic declension
- Use present, past, and future tense
- Navigate 5 cases: Nominative, Accusative, Genitive, Locative, Vocative
- Handle basic conversations about daily life
