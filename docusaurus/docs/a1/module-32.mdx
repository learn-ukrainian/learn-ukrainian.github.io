# Grammar

## Talking About Family

:::tip[–°–ø–æ—Å—Ç–µ—Ä–µ–∂–µ–Ω–Ω—è (Pattern Discovery)]

Look at these patterns:
- **–£ –º–µ–Ω–µ —î** –±—Ä–∞—Ç. (I have a brother.)
- **–ú–æ–≥–æ** –±—Ä–∞—Ç–∞ –∑–≤–∞—Ç–∏ –ú–∞–∫—Å–∏–º. (My brother's name is Maksym.)
- **–¶–µ –º–æ—è** —Å–µ—Å—Ç—Ä–∞. (This is my sister.)

We use "–£ –º–µ–Ω–µ —î" for "I have" and possessives (–º—ñ–π/–º–æ—è/–º–æ—î) for "my."
:::


### "I Have" Pattern

| Structure | Example |
|-----------|---------|
| –£ –º–µ–Ω–µ —î + Nominative | –£ –º–µ–Ω–µ —î **–±—Ä–∞—Ç**. (I have a brother.) |

### Vocative Case (–ö–ª–∏—á–Ω–∏–π –≤—ñ–¥–º—ñ–Ω–æ–∫)

When calling someone directly, Ukrainian changes the word ending. This is the **Vocative case** ‚Äî used for direct address:

:::tip[–°–ø–æ—Å—Ç–µ—Ä–µ–∂–µ–Ω–Ω—è (Pattern Discovery)]

Look at these pairs:
- –º–∞–º–∞ ‚Üí **–ú–∞–º–æ!** (Mom!)
- —Ç–∞—Ç–æ ‚Üí **–¢–∞—Ç—É!** (Dad!)
- –±–∞–±—É—Å—è ‚Üí **–ë–∞–±—É—Å—é!** (Grandma!)
- –¥—ñ–¥—É—Å—å ‚Üí **–î—ñ–¥—É—Å—é!** (Grandpa!)

The ending changes when you're calling TO someone!
:::


| Nominative | Vocative | Use |
|------------|----------|-----|
| –º–∞–º–∞ | –ú–∞–º–æ! | calling mom |
| —Ç–∞—Ç–æ | –¢–∞—Ç—É! | calling dad |
| –±–∞–±—É—Å—è | –ë–∞–±—É—Å—é! | calling grandma |
| –¥—ñ–¥—É—Å—å | –î—ñ–¥—É—Å—é! | calling grandpa |
| –±—Ä–∞—Ç | –ë—Ä–∞—Ç–µ! | calling brother |
| —Å–µ—Å—Ç—Ä–∞ | –°–µ—Å—Ç—Ä–æ! | calling sister |
| –û–∫—Å–∞–Ω–∞ | –û–∫—Å–∞–Ω–æ! | calling Oksana |
| –Ü–≤–∞–Ω | –Ü–≤–∞–Ω–µ! | calling Ivan |

**Common patterns:**
- -–∞ ‚Üí -–æ (–º–∞–º–∞ ‚Üí –º–∞–º–æ, —Å–µ—Å—Ç—Ä–∞ ‚Üí —Å–µ—Å—Ç—Ä–æ)
- -–æ ‚Üí -—É (—Ç–∞—Ç–æ ‚Üí —Ç–∞—Ç—É)
- -—è ‚Üí -—é (–±–∞–±—É—Å—è ‚Üí –±–∞–±—É—Å—é)
- consonant ‚Üí -–µ (–±—Ä–∞—Ç ‚Üí –±—Ä–∞—Ç–µ, –Ü–≤–∞–Ω ‚Üí –Ü–≤–∞–Ω–µ)

> üí° **When to Use Vocative**
>
> Use vocative when:
> - Calling someone: **–ú–∞–º–æ**, –¥–µ —Ç–∏? (Mom, where are you?)
> - Getting attention: **–¢–∞—Ç—É**, –¥–æ–ø–æ–º–æ–∂–∏! (Dad, help!)
> - In exclamations: **–ë–æ–∂–µ!** (God!/Oh my God!)
>
> Don't use with titles: –ü–∞–Ω–µ –Ü–≤–∞–Ω–µ! (Mr. Ivan!) ‚Äî not "–ü–∞–Ω –Ü–≤–∞–Ω–µ!"
