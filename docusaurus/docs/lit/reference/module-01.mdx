## 2. –°–º—ñ—Ö —è–∫ –¢–µ—Ä–∞–ø—ñ—è –¢—Ä–∞–≤–º–∏

–ö–æ–ª–∏ –Ω–∞—Ü—ñ—è –ø—Ä–æ–≥—Ä–∞—î –≤—ñ–π–Ω—É —ñ –≤—Ç—Ä–∞—á–∞—î —Å–≤–æ–±–æ–¥—É, –≤–æ–Ω–∞ –∑–∞–∑–≤–∏—á–∞–π –≤–ø–∞–¥–∞—î –≤ –¥–µ–ø—Ä–µ—Å—ñ—é —ñ –º–æ–≤—á–∞–Ω–Ω—è. –ö–æ—Ç–ª—è—Ä–µ–≤—Å—å–∫–∏–π –∑–∞–ø—Ä–æ–ø–æ–Ω—É–≤–∞–≤ –∞–Ω—Ç–∏–¥–æ—Ç: **–°–º—ñ—Ö**.

## 3. "–¢—Ä–∞–≤–µ—Å—Ç—ñ—è" —á–∏ "–ï–Ω—Ü–∏–∫–ª–æ–ø–µ–¥—ñ—è"?

–§–æ—Ä–º–∞–ª—å–Ω–æ "–ï–Ω–µ—ó–¥–∞" ‚Äî —Ü–µ **—Ç—Ä–∞–≤–µ—Å—Ç—ñ—è** (–ø–µ—Ä–µ–≤–¥—è–≥–∞–Ω–Ω—è). –ê–ª–µ —Ñ–∞–∫—Ç–∏—á–Ω–æ —Ü–µ **–µ–Ω—Ü–∏–∫–ª–æ–ø–µ–¥—ñ—è —É–∫—Ä–∞—ó–Ω–æ–∑–Ω–∞–≤—Å—Ç–≤–∞** XVIII —Å—Ç–æ–ª—ñ—Ç—Ç—è.

# –ß–∞—Å—Ç–∏–Ω–∞ IV: –ë—ñ–±–ª—ñ–æ—Ç–µ–∫–∞ –ê—Ä—Ö—ñ—Ç–µ–∫—Ç–æ—Ä–∞

# –ß–∞—Å—Ç–∏–Ω–∞ V: –°–ø–∞–¥—â–∏–Ω–∞
