# ✅ LIT - Українська література

**Завершено**

## Від читача до науковця — українська література як духовна скарбниця нації!

Цей трек — для тих, хто хоче не просто читати українську літературу, а глибоко аналізувати її. «Кобзар» Шевченка, драми Лесі Українки, проза Франка — ви вивчатимете ці тексти на рівні літературознавця.

**Чого ви навчитеся:**
- **Бурлеск та народні витоки** — Котляревський, народження сучасної мови.
- **Сентименталізм** — Квітка-Основ'яненко, ідеал української душі.
- **Пророк** — Тарас Шевченко, формування національного канону.
- **Інтелектуали** — Куліш та Костомаров, європейський вектор.
- **Реалізм** — Нечуй-Левицький, життя села «як воно є».

---

## Тематичний план

### Phase LIT.1: Бурлеск та народні витоки
*Народження сучасної літературної мови. Гумор, побут, козацтво.*

- **Іван Котляревський:** Біографія, феномен «Енеїди», словник війни та бенкету.
- **Наталка Полтавка:** Анатомія почуттів та народна пісня.

### Phase LIT.2: Сентименталізм
*Розвиток прози. Емоційність, етнографія, мораль.*

- **Квітка-Основ'яненко:** Батько української прози, «Маруся», «Конотопська відьма».
- **Мова та побут:** Весільні та поховальні обряди в текстах.

### Phase LIT.3: Пророк
*Романтизм та кристалізація мовного стандарту.*

- **Тарас Шевченко:** Шлях від кріпака до академіка, балади, історичні поеми.
- **Аналіз текстів:** «Сон», «Заповіт», «Марія», жіноча доля в «Кобзарі».
- **Спадщина:** Шевченко як маляр, щоденник, культ і міфи.

### Phase LIT.4: Інтелектуали
*Європейськість, історичний роман, мовна реформа.*

- **Пантелеймон Куліш:** «Чорна рада» — перший історичний роман.
- **Мовне питання:** «Кулішівка», Кирило-Мефодіївське братство.

### Phase LIT.5: Реалізм
*Село без прикрас. Жива мова, сімейні конфлікти.*

- **Іван Нечуй-Левицький:** «Кайдашева сім'я» (аналіз конфлікту та мови), «Микола Джеря».
