# ✅ B2 - Вищий середній

**Завершено**

## Від вивчення до володіння!

На цьому рівні ви переходите від «студента» до «користувача» української мови. B2 охоплює пасивний стан, стилістику, мовні регістри та складні комунікативні ситуації.

**Чого ви навчитеся:**
- Пасивний стан — всі способи вираження пасиву в українській
- Дієприкметники та дієприслівники — активні та пасивні форми
- Регістри мовлення — офіційний, діловий, науковий, медичний
- Стилістика — наголос, інверсія, сполучники

**Занурення:** 100% українською — англійська більше не потрібна!
