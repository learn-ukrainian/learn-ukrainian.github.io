# C2 — Рівень вільного володіння

**В розробці — 100 модулів** | **~2 500 нових слів**

На рівні C2 ви не просто володієте українською — ви творите нею. Стилістична досконалість, літературна творчість, професійна спеціалізація.

---

## Огляд програми

### Фаза C2.1: Стилістична досконалість (Модулі 01-25)

Повна система стилів:

- **7 функціональних стилів** — розмовний, офіційний, науковий, публіцистичний, художній, релігійний, епістолярний
- **Стилістичні засоби** — антоніми, синоніми, тропи
- **Милозвучність** — у/в, і/й, з/зі/із
- **Трансформація тексту** — науковий → популярний, офіційний → публіцистичний

### Фаза C2.2: Літературна майстерність (Модулі 26-40)

Творче письмо:

- **Художня проза** — оповідання, новела
- **Поезія** — віршування, ритміка
- **Драматургія** — діалог, ремарки
- **Літературна критика** — рецензії, аналіз

### Фаза C2.3: Професійна спеціалізація (Модулі 41-75)

Універсальні професійні навички:

- **Юридична українська** — договори, позови, закони
- **Медична українська** — діагнози, консультації
- **Технічна українська** — інструкції, специфікації
- **Бізнес-українська** — звіти, презентації, переговори
- **Журналістика** — репортажі, інтерв'ю, розслідування

### Фаза C2.4: Метанавички та підсумок (Модулі 76-100)

Вершина володіння:

- **Викладання української** — методика, пояснення граматики
- **Переклад** — художній, технічний, синхронний
- **Редагування** — коректура, стилістичне редагування
- **Публічний виступ** — ораторське мистецтво
- **Фінальна сертифікація** — комплексний екзамен

## Що ви опануєте

| Навичка | Досягнення C2 |
|---------|---------------|
| **Читання** | Будь-які тексти без зусиль |
| **Говоріння** | Як носій мови, з відтінками значень |
| **Аудіювання** | Швидка мова, діалекти, жарти |
| **Письмо** | Творчі та професійні тексти |
| **Стилістика** | Усі 7 стилів, усі засоби |

## Передумови

**Завершені рівні A1-C1** — академічне письмо, біографії, стилістика, народна культура, література.
