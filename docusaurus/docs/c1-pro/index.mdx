**Заплановано**

## Українська на рівні керівника — від виконавчого листування до наукових публікацій!

Цей трек для тих, хто потребує найвищого рівня професійної української. Керівники, науковці, перекладачі — цей курс для вас.

**Чого ви навчитеся:**
- Виконавча комунікація — стратегічні звіти, кризові комунікації
- Академічне письмо — наукові статті, гранти, дисертації
- Галузева спеціалізація — IT, фінанси, право, медицина
- Переклад — основи синхронного та послідовного перекладу

**Підхід:** ESP + CLIL — мова через професійні завдання
