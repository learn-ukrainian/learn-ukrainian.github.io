# 📋 B2-PRO - Професійна українська

**Заплановано**

## Українська для кар'єри — від листування до переговорів!

Цей трек розроблений для професіоналів, які хочуть використовувати українську на робочому місці. Ділове листування, технічна документація, презентації та переговори.

**Чого ви навчитеся:**
- Ділова комунікація — листи, звіти, протоколи
- Технічні домени — IT, фінанси, право, медицина
- Медіа та дискурс — аналіз новин, публічні виступи
- Нетворкінг — професійні контакти українською

**Підхід:** ESP (English for Specific Purposes) адаптований для української
