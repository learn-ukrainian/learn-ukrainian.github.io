Numbers are useful immediately: prices, ages, notebooks, and phone
numbers. Ukrainian numbers also change the noun after them, so this module
keeps the rule practical. You will learn the phrases you need first, and leave
the full grammar for later.

By the end, you can:

- count from **нуль** to **сто**;
- read useful price phrases up to **тисяча гривень**;
- ask **Скільки коштує...?** for prices;
- answer **Скільки тобі років?** with **Мені ... рік / роки / років**;
- read a Ukrainian phone number in small groups;
- repair common number traps such as **шість**, **п'ять**, **сім**,
  **сімнадцять**, **п'ятдесят**, and **дев'яносто**.

Start with one question:

**Скільки?** = How many? / How much?

In a shop, the most useful line is:

**Скільки коштує...?** = How much does ... cost?

At a market stall:

```text
Покупець: Добрий день! Скільки коштує синя сумка?
Продавець: Двісті гривень.
Покупець: А маленька зелена сумка?
Продавець: Сто п'ятдесят гривень.
Покупець: Добре, дякую.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Покупець: Добрий день! Скільки коштує синя сумка?** | Customer: Good afternoon! How much is the blue bag? |
| **Продавець: Двісті гривень.** | Seller: Two hundred hryvnias. |
| **Покупець: А маленька зелена сумка?** | Customer: And the small green bag? |
| **Продавець: Сто п'ятдесят гривень.** | Seller: One hundred fifty hryvnias. |
| **Покупець: Добре, дякую.** | Customer: Good, thank you. |

Keep **гривня / гривні / гривень** as a price phrase:

- **одна гривня**
- **дві гривні**, **три гривні**, **чотири гривні**
- **п'ять гривень**, **десять гривень**, **сто гривень**

Do not turn this into a case lesson. For now, remember the 1 / 2-4 / 5+ price
pattern.

For more than one item, you may hear **Скільки коштують три булочки?** Keep
**коштує** and **коштують** as price-question phrases for now.

:::tip
When you hear a price, first catch the number, then catch the money word.
**Двісті** is the amount. **Гривень** tells you the currency. You do not need
to translate every ending while someone is speaking.
:::

## Числа 1-20

Learn **нуль** first. In phone numbers, the first digit is **нуль**, not the
English "oh."

| Number | Ukrainian |
| --- | --- |
| 0 | **нуль** |
| 1 | **один / одна / одне** |
| 2 | **два / дві** |
| 3 | **три** |
| 4 | **чотири** |
| 5 | **п'ять** |
| 6 | **шість** |
| 7 | **сім** |
| 8 | **вісім** |
| 9 | **дев'ять** |
| 10 | **десять** |

Two numbers show gender early:

- **один олівець**, **одна ручка**, **одне тістечко**
- **два олівці**, **дві ручки**

For 5 and above, use a memorized phrase with the noun:
**п'ять зошитів**, **шість студентів**, **десять гривень**. The full grammar
comes later; the phrase is enough now.

The teens mostly end in **-надцять**:

| Number | Ukrainian |
| --- | --- |
| 11 | **одинадцять** |
| 12 | **дванадцять** |
| 13 | **тринадцять** |
| 14 | **чотирнадцять** |
| 15 | **п'ятнадцять** |
| 16 | **шістнадцять** |
| 17 | **сімнадцять** |
| 18 | **вісімнадцять** |
| 19 | **дев'ятнадцять** |
| 20 | **двадцять** |

Watch the written forms: **п'ять**, **дев'ять**, **п'ятнадцять** use an
apostrophe. **Шість** and **сім** are Ukrainian forms to keep very clean.

Use numbers with familiar objects from earlier modules:

- **один стіл**, **два столи**, **п'ять столів**
- **одна ручка**, **дві ручки**, **п'ять ручок**
- **одне тістечко**, **два тістечка**, **п'ять тістечок**

The last noun form changes at **5+**. At A1, do not analyze the case name yet.
Learn these as useful shopping and classroom phrases. The pattern is
visible enough: **1** has one form, **2-4** has another, and **5+** has another.
That is the same memory hook you need for **гривня**, **рік**, and everyday
objects.

## Десятки і сотні

The tens are:

| Number | Ukrainian |
| --- | --- |
| 20 | **двадцять** |
| 30 | **тридцять** |
| 40 | **сорок** |
| 50 | **п'ятдесят** |
| 60 | **шістдесят** |
| 70 | **сімдесят** |
| 80 | **вісімдесят** |
| 90 | **дев'яносто** |
| 100 | **сто** |

Two forms are pure memory items: **сорок** and **дев'яносто**. Do not build
them by force from **чотири** or **дев'ять**.

Combined numbers are simple:

- **двадцять один**
- **тридцять п'ять**
- **сорок сім**
- **дев'яносто дев'ять**

For prices, learn these useful phrases:

| Price | Ukrainian |
| --- | --- |
| 100 | **сто гривень** |
| 150 | **сто п'ятдесят гривень** |
| 200 | **двісті гривень** |
| 300 | **триста гривень** |
| 400 | **чотириста гривень** |
| 500 | **п'ятсот гривень** |
| 1000 | **тисяча гривень** |

For coins, recognize **копійка / копійки / копійок** the same way:
**одна копійка**, **дві копійки**, **п'ять копійок**.

You will often see rounded prices first: **сто гривень**,
**двісті гривень**, **триста гривень**. Then the middle number appears:
**сто п'ятдесят гривень**, **двісті п'ятдесят гривень**. Say the hundreds
first, then the tens. The order is friendly:

- **200** = **двісті**
- **250** = **двісті п'ятдесят**
- **297** = **двісті дев'яносто сім**

If you miss a number in a shop, ask with one short line:
**Повторіть, будь ласка.** You already know **будь ласка**, so this is a useful
repair phrase for real conversations.

### Скільки років?

Use age as one ready formula:

**Мені / тобі / їй + number + рік / роки / років**

| Age | Ukrainian |
| --- | --- |
| 1 | **Мені один рік.** |
| 2-4 | **Мені два роки.** / **Мені три роки.** / **Мені чотири роки.** |
| 5+ | **Мені п'ять років.** / **Мені двадцять років.** |
| 21 | **Мені двадцять один рік.** |
| 22-24 | **Мені двадцять два роки.** / **двадцять три роки** / **двадцять чотири роки** |
| 25+ | **Мені двадцять п'ять років.** |

Read the dialogue:

```text
Оля: Скільки тобі років?
Дмитро: Мені двадцять п'ять. А тобі?
Оля: Мені тридцять два.
Дмитро: А твоя сестра?
Оля: Їй вісімнадцять.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Оля: Скільки тобі років?** | Olia: How old are you? |
| **Дмитро: Мені двадцять п'ять. А тобі?** | Dmytro: I am twenty-five. And you? |
| **Оля: Мені тридцять два.** | Olia: I am thirty-two. |
| **Дмитро: А твоя сестра?** | Dmytro: And your sister? |
| **Оля: Їй вісімнадцять.** | Olia: She is eighteen. |

The word **років** is often left understood after the first question. The safe
full answer is still **Мені двадцять п'ять років**.

Use the same person phrases for other people:

- **Тобі двадцять два роки.**
- **Вам двадцять п'ять років.**
- **Їй вісімнадцять років.**
- **Йому тридцять один рік.**
- **Мамі п'ятдесят років.**

These are not possession sentences. English says "I am twenty-five." Ukrainian
uses a different frame: **Мені двадцять п'ять років**. Treat **Мені** as part
of the age formula, not as a new grammar topic to unpack today.

### Номери телефонів

Ukrainian mobile numbers often appear as **+380 XX XXX XX XX**. When speaking,
read them in small groups. Say **нуль**, not "oh."

```text
Ліза: Дай, будь ласка, свій номер телефону.
Марко: Запиши: нуль дев'яносто сім, три два один, сорок п'ять, шістдесят сім.
Ліза: Повторюю: нуль дев'яносто сім, три два один, сорок п'ять, шістдесят сім. Так?
Марко: Так, усе правильно.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Ліза: Дай, будь ласка, свій номер телефону.** | Liza: Please give me your phone number. |
| **Марко: Запиши: нуль дев'яносто сім, три два один, сорок п'ять, шістдесят сім.** | Marko: Write it down: zero ninety-seven, three two one, forty-five, sixty-seven. |
| **Ліза: Повторюю: нуль дев'яносто сім, три два один, сорок п'ять, шістдесят сім. Так?** | Liza: I repeat: zero ninety-seven, three two one, forty-five, sixty-seven. Right? |
| **Марко: Так, усе правильно.** | Marko: Yes, everything is correct. |

For A1, reading by groups is enough. You do not need every phone-number style.
The key habits are **нуль** at the start and clear phrases such as
**сорок п'ять**, **шістдесят сім**.

Try these aloud:

- **нуль шістдесят сім**
- **два три чотири**
- **п'ятдесят шість**
- **сімдесят вісім**

Both styles can appear: digit by digit (**два три чотири**) or as a two-digit
number (**п'ятдесят шість**). The practical goal is clarity. Speak slowly,
pause between groups, and repeat the number back before you trust that you
wrote it correctly.

## Числові пастки

Keep these repairs automatic:

| Avoid | Use |
| --- | --- |
| **п'ять столи** | **п'ять столів** |
| **Я маю 20 років** / **Я є 20 років** | **Мені 20 років** |
| **дві гривень** | **дві гривні** |
| **сімдесят гривні** | **сімдесят гривень** |
| nonstandard 50/60 spellings | **п'ятдесят** / **шістдесят** |
| **Скільки вам год?** | **Скільки вам років?** |

Also keep the common spelling and sound traps clean:

- **шість**, not a look-alike form from another language;
- **п'ять**, with apostrophe;
- **сім**, not a softened look-alike;
- **сімнадцять**, not a mixed form;
- **п'ятдесят**, no extra soft sign;
- **дев'яносто**, not a forced made-up pattern;
- **пів на шосту**, not a copied half-hour phrase.

Collective numerals such as **двоє**, **троє**, **четверо** are passive
recognition only for now. If you see them, recognize them. Do not try to
produce them yet.

One more trap is the money word itself. Use **гривня**, **гривні**,
**гривень** for Ukrainian money. Keep other currencies out of these A1 price
drills unless a later lesson gives you a specific travel context.

## 📋 Підсумок

Use numbers in three practical ways:

1. Prices:
   **Скільки коштує? — Двісті гривень. Сто п'ятдесят гривень.**
2. Age:
   **Скільки тобі років? — Мені двадцять три роки.**
3. Phone:
   **Мій номер — нуль дев'яносто сім, три два один, сорок п'ять,
   шістдесят сім.**

Now check yourself:

- Say your age with **Мені ... рік / роки / років**.
- Say a price: **250 гривень** = **двісті п'ятдесят гривень**.
- Read this number aloud:
  **нуль шістдесят сім, чотири п'ять шість, двадцять один, тридцять два**.

Keep the scope small. You can count, buy something, give your age, and read a
phone number. That is already a lot for one A1 module.

A strong final routine has four lines:

```text
Мені двадцять три роки.
Синя сумка коштує двісті гривень.
Мій номер телефону — нуль дев'яносто сім, три два один, сорок п'ять, шістдесят сім.
Повторіть, будь ласка.
```

Support after the Ukrainian lines:

| Українська | English support |
| --- | --- |
| **Мені двадцять три роки.** | I am twenty-three years old. |
| **Синя сумка коштує двісті гривень.** | The blue bag costs two hundred hryvnias. |
| **Мій номер телефону — нуль дев'яносто сім, три два один, сорок п'ять, шістдесят сім.** | My phone number is zero ninety-seven, three two one, forty-five, sixty-seven. |
| **Повторіть, будь ласка.** | Please repeat. |

Say those lines until the phrases feel boring. That is the point: numbers should
become quick enough that you can use them while buying bread, writing a phone
number, or answering a simple personal question.
