У попередньому модулі родовий відмінок допомагав говорити про джерело:
`з України`, `від мами`, `після уроку`. Тепер додаємо ще одну групу
частих прийменників:

- `для` - для кого? для чого?
- `без` - без кого? без чого?
- `біля`, `навпроти`, `коло` - біля чого? навпроти чого?

Це все родовий відмінок. Але значення різні: призначення, відсутність і
місце поруч з орієнтиром.

кого, без чого не можна їхати і де буде табір.

### Помітьте перед правилом

У діалозі всі форми короткі й практичні. Ніхто не пояснює таблицю
відмінків під час походу. Люди просто питають:

> Для кого ця ковдра?
>
> Без чого не можна їхати?
>
> Де буде намет?

Відповіді теж короткі:

> Для Олени.
>
> Без ліхтарика.
>
> Біля річки.

На А2 це правильний формат тренування. Учитель може дати словникову
форму `річка`, а ви повторюєте готову фразу `біля річки`. Потім
переносите модель на нове слово: `школа - біля школи`, `дорога -
навпроти дороги`, `хата - коло хати`.

Скажіть у парі:

- `Це для ...`
- `Я не можу без ...`
- `Ми зустрічаємося біля ...`

Якщо не знаєте форму, зупиніться і попросіть: `Як правильно: біля
школи чи біля школа?` Така коротка граматична пауза нормальна для
роботи на А2.

## Для кого? Для чого?

`Для` показує призначення, мету або людину, для якої щось зроблено.

> Це подарунок для мами.
>
> У мене є час для відпочинку.
>
> Це важливо для мене.
>
> Ця вправа для студентів А2.

Після `для` стоїть родовий відмінок:

| Називний | Для плюс родовий |
| --- | --- |
| мама | для мами |
| друг | для друга |
| сестра | для сестри |
| Марія | для Марії |
| навчання | для навчання |
| здоров'я | для здоров'я |

`Для` часто звучить у маленьких, дуже корисних фразах:

> Це для тебе.
>
> Це для мене.
>
> Час для кави.
>
> Місце для намету.

На А2 не треба одразу будувати довгі речення. Почніть з коротких
відповідей: `Для кого? - Для Олени. Для чого? - Для відпочинку.`

| Питання | Приклад | Що маємо на увазі |

| для чого? | час для відпочинку | мета або призначення |

Обидва варіанти використовують той самий прийменник і той самий
відмінок. Змінюється тільки ситуація.

## Без кого? Без чого?

`Без` показує відсутність або нестачу.

> Кава без цукру.
>
> Чай без молока.
>
> День без дощу.
>
> Ми поїхали без нього.

У побуті `без` дуже часте. Воно допомагає говорити про їжу, речі,
плани і проблеми.

| Називний | Без плюс родовий |
| --- | --- |
| цукор | без цукру |
| молоко | без молока |
| вода | без води |
| сіль | без солі |
| парасолька | без парасольки |
| допомога | без допомоги |

Є також сталі фрази:

> Без проблем.
>
> Без сумніву.
>
> Без паніки.

Вони корисні як готові блоки. Запам'ятайте їх цілими.

### Без: від їжі до плану

`Без` легко тренувати в побутових фразах. Почніть з їжі:

> Кава без цукру.
>
> Чай без молока.
>
> Салат без солі.

Потім перейдіть до речей у сумці:

> Я без карти.
>
> Вона без парасольки.
>
> Ми без ліхтарика.

Нарешті скажіть про допомогу або проблему:

> Без допомоги важко.
>
> Без сумніву, це правильна адреса.
>
> Без проблем, я зачекаю.

Це три рівні, але граматична рамка одна: `без` плюс родовий. Якщо
речення довше, спочатку знайдіть маленьку частину з `без`, а вже потім
читайте решту.

## Біля, навпроти, коло

Ці прийменники допомагають описати місце. Вони теж вимагають родового
відмінка.

### Біля

`Біля` - найпоширеніший безпечний варіант для місця поруч.

> Аптека біля ринку.
>
> Зупинка біля школи.
>
> Ми стоїмо біля будинку.
>
> Намет буде біля річки.

### Навпроти

`Навпроти` означає, що два місця розташовані одне напроти одного.

> Кафе навпроти бібліотеки.
>
> Банк навпроти вокзалу.
>
> Аптека навпроти лікарні.

Якщо ви стоїте на одному боці вулиці, а кафе на іншому, це може бути
`навпроти`.

### Коло

`Коло` також означає близькість. Воно менш нейтральне, ніж `біля`, але
його варто впізнавати.

> Коло хати росте дерево.
>
> Коло дороги стоїть автобус.
>
> Ми сидимо коло вогню.

У сучасній розмові для більшості ситуацій безпечніше сказати `біля`.
Але `коло` часто трапляється в піснях, літературі і живих регіональних
контекстах.

### Орієнтир і позиція

Коли описуєте місто або табір, спершу назвіть орієнтир. Орієнтир - це
місце, відносно якого ви пояснюєте позицію: школа, ринок, річка,
лікарня, бібліотека.

Потім виберіть прийменник:

| Позиція | Прийменник | Приклад |
| --- | --- | --- |
| поруч | біля | біля школи |
| на іншому боці | навпроти | навпроти бібліотеки |
| поруч, часто розмовно або образно | коло | коло хати |

У парі зробіть мінікарту без малюнка. Один студент каже орієнтир,
інший додає місце:

Після двох-трьох прикладів поміняйте ролі. Не треба описувати все
місто. Ціль - автоматично сказати родовий після просторового
прийменника.

### Запитання для класу

Відповідайте коротко, але повним українським фрагментом:

> Що є біля вашого дому?
>
> Що є навпроти вашої роботи або школи?
>
> Без чого ви не виходите з дому?
>
> Для кого або для чого у вас цей зошит?

Можливі відповіді:

> Біля мого дому є магазин.
>
> Навпроти моєї роботи є парк.
>
> Я не виходжу без телефона.
>
> Цей зошит для української мови.

Якщо форма після прийменника ще не готова, використайте безпечний
короткий варіант: `біля дому`, `без телефона`, `для мови`. Довгі
описи прийдуть пізніше.

## Типові помилки

### Не залишайте словникову форму після прийменника

Не кажемо:

> *для мама*
>
> *без молоко*
>
> *біля школа*

Кажемо:

> для мами
>
> без молока
>
> біля школи

### Не змішуйте `біля` і `в/у`

`В школі` означає, що людина всередині або на занятті. `Біля школи`
означає місце поруч зі школою.

> Я в школі. - я всередині або на занятті.
>
> Я біля школи. - я поруч зі школою.

Цей контраст практичний: якщо ви пояснюєте, де зустрітися, різниця
важлива.

## Контрольована продукція

Складіть коротке повідомлення для групового чату. Використайте всі три
значення: призначення, відсутність і місце.

Рамка:

> Привіт! Це ... для ...
>
> Я буду без ...
>
> Зустрічаємося біля ... / навпроти ...

Приклад:

> Привіт! Це ковдра для Олени.
>
> Я буду без карти, але з телефоном.
>
> Зустрічаємося біля станції, навпроти бібліотеки.

Це не вільний твір. Це контрольована продукція: ви берете готові
частини і змінюєте тільки кілька слів. Перевірте кожну фразу з
прийменником:

- `для кого?` - для Олени, для групи, для брата;
- `без чого?` - без карти, без телефона, без ліхтарика;
- `біля/навпроти чого?` - біля станції, навпроти бібліотеки.

Після цього прочитайте повідомлення вголос. Якщо речення звучить
занадто довго, розділіть його на два. Добра A2-відповідь може бути
короткою.

## Форма після прийменника

У цьому модулі багато жіночих іменників: `мама`, `річка`, `дорога`,
`карта`, `лікарня`, `станція`, `площа`, `бібліотека`. Після прийменника
вони часто мають закінчення `-и` або `-і`:

| Називний | Після прийменника |
| --- | --- |
| мама | для мами |
| річка | біля річки |
| дорога | навпроти дороги |
| карта | без карти |
| лікарня | біля лікарні |
| станція | біля станції |
| площа | біля площі |
| бібліотека | навпроти бібліотеки |

Не потрібно виводити правило для кожної групи самостійно. На А2
корисно тренувати пари вголос. Спочатку словникова форма, потім фраза:
`карта - без карти`, `станція - біля станції`, `бібліотека - навпроти
бібліотеки`.

Для чоловічих і середніх іменників також використовуйте готові пари:

> ліхтарик - без ліхтарика;
>
> намет - для намету;
>
> будинок - біля будинку;
>
> навчання - для навчання.

Коли в реченні є одразу два прийменники, розбивайте його на частини:

> Я без карти, але біля станції.

Перша частина: `без карти`. Друга частина: `біля станції`. Так довше
речення залишається зрозумілим.

### Швидка самоперевірка

Перед зошитом закрийте таблиці й завершіть фрази усно:

> Для кого ця річ? - Для ...
>
> Без чого важко? - Без ...
>
> Де зустрічаємося? - Біля ...
>
> Що на іншому боці? - Навпроти ...

Після кожної відповіді назвіть тільки форму після прийменника:
`мами`, `ліхтарика`, `станції`, `бібліотеки`. Якщо ви назвали
словникову форму, поверніться до пари в таблиці і повторіть її тричі.
Це коротка, але корисна підготовка до письмових вправ.

Додатково зробіть один власний рядок про клас: `Зошит для ...`,
`Я без ...`, `Стілець біля ...`. Учитель або партнер перевіряє тільки
слово після прийменника, не весь стиль речення.
Якщо рядок правильний, додайте другий із новим прийменником. Так ви
бачите, що система повторюється в різних ситуаціях: річ, людина, місце,
відсутність.
Працюйте повільно й точно.

## 📋 Підсумок

| Прийменник | Значення | Приклади |
| --- | --- | --- |
| для | призначення, користь, одержувач | для мами, для відпочинку |
| без | відсутність | без цукру, без допомоги |
| біля | поруч | біля школи, біля ринку |
| навпроти | напроти | навпроти бібліотеки |
| коло | поруч, біля | коло хати |

Питання для самоперевірки:

- Що ви п'єте без цукру або без молока?
- Що важливо для вас?
- Що є біля вашого дому?
- Що стоїть навпроти вашої школи, роботи або станції?
