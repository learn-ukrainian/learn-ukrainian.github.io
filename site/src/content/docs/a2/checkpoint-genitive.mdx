Це checkpoint для всього блоку A2.2 про родовий відмінок. Це не нова
велика тема, а спокійна перевірка: що вже працює, а що треба повторити.
Урок українською, але темп повільний. Спочатку дивимося на ситуацію,
потім упізнаємо форми, потім вибираємо правильну відповідь, і тільки
після цього будуємо власні короткі речення.

Ви перевіряєте чотири вміння:

- прийменники з родовим: `з/із/зі`, `від`, `після`, `для`, `без`,
  `біля`, `навпроти`, `коло`, `до`;
- повні фрази з прикметниками й займенниками: `мого нового друга`,
  `цієї старої книги`;
- родовий множини: `багато студентів`, `немає булок`, `кілька днів`;
- реальні ситуації: покупки, здоров'я, місто, час.

Працюйте як на тренуванні, не як на іспиті з несподіванками. Якщо форма
не приходить одразу, поверніться до питання `кого? чого?` і до знайомої
моделі.

## Частина 1: впізнавання

Спочатку не творіть форму, а впізнайте її. Це зменшує навантаження і
допомагає побачити знайомі сигнали.

Три питання:

1. Який прийменник або сигнал у реченні?
2. Яке значення він має?
3. Яка форма після нього?

Приклади:

| Речення | Сигнал | Значення | Форма |
| --- | --- | --- | --- |
| Я йду до аптеки. | до | напрямок | аптеки |
| Ми сидимо біля вікна. | біля | місце | вікна |
| Після уроку я вільний. | після | час | уроку |
| Це подарунок для мого брата. | для | призначення | мого брата |
| Я прийшов із магазину. | із | звідки | магазину |
| Ми чекаємо до шостої. | до | часова межа | шостої |

Якщо бачите `немає`, теж думайте про родовий:

> Немає квитків.
>
> Немає температури.
>
> Немає часу.

Класна інструкція:

> Підкресліть сигнал.
>
> Обведіть форму після сигналу.
>
> Скажіть функцію: місце, напрямок, час, відсутність, призначення.

Це перший рівень checkpoint. Тут не треба писати довгі речення. Треба
впізнати, що перед вами родовий.

## Частина 2: вибір правильної форми

Тепер перевіряйте повну фразу. Родовий відмінок змінює не тільки іменник,
а й прикметник або займенник.

| Називний | Родовий |
| --- | --- |
| мій новий друг | мого нового друга |
| ця стара книга | цієї старої книги |
| наш великий будинок | нашого великого будинку |
| ваша нова лікарня | вашої нової лікарні |
| це нове місце | цього нового місця |

Міні-алгоритм:

1. Знайдіть іменник.
2. Визначте рід: чоловічий, жіночий, середній.
3. Поставте іменник у родовий.
4. Узгодьте прикметник або займенник.

Приклади:

> без мого нового друга
>
> біля цієї старої школи
>
> до вашої нової лікарні
>
> коло цього нового ринку

Поширена помилка - узгодити тільки іменник і забути слово перед ним.

| Помилка | Краще |
| --- | --- |
| без мій новий друга | без мого нового друга |
| біля ця стара школи | біля цієї старої школи |
| до наша нова лікарні | до нашої нової лікарні |

Контрольована усна практика:

> Викладач: `друг`.
>
> Студент: `без мого нового друга`.
>
> Викладач: `школа`.
>
> Студент: `біля цієї старої школи`.

Тримайте фрази короткими. На цьому етапі не потрібно будувати довгий
текст.

## Частина 3: родовий множини

Родовий множини треба повторювати групами. Це checkpoint, тому мета не
вивчити всі винятки, а впевнено впізнати й ужити найчастіші форми.

| Модель | Приклади |
| --- | --- |
| `-ів` | студентів, столів, зошитів, квитків |
| нульове закінчення | книг, булок, міст, яблук |
| зміна основи | днів, слів, людей |
| м'який знак | питань, завдань, пісень |

Після кількості:

> багато студентів
>
> мало книг
>
> кілька днів
>
> п'ять пляшок води
>
> скільки людей?

Після `немає`:

> немає квитків
>
> немає булок
>
> немає питань
>
> немає вільних місць

Помічайте не тільки закінчення, а й ситуацію. На ринку: `п'ять пляшок
води`. У класі: `немає питань`. У місті: `багато людей`. У розкладі:
`кілька днів`.

Міні-диктант для пари:

1. Один студент читає: `багато ...`, `немає ...`, `кілька ...`.
2. Другий студент додає форму: `студентів`, `питань`, `днів`.
3. Потім міняються ролями.

Це контрольована продукція: відповідь коротка, але форма ваша.

## Частина 4: реальні ситуації

Тепер з'єднайте граматику з короткими сценаріями. Не додавайте складних
деталей. Одне-два речення достатньо.

**На ринку**

> Дайте, будь ласка, кілограм яблук і пляшку води.
>
> У вас немає свіжих огірків?

**У лікаря**

> Температури немає, але немає сил.
>
> Мені потрібно щось від кашлю.

**У місті**

> Як пройти до аптеки?
>
> Вона біля цього нового ринку?
>
> Музей навпроти старої школи.

**Про час**

> Після роботи я йду до лікаря.
>
> Чекайте до шостої.

Контрольоване письмо:

1. Напишіть одну фразу про місце з `біля`, `навпроти` або `коло`.
2. Напишіть одну фразу про напрямок з `до`.
3. Напишіть одну фразу про відсутність з `немає` або `без`.
4. Напишіть одну фразу про кількість.

Модель:

> Аптека біля нового ринку. Я йду до аптеки після роботи. У мене немає
> рецепта. Мені потрібні ліки від кашлю і пляшка води.

## Як працювати з помилками

Checkpoint корисний тільки тоді, коли помилка стає підказкою. Не
переписуйте весь блок. Знайдіть тип помилки й повторіть два-три приклади
цього типу.

| Якщо помилка тут | Повторіть |
| --- | --- |
| прийменник | `до аптеки`, `біля школи`, `після уроку` |
| узгодження | `мого нового друга`, `цієї старої книги` |
| множина | `багато студентів`, `немає питань` |
| ситуація | `кілограм яблук`, `ліки від кашлю` |

Міні-протокол після вправи:

1. Позначте одну неправильну відповідь.
2. Напишіть правильну форму.
3. Назвіть сигнал: `до`, `без`, `немає`, `багато`, `після`.
4. Складіть нове речення з тією самою моделлю.

Приклад:

> Помилка: `до аптека`.
>
> Правильно: `до аптеки`.
>
> Сигнал: `до`.
>
> Нове речення: `Я йду до школи`.

Це коротка самоперевірка. Вона тримає матеріал на рівні A2: ви не
пояснюєте всі винятки, а закріплюєте найчастіші моделі.

Після виправлення скажіть правильну фразу вголос двічі: один раз повільно,
один раз у нормальному темпі. Потім змініть тільки одне слово. Наприклад:
`до аптеки` -> `до школи`, `біля ринку` -> `біля музею`. Так ви
перевіряєте не пам'ять однієї відповіді, а саму модель.

Позначте для себе:

| Я можу... | Так / ще повторити |
| --- | --- |
| вибрати прийменник `до`, `для`, `без`, `біля`, `після` |  |

| утворити `мого нового друга` / `цієї старої книги` |  |
| сказати `багато студентів`, `немає питань` |  |
| попросити продукти на ринку |  |

| запитати дорогу до аптеки або музею |  |

Поверніться до відповідної частини й зробіть два приклади, не весь модуль
