## Коли потрібен родовий множини?

Родовий множини часто з'являється після слів кількості:

> багато студентів
>
> мало книг
>
> кілька днів
>
> скільки людей?
>
> немає квитків

З числами `п'ять` - `десять` теж часто потрібен родовий множини:

> п'ять столів
>
> шість книжок
>
> десять міст
>
> двадцять студентів

На А2 просто запам'ятайте контраст:

| Фраза | Форма |
| --- | --- |
| один студент | однина |
| два студенти | інша модель, не фокус цього модуля |
| п'ять студентів | родовий множини |
| багато студентів | родовий множини |

Не розширюйте це правило занадто швидко. Числа `два`, `три`, `чотири`
мають свою модель, і вона не є темою цього уроку. Тут тренуємо
ситуації, які часто потрібні в магазині, у класі, у подорожі:
`багато`, `мало`, `кілька`, `скільки`, `немає`, а також числа від
`п'ять` до `десять`.

## Чоловічий рід: найчастіше -ів

Для багатьох іменників чоловічого роду родовий множини має закінчення
`-ів`.

| Називний однини | Родовий множини |
| --- | --- |
| студент | студентів |
| брат | братів |
| стіл | столів |
| будинок | будинків |
| підручник | підручників |
| олівець | олівців |

Деякі слова змінюють основу:

| Називний | Родовий множини | Коментар |
| --- | --- | --- |
| день | днів | голосний зникає |
| хлопець | хлопців | е зникає |
| українець | українців | е зникає |

Є невелика група людей на `-ин/-їн`, де закінчення нульове:

> громадянин - громадян
>
> селянин - селян

Але це не основна модель. Для більшості звичайних чоловічих слів на А2
безпечна перша гіпотеза: `-ів`.

Карта першого вибору:

| Якщо бачите... | Спершу пробуйте... | Приклади |
| --- | --- | --- |
| чоловічий рід, предмет або людина | `-ів` | студентів, братів, столів |
| слово з випадним `е` | основу без `е` + `-ів` | хлопців, олівців |
| людина на `-ин/-їн` | коротку групову форму | громадян, селян |

Це не повний словник. Це робоча карта. Якщо слово часте для вашого
життя, запишіть його в маленьку особисту таблицю:

| Мені потрібно | Форма |
| --- | --- |
| багато квитків | квиток - квитків |
| п'ять уроків | урок - уроків |
| кілька номерів | номер - номерів |

Так родовий множини стає не абстрактною темою, а набором потрібних
фраз.

## Жіночий рід: часто нульове закінчення

Багато іменників жіночого роду в родовому множини втрачають `-а` або
`-я`.

| Називний однини | Родовий множини |
| --- | --- |
| книга | книг |
| школа | шкіл |
| сестра | сестер |
| банка | банок |
| сумка | сумок |
| зупинка | зупинок |
| пісня | пісень |

Іноді з'являється вставний голосний `о` або `е`, щоб кінець слова було
легше вимовити:

> банка - банок
>
> сумка - сумок
>
> сестра - сестер
>
> зупинка - зупинок

Деякі часті слова мають форму на `-ей`:

> стаття - статей
>
> сім'я - сімей
>
> миша - мишей

Слова на `-ія` часто мають форму без `-я`:

> лекція - лекцій
>
> станція - станцій
>
> історія - історій

Не намагайтесь вивести все з одного правила. Вивчайте модель плюс
кілька частих прикладів.

Для A2 корисно бачити три маленькі групи:

| Група | Приклади | Що запам'ятати |
| --- | --- | --- |
| просто без `-а` | книга - книг | коротка форма |
| вставний голосний | банка - банок, сестра - сестер | слово легше вимовити |
| м'яка форма | пісня - пісень | часто з `ь` |

Керуйте увагою студентів питаннями:

- `Чи є в називному -а або -я?`
- `Чи важко вимовити кінець без голосного?`
- `Чи це слово з короткого списку на -ей?`

Відповідь може бути невпевненою. Це нормально. Родовий множини жіночого
роду вчиться через багато знайомих слів.

## Середній рід: нульове, -ів або особлива основа

У середньому роді є кілька дуже частих моделей.

| Називний однини | Родовий множини |
| --- | --- |
| місто | міст |
| слово | слів |
| вікно | вікон |
| яблуко | яблук |
| море | морів |
| поле | полів |
| завдання | завдань |
| питання | питань |
| ім'я | імен |

Зверніть увагу:

> багато міст
>
> мало слів
>
> кілька вікон
>
> немає питань

Форми `завдань` і `питань` дуже корисні для навчання:

> У мене багато завдань.
>
> У вас є кілька питань?
>
> Немає питань.

Три безпечні фрази для щоденного вжитку:

> У мене багато завдань.
>
> У нас мало місць.
>
> Немає питань.

Вони важливіші за рідкісні винятки. Якщо студент може правильно
сказати `багато завдань`, `кілька питань`, `десять міст`, він уже має
корисний A2-інструмент.

> Треба шість пляшок води.

Це контрольована продукція: студент не мусить створювати довгий текст,

## Особистий список: що треба саме вам?

Родовий множини краще запам'ятовується в особистих фразах. Не треба
вчити всі можливі слова одразу. Виберіть десять слів, які потрібні вам
цього тижня: для навчання, роботи, дому або подорожі.

Модель особистої таблиці:

| Однина | Родовий множини | Моя фраза |
| --- | --- | --- |
| квиток | квитків | треба п'ять квитків |
| зошит | зошитів | багато зошитів |
| пляшка | пляшок | кілька пляшок води |
| яблуко | яблук | мало яблук |
| питання | питань | немає питань |

Після таблиці скажіть три речення про реальну ситуацію:

> Для уроку треба десять зошитів.
>
> Для подорожі треба кілька квитків.
>
> У мене немає питань.

Якщо ви не знаєте форму, не вигадуйте складне правило. Запитайте:

- `Як буде родовий множини від слова ...?`
- `Це часта форма чи виняток?`
- `Можна сказати це простіше?`

Іноді можна вибрати простішу фразу. Замість рідкісного слова в родовому
множини студент може сказати знайому кількість з іншим іменником. Це
не втеча від граматики, а нормальна A2-стратегія для живого мовлення.

Контрольований обмін у парі:

<DialogueBox
  client:only="react"
  title="Діалог"
  exchanges={JSON.parse('[{"speaker":"А","text":"У мене багато завдань. А в тебе?"},{"speaker":"Б","text":"У мене мало завдань, але багато питань."},{"speaker":"А","text":"Скільки питань?"},{"speaker":"Б","text":"П\'ять питань."}]')}
/>

Після обміну поміняйте тільки іменники: `квитки`, `уроки`, `пляшки`,
`яблука`, `місця`, `зошити`. Не змінюйте граматичну рамку, поки форми
не стануть автоматичними.

Міні-завдання для зошита:

1. Напишіть список із п'яти речей, які треба купити або взяти.
2. Використайте `треба п'ять...`, `кілька...`, `немає...`,
   `багато...`, `мало...`.
3. Підкресліть форму родового множини.
4. Перевірте одну незнайому форму з учителем або словником.

Для швидкої усної практики працюйте серіями. Серія 1: `один квиток -
п'ять квитків`, `один урок - п'ять уроків`, `один пакет - п'ять
пакетів`. Серія 2: `одна пляшка - п'ять пляшок`, `одна банка - п'ять
банок`, `одна булка - п'ять булок`. Серія 3: `одне місто - п'ять
міст`, `одне питання - п'ять питань`, `одне завдання - п'ять завдань`.
Не змішуйте всі серії одразу, поки форми не стануть знайомими.
Після кожної серії студент складає одне коротке речення про себе:
`Мені треба п'ять квитків`, `У мене багато завдань`, `У нас немає
пляшок води`.
Так форма не залишається в таблиці. Вона одразу переходить у прохання,
список або відповідь на просте питання.
Повторіть найпотрібнішу фразу на початку наступного уроку.

## 📋 Підсумок

Родовий множини має багато варіантів, але A2-стратегія проста:

| Тип | Частий результат | Приклади |
| --- | --- | --- |
| чоловічий рід | -ів | студентів, столів, підручників |
| жіночий рід | нульове закінчення, часто зі вставним голосним | книг, банок, сестер |
| середній рід | нульове, -ів або особлива форма | міст, морів, питань |
| кількість | часто родовий множини | багато книг, п'ять студентів |

Самоперевірка:

- Як сказати українською: багато + студент?
- Як сказати українською: немає + булка?
- Яка форма після `скільки`: `квитки` чи `квитків`?

Далі ви використаєте ці форми в живих ситуаціях: покупки, здоров'я,
наявність і відсутність речей у коротких діалогах.

Фінальна A2-ціль: ви не знаєте всі форми родового множини напам'ять.
Але ви впізнаєте слова кількості, правильно використовуєте часті
форми і вмієте перевірити нове слово в словнику або вчителя.
