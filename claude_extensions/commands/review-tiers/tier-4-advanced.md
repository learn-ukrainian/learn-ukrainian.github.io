## ADVANCED LEARNING EXPERIENCE AUDIT (Tier 4)

> **Key Question:** "Does this treat me as an advanced learner?"
>
> At C1/C2, learners are near-native. They need:
> - Sophisticated linguistic analysis
> - Nuanced explanations of subtle distinctions
> - Stylistic and register awareness
> - Academic and professional contexts
> - Intellectual challenge without condescension

### The "Did This Stretch Me?" Test

**Read the module as an advanced learner.** Ask:

| Question | Pass | Fail |
|----------|------|------|
| Was this at my level? | Appropriately challenging | Too easy or talked down to |
| Did I learn subtle distinctions? | Nuances I didn't know | Only reviewed basics |
| Was the linguistic analysis sophisticated? | Insightful, not surface-level | Oversimplified |
| Could I use this professionally? | Academic/professional application | Only casual usage |
| Did this respect my competence? | Treated as advanced learner | Felt like B1/B2 content |

**Scoring:**
- 5/5 Pass ‚Üí Learning Quality 10/10
- 4/5 Pass ‚Üí Learning Quality 9/10
- 3/5 Pass ‚Üí Learning Quality 8/10
- 2/5 Pass ‚Üí Learning Quality 7/10
- 0-1/5 Pass ‚Üí Learning Quality ‚â§6/10

### Advanced Learning Arc

**C1/C2 modules need intellectual depth:**

```
PROVOCATION ‚Üí EXPLORATION ‚Üí NUANCE ‚Üí MASTERY ‚Üí APPLICATION
```

| Element | What to Look For | Red Flag |
|---------|------------------|----------|
| **PROVOCATION** | Challenging question, counterintuitive claim, or complex problem | Simple topic introduction |
| **EXPLORATION** | Deep dive into linguistic subtleties | Surface-level overview |
| **NUANCE** | Register, style, regional, historical variations | One-size-fits-all rules |
| **MASTERY** | Sophisticated understanding demonstrated | Basic comprehension only |
| **APPLICATION** | Academic, professional, literary contexts | Only everyday usage |

## C1/C2 Specific Quality Markers

### Good C1/C2 Module Signs

- Opens with intellectual challenge, not basic intro
- Explores subtle distinctions (synonyms, near-synonyms, register)
- Shows formal vs. informal, written vs. spoken, regional variation
- Uses authentic texts (not simplified)
- Includes production activities (essays, stylistic analysis)
- Treats learner as near-native, not student
- Acknowledges complexity and exceptions
- Connects to academic/professional contexts

### Red Flags (Advanced)

- ‚ùå Starts with "In this lesson, you will learn..."
- ‚ùå Explains basic concepts (cases, aspects) as if new
- ‚ùå Uses simplified examples for advanced content
- ‚ùå Only recognition activities (no production)
- ‚ùå Ignores register and style
- ‚ùå Treats exceptions as "just memorize"
- ‚ùå No authentic text samples
- ‚ùå Condescending tone ("Don't worry, this is easy!")

## Production Activity Requirements

**C1/C2 MUST include production activities:**

### Acceptable Types
- Essay response (stylistic, analytical)
- Text transformation (register shift, paraphrase)
- Creative writing (within constraints)
- Stylistic analysis (identify and explain)
- Error correction (nuanced, not basic)

### Unacceptable for C1/C2
- ‚ùå Simple fill-in-the-blank (unless testing nuance)
- ‚ùå Basic matching (unless subtle distinctions)
- ‚ùå True/false (unless deeply analytical)
- ‚ùå Recognition-only activities dominating

**Minimum production:** ‚â•30% of activities must require production, not just recognition.
