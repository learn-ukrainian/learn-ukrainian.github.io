## Immersion Rules

**Grammar Modules (65-70%):**
- UKRAINIAN: Narratives, dialogues, examples, grammar rules in Ukrainian
- ENGLISH: Complex theory (passive voice, participles, register)

**Vocab Modules (80-85%):**
- UKRAINIAN: All narratives, dialogues, explanations, cultural content
- ENGLISH: Only historical/cultural context requiring clarification

## Structure (TTT/CLIL)

### TTT Structure
1. `## Діагностика` - Diagnostic task
2. `## Аналіз` - Analysis of patterns
3. `## Поглиблення` - Deep dive
4. `## Практика` - Controlled practice
5. `# Вправи` - Activities
6. `# Підсумок` - Summary
7. `# Словник` - Vocabulary table

### CLIL Structure (vocab modules)
1. `## Вступ` - Introduction with content hook
2. `## Наратив` - Extended narrative/reading
3. `## Аналіз` - Analysis and discussion
4. `## Граматика в контексті` - Grammar in context
5. `# Вправи` - Activities
6. `# Підсумок` - Summary
7. `# Словник` - Vocabulary table

## Activity Format Reminders (CRITICAL)

### Error-correction (MUST use all 4 callouts)
```markdown
## error-correction: Title
> Instructions here.

1. Sentence with error.
   > [!error] wrong_word
   > [!answer] correct_word
   > [!options] wrong | correct | distractor1 | distractor2
   > [!explanation] Why it's wrong and the rule.
```

### Unjumble (MUST use [!answer] callout)
```markdown
## unjumble: Title
> Instructions here.

1. слова / в / перемішаному / порядку
   > [!answer] Слова в правильному порядку.
```
