## Content-Heavy Modules (History M71-131)

**History modules have DIFFERENT requirements:**

| Aspect         | Standard B2 | History (M71-131) |
| -------------- | ----------- | ----------------- |
| Activity Count | 14+         | **10-12**         |
| Activity Mix   | All types   | Comprehension     |
| Quiz Questions | General     | **Text-based**    |

### Golden Rule

**"Can the learner answer without reading the Ukrainian text?"**

- If YES → Rewrite (tests history, not language)
- If NO → Keep (tests Ukrainian)

### Forbidden Patterns (❌ Tests Content)

- "У якому році..." (dates)
- "Хто був..." (names)
- "Скільки..." (numbers)

### Required Patterns (✅ Tests Language)

- "Згідно з текстом, як автор..."
- "У тексті модуля автор характеризує..."
- "Який аргумент автор наводить..."

## Immersion Rules (100% Ukrainian)

**All B2 Modules:**

- UKRAINIAN: ALL content — narratives, dialogues, examples, grammar theory, cultural content, explanations, tips
- ENGLISH: **ONLY** vocabulary table translations (in YAML)

**Why 100%?** B2 learners have internalized all grammar terminology from B1. English scaffolding is no longer needed and would be a step backwards.

## Structure (TTT/CLIL)

### TTT Structure

1. `## Діагностика` - Diagnostic task
2. `## Аналіз` - Analysis of patterns
3. `## Поглиблення` - Deep dive
4. `## Практика` - Controlled practice
5. `## Підсумок` - Summary

### CLIL Structure (vocab/history modules)

1. `## Вступ` - Introduction with content hook
2. `## Наратив` - Extended narrative/reading
3. `## Аналіз` - Analysis and discussion
4. `## Граматика в контексті` - Grammar in context
5. `## Підсумок` - Summary
