## Workflow Integration

**B2 Core uses the 4-stage core workflow:**

1. **Write** → Create lesson content and activities manually
2. **Audit** → Validate content quality and completeness
3. **Generate** → Convert to MDX for Docusaurus
4. **Validate** → Final HTML testing and deployment

**Commands:**

- Start: Write content manually in curriculum/l2-uk-en/b2/
- Audit: `.venv/bin/python scripts/audit_module.py curriculum/l2-uk-en/b2/{slug}.md`
- Generate: `python scripts/generate_mdx.py l2-uk-en b2 {num}`
- Validate: Check MDX output in docusaurus/docs/b2/

**Reference:** `docs/SCRIPTS.md` for core workflow details.

## Immersion Rules (100% Ukrainian)

**All B2 Modules:**

- UKRAINIAN: ALL content — narratives, dialogues, examples, grammar theory, cultural content, explanations, tips
- ENGLISH: **ONLY** vocabulary table translations (in YAML)

**Why 100%?** B2 learners have internalized all grammar terminology from B1. English scaffolding is no longer needed and would be a step backwards.

## Activity Format

**All activities MUST be in YAML format.**
See `docs/ACTIVITY-YAML-REFERENCE.md` for full syntax.

### Common Activity Format Examples

**Quiz (min 10 items for B2):**

```yaml
- type: quiz
  title: 'Quiz title'
  instruction: 'Select the correct answer' # singular!
  items:
    - question: 'Question text here?'
      options:
        - text: 'Correct answer'
          correct: true
        - text: 'Wrong answer 1'
          correct: false
        - text: 'Wrong answer 2'
          correct: false
        - text: 'Wrong answer 3'
          correct: false
      explanation: 'Explanation why answer is correct'
```

**Fill-in (min 10 items, MUST have options):**

```yaml
- type: fill-in
  title: 'Fill-in title'
  instruction: 'Fill in the blanks'
  items:
    - sentence: 'Text with [___] blank here'
      answer: правильна_відповідь
      options: # REQUIRED - exactly 4 options
        - правильна_відповідь
        - неправильна1
        - неправильна2
        - неправильна3
```

**Cloze (min 16 blanks, inline format):**

```yaml
- type: cloze
  title: 'Cloze title'
  instruction: 'Select correct words'
  passage: 'Text with {correct|wrong1|wrong2|wrong3} inline choices and {another|choice|here|example} more.'
  # NO blanks: [...] array when using inline format!
```

**Unjumble (min 8 items, 6-18 words per sentence):**

```yaml
- type: unjumble
  title: 'Sentence reconstruction'
  instruction: 'Put words in correct order'
  items:
    - words: ['слово1', 'слово2', 'слово3', 'слово4', 'слово5', 'слово6']
      answer: 'слово1 слово2 слово3 слово4 слово5 слово6'
```

**Error-correction (min 8 items):**

```yaml
- type: error-correction
  title: 'Fix errors'
  instruction: 'Find and correct ONE error per sentence'
  items:
    - sentence: 'Речення з помилкою тут'
      error: 'помилкою'
      answer: 'помилкой'
      options: ['помилкою', 'помилкой', 'помилку', 'помилки']
      explanation: 'Explanation of the error'
```

**Common Pitfalls:**

- ❌ `instructions` → ✅ `instruction` (singular)
- ❌ `writing` → ✅ `essay-response` (for long-form tasks)
- ❌ Single quotes with apostrophes `'інтерв'ю'` → ✅ Double quotes `"інтерв'ю"`
- ❌ `id:` field in most activities → ✅ Remove (not in B2 schema)
