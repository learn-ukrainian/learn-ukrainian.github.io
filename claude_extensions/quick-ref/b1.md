# B1 Quick Reference

## Workflow Integration

**B1 uses the 4-stage core workflow:**

1. **Write** → Create lesson content and activities manually
2. **Audit** → Validate content quality and completeness
3. **Generate** → Convert to MDX for Docusaurus
4. **Validate** → Final HTML testing and deployment

**Commands:**

- Start: Write content manually in curriculum/l2-uk-en/b1/
- Audit: `.venv/bin/python scripts/audit_module.py curriculum/l2-uk-en/b1/{slug}.md`
- Generate: `python scripts/generate_mdx.py l2-uk-en b1 {num}`
- Validate: Check MDX output in docusaurus/docs/b1/

**Reference:** `docs/SCRIPTS.md` for core workflow details.

**Visual aids (grammar modules):** Use tables for paradigms, comparisons, and category summaries. Use mermaid flowcharts for decision logic (aspect, case selection). Not mandatory — use when they clarify better than prose.

### Content Quality Standards (B1+ Modules)

- **Equal treatment**: all items in a category get same depth and format (e.g., all 10 POS in one table, not 6 table + 3 list + 1 casual)
- **No English contamination** in prose, titles, or callouts (100% Ukrainian for B1.1+)
- **Example variety**: mix standalone, table, inline, dialogue formats — no 5+ consecutive `_Приклад:_`
- **Tables must have narrative context** (2+ sentences before and after)
- **Parallel sections** use identical internal structure (if Section A = intro → table → examples → callout, Section B = same)

## Immersion Rules (B1.1+ = 100%)

**From B1.1 onwards, content is 100% Ukrainian:**

- UKRAINIAN: ALL explanations, grammar theory, dialogues, narratives, engagement boxes
- ENGLISH: **Only** vocabulary table translations

### ⛔ FORBIDDEN: English Annotations in Explanations

**DO NOT use English in parentheses to clarify Ukrainian concepts:**

```markdown
❌ WRONG: - **поки** — дія на тлі іншої дії (While she was cooking...)
❌ WRONG: - **перш ніж** — дія перед іншою (Before)
❌ WRONG: - **як тільки** — негайна послідовність (As soon as)

✅ CORRECT: - **поки** — дія на тлі іншої дії, тобто одночасні процеси
✅ CORRECT: - **перш ніж** — дія перед іншою, попередня дія
✅ CORRECT: - **як тільки** — негайна послідовність, миттєва реакція
```

**Exception - B1.0 Metalanguage Bridge (M01-05):**

- English grammar term explanations allowed
- Transition from A2's mixed approach

## B1 Critical Requirements

### All Content in Ukrainian (100% Immersion)

**WRONG (uses English):**

```markdown
> 🌍 **Real World**
> In Ukrainian job interviews, interviewers often ask...
```

**CORRECT (100% Ukrainian):**

```markdown
> 🌍 **У реальному житті**
> На українських співбесідах часто питають:
> «Що ви робили?» (What did you do?)
```

### Grammar Theory in Ukrainian

All explanations must be in Ukrainian. This is how Ukrainians learn grammar in school. English is ONLY for:

- Vocabulary table translations

### Sentence Length

- **Max 20 words** per sentence
- Check tip boxes and summary sections

### Pre-Delivery Check

Run audit to verify immersion: `.venv/bin/python scripts/audit_module.py path/to/module.md`
If immersion < 95%, check for English content outside vocabulary tables.
