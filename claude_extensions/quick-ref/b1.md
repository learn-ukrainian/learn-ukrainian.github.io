| Module Type | Reference       | Look For                                            |

**How to use references:**

1. Read 1-2 similar modules before starting
2. Note activity variety and sequencing
3. Copy structural patterns (not content)
4. Adapt engagement box styles
5. Use YAML format for new modules (M51+)
