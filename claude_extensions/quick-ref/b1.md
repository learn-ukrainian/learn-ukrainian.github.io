## Reference Patterns from Existing Modules

**Before writing new B1 modules (M51+), study existing modules for patterns:**

| Module Type | Reference       | Look For                                            |
| ----------- | --------------- | --------------------------------------------------- |
| Grammar     | M06-10 (Aspect) | Activity structure, TTT flow, engagement boxes      |
| Vocabulary  | M52-53          | YAML format, semantic grouping, cloze passage style |
| Cultural    | Any A2 cultural | Cultural angle, real-world boxes                    |

**How to use references:**

1. Read 1-2 similar modules before starting
2. Note activity variety and sequencing
3. Copy structural patterns (not content)
4. Adapt engagement box styles
5. Use YAML format for new modules (M51+)
