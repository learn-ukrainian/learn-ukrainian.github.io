## Immersion Rules (Graduated 3-Band)

A2 uses a graduated immersion ramp that increases Ukrainian exposure across the level:

| A2 Range | Modules | Immersion | English Policy |
|----------|---------|-----------|----------------|
| Core grammar | M01-20 | 50-60% | English for grammar theory (cases, aspect); Ukrainian for dialogues, examples, practice |
| Applied grammar | M21-50 | 60-75% | English only for abstract concepts (conditionals, reported speech); Ukrainian for everything else |
| Consolidation | M51-70 | 75-90% | English only in vocabulary tables; near-full Ukrainian prose |

**Ramp continuity:** A1 (~40%) → A2 start (50%) → A2 end (75-90%) → B1 bridge (70-85%, metalanguage) → B1 M06+ (100%)

### Band 1: Core Grammar (M01-20, 50-60%)

Write IN UKRAINIAN:
- Dialogues and examples
- Simple instructions and practice prompts
- Familiar vocabulary in context

Write IN ENGLISH:
- Grammar theory (all 7 cases, aspect basics)
- Complex explanations and rules

### Band 2: Applied Grammar (M21-50, 60-75%)

Write IN UKRAINIAN:
- All dialogues, examples, and practice
- Grammar explanations for familiar concepts
- Section introductions and transitions

Write IN ENGLISH:
- Abstract grammar concepts (conditionals, reported speech, word formation)
- Complex comparative explanations

### Band 3: Consolidation (M51-70, 75-90%)

Write IN UKRAINIAN:
- Nearly everything: prose, explanations, examples, dialogues
- Grammar explanations (even for new concepts)

Write IN ENGLISH:
- Vocabulary tables (translation column only)
- Brief clarifications where Ukrainian-only would cause confusion

## Structure (PPP)

1. `## Warm-up` - Hook and context
2. `## Presentation` - New material
3. `## Practice` - Controlled practice
4. `## Cultural Insight` - Optional
5. `## Summary` - Recap

## A2 Critical Requirements

### NO Transliteration (A2 = modules 35+)

- **NO** `слово (English)` patterns in body text
- **NO** `(hard)` / `(soft)` patterns - use "for hard stems" instead
- **NO** parenthetical translations anywhere

### Sentence Length

- **Max 15 words** per sentence
- Check tip boxes and summary sections especially

### Grammar Terms in Vocabulary

If you explain grammar concepts, add the terms to vocabulary:

- відмінок (case), називний (nominative), родовий (genitive)
- давальний (dative), знахідний (accusative), орудний (instrumental)
- місцевий (locative), кличний (vocative), час (tense)
