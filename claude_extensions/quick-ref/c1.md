## Content-Heavy Modules (Folk/Literature)

**Content-heavy C1 modules have DIFFERENT requirements:**

| Module Type    | Range   | Activity Count | Focus                  |
| -------------- | ------- | -------------- | ---------------------- |
| **Folk**       | M56-85  | **10-12**      | Cultural analysis      |
| **Literature** | M86-105 | **10-12**      | Textual interpretation |

### Golden Rule

**"Can the learner answer without reading the Ukrainian text?"**

- If YES ‚Üí Rewrite (tests content, not language)
- If NO ‚Üí Keep (tests Ukrainian)

### Forbidden Patterns (Tests Content Recall)

- "–£ —è–∫–æ–º—É —Ä–æ—Ü—ñ..." (dates)
- "–•—Ç–æ –±—É–≤..." (names/facts)
- "–©–æ —Å–∏–º–≤–æ–ª—ñ–∑—É—î..." (without text reference)

### Required Patterns (Tests Ukrainian Language)

- "–ó–≥—ñ–¥–Ω–æ –∑ —Ç–µ–∫—Å—Ç–æ–º, —è–∫ –∞–≤—Ç–æ—Ä..."
- "–£ —Ç–µ–∫—Å—Ç—ñ –º–æ–¥—É–ª—è –∞–≤—Ç–æ—Ä —ñ–Ω—Ç–µ—Ä–ø—Ä–µ—Ç—É—î..."
- "–Ø–∫—É —Å—Ç–∏–ª—ñ—Å—Ç–∏—á–Ω—É —Ñ—É–Ω–∫—Ü—ñ—é –≤–∏–∫–æ–Ω—É—î..."

### Activity Focus

Content-heavy modules test **vocabulary usage** and **grammatical comprehension**, not cultural/literary knowledge:

- **quiz (4-5):** MUST reference module text
- **fill-in/cloze (3-4):** Test collocations in context
- **error-correction (2-3):** Fix grammar, NOT factual errors

**Full Ukrainian immersion at C1.** All content is in Ukrainian:

- All narratives and dialogues

- All activity instructions
- All tips and engagement boxes
