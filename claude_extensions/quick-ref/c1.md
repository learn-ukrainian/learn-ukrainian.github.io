## Content-Heavy Modules (Biography/Folk/Literature)

**Content-heavy C1 modules have DIFFERENT requirements:**

| Module Type   | Range    | Activity Count | Focus                   |
| ------------- | -------- | -------------- | ----------------------- |
| **Biography** | M36-100  | **10-12**      | Language comprehension  |
| **Folk**      | M121-145 | **10-12**      | Cultural analysis       |
| **Literature** | M146-160 | **10-12**      | Textual interpretation  |

### Golden Rule

**"Can the learner answer without reading the Ukrainian text?"**

- If YES → Rewrite (tests content, not language)
- If NO → Keep (tests Ukrainian)

### Forbidden Patterns (❌ Tests Content Recall)

- "У якому році..." (dates)
- "Хто був..." (names/facts)
- "Що символізує..." (without text reference)

### Required Patterns (✅ Tests Ukrainian Language)

- "Згідно з текстом, як автор..."
- "У тексті модуля автор інтерпретує..."
- "Яку стилістичну функцію виконує..."

### Activity Focus

Content-heavy modules test **vocabulary usage** and **grammatical comprehension**, not biographical/cultural/literary knowledge:

- **quiz (4-5):** MUST reference module text
- **fill-in/cloze (3-4):** Test collocations in context
- **error-correction (2-3):** Fix grammar, NOT factual errors

## Immersion Rules (100%)

**Full Ukrainian immersion at C1.** All content is in Ukrainian:

- All grammar explanations
- All narratives and dialogues
- All cultural content, analyses, literary critiques
- All activity instructions
- All tips and engagement boxes

**English ONLY in:**

- Vocabulary table "translation" column (in YAML)

**No Language Link boxes at C1.** Students learned all grammar terminology by B1.
