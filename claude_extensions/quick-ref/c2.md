## Immersion Rules (100%)

**Full Ukrainian immersion at C2.** All content is in Ukrainian:
- EVERYTHING - learner operates as near-native
- All grammar, explanations, cultural content
- All activity instructions and tips
- All stylistic analysis and literary critique

**English ONLY in:**
- Vocabulary table "Переклад" column (translations)

**Note:** Latin/Greek scholarly terms (e.g., "damnatio memoriae", "genius loci") are acceptable in academic contexts.

## Structure (TTT/CLIL)

### TTT Structure
1. `## Діагностика` - Diagnostic task (native-level)
2. `## Аналіз` - Deep linguistic analysis
3. `## Поглиблення` - Stylistic/register nuances
4. `## Практика` - Production practice
5. `## Вільна практика` - Free production
6. `# Вправи` - Activities
7. `# Підсумок` - Summary
8. `# Словник` - Vocabulary table

### CLIL Structure
1. `## Вступ` - Academic/professional context
2. `## Наратив` - Extended authentic text
3. `## Аналіз` - Critical analysis
4. `## Граматика в контексті` - Advanced grammar
5. `# Вправи` - Activities
6. `# Підсумок` - Summary
7. `# Словник` - Vocabulary table

## Activity Format Reminders (CRITICAL)

### Error-correction (MUST use all 4 callouts)
```markdown
## error-correction: Title
> Instructions here.

1. Sentence with error.
   > [!error] wrong_word
   > [!answer] correct_word
   > [!options] wrong | correct | distractor1 | distractor2
   > [!explanation] Why it's wrong and the rule.
```

### Unjumble (MUST use [!answer] callout)
```markdown
## unjumble: Title
> Instructions here.

1. слова / в / перемішаному / порядку
   > [!answer] Слова в правильному порядку.
```

**See `docs/ACTIVITY-MARKDOWN-REFERENCE.md` for complete syntax.**
