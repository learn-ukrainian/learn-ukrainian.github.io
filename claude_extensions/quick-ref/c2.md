## Immersion Rules (95-100%)

ALLOWED (up to 5%):

## Structure (TTT/CLIL)

### TTT Structure
1. `## Діагностика` - Diagnostic task (native-level)
2. `## Аналіз` - Deep linguistic analysis
3. `## Поглиблення` - Stylistic/register nuances
4. `## Практика` - Production practice
5. `## Вільна практика` - Free production
6. `# Вправи` - Activities
7. `# Підсумок` - Summary
8. `# Словник` - Vocabulary table

### CLIL Structure
1. `## Вступ` - Academic/professional context
2. `## Наратив` - Extended authentic text
3. `## Аналіз` - Critical analysis
4. `## Граматика в контексті` - Advanced grammar
5. `# Вправи` - Activities
6. `# Підсумок` - Summary
7. `# Словник` - Vocabulary table
