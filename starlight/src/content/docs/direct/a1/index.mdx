# Українська з нуля

Курс української мови для дорослих. Від нуля до рівня A1.

Мова навчання — українська. Без перекладу. Значення слів — через зображення, відео та контекст.

---

## Як працює курс

1. **Дивись і слухай** — відео з вимовою, зображення, емоджі
2. **Повторюй уголос** — вимова з першого уроку
3. **Виконуй вправи** — інтерактивні завдання після кожної теми
4. **Крок за кроком** — від букв до речень, від простого до складного
