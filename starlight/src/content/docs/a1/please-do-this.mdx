You can now call a person naturally: **Олено! Тарасе! Друже!** This module
adds the next useful piece: asking someone to do something.

Ukrainian uses short command/request forms:

- **Читай, будь ласка.** - Read, please.
- **Читайте, будь ласка.** - Please read. (polite or plural)
- **Дай мені меню, будь ласка.** - Give me the menu, please.
- **Дайте рахунок, будь ласка.** - Please give the bill.

By the end, you can:

- recognize the school term **наказовий спосіб**;
- use common **ти** commands with one familiar person;
- use common **ви** commands with a teacher, server, stranger, or group;
- soften a command with **будь ласка**;
- keep the lesson practical: commands and polite requests, not a full verb
  paradigm.

:::tip[Polite habit]
For A1, build the whole line: **Олено, прочитай, будь ласка.** Address +
action + **будь ласка** is clear and friendly.
:::

## Наказовий спосіб

The school term is **наказовий спосіб**. It covers commands, requests,
instructions, and invitations. At A1, use only two active forms:

| Use | One person | Polite or plural |
| --- | --- | --- |
| read | **читай** | **читайте** |
| write | **пиши** | **пишіть** |
| listen | **слухай** | **слухайте** |
| speak | **говори** | **говоріть** |
| look | **дивись** | **дивіться** |
| go | **іди** | **ідіть** |
| give | **дай** | **дайте** |
| say/tell | **скажи** | **скажіть** |

The social choice is the important part:

- Use **ти** with a friend, child, close family member, or one teammate:
  **Тарасе, читай.**
- Use **ви** with a teacher, server, stranger, older person, or more than one
  person: **Пане Іване, читайте.**
- Add **будь ласка** to make the request polite:
  **Дайте меню, будь ласка.**

This is direct Ukrainian, not rude Ukrainian. **Читай!** can be a normal class
instruction. **Дай, будь ласка** can be a friendly request. Tone, relationship,
and **будь ласка** do the politeness work.

## Як утворити?

For A1, learn the common forms as pairs.

### The ti form

Use **ти** commands with one familiar person:

| Infinitive | ти command | A1 line |
| --- | --- | --- |
| **читати** | **читай** | **Олено, читай.** |
| **слухати** | **слухай** | **Тарасе, слухай.** |
| **писати** | **пиши** | **Пиши речення.** |
| **говорити** | **говори** | **Говори повільно.** |
| **дивитися** | **дивись** | **Подивись сюди.** |
| **іти** | **іди** | **Іди додому.** |
| **дати** | **дай** | **Дай меню, будь ласка.** |
| **сказати** | **скажи** | **Скажи, що ти хочеш.** |
| **сісти** | **сядь** | **Сядь тут.** |
| **відкрити** | **відкрий** | **Відкрий двері.** |

### The vy form

For polite speech or a group, the form often adds **-те** or **-іть**:

| ти | ви |
| --- | --- |
| **читай** | **читайте** |
| **слухай** | **слухайте** |
| **пиши** | **пишіть** |
| **говори** | **говоріть** |
| **дивись** | **дивіться** |
| **ходи** | **ходіть** |
| **дай** | **дайте** |
| **скажи** | **скажіть** |
| **іди** | **ідіть** |
| **сядь** | **сядьте** |
| **відкрий** | **відкрийте** |

Do not overbuild the rule. Many forms are short and frequent. Memorize the
ones you need for real requests: **дайте**, **скажіть**, **відкрийте**,
**пишіть**, **читайте**.

### Recognition-only repairs

The wiki mentions wider imperative material so writers do not create mistakes.
At A1, recognize these patterns; do not produce a full table:

- **Recognition note 1: Базові інструкції (2-га особа однини і множини).** Start with
  **ти/ви** contrasts such as **пробач - пробачте**, **допоможи -
  допоможіть**, **читай**, **пиши**, **слухай**. Forms like **доповідай** and
  **розповідай** from **доповісти** and **розповісти** show that not every
  command is formed mechanically from the infinitive.
- **Recognition note 2: Форми спільної дії (1-ша особа множини).** Later you will meet
  **-мо** forms such as **співаймо** and **робімо**. For now, just know that
  Ukrainian has native forms for "let's"; do not build them with **давай**.
- **Recognition note 3: Третя особа та частки «хай/нехай».** Later you will recognize
  **хай пробачить**, **нехай допоможе**, **хай пишуть**, **хай скаже**, and
  **хай сядуть**. Do not use an infinitive after **хай**.
- **Recognition note 4: Фонетичні та орфографічні обмеження.** In imperative forms after
  **в, п, ж, ч, ш, р**, do not write a soft sign: **поставмо**, **сипмо**,
  **їжмо**, **кличмо**, **облишмо**, **вірмо**. Other forms may have a soft
  sign for sounds like [д'], [т'], [з'], [с'], [ц'], [л'], [н'], as in
  **виходьте**, **погодься**, **посунься**, **підносься**, **погляньмо**.
- **Recognition note 5: Поглиблення через паралельні форми.** Later, A2 can recognize
  pairs such as **ходіть - ходіте** and **варіть - `варіте`**, and strict
  commands such as **Стояти!**. Do not make these the active A1 target.

## 📋 Підсумок

<span id="підсумок"></span>

Use this as your active A1 request map:

| Situation | Say this |
| --- | --- |
| to a friend | **Олено, читай, будь ласка.** |
| to a teacher or stranger | **Читайте, будь ласка.** |
| asking for a menu | **Дайте меню, будь ласка.** |
| asking a friend for the menu | **Дай меню, будь ласка.** |
| asking someone to say something | **Скажи / скажіть, будь ласка.** |
| class instruction | **Відкрийте підручники. Пишіть речення.** |
| cafe request | **Дайте рахунок, будь ласка.** |

Common repairs:

| Problem | Use |
| --- | --- |
| infinitive as request | **Будь ласка, читай текст. / Будь ласка, читайте текст.** |
| copied "let's" pattern | **Робімо це зараз!** |
| infinitive after **хай** | **Хай він працює.** |
| soft sign copied into a command | **Постав книгу на стіл.** |
| mechanical form from **доповісти** | **Доповідай керівнику.** |
| mixed ending | **Читаймо разом!** |

The active lesson remains simple: **читай / читайте**, **пиши / пишіть**,
**дай / дайте**, **скажи / скажіть**, plus **будь ласка**. Use the wider
repairs only to avoid common interference.

Keep the decolonized grammar habit: Ukrainian does not need **«давай»** as a
form-building shortcut for commands. Use the Ukrainian command form itself:
**слухаймо** when that later form is in scope, and **слухай / слухайте** here.
For spelling, do not explain Ukrainian by saying "like another language but
without a soft sign." Treat forms such as **постав**, **вір**, and **їж** as
Ukrainian patterns in their own right. The learner builds Ukrainian from
Ukrainian.

Next module links short ideas. You will be able to say:
**Олено, читай і пиши, будь ласка.** For now, keep one clear action at a time.
