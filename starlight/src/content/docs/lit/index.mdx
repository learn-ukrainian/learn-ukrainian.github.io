# 🔍 LIT - Українська література

**На перевірці — 0/221 готово, 221 у QA**

## Від читача до науковця — українська література як духовна скарбниця нації!

Цей трек — для тих, хто хоче не просто читати українську літературу, а глибоко аналізувати її. «Кобзар» Шевченка, драми Лесі Українки, проза Франка — ви вивчатимете ці тексти на рівні літературознавця.

**Чого ви навчитеся:**
- Бурлескна традиція — Котляревський, «Енеїда»
- Сентименталізм — Квітка-Основ'яненко
- Романтизм — Шевченко, «Кобзар»
- Реалізм — Нечуй-Левицький, Панас Мирний
