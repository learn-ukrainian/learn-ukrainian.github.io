Головна ідея проста:

- кількість часто бере родовий: `кілограм яблук`, `пляшка води`;

## На ринку: скільки вам?

На ринку родовий відмінок часто з'являється після слів міри.

| Міра | Приклад |
| --- | --- |
| кілограм | кілограм яблук, кілограм помідорів |
| пів кіло | пів кіло сиру, пів кіло огірків |
| літр | літр молока |
| пляшка | пляшка води |
| пачка | пачка масла |
| банка | банка меду |
| шматок | шматок хліба |
| десяток | десяток яєць |

Природний діалог має не тільки граматику, а й ввічливість.

> Добрий день. Що сьогодні свіже?
>
> Дайте, будь ласка, кілограм яблук.
>
> Мені пів кіло сиру.
>
> Скільки коштує кілограм?
>
> Це дорого чи нормально?

`Пів кіло` в розмові дуже часте. Також можна почути `півкіло`.

> Пів кіло помідорів, будь ласка.
>
> Мені півкіло сиру.

Не плутайте:

| Називний | Після міри |
| --- | --- |
| яблука | кілограм яблук |
| помідори | пів кіло помідорів |
| сир | пів кіло сиру |
| вода | пляшка води |
| яйця | десяток яєць |

## У лікаря: що вас турбує?

У лікаря вам потрібні короткі й точні речення. Родовий відмінок часто
працює з `немає`.

> Температури немає.
>
> Немає апетиту.
>
> Немає сил.
>
> Немає кашлю, але болить горло.

Питання лікаря:

> Що вас турбує?
>
> Як давно?
>
> Чи є температура?
>
> Чи є алергія на ліки?
>
> Ви маєте рецепт?

Відповіді пацієнта:

> У мене болить горло.
>
> У мене кашель.
>
> Температури немає.
>
> У мене алергія на ці ліки.
>
> Рецепта немає.

`Від` може показувати причину:

> болить від холоду
>
> кашель від алергії
>
> голова болить від стресу

Але в аптеці `від` часто означає "проти":

> ліки від кашлю
>
> краплі від нежитю
>
> таблетки від головного болю

## В аптеці

В аптеці корисні три моделі:

| Модель | Приклад |
| --- | --- |
| Мені потрібно ... | Мені потрібні краплі. |
| Що ви порадите від ...? | Що ви порадите від кашлю? |
| Чи це без ...? | Це без рецепта? |

## Один день: ринок і здоров'я

Ринок і здоров'я виглядають різними темами, але граматика повторюється.

| Ситуація | Родовий |
| --- | --- |
| покупка | кілограм яблук |
| упаковка | пляшка води |
| відсутність | немає температури |
| ліки | таблетки від головного болю |
| режим | після їжі |
| обмеження | без цукру, без кофеїну |

Міні-завдання:

1. Складіть список із трьох продуктів: `кілограм ...`, `пляшка ...`,
   `пачка ...`.
2. Скажіть лікарю дві фрази: `болить ...`, `немає ...`.
3. Запитайте в аптеці: `Що ви порадите від ...?`

## 📋 Підсумок

У покупках і здоров'ї родовий відмінок працює дуже практично.

| Функція | Модель | Приклади |
| --- | --- | --- |
| кількість | міра + родовий | кілограм яблук, пляшка води |
| відсутність | немає / без + родовий | немає температури, без рецепта |
| засіб | від + родовий | ліки від кашлю |
| режим | після + родовий | після їжі |

Самоперевірка:

- Як попросити `десяток яєць`?
- Як сказати лікарю `I have no fever`?
- Як попросити в аптеці `something for a headache`?
