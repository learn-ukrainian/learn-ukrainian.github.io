У попередніх модулях ви вже бачили родовий відмінок після прийменників:
`без друга`, `для сестри`, `до лікаря`, `біля школи`.

Тепер додаємо повну фразу:

> без мого нового друга
>
> для цієї молодої вчительки
>
> біля нашого старого будинку

У родовому змінюється не тільки іменник. Прикметник і займенник теж
мають узгодитися з іменником.

## Прикметник: який? якого? якої?

У родовому відмінку однини прикметники мають дуже впізнавані закінчення.

| Називний | Рід | Родовий |
| --- | --- | --- |
| новий друг | чоловічий | нового друга |
| велике місто | середній | великого міста |
| стара школа | жіночий | старої школи |
| синя сукня | жіночий | синьої сукні |

Для чоловічого і середнього роду форма прикметника однакова:
`-ого` або `-ього`.

> без нового підручника
>
> біля старого будинку
>
> з великого міста
>
> до синього моря

Для жіночого роду:
`-ої` або `-ьої`.

> без нової книги
>
> біля старої церкви
>
> для молодої вчительки
>
> від синьої сумки

Практична порада: спочатку знайдіть іменник і його рід, а потім
поставте прикметник у ту саму форму.

| Крок | Приклад |
| --- | --- |
| Іменник | друг |
| Родовий іменника | друга |
| Прикметник | новий |
| Повна фраза | нового друга |

## Присвійні займенники: мого, моєї

Присвійні займенники працюють майже як прикметники.

| Називний | Родовий чоловічий/середній | Родовий жіночий |
| --- | --- | --- |
| мій / моє / моя | мого | моєї |
| твій / твоє / твоя | твого | твоєї |
| наш / наше / наша | нашого | нашої |
| ваш / ваше / ваша | вашого | вашої |
| їхній / їхнє / їхня | їхнього | їхньої |

Приклади:

> без мого телефона
>
> для твоєї сестри
>
> біля нашого будинку
>
> до вашої лікарні
>
> з їхнього міста

`Його` та `її` як присвійні слова не змінюються:

> його книга - без його книги
>
> її сумка - біля її сумки

Тут змінюється іменник, але `його` / `її` лишаються ті самі.

## Вказівні займенники: цього, цієї

Вказівні займенники `цей` і `той` теж узгоджуються з іменником.

| Називний | Родовий чоловічий/середній | Родовий жіночий |
| --- | --- | --- |
| цей / це / ця | цього | цієї |
| той / те / та | того | тієї |

Приклади:

> біля цього будинку
>
> після того дня
>
> для цієї дівчини
>
> без тієї книги

Повна фраза може мати кілька слів перед іменником:

> біля цього нового ринку
>
> для моєї старшої сестри
>
> без того важливого документа
>
> до нашої нової лікарні

Порядок слів зазвичай простий:

> вказівний займенник + присвійний або прикметник + іменник

Але на А2 не треба будувати дуже довгі фрази. Дві частини перед
іменником уже достатньо:

> цього нового ринку
>
> моєї старшої сестри
>
> тієї старої школи

## Конструктор фраз

Будуйте фразу поступово.

| Крок | Приклад |
| --- | --- |
| Називний | цей новий ринок |
| Родовий іменника | ринку |
| Родовий прикметника | нового |
| Родовий займенника | цього |
| Повна фраза | біля цього нового ринку |

Ще приклади:

| Називний | Родовий з прийменником |
| --- | --- |
| моя молодша сестра | для моєї молодшої сестри |
| наш старий будинок | біля нашого старого будинку |
| той важливий документ | без того важливого документа |
| ця червона парасолька | без цієї червоної парасольки |

Типові помилки:

| Помилка | Краще | Чому |
| --- | --- | --- |
| без нової друга | без нового друга | `друг` чоловічого роду |
| біля цей будинку | біля цього будинку | після `біля` родовий |
| для моєї брата | для мого брата | `брат` чоловічого роду |
| без цього книги | без цієї книги | `книга` жіночого роду |

## 📋 Підсумок

У родовому відмінку повна фраза має узгодження.

| Рід іменника | Прикметник | Присвійний | Вказівний |
| --- | --- | --- | --- |
| чоловічий | нового друга | мого друга | цього друга |
| середній | нового міста | мого міста | цього міста |

Самоперевірка:

- Як сказати `без мій новий друг` правильно?
- Чому `біля цієї школи`, але `біля цього будинку`?
- Напишіть три фрази: з `мого`, з `моєї`, з `цього` або `цієї`.
