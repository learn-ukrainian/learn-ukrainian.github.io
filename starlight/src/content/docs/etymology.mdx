## Що це таке

Кожна стаття показує походження слова — праслов'янський корінь, давньоруські форми,
запозичення з грецької / латини / тюркських мов, і когнати в споріднених мовах (російська,
польська, чеська, болгарська, сербохорватська, словацька, словенська, білоруська).

Дані з **Етимологічного словника української мови (ЕСУМ)**, томи 1–6 (видавництво
«Наукова думка», Київ, 1982–2012). Cтатті індексовано локально, без зовнішніх запитів.

## Як шукати

Поки що — за URL: `/etymology/<слово-латиницею>/`. Транслітерація проста (BGN/PCGN
спрощена): `серце → sertse`, `жінка → zhinka`, `хата → khata`. Слова з омонімією
(наприклад, `мати`) ведуть на сторінку розгалуження, де перелічено всі статті.

Інтерактивний пошук, абетковий перелік та дослідник дерев когнатів — у наступних фазах
(Phase 2, Phase 3).

## Дизайн

Кожна сторінка `/etymology/<slug>/` — це лаконічна HTML-сторінка зі своїми CSS
(легка/темна тема, мобільна верстка), без обтяжливих залежностей. Файли генеруються
скриптом `scripts/etymology/extract_static_pages.py` з канонічної бази
`data/sources.db` і викладаються як статичні асети Astro/Starlight. Цей підхід
обрано після виміру: повний прогін через Starlight content collection падав із
SIGABRT уже на 16 ГБ heap (понад 19 хвилин на повний корпус); статичні файли — 3 с.

Архітектурні рішення задокументовано у `docs/decisions/pending/2026-05-15-etymology-feature-design.md`.
