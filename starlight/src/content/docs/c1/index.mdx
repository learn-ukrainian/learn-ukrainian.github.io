# 🔍 C1 - Просунутий

**На перевірці — 1/106 готово, 105 у QA**

## Просунутий рівень — вільне володіння мовою!

C1 — це рівень, на якому ви думаєте українською. Ви опановуєте стилістичні тонкощі, літературну мову та культурний контекст на рівні освіченого носія.

**Чого ви навчитеся:**
- Стилістика — емфаза, інверсія, риторичні прийоми
- Літературна мова — аналіз класичних текстів
- Культурні модулі — фольклор, звичаї, сучасна культура
- Діалектологія — особливості регіональних варіантів

**Занурення:** 100% українською без будь-яких пояснень англійською
