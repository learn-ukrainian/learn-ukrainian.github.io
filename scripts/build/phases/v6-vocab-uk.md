# V6 Vocabulary Generation — Ukrainian-Canonical Словник

Ви щойно написали український модуль. Тепер складіть словник **для читача, який уже працює українською**, а не для англомовного bridge-модуля.

## Завдання

Прочитайте текст модуля нижче й виведіть YAML-список лексики.

## Головні правила

1. **Даємо українські тлумачення, не англійські переклади.**
2. **Мова словника — 100% українська.** Без English glosses, without “means”, without bilingual labels.
3. **Планова лексика — мінімум.** Усе з `vocabulary_hints` має з'явитися, а також усі нові слова й вирази, які реально введено в модулі.
4. **Тлумачення мають бути короткі й природні.** Стиль: шкільний словник / СУМ-11 light.
5. **Вирази позначайте окремо.** Для фраз ставте `expression: true`.
6. **Не вигадуйте слова, яких немає в модулі або плані.**

## Формат виводу

Виведіть **лише YAML**.

```yaml
vocabulary:
  - word: "абетка"
    definition: "сукупність літер мови в усталеному порядку"
    expression: false
  - word: "Як справи?"
    definition: "звичне запитання про стан або настрій співрозмовника"
    expression: true
```

Дозволені додаткові поля:
- `additional: true` якщо слово не було в plan vocabulary, але було введене у прозі.

## Чого НЕ робити

- НЕ пишіть `translation: "alphabet"` або інші англійські переклади.
- НЕ дублюйте те саме слово кілька разів із різними тлумаченнями.
- НЕ давайте енциклопедичних абзаців замість короткого словникового тлумачення.
