## What an Academic Brief IS

A reference guide for the writer covering:
- Academic Ukrainian conventions for this topic
- Stylistic nuances and register requirements
- Advanced grammar, syntax, or vocabulary systems
- How this topic is taught at Ukrainian university level
- Scholarly debates and authoritative references

```markdown
## Академічний контекст (Academic Context)
Where this topic fits in Ukrainian higher education. What courses cover it, what level of sophistication is expected.

## Основний зміст (Main Content)
The core reference material. For stylistics: register analysis with examples. For academic writing: conventions and patterns. For professional contexts: domain-specific terminology and usage.

## Тонкощі та нюанси (Subtleties)
What makes this topic HARD at C1-C2. Nuances that separate competent from masterful use. Stylistic choices, connotation shifts, register mismatches.

## Наукові дискусії (Scholarly Debates)
Where experts disagree. Multiple positions with citations. The writer should present these fairly.

## Приклади високого рівня (Advanced Examples)
Examples from Ukrainian academic writing, journalism, literature. These demonstrate mastery-level usage.

## Пов'язані статті (Related Articles)
Cross-links to other wiki articles.
```

## Quality Standards
- **Write in Ukrainian.** This is C1-C2 content — full immersion.
- **Academic register.** Match the sophistication expected at university level.
- **Cite Ukrainian scholarship.** Not Western textbooks about Ukrainian.
- **Minimum 1,500 words.**

## Output

Return ONLY the markdown article. No preamble. Start with `# Title`.
