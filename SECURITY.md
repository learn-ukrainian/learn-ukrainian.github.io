# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do NOT open a public issue** for security vulnerabilities
2. Email the maintainer directly or use [GitHub's private vulnerability reporting](https://github.com/learn-ukrainian/learn-ukrainian.github.io/security/advisories/new)
3. Include a description of the vulnerability and steps to reproduce

We will acknowledge your report within 48 hours and aim to release a fix promptly.

## Scope

This is a static educational website. The primary security concerns are:

- **Supply chain attacks** via compromised dependencies
- **Content injection** via pull requests
- **Credential leaks** in committed files

## Supported Versions

Only the latest version on the `main` branch is supported.
